## 1. 전략 배경

### 왜 링크드인이 B2B 컨설팅의 핵심인가?

- **PDF 캐러셀이 텍스트 대비 596% 더 높은 참여율** (Linkboost, 2026)
- **네이티브 비디오가 링크 포스트 대비 5배 참여** (SearchLab, 2026)
- **이미지 포스트 = 텍스트 대비 87% 더 높은 참여, 2.45배 좋아요** (Linkboost)
- **컨설턴트/코치/프랙셔널 리더 = 주 3회 포스팅이 최적** (TrueFuture Media, 2026)
- **2026 알고리즘: Depth Score(체류시간)가 핵심** — 캐러셀과 긴 텍스트가 유리 (Kawaak)
- **인공적 참여(댓글 포드 등) 감지/페널티** — 진짜 가치 제공만 살아남음 (LaGrowthMachine)

### 하우제로의 링크드인 강점

```
[B2B 의사결정자가 링크드인에서 찾는 것]

1. "내 업종에 맞는 실전 인사이트" → 하우제로: 업종별 사례 보유
2. "이론이 아닌 증명된 성과" → 하우제로: 10억 실적
3. "솔직한 경험 공유" → 하우제로: Build in Public
4. "전문가의 역발상" → 하우제로: "AI 80%는 프로세스 문제"

→ 링크드인 알고리즘이 좋아하는 것 = 하우제로가 잘하는 것
```

### 3.3 PDF 캐러셀 제작 (최강 포맷)

```
[캐러셀 제작 워크플로우]

1. Canva에서 LinkedIn 캐러셀 템플릿 사용 (1080×1350 or 정사각형)
2. 브랜드 컬러/폰트 통일 (인스타 캐러셀과 동일 디자인 시스템)
3. 7~10장 구성:
   Slide 1: 훅 ("아직도 CS를 사람이 합니까?")
   Slide 2~8: 핵심 내용 (1슬라이드 = 1포인트)
   Slide 9: 요약 (체크리스트/핵심 정리)
   Slide 10: CTA ("저장📌 + 무료 오딧 → 댓글에 링크")
4. PDF로 내보내기 → LinkedIn에 Document로 업로드

[핵심]: 슬라이드마다 스와이프 유도 문구 ("다음 장에서 →")
→ 스와이프 = 체류시간 ↑ = Depth Score ↑ = 알고리즘 보상
```

## 4. 링크드인 특화 전략

### 4.1 DM 세일즈 파이프라인

```
[링크드인 DM 자연 전환 퍼널]

Step 1: 가치 있는 콘텐츠 → 댓글 유도
  "이 캐러셀이 도움 됐다면 '도움됨'이라고 댓글 남겨주세요"

Step 2: 댓글 단 사람에게 DM (24시간 내)
  "댓글 감사합니다! 혹시 어떤 업종에서 일하시나요?
   업종에 맞는 AI 자동화 사례를 보내드릴까요?"

Step 3: 관심 표현 시 → 가치 먼저 제공
  "참고가 될 만한 [업종] AI 자동화 가이드 PDF 보내드립니다"
  (리드 마그넷 직접 전달)

Step 4: 자연스럽게 오딧 제안
  "혹시 우리 회사에도 적용해볼 수 있을지 궁금하시면
   30분 무료 AI 오딧으로 확인해보시겠어요?"

[원칙]
  • 절대 첫 DM에서 판매하지 않는다
  • 가치를 먼저 주고, 관심을 확인한 후 제안한다
  • "세일즈 느낌"이 나면 즉시 중단
```

### 4.2 ABM(Account-Based Marketing) 미니 전략

특정 타겟 기업의 의사결정자를 링크드인에서 직접 공략:

```
[ABM 워크플로우]

1. 타겟 기업 10개 선정 (이커머스/법률/부동산 중 적합 기업)
2. 각 기업의 대표/팀장 프로필 찾기
3. 그들의 포스트에 가치 있는 댓글 남기기 (2주 워밍업)
4. 연결 요청 + 간단한 자기소개
5. 연결 후 → DM 세일즈 파이프라인 진입
6. 콘텐츠에서 해당 업종 사례 언급 (간접 타겟팅)

→ 월 2~3명 ABM 타겟 → 분기 1건 고가 계약 목표
```

### 4.3 링크드인 뉴스레터 (선택, Q3~)

```
[링크드인 뉴스레터]

링크드인 자체 뉴스레터 기능 활용:
  • 이름: "하우제로 위클리 — 비개발자 AI 자동화"
  • 주기: 격주 (스티비 뉴스레터와 교차)
  • 장점: 링크드인 알림으로 팔로워에게 직접 푸시
  • 구독자 = 링크드인 팔로워와 별도 자산

→ 스티비 뉴스레터(이메일)와 다른 오디언스에 도달
→ 링크드인 전용 B2B 인사이트 중심
```

## 5. 댓글 전략 (Reply Guy 링크드인 버전)

### 5.1 타겟 인플루언서 15명

AI/자동화/비즈니스 분야 한국 링크드인 인플루언서 15명을 선정하고, 그들의 포스트에 가치 있는 댓글을 남긴다.

| 기준 | 설명 |
|------|------|
| 팔로워 | 5,000~50,000명 (대형보다 중형이 댓글 노출 유리) |
| 주제 | AI, 자동화, 디지털 전환, 스타트업, 1인 기업 |
| 활동 | 주 2회 이상 포스팅하는 활성 계정 |

### 5.2 댓글 작성 원칙

```
[좋은 댓글]
"동의합니다. 저도 컨설팅하면서 느끼는 건데,
 AI 도입 실패의 80%는 기술이 아니라
 기본 프로세스가 정리 안 된 게 원인이더라고요.
 특히 CS 쪽에서 이 패턴이 반복됩니다."
→ 구체적 경험 + 추가 인사이트 + 전문가 포지셔닝

[나쁜 댓글]
"좋은 글이네요! 👍"
→ 가치 0, 알고리즘 무시, 시간 낭비
```
