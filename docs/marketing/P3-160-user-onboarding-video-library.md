# P3-160: 사용자 온보딩 영상 라이브러리 — 모든 서비스의 시작 영상 체계화
> P3 | 초기 10만원 | 대표
각 서비스(코스/템플릿/커뮤니티/리테이너)별 온보딩 영상 1개씩 제작(5분). "이 영상 1개만 보면 시작할 수 있습니다." 라이브러리: howzero.com/start에 정리. 영상: ①코스 시작 가이드 ②Claude Code 템플릿 설치법 ③커뮤니티 참여법 ④리테이너 소통법 ⑤AI 진단 활용법. 온보딩 이메일에 해당 영상 링크 자동 삽입. 지원 문의 50% 감소 + Aha Moment 가속. KPI: 온보딩 영상 시청률 70%+, 지원 문의 -50%.
