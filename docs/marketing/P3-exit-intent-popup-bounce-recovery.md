# P3: 이탈 방지 팝업(Exit Intent) + 바운스 복구 — 떠나는 방문자 붙잡기

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: ₩0 | **담당**: 대표

---

## 핵심: 이탈하려는 방문자의 30%를 붙잡으면 → 리드 30% 증가

## Exit Intent 팝업 설계

```
[마우스가 브라우저 상단으로 이동 = 이탈 시도 감지]

┌──────────────────────────────────┐
│  잠깐! 👋                        │
│                                  │
│  떠나시기 전에,                    │
│  AI 자동화 무료 가이드 받아가세요.  │
│                                  │
│  "비개발자를 위한 AI 자동화        │
│   시작 가이드" (PDF, 20페이지)     │
│                                  │
│  [이메일 입력] [무료 다운로드]      │
│                                  │
│  ✕ 괜찮아요, 다음에 올게요        │
└──────────────────────────────────┘
```

## 페이지별 맞춤 팝업

| 페이지 | 팝업 오퍼 |
|--------|----------|
| **홈페이지** | "AI 자동화 시작 가이드 (무료 PDF)" |
| **가격 페이지** | "결정 어려우세요? 무료 AI 오딧 30분으로 확인해보세요" |
| **블로그** | "매주 AI 자동화 팁 → 뉴스레터 구독" |
| **오딧 페이지** (미신청) | "지금 안 해도 괜찮아요. 먼저 AI 준비도만 진단해보세요 (3분)" |
| **코스 페이지** (미구매) | "아직 고민 중? 첫 모듈 무료로 체험해보세요" |

## 구현: Vercel Edge + 자체 JS or OptinMonster 대체

```
[팝업 트리거 조건]
  1. Exit intent (마우스 이탈) — 데스크톱
  2. 스크롤 80%+ (모바일 — exit intent 불가)
  3. 체류 30초+ (너무 빨리 뜨면 짜증)
  4. 세션당 1회만 (반복 표시 금지)
  5. 이미 구독자면 → 팝업 안 뜸 (쿠키 체크)
```

## KPI

| 지표 | 목표 |
|------|------|
| 팝업 노출률 (이탈자 중) | 50%+ |
| 팝업 전환율 | 5~10% |
| 팝업 경유 이메일 수집 | 50+/월 |
| 팝업 경유 오딧 신청 | 5+/월 |
| 전체 사이트 바운스율 개선 | -10% |

## 예산: ₩0 (자체 구현 or 무료 도구)
