## 1. 전략 배경

### 왜 카카오톡인가?

한국에서 **카카오톡 = 인프라**. 이메일 오픈율 20~40%인 반면, 카카오톡 메시지 열람율은 **90%+**. 비개발자 중소기업 대표들의 1차 커뮤니케이션 채널은 카카오톡이다.

| 채널 | 한국 도달률 | 열람률 | 전환 적합성 |
|------|:---:|:---:|:---:|
| 이메일 | 높음 | 20~40% | ★★★ |
| 유튜브 | 높음 | 알고리즘 의존 | ★★ |
| 링크드인 | 중간 (B2B) | 5~10% | ★★★ |
| **카카오톡** | **최고 (95%+)** | **90%+** | **★★★★★** |

### 카카오 비즈보드란?

카카오톡 채팅 목록 상단에 노출되는 네이티브 광고. 한국인이 하루에 수십 번 보는 화면에 직접 노출.

- **노출 위치**: 카톡 채팅탭 최상단 (가장 높은 주목도)
- **과금 방식**: CPM/CPC
- **타겟팅**: 연령, 성별, 관심사, 지역, 카카오 행동 데이터
- **랜딩**: 카카오톡 채널 추가 / 외부 URL / 카카오 비즈폼

## 3. 카카오 비즈보드 광고 전략

### 3.1 3단계 퍼널 (Meta 광고와 동일 구조, 카카오 특화)

```
━━━━━ TOFU: 인지 ━━━━━
  목표: 채널 친구 추가
  타겟: 25~50세, 관심사(창업/마케팅/IT/1인기업)
  소재: "아직도 사람이 CS를 하나요? AI가 80% 대신합니다"
  CTA: 카카오톡 채널 추가
  예산: 월 15만원

━━━━━ MOFU: 관심 ━━━━━
  목표: AI 진단 / 뉴스레터 구독
  타겟: 채널 추가자 (리타겟)
  소재: "3분 AI 준비도 진단 → 맞춤 리포트 무료"
  CTA: AI 준비도 진단 링크
  예산: 월 10만원

━━━━━ BOFU: 전환 ━━━━━
  목표: 무료 오딧 신청
  타겟: 진단 완료자 / 채널 활성 사용자
  소재: "무료 AI 오딧 30분 — 이번 달 5슬롯만"
  CTA: Cal.com 예약
  예산: 월 5만원
```

### 3.2 비즈보드 크리에이티브

| 소재 | 카피 | 이미지 | 단계 |

## 4. 카카오톡 메시지 자동화 (Claude Code)

### 4.1 생애주기별 카카오톡 알림톡

| 트리거 | 메시지 | 발송 시점 |
|--------|--------|----------|
| **채널 추가** | 웰컴 메시지 + 메뉴 안내 | 즉시 |
| **AI 진단 완료** | "결과: [X]점! 개선 가이드 → [링크]" | 즉시 |
| **오딧 예약** | "예약 확인! [날짜/시간]. 사전 설문 → [링크]" | 즉시 |
| **오딧 D-1** | "내일 오딧입니다! 준비사항: [체크리스트]" | D-1 |
| **오딧 완료** | "오딧 리포트 이메일 발송됨. 궁금하신 점은 카톡 주세요" | D+0 |
| **유료 제안** | "맞춤 자동화 로드맵 → [제안서 링크]" | D+3 |
| **뉴스레터 발행** | "이번 주 AI 자동화 팁 → [블로그 링크]" | 매주 화 |
| **웨비나 알림** | "오늘 저녁 8시 라이브! 참여 → [Zoom 링크]" | D-0 |
| **비활성 재참여** | "최근 놓친 AI 자동화 팁 3가지 → [링크]" | 30일 비활성 |

### 4.2 Claude Code 카카오톡 알림톡 워크플로우

```
[Claude Code → 카카오 알림톡 API]

1. Notion CRM에서 트리거 이벤트 감지
2. 해당 단계의 알림톡 템플릿 선택
3. 카카오 알림톡 API (Solapi/NHN 등) 호출
4. 발송 결과 → Notion DB 기록
5. 실패 시 → 이메일 폴백 발송
```

### 4.3 카카오 알림톡 vs 이메일 병행

```
[멀티 채널 시퀀스]

이벤트 발생
    → 카카오톡 알림톡 (즉시, 열람률 90%+)
    → 이메일 (동시 or 1시간 후, 상세 내용)
    → 카톡 미열람 시 → 24시간 후 SMS 폴백

→ 도달률 극대화: 카톡 90% + 이메일 40% + SMS 95%
→ 중복 제거: 카톡 열람 시 이메일/SMS 자동 스킵
```
