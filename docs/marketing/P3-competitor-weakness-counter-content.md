# P3: 경쟁사 약점 카운터 콘텐츠 — 간접 비교로 하우제로 선택 유도

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: ₩0 | **담당**: 대표

---

## 핵심: 경쟁사를 직접 언급하지 않으면서 약점을 부각하는 콘텐츠

## 원칙: 경쟁사 이름 절대 언급 안 함. "일반적인 문제"로 프레이밍.

## 카운터 콘텐츠 매트릭스

| 경쟁자 약점 | 카운터 콘텐츠 | 하우제로 강점 |
|-----------|-------------|-------------|
| 대형 컨설팅 (수억원) | "AI 컨설팅에 수억 쓸 필요 없는 이유" | 200만원으로 같은 효과 |
| 노코드 강사 (이론만) | "강의만 듣고 실행 못 하는 이유" | 교육+구축+운영 올인원 |
| 프리랜서 (일회성) | "일회성 외주의 숨은 비용" | 지속 지원+내재화 |
| AI SaaS (도구만) | "도구만 있으면 자동화가 될까?" | 전략+도구+실행 통합 |
| RPA 업체 (개발자 필요) | "개발자 없이 자동화하는 법" | 비개발자 전문 |

## 콘텐츠 예시

```
[유튜브] "AI 컨설팅 고를 때 피해야 할 5가지 함정"
  함정 1: "이론만 알려주고 실행은 안 해주는 곳" (← 강사 약점)
  함정 2: "구축만 하고 교육/이관은 안 하는 곳" (← 프리랜서 약점)
  함정 3: "비용만 비싸고 중소기업을 무시하는 곳" (← 대형 컨설팅)
  함정 4: "도구만 팔고 전략은 모르는 곳" (← SaaS)
  함정 5: "개발자가 없으면 못 쓰는 솔루션" (← RPA)

  → "이 5가지 함정을 피하려면? 하우제로는 이렇게 다릅니다."
```

## 톤 가이드

```
❌ "A 컨설팅은 비싸기만 하고 쓸모없다" (비방 — 절대 금지)
✅ "고가 컨설팅이 항상 효과적인 건 아닙니다" (일반적 문제 제기)

❌ "B 강사는 이론만 가르친다" (개인 공격)
✅ "강의만으로는 실행까지 가기 어렵습니다" (구조적 한계 지적)
```

## 활용

| 콘텐츠 | 포맷 |
|--------|------|
| "AI 컨설턴트 고르는 법 5가지" | 유튜브 롱폼 + 블로그 |
| "AI 도입 외주 vs 내재화, 뭐가 나을까?" | 캐러셀 + 링크드인 |
| "저렴한 AI 자동화가 더 비싼 이유" | 숏폼 + 뉴스레터 |
| "도구만 사면 자동화가 될까? (역발상)" | 숏폼 |
