# P3-144: Notion 퍼블릭 페이지 포트폴리오 — 투명한 시스템 공개
> P3 | ₩0 | 대표
하우제로의 Notion CRM/캘린더/프로세스를 퍼블릭 읽기 전용으로 공개(민감정보 제외). "이게 저희가 실제로 쓰는 시스템입니다" → 컨설팅 데모 + 콘텐츠 소재. 공개 항목: 콘텐츠 캘린더(일정만), AI 도구 추천 맵, 프로세스 맵핑 템플릿, 자동화 로드맵 템플릿. 비공개: 고객 데이터, 매출, 개인정보. Notion → Share → 퍼블릭 링크 → howzero.com/systems에 임베드. KPI: 퍼블릭 페이지 월 300+ 방문, "이런 시스템 만들어달라" 문의 3건+/월.
