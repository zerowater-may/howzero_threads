## 1. 전략 배경 & 리서치 인사이트

### 왜 PR/퍼스널 브랜딩인가?

하우제로의 모든 퍼널(숏폼, 이메일, 파트너, SEO, 광고)의 전환율은 **"이 사람 믿을 수 있나?"**에 달려 있다. PR/미디어 노출은 이 신뢰를 **한 방에 올려주는 가속기**다.

```
[신뢰 수준과 전환율 상관관계]

"유튜브에서 본 사람"         → 오딧 신청 전환율 3%
"뉴스레터 구독 중인 사람"    → 오딧 신청 전환율 7%
"언론에 나온 사람"           → 오딧 신청 전환율 15%+
"팟캐스트에서 1시간 들은 사람" → 오딧 신청 전환율 20%+
```

### 핵심 데이터

- **팟캐스트 게스팅은 복리형 가시성을 만든다** — 한 번의 인터뷰가 수년간 리드/기회/브랜드 인지 생성 (Expert Bookers, 2026)
- **90일 만에 업계 권위를 구축하는 파운더 가이드**: 팟캐스트 게스팅이 가장 빠른 전문가 포지셔닝 수단 (PodcastVideos, 2026)
- **미러 오디언스 팟캐스트에 출연하라** — 타겟이 겹치는 팟캐스트에 반복 출연하면 권위 + 전환 동시 달성 (Smooth Business Growth, 2026)
- **Earned Media가 AEO(AI 검색 최적화)에 직결** — 팩트 기반·구조화된·교차 링크된 스토리가 AI 인용 확률 극대화 (Inkhouse, 2026)
- **파운더 취약성(Vulnerability) 공개가 신뢰 신호** — 실패 스토리가 자동화된 콘텐츠와 차별화하는 핵심 (PodcastVideos, 2026)
- **Founder-Led Content가 2026 대세** — CEO가 곧 인플루언서. 개인 브랜드 = 회사 브랜드 (Topteny, 2026)

### 하우제로의 PR 자산

|---------|-----------|
| **연매출 10억 달성** | "코딩 모르는 비개발자가 GPT-3로 1년 만에 10억 SaaS 구축" |
| **CX/CS 자동화 실전** | "AI로 CS 직원 5명분 대체한 실제 사례" |

| **실패 스토리** | "10억 벌기 전에 망한 3가지 시도" (취약성 공개) |

## 2. 3대 PR 채널 전략

### 2.1 팟캐스트 게스팅 (최우선)

#### 왜 팟캐스트가 1순위인가?

| 장점 | 설명 |
|------|------|
| **깊은 신뢰 구축** | 30~60분 대화 → 유튜브 숏폼 100개보다 강한 신뢰 |
| **복리형 자산** | 에피소드가 몇 년간 검색/재생되며 리드 생성 |
| **AEO 시너지** | 팟캐스트 트랜스크립트가 AI 검색 인용 소스가 됨 |
| **무료** | 비용 0원. 시간만 투자 |
| **파트너십 문 열기** | 호스트와 관계 구축 → B2B 파트너/콜라보 자연 연결 |

#### 타겟 팟캐스트 카테고리

| 카테고리 | 예시 | 왜 시너지가 나는가 |
|---------|------|-----------------|
| **비즈니스/창업** | 스타트업 스토리, 창업가 인터뷰 팟캐스트 | "10억 달성 비하인드" 스토리 |
| **AI/테크** | AI 트렌드, 노코드 자동화 팟캐스트 | "비개발자 AI 자동화 실전" |
| **1인 기업/부업** | 프리랜서, 사이드 프로젝트 팟캐스트 | "AI로 시간 확보하는 법" |
| **이커머스/마케팅** | 온라인 셀러, 디지털 마케팅 팟캐스트 | "CS/소싱 자동화 사례" |
| **경영/HR** | 기업 혁신, 디지털 전환 팟캐스트 | "AI 오딧으로 조직 효율화" |

[호스트명]님, 안녕하세요.

## 3. Founder-Led Content 전략

### 3.1 대표 = 브랜드 포지셔닝

2026년 트렌드: **CEO가 곧 인플루언서**. 하우제로의 모든 PR은 "하우제로"가 아니라 **대표 개인의 스토리**로 풀어야 한다.

| 전통 기업 PR | Founder-Led PR (하우제로) |
|------------|------------------------|
| "하우제로가 AI 오딧을 런칭했다" | "코딩 모르던 내가 10억 벌고 AI 오딧을 만든 이유" |
| 3인칭 보도자료 | 1인칭 여정 공유 |
| 기업 로고 중심 | 대표 얼굴 중심 |
| 제품 기능 나열 | 실패 → 성공 스토리 아크 |

### 3.2 대표 미디어 킷

모든 미디어 출연/기고를 위한 표준 자료 세트:

```
[하우제로 대표 미디어 킷]

1. 프로필 사진 (고해상도, 2종 — 정면/반측면)
2. 약력 (100자/300자/500자 3가지 버전)
3. 핵심 토킹 포인트 5가지
4. FAQ (자주 묻는 질문 + 답변 10개)
5. 로고 파일 (PNG/SVG)
6. 실적 데이터 요약 1-pager
7. 이전 미디어 출연 리스트 (포트폴리오)
8. 연락처 + 스케줄링 링크
```

| 미디어 | 기고 주제 | 목표 |

## 4. PR → 비즈니스 전환 퍼널

```
[PR 노출]
├── 팟캐스트 출연 → 쇼노트에 CTA 링크
├── 언론 보도 → 기사에 howzero.com 링크
├── 어워드 수상 → 보도자료 + SNS + 뉴스레터
└── 기고문 → 본문 내 AI 준비도 진단 CTA
        │
        ▼
[신뢰 폭발]
    "언론에도 나온 사람이구나"
    "팟캐스트에서 1시간 들어봤는데 진짜 실력자네"
    "어워드도 받았어? 믿을 수 있겠다"
        │
        ▼
[기존 퍼널 전환율 부스트]
    모든 채널(숏폼, 이메일, 광고, 파트너)의
    전환율이 PR 권위 덕분에 1.5~2배 상승
        │
        ├── AI 준비도 진단 → 이메일 수집
        ├── 뉴스레터 구독
        ├── 무료 AI 오딧 30분 신청
        └── 유료 컨설팅/코스 구매
```

### PR 자산의 2차 활용

| PR 산출물 | 2차 활용 |
|----------|---------|
| 팟캐스트 에피소드 | 유튜브 롱폼 업로드 + 숏폼 클립 10개 추출 |
| 언론 기사 | 웹사이트 "미디어" 섹션 + 이메일 서명 + 광고 크리에이티브 |
| 어워드 로고 | 랜딩페이지 신뢰 배지 + 뉴스레터 헤더 |
| 기고문 | 블로그 재게시 + AEO 백링크 |
| 미디어 인터뷰 클립 | BOFU 광고 소재 (사회적 증거) |
