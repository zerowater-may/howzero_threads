## 1. 전략 배경

### 왜 정부 지원사업인가?

**하우제로의 가장 큰 세일즈 장벽**: "좋은 건 알겠는데, 예산이..."

정부 바우처가 이 장벽을 **제거**한다:

```
[바우처 없이]
  기업: "AI 컨설팅 200만원이요? 예산 없어요..."
  → 전환 실패

[바우처 있으면]
  기업: "AI 바우처로 70% 지원 받으면 60만원이네? 해볼게요!"
  → 전환 성공
  → 하우제로: 200만원 전액 수령 (70%는 정부가 지불)
```

### 2026년 주요 AI/디지털 바우처

| 사업명 | 주관 | 기업당 지원 | 지원 비율 |
|--------|------|-----------|----------|
| **AI 바우처 지원사업** | NIPA(정보통신산업진흥원) | 최대 2억원 | 70~90% |
| **AI 통합 바우처** | NIPA | 최대 3억원 | 70~80% |
| **데이터 바우처** | K-DATA(한국데이터산업진흥원) | 최대 7,000만원 | 70~90% |
| **산업맞춤형 혁신바우처** | NIPA | 과제별 상이 | 70~80% |
| **중소기업 디지털 전환** | 중기부 | 최대 5,000만원 | 50~70% |
| **K-디지털 트레이닝** | 고용부 | 교육비 전액 | 100% |

### 핵심 숫자

- **AI 바우처 공급기업 등록**: NIPA pms.ai-voucher.or.kr
- **수요기업당 최대 2억원**: 하우제로 서비스 전액을 바우처로 결제 가능
- **수요기업 부담: 10~30%만**: 200만원 프로젝트 → 기업 실부담 20~60만원

## 3. 수요기업 연결 전략

### 3.1 수요기업 발굴 채널

```
[수요기업 찾는 곳]

1. NIPA 바우처 매칭 플랫폼 (공급-수요 매칭 시스템)
2. 하우제로 기존 오딧 고객 → "바우처로 비용 절감 가능해요"
3. 뉴스레터/유튜브 CTA → "AI 바우처로 70% 할인받기"
4. 네이버 블로그 SEO → "AI 바우처 신청 방법" 키워드
5. 링크드인 → 중소기업 대표 대상 "바우처 활용법" 콘텐츠
6. 파트너(강사/코치) → 수강생 기업에 바우처 안내
7. 정부 지원사업 설명회 참석 → 네트워킹
```

### 3.2 바우처 마케팅 콘텐츠

| 콘텐츠 | 채널 | 목적 |
|--------|------|------|
| "AI 바우처로 AI 도입 비용 70% 줄이는 법" | 유튜브 롱폼 | SEO + 교육 |
| "AI 바우처 신청 가이드 2026 (5분 정리)" | 숏폼 | 검색 유입 |
| "AI 바우처 활용 실제 사례 — 200만원으로 2억 절감" | 블로그 | SEO |
| "우리 회사도 AI 바우처 받을 수 있을까?" | 캐러셀 | 인스타/링크드인 |
| "AI 바우처 Q&A — 자주 묻는 질문 10가지" | 뉴스레터 | 기존 구독자 전환 |

### 3.3 기존 오딧 고객 → 바우처 연결

```
[오딧 완료 후 유료 제안 시]

기존: "스탠다드 패키지 ₩2,000,000입니다"
  → 일부 기업: "예산이..."

바우처 버전: "스탠다드 패키지 ₩2,000,000인데,
  AI 바우처를 활용하면 귀사 부담은 ₩400,000~600,000입니다.
  바우처 신청도 저희가 도와드립니다."
  → 전환율 3배+ 예상

[바우처 신청 대행]
  수요기업이 바우처 신청 절차를 모르는 경우가 대부분
  → 하우제로가 신청서 작성/제출까지 대행
  → "원스톱" 서비스 = 추가 가치
```
