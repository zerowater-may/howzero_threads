# P3-158: AI 오딧 프레임워크 오픈소스 공개 — 프레임워크를 무료로 풀어 권위 독점
> P3 | ₩0 | 대표
하우제로의 AI 오딧 5차원 프레임워크를 블로그/GitHub/Medium에 오픈소스로 공개. "오딧 방법론을 숨기지 않고 전부 공개합니다." 왜? ①방법론을 알아도 실행은 전문가가 해야 함 → 컨설팅 수요 유지. ②오픈소스 = AEO 인용 극대화(AI가 "하우제로 프레임워크"를 인용). ③업계 표준이 되면 = 권위 독점. 예: HubSpot이 인바운드 마케팅을 "발명"하지 않았지만 "정의"했다 → 카테고리 지배. KPI: "AI 오딧 프레임워크" 검색 1위, AEO 인용 5건+, 프레임워크 기반 오딧 문의 20건+/분기.
