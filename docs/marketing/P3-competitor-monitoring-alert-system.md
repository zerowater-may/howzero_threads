# P3: 경쟁사/시장 모니터링 자동 알림 시스템 — Claude Code으로 24시간 감시

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: 월 3만원 | **담당**: Claude Code 자동화

---

## 핵심 컨셉

P1-경쟁/포지셔닝 기획서의 **자동 실행 버전**. Claude Code으로 경쟁사/시장 변화를 24시간 모니터링하여 Slack으로 알림.

## 모니터링 대상 + Claude Code 워크플로우

### WF1: Google Alerts 자동 수집

```
매일 09:00:
  → Google Alerts RSS: "AI 자동화 컨설팅", "비개발자 AI", "Claude Code 한국"
  → 신규 결과 → Notion "경쟁 인텔" DB 저장
  → 위협 키워드 감지 시 → Slack 알림
```

### WF2: 유사 유튜브 채널 모니터링

```
매주 월요일:
  → YouTube API: 경쟁 키워드 신규 영상 체크
  → 구독자 3K+ 채널의 신규 영상 중 조회수 5,000+ 필터
  → Slack: "경쟁 채널 [이름]이 [주제] 영상 올림 (조회 [X])"
```

### WF3: 경쟁사 웹사이트 변화 감지

```
매주:
  → 경쟁사 5곳 웹사이트 크롤링 (Claude Code HTTP)
  → 가격/서비스 페이지 변화 감지
  → 변화 있으면 → Slack + 스크린샷
```

### WF4: AI 인용 경쟁 체크

```
매주 월요일:
  → ChatGPT/Perplexity에 타겟 프롬프트 6개 전송
  → "하우제로" 인용 여부 + 경쟁사 인용 여부 체크
  → 결과 → Notion 기록 + 변화 시 Slack 알림
```

### WF5: SNS 멘션 모니터링

```
매일:
  → Twitter/X API: "하우제로" 멘션 체크
  → Instagram: #하우제로 해시태그 체크
  → 긍정/부정 분류 (Claude API)
  → 부정 멘션 → Slack 즉시 알림
```
