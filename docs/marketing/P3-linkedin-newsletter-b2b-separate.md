# P3: 링크드인 뉴스레터 (별도 채널) — 링크드인 내 Owned Audience

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: ₩0 | **담당**: 대표

---

## 핵심: 링크드인 자체 뉴스레터 = 링크드인 팔로워에게 직접 알림 푸시

링크드인 뉴스레터는 일반 포스트와 다르게 **구독자에게 이메일+인앱 알림**으로 전달. 알고리즘 무관하게 도달 가능.

## 뉴스레터 설계

| 항목 | 설계 |
|------|------|
| **이름** | "하우제로 B2B AI 인사이트" |
| **주기** | 격주 수요일 (스티비 뉴스레터와 교차) |
| **톤** | B2B 의사결정자 대상 — 데이터 기반, 전략적 |
| **길이** | 800~1,200자 (3분 읽기) |

## 콘텐츠 차별화

| | 스티비 뉴스레터 (P2) | 링크드인 뉴스레터 |
|---|---|---|
| 타겟 | 전체 (개인+기업) | B2B 의사결정자 |
| 톤 | 실용적 팁 중심 | 전략적 인사이트 중심 |
| CTA | 오딧/코스/템플릿 | 기업 교육/컨설팅/오딧 |
| 소스 | 오리지널 | 스티비 내용을 B2B 앵글로 리프레이밍 |

## 주제 예시 (격주)

| # | 주제 |
|---|------|
| 1 | "중소기업 대표가 AI 도입 전에 반드시 읽어야 할 3가지" |
| 2 | "AI 오딧 실제 결과: 50개 기업 평균 데이터 공개" |
| 3 | "개발자 없는 회사의 AI 전환, 현실적 로드맵" |
| 4 | "AI 바우처로 컨설팅 비용 70% 절감하는 법" |
