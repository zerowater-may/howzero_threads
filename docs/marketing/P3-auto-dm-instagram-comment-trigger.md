# P3: 인스타 자동 DM + 댓글 트리거 — "댓글에 '방법' 쓰면 DM으로 보내드립니다"

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: 월 5만원 | **담당**: 대표+ManyChat

---

## 핵심: 릴스/포스트 댓글 키워드 → 자동 DM 발송 → 리드 수집

```
[릴스 CTA] "이 Claude Code 템플릿 받고 싶으면 댓글에 '템플릿'이라고 쓰세요!"

[사용자] 댓글: "템플릿"

[ManyChat 자동 DM]
"안녕하세요! 🎉 요청하신 Claude Code 템플릿 링크입니다:
→ [Gumroad 무료 다운로드 링크]

혹시 우리 회사에도 AI 자동화를 적용하고 싶으시면:
→ 무료 AI 오딧 30분: [링크]

매주 AI 자동화 팁 받기:
→ 뉴스레터 구독: [링크]"
```

## 왜 효과적인가

1. **댓글 폭발** → 알고리즘이 "이 포스트 인기!" → 노출 확대
2. **DM = 1:1 대화 시작** → 인스타 알고리즘 "관계 형성" 신호
3. **리드 수집** → DM으로 이메일/링크 전달 → 퍼널 편입
4. **자동화** → 대표 수동 DM 불필요

## 도구: ManyChat ($15/월) or 소셜비즈

## 트리거 키워드 목록

| 키워드 | 자동 DM 내용 |
|--------|-------------|
| "템플릿" | Claude Code 무료 템플릿 링크 |
| "방법" | 해당 영상 상세 가이드 링크 |
| "오딧" | 무료 AI 오딧 예약 링크 |
| "코스" | 셀프페이스 코스 안내 |
| "가격" | 서비스 가격 페이지 링크 |
