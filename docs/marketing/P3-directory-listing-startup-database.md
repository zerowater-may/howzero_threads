# P3-111: 스타트업/AI 디렉토리 등록 — 무료 노출 채널 극대화
> P3 | ₩0 | 대표
하우제로를 AI/스타트업/컨설팅 디렉토리에 등록하여 검색 노출 + 백링크 확보.

등록 대상:
- Product Hunt (런칭)
- Indie Hackers (프로필)
- BetaList / OpenHunts
- Crunchbase (회사 프로필)
- AngelList / Wellfound
- AI Tool Directory (futuretools.io, theresanaiforthat.com)
- 한국: 로켓펀치, 원티드스페이스, 스타트업위키
- Google 비즈니스 프로필
- Clutch / DesignRush (컨설팅)

각 디렉토리: 설명 + 로고 + 웹사이트 링크 + 키워드 최적화.
1회 등록 → 영구 무료 노출 + 백링크 DA 향상.
KPI: 15개+ 디렉토리 등록, 디렉토리 경유 월 30+ 유입.
