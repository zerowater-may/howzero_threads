# P3-114: 틱톡 전용 최적화 — 유튜브 쇼츠 리포맷이 아닌 틱톡 네이티브 전략
> P3 | 월 5만원 | 대표

## 핵심: 유튜브 쇼츠 리포맷만으로는 틱톡에서 터지지 않는다. 틱톡 네이티브 전략 필요.

## 유튜브 쇼츠 vs 틱톡 차이

| | 유튜브 쇼츠 | 틱톡 |
|---|---|---|
| 톤 | 교육적/전문적 | 캐주얼/유머/리얼 |
| 편집 | 깔끔한 편집 | 로우파이/자연스러운 |
| 훅 | 정보형 ("이거 아세요?") | 감정형 ("이거 보고 충격받음") |
| 음악 | 없거나 배경 | 트렌딩 사운드 필수 |
| 댓글 문화 | 진지 | 밈/유머/자기주장 |

## 틱톡 네이티브 포맷

| 포맷 | 예시 |
|------|------|
| **스티치/듀엣** | 다른 AI 틱톡에 "비개발자 관점에서는..." 스티치 |
| **트렌드 사운드** | 인기 사운드에 AI 자동화 영상 매치 |
| **POV** | "POV: 사장님이 AI 자동화 하라고 하는데 개발자가 없다" |
| **스토리타임** | "코딩 1도 모르던 내가 10억 번 이야기 #파트1" |
| **그린스크린** | 뉴스 기사/데이터 위에 해설 |
