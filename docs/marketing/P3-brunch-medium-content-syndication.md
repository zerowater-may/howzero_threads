# P3: 브런치/Medium 콘텐츠 신디케이션 전략 — 블로그 플랫폼으로 SEO + 권위 확장

> **작성일**: 2026-03-29 | **우선순위**: P3 (중간 — 기존 콘텐츠 재배포)
> **기간**: 2026 Q2 (5월 시작)
> **예산**: 월 5만원
> **담당**: 하우제로 대표 + AI 콘텐츠 엔진

---

## 1. 전략 배경

### 왜 브런치 + Medium인가?

| 플랫폼 | 장점 | 타겟 |
|--------|------|------|
| **브런치** | 네이버 검색 노출, 한국 비즈니스 독자, 작가 권위 | 한국 B2B/비즈니스 |
| **Medium** | 구글/AI 검색 노출, 글로벌, 영문 AEO | 글로벌 + AEO |

### 핵심: 신규 작성이 아닌 "재배포"

```
기존 콘텐츠 자산:
  유튜브 대본 → 블로그화 (이미 P2-SEO에서 진행)
  뉴스레터 → 브런치/Medium 재게시
  링크드인 포스트 → 확장판 기고

→ 추가 작성 시간: 주 1시간 (편집+포맷팅만)
→ 노출 면적: 2~3배 확장
```

## 4. 신디케이션 워크플로우 (Claude Code 자동화)

```
[유튜브 롱폼 발행]
    ↓ Claude Code 트리거
[Claude API: 대본 → 브런치 포맷 변환]
    → 한국어, 브런치 톤, 이미지 위치 표시
    → 대표 5분 검토 → 브런치 발행
    ↓
[Claude API: 대본 → Medium 영문 변환]
    → 영문 로컬라이제이션 (번역 아님)
    → 대표 5분 검토 → Medium 발행
    ↓
[자동 크로스링크]
    브런치 글 → howzero.com 링크 삽입
    Medium 글 → howzero.com/en 링크 삽입
    → DA 향상 + 트래픽 유입
```
