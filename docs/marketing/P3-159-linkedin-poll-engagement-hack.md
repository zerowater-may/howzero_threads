# P3-159: 링크드인 투표(Poll) 참여 해킹 — 가장 쉬운 참여 유도 포맷
> P3 | ₩0 | 대표
링크드인 투표 = 클릭 1번으로 참여 가능 = 가장 낮은 참여 장벽. 주 1회 투표 게시. 예: "비개발자에게 가장 필요한 AI 도구는? ①Claude Code ②Make.com ③Zapier ④ChatGPT API". 투표 결과 → 다음 콘텐츠 주제로 활용 → "여러분이 뽑은 [1위]를 이번 주에 다룹니다." 투표 = 알고리즘 참여 신호 → 다음 포스트 노출↑. 투표 참여자 = 관심 있는 사람 → DM 파이프라인(#29) 타겟. KPI: 투표당 참여 50+, 투표 경유 프로필 방문 30+.
