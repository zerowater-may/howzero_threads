# P3: 전략적 커피챗/네트워킹 — 주 2회 1:1 미팅으로 기회 발굴

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: 월 10만원 | **담당**: 대표

---

## 핵심: 주 2회 × 15분 커피챗 = 연 100명과 1:1 관계 구축

## 커피챗 타겟 4그룹

| 그룹 | 목적 | 빈도 |
|------|------|:---:|
| **잠재 고객** | 오딧 제안 + 니즈 파악 | 주 1회 |
| **파트너 후보** | 콜라보/크로스프로모 | 격주 |
| **업계 전문가** | 인사이트 교환 + 추천 | 월 1회 |
| **기자/미디어** | PR 관계 구축 | 월 1회 |

## 커피챗 프로세스

```
1. Notion "커피챗 타겟" DB에 후보 추가
2. 링크드인/이메일 DM: "15분 커피챗 하실 수 있나요?"
3. Cal.com 예약 (15분 슬롯)
4. 미팅: 10분 상대 이야기 듣기 + 5분 하우제로 소개
5. 미팅 후 24시간 내: 감사 이메일 + 약속한 자료 전달
6. Notion에 미팅 메모 + 후속 액션 기록

[원칙]
  "무엇을 팔 수 있을까"가 아니라 "어떻게 도울 수 있을까" 마인드
  상대에게 먼저 가치를 제공 → 관계 → 나중에 기회
```

## ROI: 연 100명 × 10% 기회 전환 = 10건 비즈니스 기회

## KPI: 주 2회 커피챗, 월 1건 비즈니스 기회 발생
