# P3: 바이럴 콘텐츠 리버스 엔지니어링 — MrBeast식 패턴 분석 시스템

> **우선순위**: P3 | **예산**: 월 3만원 | **담당**: 대표+Claude Code

## 핵심: 경쟁/유사 채널의 바이럴 영상을 분석하여 성공 패턴을 추출 → 하우제로에 적용

> "MrBeast는 바이럴 패턴을 스프레드시트로 분석했다" — Leaked MrBeast Document

## Claude Code 자동 바이럴 분석 워크플로우

```
매주 월요일:
  → YouTube API: AI/자동화/비즈니스 카테고리 인기 영상 TOP 50 수집
  → 필터: 조회수 10,000+ & 구독자 대비 조회 비율 5x+
  → Claude API 분석:
    각 영상의:
    1. 제목 패턴 (숫자? 질문? 도발?)
    2. 썸네일 패턴 (텍스트? 얼굴? Before/After?)
    3. 첫 10초 훅 패턴 (성과? 질문? 충격?)
    4. 영상 길이
    5. CTA 유형
  → 패턴 요약 → Notion "바이럴 패턴 DB" 저장
  → 주간 TOP 3 패턴 → Slack 알림
```

## 패턴 DB 구조

| 필드 | 예시 |
|------|------|
| 패턴 ID | VP-042 |
| 패턴 유형 | "숫자 + 도발" 제목 |
| 예시 | "90%가 모르는 AI 자동화 비밀 5가지" |
| 발견 빈도 | 이번 주 3회 |
| 적용 예시 | "90%가 모르는 CS 자동화 비밀 5가지" |
| 사용 여부 | 미사용 → 이번 주 적용 예정 |

## 활용 루틴

```
매주 화요일 (콘텐츠 배칭 데이):
  1. Slack에서 주간 TOP 3 패턴 확인
  2. 패턴 DB에서 미사용 패턴 선택
  3. 하우제로 버전으로 변형
  4. 대본 템플릿(P1-#45)에 적용
  5. 촬영 → 게시
  6. 성과 기록 → 패턴 DB 업데이트
```
