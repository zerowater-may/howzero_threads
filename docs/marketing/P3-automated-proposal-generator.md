# P3: 제안서 자동 생성 시스템 — 오딧 결과 → AI가 맞춤 제안서 즉시 생성

> **우선순위**: P3 | **예산**: 월 3만원 | **담당**: 대표+Claude Code+Claude

---

## 핵심: 오딧 완료 후 제안서 작성에 2시간 → AI 자동화로 10분

## Claude Code 제안서 자동 생성 워크플로우

```
[오딧 완료 → Notion CRM 업데이트]
  → Claude Code 트리거
  → 오딧 데이터 추출: 업종, 규모, AI 준비도 점수, 핵심 비효율 3개, 예산
  → Claude API: "이 데이터 기반으로 맞춤 컨설팅 제안서를 작성해줘"

[프롬프트]
"다음 오딧 결과를 기반으로 하우제로 컨설팅 제안서를 작성해줘:
 고객사: [이름]
 업종: [업종]
 규모: [규모]
 AI 준비도: [점수]/90
 핵심 비효율:
   1. [비효율1] — 주 [X]시간 소모
   2. [비효율2] — 월 [Y]원 낭비
   3. [비효율3]

 구조:
 1. 진단 요약 (1페이지)
 2. 추천 솔루션 3가지 (각 1페이지)
 3. 3-티어 패키지 비교표
 4. ROI 예측 (투자 vs 절감)
 5. 타임라인
 6. 다음 단계 CTA

 톤: 하우제로 — 직설적, 데이터 기반, 과장 없이"

  → Claude 출력 → Notion에 저장
  → 대표 10분 검토/수정
  → PDF 자동 변환 (Claude Code + Puppeteer)
  → 고객에게 이메일 자동 발송
```

## 효과

```
현재: 오딧 → 제안서 수동 작성 (2시간) → 발송
자동화 후: 오딧 → AI 제안서 (5분) → 검토 (10분) → 자동 발송

절감: 건당 1시간 45분 → 월 10건 = 17.5시간 절감
```

## KPI: 제안서 발송 D+0 (오딧 당일), 제안서 작성 시간 -85%
## 예산: 월 ₩30,000 (Claude API)
