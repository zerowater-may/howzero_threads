# P3: 기업 교육(B2B Training) + 사내 AI 아카데미 패키지 — 개인 컨설팅을 넘어 기업 단위 교육 수익원

> **작성일**: 2026-03-29 | **우선순위**: P3 (중간 — 개인 사례 축적 후 기업 시장 진입)
> **기간**: 2026 Q4 파일럿, 2027 Q1~ 본격
> **예산**: 초기 50만원 + 월 20만원
> **담당**: 하우제로 대표 (커리큘럼 설계/강의)

---

## 1. 전략 배경

### $400B 기업 교육 시장의 AI 전환

- **글로벌 기업 교육 시장 $4,000억** — AI가 이 시장을 근본적으로 재편 중 (Josh Bersin, 2026.02)
- **95% AI 파일럿이 실패** — 기술이 아니라 **인력의 AI 리터러시 부족**이 원인 (Gloat, 2026)
- **FedEx: 40만명에게 AI 교육** — 대기업도 AI 리터러시가 급선무 (CNBC, 2026.03)
- **한국 정부: 공장 AI 통합 프로그램 발표** — AI 인력 양성 국가 전략 (OurAnimeWorld, 2026)
- **효과적인 AI 교육 = 강의가 아닌 실습** — 수동 학습이 아닌 실전 적용 중심 (DataCamp, 2026)
- **2026년 기업 교육 트렌드: 마이크로 모듈 + 실시간 코칭 + 워크플로우 내 학습** (TechClass)

### 왜 하우제로가 기업 교육을 해야 하는가?

```
[하우제로의 기업 교육 적합성]

1. 실전 검증: 연매출 10억 SaaS + 다수 컨설팅 사례
2. 비개발자 전문: 기업의 95%는 비개발자 — 정확한 타겟
3. 교육+구축 올인원: 교육 후 "이제 뭐 하죠?" → 컨설팅 자연 연결
4. 도구 불가지론: 특정 도구가 아닌 "AI 자동화 사고방식" 교육
5. 콘텐츠 자산: 유튜브/코스 콘텐츠를 기업 교육으로 재패키징
```

### 수익 구조 비교

| 모델 | 단가 | 참여자 | 시간 | 시간당 수익 |
|------|------|--------|------|-----------|

## 2. 기업 교육 상품 설계

### 2.1 3-티어 기업 교육 패키지

| | 🥉 AI 리터러시 워크샵 | 🥈 AI 자동화 부트캠프 ⭐ | 🥇 AI 전환 아카데미 |
|---|---|---|---|
| **가격** | ₩3,000,000 | ₩8,000,000 | ₩20,000,000 |
| **기간** | 반일 (4시간) | 2일 (16시간) | 4주 (주 1회 4시간) |
| **참여 인원** | 10~30명 | 10~20명 | 10~15명 |
| **수준** | 입문 (AI 이해) | 중급 (직접 실습) | 고급 (실전 구축) |
| **내용** | AI 개념/트렌드 + 데모 + Q&A | 실습 워크샵 + 프로세스 맵핑 + 자동화 체험 | 사내 프로세스 진단 + 자동화 구축 + 내재화 |
| **산출물** | AI 활용 가이드 PDF | 프로세스 맵 + 자동화 프로토타입 | 운영 가능한 자동화 시스템 + 운영 매뉴얼 |
| **후속** | 뉴스레터 구독 연결 | 1개월 이메일 지원 | 3개월 지원 + 월간 체크인 |
| **업셀** | → 부트캠프/컨설팅 | → 아카데미/컨설팅 | → 유지보수 구독 |

### 2.2 AI 리터러시 워크샵 (₩300만, 4시간)

```
[대상] 전 직원 (비개발자 중심)
[목표] "AI가 뭔지, 우리 업무에 어떻게 쓰는지" 이해

[09:00~09:30] 오프닝
    "AI 시대, 우리 회사는 준비됐나?" — AI 준비도 라이브 진단

[09:30~10:30] 세션 1: AI 이해하기 (비개발자 눈높이)
    • AI/ChatGPT/자동화가 뭔지 쉽게
    • "개발자 없이 가능한 것 vs 불가능한 것"
    • 실제 기업 사례 3가지 (Before/After)

[10:30~10:45] 휴식

[10:45~11:45] 세션 2: 우리 업무에서 AI 찾기 (그룹 실습)
    • 부서별 반복 업무 리스트업 (포스트잇 워크샵)
    • 난이도/가치 매트릭스로 우선순위 정리
    • "우리 회사의 Low-hanging Fruit은 이것!"

[11:45~12:30] 세션 3: AI 도구 체험 (라이브 데모)
    • ChatGPT 실습 (업무 프롬프트 만들기)
    • Claude Code 자동화 데모 (CS 자동 분류 30초)
    • "이거 우리 회사에도 되겠다!" 순간 만들기

[12:30~13:00] Q&A + 클로징
    • 액션 아이템: "다음 주까지 이것만 해보세요"
    • CTA: AI 자동화 부트캠프 안내
```

### 2.3 AI 자동화 부트캠프 (₩800만, 2일)

```
[대상] 팀 리더, 운영 담당자, 관리자 (10~20명)
[목표] "직접 프로세스 맵을 그리고, 자동화 프로토타입을 만든다"

[Day 1: 진단 + 설계]
  09:00~10:30 | AI 오딧 실습 — 우리 팀 프로세스 분석
  10:45~12:00 | 자동화 로드맵 설계 — Quick Wins 3개 선정
  13:00~14:30 | Claude Code 기초 — 첫 워크플로우 만들기 (핸즈온)
  14:45~16:00 | 그룹 프로젝트 — 팀별 자동화 설계

[Day 2: 구축 + 발표]
  09:00~10:30 | Claude Code 심화 — AI API 연동 (ChatGPT/Claude)
  10:45~12:00 | 팀별 프로토타입 구축 (멘토링)
  13:00~14:30 | 팀별 발표 — "우리 팀의 자동화 계획"
  14:45~16:00 | 피드백 + ROI 측정법 + 실행 로드맵 확정

[산출물]
  • 부서별 프로세스 맵 (Notion)
  • 자동화 프로토타입 1~3개 (Claude Code)
  • 90일 자동화 로드맵
  • 참여자 전원 수료증
```

### 2.4 AI 전환 아카데미 (₩2,000만, 4주)

```
[대상] 디지털 전환 추진팀, CTO/COO, 핵심 인력 (10~15명)
[목표] "사내에 실제 운영 가능한 AI 자동화 시스템을 구축하고 내재화"

Week 1: 전사 AI 오딧
  → 주요 프로세스 5~10개 진단
  → 자동화 우선순위 매트릭스 작성
  → 경영진 리포트 발표

Week 2: 핵심 자동화 구축 (3개)
  → 대표가 직접 + 사내 담당자와 페어 구축
  → "하는 법을 보여주고, 함께 만들고, 혼자 하게 한다"

Week 3: 확장 자동화 + 연동
  → 추가 자동화 2~3개 구축
  → 기존 시스템(ERP/CRM/슬랙 등)과 연동
  → 담당자가 독립적으로 수정/관리 가능하도록 교육

Week 4: 내재화 + 졸업
  → 운영 매뉴얼 작성 (Notion)
  → 담당자 독립 운영 테스트
  → 경영진 최종 발표 (ROI 예상치 포함)
  → 3개월 후속 지원 계약

[산출물]
  • 운영 가능한 자동화 시스템 5~6개
  • 운영 매뉴얼 (Notion)
  • 사내 AI 담당자 2~3명 양성
  • 경영진 ROI 리포트
```

## 3. B2B 세일즈 퍼널

### 3.1 기업 고객 획득 경로

```
[기업 고객 유입 경로]

Path 1: 기존 개인 고객 → 소속 회사에 추천
  "코스에서 배웠는데, 우리 회사 전체가 배우면 좋겠다"
  → 가장 높은 전환율 (이미 가치 검증)

Path 2: 링크드인 B2B 콘텐츠 → 기업 의사결정자 유입
  "AI 자동화 교육 필요한데 어디서 받지?"
  → 링크드인 Thought Leadership이 핵심

Path 3: 파트너(강사/코치) 추천
  "제 고객사가 AI 교육을 찾고 있어요"
  → B2B 파트너 어필리에이트 (기획서 #6)

Path 4: PR/미디어 노출 → 기업 인바운드
  "뉴스에서 본 하우제로, 우리 회사 교육 해줄 수 있나요?"
  → PR 기획서 (#13)와 연동

Path 5: 정부 지원 사업/바우처
  "AI 교육 바우처로 쓸 수 있나요?"
  → 중소기업 AI 바우처, 디지털 전환 지원금 연동
```

### 3.2 기업 세일즈 프로세스

```
[인바운드 문의 or 아웃바운드 접촉]
    ↓
[15분 디스커버리 콜] — 무료
    "현재 AI 활용 수준은? 교육 목표는? 인원은? 예산은?"
    ↓
[맞춤 제안서 발송] — 24시간 내
    3-티어 중 추천 패키지 + ROI 프레이밍
    ↓
[의사결정자 미팅] — 30분
    경영진에게 ROI 중심 프레젠테이션
    "교육 비용 ₩800만 → 연간 절감 ₩5,000만 = ROI 525%"
    ↓
[계약]
    ↓
[교육 진행] → [만족도 설문] → [후속 제안(유지보수/추가 교육)]
```

### 3.3 기업 제안서 핵심 프레이밍

