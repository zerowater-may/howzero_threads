# P3: 스와이프 파일 + 광고 크리에이티브 라이브러리 — 검증된 소재 자산 DB

> **우선순위**: P3 | **예산**: ₩0 | **담당**: 대표

## 핵심: 성과 좋은 크리에이티브를 DB로 관리 → 재사용/변형/스케일

## Notion "크리에이티브 라이브러리" DB

| 필드 | 내용 |
|------|------|
| ID | CR-001 |
| 포맷 | 릴스/캐러셀/텍스트/광고 |
| 채널 | 유튜브/인스타/링크드인/Meta/카카오 |
| 훅 문구 | "CS 인건비 80% 줄이는..." |
| 성과 | 조회수/CTR/전환/CPA |
| 태그 | 교육/역발상/사례/비교 |
| 상태 | 활성/피로/아카이브 |
| 변형 히스토리 | A→B→C 변형 경로 |

## 활용

```
새 캠페인 기획 시:
  → 라이브러리에서 "CTR 상위 10%" 필터
  → 검증된 훅/포맷 재사용
  → 변형 제작 (훅만 교체 or 비주얼만 교체)
  → 성과 기록 → 라이브러리 업데이트

→ "백지에서 시작"이 아니라 "검증된 것에서 변형"
→ 크리에이티브 제작 시간 50% 절감 + 성과 예측 가능
```

## 경쟁사 스와이프 파일

```
경쟁사/벤치마크 채널의 인기 콘텐츠도 수집:
  → 훅 패턴 분석
  → 포맷 분석
  → 하우제로 버전으로 변형

도구: Meta Ad Library (무료) + TikTok Creative Center (무료)
```

## KPI: 라이브러리 100개+ 소재, 크리에이티브 제작 시간 -50%
