> **작성일**: 2026-03-29 | **우선순위**: P3 (중간 — 오가닉 기반 확보 후 가속)

> **예산**: 월 100만원 (광고비 80만 + 운영 20만)
> **담당**: 하우제로 대표 (크리에이티브 기획) + AI 콘텐츠 엔진 (변형 제작)

## 1. 전략 배경 & 리서치 인사이트

### 왜 P3(중간 우선순위)인가?

유료 광고는 **이미 작동하는 오가닉 퍼널을 가속**할 때 가장 효과적이다. P1(숏폼), P2(이메일/커뮤니티/파트너/SEO) 퍼널이 초기 데이터를 확보한 후(6월~), 검증된 콘텐츠를 광고로 밀어주는 전략.

```
[P1~P2에서 검증된 콘텐츠]
    "이 숏폼은 조회수 10,000, 시청 완료율 50%"
    "이 롱폼은 유튜브 구독 전환율 5%"
    "이 AI 준비도 진단은 이메일 전환율 70%"
        ↓
[P3: 검증된 콘텐츠만 광고로 부스팅]
    → 월 80만원으로 최대 효율 → CAC 최적화
```

### 핵심 데이터

- **Meta 광고 CPC 평균 $1.11 (약 1,500원)** / **Instagram CPM $7.68** (Pravaah, 2026)
- **Meta 2026 알고리즘: 수동 타겟팅보다 AI 자동 배치(Advantage+)가 우세** — 크리에이티브 다양성이 핵심 (Nice Digitals, Flighted)
- **UGC 스타일 광고가 2026년에도 최고 전환율** — 스마트폰 촬영, 네이티브 느낌, 진정성 (Marpipe, Foxwell Digital)
- **주 2~4개 크리에이티브 테스트가 소규모 예산의 최적 속도** (Flighted, 2026)
- **YouTube Creator Partnerships: 오가닉 크리에이터 영상을 광고로 전환(boost) 가능** (YouTube NewFronts 2026)
- **YouTube는 AI 검색 인용의 핵심 자산** — "YouTube는 기계 판독이 쉽고, AI가 인용하기 낮은 리스크 소스" (Digiday, 2026)
- **크리에이터 게시물 부스팅이 ROI 1위 마케팅 활동 (39%)** (CreatorIQ, 2025)

### 왜 UGC 스타일인가?

하우제로의 강점은 "대표가 직접 경험한 실전 이야기"다. 이건 자연스럽게 **UGC(사용자 생성 콘텐츠) 스타일**이다.

| 전통 광고 | UGC 스타일 광고 (하우제로 맞춤) |
|----------|--------------------------|
| 스튜디오 촬영, 정제된 영상 | 스마트폰 셀카, 화면 녹화, 실전 데모 |
| "우리 서비스가 최고입니다" | "저도 처음엔 수동으로 다 했습니다" |
| 높은 제작 비용 | 제작 비용 거의 0원 |
| 광고처럼 보임 → 스킵 | 일반 콘텐츠처럼 보임 → 시청 |

## 2. 3단계 유료 광고 퍼널

### 2.1 퍼널 구조

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  TOFU (인지) — "이 사람 누구지?"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  목표: 도달 + 영상 조회 (브랜드 인지)
  타겟: 관심사 기반 (AI, 자동화, 1인 기업, 마케팅)
  광고: 검증된 숏폼 콘텐츠 부스팅
  CTA: 없음 (순수 가치 제공)
  예산: 월 30만원 (전체 광고비의 37.5%)
  KPI: CPV(조회당 비용) < 50원, 영상 25% 이상 시청

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  MOFU (고려) — "이 사람 말이 되네"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  목표: 리드 확보 (이메일 / AI 준비도 진단)
  타겟: TOFU 리타겟팅 (영상 50%+ 시청자 / 프로필 방문자)
  광고: "AI 준비도 점수 무료 진단" + "Claude Code 템플릿 무료"
  CTA: AI 준비도 진단 / 뉴스레터 구독
  예산: 월 30만원 (37.5%)
  KPI: CPL(리드당 비용) < 3,000원

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  BOFU (전환) — "한번 해볼까?"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  목표: 무료 AI 오딧 신청
  타겟: MOFU 리타겟팅 (이메일 구독자 / 진단 완료자 / 웹사이트 방문자)
  광고: 사례 기반 — "CS 인건비 0원으로 만든 실제 과정"
  CTA: "무료 AI 오딧 30분 예약"
  예산: 월 20만원 (25%)
  KPI: CPA(오딧 신청당 비용) < 50,000원
```

|---------|------|-----------|------|

## 3. 플랫폼별 광고 전략

### 3.1 Meta Ads (인스타그램/페이스북)

#### 왜 Meta인가?

- 고가치 서비스(컨설팅, 코칭, 코스)에 최적 — 신뢰·교육·다중 터치포인트 필요한 구매에 강점 (Digital Poonam)
- 2026년 Meta: **Advantage+ 자동 배치가 수동 타겟팅 능가** — 크리에이티브 품질이 곧 타겟팅
- 리타겟팅 정밀도 최고: 영상 조회, 인스타 프로필 방문, Pixel 이벤트 등 행동 기반

#### Meta 광고 크리에이티브 포맷

| 포맷 | TOFU | MOFU | BOFU |
|------|:---:|:---:|:---:|
| **숏폼 릴스 (UGC 스타일)** | ★★★ | ★★ | ★ |
| **카드뉴스/캐러셀** | ★★ | ★★★ | ★★ |
| **스토리 광고** | ★★ | ★★ | ★★★ |
| **사례 스크린샷 + 텍스트** | ★ | ★★ | ★★★ |

#### Meta 크리에이티브 예시

**TOFU — 릴스 부스팅**:
```
[0~3초] Hook: "남들이 챗GPT로 장난칠 때, 저는 10억을 벌었습니다"
[3~15초] 데모: CS 자동화 화면 녹화
[15~25초] 결과: "CS 담당자 5명 → 0명. 인건비 절감액: 월 2,000만원"
[25~30초] CTA 없음 (순수 가치 — TOFU)
```

**MOFU — 리드 수집**:
```
[카드뉴스 5장]
1. "당신의 회사, AI 도입 준비 됐나요?"
2. "5가지 질문으로 3분 만에 알 수 있습니다"
3. "비개발자 기준으로 설계된 자가진단"
4. "지금까지 1,000명이 테스트했습니다"
5. "→ 무료 AI 준비도 진단 받기" (CTA)
```

**BOFU — 전환**:
```
[스토리 광고]
"저도 처음엔 개발자 없이 어떻게 하나 막막했습니다.
 그래서 제가 직접 무료로 진단해드립니다.
 30분이면 우리 회사에 맞는 AI 자동화 로드맵을 드립니다.
 → 무료 AI 오딧 예약하기"
```

### 3.2 YouTube Ads

#### 왜 YouTube인가?

- "YouTube는 AI 검색에서 기계 판독이 쉬운 저위험 인용 소스" — SEO/AEO 시너지 (Digiday)
- Creator Partnerships Boost: **오가닉 영상을 그대로 광고로 전환** 가능 (YouTube 2026)
- 롱폼 → 신뢰 구축에 최적. 숏폼 → 발견(discovery)에 최적

#### YouTube 광고 전략

| 포맷 | 사용 단계 | 설명 |
|------|----------|------|
| **In-feed Discovery** | TOFU | 검색 결과/추천 영상 옆에 노출. "비개발자 AI 자동화" 키워드 타겟 |
| **Shorts 부스팅** | TOFU | 인기 숏폼을 광고로 전환. CPV 최적화 |
| **Skippable In-stream** | MOFU | 5초 후 스킵 가능. 첫 5초 훅이 전부 |
| **Bumper (6초)** | 리마인더 | MOFU→BOFU 리타겟. "무료 AI 오딧 → howzero.com" |

## 4. 크리에이티브 테스팅 프레임워크

### 4.1 주간 테스팅 루틴

Meta 2026 베스트 프랙티스: **소규모 예산은 주 2개 크리에이티브 테스트** (Flighted)

```
[월요일] 지난 주 데이터 분석
    → 승자(Winner) 크리에이티브 식별
    → 패자 OFF, 승자 예산 증액

[화요일] 새 크리에이티브 2개 제작
    → 변형 축: 훅 교체 / 포맷 변경 / CTA 변경
    → AI 도구로 변형 빠르게 생산

[수요일] 새 크리에이티브 배포
    → Advantage+ 캠페인에 추가
    → 초기 24시간 학습 기간

[목~일] 데이터 수집
    → CTR, CPV, CPL, CPA 트래킹
    → 충분한 데이터 축적 (최소 50 이벤트/광고셋)
```

### 4.2 크리에이티브 변형 매트릭스

| 변형 축 | A 버전 | B 버전 |
|---------|--------|--------|
| **훅** | "코딩 없이 CS 자동화" | "CS 직원 5명분 → AI 하나로" |
| **포맷** | 릴스 (세로 영상) | 캐러셀 (카드뉴스) |
| **톤** | 교육적 ("이렇게 하세요") | 도발적 ("아직도 사람이 해요?") |
| **CTA** | "뉴스레터 구독" | "AI 준비도 진단" |
| **사회적 증거** | 수치 ("연매출 10억") | 사례 ("A 회사 월 200시간 절감") |

### 4.3 "오가닉 → 광고" 승격 기준

숏폼/롱폼 콘텐츠가 아래 기준을 넘으면 광고로 승격:

| 기준 | 숏폼 (쇼츠/릴스) | 롱폼 (유튜브) |
|------|:---:|:---:|
| 오가닉 조회수 | 5,000+ | 2,000+ |
| 시청 완료율 | 40%+ | 평균 시청 8분+ |
| 댓글/공유 | 50+ | 20+ |
| 구독 전환 | 상위 20% | 상위 20% |

→ 이 기준을 넘긴 콘텐츠만 광고비를 투입 = **검증된 콘텐츠에만 돈 쓰기**

## 5. 기술 인프라

### 5.1 트래킹 스택

| 도구 | 역할 | 비용 |
|------|------|------|
| **Meta Pixel** | 웹사이트 행동 추적 (페이지뷰, 가입, 오딧 신청) | 무료 |
| **Meta Conversions API (CAPI)** | 서버사이드 이벤트 전송 (쿠키 차단 대비) | 무료 (Claude Code 구현) |
| **GA4** | 전체 퍼널 분석 | 무료 |
| **UTM 파라미터** | 채널별 성과 구분 | 무료 |
| **Claude Code 자동화** | CAPI 구현 + 리드 데이터 CRM 연동 | 무료 (셀프호스팅) |

### 5.2 CAPI 구현 (Claude Code 워크플로우)

```
[사용자 행동]
    howzero.com 방문 / AI 준비도 진단 완료 / 오딧 신청
        ↓
[Claude Code 워크플로우]
    1. 이벤트 수신 (Webhook)
    2. Meta CAPI 엔드포인트로 서버사이드 전송
    3. GA4 Measurement Protocol로 동시 전송
    4. 스티비 API로 이메일 리스트 추가
    5. Notion DB에 리드 기록
```
