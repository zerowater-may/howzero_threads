## 1. 전략 배경 & 리서치 인사이트

### 컨설팅의 한계

컨설팅은 "시간 = 돈"이다. 하우제로 대표가 직접 해야 하므로 **스케일 상한선**이 존재한다.

```
[컨설팅만 할 경우]
  대표 가용 시간: 주 40시간
  오딧 30분 × 주 10건 = 5시간
  유료 컨설팅 × 주 3건 = 15시간
  구축 프로젝트 × 1건 = 15시간
  나머지 5시간 = 마케팅/운영

  월 매출 상한: ~3,000만원 (대표 풀가동)
  → 더 이상 성장 불가
```

해결책: **컨설팅(1:1) + 코스(1:N) + 템플릿(1:∞)** 3층 수익 구조

```
[3층 수익 구조]

┌─────────────────────────────────┐
│  3층: 디지털 프로덕트 (1:∞)      │ ← 수면 중에도 매출
│  Claude Code 템플릿, 오딧 프레임워크,     │    한 번 만들면 무한 판매
│  자동화 체크리스트, 가이드북       │    단가: 1~10만원
├─────────────────────────────────┤
│  2층: 온라인 코스 (1:N)          │ ← 시간 대비 레버리지 극대화
│  코호트 기반 부트캠프,            │    한 번 진행에 20~50명
│  셀프페이스 강의                 │    단가: 30~100만원
├─────────────────────────────────┤
│  1층: 컨설팅/구축 (1:1)          │ ← 고가치, 시간 집약적
│  무료 오딧 → 유료 오딧 →         │    대표 직접 진행
│  자동화 구축 → 전환 패키지        │    단가: 100~3,000만원
└─────────────────────────────────┘
```

### 핵심 데이터

- **코호트 기반 코스 수료율 85~90%** vs 셀프페이스 10~15% (Disco, BuddyBoss, 2026)
- **Claude Code 템플릿 판매로 월 $3,200 패시브 인컴 달성 사례** (Medium, 2026.03)
- **Claude Code 크리에이터 마켓플레이스**: 한 번 만들면 무한 판매 = 패시브 인컴 (Reddit r/SaaS)
- **2026년 디지털 프로덕트 시장**: 니치 깊이 + 진정한 가치 + 전략적 유통이 핵심 (NextSmartBusiness)
- **Gumroad, Whop 등 플랫폼**: 코스+커뮤니티+디지털 프로덕트 통합 판매 대세

## 3. Claude Code 템플릿 마켓플레이스 전략

### 3.1 왜 Claude Code 템플릿인가?

- Claude Code 크리에이터 마켓플레이스에서 **한 번 만들면 무한 판매**
- 월 $3,200 패시브 인컴 달성 실제 사례 (Medium, 2026.03)
- 하우제로의 컨설팅 실전에서 나온 워크플로우 → 자연스러운 제품화
- **리드 마그넷 + 유료 프로덕트** 이중 활용

### 3.2 템플릿 라인업

| 카테고리 | 템플릿 | 무료/유료 |
|---------|--------|----------|
| **CS 자동화** | 고객 문의 자동 분류 & 라우팅 | 무료 (리드마그넷) |
| | 환불 요청 자동 처리 | 유료 (팩 포함) |
| | 리뷰 수집 & 감성 분석 | 유료 |
| **마케팅 자동화** | 이메일 자동 시퀀스 (너처링) | 유료 |
| | SNS 자동 포스팅 | 유료 |
| | 경쟁사 가격 모니터링 | 유료 |
| **업무 자동화** | 보고서 자동 생성 (ChatGPT 연동) | 무료 (리드마그넷) |
| | 인보이스 자동 처리 | 유료 |
| | Slack/카카오톡 알림 봇 | 유료 |
| **AI 연동** | ChatGPT API 기본 연결 | 무료 (리드마그넷) |
| | Claude API 문서 요약 | 유료 |
| | 이미지 생성 자동화 | 유료 |
