# P3: 첫 팀원 채용 전략 — 마케팅 운영 담당자 확보 시점과 방법

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: 월 150~300만원 (채용 시) | **담당**: 대표

---

## 핵심 컨셉

대표 혼자 운영하다가 **MRR ₩500만 or 월매출 ₩1,500만 달성 시** 첫 팀원을 채용한다. 너무 빨리 뽑으면 비용 부담, 너무 늦으면 성장 병목.

## 채용 시점 기준

```
[채용 트리거]
  □ 월매출 ₩1,500만+ 3개월 연속
  □ 대표 주 50시간+ 근무 2개월 연속
  □ 콘텐츠 게시 목표 달성률 80% 미만 2개월 연속
  □ 오딧 대기 2주+ 백로그

→ 3개 이상 해당 → 채용 시작
```

## 첫 채용 역할: "마케팅 운영 매니저"

| 항목 | 설계 |
|------|------|
| **역할** | 콘텐츠 편집/예약, 뉴스레터 발행, CRM 관리, 커뮤니티 운영 |
| **유형** | 파트타임 (주 20시간) or 프리랜서 |
| **급여** | 월 ₩150~200만 (파트타임) |
| **필수 스킬** | Notion, Canva, 기본 영상 편집, 이메일 도구, 한국어+영어 |
| **우대 스킬** | Claude Code 경험, SNS 운영 경험, AI 도구 관심 |

## 위임할 업무 (대표 시간 해방)

| 업무 | 현재 시간 | 위임 후 |
|------|:---:|:---:|
| 숏폼 편집/업로드 | 주 3시간 | → 매니저 |
| 뉴스레터 포맷팅/발송 | 주 1시간 | → 매니저 |
| SNS 예약 게시 | 주 1시간 | → 매니저 |
| 블로그 포맷팅/발행 | 주 1시간 | → 매니저 |
| CRM 데이터 관리 | 주 1시간 | → 매니저 |
| 커뮤니티 모니터링 | 주 2시간 | → 매니저 |
| **합계** | **주 9시간** | **대표 0시간** |

→ **대표는 촬영+컨설팅+전략에만 집중**

| 채널 | 적합도 |
|------|:---:|
| 하우제로 커뮤니티 내 모집 | ★★★★★ (이미 팬) |
| 링크드인 공고 | ★★★★ |
| 원티드/로켓펀치 | ★★★ |
| 크몽/숨고 (프리랜서) | ★★★ |
| 대학교 인턴 | ★★★ |

## 2단계 확장 (월매출 ₩3,000만+)

| 역할 | 시기 | 급여 |
|------|------|------|
| 마케팅 운영 매니저 (→ 풀타임) | MRR ₩500만 | ₩250~350만 |
| 영상 PD / 편집자 | 월매출 ₩3,000만 | ₩200~300만 |
| 세일즈 / CS 담당 | B2B 10건+/월 | ₩250~350만 |

## KPI (채용 후 기대 효과)

| 지표 | 채용 전 | 채용 후 |
|------|--------|--------|
| 주간 콘텐츠 게시 | 15~20개 | 25~30개 (+50%) |
| 대표 주간 가용 시간 | 주 40시간 | 주 49시간 (+9) |
| 콘텐츠 목표 달성률 | 80% | 95%+ |
| 대표 번아웃 리스크 | 높음 | 낮음 |
