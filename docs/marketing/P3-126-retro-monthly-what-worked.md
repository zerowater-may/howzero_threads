# P3-126: 월간 "뭐가 됐고 뭐가 안 됐나" 회고 콘텐츠 — Build in Public 심화
> P3 | ₩0 | 대표
매월 1일 "지난 달 마케팅 회고" 콘텐츠 발행. 유튜브 롱폼 + 링크드인 + 뉴스레터. 수치 전부 공개: 조회수, 구독자, 매출, 오딧 건수, 전환율, 실패한 것. "이번 달 가장 잘 된 것 3가지 + 가장 안 된 것 3가지 + 다음 달 계획." 진정성 콘텐츠 = 가장 높은 참여율. 시청자: "이 사람 진짜 투명하다" → 신뢰 → 팔로우/오딧. KPI: 회고 콘텐츠 조회수 일반 대비 2배, 댓글 참여 3배.
