# P3: B2B 콜드 아웃리치 + 이메일 프로스펙팅 — 인바운드 보완 아웃바운드 채널

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: 월 15만원 | **담당**: 대표 + Claude Code

---

## 핵심 컨셉

인바운드(콘텐츠/SEO/PR)가 성장하는 동안, **아웃바운드 이메일로 특정 타겟 기업에 직접 접근**하여 파이프라인을 보충한다. 스팸이 아닌 **초개인화된 가치 제공** 방식.

## 아웃바운드 vs 인바운드

| | 인바운드 | 아웃바운드 |
|---|---|---|
| 속도 | 느림 (3~6개월) | 빠름 (즉시) |
| 규모 | 자동 스케일 | 수동/반자동 |
| 비용 | 콘텐츠 제작비 | 도구+시간 |
| 용도 | 장기 성장 | 단기 파이프라인 보충 |

→ **인바운드가 메인, 아웃바운드는 보조** (전체 리드의 20% 이하)

## 타겟 리스트 구축

```
[Claude Code 리서치 워크플로우]

1. LinkedIn Sales Navigator or Apollo.io:
   필터: 한국 / 대표/팀장 / 10~100명 / 이커머스·서비스업
2. 타겟 50명/월 추출 → Notion "아웃바운드 DB"
3. 각 타겟의 웹사이트/SNS 자동 스캔 (Claude Code + Brave)
4. Claude API: 타겟별 초개인화 이메일 초안 생성
```

## 콜드 이메일 3-터치 시퀀스

```
[Email 1: 가치 제공] (D+0)
  제목: "[회사명]의 [업무]를 자동화하면 월 [X]시간 아끼실 수 있습니다"
  본문: "안녕하세요 [이름]님, [회사]의 [관찰한 구체적 포인트]를 보고 연락드립니다.
  비슷한 [업종] 기업에서 [자동화명]으로 월 [X]시간을 절감한 사례가 있어
  [이름]님에게도 도움이 될 것 같아 공유드립니다.
  → [사례 PDF or 블로그 링크]"

[Email 2: 후속] (D+3, 미응답 시)
  제목: "Re: [이전 제목]"
  본문: "혹시 바쁘셨을까 싶어 한 번 더 보내드립니다.
  15분 통화로 [회사명]에 맞는 자동화 가능성을 빠르게 살펴볼 수 있습니다.
  → [캘린더 링크]"

[Email 3: 마지막] (D+7, 미응답 시)
  제목: "마지막으로 한 가지만"
  본문: "더 이상 연락드리지 않겠습니다.
  혹시 나중에 AI 자동화가 필요해지시면
  이 무료 가이드가 도움이 될 겁니다: [가이드 링크]
  좋은 하루 되세요!"
```

## Claude Code 자동화

```
[주간 아웃바운드 Claude Code 워크플로우]

매주 월요일:
  → Notion DB에서 이번 주 타겟 10명 추출
  → Claude API: 각 타겟 정보 기반 초개인화 이메일 생성
  → 대표 검토 (10분) → 승인
  → Gmail API: 이메일 발송 (10명)
  → D+3: 미오픈자 자동 팔로업
  → D+7: 미응답자 마지막 이메일
  → 응답자 → Notion CRM "HOT 리드" 자동 이동
```

## 원칙

1. **주 10명 이하** (질 > 양)
2. **초개인화 필수** (대량 스팸 절대 금지)
3. **가치 먼저** (첫 이메일에 판매 언급 NO)
4. **3터치 후 종료** (끈질기되 괴롭히지 않음)
5. **인바운드 리드가 충분해지면 아웃바운드 축소**
