## 핵심 컨셉

P1-AI 준비도 진단 도구를 Product Hunt에 런칭하여 **하루 만에 글로벌 인지도** 확보.

## 런칭 제품: "AI Readiness Score by HowZero"

- **카테고리**: Productivity / AI / Business Tools
- **무료 도구**: 3분 자가진단 → 즉시 결과 → 이메일 수집
- **PH 적합성**: 무료, 인터랙티브, 즉각적 가치 = PH가 좋아하는 포맷

## D-60 → D-Day 런칭 타임라인

| 시점 | 작업 |
|------|------|
| D-60 | PH 계정 활성화, 매일 5개 제품 업보트/댓글 |
| D-30 | 런칭 페이지 사전 제작 (스크린샷 5장 + 소개 영상 60초) |
| D-14 | Hunter 섭외 (PH 파워유저에게 DM) |
| D-7 | 전 채널 "다음 주 PH 런칭" 사전 알림 |
| D-1 | 이메일 리스트 + SNS "내일 PH 런칭!" 알림 |
| **D-Day** | **00:01 PST 런칭** + 댓글 실시간 응답 (12시간) |
| D+1 | 결과 발표 "PH #X 달성!" 콘텐츠 |
| D+7 | PH 유입 → 이메일 시퀀스 가동 |

