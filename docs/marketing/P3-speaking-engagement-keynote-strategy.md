## 핵심: 강연 1회 = PR + 리드 + 후기 + 콘텐츠 + 네트워킹 동시 달성

## 타겟 강연 기회

| 유형 | 예시 | 기대 청중 | 보수 |
|------|------|:---:|------|
| **스타트업 컨퍼런스** | 서울창업허브, 벤처스퀘어 | 100~500명 | 무료~₩100만 |
| **기업 사내 특강** | 중소기업 AI 교육 | 20~50명 | ₩100~300만 |
| **대학/교육기관** | AI 특강, 창업 강의 | 50~200명 | 무료~₩50만 |
| **협회 세미나** | P3-협회 MOU 연동 | 30~100명 | 무료 (MOU) |
| **온라인 서밋** | AI/노코드 온라인 컨퍼런스 | 500~5,000명 | 무료~₩200만 |

## 키노트 대본 구조 (45분)

```
[0~5분] 오프닝 + 자기소개
  "코딩 1도 모르던 제가 GPT-3로 10억을 벌었습니다"

[5~15분] 문제 제기
  "AI 도입의 80%가 실패합니다. 왜?"
  → 데이터 + 사례 + 역발상

[15~30분] 프레임워크 제시
  "AI 오딧 5차원 프레임워크"
  → 실전 데모 or 참석자 실습

[30~40분] 사례 3가지
  이커머스 / 법률 / 1인 기업 Before/After

[40~45분] CTA + Q&A
  "오늘 참석자 한정 무료 AI 오딧"
  → QR코드 → howzero.com/audit?ref=keynote
```

## 강연 → 콘텐츠 변환

```
강연 1회 (45분 녹화)
  → 유튜브 롱폼 1개
  → 숏폼 클립 5~10개
  → 블로그 포스트 1개 (대본 기반)
  → 링크드인 포스트 3개
  → 캐러셀 2개 (핵심 프레임워크)
  = 15~20개 콘텐츠
```

## 강연 섭외 전략

```
Step 1: 미디어 킷(P2-#50) + 키노트 주제 3개 준비
Step 2: 행사 주최자에게 이메일:
  "연매출 10억 달성한 비개발자 AI 실전가의 키노트"
Step 3: 첫 3회는 무료 (포트폴리오 구축)
Step 4: 4회차부터 유료 (₩100만~)
Step 5: 강연 포트폴리오 → 더 큰 무대 → 선순환
```
