## 핵심 컨셉

하우제로의 AI 오딧 워크시트, 프로세스 맵, ROI 계산기, CRM 등을 **Notion 템플릿**으로 만들어 Notion 마켓플레이스에 유통. Notion 사용자(비개발자 타겟과 정확히 겹침)에게 자연 도달.

## 템플릿 라인업

| 템플릿 | 가격 | 목적 |
|--------|:---:|------|
| AI 준비도 자가진단 워크시트 | 무료 | 리드마그넷 |
| 비즈니스 프로세스 맵핑 보드 | 무료 | 리드마그넷 |
| 자동화 ROI 계산 대시보드 | ₩9,900 | 소액 첫 결제 |
| 1인 기업 AI 도구 관리 DB | ₩9,900 | 소액 결제 |
| AI 오딧 진단 리포트 템플릿 | ₩19,000 | 컨설턴트용 |
| Claude Code 워크플로우 관리 대시보드 | ₩19,000 | 자동화러용 |
| 콘텐츠 캘린더 (크리에이터용) | ₩9,900 | 크리에이터 타겟 |

## 유통 채널

| 채널 | 장점 |
|------|------|
| **Notion 공식 템플릿 갤러리** | Notion 내 검색 노출 |
| **Gumroad** | 결제 + 이메일 수집 |
| **하우제로 자체 스토어** | 마진 100% |

## 전환 퍼널

```
Notion 템플릿 다운로드 (무료)
  → 템플릿 안에 CTA: "이 진단 결과를 전문가에게 보여주세요 → 무료 AI 오딧"
  → howzero.com/audit
  → 오딧 → 컨설팅
```
