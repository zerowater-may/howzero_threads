## 1. 전략 배경

### 왜 무료 도구인가?

> "무료 도구와 자가진단이 일반 다운로드 PDF보다 항상 높은 전환율. 즉각적 가치를 제공하기 때문." (VentureHarbour, 2026)

HubSpot의 Website Grader가 증명한 공식:
```
무료 도구 사용 → 즉각 결과 → "더 개선하려면?" → 유료 서비스 전환
```

P1-AI 준비도 진단은 이 공식의 첫 번째 도구. 이 기획서는 **추가 무료 도구 5개**를 만들어 리드 유입 면적을 5배 확장.

### 핵심: 각 도구 = 독립적 리드마그넷 + SEO 자산

```
[무료 도구 5개 = 5개의 독립 유입 경로]

도구 1: AI 준비도 진단 (이미 있음)
도구 2: 자동화 ROI 계산기 ← NEW
도구 3: AI 도구 추천 매칭 ← NEW
도구 4: 프로세스 맵핑 도구 ← NEW
도구 5: 자동화 절감액 시뮬레이터 ← NEW

→ 각각 독립 URL + 독립 SEO + 독립 SNS 공유
→ 5개 입구 → 하나의 퍼널 (이메일 → 오딧 → 컨설팅)
```

## 2. 무료 도구 5개 설계

### 도구 1: AI 준비도 진단 (기존 — P1 기획서)

howzero.com/ai-score — 이미 구축 완료

### 도구 2: 자동화 ROI 계산기

```
howzero.com/roi-calculator

[입력]
  1. 자동화하고 싶은 업무: [드롭다운 선택]
     (CS 응대/이메일 관리/보고서/데이터 입력/기타)
  2. 현재 이 업무에 쓰는 시간: [___] 시간/주
  3. 담당자 시급 (or 월급): [___] 원
  4. 외주 비용 (있으면): [___] 원/월
  5. 자동화 도입 예산: [___] 원

[즉시 결과]
  ┌──────────────────────────────┐
  │  💰 자동화 ROI 분석 결과      │
  │                              │
  │  현재 연간 비용: ₩36,000,000  │
  │  자동화 후 비용: ₩7,200,000   │
  │  연간 절감액: ₩28,800,000    │
  │  투자 대비 ROI: 1,340%       │
  │  투자 회수 기간: 0.9개월      │
  │                              │
  │  📊 상세 리포트 → 이메일 입력  │
  │  🔍 무료 AI 오딧 → 예약하기   │
  └──────────────────────────────┘
```

**SEO 키워드**: "자동화 ROI 계산", "업무 자동화 비용", "AI 도입 비용 효과"

### 도구 3: AI 도구 추천 매칭

```
howzero.com/tool-match

[입력] (5문항, 1분)
  1. 업종: [선택]
  2. 가장 자동화하고 싶은 업무: [선택]
  3. 기술 수준: 완전 초보 / 기본 / 중급
  4. 월 예산: 무료만 / ~5만원 / ~20만원 / 상관없음
  5. 혼자 vs 팀: 혼자 / 2~5명 / 5명+

[즉시 결과]
  "당신에게 맞는 AI 자동화 도구 TOP 3"

  1위: Claude Code (추천도 95%)
     → 이유: 무료, 비개발자 친화적, CS 자동화에 최적
     → 하우제로 가이드: [블로그 링크]

  2위: Make.com (추천도 80%)
     → 이유: 시각적 UI, 1인 기업에 적합
     → 하우제로 가이드: [블로그 링크]

  3위: Zapier (추천도 65%)
     → 이유: 가장 쉬움, 연동 7,000+
     → 하우제로 가이드: [블로그 링크]

  📧 맞춤 세팅 가이드 → 이메일 입력
  🔍 어떤 도구를 써야 할지 모르겠다면 → 무료 AI 오딧
```

**SEO 키워드**: "AI 도구 추천", "업무 자동화 도구 비교", "Claude Code vs Make vs Zapier"

### 도구 4: 프로세스 맵핑 도구

```

[인터랙티브 캔버스]

  [시작] → [단계 1: 고객 문의 접수]

[결과]
  "당신의 프로세스 5단계 중 3단계가 자동화 가능합니다!
```

**SEO 키워드**: "업무 프로세스 맵핑", "업무 흐름 분석", "프로세스 개선"

## 3. 도구 → 전환 공통 퍼널

```
[무료 도구 5개 중 아무거나 사용]
    ↓
[즉각적 결과 제공] — 가치 먼저!
    ↓
[상세 리포트 → 이메일 입력] — 리드 수집
    ↓
[CTA: 무료 AI 오딧 30분] — 전환
    ↓
[이메일 너처링 시퀀스] — 장기 육성
    ↓
[유료 컨설팅/코스/커뮤니티]
```

### 이메일 수집 문구 (도구별)

| 도구 | 이메일 수집 문구 |
|------|----------------|
| ROI 계산기 | "상세 ROI 리포트를 이메일로 보내드립니다" |
| 도구 매칭 | "맞춤 세팅 가이드를 이메일로 받으세요" |
| 프로세스 맵 | "프로세스 맵 PDF를 다운로드하세요" |
| 절감 시뮬레이터 | "시뮬레이션 결과 + 실제 사례를 보내드립니다" |

## 4. 기술 구현

### 4.1 스택

| 도구 | 기술 | 비용 |
|------|------|------|
| ROI 계산기 | Next.js 정적 페이지 + JS 계산 | Vercel 무료 |
| 도구 매칭 | Typeform or 자체 폼 + 조건 로직 | 무료~₩30K/월 |
| 프로세스 맵 | React Flow (오픈소스) or Miro 임베드 | 무료 |
| 절감 시뮬레이터 | Next.js + 슬라이더 UI | Vercel 무료 |
| 이메일 수집 | Claude Code Webhook → 스티비 API | 기존 |
| 분석 | GA4 퍼널 이벤트 | 무료 |

### 4.2 Claude Code 연동

```
[각 도구 완료 시]
    → Claude Code Webhook: 결과 데이터 + 이메일 전송
    → Notion CRM: 리드 자동 등록 (유입 소스: 도구명)
    → 스티비: 도구별 맞춤 이메일 시퀀스 자동 시작
    → 리드 스코어: 도구 사용 = +15점
```
