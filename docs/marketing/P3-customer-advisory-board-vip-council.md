# P3: 고객 자문단(Advisory Board) + VIP 카운슬 — 핵심 고객을 전략 파트너로

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: 월 5만원 | **담당**: 대표

---

## 핵심: 최고 고객 5~10명을 "자문단"으로 만들면 → 제품 개선 + 추천 + 케이스 스터디 동시 해결

## 자문단 구성

| 항목 | 설계 |
|------|------|
| **인원** | 5~10명 |
| **자격** | NPS 9~10 고객 / 리테이너 고객 / 케이스 스터디 협조 고객 |
| **미팅** | 분기 1회 온라인 (1시간) |
| **혜택** | 신규 서비스 Early Access + 1:1 전략 미팅 연 2회 + 이벤트 VIP 초대 |

## 자문단 미팅 아젠다

```
분기 1회 (1시간):
  [10분] 하우제로 분기 실적 공유 (투명)
  [20분] "다음 분기 이런 서비스를 기획 중인데 어떻게 생각하세요?"
  [20분] "현재 서비스에서 개선할 점은?"
  [10분] 자유 네트워킹 + 자문단 간 교류
```

## 가치 교환

| 하우제로 → 자문단 | 자문단 → 하우제로 |
|----------------|----------------|
| VIP 대우 + 특별감 | 솔직한 제품 피드백 |
| Early Access | 케이스 스터디 참여 |
| 네트워킹 기회 | 추천/소개 |
| 연 2회 1:1 전략 미팅 | 시장 인사이트 |

## KPI: 자문단 5명 구성, 분기 1회 미팅, 자문단 경유 추천 3건+/분기
## 예산: 월 ₩50,000 (미팅 운영 + 선물)
