## 1. 전략 배경

### 왜 미디어 킷 + 아카이브인가?

팟캐스트 호스트, 기자, 협회, 파트너가 하우제로를 소개할 때 **매번 처음부터 설명하면 시간 낭비**. 미디어 킷 1개를 만들면 모든 외부 접점에서 **통일된 브랜드**로 소개된다.

```
[미디어 킷이 없으면]
  팟캐스트 호스트: "소개 자료 보내주세요" → 매번 새로 작성 (30분)
  기자: "약력과 사진 주세요" → 찾아서 보내기 (15분)
  파트너: "우리 구독자에게 어떻게 소개하면 돼요?" → 고민 (20분)

[미디어 킷이 있으면]
  "howzero.com/press에 다 있습니다" → 0분
```

### 2.3 핵심 토킹 포인트 5가지

```
1. "코딩 0줄로 GPT-3 SaaS를 만들어 연매출 10억을 달성했다"
   → 비개발자도 AI로 비즈니스를 만들 수 있다는 실전 증거

2. "AI 도입의 80%는 기술이 아니라 프로세스 문제"
   → AI 도구보다 먼저 비즈니스 프로세스를 진단해야 한다

3. "AI 오딧 30분이면 기업의 비효율을 찾을 수 있다"
   → 무료 진단 → 맞춤 자동화 로드맵 제공

4. "CS 직원 5명분의 업무를 AI 하나로 대체한 실제 사례"
   → 비개발자도 Claude Code으로 자동화 시스템을 구축할 수 있다

5. "코딩이 아니라 비즈니스 원리만 알면 된다"
   → 기술 장벽은 낮아졌다. 실행하는 사람만 성과를 낸다
```

### 2.4 샘플 인터뷰 질문 10개

```
1. 코딩을 전혀 모르셨는데 어떻게 SaaS를 만들 수 있었나요?

3. 연매출 10억까지 얼마나 걸렸나요? 타임라인을 알려주세요.
4. 가장 큰 실패 경험은 뭐였나요?
5. "AI 도입의 80%가 프로세스 문제"라고 하셨는데 무슨 뜻인가요?
6. AI 오딧은 구체적으로 어떻게 진행되나요?
7. 비개발자가 AI 자동화를 시작하려면 첫 번째로 뭘 해야 하나요?
8. 가장 보람 있었던 컨설팅 사례를 알려주세요.
9. AI 도입에 가장 적합한 업종이 있나요?
```
