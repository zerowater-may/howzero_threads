## 1. 전략 배경 & 리서치 인사이트

### 왜 오프라인/라이브인가?

온라인 채널(숏폼, 뉴스레터, SEO)이 **양(Volume)**을 만든다면, 오프라인/라이브는 **깊이(Depth)**를 만든다.

```
[신뢰 깊이 피라미드]

      ┌──────────┐
      │ 오프라인  │ ← 가장 깊은 신뢰
      │ 만남/밋업 │    "직접 만나봤는데 진짜야"
      ├──────────┤
      │ 라이브   │ ← 실시간 상호작용
      │ 웨비나   │    "질문에 즉답하는 걸 보니 실력자"
      ├──────────┤
      │ 팟캐스트 │ ← 1시간 깊은 대화
      │ 롱폼영상 │    "이 사람 철학이 맞네"
      ├──────────┤
      │ 숏폼    │ ← 30초 임팩트
      │ SNS포스트│    "오 이거 흥미롭다"
      └──────────┘
```

### 핵심 데이터

- **"낮은 신뢰, 높은 노이즈 환경에서 이벤트는 브랜드가 광고나 알고리즘으로 살 수 없는 것을 만든다: 신념, 신뢰, 진짜 연결"** (Smart Meetings, 2026)
- **웨비나 평균 전환율 56%** — 디지털 마케팅 채널 중 최고 전환율 (DemandSage, 2026)
- **웨비나 참석율 35~45%**, 최적화 시 **58%까지** 가능 (ClickMeeting)
- **High-Ticket 웨비나 프레임워크: $5K~$10K+ 서비스 판매에 최적** (HighTicketHQ)
- **500명의 40% 참여 커뮤니티 > 5,000명의 2% 참여 커뮤니티** (OnlineCommunityHub, 2026)
- **컨설팅 컨퍼런스: 이론보다 실전 챌린지 중심 워크샵이 전환율 극대화** (Vendelux, 2026)

### 왜 Q3인가?

| Q2 (4~6월) | Q3 (7~9월) |
|-----------|-----------|
| 온라인 오디언스 구축 중 | 구독자 500+, 이메일 300+ 확보 |
| 아직 "모이라"고 할 사람이 부족 | 웨비나 30~50명 모집 가능 |
| 대표 콘텐츠 트레이닝 중 | 라이브 진행 경험 축적 (Q&A 등) |

#### 웨비나 전환 퍼널

```
[등록] 50~80명
    ↓ 참석율 45%
[참석] 25~36명
    ↓ 라이브 중 투표: "무료 오딧 관심 있으신 분?"
[CTA 클릭] 25% = 6~9명
    ↓ 오딧 신청 전환
[오딧 신청] 50% = 3~5명
    ↓ 유료 전환
[유료 계약] 25% = 1건

→ 웨비나 1회당 유료 계약 약 1건
→ 월 2회 = 월 2건 × 200만원 = 월 400만원
```

#### 오프라인 밋업 프로그램

```
[19:00~19:10] 오프닝
    대표 인사 + 아이스브레이킹 ("1분 자기소개 릴레이")

[19:10~19:50] 키노트 (40분)
    "비개발자의 AI 자동화, 실전에서 뭐가 됐고 뭐가 안 됐나"
    → 대표의 실전 사례 + Before/After + 실패담

[19:50~20:00] 휴식 + 네트워킹

[20:00~20:50] 미니 워크샵 (50분)
    "내 업무 프로세스 맵 그려보기 + 자동화 포인트 찾기"
    → 3~4인 그룹 테이블, 워크시트 배포
    → 각 그룹 발표 (2분씩)

[20:50~21:00] 클로징 + CTA
    "오늘 워크시트 기반으로 무료 AI 오딧 → QR 코드 스캔"
    → 참석자 전원 오딧 할인 쿠폰 (50% OFF)

[21:00~22:00] 자유 네트워킹 + 명함 교환
    대표가 1:1로 돌아다니며 인사 → 관계 구축
```

#### 밋업 → 전환 퍼널

```
[참석] 30~50명
    ↓ 밋업 후 오딧 신청율 30% (대면 신뢰)
[오딧 신청] 10~15명
    ↓ 유료 전환율 30% (이미 대면으로 신뢰 구축)
[유료 계약] 3~5건

→ 밋업 1회당 유료 계약 3~5건
→ 3~5건 × 200만원 = 600~1,000만원/밋업
→ 분기 1회 = 분기 600~1,000만원

밋업 CAC: (대관 30만 + F&B 10만 + 운영 10만) ÷ 4건 = 12.5만원
→ 광고 CAC(20만원)의 63% — 더 저렴하면서 더 깊은 신뢰
```

## 3. 이벤트 → 콘텐츠 자산 변환

### 모든 이벤트는 콘텐츠 공장이다

| 이벤트 산출물 | 콘텐츠 변환 | 채널 |
|------------|-----------|------|
| 웨비나 녹화 | 유튜브 롱폼 업로드 | 유튜브 |
| 웨비나 핵심 클립 | 숏폼 5~10개 추출 | 쇼츠/틱톡/릴스 |
| 밋업 사진/영상 | "하우제로 밋업 후기" 인스타/링크드인 | 인스타/링크드인 |
| 밋업 참석자 인터뷰 | 30초 후기 숏폼 (사회적 증거) | 전 채널 |
| 워크샵 워크시트 | 리드마그넷으로 재활용 | 이메일 |
| Q&A 질문 모음 | "가장 많이 물어본 AI 질문 TOP 10" 블로그 | SEO |
| 컨퍼런스 키노트 | 유튜브 메가 롱폼 (SEO + AEO) | 유튜브/블로그 |
| 컨퍼런스 보도 | PR 보도자료 | 언론 |

```
웨비나 1회 = 롱폼 1개 + 숏폼 5개 + 블로그 1개 + 뉴스레터 소재
밋업 1회 = 롱폼 1개 + 숏폼 10개 + 인스타 5개 + 링크드인 3개
컨퍼런스 = 롱폼 3개 + 숏폼 20개 + 보도자료 + 연간 하이라이트 영상

→ 이벤트 3개월 치 콘텐츠 = 일반 콘텐츠 제작 2개월 분량
```

## 4. 이벤트 마케팅 퍼널

### 4.1 등록 → 참석 → 전환 최적화

```
[등록 확보]
├── 뉴스레터 공지 (기존 구독자 — 가장 높은 전환)
├── 숏폼 CTA: "다음 주 라이브에서 직접 보여드립니다"
├── 링크드인 이벤트 생성 + 포스트
├── 커뮤니티 (카카오톡/디스코드) 공지
├── 파트너 크로스 프로모션
└── (Q3~) 유료 광고 리타겟팅 (MOFU→이벤트)

[참석율 극대화]
├── D-7: 등록 확인 이메일 + 캘린더 초대(.ics 파일)
├── D-3: "준비물 안내" 이메일 (기대감 조성)
├── D-1: 리마인더 이메일 + 카카오톡 알림
├── D-0 (당일): 30분 전 "곧 시작합니다" 알림
└── 노쇼 대응: D+1 "놓치셨나요? 녹화본 보기" 이메일

[전환 극대화]
├── 라이브 중 투표: "무료 오딧 관심 있으신 분?" (핸드레이즈)
├── 라이브 중 한정 오퍼: "오늘 참석자만 오딧 50% 할인"
├── D+0: 녹화본 + 리소스 + CTA 이메일
├── D+2: "어제 못 다룬 질문 답변" 이메일 (가치+권위)
└── D+5: 미전환자 → 뉴스레터 너처링으로 편입
```

### 4.2 에버그린 웨비나 (Q4~)

인기 웨비나를 녹화본으로 전환 → 24/7 자동 재생:

```
[에버그린 웨비나]
    인기 세션 녹화본 → 마치 라이브처럼 재생
    자동 등록 → 자동 리마인더 → 자동 재생 → 자동 CTA
    → 대표가 자는 동안에도 오딧 신청 유입

도구: WebinarJam / EasyWebinar (에버그린 기능)
```
