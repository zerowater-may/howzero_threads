## 1. 전략 배경

### 왜 마이크로 인플루언서인가?

| | 메가 인플루언서 | 마이크로 (1K~30K) |
|---|---|---|
| 비용 | ₩500만~수천만 | ₩10~50만 (or 무료 교환) |
| 참여율 | 1~2% | **4~8%** |
| 타겟 정확도 | 넓고 분산 | **니치에 정확** |
| 신뢰도 | 광고 느낌 | **친구 추천 느낌** |
| 전환율 | 낮음 | **높음** |
| 접근성 | DM 무시 | **직접 소통 가능** |

### 하우제로에 맞는 마이크로 인플루언서

| 카테고리 | 예시 | 팔로워 | 시너지 |
|---------|------|:---:|------|
| **1인 기업/프리랜서 유튜버** | 생산성/자동화 콘텐츠 | 3K~20K | "이 도구로 시간 아꼈어요" |
| **노코드/자동화 크리에이터** | Claude Code/Make 튜토리얼 채널 | 2K~15K | "이 템플릿 써봤는데 좋아요" |
| **비즈니스 코치 인스타** | 경영/마케팅 팁 | 5K~30K | "AI 오딧 받아봤는데..." |
| **이커머스 셀러 유튜버** | 스마트스토어/쿠팡 | 5K~30K | "CS 자동화 실제 적용기" |
| **직장인 부업 크리에이터** | 사이드 프로젝트/자동화 | 3K~20K | "비개발자도 가능한 AI" |

## 2. 협업 모델 3가지

### 2.1 제품/서비스 교환 (비용 ₩0)

```
하우제로 제공: 무료 AI 오딧 30분 + Claude Code 템플릿 팩 (₩29K)
크리에이터 제공: 솔직한 후기 영상/포스트 1개

→ 비용 0원으로 진정성 있는 UGC 확보
→ 크리에이터도 "AI 오딧 체험기"라는 콘텐츠 소재 얻음
```

### 2.2 유료 협업 (₩10~50만)

```
하우제로 제공: ₩10~50만 + 무료 오딧 + 템플릿
크리에이터 제공:
  - 전용 리뷰 영상 1개 (5~10분)
  - 숏폼 2~3개 (30~60초)
  - 스토리/포스트 3개
  - 추천 링크 (어필리에이트 추적)

→ 구독자 5K~20K 크리에이터 기준
→ CPM 환산: ₩5,000~10,000 (일반 광고 대비 저렴 + 높은 전환)
```

### 2.3 어필리에이트 (성과 연동)

```
하우제로 제공: 전용 추천 링크 + 유료 전환 시 20% 커미션
크리에이터 제공: 자발적 추천/리뷰

→ 사전 비용 0원
→ 전환이 일어나야 비용 발생 = 리스크 제로
→ P2-B2B 파트너 어필리에이트 시스템과 통합
```

## 3. 크리에이터 섭외 플레이북

### 3.1 타겟 리스트 구축

```
[Claude Code 자동 리서치 워크플로우]

1. YouTube API: "AI 자동화", "Claude Code", "1인 기업 자동화" 검색
   → 구독자 2K~30K 채널 필터
2. Instagram: #노코드, #업무자동화, #1인기업 해시태그
   → 팔로워 3K~30K 계정 필터
3. 결과 → Notion "크리에이터 타겟" DB 자동 저장
4. 참여율/콘텐츠 품질 수동 검증
5. 최종 타겟 20명/분기 선정
```

### 3.2 아웃리치 시퀀스

```
Step 1 (워밍업 2주):
  크리에이터 영상 3개에 가치 있는 댓글
  인스타 스토리 답장 / 포스트 공유

Step 2 (DM 접근):
  "[이름]님, 최근 [영상 제목]에서 다루신 [주제] 정말 공감했습니다.
   저는 비개발자 AI 자동화 전문 하우제로입니다.
   [이름]님 구독자분들에게 도움 될 만한 제안이 있는데,
   잠깐 이야기할 수 있을까요?"

Step 3 (제안):
  "무료로 AI 오딧 30분을 체험해보시겠어요?
   [이름]님의 업무 프로세스를 진단해드리고,
   그 과정과 결과를 콘텐츠로 만드시면
   구독자분들에게도 가치 있을 거예요.

   + Claude Code 템플릿 팩도 무료로 드립니다."

Step 4 (협업 진행):
  오딧 → 후기 영상 촬영 → 게시 → 추적
```

## 4. UGC 콘텐츠 활용

### 4.1 크리에이터 UGC 재가공

```
[크리에이터가 만든 후기 영상 (5분)]
    ├── 하우제로 유튜브: "크리에이터 리뷰" 플레이리스트에 추가
    ├── 숏폼: 핵심 15~30초 컷 추출 → 쇼츠/릴스/틱톡
    ├── 인스타 캐러셀: 크리에이터 후기 인용 + 결과 수치
    ├── 가격 페이지: 영상 임베드 (사회적 증거)
    ├── BOFU 광고 소재: "실제 크리에이터의 솔직 리뷰"
    └── 뉴스레터: "이번 주 크리에이터 [이름]님의 후기"

→ UGC 1개 = 6~8개 자산
```

### 4.2 시딩 캠페인 (대량 배포)

분기 1회, 10명 크리에이터에게 동시 시딩:

```
[시딩 캠페인]
  10명에게 동시에 무료 오딧 + 템플릿 제공
  → 10개 리뷰/후기가 2~3주 내 동시 게시
  → 타겟 오디언스의 피드에 "하우제로"가 여러 곳에서 보임
  → "여기저기서 보이네" = 브랜드 인지도 폭발
  → Mere Exposure Effect (단순 노출 효과)
```

## 7. 12주 타임라인

| 주차 | 작업 |
