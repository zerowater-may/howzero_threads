# P3: SMS/알림톡 + 푸시 알림 마케팅 — 이메일이 안 되는 순간의 백업 채널

> **작성일**: 2026-03-29 | **우선순위**: P3 (중간 — 이메일/카톡과 병행)
> **기간**: 2026 Q3 (CRM 안정화 후)
> **예산**: 월 10만원
> **담당**: 하우제로 대표 + Claude Code 자동화

---

## 1. 전략 배경

### 채널별 도달률 비교

| 채널 | 오픈/열람률 | 즉시성 | 비용/건 |
|------|:---:|:---:|:---:|
| 이메일 | 20~40% | 느림 | ₩0 |
| 카카오 알림톡 | 85~90% | 빠름 | ₩8~15 |
| **SMS/LMS** | **95%+** | **즉시** | **₩15~30** |
| 웹 푸시 | 5~15% | 즉시 | ₩0 |

### 언제 SMS/알림톡을 쓰는가?

```
[이메일로 충분한 경우] → 이메일만 (비용 ₩0)
  뉴스레터, 콘텐츠, 장기 너처링

[즉시 전달이 필요한 경우] → 알림톡 + 이메일
  오딧 예약 확인/리마인더
  웨비나 D-1/당일 알림
  기한 한정 프로모션 (마감 24시간)

[최후의 수단] → SMS
  알림톡 발송 실패 시 SMS 폴백
  오딧 노쇼 방지 D-0 알림
  긴급 시스템 알림
```

## 2. 멀티 채널 시퀀스 설계

### 2.1 우선순위 폭포 (Waterfall)

```
[이벤트 발생] (예: 오딧 D-1 리마인더)
    ↓
[1차] 카카오 알림톡 발송 (₩8~15)
    → 열람 확인
    → 열람 O → 완료
    → 열람 X (4시간 내)
        ↓
[2차] 이메일 발송 (₩0)
    → 오픈 확인
    → 오픈 O → 완료
    → 오픈 X (12시간)
        ↓
[3차] SMS 발송 (₩15~30) ← 최후 수단
    → 거의 100% 도달
```

→ **비용 최적화**: 알림톡 먼저(저렴+높은 열람) → 실패 시만 SMS(비싸지만 확실)

### 2.2 트리거별 채널 매핑

| 트리거 | 1차 | 2차 | 3차 |
|--------|-----|-----|-----|
| 뉴스레터 발행 | 이메일 | - | - |
| 오딧 예약 확인 | 알림톡 | 이메일 | - |
| 오딧 D-1 리마인더 | 알림톡 | 이메일 | SMS |
| 오딧 D-0 (2시간 전) | 알림톡 | SMS | - |
| 웨비나 D-1 | 알림톡 | 이메일 | - |
| 웨비나 시작 30분 전 | 알림톡 | SMS | - |
| 프로모션 마감 24시간 | 알림톡 | 이메일 | - |
| 프로모션 마감 1시간 | SMS | - | - |
| 결제 완료 감사 | 알림톡 | 이메일 | - |
| 14일 비활성 재참여 | 이메일 | 알림톡 | - |

## 3. Claude Code 자동화 워크플로우

### 3.1 알림톡/SMS 통합 발송 워크플로우

```
[Claude Code 워크플로우: 멀티채널 알림]

[트리거] CRM 이벤트 감지 (Notion Webhook)
    → 이벤트 유형 판별 (예약확인/리마인더/프로모/재참여)
    → 채널 우선순위 결정 (이벤트 유형별 매핑 테이블)
    → 1차 채널 발송:
        알림톡 → Solapi API
        이메일 → 스티비 API
        SMS → Solapi API
    → 열람/오픈 확인 (4~12시간 대기)
    → 미열람 시 → 2차 채널 자동 발송
    → 결과 → Notion CRM 발송 이력 기록
```

### 3.2 알림톡 템플릿 (사전 등록 필수)

| 템플릿 ID | 용도 | 본문 |
|----------|------|------|
| HZ-001 | 오딧 예약 확인 | "[이름]님, AI 오딧이 예약되었습니다. 일시: [날짜] [시간]. 사전 설문: [링크]" |
| HZ-002 | 오딧 D-1 | "[이름]님, 내일 [시간] AI 오딧이 있습니다. Zoom 링크: [링크]" |
| HZ-003 | 웨비나 알림 | "오늘 [시간] 하우제로 AI 자동화 라이브! 참여: [링크]" |
| HZ-004 | 결제 감사 | "[이름]님, 결제 완료! 시작 가이드: [링크]. 궁금한 점은 카톡 주세요." |
| HZ-005 | 프로모션 마감 | "⏰ [프로모명] 오늘 자정 마감! 지금 신청: [링크]" |

### 3.3 웹 푸시 알림 (보조)

```
[웹 푸시 셋업]
  howzero.com 방문자에게 푸시 알림 구독 요청
  도구: OneSignal (무료 플랜)

  용도:
  - 새 블로그 발행 알림
  - 웨비나 시작 알림
  - 프로모션 알림

  장점: 무료, 이메일 미오픈자에게 도달
  단점: 구독률 5~10%, 오픈률 낮음

  → 보조 채널로만 활용
```
