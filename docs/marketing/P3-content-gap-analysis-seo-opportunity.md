# P3-112: 콘텐츠 갭 분석 — 경쟁사가 다루지 않은 키워드 선점
> P3 | 월 3만원 | 대표

## 핵심: 경쟁사 블로그/유튜브가 다루는 키워드 vs 하우제로가 다루는 키워드 → 갭 = 기회

## Claude Code 자동 갭 분석

```
분기 1회:
  → Ubersuggest: 경쟁사 3곳의 랭킹 키워드 추출
  → 하우제로 랭킹 키워드와 비교
  → "경쟁사는 랭킹하지만 하우제로는 없는 키워드" = 갭
  → 갭 키워드 → 콘텐츠 제작 우선순위에 반영
```

## 갭 유형

| 갭 유형 | 예시 | 액션 |
|--------|------|------|
| 키워드 갭 | "Make.com 튜토리얼" (경쟁사 있음, 우리 없음) | 블로그+영상 제작 |
| 포맷 갭 | 경쟁사는 블로그만, 영상 없음 | 영상으로 선점 |
| 언어 갭 | 영문만 있고 한국어 없음 | 한국어 버전 제작 |
| 업종 갭 | "부동산 AI 자동화" 아무도 안 다룸 | 니치 선점 |

## KPI: 분기 갭 키워드 20개 발견, 3개월 내 10개 선점
