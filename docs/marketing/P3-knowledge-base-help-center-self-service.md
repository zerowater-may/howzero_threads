
## SEO 효과

- FAQ 각 항목 = 독립 URL = 검색 노출
- FAQ Schema.org 마크업 → Google Rich Snippet
- "AI 컨설팅 비용", "Claude Code 사용법" 등 롱테일 키워드 선점
- AEO: 구조화된 Q&A = AI 인용에 최적

## CS 부하 감소

```
현재: 모든 질문 → 대표 직접 답변 (주 2~3시간)
Help Center 후: 90% → "여기 보세요 → [링크]" (주 15분)
절감: 주 2시간+ → 연 100시간+
```
