## 1. 전략 배경

### 왜 사회적 증거가 전환의 핵심인가?

> "영상 후기는 가장 강력한 마케팅 도구. 진짜 고객이 진심으로 추천하는 모습 — 이걸 이기는 광고는 없다." (Danny Avila, 2026)

- **UGC(사용자 생성 콘텐츠) = 2026년 최고 전환 도구** — 리뷰, 증언, 실제 사진/영상 (NetLZ, 2026)
- **전환 목표라면 영상 후기 + 케이스 스터디가 최강** — 구매 전 마지막 신뢰 질문에 답변 (Vidlo Video)
- **60초 후기 → 15초 틱톡 훅 + 스틸 이미지 캐러셀 + 상품 페이지 삽입** — 1개 후기 = 5개 자산 (JoinBrands)
- **B2B 마케터에게 사회적 증거는 2026년 필수** — 후기/사례/전문가 추천이 모든 터치포인트에 통합 (Kacerr)

### 하우제로의 사회적 증거 자산

```
[현재 보유 가능한 증거 유형]

1. 대표 본인의 성과: "연매출 10억 SaaS" ← 이미 활용 중
2. 컨설팅 고객 후기: ← 아직 체계적 수집 안 됨
3. 코스 수강생 후기: ← Q3~ 축적 시작
4. AI 준비도 진단 결과 공유: ← P1-진단에서 자동 수집 중
5. 챌린지 참여자 인증: ← P2-챌린지에서 UGC 생산

→ 2~5번을 체계적으로 수집+활용하는 시스템이 없다
→ 이 기획서가 해결
```

## 2. 사회적 증거 5가지 유형 + 수집 방법

### 2.1 유형별 수집 전략

| # | 유형 | 수집 방법 | 활용 채널 | 전환 영향 |
|---|------|----------|----------|:---:|
| 1 | **영상 후기 (30~60초)** | 프로젝트 완료 후 Zoom 녹화 요청 | 유튜브/숏폼/광고/가격페이지 | ★★★★★ |
| 2 | **텍스트 후기 (카톡/이메일)** | 프로젝트 완료 D+3 자동 요청 | 웹사이트/뉴스레터/SNS | ★★★★ |
| 3 | **Before/After 수치** | 오딧 전 vs 3개월 후 데이터 | 케이스 스터디/캐러셀/블로그 | ★★★★★ |
| 4 | **SNS 멘션/태그** | #하우제로 해시태그 모니터링 | 리포스트/스토리 공유 | ★★★ |
| 5 | **NPS 점수 + 한줄평** | 프로젝트 완료 D+7 자동 설문 | 가격 페이지/제안서 | ★★★★ |

### 2.2 영상 후기 수집 프로세스

```
[프로젝트 완료]
    ↓
[D+3] 자동 이메일: "감사합니다! 30초 영상 후기 부탁드려도 될까요?"
    → 후기 질문 가이드 포함:
      1. "하우제로를 알기 전 가장 큰 고민은?"
      2. "어떤 서비스를 받으셨나요?"
      3. "가장 큰 변화/성과는?"
      4. "비슷한 고민을 하는 분에게 한 마디?"
    → Zoom 녹화 링크 or 스마트폰 셀카 영상 업로드 링크
    ↓
[수집 완료]
    → 편집 (30초/15초/60초 3버전)
    → 자막 자동 생성
    → 활용 채널별 배포
```

### 2.3 텍스트 후기 자동 수집 (Claude Code)

```
[Claude Code 워크플로우: 후기 자동 수집]

[트리거] Notion CRM Stage 8(졸업) 업데이트
    → D+3: 자동 이메일 "후기 한 줄만 부탁드립니다"
        "[서비스명] 덕분에 [변화]했습니다" 형식 가이드
    → D+5: 미응답 시 카카오톡 리마인더
    → D+7: NPS 설문 (1~10점) + "한줄평" 자동 발송
    → 수집된 후기 → Notion "후기 DB"에 자동 저장
    → NPS 9~10 → 영상 후기 추가 요청 자동 트리거
    → Slack 알림: "새 후기 수집! [고객명] NPS [점수]"
```

## 3. 사회적 증거 활용 매트릭스

### 3.1 터치포인트별 배치

| 터치포인트 | 배치할 증거 | 목적 |
|-----------|-----------|------|
| **가격 페이지** | 영상 후기 2~3개 + NPS 평균 + 텍스트 후기 5개 | 전환 직전 신뢰 |
| **오딧 신청 페이지** | Before/After 수치 + "이 오딧으로 ₩2,400만 절감" | 오딧 신청 유도 |
| **유튜브 영상 설명란** | "고객 후기: [한줄]" + 후기 영상 링크 | 채널 신뢰 |
| **뉴스레터** | 주간 "고객 스포트라이트" 코너 | 구독 유지 + 전환 |
| **인스타 캐러셀** | Before/After 카드뉴스 (7장) | 저장/공유 |
| **링크드인 포스트** | "고객 A의 이야기: CS 인건비 80% 절감" | B2B 신뢰 |
| **BOFU 광고 소재** | 영상 후기 15초 컷 | 리타겟 전환 |
| **제안서/세일즈 자료** | 동종 업계 사례 + NPS + 후기 | 기업 세일즈 |
| **AI 준비도 결과 페이지** | "같은 점수에서 시작한 A 기업은 지금..." | 오딧 연결 |

### 3.2 후기 재가공 매트릭스

```
[영상 후기 1개 (60초)]
    ├── 유튜브 숏폼 (30초 편집) ──── 1개
    ├── 틱톡/릴스 (15초 훅 버전) ── 1개
    ├── 가격 페이지 임베드 ──────── 1개
    ├── 텍스트 추출 → 캐러셀 ────── 1개
    ├── 링크드인 인용 포스트 ────── 1개
    ├── 뉴스레터 인용 ──────────── 1개
    ├── 광고 소재 (BOFU) ─────── 1개
    └── 제안서 삽입 ──────────── 1개

    = 영상 후기 1개 → 8개 자산
```
