# P3: 마이크로 콘텐츠 데일리 — 매일 1분 콘텐츠로 상시 존재감

> **우선순위**: P3 | **예산**: 월 3만원 | **담당**: 대표+AI

---

## 핵심: 매일 1개 마이크로 콘텐츠(1~3문장)를 X/Threads/링크드인에 게시 → "매일 보이는 사람"

## Claude Code 자동 생성 워크플로우

```
매일 07:00:
  → Claude API: "비개발자 AI 자동화 관련 인사이트 1개를 2~3문장으로"
  → 하우제로 톤 적용
  → 3개 버전 생성 (X/Threads/LinkedIn 각 톤)
  → 대표 Slack 알림: "오늘의 마이크로 콘텐츠 3개 — 검토 후 '승인' 누르세요"
  → 승인 → Buffer API 자동 게시 (X 08:00 / Threads 09:00 / LinkedIn 10:00)
```

## 마이크로 콘텐츠 유형 7가지 (요일별 로테이션)

| 요일 | 유형 | 예시 |
|------|------|------|
| 월 | 💡 팁 | "Claude Code에서 HTTP Request 노드 하나면 웬만한 API 다 연결됩니다" |
| 화 | 📊 수치 | "이번 주 오딧 고객 3곳의 평균 AI 준비도: 44점" |
| 수 | 🔥 역발상 | "GPT-5 나왔다고 흥분하기 전에, GPT-4 제대로 써보셨나요?" |
| 목 | ❓ 질문 | "비개발자분들, AI로 가장 자동화하고 싶은 업무가 뭔가요?" |
| 금 | 📖 여정 | "오늘 컨설팅 중 깨달은 것: 진짜 문제는 도구가 아니라 프로세스" |
| 토 | 🛠️ 도구 | "오늘의 추천 도구: Cal.com — 예약 자동화의 끝판왕" |
| 일 | 💬 격려 | "코딩 몰라도 됩니다. 비즈니스 감각이면 충분합니다" |

## 대표 투입 시간: 매일 2분 (검토+승인만)

## KPI: 매일 3개 플랫폼 게시 (365일), 마이크로 → 프로필 방문 10+/일
## 예산: 월 ₩30,000 (Claude API + Buffer)
