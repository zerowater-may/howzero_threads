## 1. 전략 배경

### 왜 CRO가 P2인가?

모든 기획서(숏폼/SEO/광고/PR/파트너)가 트래픽을 howzero.com으로 보내고 있다. 그런데 **웹사이트가 전환 최적화 안 되어 있으면 = 새는 양동이에 물 붓기**.

```
[CRO 전후 비교]

CRO 전: 월 방문 1,000명 × 전환율 2% = 리드 20명
CRO 후: 월 방문 1,000명 × 전환율 5% = 리드 50명
→ 같은 트래픽에서 2.5배 리드 = 모든 마케팅 ROI 2.5배 상승
```

## 4. A/B 테스트 로드맵

### 4.1 테스트 우선순위 (가장 큰 임팩트 순)

| 순위 | 테스트 | 가설 | 측정 |
|:---:|--------|------|------|
| 1 | **히어로 헤드라인** | "코딩 없이 AI 자동화" vs "CS 인건비 80% 절감" | CTA 클릭율 |
| 2 | **CTA 문구** | "무료 AI 오딧 신청" vs "30분 무료 진단 받기" | 신청 전환율 |
| 3 | **CTA 색상** | 녹색 vs 주황 vs 파랑 | 클릭율 |
| 4 | **사회적 증거 위치** | 히어로 아래 vs 히어로 옆 | 스크롤 유지율 |
| 5 | **폼 필드 수** | 3개(이름/이메일/업종) vs 2개(이름/이메일) | 폼 완료율 |
| 6 | **이탈 팝업 유무** | 팝업 있음 vs 없음 | 이메일 수집률 |
| 7 | **영상 후기 유무** | 있음 vs 없음 | 오딧 신청률 |
| 8 | **가격 표시 방식** | 월 환산 vs 총액 | 상담 신청률 |

### 4.2 A/B 테스트 도구

| 도구 | 비용 | 용도 |
|------|------|------|
| **Vercel Edge Config** | 무료 | 서버사이드 A/B (빠름) |
| **Google Optimize 대체 (VWO Lite)** | 무료~₩50K/월 | 비주얼 A/B 에디터 |
| **GA4 실험** | 무료 | 간단한 리다이렉트 테스트 |

### 4.3 테스트 루틴

```
[격주 A/B 테스트 루틴]

Week 1 (월): 이전 테스트 결과 분석 → 승자 적용
Week 1 (수): 새 테스트 가설 설정 → 변형 제작
Week 1 (금): 새 테스트 배포
Week 2: 데이터 수집 (최소 100 방문/변형)
→ 격주 1개 테스트 = 연 26개 테스트 = 지속 개선 엔진
```

## 6. 히트맵/세션 녹화 분석

### 도구: Microsoft Clarity (무료)

```
[Clarity 분석 포인트]

1. 히트맵: 어디를 가장 많이 클릭하는가? → CTA 위치 최적화
2. 스크롤맵: 어디까지 스크롤하는가? → CTA를 스크롤 중단점 위에 배치
3. 세션 녹화: 방문자가 어디서 이탈하는가? → 이탈 원인 식별
4. Rage Click: 어디를 반복 클릭하는가? → UX 문제 발견
5. Dead Click: 클릭했는데 반응 없는 곳 → 링크 누락 수정

→ 매주 10개 세션 녹화 리뷰 (15분) → CRO 인사이트 발굴
```
