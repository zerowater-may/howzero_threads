# P3: 영상 후기 제작 시스템 — 고객이 쉽게 촬영하고 하우제로가 자동 편집

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: 월 5만원 | **담당**: 대표+AI

---

## 핵심: 고객이 "뭐라고 말하면 돼요?" → 가이드+질문 4개 보내면 끝

## 영상 후기 수집 프로세스

```
[D+3: 프로젝트 완료 후]
  자동 이메일: "30초만 시간 되시나요? 4가지 질문에 답변해주세요."

[질문 4개]
  1. "하우제로를 알기 전 가장 큰 고민은 뭐였나요?"
  2. "어떤 서비스를 받으셨나요?"
  3. "가장 큰 변화/성과는?"
  4. "비슷한 고민 하는 분에게 한 마디?"

[촬영 가이드]
  "스마트폰으로 편하게 셀카 모드로 찍어주세요.
   조용한 곳, 밝은 곳이면 충분합니다.
   완벽하지 않아도 됩니다 — 진짜가 가장 강력합니다."

[업로드] → Google Drive or VocalVideo 링크
```

## 자동 편집 (Claude Code + AI)

```
영상 수신
  → 자동 자막 생성 (Whisper)
  → AI 편집: 핵심 30초/15초 컷 자동 추출
  → 하우제로 로고 + 이름/업종 오버레이
  → 3버전 자동 출력:
    60초 (풀 버전 → 가격 페이지)
    30초 (숏폼 → 유튜브/틱톡/릴스)
    15초 (광고 소재 → BOFU)
```

## 활용: 1 후기 = 8개 자산

| 자산 | 채널 |
|------|------|
| 풀버전 60초 | 가격 페이지 임베드 |
| 숏폼 30초 | 유튜브 쇼츠/틱톡/릴스 |
| 광고 15초 | Meta BOFU 광고 |
| 텍스트 인용 | 링크드인/캐러셀/뉴스레터 |
| 스틸 이미지 | 인스타 포스트/스토리 |
| 제안서 삽입 | B2B 세일즈 자료 |
| Wall of Love | howzero.com/reviews |
| NPS 데이터 | 리포트/PR 소재 |

## 인센티브: 영상 후기 제출 → Claude Code 프리미엄 템플릿 1개 무료 (₩29K 상당)
