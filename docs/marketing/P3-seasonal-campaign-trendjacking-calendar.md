## 1. 전략 배경

### 에버그린 콘텐츠 vs 시즌 콘텐츠

하우제로의 기존 15개 기획서는 **에버그린(상시)** 전략이다. 시즌 캠페인은 이 위에 **시기별 폭발력**을 얹는다.

```
[에버그린] ═══════════════════════════════════ (꾸준한 기본 트래픽)
            ↑시즌 스파이크↑         ↑시즌 스파이크↑
[시즌 캠페인]    ████                   ████████
              새해 결심              블프/연말
```

### 핵심 원칙

- **TikTok 분기별 마인드셋**: Q1 리셋(자기개발) → Q2 감정/선물 → Q3 전환(백투스쿨) → Q4 소비 폭발 (TLinky, 2026)
- **시즌 콘텐츠는 1~3개월 전 준비** (PageRelease, 2026)
- **에버그린 70% + 시즌 30%가 최적 믹스** (TrySight, 2026)
- **니치 홀리데이 활용** — 대형 시즌뿐 아니라 업계 특화 이벤트에 연결 (TrySight)

## 3. 트렌드잭킹 (Trendjacking) 전략

### 3.1 트렌드잭킹이란?

**실시간 트렌드(뉴스, 밈, AI 신제품)에 올라타서 하우제로의 메시지를 연결하는 기법.**

```
[트렌드] "OpenAI가 GPT-5 발표!"
    ↓
[하우제로 트렌드잭킹]
    "GPT-5가 나왔다고 흥분하기 전에,
     아직 GPT-4도 제대로 안 쓰고 있잖아요.
     비개발자가 지금 당장 GPT로 할 수 있는 3가지 자동화"
    → 역발상(Contrarian) + 실전(Practical) + CTA(진단/오딧)
```

### 3.2 트렌드잭킹 대상

| 트렌드 유형 | 예시 | 반응 속도 | 콘텐츠 포맷 |
|-----------|------|----------|-----------|
| **AI 신제품 발표** | GPT-5, Claude 5, Gemini 3 | 24시간 내 | 숏폼 + 링크드인 핫테이크 |
| **AI 뉴스/논란** | AI 일자리 대체, 규제, 윤리 | 24~48시간 | 숏폼 역발상 + 뉴스레터 |
| **바이럴 밈/챌린지** | AI 밈, 업무 자동화 밈 | 12시간 내 | 틱톡/릴스 (밈 + 교육) |
| **업계 이벤트** | CES, Google I/O, WWDC | 이벤트 당일 | 롱폼 큐레이션 + 숏폼 |
| **한국 사회 이슈** | 주 4일제, 최저임금, 인력난 | 24~48시간 | 링크드인 + 뉴스레터 |

### 3.3 트렌드잭킹 대본 공식

```
[공식: TRAC]

T — Trend (트렌드 언급): "어제 OpenAI가 GPT-5를 발표했습니다"
R — Reframe (리프레이밍): "근데 솔직히, GPT-4도 제대로 못 쓰는 회사가 99%입니다"
A — Application (실전 적용): "지금 당장 GPT-4로 할 수 있는 자동화 3가지를 알려드립니다"
C — CTA (행동 촉구): "더 자세히 알고 싶으면 → 뉴스레터/오딧"
```

### 3.4 트렌드 모니터링 시스템

```
[Claude Code 워크플로우: 트렌드 감지]

매일 08:00 자동 실행
    → Google Alerts: "AI 자동화", "GPT", "Claude Code", "노코드"
    → Twitter/X API: AI 관련 트렌딩 토픽
    → YouTube Trending: AI/테크 카테고리
    → 결과 → Slack 채널 "#trends"에 자동 포스팅
    → 대표가 확인 → 반응할 가치가 있으면 → 즉시 콘텐츠 제작

[반응 기준]
  "이 트렌드에 하우제로의 관점을 더할 수 있는가?"
  YES → TRAC 공식으로 24시간 내 숏폼 1개 제작
  NO → 무시 (모든 트렌드에 반응하면 브랜드 희석)
```

## 5. 시즌별 프로모션 구조

### 5.1 할인/오퍼 원칙

```
[원칙]
1. 가격은 절대 내리지 않는다 — "할인"이 아니라 "보너스 추가"
2. 예외: 블프/연말은 연 1회 한정 직접 할인 허용
3. 시즌 오퍼는 희소성 + 기한 한정 필수

[오퍼 유형]
  A형 — 보너스 추가: "스탠다드 패키지 + 코스 무료 수강 (이번 달만)"
  B형 — 기한 할인: "블프 한정 30% OFF (72시간)"
  C형 — 무료 확대: "이번 달 무료 오딧 5건 추가 (선착순)"
  D형 — 번들: "코스 + 템플릿 + 커뮤니티 3개월 = 패키지 가격"
```

### 5.2 시즌별 오퍼 매핑

| 시즌 | 오퍼 유형 | 구체 내용 |
|------|----------|----------|
