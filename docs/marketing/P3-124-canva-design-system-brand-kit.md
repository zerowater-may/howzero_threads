# P3-124: Canva 디자인 시스템 + 브랜드 킷 — 전 채널 비주얼 통일
> P3 | 월 1.5만원(Canva Pro) | 대표
Canva Brand Kit에 하우제로 컬러/폰트/로고/템플릿 등록 → 캐러셀/썸네일/배너/인포그래픽 제작 시 자동 브랜드 적용. 템플릿 10종: 유튜브 썸네일(3종), 인스타 캐러셀(3종), 링크드인 캐러셀(2종), 스토리(1종), 배너(1종). "어떤 채널에서든 하우제로 콘텐츠는 한눈에 알아본다" = 브랜드 일관성. 팀원/앰배서더가 제작해도 동일 퀄리티 유지. KPI: 전 채널 디자인 통일률 100%, 템플릿 사용으로 디자인 시간 -70%.
