# P3: AI 도구 회사 전략적 파트너십 — Claude Code/Make.com 등과 공식 파트너 관계

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: 월 0원 | **담당**: 대표

## 핵심 컨셉

하우제로가 추천/교육하는 AI 도구 회사와 **공식 파트너십**을 맺어, 양방향 가치를 교환한다.

```
[하우제로 → 도구 회사]
  한국 시장에서 우리 도구를 추천/교육하는 전문가
  → 한국 사용자 유치 + 교육 콘텐츠 제공

[도구 회사 → 하우제로]
  공식 파트너 인증 + 어필리에이트 우대 + 공동 마케팅
  → 권위 + 수익 + 공동 노출
```

## 타겟 파트너 도구

| 도구 | 파트너 프로그램 | 하우제로 가치 |
|------|-------------|-------------|
| **Claude Code** | Claude Code Partner Program | 한국 1위 Claude Code 교육자/컨설턴트 |
| **Make.com** | Make Partner Program | 한국 비개발자 대상 Make 교육 |
| **Zapier** | Zapier Experts | 비교 콘텐츠 + 가이드 |
| **Lovable** | 아직 미정 | 한국 시장 진출 파트너 |
| **Chatbase** | 아직 미정 | 한국 챗봇 시장 |
| **Notion** | Notion Consulting Partners | AI 연동 자동화 |

## 파트너 혜택 (일반적)

| 혜택 | 내용 |
|------|------|
| **공식 파트너 배지** | "Certified Claude Code Partner" → 웹사이트/명함에 표시 |
| **어필리에이트 우대** | 일반 20% → 파트너 30~40% 커미션 |
| **공동 마케팅** | 도구 회사 블로그/뉴스레터에 하우제로 피처 |
| **얼리 액세스** | 신기능 베타 테스트 → 콘텐츠 선점 |
| **공동 웨비나** | 도구 회사 + 하우제로 합동 세미나 |
| **한국 마켓 지원** | 한국어 번역/로컬라이제이션 협업 |
| **리드 공유** | 도구 회사의 한국 문의 → 하우제로에 연결 |

## 파트너십 활용

1. **"Claude Code 공식 파트너" 타이틀** → 모든 콘텐츠/제안서에 표시
2. **도구 회사 블로그에 피처** → SEO 백링크 + 글로벌 노출
3. **공동 웨비나** → 도구 회사 오디언스 접근
4. **한국 리드 연결** → 도구 회사 고객 → 하우제로 컨설팅
