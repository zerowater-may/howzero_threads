# P3: Freemium 미니 SaaS 도구 — Product-Led Growth로 리드 무한 생성

## 핵심: AI 준비도 진단을 넘어, 독립적인 미니 SaaS 도구를 만들어 PLG

P1-AI 준비도 진단이 성공하면, 같은 공식으로 **추가 무료 도구**를 만든다. 각 도구가 독립적인 리드 생성 엔진이 된다.

## 미니 SaaS 후보 3가지

### 1. "AI 자동화 ROI 시뮬레이터" (SaaS 버전)

```
howzero.com/roi
  기능: 업종/규모/업무 입력 → AI가 맞춤 ROI 예측 + 로드맵 생성
  차별점: 단순 계산기가 아닌 AI 기반 맞춤 분석
  무료: 기본 분석
  프리미엄: 상세 리포트 (이메일 수집)
  → 리드 → 오딧 → 컨설팅
```

### 2. "프로세스 비효율 스캐너"

```
howzero.com/scan
  기능: 회사 프로세스 5개 입력 → AI가 자동화 가능 포인트 식별
  출력: 비효율 점수 + "자동화하면 이만큼 아낍니다"
  무료: 3개 프로세스까지
  프리미엄: 무제한 + PDF 리포트 (이메일)
```

### 3. "AI 도구 추천 매칭 엔진"

```
howzero.com/match
  기능: 5개 질문 → AI가 업종/예산/수준에 맞는 도구 스택 추천
  출력: 맞춤 AI 도구 추천 3개 + 세팅 가이드
  무료: 기본 추천
  프리미엄: 상세 비교표 + 템플릿 (이메일)
```

## PLG 퍼널

```
도구 사용 (무료) → "더 자세한 분석은?" → 이메일 입력
  → 리포트 수신 → 너처링 시퀀스 → 오딧/코스/컨설팅
```

## SEO/AEO: 각 도구 = 독립 키워드 선점 + AI 인용 대상

## KPI: 도구당 월 500+ 사용, 이메일 수집 200+/도구
## 예산: 도구당 ₩500,000 개발 (Vercel + Claude API)
