# P3: 케이스 스터디 다큐멘터리 시리즈 — "AI 전환 그 후" 미니 다큐

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: 월 10만원 | **담당**: 대표

---

## 핵심: 컨설팅 고객의 Before→During→After를 5~10분 미니 다큐로 제작

## 포맷: "AI 전환 그 후" 시리즈

```
[EP.01] "이커머스 A 대표 — CS 직원 3명 대신 AI가 일합니다"

[0:00] Before: A 대표의 고민 (인터뷰)
  "하루 4시간을 CS에 쓰고 있었어요. 정말 지쳤습니다."

[2:00] 하우제로와의 만남
  "무료 AI 오딧을 받았는데, 비효율이 5곳이나 있더라고요."

[4:00] 자동화 구축 과정 (타임랩스 + 화면 녹화)
  Claude Code 워크플로우 세팅 → 테스트 → 가동

[6:00] After: 3개월 후 (인터뷰 + 수치)
  "CS 시간이 4시간에서 30분으로 줄었어요.
   그 시간에 신상품 소싱을 하니까 매출도 올랐습니다."
  → Before/After 수치 오버레이

[8:00] A 대표의 한 마디
  "코딩 모르는 저도 됐으니까, 여러분도 됩니다."

[9:00] CTA
  "무료 AI 오딧 30분 → QR / 링크"
```

## 제작 워크플로우

```
1. 고객 동의 (프로젝트 완료 D+30에 제안)
2. Zoom 인터뷰 30분 (Before/After 질문)
3. 화면 녹화 자료 수집 (Claude Code 워크플로우 시연)
4. AI 편집 (OpusClip + CapCut) → 8~10분 롱폼 + 숏폼 5개
5. 고객 검수 → 승인 → 게시
```

## 활용: 1 다큐 = 유튜브 롱폼 + 숏폼 5개 + 블로그 + 캐러셀 + 제안서 + 가격페이지

## KPI: 분기 2편 제작 → 연 8편 → 가장 강력한 전환 콘텐츠
## 예산: 월 ₩100,000 (편집)
