# P3-115: 코워킹 스페이스 파트너십 — 1인 기업/스타트업 밀집 공간 공략
> P3 | 월 10만원 | 대표

## 핵심: WeWork/패스트파이브/스파크플러스 등과 제휴 → 입주사 대상 무료 세미나/오딧

## 파트너십 모델

```
하우제로 → 코워킹 스페이스:
  월 1회 무료 AI 자동화 세미나 (입주사 대상)
  입주사 전용 AI 오딧 10% 할인

코워킹 → 하우제로:
  세미나 공간 무료 제공
  입주사 뉴스레터에 하우제로 소개
  게시판/디지털 사이니지 포스터 노출
```

## 타겟 코워킹 (서울)

| 코워킹 | 입주사 수 | 위치 |
|--------|:---:|------|
| WeWork | 수백 | 강남/종로 |
| 패스트파이브 | 수백 | 전국 |
| 스파크플러스 | 수십 | 강남/성수 |
| 서울창업허브 | 수십 | 마포 |
| 구글 캠퍼스(전) | 수십 | 삼성 |

## 전환 퍼널

```
코워킹 세미나 (30명)
  → 오딧 신청 20% = 6건
  → 유료 전환 30% = 2건
  → ₩400만/세미나

월 1회 × 12개월 = ₩4,800만
비용: 공간 무료 + 교통/다과 ₩10만 = 연 ₩120만
ROI: 3,900%
```

## KPI: 코워킹 3곳 파트너십, 분기 3회 세미나, 세미나당 오딧 5건+
