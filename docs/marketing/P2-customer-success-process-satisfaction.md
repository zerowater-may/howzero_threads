# P2: 고객 성공 프로세스 — 1인이지만 시스템으로 CS를 운영

> **작성일**: 2026-03-29 | **우선순위**: P2 (높음 — 리텐션+추천의 근간)
> **기간**: 2026 Q2 즉시
> **예산**: 월 5만원
> **담당**: 하우제로 대표 + Claude Code

---

## 1. 전략 배경

### 고객 성공(Customer Success)이 왜 마케팅인가?

```
만족한 고객 1명 =
  ✅ 후기 1개 (사회적 증거)
  ✅ 추천 2~3명 (레퍼럴)
  ✅ 재구매/업셀 1건 (MRR)
  ✅ 케이스 스터디 1개 (콘텐츠)
  ✅ NPS 9~10 (브랜드 지표)

불만족 고객 1명 =
  ❌ 환불 (매출 손실)
  ❌ 부정 후기 (브랜드 훼손)
  ❌ 안티 (SNS 부정 바이럴)
```

→ **고객 성공 = 마케팅의 가장 효율적인 투자**

---

## 2. 고객 성공 3원칙

```
1. 선제적(Proactive): 고객이 불만을 말하기 전에 먼저 체크
2. 체계적(Systematic): 감이 아니라 프로세스로 관리
3. 자동화(Automated): 1인이어도 100명 고객을 관리 가능
```

---

## 3. 고객 성공 타임라인 (프로젝트 기준)

```
[계약 체결]
  → D+0: 웰컴 (P2-온보딩 연동)
  → D+3: 킥오프 미팅
  → D+5: 첫 성과 데모
  → D+7: 첫 주 체크인 설문 (1~5점)

[프로젝트 진행 중]
  → 매주: 진행 상황 이메일 (자동 생성)
  → 2주차: 중간 체크인 통화 (15분)
  → 완료 1주 전: "마무리 준비 안내" 이메일

[프로젝트 완료]
  → D+0: 완료 축하 + 결과 리포트 이메일
  → D+3: NPS 설문 (1~10) + 후기 요청
  → D+7: 리테이너 제안 (P2-MRR)
  → D+14: "2주 후 어떠세요?" 체크인
  → D+30: "1개월 후 성과는?" + 케이스 스터디 제안
  → D+90: "3개월 결산" + 추천 인센티브 리마인더

[장기]
  → 분기: 계절 인사 + "새 서비스/코스 안내"
  → 연말: Spotify Wrapped式 연간 리캡
```

## 4. Claude Code 고객 성공 자동화

### 4.1 자동 체크인 워크플로우

```
[트리거] Notion CRM 프로젝트 일정 기반

  프로젝트 시작 D+7:
    → 자동 이메일: "첫 주 만족도 설문" (Typeform 링크)
    → 점수 3 이하 → Slack 즉시 알림 → 대표 직접 연락

  프로젝트 완료 D+3:
    → 자동 이메일: NPS 설문
    → NPS 9~10 → 자동 후기/추천 요청 이메일
    → NPS 7~8 → "개선할 점?" 추가 질문
    → NPS 1~6 → Slack 긴급 알림 → 대표 30분 내 연락

  완료 D+30:
    → 자동 이메일: "자동화 1개월 성과 어떠세요?"
    → Before/After 수치 요청 → 케이스 스터디 소재

  완료 D+90:
    → 자동 이메일: "3개월 결산 + 추천 인센티브"
```

### 4.2 고객 건강 점수 (Health Score)

```
[자동 계산 — Notion + Claude Code]

건강 점수 = 다음 5가지 합산 (각 20점, 총 100점):

1. 응답성 (20점): 이메일/카톡 응답 속도
   24시간 내 = 20 / 48시간 = 10 / 무응답 = 0

2. 참여도 (20점): 뉴스레터 오픈 + 커뮤니티 활동
   활성 = 20 / 가끔 = 10 / 비활성 = 0

3. NPS (20점): 최근 NPS 점수
   9~10 = 20 / 7~8 = 10 / 1~6 = 0

4. 추가 구매 (20점): 업셀/재구매 이력
   있음 = 20 / 관심 표명 = 10 / 없음 = 0

5. 추천 활동 (20점): 추천 건수
   1건+ = 20 / 후기 작성 = 10 / 없음 = 0

→ 80~100: 🟢 건강 (핵심 고객 — VIP 대우)
→ 50~79: 🟡 주의 (정기 체크인 필요)
→ 0~49: 🔴 위험 (즉시 개입)
```

## 5. 고객 불만 대응 프로세스

```
[불만 접수]
  카카오톡 / 이메일 / 전화
    ↓
[30분 내 1차 응답] (자동 or 수동)
  "말씀 감사합니다. 확인 후 [시간] 안에 답변드리겠습니다."
    ↓
[24시간 내 해결 or 해결 계획 공유]
  "이렇게 해결하겠습니다: [구체적 계획]"
    ↓
[해결 완료]
  "해결되었습니다. 다시 한번 불편 드려 죄송합니다.
   [보상: 다음 서비스 10% 할인 or 템플릿 1개 무료]"
    ↓
[D+3 팔로업]
  "그 후로 문제 없으시죠? 추가 도움 필요하시면 언제든 말씀주세요."

[원칙]
  1. 30분 내 1차 응답 (자동이라도)
  2. 변명하지 않고 해결에 집중
  3. 과잉 보상 (기대 이상의 대응)
  4. 팔로업으로 관계 복구 확인
```

## 6. 고객 성공 → 마케팅 자산 변환

| 고객 성공 산출물 | 마케팅 활용 |
|--------------|-----------|
| NPS 9~10 고객 | 후기 요청 → 영상/텍스트 후기 (P2-UGC) |
| 케이스 스터디 데이터 | 블로그/캐러셀/영상 (P2-니치) |
| 추천 인센티브 | 레퍼럴 프로그램 연동 (P3-레퍼럴) |
| 고객 건강 점수 데이터 | 리포트/콘텐츠 소재 (P3-리포트) |
| 불만 해결 스토리 | "실패를 이렇게 극복했습니다" 콘텐츠 |
| 재구매/업셀 사례 | "리테이너로 지속 가치 제공" 사례 |
