# P2: AI 챗봇 + 자동 리드 퀄리파이 — 24시간 세일즈봇으로 오딧 신청 자동화

> **작성일**: 2026-03-29 | **우선순위**: P2 (높음 — 전환율 2~4배 증가)
> **기간**: 2026 Q2 (5월 구축, 6월~ 운영)
> **예산**: 초기 10만원 + 월 5만원
> **담당**: 하우제로 대표 + Claude Code 자동화

---

## 1. 전략 배경

### 왜 AI 챗봇인가?

- **AI 챗봇 = 정적 폼 대비 전환율 2~4배** (FastBots, 2026 실제 케이스 스터디)
- **24시간 작동** — 대표가 자는 동안에도 리드 퀄리파이 + 오딧 예약
- **즉각 응답** — 방문자가 "궁금한 순간"에 바로 답변 → 이탈 방지
- **자동 퀄리파이** — 업종/규모/예산/긴급도 자동 파악 → CRM 자동 태깅

### 하우제로 챗봇 = 자체 쇼케이스

하우제로는 **AI 자동화를 파는 회사**다. 자체 웹사이트에 AI 챗봇이 없으면 설득력이 떨어진다. 챗봇 자체가 **"우리가 만드는 게 이런 것"**의 라이브 데모.

### 2.2 챗봇 대화 시나리오

#### 시나리오 A: 첫 방문자 → 오딧 신청 유도

```
봇: "안녕하세요! 하우제로입니다 👋
     비개발자를 위한 AI 자동화 전문가예요.

     어떤 것이 궁금하세요?
     1️⃣ AI 자동화가 우리 회사에도 되나요?
     2️⃣ 서비스/가격이 궁금해요
     3️⃣ 무료 AI 오딧 신청하고 싶어요
     4️⃣ 그냥 둘러보는 중이에요"

사용자: "1" (AI 자동화 관심)

봇: "좋은 질문이에요!
     AI 자동화는 거의 모든 업종에 적용 가능합니다.

     혹시 어떤 업종에서 일하시나요?
     (이커머스/서비스업/법률·회계/부동산/병원/교육/기타)"

사용자: "이커머스"

봇: "이커머스라면 이런 자동화가 많이 됩니다:
     • CS 문의 자동 분류/응답 (인건비 80% 절감)
     • 리뷰 자동 수집/분석
     • 경쟁사 가격 자동 모니터링
     • 주문/환불 자동 처리

     실제로 이커머스 고객이 월 200시간을 절감한 사례가 있어요.

     우리 회사에도 적용 가능한지 확인하려면
     무료 AI 오딧 30분을 받아보세요. (무료, 부담 없이)

     👉 [오딧 예약하기]

     아니면 먼저 AI 준비도를 진단해볼 수도 있어요. (3분)
     👉 [AI 준비도 진단]"
```

## 3. 기술 구현

### 3.1 옵션 A: Chatbase (노코드, 빠른 셋업)

```
[Chatbase 구현]
1. Chatbase 계정 생성 ($19/월)
2. howzero.com 전체를 크롤링 → 지식 기반 자동 학습
3. FAQ/가격/서비스 정보 수동 추가
4. 톤 설정: "친근하고 직설적인 AI 자동화 전문가"
5. 위젯 코드 → howzero.com에 삽입
6. 리드 데이터 → Zapier/Claude Code → Notion CRM 연동

장점: 1~2시간 셋업, 노코드, 빠름
단점: 커스텀 한계, 월 $19 비용
```

### 3.2 옵션 B: 자체 구축 (Claude Code + Claude API, 추천)

```
[Claude Code 자체 챗봇 구현]
1. howzero.com에 간단한 채팅 UI 구현 (오픈소스 위젯)
2. 사용자 메시지 → Claude Code Webhook
3. Claude Code에서 Claude API 호출:
   - 시스템 프롬프트: 하우제로 페르소나 + 서비스 정보 + FAQ
   - 대화 맥락 유지 (세션별)
4. Claude 응답 → 채팅 UI로 반환
5. 리드 퀄리파이 정보 감지 → Notion CRM 자동 저장
6. 고가치 리드 → Slack 즉시 알림

장점: 완전 커스텀, 비용 저렴 (API만), 쇼케이스 가치
단점: 구축 시간 1~2일
```

### 3.3 자동 리드 퀄리파이 로직

```
[Claude Code 리드 스코어링]

대화 중 자동 추출:
  업종: [이커머스/법률/부동산/병원/교육/기타]
  규모: [1인/소규모/중소/중견]
  예산: [미정/50만 미만/50~200만/200만+]
  긴급도: [탐색 중/3개월 내/즉시]
  핵심 니즈: [CS/마케팅/업무/데이터/기타]

→ 리드 점수 자동 계산:
  업종 적합도 × 규모 × 예산 × 긴급도 = 점수

→ 점수별 자동 라우팅:
  80+: 🔴 HOT → Slack 즉시 알림 → 대표 직접 연락
  50~79: 🟡 WARM → 자동 이메일 시퀀스 → 오딧 CTA
  < 50: 🟢 COLD → 뉴스레터 편입 → 장기 너처링
```
