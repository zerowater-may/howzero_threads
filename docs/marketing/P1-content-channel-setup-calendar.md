## 2. 콘텐츠 제작 워크플로우

### 주간 배칭 (매주 화요일)

```
화요일 오전 (3시간):
  1. 롱폼 촬영 + 편집 (2시간)
  2. OpusClip으로 쇼츠 5개 클리핑 (20분)
  3. 쇼츠 → 틱톡 리포맷 3개 + 네이티브 2개 (20분)
  4. 캐러셀 2개 제작 - Canva (30분)
  5. 쓰레드 포스트 3개 작성 (10분)

화요일 오후:
  6. 콘텐츠 예약 업로드 (Buffer/Later)
  7. 유튜브 롱폼 업로드 + 쇼츠 예약

매일 (5분):
  - 인스타 스토리 1개 (날것 OK)
  - 댓글/DM 응대
```

### 원소스 멀티유즈 변환 맵

```
유튜브 롱폼 1개 (원본)
  ├── 유튜브 쇼츠 ×5
  ├── 틱톡 ×5 (쇼츠 3 + 네이티브 2)
  ├── 인스타 릴스 ×3
  ├── 인스타 캐러셀 ×2
  ├── 쓰레드 ×2
  └── 인스타 스토리 ×3
  = 총 20개 콘텐츠/주
```

### 인스타 캐러셀 포맷 로테이션

```
Week N:
  화 — 교육형 (스텝바이스텝 or 비교표)
  금 — 저장형 (체크리스트 or 도구 추천)

  체크리스트 → 비교표 → 스텝바이스텝 → Before/After → 도구 추천
```
