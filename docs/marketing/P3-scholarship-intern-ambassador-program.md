## 대학생 앰배서더 프로그램

| 항목 | 설계 |
|------|------|
| **인원** | 5~10명/분기 |
| **자격** | AI/자동화/마케팅 관심 대학생 |
| **활동** | 주 3시간 — 콘텐츠 제작/SNS 홍보/커뮤니티 관리 |
