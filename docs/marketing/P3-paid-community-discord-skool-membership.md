## 1. 전략 배경

### 무료 커뮤니티 → 유료 커뮤니티 전환

P2-이메일/커뮤니티 기획서에서 **무료 카카오톡 오픈채팅 → 디스코드**를 구축했다. 이 기획서는 그 위에 **유료 프리미엄 레이어**를 얹는 전략.

```
[커뮤니티 사다리]

FREE: 카카오톡 오픈채팅 (진입)
    → "AI 자동화 기본 Q&A + 네트워킹"
    → 누구나 무료 참여

FREE+: 디스코드 (중급)
    → "주간 도전 + 템플릿 공유 + 사례 토론"
    → 이메일 구독자 대상

PAID: Skool 프리미엄 (고급) ← 이 기획서
    → "월간 라이브 + 전용 템플릿 + 대표 직접 Q&A + 코스"
    → 월 ₩49,000~99,000
```

### 핵심 데이터

- **2026: AI 커뮤니티가 코스를 대체하는 새 지식 수익화 모델** (HumAI Blog)
- **Skool Pro $99/월**: 게이미피케이션 내장, 코스+커뮤니티 통합, 주간 정산 (Skool)
- **Skool 커뮤니티 수익화: 월 $5,000 달성 로드맵** (CommuniPass, 2026)
- **유료 커뮤니티의 핵심: 코스보다 "접근성"** — 전문가에게 직접 질문할 수 있는 권리 (CommuniPass)
- **멀티 플랫폼 스택: Skool(커뮤니티) + CommuniPass(챌린지) = 최적** (CommuniPass)

    ↓ "이거 계속 받고 싶으면 → 프로 멤버십"
    ↓
[유료 전환]
  프로 ₩49,000/월 or VIP ₩99,000/월

## 5. MRR(월간 반복 매출) 시뮬레이션

```
[런칭 월 (10월)]
  프로 멤버: 15명 × ₩49,000 = ₩735,000
  VIP 멤버: 5명 × ₩99,000 = ₩495,000
  MRR: ₩1,230,000
  비용: ₩150,000 (Skool + 운영)
  순이익: ₩1,080,000

[3개월 후 (1월)]
  프로: 40명 × ₩49,000 = ₩1,960,000
  VIP: 15명 × ₩99,000 = ₩1,485,000
  MRR: ₩3,445,000

[6개월 후 (4월)]
  프로: 80명 × ₩49,000 = ₩3,920,000
  VIP: 30명 × ₩99,000 = ₩2,970,000
  MRR: ₩6,890,000

[12개월 후]
  프로: 150명 + VIP: 50명
  MRR: ₩12,300,000 ← 커뮤니티만으로 월 1,230만원
```

### 이탈 방지 핵심

| 전략 | 설명 |
|------|------|
| **매주 새로운 가치** | 라이브/템플릿/도전 — 한 주도 빈 주 없음 |
| **커뮤니티 관계** | 멤버 간 관계 형성 → 떠나기 어려움 (스위칭 비용) |
| **레벨 시스템** | 포인트 축적 → "여기서 떠나면 레벨이 리셋" |
| **연간 결제 할인** | 월 ₩49K → 연 ₩490K (2개월 무료) → 이탈 잠금 |
| **졸업 없음** | 코스는 끝나지만 커뮤니티는 계속 → 영구 MRR |
