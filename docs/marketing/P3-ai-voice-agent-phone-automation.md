# P3: AI 음성 에이전트 + 전화 자동화 — 데모 서비스 겸 세일즈 쇼케이스

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: 월 10만원 | **담당**: 대표

---

## 핵심 컨셉

하우제로 웹사이트에 **AI 음성 에이전트(전화봇)**를 구축하여 2가지 목적을 달성:
1. **세일즈 쇼케이스**: "이게 저희가 구축할 수 있는 것입니다" 라이브 데모
2. **리드 퀄리파이**: 전화 문의를 AI가 1차 응대 → 핫 리드만 대표에게 연결

## AI 음성 에이전트 설계

```
[방문자가 웹사이트에서 "전화 상담" 클릭]
  → AI 음성 에이전트 연결 (Vapi or Bland.ai)

AI: "안녕하세요! 하우제로 AI 어시스턴트입니다.
     AI 자동화에 관심이 있으시군요.
     어떤 업종에서 일하시나요?"

사용자: "이커머스요"

AI: "이커머스라면 CS 자동화, 리뷰 분석, 재고 관리 등
     자동화할 수 있는 것이 많습니다.
     현재 가장 시간을 많이 빼앗기는 업무가 뭔가요?"

사용자: "CS 문의 처리요"

AI: "그렇군요. CS 자동화만으로도 월 200시간 이상
     절감한 사례가 있습니다.
     무료 AI 오딧 30분을 예약하시면
     정확한 절감 효과를 알려드릴 수 있어요.
     예약하시겠어요?"

사용자: "네"

AI: "성함과 이메일을 알려주시면 예약 링크를 보내드립니다."
  → 정보 수집 → Claude Code → CRM + Cal.com 예약 링크 자동 발송
  → 핫 리드 → Slack 즉시 알림
```

## 쇼케이스 가치

```
유튜브 콘텐츠:
  "우리 웹사이트에 AI 전화봇을 만들었습니다 (실제 통화 시연)"
  → 조회수 폭발 (신기함 + 실전)

컨설팅 세일즈:
  "이 전화봇도 저희가 만든 겁니다.
   귀사 CS에도 이런 AI 전화봇을 달아드릴 수 있어요."
  → 데모 → 전환

템플릿 판매:
  "AI 음성 에이전트 구축 가이드" → 코스/템플릿 소재
```
