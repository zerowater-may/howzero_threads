## 1. 퍼널 전체 구조

```
┌─────────────────────────────────────────────────────────────────┐
│                      TOFU (Top of Funnel)                       │
│                                                                 │
│   유튜브 롱폼/쇼츠 → "프로필 링크"                               │
│   인스타 캐러셀/릴스 → "프로필 링크" / 자동 DM                    │
│   틱톡 → "프로필 링크"                                           │
│   쓰레드 → DM / 프로필 링크                                      │
│                                                                 │
│   모든 CTA → howzero.com/link (링크트리)                         │
├─────────────────────────────────────────────────────────────────┤
│                      MOFU (Middle of Funnel)                    │
│                                                                 │
│   경로 A: howzero.com/ai-score (AI 준비도 진단)                  │
│           → 결과 페이지 → "무료 오딧 신청" CTA                    │
│           → howzero.com/audit                                    │
│                                                                 │
│   경로 B: howzero.com/audit (직접 진입)                          │
│           → 신청 폼 작성 → 확인 이메일                            │
│                                                                 │
│   경로 C: 뉴스레터 구독 → 3일 미니코스 이메일                     │
│           → Day 3 CTA: "무료 오딧 신청"                           │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                      BOFU (Bottom of Funnel)                    │
│                                                                 │
│   무료 오딧 30분 화상 진행                                       │
│   → 진단 결과 리포트 이메일 발송                                  │
│   → 후속 이메일 너처링 (5일 시리즈)                               │
│   → 유료 오딧/컨설팅/구축 제안                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 3. 랜딩페이지 개선안 — howzero.com/audit

### 현재 상태

기존 페이지(`src/app/(marketing)/audit/page.tsx`)에 히어로 + 혜택 리스트 + 신청 폼 + FAQ 구성 완료. 다만 전환율 극대화를 위해 아래 요소 추가 필요.

### 추가 필요 섹션

#### 3.1 사회적 증거 섹션 (Social Proof)

```
[히어로 바로 아래]

"100+ 기업이 이미 진단받았습니다"

[로고 슬라이더 or 숫자 카운터]
  진단 완료: 100+건
  평균 발견 비효율: 5.2건/기업
  평균 절감 예상액: 월 ₩320만

[고객 후기 카드 2~3개]
  "CS에 하루 3시간 쓰던 게 AI로 30분이 됐습니다" — 김○○, 스마트스토어 대표
  "오딧 하나로 월 200시간 절감 포인트를 찾았습니다" — 박○○, 마케팅 에이전시
```

#### 3.2 프로세스 시각화

```
[신청 폼 위]

"오딧은 이렇게 진행됩니다"

Step 1: 신청 (2분)
  → 아래 폼으로 간단한 정보 입력

Step 2: 일정 확인 (1일 이내)
  → 이메일로 화상 미팅 일정 안내

Step 3: 화상 오딧 (30분)
  → 비즈니스 프로세스 진단 + Quick Win 발견

Step 4: 리포트 수령 (오딧 후 2일)
  → 비효율 진단 결과 + 자동화 로드맵 PDF
```

#### 3.3 긴급성/제한 요소

```
[신청 폼 바로 위]

"이번 달 무료 오딧 잔여: 7/10건"
"다음 달부터 유료(₩50만) 전환 예정"
```

#### 3.4 이탈 방지 팝업

```
[이탈 감지 시]

"잠깐만요! 2분이면 됩니다."
"무료 AI 준비도 진단만 먼저 해보세요"
→ howzero.com/ai-score 로 리다이렉트
```

## 4. AI 준비도 진단 → 오딧 전환 브릿지

### 현재 상태

`/ai-score` 페이지에서 6개 질문 → 결과(A~D등급) → "무료 AI 오딧 신청" 버튼 연결 완료.

### 전환율 개선안

#### 4.1 결과 페이지에 이메일 수집 추가

```
[결과 표시 후]

"상세 진단 리포트를 이메일로 받아보세요"
[이메일 입력] [리포트 받기]

→ 이메일 입력 시:
  1. 즉시: 맞춤 진단 리포트 PDF 발송
  2. Day 1~3: 미니코스 이메일 시리즈 시작
  3. Day 3 CTA: 무료 오딧 신청 유도
```

#### 4.2 등급별 맞춤 CTA

| 등급 | CTA 메시지 | 연결 |
|:---:|-----------|------|
| **D** (시작) | "기본부터 시작해야 합니다. 전문가의 무료 진단으로 로드맵을 받으세요." | /audit |
| **C** (준비) | "Quick Win이 있습니다. 30분 오딧으로 당장 절감할 수 있는 포인트를 찾으세요." | /audit |
| **B** (성장) | "본격 자동화 시점입니다. 심화 진단으로 최적의 자동화 전략을 세우세요." | /audit |
| **A** (선도) | "고급 AI 전략이 필요합니다. 전략 자문으로 경쟁 우위를 강화하세요." | /audit |

```

───────────────────────────────────

{이름}님, 안녕하세요. 하우제로입니다.

무료 AI 오딧 신청을 확인했습니다.
영업일 기준 1일 이내로 화상 미팅 일정을 안내드리겠습니다.

■ 오딧 준비 체크리스트

미팅 전에 아래 3가지만 정리해 주시면 30분을 100% 활용할 수 있습니다:

1. 하루 중 가장 많은 시간을 쓰는 반복 업무 3가지
2. 현재 사용 중인 주요 도구 (엑셀, CRM, 메신저 등)
3. "이것만 자동화되면 좋겠다" 하는 것 1가지

■ 그동안 읽어보시면 좋은 자료

- AI 준비도 자가진단: howzero.com/ai-score
- [유튜브] "코딩 몰라도 10억 번 방법": [링크]

궁금한 점이 있으면 이 메일에 직접 답장해 주세요.

감사합니다.
하우제로 드림

───────────────────────────────────
```

■ 핵심 발견

1. {비효율 #1}: {설명}
   → 예상 절감: 월 {X}시간 / ₩{Y}만원

## 8. 전환 목표 & KPI

### 퍼널 단계별 전환율 목표

| 단계 | 지표 | 목표 (월) |
|------|------|----------|
| TOFU → 링크 클릭 | 프로필 링크 클릭 | 500회 |
| 링크 → AI 진단 완료 | 진단 완료율 | 150명 (30%) |
| 링크 → 오딧 신청 | 신청 전환율 | 40명 (8%) |
