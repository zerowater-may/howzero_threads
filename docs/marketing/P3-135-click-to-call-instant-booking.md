# P3-135: 즉시 예약(Click-to-Book) 위젯 — 모든 페이지에서 1클릭 오딧 예약
> P3 | ₩0 | 대표
howzero.com 모든 페이지 우하단에 "무료 오딧 예약" 플로팅 버튼. 클릭 → Cal.com 예약 팝업 (페이지 이탈 없음). 모바일: 전화 아이콘 + 예약 아이콘 동시 표시. "지금 이 순간 오딧 받고 싶다" → 1클릭이면 예약 완료. 현재: CTA 클릭 → 페이지 이동 → 예약 페이지 → 날짜 선택 (3단계). 개선: 플로팅 버튼 → 팝업 → 날짜 선택 (1단계). 전환 장벽 -66%. KPI: 플로팅 버튼 클릭 → 예약 전환 15%+, 전체 오딧 신청 +20%.
