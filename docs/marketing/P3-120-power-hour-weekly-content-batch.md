# P3-120: 파워 아워 주간 콘텐츠 배칭 — 화요일 3시간으로 1주일치 완성
> P3 | ₩0 | 대표

## 핵심: "매일 조금씩"이 아니라 "화요일 3시간 몰아서 다 끝내기"

## 파워 아워 스케줄 (매주 화요일)

```
09:00~09:30 (30분) — 롱폼 촬영 1개
09:30~10:00 (30분) — 숏폼 5개 촬영 (배치)
10:00~10:30 (30분) — Claude Code 자동 리퍼포징 트리거 + 초안 검토
10:30~11:00 (30분) — 뉴스레터 초안 검토/수정
11:00~11:30 (30분) — 캐러셀 2개 검토 + 링크드인 포스트 2개 검토
11:30~12:00 (30분) — 전체 예약 게시 승인 + 다음 주 주제 결정

총 3시간 → 주 25~30개 콘텐츠 완성 → 나머지 6일은 컨설팅/오딧/세일즈에 집중
```
