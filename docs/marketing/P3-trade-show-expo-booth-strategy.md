## 1. 전략 배경

### 왜 박람회/전시회인가?

- **하루에 수백~수천 명의 타겟 방문자**와 직접 대면
- **데모 시연**: AI 자동화를 직접 보여줄 수 있음 → 숏폼보다 강한 임팩트
- **리드 대량 수집**: 명함/QR → 수백 건 이메일 수집
- **PR/미디어**: 박람회 참가 자체가 보도 소재
- **경쟁사 벤치마킹**: 같은 분야 다른 업체 관찰

### 하우제로 부스 = 라이브 쇼케이스

```
[부스 방문자 경험]
  "이 화면 보세요. 지금 고객 문의가 들어왔죠?
   AI가 5초 만에 분류하고, 자동 답변 초안을 만들고,
   담당자한테 알림을 보냅니다. 코딩 0줄."

→ 30초 라이브 데모 = 유튜브 100개보다 강한 전환력
```

## 3. 부스 설계

### 3.1 미니 부스 (1~2테이블)

```
┌──────────────────────────────┐
│  [배너]                       │
│  "코딩 없이 AI 자동화          │
│   연매출 10억 실전가의         │
│   라이브 데모"                │
│                              │
│  [노트북 1] 라이브 데모        │
│  → Claude Code CS 자동화 실시간 시연   │
│                              │
│  [노트북 2] AI 준비도 진단     │
│  → 방문자 즉석 진단 (3분)     │
│                              │
│  [QR코드 스탠드]              │
│  → 무료 오딧 신청             │
│  → AI 준비도 진단             │
│  → 뉴스레터 구독              │
│                              │
│  [무료 굿즈]                  │
│  → 스티커 + 워크시트           │
│  → 명함 (QR 포함)            │
│                              │
│  [대표] 직접 설명 + 데모       │
└──────────────────────────────┘
```

### 3.2 핵심 데모 시나리오 (30초)

```
[라이브 데모: "CS 문의 자동 분류"]

Step 1: "지금 이 이메일 보이시죠? 고객이 환불 요청을 보냈습니다."
Step 2: "Claude Code이 자동으로 '환불'로 분류합니다. 3초."
Step 3: "AI가 응답 초안을 만듭니다. '환불 절차 안내' 5초."
Step 4: "담당자한테 카카오톡 알림이 갑니다."
Step 5: "이 전체 과정에 코딩 0줄. 세팅 시간 15분."

→ "우리 회사에도 되나요?"
→ "QR 찍으시면 무료 AI 오딧 30분 예약할 수 있습니다."
```

### 3.3 즉석 AI 준비도 진단

```
[부스 방문자 참여 이벤트]

"3분 만에 우리 회사 AI 준비도를 알 수 있습니다.
 노트북으로 바로 진단해보세요!"

→ 방문자가 직접 진단 → 결과 카드 즉석 출력 (프린터 or 이메일)
→ 이메일 자동 수집 → CRM 편입
→ "점수가 42점이시네요. 개선 방법은 무료 오딧에서 안내드립니다."
```

## 4. 리드 수집 자동화

```
[박람회 리드 수집 Claude Code 워크플로우]

[방문자] QR 스캔 or 명함 촬영
    → Google Form or Typeform (이름/이메일/업종/관심사)
    → Claude Code Webhook → Notion CRM 자동 등록
    → 태그: "출처: [박람회명]"
    → 즉시 자동 이메일: "만나서 반가웠습니다! 데모에서 보신 자동화 자료: [링크]"
    → D+3: "박람회 특별 무료 오딧 안내" 이메일
    → D+7: "박람회 한정 코스 10% 할인" 이메일
    → 이후: 일반 너처링 시퀀스 편입
```

## 5. 비용 최적화

### 5.1 무료/저비용 참가 전략

| 전략 | 비용 | 설명 |
|------|:---:|------|
| **데모데이 발표자** | ₩0 | 서울창업허브 등에 발표자로 지원 → 무료 부스 |
| **스폰서 파트너** | ₩0 | Claude Code/Make.com의 한국 파트너로 공동 부스 |
| **강연자 초청** | ₩0 | 박람회 세미나 강연자로 초청 → 무료 참가 + 노출 |
| **미디어 파트너** | ₩0 | 뉴스레터/유튜브 미디어 파트너로 등록 → 무료 패스 |

### 5.2 ROI 기준

```
[박람회 ROI 계산]

투자: ₩300만 (부스 + 제작물 + 교통)
리드 수집: 200명
→ 오딧 전환: 200 × 10% = 20건
→ 유료 전환: 20 × 25% = 5건
→ 매출: 5 × ₩200만 = ₩1,000만

ROI: (1,000 - 300) / 300 = 233%

→ ROI 200%+ 달성 못 하면 해당 박람회 불참
```

## 6. 콘텐츠 수확

```
[박람회 1회 = 콘텐츠 20개+]

  부스 현장 영상 → 숏폼 5개 (틱톡/쇼츠/릴스)
  라이브 데모 녹화 → 유튜브 롱폼 1개
  방문자 인터뷰 → 후기 숏폼 5개
  현장 사진 → 인스타 캐러셀 3개
  참가 후기 → 블로그 1개 + 링크드인 2개
  리드 데이터 → 뉴스레터 + PR 보도자료 1개
```
