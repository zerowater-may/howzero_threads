# P2: 인스타그램 캐러셀 + 릴스 전략 — 교육형 콘텐츠로 저장/공유 극대화

> **작성일**: 2026-03-29 | **우선순위**: P2 (높음 — 숏폼 크로스포스팅 + 독립 채널)
> **기간**: 2026 Q2 즉시 (P1-숏폼과 동시)
> **예산**: 월 10만원
> **담당**: 하우제로 대표 + AI 콘텐츠 엔진

---

## 1. 전략 배경

### 2026 인스타그램 알고리즘 핵심 변화

- **"Views"가 전 포맷(릴스/캐러셀/스토리/사진) 통합 지표로 전환** (Sprout Social, Catalyst, 2026)
- **저장(Saves)과 공유(Sends)가 좋아요보다 중요** — 교육/실용 콘텐츠에 유리 (PostEverywhere, OrangeMonke)
- **캐러셀 = 참여율 최고 포맷** — 교육/인사이트/체크리스트에 최적 (Social Insider, 2026)
- **릴스 = 발견/도달 최고** — 신규 팔로워 확보 (TryMyPost)
- **Trial Reels**: 팔로워 아닌 사람에게만 먼저 테스트 → 반응 좋으면 팔로워에게도 노출 (PostEverywhere)

### 하우제로에 인스타가 맞는 이유

```
[인스타그램 콘텐츠 적합도]

교육형 캐러셀 → 저장률 극대화 → "나중에 보려고 저장"
  "AI 자동화 5단계 체크리스트"
  "Claude Code vs Make.com 비교표"
  "CS 자동화 Before/After"

→ 저장 = 알고리즘 최고 신호
→ 교육 콘텐츠가 강한 하우제로에 최적
```

## 2. 포맷별 전략

### 2.1 캐러셀 (교육/저장 최적화)

| 항목 | 설계 |
|------|------|
| **비중** | 전체 콘텐츠의 40% |
| **슬라이드 수** | 7~10장 (최적 참여) |
| **비율** | 4:5 (세로, 피드 최대 면적) |
| **디자인** | Canva 통일 템플릿 (브랜드 컬러/폰트) |
| **CTA 마지막 슬라이드** | "저장하고 나중에 적용하세요 📌" + 프로필 링크 |

#### 캐러셀 10가지 포맷

| # | 포맷 | 예시 | 저장 유도 |
|---|------|------|----------|
| 1 | **체크리스트** | "AI 자동화 시작 전 확인할 것 7가지" | ★★★★★ |
| 2 | **Before/After** | "CS 자동화 전 vs 후 (수치 비교)" | ★★★★ |
| 3 | **비교표** | "Claude Code vs Make.com vs Zapier" | ★★★★★ |
| 4 | **스텝바이스텝** | "Claude Code 첫 워크플로우 5단계" | ★★★★★ |
| 5 | **미신 파괴** | "AI 자동화에 대한 5가지 오해" | ★★★★ |
| 6 | **도구 추천** | "비개발자 AI 도구 TOP 5" | ★★★★★ |
| 7 | **데이터/통계** | "한국 기업 AI 준비도 현황 (100명 분석)" | ★★★★ |
| 8 | **실패 사례** | "AI 도입 실패하는 3가지 패턴" | ★★★ |
| 9 | **ROI 계산** | "자동화 투자 vs 절감 비교표" | ★★★★ |
| 10 | **로드맵** | "비개발자 AI 자동화 30/60/90일 플랜" | ★★★★★ |

#### 캐러셀 슬라이드 구조 (표준 7장)

```
[Slide 1] 훅 — "아직도 CS를 사람이 하나요?" (호기심 유발)
[Slide 2] 문제 — "하루 4시간 CS에 쓰는 당신" (공감)
[Slide 3~5] 솔루션 — 3가지 자동화 방법 (가치)
[Slide 6] 결과 — "월 200시간 절감 실제 사례" (증거)
[Slide 7] CTA — "저장📌 + 자세한 가이드는 프로필 링크" (전환)
```

### 2.2 릴스 (발견/도달 최적화)

| 항목 | 설계 |
|------|------|
| **비중** | 전체 콘텐츠의 40% |
| **길이** | 15~60초 (완시청률 우선) |
| **소스** | 유튜브 쇼츠 리포맷 (비율 조정 + 워터마크 제거) |
| **훅** | 첫 1~3초에 텍스트 오버레이 ("이거 아세요?") |
| **자막** | 항상 ON (무음 시청 70%+) |

#### 릴스 콘텐츠 유형

| 유형 | 비중 | 예시 |
|------|:---:|------|
| **30초 팁** | 40% | "Claude Code으로 이메일 자동 분류 30초 세팅" |
| **Before/After** | 30% | "CS 자동화 전 vs 후 (화면 녹화)" |
| **역발상 훅** | 20% | "AI 도구 100개 써봤는데 비개발자에게 되는 건 3개" |
| **트렌드잭킹** | 10% | AI 뉴스/밈에 하우제로 관점 연결 |

### 2.3 스토리 (관계/참여 강화)

| 항목 | 설계 |
|------|------|
| **비중** | 매일 1~3개 |
| **유형** | 투표, 퀴즈, 질문 스티커, 비하인드, 릴스 공유 |
| **목적** | 팔로워와 DM 대화 유도 → 알고리즘 관계 신호 강화 |

#### 스토리 시리즈

| 요일 | 스토리 시리즈 |
|------|-------------|
| 월 | "이번 주 자동화 팁 미리보기" + 투표 ("이거 해봤어요?") |
| 수 | "릴스 비하인드" + 질문 스티커 ("자동화 고민 있으세요?") |
| 금 | "이번 주 베스트 DM 질문 답변" + 퀴즈 |

## 4. 인스타 → 전환 퍼널

```
[릴스] → 신규 팔로워 확보 (발견)
    ↓
[캐러셀] → 저장/공유 → 신뢰 구축 (교육)
    ↓
[스토리] → DM 대화 → 관계 심화 (참여)
    ↓
[프로필 링크] → 링크트리 or howzero.com
    ├── AI 준비도 진단 → 이메일 수집
    ├── 뉴스레터 구독
    ├── 무료 오딧 신청
    └── 코스/템플릿 구매
```

### 프로필 최적화

```
[인스타 프로필]

이름: 하우제로 | AI 자동화 컨설턴트
바이오:
  "코딩 없이 AI로 10억 벌어본 실전가
   비개발자를 위한 AI 자동화 전문
   ↓ 무료 AI 준비도 진단 받기"
링크: howzero.com/link (링크트리)
하이라이트:
  📊 AI 진단 | 🛠️ 자동화 사례 | 🎓 무료 가이드 | 💬 후기
```
