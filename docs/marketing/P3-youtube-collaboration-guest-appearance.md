## 핵심 컨셉

P2-파트너 기획서의 **유튜브 특화 버전**. 구독자 3K~50K 유튜버 채널에 게스트로 출연하여 해당 채널의 오디언스를 흡수.

## 콜라보 포맷 4종

### 1. 인터뷰 게스트 (가장 효과적)

```
파트너 채널에서:
  "[하우제로] 코딩 모르는데 AI로 10억 벌었다고?"
  → 30분 인터뷰 → 파트너 오디언스에 하우제로 깊은 노출
```

### 2. 공동 튜토리얼

```
파트너(노코드 강사) + 하우제로:
  "Claude Code으로 이커머스 CS 완전 자동화 (합동 튜토리얼)"
  → 양쪽 채널에 동시 게시 or 파트 분할
```

### 3. VS 대결/토론

```
"AI 자동화 vs 수동 관리: 어떤 게 이길까?"
  하우제로(자동화) vs 파트너(수동 옹호)
  → 토론형 = 높은 참여율 + 댓글 폭발
```

### 4. 채널 테이크오버

```
파트너 채널에 하우제로가 1편 단독 제작:
  "오늘은 특별 게스트 하우제로가 진행합니다"
  → 파트너 구독자에게 100% 노출
```

## 타겟 유튜버 15명

| 카테고리 | 구독 규모 | 콜라보 앵글 |
|---------|:---:|------|
| 노코드/자동화 채널 | 3K~20K | 공동 튜토리얼 |
| 1인 기업/부업 채널 | 5K~50K | 인터뷰 ("10억 비하인드") |
| 이커머스 강사 | 5K~30K | CS 자동화 데모 |
| 생산성 채널 | 5K~30K | AI 도구 소개 |
| 비즈니스/창업 채널 | 10K~50K | 인터뷰 + 스토리 |

## 섭외 프로세스

```
Step 1: 워밍업 — 댓글 10개 + 영상 공유 (2주)
Step 2: DM — "콜라보 하면 서로 좋을 것 같아요"
Step 3: 기획 미팅 (15분) — 주제/포맷/일정 합의
Step 4: 촬영 — 원격 or 대면
Step 5: 편집 → 양쪽 채널 게시
Step 6: 성과 공유 → 정기 콜라보 검토
```
