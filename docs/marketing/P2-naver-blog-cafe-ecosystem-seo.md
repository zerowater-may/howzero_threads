## 1. 전략 배경

### 왜 네이버인가?

- **한국인 70%+가 네이버를 1차 검색 엔진으로 사용** (InterAd, 2026)
- **네이버 블로그 = 한국에서 가장 강력한 오가닉 SEO 도구** (CliffsNotes/Oban)
- 네이버는 **자체 생태계(블로그/카페/지식iN/쇼핑)** 콘텐츠를 우선 노출 — 외부 사이트보다 네이버 블로그가 유리
- **"한국에서 SEO = 네이버 블로그 + 카페"** (iCrossBorder, 2026)
- 네이버 유저는 **진정성 있는 정보형 콘텐츠 + 스토리텔링**을 선호, 하드 셀링 기피 (TranslatedRight)

### howzero.com SEO와의 차이

| | howzero.com (구글 SEO) | 네이버 블로그 |
|---|---|---|
| 검색엔진 | 구글 + ChatGPT/Perplexity | 네이버 |
| 한국 도달률 | 30% | **70%+** |
| 인덱싱 속도 | 수일~수주 | **수시간~1일** |
| 신뢰도 | 외부 사이트 취급 | **네이버 자체 콘텐츠 우대** |
| AEO | 영문 AI 인용 | 네이버 AI 큐: 답변 인용 |

→ **두 채널 병행이 필수**: 구글/AI 검색은 howzero.com, 네이버 검색은 네이버 블로그

### 2.3 네이버 블로그 SEO 최적화 체크리스트

| 요소 | 최적화 방법 |
|------|-----------|
| **제목** | 핵심 키워드 + 수치 + 연도 ("Claude Code 사용법 완전 가이드 2026") |
| **본문 서두** | 첫 2~3문장에 핵심 키워드 자연 삽입 |
| **소제목** | H2/H3로 구조화 (네이버도 구조화 콘텐츠 선호) |
| **이미지** | 3~5장 포함 + ALT 텍스트에 키워드 |
| **분량** | 2,000~4,000자 (너무 짧으면 저품질, 너무 길면 이탈) |
| **태그** | 관련 태그 5~10개 |
| **링크** | 과도한 외부 링크 금지 (저품질 판정 위험). 내부 링크 활용 |
| **발행 빈도** | 주 2~3회 (하루 1포스트 초과 금지) |
| **서로이웃** | 50명+ 확보 → 초기 노출 부스트 |

## 4. 네이버 카페 전략

### 4.1 자체 카페 vs 기존 카페 활동

| 전략 | 시기 | 설명 |
|------|------|------|
| **기존 카페 활동** (Q2) | 즉시 | 관련 카페에서 가치 있는 글/댓글 → 인지도 구축 |
| **자체 카페 개설** (Q3~) | 7월~ | "하우제로 AI 자동화 커뮤니티" 카페 운영 |

### 4.2 타겟 기존 카페 (Reply Guy 네이버 버전)

| 카페 | 회원 수(추정) | 활동 전략 |
|------|:---:|------|
| 스마트스토어 셀러 카페 | 5만+ | CS/업무 자동화 팁 공유 |
| 1인 기업/프리랜서 카페 | 3만+ | AI 도구 추천 + 활용법 |
