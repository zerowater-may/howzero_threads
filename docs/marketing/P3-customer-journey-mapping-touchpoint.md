# P3-113: 고객 여정 맵핑 + 터치포인트 최적화 — 전환까지 모든 접점 설계
> P3 | ₩0 | 대표

## 고객 여정 맵 (Awareness → Advocacy)

```
[1. 인지] "AI 자동화? 뭔지 모르겠지만 관심"
  터치포인트: 숏폼/틱톡/인스타 릴스/검색
  감정: 호기심
  CTA: 팔로우/구독

[2. 탐색] "더 알아보자"
  터치포인트: 유튜브 롱폼/블로그/뉴스레터
  감정: "이거 우리 회사에도 되나?"
  CTA: AI 진단/뉴스레터 구독

[3. 고려] "해볼까?"
  터치포인트: AI 진단 결과/가격 페이지/케이스 스터디
  감정: "비용 대비 효과가 있을까?"
  CTA: 무료 오딧 신청

[4. 결정] "하자"
  터치포인트: 오딧 리포트/제안서/가격 비교
  감정: "이 사람 믿을 수 있나?"
  CTA: 패키지 선택/계약

[5. 사용] "써보는 중"
  터치포인트: 온보딩/킥오프/중간 체크인
  감정: "진짜 되네!"
  CTA: 없음 (만족 극대화)

[6. 옹호] "남에게 추천"
  터치포인트: NPS/후기/추천 프로그램
  감정: "이거 좋았으니까 공유해야지"
  CTA: 후기 작성/추천 링크 공유
```

## 각 단계별 병목 식별 + 최적화

| 단계 | 현재 이탈률(추정) | 병목 | 해결 기획서 |
|:---:|:---:|------|------|
| 1→2 | 95% | 숏폼 → 롱폼 전환 낮음 | P1-숏폼 시리즈 |
| 2→3 | 80% | "나에게 해당되나?" 확신 부족 | P2-니치 타겟팅 |
| 3→4 | 70% | 가격 부담/결정 지연 | P2-가격/전환+바우처 |
| 4→5 | 20% | 온보딩 미흡 | P2-온보딩 |
| 5→6 | 40% | 후기 요청 안 함 | P2-UGC/후기 |

## KPI: 각 단계 전환율 10%씩 개선 = 전체 퍼널 전환 5배 향상
