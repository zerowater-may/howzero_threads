## 1. 전략 배경

### 크로스 프로모션이란?

비슷한 타겟을 가진 크리에이터끼리 **서로의 콘텐츠/서비스를 추천**하는 것. 비용 0원으로 오디언스를 교환하는 가장 효율적인 성장 전략.

```
[하우제로] ← 오디언스 교환 → [노코드 강사 A]
  "A님의 Claude Code 강의 추천합니다"    "하우제로 AI 오딧 추천합니다"
  → A의 팔로워가 하우제로 유입    → 하우제로 팔로워가 A 유입
  = Win-Win (비용 ₩0)
```

### 핵심 조건: "미러 오디언스"

크로스 프로모션은 **오디언스가 겹치되 서비스가 겹치지 않는** 파트너와 해야 효과적.

| 좋은 파트너 (미러 오디언스) | 나쁜 파트너 |
|---|---|
| 노코드 강사 (교육 → 실전 연결) | 다른 AI 컨설턴트 (직접 경쟁) |
| 생산성 유튜버 (도구 → 자동화 연결) | 전혀 다른 업종 크리에이터 |
| 비즈니스 코치 (전략 → 실행 연결) | 같은 포지셔닝의 경쟁자 |
| 이커머스 강사 (셀러 → CS 자동화) | |

## 2. 크로스 프로모션 5가지 포맷

### 2.1 뉴스레터 상호 추천

```
[하우제로 위클리]
  "이번 주 추천 뉴스레터: [파트너명]
   비개발자를 위한 노코드 실전 팁을 매주 받아보세요.
   → [구독 링크]"

[파트너 뉴스레터]
  "이번 주 추천: 하우제로
   코딩 없이 AI 자동화 실전 가이드. 연매출 10억 달성 대표가 직접 씁니다.
   → [구독 링크]"
```

### 2.2 유튜브 엔드스크린 상호 추천

```
"이 영상이 도움 됐다면, [파트너명]의 채널도 추천합니다.
 [주제]를 가장 잘 설명하는 채널이에요."
→ 엔드스크린에 파트너 영상 카드
```

### 2.3 인스타/링크드인 공동 라이브

```
[공동 라이브 30~45분]
  하우제로: AI 자동화 실전
  파트너: 비즈니스 전략/생산성/노코드
  → 양쪽 팔로워에게 동시 노출
  → 라이브 후 클립 → 숏폼으로 재가공
```

### 2.4 콘텐츠 스왑 (게스트 포스트)

```
[하우제로 → 파트너 블로그/뉴스레터]
  "AI 자동화 초보자 가이드" 기고

[파트너 → 하우제로 뉴스레터]
  "[주제] 실전 팁" 기고

→ 각자의 오디언스에 상대방 전문성 노출
```

### 2.5 번들 프로모션

```
"하우제로 코스 + [파트너] 코스 번들 = 30% 할인"
→ 양쪽 고객에게 상대방 서비스 교차 노출
→ 번들 구매 시 양쪽 모두 매출 발생
```

## 3. 파트너 타겟 20명

| 카테고리 | 예시 (구독 3K~30K) | 교환 가치 |
|---------|-------------------|----------|
| **노코드/자동화** | Claude Code/Make 튜토리얼 유튜버 | "도구 사용법 → 실전 적용" |
| **생산성** | Notion/자동화 팁 크리에이터 | "생산성 → AI 자동화" |
| **1인 기업** | 프리랜서/사이드 프로젝트 유튜버 | "시간 확보 → 자동화" |
| **비즈니스 코치** | 경영/마케팅 코치 | "전략 → 실행 도구" |
| **이커머스** | 스마트스토어/쿠팡 셀러 강사 | "CS/소싱 → 자동화" |

## 4. 파트너 관계 구축 프로세스

```
[Step 1] 워밍업 (2주)
  → 파트너 콘텐츠에 가치 있는 댓글 5~10개
  → 스토리 공유/태그

[Step 2] DM 접근
  "안녕하세요! 콘텐츠 잘 보고 있습니다.
   저도 비슷한 타겟(비개발자)에게 AI 자동화를 알려주고 있는데,
   오디언스가 겹치면서도 서비스는 겹치지 않아서
   크로스 프로모션 하면 서로 좋을 것 같아요.

   예를 들어 뉴스레터 상호 추천이나 공동 라이브는 어떠세요?"

[Step 3] 소규모 테스트
  → 뉴스레터 1회 상호 추천 → 결과 공유

[Step 4] 정기 파트너십
  → 월 1회 상호 추천 + 분기 1회 공동 라이브
```

```
   파트너명 | 프로모 유형 | 날짜 | 유입 수 | 전환 수
→ 효과 좋은 파트너 → 확대
→ 효과 없는 파트너 → 종료
```
