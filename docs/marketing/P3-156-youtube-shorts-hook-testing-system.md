# P3-156: 숏폼 훅 체계적 테스팅 — 같은 내용, 다른 훅 3개로 A/B/C 테스트
> P3 | ₩0 | 대표
동일한 숏폼 내용을 훅(첫 3초)만 다르게 3개 버전으로 제작 → 동시 게시 → 가장 높은 완시청률 버전 식별. 예: 버전A "이거 아세요?" / 버전B "90%가 모릅니다" / 버전C "CS 인건비 0원으로 만든 방법". 인스타 Trial Reels(#25) 활용: 팔로워 아닌 사람에게 먼저 테스트 → 승자만 팔로워에게 노출. 주 1회 훅 테스트 → 월 4개 → 연 48개 → "가장 잘 되는 훅 패턴" 과학적 축적. KPI: 연 48개 훅 테스트, TOP 10 훅 패턴 DB 구축, 평균 완시청률 +30%.
