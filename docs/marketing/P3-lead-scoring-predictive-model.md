## Claude Code 구현

```
매일 09:00:
  → Notion CRM 전체 리드 스캔
  → 각 리드의 행동 데이터 수집 (스티비 오픈, GA4 방문, Typeform 등)
  → 점수 자동 계산 → Notion DB 업데이트
  → 점수 변화 감지:
    새로 80+ 진입 → Slack: "🔴 HOT 리드! [이름] [업종] [점수]"
    50+ 진입 → 자동 오딧 CTA 이메일 발송
```

## 고급: Claude API 예측 모델 (Q4~)

```
축적된 전환 데이터 100건+ 후:
  → Claude API에 "이 프로필의 리드가 전환할 확률?"
  → 입력: 업종, 규모, 점수, 행동 패턴, 유입 채널
  → 출력: 전환 확률 % + 추천 액션
  → 시간이 지날수록 정확도 향상
```
