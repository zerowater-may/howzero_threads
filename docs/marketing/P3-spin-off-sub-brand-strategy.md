# P3: 서브 브랜드/스핀오프 전략 — 업종별 특화 브랜드 분리

> **우선순위**: P3 | **예산**: 초기 50만원/브랜드 | **담당**: 대표 | **시기**: 2027~

---

## 핵심: 하우제로가 성장하면 업종별 서브 브랜드를 분리하여 니치 지배력 강화

## 서브 브랜드 후보

| 서브 브랜드 | 타겟 | 런칭 조건 |
|-----------|------|----------|
| **하우제로 커머스** | 이커머스 셀러 AI 자동화 | 이커머스 고객 10건+ |
| **하우제로 리걸** | 법률사무소 AI 자동화 | 법률 고객 5건+ |
| **하우제로 메디** | 병원/의원 AI 자동화 | 병원 고객 5건+ |
| **하우제로 에듀** | 교육/코칭 AI 자동화 | 교육 고객 5건+ |
| **하우제로 글로벌** | 영문 글로벌 서비스 | 영문 매출 20%+ |

## 서브 브랜드의 장점

```
[하우제로] "AI 자동화 전문" — 범용 브랜드
  ↓ 서브 브랜드 분리
[하우제로 커머스] "이커머스 AI 자동화 전문" — 니치 1등
  → "이커머스 AI 자동화"로 검색하면 → 하우제로 커머스가 1등
  → 범용 하우제로보다 전환율 3~5배 높음 (니치 효과)
```

## 분리 기준

```
서브 브랜드 분리 트리거:
  □ 해당 업종 고객 10건+ 축적
  □ 해당 업종 전용 케이스 스터디 5개+
  □ 해당 업종 키워드 SEO 1페이지 달성
  □ 해당 업종 매출이 전체의 20%+ 차지

→ 4개 중 3개 충족 → 서브 브랜드 분리 검토
```
