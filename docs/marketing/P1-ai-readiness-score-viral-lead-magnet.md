## 1. 전략 배경 & 리서치 인사이트

### Spotify Wrapped가 증명한 것

Spotify Wrapped는 매년 **사용자 데이터를 개인화된 스토리로 변환**하여 자발적 공유를 유도하는 바이럴 마케팅의 교과서다.

| Wrapped 핵심 원리 | 하우제로 적용 |
|------------------|-------------|
| **개인화된 데이터 → 감정적 경험** | 기업/개인의 업무 프로세스 → "당신의 AI 준비도 점수" |
| **게이미피케이션 (상위 1% 리스너)** | "상위 15% AI 활용 기업입니다" / "AI 도입 시급 단계" |
| **소셜 공유 욕구 자극** | 결과 카드를 링크드인/인스타에 공유하고 싶게 디자인 |
| **CTA 없는 듯한 CTA** | 점수 자체가 가치 → 자연스럽게 "더 알고 싶다면?" 전환 |
| **연례 이벤트화** | 분기별 "AI 준비도 리포트" 업데이트로 재방문 유도 |

### 2026년 리드 마그넷 트렌드

- **Generic PDF는 죽었다**: "범용 가이드"는 더 이상 전환되지 않음. 2026년은 **속도, 명확성, 개인화**가 핵심 (USA Financial, 2026)
- **인터랙티브 자가 진단이 최고 전환율**: 개인 스코어카드/체크리스트 형태의 자가 평가 도구가 가장 높은 리드 전환율 기록 (USA Financial)
- **LinkedIn Lead Gen Form 전환율 13%**: 일반 랜딩페이지(4.02%) 대비 3배 이상 (Thunderbit, 2026)
- **73% B2B 의사결정자가 링크드인 Thought Leadership 신뢰** (Thunderbit, 2026)
- **Forbes: AI Readiness는 5개 차원으로 평가해야** — 단일 점수가 아닌 다차원 진단이 설득력 있음 (Forbes Tech Council, 2026.03)

### 왜 "AI 준비도 점수"인가?

하우제로의 핵심 서비스는 **AI 오딧(Audit)**이다. 그런데 무료 오딧 30분은 **스케일이 안 된다** (대표가 직접 해야 하므로). "AI 준비도 점수"는 이 오딧의 **셀프서비스 버전**으로:

```
[비개발자 고객의 심리 여정]

"AI 도입해야 하는 건 알겠는데, 뭐부터 해야 하지?"
    → "내 회사/내 업무가 AI 도입 준비가 되어 있나?"
    → "AI 준비도 점수"로 자가 진단 (3분)
    → 점수 + 맞춤 리포트 수령 (이메일 수집)
    → "점수가 낮네... 어떻게 올리지?"
    → 하우제로 콘텐츠/오딧/컨설팅으로 자연 전환
```

## 2. "AI 준비도 점수" 제품 설계

### 2.1 자가진단 질문 설계 (10문항, 3분 소요)

Forbes의 5차원 AI Readiness 프레임워크를 비개발자 눈높이로 번역:

| 차원 | 질문 (비개발자 언어) | 점수 |
|------|-------------------|------|
| **1. 프로세스** | "반복적으로 하는 업무 중 '이건 자동화할 수 있을 텐데'라고 느끼는 게 있나요?" | 0~10 |
| **2. 프로세스** | "업무 프로세스가 문서화(매뉴얼/SOP)되어 있나요?" | 0~10 |
| **3. 데이터** | "고객 데이터를 엑셀이 아닌 전용 시스템(CRM 등)으로 관리하나요?" | 0~10 |
| **4. 데이터** | "지난 1년간의 고객 문의/CS 내역을 바로 꺼낼 수 있나요?" | 0~10 |
| **5. 도구** | "현재 사용 중인 SaaS/도구가 서로 연동되어 있나요? (예: 이메일→CRM 자동)" | 0~10 |
| **6. 도구** | "팀에서 ChatGPT나 AI 도구를 업무에 활용하고 있나요?" | 0~10 |
| **7. 인력** | "개발자 또는 IT 담당자가 사내에 있나요?" | 0~10 |
| **8. 인력** | "새로운 도구나 시스템을 도입할 때 의사결정까지 얼마나 걸리나요?" | 0~10 |
| **9. 예산** | "AI/자동화 도입에 월 얼마까지 투자할 의향이 있나요?" | 0~10 |
| **10. 비전** | "6개월 후 AI가 해결해줬으면 하는 가장 큰 고민 1가지는?" | 주관식 |

## 3. 바이럴 퍼널 설계

### 3.1 유입 → 진단 → 전환 퍼널

```
[유입 경로]
├── 숏폼 영상 CTA: "당신의 AI 준비도는 몇 점? → 프로필 링크"
├── 링크드인 포스트: "우리 팀 AI 준비도 측정해봤는데 충격..."
├── 이메일 뉴스레터: "3분 만에 AI 도입 가능 여부 알 수 있습니다"
└── 기존 콘텐츠 CTA 통합
        ↓
[자가진단 페이지] howzero.com/ai-score
    10문항, 3분 소요, 모바일 최적화
        ↓
[이메일 입력] "맞춤 리포트를 보내드립니다"
    → 이메일 리스트 확보 (핵심 전환 포인트)
        ↓
[즉시 결과 화면]
├── 점수 + 등급 + 5차원 분석
├── 결과 공유 버튼 (링크드인/인스타/카카오톡)
└── "더 자세한 진단이 필요하면 → 무료 AI 오딧 30분 예약"
        ↓
[이메일 너처링 시퀀스 (5일)]
├── Day 0: 리포트 전문 + 등급별 맞춤 가이드
├── Day 1: "당신과 비슷한 점수의 기업이 AI로 이렇게 바뀌었습니다" (사례)
├── Day 3: "점수를 올리는 가장 쉬운 방법 3가지" (Low-hanging fruit)
├── Day 5: "무료 AI 오딧 30분, 지금 예약하세요" (전환 CTA)
└── Day 7: "혼자 해보고 싶다면 → Claude Code 템플릿 무료 다운로드"
        ↓
[전환]
├── 무료 AI 오딧 30분 → 유료 컨설팅/구축
├── 온라인 코스 구매
└── 프라이빗 커뮤니티 가입
```

### 3.2 바이럴 루프 설계 (Spotify Wrapped 벤치마킹)

```
[사용자 A가 결과 카드를 링크드인에 공유]
    → "AI 준비도 47점... 생각보다 낮네요 😅"
    → [사용자 B가 보고] "나도 해봐야겠다"
    → howzero.com/ai-score 방문
    → 진단 완료 → 결과 공유
    → [사용자 C, D, E...] 바이럴 루프 반복
```

**바이럴 촉진 장치**:
- "당신의 업종 평균은 52점입니다. 당신은?" → 비교 욕구
- "이 결과를 공유하고 동료의 점수와 비교해보세요" → 소셜 프루프
- 분기별 "AI 준비도 트렌드 리포트" 발행 → 재진단 유도

## 5. 기술 구현 스펙

### 5.1 MVP (4주 개발)

| 컴포넌트 | 기술 스택 | 비용 |
|---------|----------|------|
| 랜딩페이지 + 진단 폼 | Tally / Typeform + 커스텀 CSS | 무료~3만원/월 |
| 점수 계산 + 결과 페이지 | Next.js or 정적 페이지 | Vercel 무료 |
| 결과 카드 이미지 생성 | html2canvas / Satori | 무료 |
| 이메일 수집 + 시퀀스 | 스티비 or ConvertKit | 3만원/월 |
| 자동화 연동 | Claude Code (폼 제출 → 이메일 → CRM) | 무료 (셀프호스팅) |
| 분석 | GA4 + Mixpanel (퍼널 추적) | 무료 |

### 5.2 데이터 활용 (2차 가치)

진단 데이터는 하우제로의 **콘텐츠 + 세일즈 자산**이 된다:

- **콘텐츠**: "1,000명의 AI 준비도를 분석한 결과..." → 데이터 기반 콘텐츠 (Lenny Rachitsky式)
- **세일즈**: 점수가 낮은 응답자를 자동 세그먼팅 → 맞춤 오딧 제안
- **시장 인사이트**: 업종별/규모별 AI 준비도 트렌드 → PR 보도자료 + 권위 구축
- **Q10 주관식 답변**: "AI가 해결해줬으면 하는 고민" → 다음 콘텐츠/서비스 주제 발굴
