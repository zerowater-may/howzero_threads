# P3: AI 도구 어필리에이트 수익 전략 — 추천하는 도구에서 수수료 수익

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: 월 0원 | **담당**: 대표

---

## 핵심 컨셉

하우제로가 콘텐츠/오딧/코스에서 추천하는 AI 도구(Claude Code, Make.com, Zapier, Lovable, Chatbase 등)에 **어필리에이트 링크**를 달아 추가 수익을 만든다. 추천 자체는 이미 하고 있으므로 **추가 노력 0, 순수 부가 수익**.

## 어필리에이트 대상 도구

| 도구 | 예상 커미션 | 적용 터치포인트 |
|------|-----------|-------------|
| Claude Code Cloud | 20~30% 리커링 | 모든 튜토리얼/코스/템플릿 |
| Make.com | 20% 리커링 | 비교 콘텐츠/코스 |
| Zapier | $5~10/건 | 비교 콘텐츠 |
| Lovable | 20% | 앱 빌더 리뷰 |
| Chatbase | 20% | 챗봇 콘텐츠 |
| Cal.com | 10~20% | 예약 관련 콘텐츠 |
| Notion | 4% | 모든 생산성 콘텐츠 |

## 적용 위치

- 유튜브 설명란 "이 영상에서 사용한 도구" 섹션
- 블로그 도구 비교표 내 링크
- 뉴스레터 "이번 주 추천 도구"
- 코스 내 도구 가이드
- 템플릿 설치 가이드

## 수익 시뮬레이션

```
6개월 후 월간 트래픽 20,000명 기준:
  어필리에이트 클릭율: 2% = 400클릭
  전환율: 5% = 20건
  평균 커미션: ₩15,000/건 (리커링 월 평균)
  월 수익: ₩300,000
  리커링 누적: 6개월 후 ₩100만/월+
```

## 원칙

1. **직접 써보고 진심으로 추천하는 도구만** (진정성)
2. "이건 어필리에이트 링크입니다" 투명 공개
3. 어필리에이트 수익이 콘텐츠 방향을 왜곡하면 안 됨
4. 월 1회 수익 리포트 → 어떤 도구가 가장 전환되는지 분석

## KPI

| 지표 | 3개월 | 6개월 |
|------|------|------|
| 등록 어필리에이트 프로그램 | 5개 | 8개 |
| 월 클릭 | 100 | 400 |
| 월 전환 | 5건 | 20건 |
| **월 어필리에이트 수익** | ₩75,000 | ₩300,000+ |
