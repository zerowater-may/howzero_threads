# P3: Pinterest + 인포그래픽 비주얼 SEO — 이미지 검색에서 리드 확보

> **작성일**: 2026-03-29 | **우선순위**: P3 | **예산**: 월 5만원 | **담당**: 대표

---

## 핵심 컨셉

Pinterest = **비주얼 검색 엔진**. "AI 자동화", "업무 자동화 도구", "Claude Code" 같은 키워드의 인포그래픽/비교표를 Pinterest에 올리면 **Google 이미지 + Pinterest 검색** 동시 노출.

## 콘텐츠 유형

| 유형 | 예시 | Pinterest 적합도 |
|------|------|:---:|
| **인포그래픽** | "AI 자동화 5단계 로드맵" | ★★★★★ |
| **비교표** | "Claude Code vs Make vs Zapier" | ★★★★★ |
| **체크리스트** | "AI 도입 전 체크리스트 10가지" | ★★★★ |
| **Before/After** | "CS 자동화 Before vs After" | ★★★★ |
| **도구 맵** | "비개발자 AI 도구 추천 맵" | ★★★★★ |
| **통계/데이터** | "2026 AI 준비도 현황" | ★★★ |

## 워크플로우

```
기존 캐러셀/블로그 제작 시:
  → Canva에서 세로형(1000×1500px) 버전 추가 제작 (5분)
  → Pinterest에 업로드 (제목/설명에 키워드)
  → 링크: 해당 블로그 포스트 or howzero.com

→ 추가 시간: 주 15분
→ Pinterest Pin = Google 이미지에도 노출 = 이중 SEO
```

## Pinterest SEO

| 요소 | 최적화 |
|------|--------|
| Pin 제목 | 키워드 포함 ("비개발자 AI 자동화 5단계") |
| Pin 설명 | 200자+ 키워드 풍부한 설명 |
| 보드 이름 | "AI 자동화 가이드" "비개발자 AI 도구" |
| ALT 텍스트 | 키워드 포함 |
| 링크 | howzero.com 해당 페이지 |
