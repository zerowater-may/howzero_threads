# P3-147: 에버그린 콘텐츠 리사이클링 — 인기 콘텐츠를 주기적으로 재게시
> P3 | ₩0 | Claude Code
3개월 이상 된 인기 콘텐츠를 업데이트+재게시. 블로그: 제목에 "(2026 업데이트)" 추가 + 최신 데이터 반영 → 재발행 → SEO 신선도 유지. SNS: 인기 숏폼을 3개월 후 다시 게시 (새 팔로워는 본 적 없음). 뉴스레터: "놓치셨을 수도 있는 인기 글 TOP 3" 코너. Claude Code: 3개월 전 콘텐츠 중 조회수 TOP 10 자동 추출 → "리사이클 후보" 리스트 → 대표 선택 → 업데이트+재게시. 70% 에버그린 + 30% 시즌 = 콘텐츠 생산 부담 -30%. KPI: 리사이클 콘텐츠 원본 대비 70%+ 성과 유지.
