## 1. 전략 배경

### 왜 "시리즈"인가?

> "바이럴 1개는 운, 시리즈는 비즈니스" (12AM Agency, 2026)

단발 숏폼은 조회수가 나와도 **구독으로 이어지지 않는다**. 시리즈는:
- 알고리즘이 "다음 편" 추천 → 연속 시청
- 시청자가 "나머지도 봐야지" → 채널 방문 → 구독
- 제작자가 "이번 주 주제 뭐하지?" 고민 0 → 시리즈 다음 편 촬영

## 2. 10가지 쇼츠/틱톡 시리즈 포맷

### Series 1: "30초 AI 도구 해설" (매일 1개)

```
컨셉: 매일 1개 AI 도구를 30초로 해설
포맷: 화면 녹화 + 보이스오버
예시: "오늘의 AI 도구: Claude Code. 30초 안에 설명합니다"
CTA: "어떤 도구 해설 원해요? 댓글로!"
시리즈 태그: #AI도구해설 #Day{번호}
목표: 100편 (3개월+)
```

### Series 2: "비개발자 자동화 레시피" (주 2회)

```
컨셉: 특정 자동화를 30~60초로 시연
포맷: 화면 녹화 (Claude Code 실시간 세팅)
예시: "이메일 자동 분류 레시피 — 따라하세요"
CTA: "레시피 템플릿 → 프로필 링크"
시리즈 태그: #자동화레시피 #EP{번호}
```

### Series 3: "이거 아직도 수동으로 해요?" (주 2회)

```
컨셉: 비효율적인 수동 업무를 역발상으로 지적
포맷: 카메라 직접 말하기 (UGC 스타일)
예시: "아직도 엑셀로 보고서 만들어요? AI가 10초면 합니다"
CTA: "자동화 방법 → 다음 영상에서"
시리즈 태그: #아직도수동
```

### Series 4: "AI 오딧 현장" (주 1회)

```
컨셉: 실제 오딧 과정 일부를 (익명으로) 공개
포맷: 화면 녹화 + 보이스오버 (고객 동의)
예시: "이 회사 CS 프로세스 진단 결과... 충격"
CTA: "무료 오딧 → 프로필 링크"
시리즈 태그: #AI오딧현장 #EP{번호}
```

### Series 5: "Before vs After" (주 1회)

```
컨셉: 자동화 전후 비교를 시각적으로
포맷: 화면 분할 (좌: Before, 우: After)
예시: "CS 처리 시간 Before: 4시간 → After: 30분"
CTA: "어떻게? 댓글에 '방법' 쓰세요"
시리즈 태그: #BeforeAfter자동화
```

### Series 6: "오늘의 AI 뉴스 비개발자 해설" (주 2~3회)

```
컨셉: 오늘 나온 AI 뉴스를 비개발자 눈높이로 30초 해설
포맷: 카메라 + 뉴스 캡처 화면
예시: "GPT-5 나왔다는데, 비개발자에게 바뀌는 건? 0입니다"
CTA: "매주 AI 뉴스 해설 → 뉴스레터 구독"
시리즈 태그: #AI뉴스해설
```

### Series 7: "10억 SaaS 비하인드" (주 1회)

```
컨셉: GPT-3로 10억 달성 과정을 에피소드로 풀기
포맷: 카메라 직접 말하기 (스토리텔링)
예시: "EP3. 첫 고객이 왔는데 버그가 터졌다"
CTA: "다음 편 보려면 → 팔로우"
시리즈 태그: #10억SaaS비하인드 #EP{번호}
```

### Series 8: "1분 챌린지" (주 1회)

```
컨셉: 1분 안에 특정 자동화를 만드는 타임어택
포맷: 타이머 + 화면 녹화 (배속)
예시: "1분 안에 Slack 알림 봇 만들기 — 시작!"
CTA: "성공할까? 끝까지 보세요"
시리즈 태그: #1분자동화챌린지
```

### Series 9: "Q&A 숏앤서" (주 1~2회)

```
컨셉: 댓글/DM 질문을 숏폼으로 답변
포맷: 질문 텍스트 오버레이 + 카메라 답변
예시: "Q. 코딩 모르는데 Claude Code 쓸 수 있나요? A. 네. 이유는..."
CTA: "질문 있으면 댓글에!"
시리즈 태그: #하우제로QA
```

### Series 10: "주간 자동화 성과" (주 1회)

```
컨셉: 이번 주 하우제로의 자동화 성과 수치 공개 (Build in Public)
포맷: 카메라 + 수치 오버레이
예시: "이번 주 성과: 오딧 5건, 매출 ₩800만, 실패 1건"
CTA: "매주 월요일 업데이트 → 팔로우"
시리즈 태그: #하우제로주간리포트
```

## 3. 시리즈 + 알고리즘 최적화

### 3.1 시리즈 해시태그 전략

```
[고정 태그] #하우제로 #AI자동화 #비개발자
[시리즈 태그] #AI도구해설 #Day{번호} or #EP{번호}
[검색 태그] #Claude Code #AI자동화도구 #업무자동화 #노코드

→ 시리즈 태그 = 채널 내 "플레이리스트" 역할
→ 시청자가 시리즈 태그 클릭 → 연관 영상 연속 시청
```

### 3.2 시리즈별 게시 스케줄 (주간)

| 요일 | 시리즈 | 수 |
|------|--------|:---:|
| **월** | #주간성과 + #AI뉴스해설 | 2 |
| **화** | #AI도구해설 + #자동화레시피 | 2 |
| **수** | #아직도수동 + #AI뉴스해설 | 2 |
| **목** | #AI도구해설 + #BeforeAfter | 2 |
| **금** | #AI오딧현장 + #QA숏앤서 | 2 |
| **토** | #10억비하인드 + #1분챌린지 | 2 |
| **일** | #AI도구해설 (또는 휴식) | 0~1 |
| **합계** | | **11~12/주** |

### 3.3 첫 3초 훅 패턴 라이브러리

| 패턴 | 예시 | 사용 시리즈 |
|------|------|-----------|
| **충격 수치** | "CS 인건비 월 2,000만원을 0원으로 만들었습니다" | #BeforeAfter |
| **질문** | "아직도 엑셀로 보고서 만들어요?" | #아직도수동 |
| **도발** | "GPT-5 나왔다고? 비개발자에게 바뀌는 건 0" | #AI뉴스해설 |
| **약속** | "30초 안에 이메일 자동 분류를 만들어 보여드리겠습니다" | #자동화레시피 |
| **비밀** | "99%가 모르는 Claude Code 숨은 기능" | #AI도구해설 |
| **타임어택** | "1분 안에 Slack 봇 만들기 — 시작!" | #1분챌린지 |
| **스토리** | "첫 고객이 왔는데 버그가 터졌습니다" | #10억비하인드 |

## 4. 시리즈 → 구독/전환 연결

```
[시리즈 시청]
    → "이 시리즈 재밌다 / 도움된다"
    → "다음 편도 봐야지" → 팔로우/구독
    ↓
[시리즈 3~5편 시청 후]
    → "이 사람 진짜 실력자네"
    → 프로필 방문 → 링크 클릭
    ↓
[전환]
    → 뉴스레터 구독 / AI 진단 / 템플릿 / 오딧
```
