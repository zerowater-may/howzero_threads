> **작성일**: 2026-03-29 | **우선순위**: P1 (최우선)
> **기간**: 2026 Q2 (4월~6월, 12주)
> **예산**: 월 60만원 (전체 마케팅 예산의 30%)
> **담당**: 하우제로 대표 (기획/출연) + AI 콘텐츠 엔진 (편집/재가공)

## 1. 전략 배경 & 리서치 인사이트

### 왜 숏폼인가?

- **60초 미만 숏폼 영상이 2026년 전 플랫폼에서 가장 높은 전환율을 기록** (Sprout Social, 2026)
- MrBeast는 2025년 한 해에만 **1.17억 구독자 증가** — 숏폼 전략이 핵심 동력 (netinfluencer, 2026.02)
- KIMPRO는 숏폼 전략으로 유튜브 #1 달성 — 롱폼 1개에서 **15~20개 숏폼 클립** 자동 생성 (OpusClip)
- TikTok SEO가 구글과 동급의 검색 플랫폼으로 부상 — "비개발자 AI 툴" 키워드 선점 기회 (ALM Corp, 2026)

### 왜 "비개발자를 위한 AI 툴"인가?

- 2026년 Entrepreneur 보도: **"코드 없이 1인 비즈니스를 운영하는 7가지 AI 툴"** — 비개발자 AI 수요 폭증
- 노코드 AI 앱 빌더 시장 급성장 — Lovable이 **1년 만에 ARR $2억** 달성 (Lenny Rachitsky x Elena Verna)
- 문제: 비개발자들은 AI 툴이 있다는 건 알지만 **어떤 걸 써야 하고, 어떻게 세팅하는지 모름**
- 기회: 하우제로가 이 **지식 격차(Knowledge Gap)**를 숏폼으로 해소 → 컨설팅 전환

## 2. 핵심 컨셉: "AI 툴 해설자 하우제로"

### 포지셔닝

```
"개발자 없이는 못 쓰는 AI 툴들,
 하우제로가 비개발자도 쓸 수 있게 풀어드립니다.
 그래도 안 되면? 우리가 대신 구축해드립니다."
```

### 콘텐츠 → 컨설팅 전환 로직

```
[숏폼: AI 툴 30초 해설] → "이거 써보고 싶은데 세팅이 어려우세요?"
    → [롱폼: 실전 튜토리얼] → "직접 해보셨는데 막히셨나요?"
    → [무료 AI 오딧 30분] → "당신의 프로세스에 맞는 최적의 툴 조합을 찾아드립니다"
    → [유료 컨설팅/구축] → "저희가 세팅부터 운영까지 다 해드립니다"
```

## 3. MrBeast 벤치마킹: 훅(Hook) 프레임워크

### MrBeast 핵심 원칙 (Leaked Document 기반)

| 원칙 | 하우제로 적용 |
|------|-------------|
| **첫 10초에 시청자를 잡아라** | 툴 데모 화면 + 충격적 수치로 시작 |
| **CTR(클릭율)이 왕이다** | 썸네일에 "Before/After" 비교 + 호기심 제목 |
| **리텐션(시청 유지율) 최적화** | 30초 안에 문제→해결→결과 완결 구조 |
| **바이럴 패턴을 스프레드시트로 분석** | 경쟁 채널 인기 숏폼 훅 패턴 DB화 |

### Hook → Build → Payoff 구조 (30초 숏폼)

```
[0~5초] Hook: "이 AI 툴 하나로 CS 직원 5명이 필요 없어졌습니다"
[5~20초] Build: 실제 화면 시연 — 설정 과정 30초 요약
[20~30초] Payoff: 결과 수치 + CTA "세팅 어려우면 프로필 링크"
```

## 5. 제작 워크플로우: 원 소스 멀티 유즈

### 주간 1일 콘텐츠 배칭 프로세스

```
[월요일] 기획
    AI 엔진이 주간 5개 숏폼 + 1개 롱폼 대본 초안 생성
    훅 3가지 버전 A/B 테스트용 준비

[화요일] 촬영
    롱폼 1개 촬영 (20~30분)
    숏폼 5개 촬영 (각 30~60초)
    총 촬영 시간: 2~3시간

[수요일] 편집 & 업로드
    AI 편집 툴(OpusClip, CapCut Pro)로 자동 편집
    롱폼 → 추가 숏폼 5~10개 자동 클리핑
    자막/캡션 자동 생성

[목~일] 자동 배포
    예약 게시로 자동 배포
    댓글 모니터링 + 'Reply Guy' 활동
    퍼포먼스 데이터 수집
```

### 크로스 플랫폼 변환 매트릭스

| 원본 | → 유튜브 쇼츠 | → 틱톡 | → 릴스 | → 링크드인 | → 이메일 |
|------|:---:|:---:|:---:|:---:|:---:|
| 롱폼 영상 | 클립 5~10개 | 리포맷 | 리포맷 | 핵심 인사이트 텍스트 | 주간 뉴스레터 |
| 숏폼 영상 | 원본 | 워터마크 교체 | 비율 조정 | 대본 → 포스트 | CTA 링크 |
