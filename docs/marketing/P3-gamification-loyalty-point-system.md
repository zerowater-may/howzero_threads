## 기술 구현

```
[Claude Code 포인트 추적]

각 활동 발생 시:
  → Claude Code Webhook 수신
  → Notion "포인트 DB" 업데이트 (사용자별 누적)
  → 레벨업 감지 → 축하 이메일 자동 발송
  → 배지 달성 → 배지 이미지 + 공유 링크 이메일
  → 월말: 월간 포인트 리더보드 → 뉴스레터 공개
```
