## 1. 전략 배경

### 컨설팅의 부산물 = 판매 가능한 제품

하우제로가 컨설팅/구축 프로젝트를 할 때마다 **Claude Code 워크플로우, AI 에이전트 설정, 프로세스 템플릿**이 만들어진다. 이것들은 1회성 납품물이 아니라 **무한 복제 가능한 디지털 제품**이다.

```
[현재: 1:1 납품]
  클라이언트 A → CS 자동화 구축 → 200만원 (1회)
  클라이언트 B → CS 자동화 구축 → 200만원 (1회)
  클라이언트 C → CS 자동화 구축 → 200만원 (1회)
  = 같은 유형의 작업을 3번 반복

[제품화 후: 1:∞]
  CS 자동화 템플릿 팩 → Gumroad에 등록 → 29,000원
  → 100명 구매 = 290만원 (패시브, 반복 작업 0)
  → 템플릿으로 안 되는 커스텀 → 컨설팅 업셀
```

### 핵심 데이터

- **Claude Code 템플릿 판매로 월 $3,200 패시브 인컴 사례** (Medium, 2026.03)
- **Gumroad 수익 $2,539 달성 솔로 개발자의 전략 공개** (Medium, 2026.03) — 핵심: 니치 타겟 + SEO + 뉴스레터 연동
- **Claude Code은 2,500+ 커뮤니티 템플릿 보유** — 크리에이터 마켓플레이스 성장 중 (GeekFleet)

- **Gumroad: 월정액 없음, 판매 시 수수료만** — 리스크 0으로 시작 가능 (CartMango)

## 2. 제품 라인업: 3-Layer 스토어

### Layer 1: 무료 템플릿 (리드 마그넷)

| 템플릿 | 설명 | 목적 |
|--------|------|------|
| CS 문의 자동 분류 (Claude Code) | 이메일/카톡 문의 → AI가 카테고리 분류 → Notion에 기록 | 이메일 수집 |
| ChatGPT API 기본 연결 (Claude Code) | API 키 입력 → 첫 AI 호출 성공 | "첫 성공" 경험 → 유료 업셀 |
| 자동 보고서 생성 (Claude Code) | Google Sheets 데이터 → AI 요약 → 이메일 발송 | 가치 입증 |
| 프로세스 맵핑 워크시트 (Notion) | 업무 프로세스 시각화 + 자동화 포인트 식별 | 오딧 준비 |
| AI 도구 비교표 (PDF) | 업무별/예산별 AI 도구 추천 맵 | SEO 리드 |

**총 5개 무료 템플릿 = 이메일 수집 엔진**

### Layer 2: 유료 템플릿 팩 (₩19,000~99,000)

| 팩 이름 | 포함 템플릿 수 | 가격 | 타겟 |
|---------|:---:|------|------|
| **CS 자동화 팩** | 5개 | ₩29,000 | 이커머스/서비스업 |
| **마케팅 자동화 팩** | 5개 | ₩39,000 | 1인 기업/마케터 |
| **업무 자동화 팩** | 5개 | ₩29,000 | 사무직/관리자 |
| **AI 에이전트 스타터 팩** | 3개 | ₩49,000 | AI 입문자 |
| **CRM 자동화 팩** | 7개 | ₩49,000 | 중소기업 대표 |
| **올인원 번들** | 25개 전부 | ₩99,000 | 진지한 자동화러 |

#### CS 자동화 팩 상세 (예시)

```
[CS 자동화 팩 — ₩29,000]

1. 고객 문의 자동 분류 & 라우팅
   트리거: 이메일/카톡 수신
   → AI가 문의 유형 판별 (환불/배송/제품/기타)
   → 유형별 담당자에게 자동 할당
   → Notion DB에 티켓 생성

2. 환불 요청 자동 처리
   트리거: "환불" 키워드 감지
   → 주문 정보 자동 조회
   → 환불 조건 충족 여부 AI 판단
   → 승인 시 자동 처리 + 고객 알림

3. 리뷰 자동 수집 & 감성 분석
   트리거: 주문 완료 D+7
   → 리뷰 요청 자동 발송
   → 수집된 리뷰 감성 분석 (긍정/부정/중립)
   → 부정 리뷰 즉시 알림

4. FAQ 자동 응답 봇
   트리거: 자주 묻는 질문 감지
   → AI가 FAQ DB에서 답변 생성
   → 확신도 80%+ → 자동 발송
   → 확신도 < 80% → 담당자 에스컬레이션

5. CS 주간 리포트 자동 생성
   트리거: 매주 월요일 09:00
   → 지난 주 문의 건수/유형/처리시간 집계
   → AI가 인사이트 요약
   → 이메일/Slack으로 자동 발송

포함: 영상 설치 가이드 (15분) + Notion 연동 매뉴얼
```

### 5.1 "Build Once, Sell Forever" 워크플로우

```
[컨설팅 프로젝트 완료]
    ↓
[체크리스트] "이 워크플로우를 제품화할 수 있나?"
    □ 다른 기업에서도 동일 니즈가 있는가? → YES
    □ 커스텀 부분을 변수화할 수 있는가? → YES
    □ 비개발자가 설치할 수 있는 수준인가? → YES/개선 필요
    ↓
[제품화 작업] (2~3시간)
    1. 워크플로우에서 고객 특화 데이터 제거
```
