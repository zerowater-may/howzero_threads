## 핵심 컨셉

한국은 카카오톡, 글로벌은 **WhatsApp/Telegram**. 영문 콘텐츠(P3-글로벌) 확장 시 글로벌 커뮤니티가 필요. Telegram은 특히 AI/테크 커뮤니티에서 가장 활발.

## 플랫폼 선택

| | WhatsApp | Telegram |
|---|---|---|
| 글로벌 사용자 | 20억+ | 9억+ |
| AI/테크 커뮤니티 | 중간 | **매우 활발** |
| 그룹 기능 | 1,024명 한정 | 무제한 |
| 봇 지원 | 제한적 | **강력** (자동화 연동) |
| 채널 기능 | 없음 | **1:N 브로드캐스트** |
| **추천** | 보조 | **메인** |

## Telegram 채널 + 그룹 설계

```
[Telegram 채널: @HowZeroAI] (1:N 브로드캐스트)
  → 영문 AI 자동화 팁/뉴스 (주 3~5회)
  → 뉴스레터 핵심 요약 + 링크
  → 구독자가 보기만 하는 채널

[Telegram 그룹: HowZero Community] (N:N 대화)
  → 영문 Q&A + 사례 공유 + 네트워킹
  → 채널 구독자 중 적극 참여자가 가입
```

## 콘텐츠 운영

| 요일 | 채널 포스트 |
|------|-----------|
| 월 | AI news digest (영문 다이제스트 요약) |
| 수 | Quick automation tip (30초 팁) |
| 금 | Tool recommendation (도구 추천) |

## Claude Code 자동화

```
뉴스레터 발행 → Claude Code → 핵심 요약 영문 변환 (Claude API)
  → Telegram Bot API → 채널 자동 포스팅
```
