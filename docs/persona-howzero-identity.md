# 하우제로(HowZero) 완전 정체성 가이드

> 이 문서는 하우제로의 **누구인지, 누구를 위한 건지, 왜 존재하는지**를 정의합니다.
> 모든 콘텐츠/세일즈/파트너십/채용의 기준 문서.

---

## 1. 하우제로는 누구인가

### 한 문장 정의

**"코딩 1도 모르던 사람이 AI로 10억을 벌어본 경험을, 비개발자 기업에 이식해주는 실전가."**

### 정체성 키워드

| 하우제로는 | 하우제로는 아닌 |
|-----------|-------------|
| 실전가 (직접 해봤다) | 이론가 (강의만 한다) |
| 비개발자 출신 | 개발자/엔지니어 |
| 비즈니스 문제 해결자 | 기술 덕후 |
| 가이드/통역관 | 위에서 가르치는 선생 |
| 투명한 (실패도 공유) | 완벽한 척하는 사람 |
| 직설적 | 돌려 말하는 사람 |
| 데이터 기반 | 감 기반 |
| 실용주의자 | 이상주의자 |

### 하우제로가 세상에 존재하는 이유

```
"AI 시대에 가장 큰 불평등은 '기술을 아는 사람'과 '모르는 사람' 사이에 있다.
 개발자가 없다는 이유로 AI 혜택을 못 받는 기업이 99%다.
 나는 그 99%의 편에 서서, 기술 장벽을 없애고,
 비개발자도 AI로 돈을 벌 수 있게 만든다.
 왜냐하면 나 자신이 바로 그 비개발자였으니까."
```

## 2. 고객은 누구인가 (5가지 고객 아바타)

### 고객 아바타 1: "시간 없는 1인 사장" 민수 (34세)

```
이름: 김민수
나이: 34세
직업: 스마트스토어 셀러 (월매출 3,000만원)
상황: 혼자서 소싱+등록+CS+배송 다 함. 하루 12시간 일함.
고통: "CS 문의만 하루 2시간. 더 이상 못 하겠다."
목표: 반복 업무 줄여서 → 상품 기획/마케팅에 집중 → 월 1억
기술 수준: 엑셀은 쓸 줄 알고, ChatGPT는 써봤지만 API는 모름
예산: 월 50~200만원 (외주비 대체)
정보 채널: 유튜브 80% > 네이버 카페 > 카카오톡
의사결정: ROI 수치 보여주면 빠르게 결정
하우제로를 찾는 계기: "이커머스 CS 자동화" 유튜브 검색

→ 하우제로 메시지: "CS에 쓰는 2시간을 AI가 대신합니다.
   그 시간에 매출 올리세요."
→ 서비스: 컨설팅 스탠다드 (₩200만) + 리테이너 (₩30만/월)
```

### 고객 아바타 2: "디지털 전환 압박받는 대표" 정호 (45세)

```
이름: 박정호
나이: 45세
직업: 제조업 중견기업 대표 (직원 50명, 연매출 60억)
상황: 이사회에서 "AI 도입하라" 압박. 뭐부터 해야 할지 모름.
고통: "AI 업체 3곳 만났는데 다 수억 달라고 해서 포기했다."
목표: 합리적 비용으로 AI 도입 → 운영 효율 30% 개선
기술 수준: IT팀이 있지만 AI 전문성 없음
예산: 500~3,000만원 (정부 바우처 활용 가능)
정보 채널: 링크드인 > 지인 추천 > 기사/세미나
의사결정: ROI + 사례 + 신뢰가 필요. 2~3번 미팅 후 결정.
하우제로를 찾는 계기: 링크드인 포스트 or 파트너 추천

→ 하우제로 메시지: "수억 안 써도 됩니다. ₩200만으로
   같은 효과를 내는 방법이 있습니다."
→ 서비스: 기업 교육(₩800만) + AI 오딧(₩500만) + 바우처
```

### 고객 아바타 3: "성장 정체된 프리랜서" 수진 (29세)

```
이름: 이수진
나이: 29세
직업: 마케팅 프리랜서 (월 300~500만원)
상황: 클라이언트 3~4개 동시 관리. 성장하고 싶은데 시간 없음.
고통: "보고서 작성, 데이터 정리, 이메일에 하루가 다 간다."
목표: 반복 업무 자동화 → 클라이언트 2배 → 월 1,000만원
기술 수준: Notion, Canva는 잘 씀. Claude Code은 들어본 적 있음.
예산: 월 10~50만원
정보 채널: 인스타 > 유튜브 > 틱톡
의사결정: "진짜 되는지" 데모 보면 바로 결정
하우제로를 찾는 계기: 인스타 릴스 "프리랜서 자동화 3가지"

→ 하우제로 메시지: "보고서에 쓰는 3시간을 AI에게.
   그 시간에 클라이언트를 더 잡으세요."
→ 서비스: 셀프페이스 코스(₩99K) + 템플릿 팩(₩29K)
```

### 고객 아바타 4: "AI 교육이 필요한 강사" 현우 (42세)

```
이름: 이현우
나이: 42세
직업: 비즈니스 코치 (유튜브 3만, 수강생 500명)
상황: 수강생들이 "AI 어떻게 써요?" 계속 물어봄. 답 못 해줌.
고통: "AI 실전 경험이 없어서 가르칠 수가 없다."
목표: AI 자동화를 커리큘럼에 추가 → 경쟁 강사 대비 차별화
기술 수준: 콘텐츠 제작은 프로. AI 실전은 초보.
예산: 200~500만원 (투자 대비 수강생 매출 증가)
정보 채널: 유튜브 > 링크드인 > 팟캐스트
의사결정: "내 수강생에게 도움이 되는가?"가 기준
하우제로를 찾는 계기: 파트너 제안 or 유튜브 콜라보

→ 하우제로 메시지: "당신의 교육에 실전 AI를 더하세요.
   수강생이 성과를 내면 당신의 권위가 올라갑니다."
→ 서비스: B2B 파트너(30% 커미션) + 공동 코스 제작
```

### 고객 아바타 5: "AI 도입을 시도했다 실패한" 재민 (38세)

```
이름: 최재민
나이: 38세
직업: 서비스업 대표 (직원 15명, 월매출 1.5억)
상황: 작년에 AI 도구 3개 도입했는데 직원들이 안 씀. 돈 낭비.
고통: "AI가 좋다는 건 아는데, 우리 회사에는 왜 안 되는 거지?"
목표: 이번에는 제대로 → 실제로 작동하는 AI 시스템
기술 수준: ChatGPT는 써봤지만, 자동화 시스템은 모름
예산: 200~500만원 (이번에 되면 더 투자 의향)
정보 채널: 지인 추천 > 유튜브 > 구글 검색
의사결정: "실패 경험 있으니 더 신중. 사례가 3개 이상 있어야."
하우제로를 찾는 계기: "AI 도입 실패" 검색 → 역발상 콘텐츠 발견

→ 하우제로 메시지: "AI 도구가 문제가 아닙니다.
   프로세스부터 고쳐야 합니다. 그래서 오딧이 먼저입니다."
→ 서비스: AI 오딧(무료) → 컨설팅 스탠다드(₩200만) + 교육
```

## 3. 누가 하우제로를 봐야 하는가 (오디언스 4계층)

```
┌─────────────────────────────────────────┐
│  Layer 1: 핵심 타겟 (직접 고객)          │
│  "AI 자동화 도입하고 싶은 비개발자"       │
│  → 이 사람들이 돈을 낸다                 │
├─────────────────────────────────────────┤
│  Layer 2: 확장 타겟 (간접 고객)          │
│  "AI에 관심 있는 모든 비개발자"           │
│  → 지금은 구독자, 나중에 고객             │
├─────────────────────────────────────────┤
│  Layer 3: 파트너 타겟                    │
│  "비개발자를 가르치는 강사/코치/컨설턴트"  │
│  → 콜라보/어필리에이트/공동 코스           │
├─────────────────────────────────────────┤
│  Layer 4: 영향력 타겟                    │
│  "AI/비즈니스 미디어/기자/인플루언서"      │
│  → PR/노출/권위 부스트                   │
└─────────────────────────────────────────┘
```

### Layer 1: 핵심 타겟 — 이 사람들만 잡으면 된다

| 특성 | 설명 |
|------|------|
| **직업** | 중소기업 대표, 1인 사장, 프리랜서, 팀 리더 |
| **업종** | 이커머스 > 서비스업 > 법률/회계 > 부동산 > 병원 > 교육 |
| **규모** | 1인~50명 |
| **공통 고통** | "반복 업무에 시간 다 빼앗긴다" |
| **공통 욕구** | "AI가 대신 해줬으면" |
| **공통 장벽** | "개발자가 없어서 / 코딩을 몰라서 / 뭐부터 해야 할지 몰라서" |
| **연령** | 28~50세 |
| **소득** | 월 300만원+ (서비스 비용 지불 능력) |

### Layer 2: 확장 타겟 — 콘텐츠로 먼저 잡고, 나중에 전환

| 특성 | 설명 |
|------|------|
| **직업** | 직장인 (부업 관심), 취준생 (AI 스킬 희망), 대학생 |
| **관심** | "AI로 뭔가 해보고 싶은데 뭘 해야 할지 모르겠다" |
| **지금은** | 구독자/팔로워 → 뉴스레터 → 커뮤니티 |
| **나중에** | 코스 수강 → 이직 후 회사에 오딧 추천 → 1인 창업 시 컨설팅 |

### Layer 3: 파트너 타겟

| 특성 | 설명 |
|------|------|
| **유형** | 비즈니스 코치, 이커머스 강사, 노코드 교육자, HR 컨설턴트 |
| **수강생 수** | 500~10,000명 |
| **니즈** | "내 교육에 AI 실전을 추가하고 싶다" |
| **관계** | 하우제로의 어필리에이트 파트너 (30~40% 커미션) |

### Layer 4: 영향력 타겟

| 특성 | 설명 |
|------|------|
| **유형** | IT 기자, 스타트업 미디어, AI 유튜버, 비즈니스 팟캐스트 |
| **니즈** | "재밌는 스토리가 필요하다" |
| **관계** | PR + 인터뷰 + 게스팅 |

## 4. 브랜드 정체성

### 4.1 브랜드 에센스 (한 단어)

**"실전"**

모든 것이 이 한 단어에서 시작: 실전 경험, 실전 사례, 실전 도구, 실전 성과.

### 4.2 브랜드 약속 (Brand Promise)

```
"코딩 없이도, 개발자 없이도, AI 자동화는 됩니다.
 나는 그것을 직접 해봤고, 당신도 할 수 있게 만들어드립니다."
```

### 4.3 브랜드 성격 (Brand Personality)

| 성격 | 수준 | 표현 |
|------|:---:|------|
| **자신감** | ██████████ 100% | "10억 벌어봤으니까 자신 있습니다" |
| **직설적** | █████████░ 90% | "솔직히 말하면..." |
| **실용적** | ██████████ 100% | "지금 당장 쓸 수 있는 것만" |
| **도발적** | ████████░░ 80% | "아직도 수동으로?" |
| **친근함** | ███████░░░ 70% | "저도 처음엔 그랬습니다" |
| **유머** | █████░░░░░ 50% | 가끔 자조적 유머 |
| **학문적** | ██░░░░░░░░ 20% | 이론은 최소. 실전 위주 |

### 4.4 브랜드 보이스 (Voice) 가이드

| | 하우제로 스타일 ✅ | 하우제로 아닌 스타일 ❌ |
|---|---|---|
| **인사** | "안녕하세요, 하우제로입니다" | "안녕하세요 여러분~ 오늘도 좋은 하루~" |
| **주장** | "AI 도입의 80%는 프로세스 문제입니다" | "어쩌면 프로세스 문제일 수도 있지 않을까요?" |
| **사례** | "실제로 A 기업에서 월 200시간 절감했습니다" | "아마 효과가 있을 거예요" |
| **CTA** | "무료 오딧 30분, 지금 신청하세요" | "관심 있으시면 한번 알아보세요~" |
| **실패** | "솔직히, 이건 실패했습니다. 이유는..." | (실패를 숨김) |
| **경쟁** | "도구만 사면 자동화가 될까요? 안 됩니다" | "다른 서비스도 좋지만 저희도..." |

### 4.5 시각적 정체성

| 요소 | 설계 |
|------|------|
| **메인 컬러** | 딥 블루 (#1E3A5F) — 신뢰+전문성 |
| **액센트 컬러** | 오렌지 (#FF6B35) — 에너지+행동 유도 |
| **서브 컬러** | 라이트 그레이 (#F5F5F5) — 깔끔+모던 |
| **폰트** | Pretendard (한국어) / Inter (영문) — 깔끔+가독성 |
| **사진 톤** | 자연광, 밝은 배경, 깔끔한 데스크 — "정돈된 전문가" |
| **금지** | 네온, 과도한 이모지, 반짝이, 만화 캐릭터 |

### 4.6 하우제로가 절대 하지 않는 것

```
❌ 근거 없는 수익 보장 ("AI 쓰면 무조건 10억!")
❌ 경쟁사 직접 비방 (이름 언급+공격)
❌ 과도한 전문 용어 (API, Webhook을 설명 없이)
❌ 고객 무시 표현 ("이것도 모르세요?")
❌ AI 과대 포장 ("AI가 모든 걸 해결합니다!")
❌ 뜬구름 약속 ("엄청난 변화가...")
❌ 유행만 좇기 ("요즘 뜨는 AI 도구 100개!")
```

## 5. 하우제로의 고유한 무기 (Unfair Advantage)

### 5.1 왜 하우제로여야 하는가? (Not Why AI, but Why HowZero)

```
[고객의 선택지]

A. 혼자 한다 → "유튜브 보고 따라했는데 안 됨. 3개월 낭비."
B. 프리랜서 외주 → "만들어줬는데 관리/수정을 못 함. 또 돈 써야 함."
C. 대형 컨설팅 → "견적이 수억. 중소기업은 엄두도 못 냄."
D. AI SaaS만 구독 → "도구는 있는데 뭘 어떻게 쓰는지 모름."
E. 노코드 강사 → "강의 들었는데 실전에서 막힘. 도와줄 사람 없음."

F. 하우제로 → "진단부터 해주고, 직접 만들어주고, 내가 할 수 있게 가르쳐주고,
               막히면 도와주고, 가격은 합리적."
   = 유일한 올인원 + 비개발자 전문 + 실전 검증
```

### 5.2 하우제로만의 5가지 무기

| # | 무기 | 경쟁자가 못 따라하는 이유 |
|---|------|---------------------|
| 1 | **"10억 SaaS" 실전 스토리** | 똑같은 성과를 내려면 수년 필요 |
| 2 | **비개발자 → 비개발자 통역** | 개발자 출신은 비개발자 언어를 모름 |
| 3 | **진단→구축→교육→유지 올인원** | 대부분 이 중 1~2개만 함 |
| 4 | **축적된 오딧 데이터 (업종별)** | 데이터는 시간이 만듦. 따라잡기 불가 |

## 9. 브랜드 네임 스토리 — "왜 하우제로인가"

```
하우제로라는 이름 자체가 스토리입니다.

코딩 지식 → 제로에서 시작했고

개발자   → 제로명으로 운영하고

제로에서 시작해서, 제로로 만드는 사람.
그래서 하우제로.
```

### "5 → 0" 오리진 스토리

```
"제가 SaaS를 운영하면서 매일 5가지를 수동으로 했습니다.

 1. CS 문의 분류 — 하루 2시간
 2. 고객 응답 작성 — 하루 1시간
 3. 리뷰 모니터링 — 하루 30분
 4. 주간 보고서 — 주 2시간
 5. 데이터 정리 — 하루 1시간

 5가지. 매일. 직원 한 명이 풀타임으로 해도 모자랐습니다.

 AI를 붙이고 나서?
 5 → 0.
 5가지 업무가 전부 자동화됐습니다.
 사람이 하는 건 0.

 그래서 하우제로입니다.
 5를 0으로 만드는 방법(How)을 알려드립니다."
```
