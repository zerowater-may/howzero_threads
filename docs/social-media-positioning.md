# 하우제로 소셜 미디어 포지셔닝 가이드

> 유튜브 / 인스타그램 / 틱톡 / 쓰레드 — 채널별 정체성 + 톤 + 포맷 + 역할 완전 정의

---

## 0. 공통 원칙

```
하우제로 = "코딩 모르는 비개발자가 AI로 10억 번 실전가"

모든 채널 공통:
  ✅ 직설적, 데이터 기반, 과장 없이
  ✅ "나는 이렇게 해서 10억 벌었다" 자신감
  ✅ 실전 > 이론, 증거 > 약속
  ❌ 과대 포장, 감성 팔이, 뜬구름
```

### 채널별 역할 분담

```
[유튜브] — 신뢰 + 교육 (깊이)
    "이 사람 진짜 실력자네" → 구독 → 오딧 전환

[인스타] — 저장 + 공유 (가치)
    "이거 저장해둬야지" → 팔로우 → 뉴스레터

[틱톡] — 발견 + 바이럴 (도달)
    "이거 뭐야 신기하다" → 팔로우 → 유튜브로 이동

[쓰레드] — 대화 + 관계 (친근)
    "이 사람이랑 대화하고 싶다" → 팔로우 → DM → 오딧
```

## 2. 인스타그램 — "저장하고 싶은 AI 자동화 교과서"

### 포지셔닝 한 문장

**"비개발자가 저장해두고 나중에 적용하는 AI 자동화 실전 카드"**

| 항목 | 설계 |
|------|------|
| **계정명** | howzero.ai |
| **카테고리** | 비즈니스 / 교육 |
| **역할** | 저장(Saves) + 공유 → 팔로우 → 뉴스레터/진단 전환 |
| **핵심 지표** | 저장수 > 좋아요 > 댓글 |
| **타겟 감정** | "이거 나중에 쓸 거니까 저장" |

### 프로필

```
[프로필 사진] 유튜브와 동일 (브랜드 통일)

[바이오]
하우제로 | AI 자동화 컨설턴트

↓ 무료 AI 준비도 진단 받기

[링크] howzero.com/link (링크트리)

[하이라이트]
📊 AI 진단 | 🛠️ 자동화 사례 | 🎓 무료 가이드 | 💬 후기 | 📌 체크리스트
```

### 콘텐츠 포맷

| 포맷 | 비중 | 목적 | 핵심 지표 |
|------|:---:|------|----------|
| **캐러셀 (7~10장)** | 40% | 저장 극대화 (교육/체크리스트/비교표) | 저장수 |
| **릴스 (15~60초)** | 40% | 발견/도달 (숏폼 리포맷) | 조회수+완시청 |
| **스토리** | 매일 | 관계 유지 (투표/질문/비하인드) | 답장수+투표 |
| **피드 이미지** | 5% | Before/After 수치 인포그래픽 | 저장수 |
| **라이브** | 월 1회 | 신뢰 (Q&A) | 시청자수 |

### 인스타 톤 가이드

```
[캐러셀]
  Slide 1: 큰 텍스트 질문/도발 (호기심)
  Slide 2~8: 포인트 1개/슬라이드 (깔끔, 가독성)
  마지막: "저장📌하고 나중에 적용하세요" + CTA

  톤: 교육적이지만 쉽게. 전문가스럽지만 친근하게.
  ✅ "비개발자가 15분 만에 세팅 가능"
  ❌ "API Endpoint를 Webhook으로 연결하여..."

[릴스]
  유튜브 쇼츠를 인스타 톤으로 살짝 조정.
  자막 필수 (무음 시청 70%).
  트렌딩 음악/사운드 활용 (유튜브보다 자유롭게).

[스토리]
  일상 + 비하인드 + 투표 + 질문 스티커.
  완벽할 필요 없음. 날것 OK. "진짜 사람"이 느껴지게.
  ✅ 셀카 + "오늘 오딧 3건 했는데 이런 발견이..."
  ❌ 완벽하게 편집된 홍보 영상
```

### 인스타 캐러셀 10대 포맷

| # | 포맷 | 저장 유도 |
|---|------|:---:|
| 1 | 체크리스트 ("AI 시작 전 확인 7가지") | ★★★★★ |
| 2 | 비교표 ("Claude Code vs Make vs Zapier") | ★★★★★ |
| 3 | 스텝바이스텝 ("자동화 첫 세팅 5단계") | ★★★★★ |
| 4 | Before/After (수치 비교) | ★★★★ |
| 5 | 도구 추천 ("비개발자 TOP 5") | ★★★★★ |
| 6 | 미신 파괴 ("AI 자동화 5가지 오해") | ★★★★ |
| 7 | 로드맵 ("30/60/90일 플랜") | ★★★★★ |
| 8 | ROI 계산 ("자동화 투자 vs 절감") | ★★★★ |
| 9 | 데이터/통계 ("100명 진단 결과") | ★★★★ |
| 10 | 실패 사례 ("AI 도입 실패 3패턴") | ★★★ |

### CTA 구조

```
[캐러셀] → 마지막 슬라이드 "저장📌 + 프로필 링크"
[릴스] → "더 자세한 건 프로필 링크"
```

## 3. 틱톡 — "30초 만에 '와' 하게 만드는 AI 마법"

### 포지셔닝 한 문장

**"비개발자가 '이거 진짜야?' 하면서 팔로우 누르는 AI 자동화 쇼"**

### 채널 정체성

| 항목 | 설계 |
|------|------|
| **계정명** | @howzero |
| **역할** | 최대 도달 + 발견 → 유튜브/인스타/뉴스레터로 이동 |
| **핵심 지표** | 조회수 + 팔로워 증가 + 프로필 클릭 |
| **타겟 감정** | "이거 진짜? 더 보고 싶다" |

### 프로필

```
[프로필 사진] 유튜브/인스타와 동일

[바이오]
하우제로 🔥 AI 자동화
코딩 0줄로 10억 벌어봄
비개발자 AI 자동화 매일 공유

[링크] howzero.com/link
```

### 유튜브 쇼츠 vs 틱톡 — 차이점

| | 유튜브 쇼츠 | 틱톡 |
|---|---|---|
| **톤** | 교육적, 전문적 | **캐주얼, 리얼, 유머 OK** |
| **편집** | 깔끔한 자막 | **로우파이, 자연스러운** |
| **음악** | 없거나 배경 | **트렌딩 사운드 적극 활용** |
| **훅** | 정보형 ("이거 아세요?") | **감정형 ("이거 보고 놀람")** |
| **댓글 문화** | 진지 | **밈, 유머, 자기주장** |
| **얼굴** | 선택 | **얼굴 나오는 게 더 좋음** |

### 틱톡 네이티브 포맷 (유튜브 리포맷 X)

| 포맷 | 예시 | 빈도 |
|------|------|:---:|

| **스티치/듀엣** | 다른 AI 틱톡에 "비개발자 관점에서는..." | 주 1회 |
| **트렌딩 사운드** | 인기 사운드에 AI 자동화 데모 매치 | 주 2회 |
| **그린스크린** | 뉴스 기사/데이터 위에 해설 | 주 1회 |
| **스토리타임** | "코딩 0줄로 10억 번 이야기 #파트1" | 주 1회 |
| **라이브 빌드** | "30초 안에 자동화 만들기" 타임어택 | 주 1회 |
| **Before/After** | 자동화 전후 화면 분할 | 주 2회 |

## 4. 쓰레드(Threads) — "AI 자동화 동네 형/누나"

| 항목 | 설계 |

| **역할** | 대화 + 관계 → DM → 오딧 전환 |
| **핵심 지표** | 댓글수 + 답장수 + DM 전환 |
| **타겟 감정** | "이 사람이랑 이야기하고 싶다" |

### 유튜브/인스타/틱톡 vs 쓰레드 — 핵심 차이

```
유튜브: "선생님이 가르쳐주는 느낌"

틱톡:  "신기한 걸 보여주는 사람"
```

### 프로필

```
[바이오]
```

### 콘텐츠 유형 + 톤

| 유형 | 비중 | 톤 | 예시 |

### 쓰레드 톤 vs 다른 채널

```
[유튜브/인스타]
"AI 도입의 80%는 기술이 아니라 기본 프로세스 문제입니다."
→ 전문가 포지셔닝

[X/트위터]
"AI 도입의 80%가 실패한다. 기술이 아니라 프로세스 문제다."
→ 도발적, 직설적

[쓰레드] ← 이것
"오늘 컨설팅하면서 또 느낀 건데요,
 AI 도입에서 제일 중요한 건 기술이 아니라
 기본 프로세스가 정리되어 있느냐더라고요.
 여러분 회사는 어때요?"
→ 대화체, 친근, 동료 느낌
```

### 쓰레드 운영 원칙

```
1. 완벽한 글 쓰지 않기 — 생각나면 바로 올리기
2. 질문으로 끝내기 — 모든 포스트에 "여러분은?" 추가
3. 댓글에 진심으로 답하기 — 쓰레드는 대화가 핵심
4. 하루 1~2개면 충분 — 양보다 질, 자연스러움
5. 인스타와 톤 다르게 — 인스타는 정제, 쓰레드는 날것
6. 절대 광고 느낌 내지 않기 — CTA는 한 달에 2~3번만
```

### 주간 스케줄

| 요일 | 포스트 | 예시 |
|------|--------|------|
| **월** | 💡 주간 인사이트 | "이번 주 뉴스레터에서 다룬 건데요, CS 자동화에서 가장 중요한 건..." |
| **수** | 📖 비하인드/여정 | "오늘 법률사무소 오딧했는데, 세금계산서를 아직도 수동으로 하더라고요. 충격." |
| **금** | ❓ 질문/대화 | "주말이니까 가볍게. AI 도구 중에 '이거는 진짜 인생템'인 거 있어요?" |

### CTA (매우 자연스럽게, 드물게)

```
[평소] CTA 없음. 순수 가치/대화만.

[월 2~3회만]
"참고로, 이런 비효율을 30분 만에 찾는 게 AI 오딧인데,
 무료로 해드리고 있어요. 프로필 링크에 있습니다.
 궁금하면 DM 주셔도 됩니다."

→ "광고"가 아니라 "대화 중 자연스러운 언급"
```

## 5. 채널 간 연결 구조

```
[틱톡] 발견/도달 (가장 넓은 유입)
    ↓ "더 자세히 보려면 유튜브에서"
[유튜브] 신뢰/교육 (가장 깊은 관계)
    ↓ "매주 AI 팁 → 뉴스레터 구독"
[인스타] 저장/공유 (시각적 자산)
    ↓ "프로필 링크에서 진단/오딧"
[쓰레드] 대화/관계 (가장 친근)
    ↓ DM → 오딧 신청

[모든 채널] → howzero.com/link → 오딧/진단/뉴스레터/코스
```

### 1소스 멀티유즈 변환 맵

```
[원본: 유튜브 롱폼 1개]
    │
    ├── 유튜브 쇼츠 ×5 (교육 톤, 깔끔 자막)
    ├── 틱톡 ×5 (캐주얼 톤, 트렌딩 사운드, 리얼 느낌)
    ├── 인스타 릴스 ×3 (쇼츠 리포맷 + 인스타 최적화)
    ├── 인스타 캐러셀 ×2 (핵심 포인트 → 7장 카드뉴스)
    ├── 쓰레드 ×2 (핵심 인사이트 → 대화체 텍스트)
    └── 스토리 ×3 (비하인드 + 투표 + 클립 공유)

= 롱폼 1개 → 20개 콘텐츠
```
