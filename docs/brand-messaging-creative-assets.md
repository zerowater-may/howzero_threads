## 1. 브랜드 비주얼 시스템

### 1.1 컬러 팔레트

| 용도 | 컬러명 | HEX | RGB | 사용처 |
|------|--------|-----|-----|--------|
| **Primary** | 딥 블루 | `#1E3A5F` | 30, 58, 95 | 헤더, 배경, 주요 텍스트, 로고 |
| **Accent** | 오렌지 | `#FF6B35` | 255, 107, 53 | CTA 버튼, 강조, 숫자 하이라이트 |
| **Background** | 화이트 | `#FFFFFF` | 255, 255, 255 | 기본 배경 |
| **Surface** | 라이트 그레이 | `#F5F5F5` | 245, 245, 245 | 카드 배경, 섹션 구분 |
| **Text Primary** | 다크 네이비 | `#0D1B2A` | 13, 27, 42 | 본문 텍스트 |
| **Text Secondary** | 미디엄 그레이 | `#6B7280` | 107, 114, 128 | 부제목, 캡션 |
| **Success** | 그린 | `#22C55E` | 34, 197, 94 | Before/After "After" 강조 |
| **Danger** | 레드 | `#EF4444` | 239, 68, 68 | Before/After "Before" 강조 |

### 1.2 컬러 사용 비율

```
딥 블루(Primary)     60%  — 배경, 프레임, 텍스트
오렌지(Accent)       20%  — CTA, 강조 포인트, 숫자
화이트/그레이        20%  — 여백, 카드, 호흡
```

### 1.3 타이포그래피

| 용도 | 폰트 | 웨이트 | 크기(참고) |
|------|------|--------|-----------|
| **한국어 제목** | Pretendard | Bold (700) | 32~48px |
| **한국어 본문** | Pretendard | Regular (400) | 16~18px |
| **영문 제목** | Inter | Bold (700) | 32~48px |
| **영문 본문** | Inter | Regular (400) | 16~18px |
| **숫자 강조** | Inter | ExtraBold (800) | 48~72px |
| **코드/기술** | JetBrains Mono | Regular (400) | 14~16px |

### 1.4 톤 스펙트럼 (확정)

| 성격 | 수준 | 채널별 적용 |
|------|:---:|-----------|
| **실용적** | 100% | 전 채널 — 뜬구름 금지. 당장 쓸 수 있는 것만 |
| **자신감** | 95% | 전 채널 — "10억 벌어봤으니까" (단, 오만이 아닌 경험 기반) |
| **직설적** | 90% | 유튜브/인스타/틱톡 — "솔직히 말하면..." |
| **도발적** | 80% | 틱톡/쇼츠 — "아직도 수동으로?" (쓰레드에서는 30%로 낮춤) |
| **친근함** | 70% | 쓰레드 중심 — "저도 처음엔 그랬습니다" |
| **유머** | 50% | 틱톡/스토리 — 자조적 유머, 밈 활용 |
| **학문적** | 20% | 최소한으로 — 이론은 실전 뒷받침용으로만 |

### 1.5 비주얼 금지 목록

```
금지 항목:
  - 네온 컬러, 형광 하이라이트
  - 과도한 이모지 (제목에 1~2개까지만)
  - 반짝이, 그라데이션 남발, 3D 효과
  - 만화 캐릭터, 클립아트
  - 스톡 이미지 (어색한 악수, 가짜 미소)
  - 워터마크가 있는 이미지
  - 6종 이상 폰트 혼용
```

## 3. 캐치프레이즈 3종 (최종 확정)

### 3.1 메인 캐치프레이즈 (권위 + 스토리)

> **"제로에서 10억까지. 코딩도 제로."**

- **용도**: 유튜브 배너, 웹사이트 히어로, 명함, 발표 슬라이드
- **톤**: 자신감 95% + 직설적 90%
- **왜 이것인가**: 하우제로의 오리진 스토리(코딩 제로에서 시작 → 10억 달성)와 브랜드 네임(How + Zero)을 동시에 전달. 짧고 기억에 남는 대구법.

### 3.2 서브 캐치프레이즈 (실용 + 행동 유도)

> **"5를 0으로 만드는 방법. 하우제로."**

- **용도**: 숏폼 엔딩, 인스타 캐러셀 마지막 슬라이드, 뉴스레터 서명
- **톤**: 실용적 100% + 도발적 80%
- **왜 이것인가**: "5가지 수동 업무 → 0"이라는 구체적 스토리를 압축. 수치가 있어 신뢰도 높음. 브랜드명 자연 삽입.

### 3.3 도발형 캐치프레이즈 (바이럴 + 훅)

> **"CS 비용 제로. 야근 제로. 코딩 제로. 변명도 제로."**

- **용도**: 틱톡/쇼츠 훅, 광고 카피, SNS 도발 포스트
- **톤**: 도발적 80% + 자신감 95%
- **왜 이것인가**: "제로" 반복으로 리듬감. 마지막 "변명도 제로"로 행동 촉구. 숏폼 훅으로 강력함.

### 캐치프레이즈 사용 매트릭스

| 채널 | 메인 | 서브 | 도발형 |
|------|:---:|:---:|:---:|
| 유튜브 배너/소개 | **주력** | 보조 | - |
| 유튜브 쇼츠 | - | 엔딩 | **훅** |
| 인스타 피드 | 프로필 | **주력** | - |
| 인스타 릴스 | - | - | **훅** |
| 틱톡 | - | - | **주력** |
| 쓰레드 | - | 가끔 | - |
| 웹사이트 | **주력** | 서브 섹션 | CTA 영역 |
| 명함/발표 | **주력** | - | - |

## 6. 브랜드 보이스 Quick Reference

### DO (이렇게 쓴다)

| 상황 | 하우제로 스타일 |
|------|--------------|
| **주장** | "AI 도입의 80%는 프로세스 문제입니다" |
| **사례** | "실제로 A 기업에서 월 200시간 절감했습니다" |
| **CTA** | "무료 오딧 30분, 지금 신청하세요" |
| **실패** | "솔직히, 이건 실패했습니다. 이유는..." |
| **비교** | "도구만 사면 자동화가 될까요? 안 됩니다" |
| **인사** | "안녕하세요, 하우제로입니다" |

### DON'T (이렇게 쓰지 않는다)

| 상황 | 금지 스타일 |
|------|----------|
| **주장** | "어쩌면 프로세스 문제일 수도 있지 않을까요?" |
| **사례** | "아마 효과가 있을 거예요" |
| **CTA** | "관심 있으시면 한번 알아보세요~" |
| **실패** | (실패를 숨김) |
| **비교** | "다른 서비스도 좋지만 저희도..." |
| **인사** | "안녕하세요 여러분~ 오늘도 좋은 하루~" |

### 절대 하지 않는 것

```
- 근거 없는 수익 보장 ("AI 쓰면 무조건 10억!")
- 경쟁사 직접 비방 (이름 언급 + 공격)
- 과도한 전문 용어 (API, Webhook을 설명 없이)
- 고객 무시 표현 ("이것도 모르세요?")
- AI 과대 포장 ("AI가 모든 걸 해결합니다!")
- 뜬구름 약속 ("엄청난 변화가...")
- 유행만 좇기 ("요즘 뜨는 AI 도구 100개!")
```
