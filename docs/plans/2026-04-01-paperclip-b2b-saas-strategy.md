## Executive Summary

Paperclip(MIT 라이선스, 오픈소스)을 커스터마이징하여 B2B AI 업무자동화 SaaS 사업을 구축한다. **CLI 구독형(Claude Code Max + Codex Pro)으로 API 비용 0원 운영.** 고객사에 AI 직원 팀을 배치하고, 고객 대표가 웹 대시보드로 업무현황을 확인하는 구조. **1회성 도입비 + 월 구독 모델.** 오프라인 미팅으로 세일즈하고, 전 과정을 유튜브 다큐 콘텐츠로 제작하여 콘텐츠 = 세일즈 플라이휠을 구축한다. 12개월 목표 연매출 ₩3.24억, 마진율 65~82%.

## 배경 & 현황 (팩트 기반)

### Paperclip 기술 현황
- MIT 라이선스, 셀프호스팅, 상업적 사용 자유
- 멀티컴퍼니 지원 (현재 howzero, bulsaja 2개 회사 운영 중)
- 지원 어댑터: `claude_local`, `codex_local`, claude_api, gemini, cursor 등
- 웹 UI 내장 (이슈보드, 에이전트 상태, 코멘트, 대시보드)
- 스택: Node.js + PostgreSQL + Redis
- Export/Import API로 회사 설정 통째로 이동 가능

### 비용 구조 — CLI 구독형 (API 0원)

| 항목 | 비용 | 비고 |
|------|------|------|
| Claude Code Max | $100~200/월 | 구독 1~2개, 시간 분산으로 여러 고객 커버 |
| ChatGPT Pro (Codex) | $200/월 | 구독 1~2개 |
| VPS (Hetzner) | $30~60/월 | Docker 격리 포함 |
| **합계** | **₩45~80만/월 (고정)** | **고객 3사든 10사든 동일** |

**핵심 차별점:** API 과금 모델과 달리, CLI 구독은 고정비 → 고객 늘수록 마진 폭발.

### 기술 아키텍처

```
[모바일/PC 브라우저]
      │
[nginx — SSL termination, rate limiting, HSTS]
      │
[Paperclip 앱 서버 — Linux VPS]
      │
[PostgreSQL — 별도 프로세스, 고객사별 schema]
      │
[에이전트 실행 레이어 — Docker 컨테이너 격리]
├── [고객 A 컨테이너] → claude_local 어댑터 → Claude Code Max (구독)
├── [고객 B 컨테이너] → codex_local 어댑터 → Codex/ChatGPT Pro (구독)
└── [고객 C 컨테이너] → claude_local + codex_local 혼용
```

### 프로덕션 설정 변경 (현재 → 목표)

| 설정 | 현재 | 목표 |
|------|------|------|
| deploymentMode | local_trusted | cloud |
| host | 127.0.0.1 | 0.0.0.0 |
| exposure | private | authenticated |
| disableSignUp | false | true (관리자 계정 생성) |
| database.mode | embedded-postgres | 별도 PostgreSQL 프로세스 |
| dangerouslySkipPermissions | true | false + 화이트리스트 |
| 에이전트 어댑터 | claude_local | claude_local + codex_local (CLI 구독) |

## 세일즈 프로세스 + 유튜브 콘텐츠 플라이휠

### 세일즈 퍼널

```
[유튜브 다큐 시리즈] "이 회사 AI 자동화 전 과정 공개"
      ↓ (시청자: "우리 회사도 해주세요")
[무료 AI 오딧 30분] 오프라인 or 화상
      ↓
[오프라인 미팅] 현장 방문 + 업무 진단 + 비효율 직접 확인
      ↓
[제안서 + 도입비 견적] → 도입비 ₩200~500만
      ↓
[도입 (2~4주)] 에이전트 세팅 + 교육 + (촬영)
      ↓
[월 구독 시작] 대시보드 접근 + 운영 + 월간 ROI 리포트
      ↓
[유튜브] 이 고객의 비포/애프터를 다시 콘텐츠로 → 선순환 플라이휠
```

### 유튜브 다큐 시리즈 구조

**핵심 컨셉:** "실제 고객사 AI 자동화 전 과정을 카메라에 담는다"

```
[시즌 1] 이커머스 셀러 자동화
  EP1. "하루 12시간 일하는 셀러를 만났다" — 비포 다큐
  EP2. "업무 진단: 6시간이 반복 작업이었다" — 오딧 과정
  EP3. "AI 직원 3명 투입 — 세팅 과정 전부 공개" — 구축 과정
  EP4. "2주 후: CS 시간 83% 감소" — 애프터 결과
  EP5. "사장님이 대시보드를 처음 본 반응" — 리액션 + CTA

[시즌 2] 마케팅 대행사 자동화
  EP1~5 동일 구조

[시즌 3] 유튜브 채널 운영 자동화
  EP1~5 동일 구조
```

**촬영 동의 할인:** 초기 3~5개 고객은 도입비 50% 할인 조건으로 촬영 동의 확보. 이 콘텐츠가 다음 고객을 데려오는 구조.

**하우제로 페르소나 정합:** "직접 해보고 증명한 것만 말한다" — 자동화 과정 자체를 투명하게 공개하여 신뢰 구축.

## 핵심 차별점

1. **실제 업무 수행:** 챗봇이 아님. CS 응대, 콘텐츠 작성, 데이터 정리를 끝까지 처리하는 AI 직원
2. **API 비용 0원:** CLI 구독형 고정비 모델. 고객 늘수록 마진 폭발. 경쟁사 대비 압도적 가격 경쟁력
3. **사장님 전용 화면:** 비기술인도 모바일로 "오늘 AI 직원이 뭘 했는지" 확인
4. **전 과정 유튜브 공개:** 자동화 과정을 투명하게 다큐로 제작 → 신뢰 + 세일즈 플라이휠
5. **오프라인 미팅:** 현장 방문하여 실제 업무를 직접 보고 진단 → 높은 전환율
6. **하우제로가 관리:** 고객은 관리 불필요. 하우제로 팀이 AI 직원 성과 모니터링 + 최적화

| D7~D14 | 파일럿 후보 3~5사 오프라인 미팅 | 2사 이상 파일럿 의사 (촬영 동의 포함) |

**Go/No-Go Gate:** 관심 10명 미만 OR 파일럿 0사 → 피벗 검토

### Phase 1: MVP + 파일럿 촬영 (6주)

| 주차 | 작업 | 산출물 |
|------|------|--------|
| 1~2주 | 인프라: PostgreSQL 분리, Docker 격리, 인증 강화, nginx HTTPS | 프로덕션 인프라 |
| 3~4주 | 제품: Paperclip 한국어 UI, 업종별 에이전트 프리셋 2개 (이커머스 CS, 마케팅 콘텐츠), 모바일 반응형 | 데모 가능 제품 |

- 파일럿 3사 운영 + 피드백 수집 + 제품 개선

- ROI 계산기 랜딩페이지 + 무료 AI 오딧 CTA

### Phase 3: 성장 (8주)

- 유료 고객 확보 목표: 15사
- 업종별 에이전트 템플릿 추가 (유튜브, 1인기업)
- 시즌2 (마케팅 대행사 자동화) 촬영/업로드
