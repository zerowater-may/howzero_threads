# 하우제로 첫 유튜브 숏폼 제작 플랜

> **제목:** "AI 좀 한다는 사람 vs 진짜 돈 버는 사람"
> **포맷:** 숏폼 (30~40초) | 비교/대조 스플릿 화면
> **목표:** 바이럴 → 채널 인지도 확보 → HOWAAA 브랜드 각인

---

## 1. 영상 구조 (타임라인)

### [0-3초] 훅 — 시선 잡기
- **화면:** 스플릿 화면 등장 (좌/우 동시)
- **자막 (크게, 중앙):** "AI 좀 한다는 사람 vs 진짜 돈 버는 사람"
- **BGM:** 긴장감 있는 비트 시작
- **TIP:** 첫 1초에 스플릿이 "팍" 등장해야 함. 페이드인 금지. 바로 시작.

### [3-15초] 왼쪽 — "세팅 중독자"
- **화면 녹화 소재:**
  1. 유튜브에서 "AI 자동화 세팅" 검색하는 화면 (2초)
  2. 검색 결과 영상 여러 개 빠르게 스크롤 (2초)
  3. 브라우저 탭 10개 이상 열린 화면: Notion, Zapier, Make, n8n, ChatGPT, Slack, Airtable 등 (3초)
  4. n8n이나 Make 워크플로우 복잡하게 연결된 화면 (3초)
  5. 캘린더에 "AI 세팅" 일정이 3개월째 반복되는 화면 (2초)
- **자막 순차 등장:**
  - "도구 10개 연동..."
  - "워크플로우 세팅..."
  - "3개월째 세팅 중..."
- **색감:** 약간 어둡게 / 채도 낮게 (부정적 느낌)

### [15-25초] 오른쪽 — "하우제로 방식"
- **화면 녹화 소재:**
  1. 심플한 자동화 1개 만드는 실제 화면 — 타임랩스 (3초)
  2. 완성된 자동화가 실제 작동하는 모습: CS 문의 들어오면 AI가 자동 응답하는 화면 (3초)
  3. 비용 절감 대시보드 or 스프레드시트에 절감 금액 표시 (2초)
  4. (선택) 실제 고객사 만족 메시지/리뷰 캡처 (2초)
- **자막 순차 등장:**
  - "자동화 1개"
  - "3시간 완성"
  - "월 1,000만원 절감"
- **색감:** 밝고 선명하게 (긍정적 느낌)

### [25-30초] 스플릿 결과 비교
- **왼쪽 하단:** 큰 숫자 "₩0" (빨간색, 굵은 폰트)
- **오른쪽 하단:** 큰 숫자 "-₩1,000만/월" (초록색, 굵은 폰트)
- **효과:** 숫자가 카운트업 애니메이션으로 등장
- **BGM:** 비트 드랍 or 효과음

### [30-37초] 엔딩 — 브랜드 각인
- **화면:** 스플릿 사라지고 전체 화면 전환 (검정 or 딥블루 배경)
- **자막 1 (2초):** "유행하는 AI 툴 장난칠 시간 없습니다."
- **자막 2 (2초):** "하우제로. HOWAAA."
- **로고:** HOWAAA 로고 or 텍스트
- **CTA (마지막 1초):** "팔로우" 버튼 애니메이션 or "프로필 링크에서 무료 오딧"

## 2. 촬영/소재 준비 체크리스트

### 왼쪽 화면 (세팅 중독자) — 화면 녹화
- [ ] 유튜브에서 "AI 자동화 세팅", "n8n 튜토리얼" 검색하는 화면 녹화
- [ ] 브라우저 탭 10개 이상 열기: Notion, Zapier, Make, n8n, ChatGPT, Slack, Airtable, Google Sheets, Trello, Asana
- [ ] n8n 또는 Make에서 의도적으로 복잡한 워크플로우 만들기 (실제 작동 안 해도 됨. 노드 많이 연결만)
- [ ] Google Calendar에 "AI 세팅" 반복 일정 3개월치 넣기

### 오른쪽 화면 (하우제로 방식) — 실제 작동 화면
- [ ] 실제 CS 자동화 시스템 작동 화면 녹화 (또는 데모 환경에서)
- [ ] 자동화 구축 과정 3시간 → 3~5초 타임랩스로 압축
- [ ] 비용 절감 대시보드/스프레드시트 준비 (월 1,000만원 절감 숫자)
- [ ] (선택) 고객사 만족 메시지 캡처 (개인정보 블러 처리)

### 공통
- [ ] BGM 선택: 저작권 프리 비트 (긴장감 → 해소 구조)
- [ ] 폰트: Pretendard Bold (자막용)
- [ ] 색상: 딥블루 #1E3A5F (브랜드) + 오렌지 #FF6B35 (액센트)

## 3. 편집 가이드

### 소프트웨어
- **추천:** CapCut Pro (숏폼 최적화) 또는 Premiere Pro
- **스플릿 화면:** 50:50 좌우 분할. 중앙에 얇은 흰색 라인

### 자막 스타일
- 폰트: Pretendard Bold 또는 Noto Sans KR Bold
- 크기: 화면 너비의 60~70% 차지
- 위치: 중앙~하단
- 배경: 반투명 검정 (opacity 60%)
- 색상: 흰색 기본. 숫자 강조 시 빨강(₩0) / 초록(절감액)

### 전환 효과
- 스플릿 등장: 하드컷 (애니메이션 없이 바로)
- 자막 등장: 팝업 (0.1초 스케일업)
- 결과 숫자: 카운트업 애니메이션
- 엔딩 전환: 좌우가 중앙으로 슬라이드 → 전체 화면

### 속도감
- 왼쪽 화면: 1.5~2배속 (바쁘고 정신없는 느낌)
- 오른쪽 화면: 1~1.2배속 (차분하고 확실한 느낌)
- 대비 효과로 "이쪽은 혼란, 저쪽은 깔끔" 체감

2. "3개월 세팅 ₩0 vs 3시간 자동화 월 1000만원"
3. "AI 좀 한다는 사람 vs 진짜 돈 버는 사람"

```

중요한 건 "그래서 그 AI로 오늘 얼마를 아꼈는가"입니다.
```

## 5. 성과 측정 기준

| 지표 | 1주 목표 | 1달 목표 |
|------|---------|---------|
| 조회수 | 1,000+ | 10,000+ |
| 좋아요 | 50+ | 500+ |
| 댓글 | 10+ | 50+ |
| 구독 전환 | 20+ | 100+ |
| 오딧 신청 | 1+ | 5+ |

### 댓글 대응 전략
- **"이거 진짜야?"** → "네, 실제 고객사 사례입니다. 프로필 링크에서 무료 오딧 신청하시면 귀사도 진단해드립니다."
- **"어떤 도구 썼어요?"** → "도구는 중요하지 않습니다. 중요한 건 어떤 문제를 풀었느냐입니다. 궁금하시면 무료 오딧에서 직접 보여드릴게요."
- **"과장 아니야?"** → "숫자 공개합니다. 프로필 링크에서 사례 확인해보세요."
