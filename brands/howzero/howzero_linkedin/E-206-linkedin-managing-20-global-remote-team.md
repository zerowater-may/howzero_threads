```
한국, 필리핀, 인도, 미국.

"원격으로 20명을 어떻게 관리하나요?"
가장 많이 받는 질문입니다.

답: 관리하지 않습니다.
시스템이 관리합니다.

원격 팀 운영의 5가지 원칙:

1. 비동기 우선 (Async-first)
미팅은 주 2회. 나머지는 전부 비동기.
Slack + Notion + Loom.
"지금 당장 답해" → 금지.
"24시간 내 답변" → 기본.

2. 결과 기반 평가
출근 시간 → 의미 없음.
근무 시간 → 의미 없음.
산출물 → 유일한 기준.

3. 자동화된 온보딩
새 팀원 합류 시:
AI가 30개 문서를 자동 안내.
시스템이 접근 권한 자동 부여.
첫 주 태스크 자동 할당.
사장 개입: 환영 메시지 하나.

4. 투명한 대시보드
모든 프로젝트 현황 → 실시간 공개.
모든 매출/비용 → 팀 전체 공개.
숨기는 게 없으면 — 정치가 없습니다.

5. 문화 < 시스템
"우리는 자율적인 문화야" → 이건 구호입니다.
자율적으로 일할 수 있는 시스템을 만드는 것 → 이게 진짜입니다.

결과:
팀 이직률: 5% (업계 평균 25%).
고객 만족도: 4.8/5.
사장의 팀 관리 시간: 주 3시간.

원격 팀이 안 되는 건 — 사람 문제가 아닙니다.
시스템 부재입니다.
```
