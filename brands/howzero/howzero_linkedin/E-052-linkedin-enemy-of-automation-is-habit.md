# E-052: 링크드인 포스트 — "자동화의 적은 기술이 아니라 습관이다"

> **유형**: 링크드인 텍스트 포스트 | **길이**: 1,400자 | **톤**: 인사이트 + 관찰 기반

## 포스트 본문 (복사해서 바로 게시)

```
"이거 원래 이렇게 해왔는데요."

자동화를 막는 가장 강력한 한 마디입니다.

지난 1년간 50개 이상의 기업을 진단하면서
발견한 패턴이 있습니다.

자동화를 못 하는 기업의 공통점은
기술이 부족한 게 아니었습니다.

"원래 하던 방식"에 갇혀 있었습니다.

실제 사례를 하나 들겠습니다.

한 유통 회사의 발주 프로세스:
1. 재고 확인 (엑셀)
2. 발주서 작성 (워드)
3. 이메일로 발송
4. 전화로 확인
5. 다시 엑셀에 기록

5단계. 소요 시간: 매일 3시간.

"왜 이렇게 하세요?"
"원래 이렇게 해왔어요."

"엑셀 대신 자동화 도구 쓰면 어떨까요?"
"익숙하지 않아서요. 엑셀이 편해요."

여기서 문제는 엑셀이 아닙니다.
'편하다'라는 착각입니다.

매일 3시간 × 22일 × 12개월 = 연 792시간.
시급 1.5만 원 기준 = 연 1,188만 원.

'편한 습관'의 실제 비용: 연 1,200만 원.

자동화 후: 5단계 → 1단계.
재고 확인 → 자동 발주 → 자동 기록.
매일 3시간 → 10분.

세팅 비용: 이틀 + 월 2만 원.

그런데 이 회사가 자동화하는 데
가장 오래 걸린 건 기술 세팅이 아니었습니다.

"기존 방식을 버리겠다"는 결정이었습니다.

자동화의 진짜 적:
✗ 어려운 기술
✗ 비싼 도구
✗ 부족한 인력

✓ "원래 이렇게 해왔다"는 습관

습관을 바꾸는 방법은 간단합니다.
비용을 계산하세요.

"이 습관이 연간 얼마를 소모하고 있는가?"
숫자를 보면, 습관이 흔들립니다.
```
