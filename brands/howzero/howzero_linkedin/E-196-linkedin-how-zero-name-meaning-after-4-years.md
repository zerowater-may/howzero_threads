# E-196: 링크드인 포스트 — "How + Zero — 이름의 의미, 4년 후"

> **유형**: 링크드인 텍스트 포스트 | **길이**: 1,250자 | **톤**: 철학 + 결의

---

## 포스트 본문 (복사해서 바로 게시)

```
How + Zero.
이 이름을 지은 건 4년 전입니다.
4년 후, 의미가 더 깊어졌습니다.

처음의 의미:
How — 어떻게.
Zero — 제로에서.
"제로에서 어떻게 시작할 것인가."

4년 후의 의미:

How 1: "어떻게 시작할 것인가"
여전히 유효합니다.
매일 새로운 사람이 "어디서부터 시작해야 하나요?"라고 묻습니다.
이 질문에 답하는 것이 하우제로의 첫 번째 존재 이유입니다.

How 2: "어떻게 성장할 것인가"
₩0 → ₩1억은 시작입니다.
₩1억 → ₩10억은 시스템입니다.
₩10억 → ₩100억은 팀과 문화입니다.
각 단계마다 "How"가 다릅니다.
그 각각의 "How"를 경험으로 알려드립니다.

How 3: "어떻게 계속할 것인가"
시작보다 어려운 건 지속입니다.
4년간 계속하면서 배운 것:
열정은 소비됩니다. 시스템은 유지됩니다.
"계속하는 How"는 열정이 아니라 습관과 시스템입니다.

Zero 1: "제로에서 시작한다"
모든 사업은 ₩0에서 시작합니다.
코딩 0, 경험 0, 인맥 0.
이 상태에서도 시작할 수 있다는 걸 증명합니다.

Zero 2: "제로로 돌아간다"
성공하면 교만해지기 쉽습니다.
"제로 마인드셋"으로 돌아가는 것.
매 분기마다 "우리가 오늘 시작한다면?"을 묻는 것.
이게 하우제로가 4년간 성장한 비결입니다.

Zero 3: "제로를 향해 간다"
비효율 → Zero.
지식 격차 → Zero.
진입 장벽 → Zero.
비개발자가 AI를 쓰는 데 필요한 장벽을 0으로 만드는 것.

How + Zero.
4년 전에는 이름이었습니다.
4년 후에는 철학이 됐습니다.

이 철학은 10년 후에도 변하지 않습니다.

여러분의 회사/브랜드 이름에 담긴 의미가 있다면 댓글로 공유해주세요.
```
