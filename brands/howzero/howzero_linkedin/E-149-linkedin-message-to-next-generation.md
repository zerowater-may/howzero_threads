# E-149: 링크드인 포스트 — "내가 다음 세대에게 전하고 싶은 것"

> **유형**: 링크드인 텍스트 포스트 | **길이**: 1,250자 | **톤**: 진솔함 + 멘토링

---

## 포스트 본문 (복사해서 바로 게시)

```
내가 다음 세대에게 전하고 싶은 것.

20대에 알았으면 좋았을 것들입니다.

1. "안정적인 길"은 존재하지 않습니다

대기업, 공무원, 전문직.
안정적이라고 배웠습니다.
하지만 세상이 바뀌는 속도를 보세요.

10년 전 안정적이던 직업이 지금은 불안정합니다.
안정은 "좋은 직장"이 아니라 "적응 능력"에서 옵니다.

2. 실패는 빨리, 자주 하세요

제가 사업에서 실패한 횟수: 7번.
그 7번이 없었으면 ₩10억은 못 벌었습니다.

실패 1: 시장이 없는 서비스 → "시장 검증"을 배움
실패 2: 가격을 너무 낮게 → "가치 기반 가격"을 배움
실패 3: 혼자 다 하려 함 → "위임과 시스템"을 배움

실패는 비용이 아니라 수업료입니다.
빨리 낼수록 좋습니다.

3. "무엇을 아느냐"보다 "무엇을 만들 수 있느냐"

학위, 자격증, 지식 — 중요합니다.
하지만 세상은 "만든 것"에 돈을 줍니다.

블로그 1개, 자동화 1개, 서비스 1개.
이력서에 쓸 "프로젝트"가 "학점"보다 강력합니다.

4. AI를 두려워하지 마세요

"AI가 내 일자리를 뺏을까?"
이 걱정을 하는 대신 —
"AI로 내가 뭘 만들 수 있을까?"

AI를 쓰는 사람이 쓰지 않는 사람을 대체합니다.
AI가 사람을 대체하는 게 아닙니다.

5. 시작은 "0"이어도 됩니다

돈 0원. 경험 0년. 인맥 0명.
저도 거기서 시작했습니다.

How + Zero.
어떻게(How) 제로(Zero)에서 시작할 것인가.
이 질문에 답을 찾는 여정이 사업이고, 삶입니다.

```
