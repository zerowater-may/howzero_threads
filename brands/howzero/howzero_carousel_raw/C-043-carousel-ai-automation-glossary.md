```
AI 자동화 용어 사전

```

---

## 슬라이드 2
```
API (에이피아이)

소프트웨어끼리 대화하는 통로.

예시:
쇼핑몰 → 택배사 시스템으로
"주문 접수됐어" 자동 전달

개발자가 만들어 놓으면
비개발자는 버튼 하나로 연결합니다.
```

---

## 슬라이드 3
```
워크플로우 (Workflow)

업무가 흘러가는 순서.

예시:
문의 접수 → 분류 → 담당자 배정 → 답변

이 순서를 그림으로 그리면 = 프로세스 맵
이걸 자동으로 돌리면 = 자동화
```

---

## 슬라이드 4
```
노코드 / 로우코드

노코드: 코딩 없이 블록 조립처럼 만드는 것
로우코드: 약간의 코딩으로 만드는 것

대표 툴:
노코드 → Zapier, Make
로우코드 → n8n, Retool

비개발자라면 노코드부터 시작하세요.
```

---

## 슬라이드 5
```
LLM (Large Language Model)

대규모 언어 모델.
ChatGPT, Claude가 이것입니다.

텍스트를 이해하고 생성하는 AI.
요약, 분류, 번역, 글쓰기 가능.

도구일 뿐입니다. 마법이 아닙니다.
```

---

## 슬라이드 6
```
RPA (Robotic Process Automation)

소프트웨어 로봇이 사람 대신
반복 작업을 클릭·입력하는 것.

예시:
매일 아침 엑셀 데이터 → ERP 입력
로봇이 대신 클릭합니다.

AI와 결합하면 "판단"까지 가능.
```

---

## 슬라이드 7
```
프롬프트 (Prompt)

AI에게 주는 지시문.

❌ "보고서 써줘" → 엉뚱한 결과
✅ "이커머스 주간 매출 데이터를 분석해서
    핵심 3줄 요약 + 그래프 제안" → 정확한 결과

프롬프트 = AI의 성능을 결정하는 입력값.
```
