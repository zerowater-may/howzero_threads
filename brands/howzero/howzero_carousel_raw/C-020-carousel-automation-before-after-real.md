---

## 슬라이드 2
```
👤 고객 A — 이커머스 셀러

자동화 전:
· CS 응대: 하루 4시간
· 리뷰 확인: 하루 1시간
· 재고 확인: 하루 30분
= 하루 5시간 30분
```

---

## 슬라이드 3
```
👤 고객 A — 자동화 후

· CS 자동 분류+초벌 답변: 20분
· 리뷰 자동 감시+알림: 5분
· 재고 자동 알림: 0분 (완전 자동)
= 하루 25분

절감: 하루 5시간 5분
월 절감: ₩180만 (인건비 기준)
```

---

## 슬라이드 4
```
👤 고객 B — 법률사무소

자동화 전:
· 계약서 초안: 건당 3시간
· 판례 검색: 건당 2시간
· 의뢰인 보고서: 건당 1시간
= 건당 6시간
```

## 슬라이드 5
```

· 계약서 AI 초안: 15분 (수정만)
· 판례 AI 검색+요약: 10분
· 보고서 AI 생성: 5분
= 건당 30분

절감: 건당 5시간 30분
월 10건 기준: ₩320만 절감
```
