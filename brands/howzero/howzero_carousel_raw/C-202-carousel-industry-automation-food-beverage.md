# C-202: 인스타 캐러셀 — "업종별 자동화 — 식음료 편"
> **유형**: 인스타 캐러셀 | **슬라이드**: 9장
## 슬라이드 1: 업종별 자동화 — 식음료 편 🍽️ (저장 필수)
## 슬라이드 2: 식음료 업종은 반복 업무의 보고다 — 예약, 재고, 고객 응대, 리뷰 관리까지 자동화할 수 있는 영역이 넘친다
## 슬라이드 3: 자동화 1 — 예약 관리: 네이버·카카오·전화 예약 통합 → 자동 확인 문자 → 노쇼 방지 리마인더
## 슬라이드 4: 자동화 2 — 재고/발주: 일일 판매 데이터 기반 소모량 예측 → 자동 발주 리스트 생성 → 공급업체 연동
## 슬라이드 5: 자동화 3 — 리뷰 관리: 네이버·배민·구글 리뷰 수집 → 감성 분석 → 부정 리뷰 즉시 알림 → 답변 초안 생성
## 슬라이드 6: 자동화 4 — 직원 스케줄링: 매출 예측 기반 필요 인원 산정 → 자동 시프트 배정 → 대타 요청 시스템
## 슬라이드 7: 자동화 5 — 마케팅: 시즌별 메뉴 자동 홍보 → SNS 포스팅 스케줄링 → 단골 고객 타겟 쿠폰 발송
## 슬라이드 8: 핵심 — 식음료 자동화의 시작은 '주문-재고-리뷰' 이 세 가지. 여기서부터 월 40시간을 아낄 수 있다
