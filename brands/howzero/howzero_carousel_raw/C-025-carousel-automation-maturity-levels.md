# C-025: 인스타 캐러셀 — "자동화 성숙도 5단계 — 당신은 어디?"

> **유형**: 인스타 캐러셀 | **슬라이드**: 8장 | **톤**: 자기 진단

---

## 슬라이드 1 (표지)
```
자동화 성숙도 5단계

당신은 몇 단계?

💡 현재 위치를 알아야 다음 단계가 보입니다
```

---

## 슬라이드 2
```
Level 0: 완전 수동 🔴

· 모든 업무를 사람이 직접 수행
· 엑셀 복붙이 일상
· "AI? 아직 먼 이야기"
· 전체 기업의 약 40%

→ 지금 여기라면: 종이 한 장에 반복 업무 적기부터
```

---

## 슬라이드 3
```
Level 1: 부분 자동화 🟠

· 1~2개 업무만 자동화
· "이메일 분류는 AI가 해요"
· 나머지는 여전히 수동
· 전체 기업의 약 25%

→ 지금 여기라면: 두 번째, 세 번째 자동화 추가
```

---

## 슬라이드 4
```
Level 2: 프로세스 자동화 🟡

· 한 프로세스의 전체 흐름이 자동
· 예: 문의 접수 → 분류 → 답변 → 기록 (전부 자동)
· 수동 개입은 예외 케이스만
· 전체 기업의 약 20%

→ 지금 여기라면: 다른 프로세스에도 적용 확장
```

---

## 슬라이드 5
```
Level 3: 시스템 자동화 🟢

· 여러 프로세스가 연결되어 작동
· CS + 리뷰 + 재고 + 보고서가 하나의 시스템
· 사람은 의사결정만
· 전체 기업의 약 10%

→ 지금 여기라면: 최적화와 모니터링에 집중
```

---

## 슬라이드 6
```
Level 4: 자율 운영 🔵

· 시스템이 스스로 판단하고 실행
· 이상치 감지 → 자동 대응
· 사람은 전략만 수립
· 전체 기업의 약 5%

→ 여기에 도달하면: AI가 제안하고 사람이 승인
```

---

## 슬라이드 7
```
현실적 목표:

대부분은 Level 0~1입니다.
Level 2~3이면 이미 상위 15%.

Level 0 → 1: 15분
Level 1 → 2: 1~2주
Level 2 → 3: 1~2개월

급할 필요 없습니다.
한 단계씩. 하나씩.
```
