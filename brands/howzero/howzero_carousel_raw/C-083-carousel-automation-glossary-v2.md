# C-083: 인스타 캐러셀 — "자동화 용어 사전 v2 — 심화편"
> **유형**: 인스타 캐러셀 | **슬라이드**: 9장
## 슬라이드 1: 자동화 용어 사전 v2 — 심화편 📖
## 슬라이드 2: Webhook — 이벤트 발생 시 자동으로 다른 시스템에 알림을 보내는 방식
## 슬라이드 3: ETL — 데이터를 추출(Extract)→변환(Transform)→적재(Load)하는 파이프라인
## 슬라이드 4: Orchestration — 여러 자동화를 순서대로 연결·관리하는 상위 제어
## 슬라이드 5: Idempotency — 같은 작업을 여러 번 실행해도 결과가 동일한 성질 (안정성 핵심)
## 슬라이드 6: Rate Limiting — API 호출 횟수 제한. 초과 시 자동화 멈춤 → 대비 필수
## 슬라이드 7: Retry Logic — 실패 시 자동 재시도. 횟수·간격 설정이 안정성 좌우
## 슬라이드 8: Fallback — 자동화 실패 시 대안 경로. 수동 처리 알림 등
