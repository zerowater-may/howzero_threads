# C-173: 인스타 캐러셀 — "자동화 비용 항목 v2 — 숨겨진 비용까지"
> **유형**: 인스타 캐러셀 | **슬라이드**: 10장
## 슬라이드 1: 자동화 비용 항목 v2 — 숨겨진 비용까지 💰 (저장 필수)
## 슬라이드 2: 대부분 도구 비용만 계산한다 — 하지만 진짜 비용은 눈에 안 보이는 곳에 숨어 있다
## 슬라이드 3: 보이는 비용 ① — 도구/SaaS 구독료: Zapier, Make, API 호출 비용. 월 10~100만 원 수준
## 슬라이드 4: 보이는 비용 ② — 구축 비용: 외부 개발/컨설팅비. 내부 담당자 투입 시간의 기회비용
## 슬라이드 5: 숨겨진 비용 ① — 학습 비용: 팀원 교육 시간. 초기 생산성 저하 기간 (평균 2~4주)
## 슬라이드 6: 숨겨진 비용 ② — 유지보수 비용: API 변경 대응, 오류 수정, 업데이트. 월 구축 비용의 15~20%
## 슬라이드 7: 숨겨진 비용 ③ — 기회비용: 잘못된 자동화에 투입한 시간. 다른 프로젝트 지연 비용
## 슬라이드 8: 숨겨진 비용 ④ — 전환 비용: 도구 변경 시 마이그레이션. 데이터 이전. 재교육 비용
## 슬라이드 9: 핵심 — 총 비용(TCO)으로 판단하라. 도구 비용만 보면 ROI 계산이 틀린다. 숨겨진 비용까지 포함해야 진짜 수익이 보인다
