# C-037: 인스타 캐러셀 — "팀에 AI 자동화 도입하는 법"

> **유형**: 인스타 캐러셀 | **슬라이드**: 7장

## 슬라이드 1 (표지)
```
팀에 AI 자동화 도입하는 법
(저항 없이, 효과적으로)
```

## 슬라이드 2
```
Step 1: 작게 시작
전체 팀이 아니라 1명만. 가장 열린 팀원 1명과 시작. 성공 사례를 만든 후 확산.
```

## 슬라이드 3
```
Step 2: "도움"으로 포지셔닝
"AI가 너의 일을 대체할 거야" ❌
"AI가 귀찮은 일을 해줄 거야" ✅
핵심: 위협이 아니라 도움.
```

## 슬라이드 4
```
Step 3: 15분 데모
말로 설명하지 말고, 보여주세요. 팀원의 실제 반복 업무를 15분 만에 자동화. "와!"가 나오면 성공.
```

## 슬라이드 5
```
Step 4: 자율성 보장
강제하지 마세요. "써봐" 대신 "쓸 수 있어". 좋은 도구는 강제하지 않아도 퍼집니다.
```

## 슬라이드 6
```
Step 5: 성과 공유
자동화로 절감된 시간을 공유. "이번 주에 팀 전체 15시간 절감!" 숫자가 동기부여.
```

## 슬라이드 7 (CTA)
```
팀 AI 도입이 어렵다면 같이 해드립니다.
📊 howzero.com/ai-score
🔍 팀 워크숍: howzero.com/audit
하우제로 | "한 명이 시작하면 팀이 바뀝니다."
```
