## 슬라이드 1 (표지)

```
비개발자 창업자를 위한
AI 활용 가이드

```

---

## 슬라이드 2
```
먼저 안심하세요.

2026년 AI는:
· 코딩이 필요 없습니다
· 한국어로 말하면 됩니다
· 15분이면 첫 자동화를 만듭니다

"개발자가 아니니까 못 한다"
→ 2024년까지의 이야기입니다.
```

---

## 슬라이드 3
```
Step 1: AI가 뭘 하는지 이해하기

AI는 마법이 아닙니다.
AI는 "패턴을 찾고, 반복하는 기계"입니다.

잘하는 것: 분류, 요약, 번역, 생성, 반복
못하는 것: 판단, 감정, 비전, 관계, 창의
```

---

## 슬라이드 4
```
Step 2: 내 업무 중 AI가 할 수 있는 것 찾기

종이 한 장에 매일 하는 업무를 적고:
"이거 규칙이 있나?" Y/N 체크

Y인 것 = AI가 할 수 있는 것

예:
이메일 분류 → Y (규칙 있음)
고객 미팅 → N (판단 필요)
보고서 작성 → Y (규칙 있음)
사업 전략 → N (창의 필요)
```

---

## 슬라이드 5
```
Step 3: 가장 쉬운 것 1개부터

하나만 고르세요. 가장 단순한 것.
추천: 이메일 분류 or 데이터 정리

Claude Code에:
"이 이메일들을 긴급/일반/참조로 분류해줘"

끝. 이게 첫 AI 활용입니다.
```

---

## 슬라이드 6
```
Step 4: 성공하면 2~3개 추가

첫 자동화 성공 → 자신감 ↑
두 번째: 보고서 자동 생성
세 번째: CS 답변 초안

3개면 주 10시간 이상 절감.
```

---

## 슬라이드 7
```
Step 5: 시스템으로 연결

개별 자동화 → 하나의 흐름으로
예: 이메일 분류 → 긴급 건 자동 답변 → 답변 기록 → 주간 리포트

이러면 "AI 시스템"이 됩니다.
사장님은 아침에 리포트만 확인.
```

---

## 슬라이드 8
```
비개발자 창업자의 진짜 경쟁력:

개발자는 기술을 압니다.
창업자는 비즈니스를 압니다.

AI 시대에 중요한 건:
"이 기술로 뭘 할 수 있을까?" (비즈니스 감각)
≫ "이 기술을 어떻게 만들까?" (기술 능력)

당신이 개발자보다 유리한 이유입니다.
```
