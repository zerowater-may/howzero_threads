## 슬라이드 2
```
ROI 측정 기준

투입: 도구 비용 + 세팅 시간 + 학습 시간
산출: 절감 시간 × 시급 + 매출 증가분

ROI = (산출 - 투입) ÷ 투입 × 100%

아래 수치는 하우제로 클라이언트
실제 데이터 기반 평균값입니다.
```
