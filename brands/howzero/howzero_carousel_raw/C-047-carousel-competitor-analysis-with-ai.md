```

매일 경쟁사 홈페이지 들어가서
눈으로 확인하고 계시죠?

🔍 자동으로 추적하고, 알림 받으세요
```

---

## 슬라이드 2
```
왜 경쟁사 분석을 자동화해야 하나?

수동 분석의 문제:
❌ 시간이 많이 걸린다
❌ 빠뜨리는 정보가 있다
❌ 일관성이 없다
❌ 실시간 대응이 안 된다

AI 자동화하면:
✅ 24시간 모니터링
✅ 변화 감지 → 즉시 알림
✅ 데이터 축적 → 트렌드 분석
```

---

## 슬라이드 3
```
분석 영역 1: 가격 모니터링

경쟁사 주요 상품 가격을 자동 추적

도구:
• 웹 스크래핑 + 구글 시트
• Make 시나리오 → 슬랙 알림

설정 예시:
경쟁사 가격이 5% 이상 변동 시
→ 슬랙 채널에 자동 알림

가격 전쟁에서 1시간이면 승부가 갈립니다.
```

---

## 슬라이드 4
```
분석 영역 2: 콘텐츠·SNS 추적

경쟁사가 어떤 콘텐츠를 올리는지 자동 수집

추적 대상:
• 블로그 포스트 (RSS 피드)
• 인스타·링크드인 게시물
• 뉴스레터

AI로 요약:
"경쟁사 A가 이번 주 올린 콘텐츠 3개 요약"
→ 매주 월요일 리포트 자동 생성
```

---

## 슬라이드 5
```
분석 영역 3: 리뷰·고객 반응

경쟁사 고객이 뭘 불만족하는지 = 우리의 기회

추적 대상:
• 네이버 쇼핑 리뷰
• 앱스토어 리뷰
• SNS 언급

AI 감성 분석:
부정 리뷰 키워드 자동 추출
→ "배송 느림" "CS 불만" 등 약점 파악
```

---

## 슬라이드 6
```
분석 영역 4: 채용 공고 분석

경쟁사가 어떤 포지션을 뽑는지 =
다음에 뭘 하려는지 알 수 있습니다.

"AI 엔지니어 채용" → AI 서비스 준비 중
"해외 마케터 채용" → 해외 진출 준비 중

채용 공고는 가장 솔직한 전략 발표문입니다.
```

---

## 슬라이드 7
```
5단계 세팅 가이드

1️⃣ 경쟁사 3~5개 선정
2️⃣ 분석 영역 선택 (가격/콘텐츠/리뷰/채용)
3️⃣ 데이터 수집 도구 세팅
4️⃣ AI 요약·분석 자동화
5️⃣ 주간 리포트 → 슬랙/이메일 자동 발송

한 번 세팅하면 매주 자동으로 돌아갑니다.
```
