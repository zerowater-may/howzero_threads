# C-148: 인스타 캐러셀 — "자동화 인수인계 가이드 — 담당자 바뀌어도 OK"
> **유형**: 인스타 캐러셀 | **슬라이드**: 9장
## 슬라이드 1: 자동화 인수인계 가이드 — 담당자 바뀌어도 OK 🔄 (저장 필수)
## 슬라이드 2: 담당자 퇴사 = 자동화 마비? — 인수인계 체계 없으면 자동화는 사람에 종속된다. 시스템으로 만들어라
## 슬라이드 3: 인수인계 문서 필수 항목 — ① 자동화 목록 + 각각의 목적 ② 사용 도구 + 로그인 정보 관리 방법
## 슬라이드 4: 필수 항목 계속 — ③ 각 자동화 플로우 상세 설명 (SOP) ④ 장애 대응 매뉴얼 ⑤ 비용 정보
## 슬라이드 5: 인수인계 프로세스 — 1주차: 문서 리뷰 → 2주차: 실습 운영 → 3주차: 독립 운영 + 질의응답
## 슬라이드 6: 도구 접근 권한 — 개인 계정 NO. 팀 계정/조직 계정 사용. 비밀번호 관리 도구(1Password 등) 필수
## 슬라이드 7: 자동화 팁 — 인수인계 체크리스트를 Notion 템플릿으로 만들어 놓으면 매번 재사용 가능
