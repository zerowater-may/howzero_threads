## 슬라이드 1 (표지)
```
AI 활용 주간 루틴 플래너

```

---

## 슬라이드 2
```
월요일: 주간 계획 + 데이터 점검

□ AI로 지난주 성과 데이터 요약
□ 이번 주 할 일 우선순위 정리
□ 자동화 실행 로그 확인 (에러 체크)
□ 주간 목표 설정

프롬프트 예시:
"지난주 매출 데이터를 분석해서
핵심 인사이트 3가지와 이번 주 액션 아이템을 정리해줘"
```

---

## 슬라이드 3
```
화요일: 콘텐츠 제작

□ AI로 콘텐츠 아이디어 5개 생성
□ 캐러셀/포스트 초안 작성
□ 이메일 뉴스레터 초안 생성
□ 해시태그 리서치

프롬프트 예시:
"이커머스 대표가 관심 가질 AI 자동화 주제로
인스타 캐러셀 아이디어 5개를 제안해줘"

1일 집중 → 1주일 콘텐츠 완성.
```

---

## 슬라이드 4
```
수요일: 고객 관리 + 커뮤니케이션

□ AI로 고객 문의 트렌드 분석
□ 자주 묻는 질문 업데이트
□ 팔로업 이메일 초안 작성
□ 고객 피드백 감성 분석

프롬프트 예시:
"이번 주 고객 문의 50건을 분석해서
가장 많은 불만 유형 3가지와 개선 방안을 정리해줘"
```

---

## 슬라이드 5
```
목요일: 분석 + 리포트

□ AI로 주간 매출 리포트 초안 생성
□ 광고 성과 분석 요약
□ 경쟁사 동향 자동 리포트 확인
□ KPI 대시보드 점검

프롬프트 예시:
"이번 주 광고 데이터를 분석해서
ROAS, CPC, 전환율 기준 성과 요약표를 만들어줘"
```

---

## 슬라이드 6
```
금요일: 최적화 + 다음 주 준비

□ 이번 주 자동화 성과 정리
□ 개선할 점 기록
□ 다음 주 자동화 테스트 계획
□ AI 프롬프트 라이브러리 업데이트

프롬프트 예시:
"이번 주 자동화 성과를 정리하고
다음 주 개선할 점 3가지를 제안해줘"
```

□ 새로운 AI 도구 1개 테스트
□ AI 관련 아티클 2~3개 읽기
□ 다른 업계 자동화 사례 리서치
□ 아이디어 메모

주말에 30분만 투자하면
월요일이 완전히 달라집니다.
