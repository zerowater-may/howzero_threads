```

→ claude.ai 접속
→ 계정 만들기 (이메일만 있으면 됨)
→ Pro 구독 ($20/월) — 무료도 가능하지만 Pro 추천

```

```
→ 최근 받은 이메일 10~20개 복사
→ Claude Code에 붙여넣기
```

```

"이 이메일들을 긴급/일반/참조/스팸으로
분류하고, 각각 1줄 요약해줘"

```
