# C-033: 인스타 캐러셀 — "AI에 대한 7가지 오해 (팩트로 반박)"

> **유형**: 인스타 캐러셀 | **슬라이드**: 9장

## 슬라이드 1 (표지)
```
AI에 대한 7가지 오해
팩트로 반박합니다
```

## 슬라이드 2
```
오해 #1: "AI는 만능이다"
팩트: AI는 반복 작업에 강하고, 창의적 판단은 약합니다.
적용: 반복은 AI에, 판단은 사람에.
```

## 슬라이드 3
```
오해 #2: "AI는 비싸다"
팩트: Claude Code 월 ₩27,000. 커피 6잔 값.
적용: ROI 4,000%+ 투자.
```

## 슬라이드 4
```
오해 #3: "개발자가 필요하다"
팩트: 2026년 AI 자동화는 한국어로 말하면 됩니다.
적용: 코딩 0줄. 한국어 1줄.
```

## 슬라이드 5
```
오해 #4: "AI가 일자리를 뺏는다"
팩트: AI는 반복 업무를 대체, 사람은 가치 있는 일로 이동.
적용: 해고 0명, 업그레이드 15명 (실제 사례).
```

## 슬라이드 6
```
오해 #5: "우리 데이터가 유출된다"
팩트: Claude는 대화 내용을 학습에 사용하지 않음.
적용: 민감 정보(주민번호 등)만 제외하면 안전.
```

## 슬라이드 7
```
오해 #6: "세팅이 복잡하다"
팩트: 첫 자동화 평균 세팅 시간 15분.
적용: 커피 한 잔 마시는 동안 완료.
```

## 슬라이드 8
```
오해 #7: "대기업만 쓸 수 있다"
팩트: 소규모가 더 유리 (빠른 결정, 즉시 적용, 체감 효과 큼).
적용: 1인 사업자도 15분 만에 시작 가능.
```
