## 슬라이드 2
```
ROI 기본 공식

ROI = (절감액 - 투자액) / 투자액 x 100%

예시:
월 절감: 300만원
월 투자: 50만원 (도구 구독 + 관리)

ROI = (300 - 50) / 50 x 100 = 500%

500% ROI = 1만원 투자 → 5만원 회수
```

## 슬라이드 3
```
Step 1: 현재 비용 계산

자동화 전에 들어가는 비용을 산출하세요.

인건비:
담당자 시급 x 해당 업무 시간 = 월 인건비

예시:
시급 2만원 x 주 10시간 x 4주 = 월 80만원

여기에 실수 비용, 지연 비용까지 더하면
실제 비용은 더 큽니다.
```

---

## 슬라이드 4
```
Step 2: 자동화 후 비용 계산

자동화에 들어가는 비용:

□ 도구 구독료 (월)
□ 초기 세팅 비용 (1회)
□ 유지보수 시간 (월)

예시:
Make 구독: 월 3만원
초기 세팅: 50만원 (외주 또는 자체)
유지보수: 월 2시간 x 시급 2만원 = 4만원

월 비용: 약 7만원
```

---

## 슬라이드 5
```
Step 3: 절감액 계산

절감액 = 현재 비용 - 자동화 후 비용

예시:
현재: 월 80만원
자동화 후: 월 7만원

월 절감액: 73만원
연 절감액: 876만원

초기 세팅비 50만원은 1개월 만에 회수.
```

---

## 슬라이드 6
```
Step 4: 숨은 절감 효과 추가

숫자로 바로 안 보이는 효과:

⏱️ 응답 속도 향상 → 고객 만족도 상승
📉 인적 실수 감소 → 클레임 비용 절감
📈 처리량 증가 → 매출 기회 확대
😌 직원 스트레스 감소 → 이직률 감소

이 효과까지 포함하면
실제 ROI는 계산보다 높습니다.
```

## 슬라이드 7
```
ROI 판단 기준

ROI 300% 이상 → 즉시 시작
ROI 100~300% → 시작 추천 (3개월 내 회수)
ROI 50~100% → 검토 필요
ROI 50% 미만 → 다른 업무부터

하우제로 기준:
ROI 300% 이상인 업무만 먼저 자동화합니다.
낮은 ROI는 나중에.
```
