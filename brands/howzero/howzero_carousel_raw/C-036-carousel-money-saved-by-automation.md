# C-036: 인스타 캐러셀 — "자동화로 절약한 금액 계산기"

> **유형**: 인스타 캐러셀 | **슬라이드**: 7장

## 슬라이드 1 (표지)
```
내가 자동화로 절약할 수 있는 금액
직접 계산해보세요
💰 저장 필수
```

## 슬라이드 2
```
Step 1: 반복 업무 시간 계산
하루에 반복 업무에 쓰는 시간: ___시간
예: 이메일 1시간 + CS 1.5시간 + 보고서 0.5시간 = 3시간
```

## 슬라이드 3
```
Step 2: 연간 시간 환산
하루 ___시간 × 250일(연 근무일) = ___시간/년
예: 3시간 × 250일 = 750시간/년
```

## 슬라이드 4
```
Step 3: 금액 환산
___시간 × 시급 ₩_____ = 연 절감액
예: 750시간 × ₩30,000 = ₩22,500,000/년
```

## 슬라이드 5
```
Step 4: 자동화 비용 차감
자동화 비용: ₩27,000 × 12개월 = ₩324,000/년
순 절감액: ₩22,500,000 - ₩324,000 = ₩22,176,000/년
ROI: 6,844%
```

## 슬라이드 6
```
실제 계산 예시:
| 반복 시간 | 시급 | 연 절감 | ROI |

| 4시간/일 | ₩50,000 | ₩49,676,000 | 15,226% |
```
