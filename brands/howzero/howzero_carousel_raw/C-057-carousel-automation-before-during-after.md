# C-057: 인스타 캐러셀 — "자동화 전/중/후 체크리스트"

> **유형**: 인스타 캐러셀 | **슬라이드**: 8장 | **톤**: 체계적+실용

---

## 슬라이드 1 (표지)
```
자동화 전 / 중 / 후
체크리스트

세팅만 하면 끝이 아닙니다

✅ 각 단계에서 확인할 것 총정리
```

---

## 슬라이드 2
```
🔵 자동화 전 (Before) — 준비

□ 자동화할 업무 1개 선정
□ 현재 프로세스 단계별 정리
□ 수동 처리 시간 측정
□ 예상 ROI 계산
□ 담당자 지정
□ 도구 선택 및 계정 생성

이 단계를 건너뛰면
자동화가 아니라 혼란이 시작됩니다.
```

---

## 슬라이드 3
```
🔵 자동화 전 — 설계

□ 입력 데이터 형식 확인
□ 출력 결과 형식 정의
□ 예외 상황 목록 작성
□ 에러 시 대응 프로세스 설계
□ 테스트 시나리오 준비

설계 없이 세팅하면
"왜 이상하게 작동하지?" 반복합니다.
```

---

## 슬라이드 4
```
🟡 자동화 중 (During) — 세팅

□ 도구 간 연동 설정
□ 트리거 조건 설정
□ 워크플로우 단계 구성
□ 테스트 데이터로 실행
□ 에러 처리 로직 확인
□ 알림 설정 (성공/실패)

테스트 없이 라이브 배포 = 사고.
반드시 테스트 데이터로 먼저 돌리세요.
```

---

## 슬라이드 5
```
🟡 자동화 중 — 테스트

□ 정상 케이스 3회 이상 테스트
□ 에러 케이스 테스트 (빈 데이터, 잘못된 형식)
□ 대량 데이터 테스트
□ 실행 시간 확인
□ 출력 결과 품질 확인

테스트 통과 기준:
정상 케이스 100% + 에러 케이스 대응 완료
```

---

## 슬라이드 6
```
🟢 자동화 후 (After) — 1주차

□ 실제 데이터로 운영 시작
□ 매일 실행 로그 확인
□ 출력 품질 샘플 체크
□ 팀 피드백 수집
□ 소소한 버그 수정

첫 1주일이 가장 중요합니다.
이때 잡지 않으면 나중에 더 큽니다.
```

---

## 슬라이드 7
```
🟢 자동화 후 — 안정화 (2주~)

□ 주간 점검 루틴 확립
□ 실제 ROI 측정 (예상 vs 실제)
□ 최적화 포인트 발굴
□ 문서화 (다음 자동화 참고용)
□ 다음 자동화 대상 선정

1개 성공하면 문서로 남기세요.
다음 건은 절반의 시간으로 끝납니다.
```

---
