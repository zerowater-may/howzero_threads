## 슬라이드 3
```
자동화 가능한 데이터 정리 업무

✅ 중복 데이터 제거
✅ 형식 통일 (날짜, 전화번호, 주소)
✅ 빈 칸 자동 채우기 / 표시
✅ 여러 시트 → 하나로 병합
✅ 이상값 탐지 및 플래그
✅ 카테고리 자동 분류

전부 사람이 할 필요 없습니다.
```

## 슬라이드 4
```
단계 1: 데이터 소스 파악

어디서 데이터가 들어오는지 정리하세요.

□ 구글 시트 / 엑셀
□ CRM (HubSpot, Salesforce 등)
□ 쇼핑몰 관리자 페이지
□ 이메일 / 폼 응답
□ ERP / 회계 프로그램

소스를 알아야 자동화 설계가 가능합니다.
```

---

## 슬라이드 5
```
단계 2: 정리 규칙 문서화

AI에게 시키려면 "규칙"이 있어야 합니다.

예시:
• 전화번호 → 010-XXXX-XXXX 형식
• 날짜 → YYYY-MM-DD 통일
• 중복 기준 → 이름+전화번호 동일 시 제거
• 빈 칸 → "미입력"으로 표시

규칙을 정하면 자동화는 쉽습니다.
```
