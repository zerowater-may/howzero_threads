## 슬라이드 1: AI 에이전트 활용 — 실전 5가지 시나리오 🤖 (저장 필수)
## 슬라이드 2: AI 에이전트는 단순 챗봇이 아니다 — 스스로 판단하고, 실행하고, 보고까지 하는 '디지털 직원'이다
## 슬라이드 3: 시나리오 1 — CS 자동 응대: 고객 문의 접수 → 유형 분류 → 답변 생성 → 에스컬레이션까지 자동. 응답 시간 90% 단축
## 슬라이드 4: 시나리오 2 — 리드 관리: 웹사이트 방문자 행동 분석 → 리드 스코어링 → CRM 자동 입력 → 영업팀 알림까지 원스톱
## 슬라이드 5: 시나리오 3 — 보고서 자동 생성: 매출·재고·마케팅 데이터 수집 → 주간/월간 보고서 자동 작성 → 슬랙 전송
## 슬라이드 6: 시나리오 4 — 채용 스크리닝: 이력서 접수 → 직무 적합도 분석 → 1차 필터링 → 면접 일정 조율까지 자동
## 슬라이드 7: 시나리오 5 — 재고 관리: 판매 데이터 분석 → 수요 예측 → 자동 발주 제안 → 공급업체 이메일 발송
