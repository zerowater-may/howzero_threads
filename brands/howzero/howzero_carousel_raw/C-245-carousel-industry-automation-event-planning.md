# C-245: 인스타 캐러셀 — "업종별 자동화 — 이벤트 기획 편"
> **유형**: 인스타 캐러셀 | **슬라이드**: 10장
## 슬라이드 1: 업종별 자동화 — 이벤트 기획 편 🎪 (저장 필수)
## 슬라이드 2: 이벤트 기획의 70%는 반복 업무다 — 참가자 관리, 이메일, 체크리스트. 전부 자동화 가능
## 슬라이드 3: 자동화 1 — 참가 신청 → 확인 메일 → 리마인드 시퀀스 자동화. 노쇼율 40% → 12%로 감소
## 슬라이드 4: 자동화 2 — 협력업체 커뮤니케이션 자동화. 일정·장소·요구사항을 한 번 입력하면 전체 벤더에 자동 전달
## 슬라이드 5: 자동화 3 — 참가자 데이터 수집 → 명찰·좌석배치·키트 자동 생성. 전날 밤샘 작업 제로
## 슬라이드 6: 자동화 4 — 이벤트 후 설문 자동 발송 + 감사 메일 + 사진 갤러리 공유. 후속 관리까지 자동
## 슬라이드 7: 자동화 5 — 이벤트 ROI 자동 산출. 비용 대비 참가자 수·만족도·리드 전환율 대시보드 자동 생성
## 슬라이드 8: 실전 데이터 — 이벤트 기획사 2곳 도입: 준비 시간 55% 단축, 참가자 만족도 18%↑, 반복 이벤트 셋업 80% 단축
## 슬라이드 9: 핵심 — 이벤트의 가치는 '경험'이다. 반복 업무를 자동화해서 경험 설계에 집중하자
