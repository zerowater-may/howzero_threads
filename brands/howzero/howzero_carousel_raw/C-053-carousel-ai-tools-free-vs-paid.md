# C-053: 인스타 캐러셀 — "AI 도구 무료 vs 유료 비교"

> **유형**: 인스타 캐러셀 | **슬라이드**: 8장 | **톤**: 비교 분석+실용

---

## 슬라이드 1 (표지)
```
AI 도구
무료 vs 유료 비교

무료로 충분한 것
유료가 필요한 것

💰 돈 아끼려다 시간 날리지 마세요
```

---

## 슬라이드 2
```
무료로 충분한 경우

✅ 개인 업무 보조 (요약, 번역, 초안)
✅ 간단한 데이터 정리 (소량)
✅ 프롬프트 연습·학습
✅ 아이디어 브레인스토밍
✅ 1일 사용량이 적은 경우

무료 도구도 잘 쓰면 월 10시간은 아낍니다.
```

---

## 슬라이드 3
```
유료가 필요한 경우

❗ 팀 단위로 사용
❗ API 연동 (자동화 워크플로우)
❗ 대량 데이터 처리
❗ 응답 속도·안정성이 중요한 업무
❗ 고객 대면 서비스 (챗봇 등)

비즈니스에 직결되면 유료가 답입니다.
무료 불안정성 = 고객 이탈 리스크.
```

---

## 슬라이드 4
```
텍스트 AI 비교

ChatGPT:
무료: GPT-4o mini (기본)
유료: GPT-4o + 고급 기능 / 월 $20

Claude:
무료: 일일 사용량 제한
유료: 더 긴 컨텍스트 + 빠른 응답 / 월 $20

비즈니스용이면?
→ 유료 + API 연동 추천
개인 학습용이면?
→ 무료로 충분
```
