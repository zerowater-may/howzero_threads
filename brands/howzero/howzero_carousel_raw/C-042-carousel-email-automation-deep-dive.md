# C-042: 인스타 캐러셀 — "이메일 자동화 심화 — 분류를 넘어서"

> **유형**: 인스타 캐러셀 | **슬라이드**: 8장 | **톤**: 심화 가이드

---

## 슬라이드 1 (표지)
```
이메일 자동화 심화

분류만으로 끝? ❌
여기서부터가 진짜입니다.

💡 이미 분류 자동화 하신 분만 보세요
```

---

## 슬라이드 2
```
Level 1: 분류 (이미 하고 있죠?)
· 긴급/일반/참조/스팸 자동 분류
· 절감: 하루 30분

→ 여기까지가 기본.
→ 이제 Level 2로.
```

---

## 슬라이드 3
```
Level 2: 자동 답변 초안

분류된 이메일 → AI가 답변 초안 생성
· 긴급 건: 즉시 답변 초안
· 일반 건: 오후에 일괄 답변 초안
· 사람은 검토+수정만

절감: 하루 추가 30분
```

---

## 슬라이드 4
```
Level 3: 자동 라우팅

이메일 내용 → 적절한 담당자에게 자동 전달
· 기술 문의 → 기술팀
· 결제 문의 → 경리팀
· 협업 제안 → 대표에게

절감: 하루 추가 15분 + 오분류 방지
```

---

## 슬라이드 5
```
Level 4: 자동 팔로업

답변 후 3일 무응답 → 자동 리마인드
"혹시 추가 궁금한 점 있으시면 알려주세요"

효과: 팔로업 누락 0%
전환율: 15~20% 상승
```

---

## 슬라이드 6
```
Level 5: 인사이트 추출

주간 이메일 분석:
· 가장 많은 문의 유형 TOP 3
· 새로 등장한 이슈
· 응답 시간 평균
· 고객 감정 트렌드

→ 비즈니스 인사이트가 됩니다
```

---

## 슬라이드 7
```
전체 파이프라인:

이메일 수신
↓ AI 분류
↓ AI 답변 초안
↓ 담당자 자동 배정
↓ 사람 검토 + 발송
↓ 팔로업 자동
↓ 주간 인사이트 리포트

이게 "이메일 시스템"입니다.
```
