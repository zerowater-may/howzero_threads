# C-034: 인스타 캐러셀 — "AI 준비도 점수 — 이렇게 해석하세요"

> **유형**: 인스타 캐러셀 | **슬라이드**: 8장

## 슬라이드 1 (표지)
```
AI 준비도 점수
이렇게 해석하세요
(진단 후 이 가이드 보세요)
```

## 슬라이드 2
```
0~20점: AI 초보 🔴
현재 상태: 거의 수동. AI 경험 없음.
추천 액션: 이메일 분류 자동화 1개부터
예상 효과: 월 10시간+ 절감
시작: howzero.com/ai-score
```

## 슬라이드 3
```
21~40점: 시작 단계 🟠
현재 상태: ChatGPT 써본 적은 있음. 업무 적용은 아직.
추천 액션: 가장 반복적인 업무 3개 자동화
예상 효과: 월 20시간+ 절감
```

## 슬라이드 4
```
41~60점: 활용 단계 🟡
현재 상태: 1~2개 자동화 사용 중.
추천 액션: 프로세스 전체 자동화 (한 업무의 시작→끝)
예상 효과: 월 30시간+ 절감
```

## 슬라이드 5
```
61~80점: 시스템 단계 🟢
현재 상태: 여러 자동화가 연결되어 작동.
추천 액션: 모니터링 + 최적화 + 팀 확산
예상 효과: 월 40시간+ 절감
```

## 슬라이드 6
```
81~100점: 마스터 단계 🔵
현재 상태: AI가 일상의 일부. 대부분 자동.
추천 액션: 고급 자동화 + AI 전략 수립
예상 효과: 업무 방식 자체의 전환
```

## 슬라이드 7
```
중요한 건 점수가 아닙니다.
"다음 단계로 가는 것"입니다.

0점이든 50점이든
다음 단계로 가는 데 필요한 건
15분과 종이 한 장입니다.
```

## 슬라이드 8 (CTA)
```
내 AI 준비도 점수 확인:
📊 howzero.com/ai-score (3분)

점수가 낮아도 괜찮습니다.
같이 올려드립니다:
🔍 howzero.com/audit

하우제로 | "점수도 제로부터."
```
