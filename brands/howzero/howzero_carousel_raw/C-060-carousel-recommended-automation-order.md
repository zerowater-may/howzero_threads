```

자동화 순서

"뭐부터 해야 하나요?"

```

---

## 슬라이드 2
```
왜 순서가 중요한가?

자동화를 실패하는 가장 큰 이유:
순서를 무시하고 어려운 것부터 시작

쉬운 것부터 성공 경험을 쌓아야
어려운 것도 할 수 있습니다.

난이도 순서대로.
이게 하우제로가 10억을 만든 방식입니다.
```

---

## 슬라이드 3
```
1순위: 알림 자동화

난이도: ★☆☆☆☆
효과: 즉시 체감

예시:
• 주문 접수 → 슬랙 알림
• 재고 부족 → 담당자 알림
• 문의 접수 → 이메일 알림

세팅 시간: 30분~1시간
도구: Zapier, Make

가장 쉽고, 가장 빨리 효과를 느낍니다.
```

---

## 슬라이드 4
```
2순위: 데이터 수집·정리 자동화

난이도: ★★☆☆☆
효과: 주 3~5시간 절감

예시:
• 여러 시트 데이터 → 하나로 병합
• 폼 응답 → 구글 시트 자동 입력
• 주문 데이터 → 일간 리포트 자동 생성

세팅 시간: 2~4시간
도구: Google Apps Script, Make

데이터가 깨끗해야 다음 자동화가 가능합니다.
```

---

## 슬라이드 5
```
3순위: 커뮤니케이션 자동화

난이도: ★★★☆☆
효과: CS 부담 50% 감소

예시:
• FAQ 자동 응답 챗봇
• 예약 확인 자동 발송
• 리뷰 요청 자동 발송
• 결제 완료 안내 자동화

세팅 시간: 1~2일
도구: 챗봇 빌더, Make + 알림톡

고객 접점을 자동화하면 매출에 직결됩니다.
```

---

## 슬라이드 6
```
4순위: 보고·분석 자동화

난이도: ★★★☆☆
효과: 보고서 작성 70% 절감

예시:
• 주간 매출 리포트 자동 생성
• 광고 성과 자동 요약
• 경쟁사 가격 모니터링 리포트
• KPI 대시보드 자동 업데이트

세팅 시간: 1~3일
도구: Google Sheets + Claude API, Make

의사결정 속도가 빨라집니다.
```

---

## 슬라이드 7
```
5순위: 워크플로우 자동화

난이도: ★★★★☆
효과: 프로세스 전체 자동화

예시:
• 주문 → 결제 → 재고 → 배송 → 알림 전체 연결
• 채용 → 서류 검토 → 면접 조율 → 합격 통보
• 리드 유입 → 스코어링 → 영업팀 배정

세팅 시간: 1~2주
도구: Make, n8n

1~4순위 경험이 있어야 가능합니다.
```
