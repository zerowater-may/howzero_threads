# C-049: 인스타 캐러셀 — "비개발자가 AI 전문가처럼 보이는 법"

> **유형**: 인스타 캐러셀 | **슬라이드**: 8장 | **톤**: 직설+실용

---

## 슬라이드 1 (표지)
```
비개발자가
AI 전문가처럼 보이는 법

코딩 1줄 몰라도 됩니다

💡 아는 척이 아니라 진짜 쓸 줄 아는 사람
```

---

## 슬라이드 2
```
1. 용어를 정확히 쓰세요

❌ "그 AI 뭐시기... 챗봇 같은 거"
✅ "LLM 기반 챗봇으로 CS 자동화하려고 합니다"

외울 용어:
API, 워크플로우, 프롬프트, LLM,
노코드, RPA, 파인튜닝

7개만 알면 미팅에서 밀리지 않습니다.
```

---

## 슬라이드 3
```
2. 직접 써본 사례를 말하세요

❌ "AI가 좋다더라"
✅ "저번 달에 Claude로 주간 보고서 초안 만들었더니
    작성 시간이 2시간 → 20분으로 줄었습니다"

직접 써본 경험 1개 >
뉴스에서 본 이야기 100개.
```

---

## 슬라이드 4
```
3. 한계를 아는 사람이 진짜 전문가

"AI 만능이에요" → 초보 티
"AI는 이건 잘하고, 이건 못해요" → 전문가 티

AI가 못하는 것:
• 100% 정확한 사실 보장
• 최신 정보 실시간 반영
• 감정적 판단이 필요한 의사결정
• 규정·법률 최종 확인
```

---

## 슬라이드 5
```
4. 결과를 숫자로 보여주세요

❌ "자동화하니까 편해졌어요"
✅ "월 40시간 절감, 인건비 200만원 절약"

Before/After를 숫자로 말하면
대표도, 팀장도, 투자자도 귀 기울입니다.
```

---

## 슬라이드 6
```
5. 도구 3개만 깊게 파세요

100개 도구 이름 아는 사람보다
3개 도구를 실무에서 쓰는 사람이 전문가.

추천 조합:
• ChatGPT 또는 Claude (텍스트 AI)
• Make 또는 Zapier (자동화)
• Notion 또는 Google Sheets (데이터 관리)

이 3개면 웬만한 업무 자동화 가능합니다.
```

---

## 슬라이드 7
```
6. 트렌드를 읽되, 쫓지는 마세요

매주 새로운 AI 도구가 나옵니다.
전부 따라가면 지칩니다.

✅ 뉴스레터 2개만 구독
✅ 한 달에 1개만 새로 테스트
✅ 기존 도구 활용도를 먼저 끌어올리기

유행 쫓는 사람 vs 실행하는 사람
후자가 전문가입니다.
```
