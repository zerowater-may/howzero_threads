## 슬라이드 2
```
왜 설득이 어려운가?

상사/대표가 거부하는 이유는 3가지:

1. "잘 되고 있는데 왜 바꿔?"
   → 현상 유지 편향

2. "비용이 얼마나 드는데?"
   → 비용 두려움

3. "그런 거 도입하면 일자리 없어지는 거 아냐?"
   → 인력 불안

이 3가지만 해결하면 됩니다.
```

---

## 슬라이드 3
```
설득 전략 1: 비용이 아니라 "낭비"를 보여줘라

❌ "자동화 도구에 월 10만 원이면 됩니다"
✅ "지금 매달 300만 원을 낭비하고 있습니다"

현재 반복 업무에 쓰는 시간 × 시급
= 매달 버리는 돈

이 숫자를 먼저 보여주세요.
도구 비용은 그 다음입니다.
```

---

## 슬라이드 4
```
설득 전략 2: 작게 시작하자고 제안하라

❌ "전사 시스템을 바꿉시다"
✅ "알림 자동화 1개만 테스트합시다"

대표는 "큰 변화"를 두려워합니다.

30분이면 세팅되는 것 1개.
비용 0원. 리스크 0.

이걸 거부할 대표는 없습니다.
```

## 슬라이드 6
```
설득 전략 4: 일자리 불안을 해소하라

❌ "사람이 할 필요 없는 일입니다"
✅ "사람이 더 가치 있는 일을 할 수 있습니다"

자동화 = 해고가 아니라
자동화 = 재배치

반복 업무에서 해방된 직원이
영업, 고객 관리, 기획에 집중하면
매출이 올라갑니다.
```

## 슬라이드 7
```
설득 전략 5: 1장짜리 제안서를 만들어라

대표는 바쁩니다.
PPT 30장은 안 봅니다.

A4 1장에 넣을 것:
• 현재 낭비되는 비용 (숫자)
• 자동화 대상 업무 1개
• 예상 절감 효과 (숫자)
• 필요 비용 (0원 또는 최소 금액)
• 테스트 기간 (2주)

숫자가 설득합니다.
감정이 아니라.
```
