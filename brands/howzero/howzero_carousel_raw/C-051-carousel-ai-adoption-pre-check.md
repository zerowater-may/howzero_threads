# C-051: 인스타 캐러셀 — "AI 도입 전 체크해야 할 5가지"

> **유형**: 인스타 캐러셀 | **슬라이드**: 7장 | **톤**: 직설+체크리스트

---

## 슬라이드 1 (표지)
```
AI 도입 전
체크해야 할 5가지

이거 안 하고 AI 도입하면
돈만 날립니다

⚠️ 90%가 3번을 빠뜨립니다
```

---

## 슬라이드 2
```
체크 1: 해결할 문제가 명확한가?

❌ "AI 도입해야 할 것 같아서"
✅ "CS 응답 시간을 3시간 → 30분으로 줄이고 싶다"

문제가 불명확하면 도구만 사고
결과는 없습니다.

"왜 AI를 쓰려는가?"
이 질문에 1문장으로 답할 수 있어야 합니다.
```

---

## 슬라이드 3
```
체크 2: 데이터가 있는가?

AI는 마법이 아닙니다.
데이터가 있어야 작동합니다.

확인 사항:
□ 고객 문의 기록이 정리되어 있는가?
□ 매출·주문 데이터가 디지털로 존재하는가?
□ 업무 프로세스가 문서화되어 있는가?

데이터가 없으면 AI 도입이 아니라
데이터 수집이 먼저입니다.
```

---

## 슬라이드 4
```
체크 3: 현재 프로세스를 아는가?

⚠️ 가장 많이 빠뜨리는 항목

"대충 이렇게 하고 있어요"는 안 됩니다.

□ 업무 순서를 단계별로 적을 수 있는가?
□ 각 단계에 걸리는 시간을 아는가?
□ 누가 어떤 역할을 하는지 명확한가?

프로세스를 모르면
자동화할 대상도 모릅니다.
```

---

## 슬라이드 5
```
체크 4: 예산과 기대 효과가 현실적인가?

"무료 AI로 전부 해결하겠다" → 비현실적
"AI 도입하면 바로 10배 성장" → 과장

현실적 기대:
• 노코드 도구: 월 5~30만원
• 자동화 세팅: 1~4주
• 첫 ROI 체감: 1~3개월

작게 시작하고 점진적으로 확대하세요.
```

---

## 슬라이드 6
```
체크 5: 담당자가 있는가?

AI 도구를 사놓고 아무도 안 쓰면
그냥 비용입니다.

□ 자동화를 세팅하고 관리할 담당자가 있는가?
□ 그 담당자가 시간을 투자할 수 있는가?
□ 외부 도움이 필요한지 판단했는가?

도구가 문제가 아닙니다.
쓰는 사람이 핵심입니다.
```
