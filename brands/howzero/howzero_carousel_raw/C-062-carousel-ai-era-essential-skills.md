# C-062: 인스타/링크드인 캐러셀 — "AI 시대 필수 스킬 5가지"

> **유형**: 캐러셀 (8장) | **채널**: 인스타 + 링크드인 | **목적**: 저장 유도 + 자기계발 동기

---

## 슬라이드 1 (표지)
```
AI 시대
살아남으려면 필요한
스킬 5가지

코딩이 아닙니다.

→ 스와이프
```

---

## 슬라이드 2
```
1. 프롬프트 설계 능력

AI를 쓰는 건 누구나 합니다.
AI를 "잘" 쓰는 건 소수입니다.

핵심은 질문의 질.
구체적 맥락 + 원하는 출력 형식
+ 제약 조건을 줄 수 있는 능력.

이건 코딩이 아니라 "사고력"입니다.
```

---

## 슬라이드 3
```
2. 프로세스 분석 능력

자동화의 80%는 기술이 아니라
프로세스 이해입니다.

우리 회사의 업무 흐름을
종이 한 장에 그릴 수 있는가?

병목이 어디인지 찾을 수 있는가?

이 능력 하나로 AI 컨설팅 비용
수천만 원을 아낄 수 있습니다.
```

---

## 슬라이드 4
```
3. 데이터 해석 능력

AI가 뽑아준 데이터를
비즈니스 의사결정으로 바꾸는 능력.

숫자를 보고 "그래서 뭘 해야 하는데?"
라고 말할 수 있어야 합니다.

통계 전공이 아니어도 됩니다.
트렌드와 이상치만 읽으면 충분합니다.
```

---

## 슬라이드 5
```
4. 도구 조합 능력

최고의 도구 1개 < 적절한 도구 3개 조합

Make + Google Sheets + Claude API
이 조합만으로도 월 200시간 절감 가능.

도구를 "깊게" 파는 것보다
"넓게" 연결하는 능력이 더 중요합니다.
```

---

## 슬라이드 6
```
5. 검증 및 피드백 능력

AI는 틀립니다. 자주 틀립니다.

AI 결과물을 검증하고
피드백을 주어 개선하는 능력.

"AI가 알아서 해주겠지"가 아니라
"AI가 한 걸 내가 확인한다."

이 차이가 성과의 차이입니다.
```

---

## 슬라이드 7 (요약)
```
AI 시대 필수 스킬 5가지 요약

1. 프롬프트 설계 — 좋은 질문
2. 프로세스 분석 — 업무 흐름 파악
3. 데이터 해석 — 숫자를 행동으로
4. 도구 조합 — 연결하는 능력
5. 검증/피드백 — AI 결과물 확인

공통점: 전부 코딩과 무관합니다.
전부 "비즈니스 감각"입니다.
```
