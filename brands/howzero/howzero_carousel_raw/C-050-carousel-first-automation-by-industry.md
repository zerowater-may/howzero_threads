## 슬라이드 2
```
🛒 이커머스 / 쇼핑몰

첫 자동화: 주문 후 자동 리뷰 요청

왜?
→ 리뷰 수 = 매출과 직결
→ 수동 요청 시 누락률 70% 이상

세팅:
주문 완료 3일 후 → 카카오/문자 자동 발송
도구: Zapier + 쇼핑몰 API

기대 효과: 리뷰 수 2~3배 증가
```

## 슬라이드 3
```
🏥 병원 / 클리닉

첫 자동화: 예약 확인·리마인드 자동 발송

왜?

→ 리마인드만으로 노쇼 50% 감소

세팅:
예약 1일 전 + 2시간 전 자동 문자/알림톡
도구: 예약 시스템 + Make

기대 효과: 노쇼 50% 감소 → 매출 직결
```

## 슬라이드 4
```
📐 부동산 / 중개

첫 자동화: 매물 문의 자동 응답

왜?
→ 문의 후 5분 안에 응답해야 계약 확률 상승
→ 현장에 있으면 전화를 못 받음

세팅:
문의 접수 → 매물 정보 + 방문 가능 일정 자동 회신
도구: 챗봇 + Google Form 연동

기대 효과: 응답 속도 90% 개선
```

## 슬라이드 5
```
🍽️ 음식점 / F&B

첫 자동화: 재고 부족 자동 알림

왜?
→ 재료 떨어져서 메뉴 품절 = 매출 손실
→ 수동 체크는 빠뜨리기 쉬움

세팅:
재고 수량 임계치 이하 → 슬랙/문자 자동 알림
도구: POS 연동 + 알림 자동화

기대 효과: 품절 사고 80% 감소
```

## 슬라이드 6
```
💼 컨설팅 / 전문서비스

첫 자동화: 미팅 노트 자동 요약

왜?
→ 미팅 후 정리에 30분~1시간 소요
→ 내용 누락·왜곡 발생

세팅:
미팅 녹음 → AI 자동 전사 + 요약 + 액션 아이템 추출
도구: Otter.ai / Clova Note + Claude

기대 효과: 미팅 정리 시간 80% 절감
```

## 슬라이드 7
```
🎓 교육 / 학원

숙제 미제출 → 리마인드 자동 발송
도구: Google Sheets + 알림톡

기대 효과: 학부모 만족도 상승 + 재등록률 증가
```
