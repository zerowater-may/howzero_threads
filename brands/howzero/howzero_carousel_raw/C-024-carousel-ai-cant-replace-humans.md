# C-024: 인스타 캐러셀 — "AI가 절대 못 하는 5가지 (사람만 할 수 있는 것)"

> **유형**: 인스타 캐러셀 | **슬라이드**: 8장 | **톤**: 균형 잡힌 관점

---

## 슬라이드 1 (표지)
```
AI가 절대 못 하는 5가지

(사람만 할 수 있는 것)

💡 자동화 전문가가 말하는 AI의 한계
```

---

## 슬라이드 2
```
#1 비전을 세우는 것

AI: 데이터 분석, 트렌드 정리 → 잘함
AI: "그래서 우리 회사가 어디로 가야 하는가?" → 못함

비전은 사장님의 경험, 직관, 가치관에서 나옵니다.
AI는 보조할 수 있지만 대체할 수 없습니다.
```

---

## 슬라이드 3
```
#2 고객과 신뢰를 쌓는 것

AI: CS 80% 자동 처리 → 잘함
AI: 고객이 "이 사람이니까 믿고 산다" → 못함

거래는 AI가 처리하지만
관계는 사람이 만듭니다.
핵심 고객 미팅은 직접 하세요.
```

---

## 슬라이드 4
```
#3 팀을 이끄는 것

AI: 업무 분배, 일정 관리 → 잘함
AI: "오늘 기분이 안 좋은 팀원 챙기기" → 못함

리더십은 감정, 공감, 동기부여입니다.
이건 사람만 할 수 있습니다.
```

---

## 슬라이드 5
```
#4 "왜?"를 묻는 것

AI: "매출이 10% 떨어졌습니다" → 분석 잘함
AI: "왜 우리가 이 사업을 해야 하는가?" → 못함

전략적 질문, 존재 이유, 가치 판단.
이건 사람의 영역입니다.
```

---

## 슬라이드 6
```
#5 실패에서 의미를 찾는 것

AI: 실패 데이터 분석 → 잘함
AI: "이 실패가 나에게 어떤 의미인가?" → 못함

실패를 겪고, 의미를 부여하고,
다시 일어서는 건 사람만 할 수 있습니다.
이게 기업가 정신입니다.
```

---

## 슬라이드 7
```
결론:

AI에 맡길 것:
✅ 반복 업무, 데이터 처리, 분류, 보고서, CS

사람이 할 것:
🎯 비전, 관계, 리더십, 전략, 의미

AI는 "어떻게(How)"를 잘합니다.
사람은 "왜(Why)"를 결정합니다.

둘 다 필요합니다.
```
