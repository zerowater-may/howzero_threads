> **유형**: 캐러셀 7장 | **앵글**: 역발상 → "그래서 사람이 더 중요" → 신뢰

## Slide 1
```
AI가 못하는 것
5가지

"AI 만능론에 속지 마세요.
 이건 사람만 할 수 있습니다."

→
```

## Slide 2
```
❌ 1. 고객의 감정을 읽는 것

AI는 "환불 요청"이라고 분류합니다.
근데 "화가 났는지, 실망했는지, 급한지"는 모릅니다.

진짜 CS는 분류가 아니라 공감입니다.
AI가 분류하고 → 사람이 공감하세요.
```

## Slide 3
```
❌ 2. 전략적 결정

"이 시장에 진출할까 말까?"
"가격을 올릴까 내릴까?"
"이 파트너와 손잡을까?"

AI는 데이터를 정리해줍니다.
결정은 사람이 합니다.
```

## Slide 4
```
❌ 3. 신뢰 관계 구축

거래처 사장님과 밥 먹으면서
쌓는 관계.

고객이 "이 사람이니까" 하고
다시 찾아오는 신뢰.

AI는 이메일은 잘 쓰지만,
악수는 못합니다.
```

## Slide 5
```
❌ 4. 창의적 기획

"이런 상품은 어때?"
"이런 마케팅 해볼까?"
"이런 서비스를 만들면?"

AI는 "비슷한 것"은 잘 만듭니다.
"처음 보는 것"은 사람이 만듭니다.
```

## Slide 6
```
❌ 5. 책임

AI가 실수하면 → 누가 책임?
AI가 이상한 답변 보내면 → 누가 사과?

최종 책임은 항상 사람에게 있습니다.
그래서 AI 답변을 "검토"하는 과정이 필수.
```

## Slide 7 (CTA)
```
그래서 AI는 "대체"가 아니라 "분업"입니다.

AI가 할 일: 분류, 복붙, 정리, 반복
사람이 할 일: 판단, 공감, 창의, 관계

이 구분이 명확하면
AI는 최고의 팀원이 됩니다.

📊 우리 회사에서 AI한테 맡길 것은?
→ 프로필에서 무료 진단

하우제로
"AI는 도구. 주인은 사람."
```

## 캡션
```
AI가 다 해준다고요?
아닙니다. AI가 못하는 것 5가지.

1. 고객 감정 읽기
2. 전략적 결정
3. 신뢰 관계 구축
4. 창의적 기획
5. 책임

AI = 분류, 복붙, 정리, 반복
사람 = 판단, 공감, 창의, 관계
```
