# C-055: 인스타 캐러셀 — "고객 여정별 자동화 맵"

> **유형**: 인스타 캐러셀 | **슬라이드**: 9장 | **톤**: 전략+시각화

---

## 슬라이드 1 (표지)
```
고객 여정별 자동화 맵

인지 → 관심 → 구매 → 재구매

🗺️ 각 단계에서 자동화할 것 총정리
```

---

## 슬라이드 2
```
고객 여정이란?

고객이 우리 브랜드를 알게 되고
구매하고, 다시 돌아오는 전체 과정.

인지 → 관심 → 비교 → 구매 → 사용 → 재구매

각 단계마다 자동화할 수 있는 업무가 있습니다.
전부 사람이 할 필요 없습니다.
```

---

## 슬라이드 3
```
1단계: 인지 (Awareness)

고객이 우리를 처음 아는 단계

자동화:
□ SNS 콘텐츠 자동 예약·게시
□ 블로그 SEO 콘텐츠 초안 생성
□ 광고 성과 자동 리포트
□ 경쟁사 콘텐츠 모니터링

목표: 더 많은 사람에게 노출
자동화 효과: 콘텐츠 생산량 3배 증가
```

---

## 슬라이드 4
```
2단계: 관심 (Interest)

고객이 우리에게 관심을 갖는 단계

자동화:
□ 리드 마그넷 자동 발송 (PDF, 가이드)
□ 웰컴 이메일 시퀀스
□ 문의 자동 응답 (챗봇)
□ 리드 스코어링 자동 분류

목표: 관심 → 연락처 확보
자동화 효과: 리드 전환율 2배 향상
```

---

## 슬라이드 5
```
3단계: 비교·검토 (Consideration)

고객이 우리 vs 경쟁사를 비교하는 단계

자동화:
□ 맞춤형 제안서 자동 생성
□ 사례 연구 자동 발송
□ FAQ 챗봇 응답
□ 리마인드 이메일 (장바구니 이탈)

목표: 우리를 선택하게 만들기
자동화 효과: 이탈률 40% 감소
```

---

## 슬라이드 6
```
4단계: 구매 (Purchase)

고객이 결제하는 단계

자동화:
□ 결제 확인 자동 안내
□ 주문 처리 워크플로우
□ 배송 추적 자동 알림
□ 세금계산서 자동 발행

목표: 매끄러운 구매 경험
자동화 효과: CS 문의 50% 감소
```

---

## 슬라이드 7
```
5단계: 사용·유지 (Retention)

고객이 제품/서비스를 쓰는 단계

자동화:
□ 온보딩 가이드 자동 발송
□ 사용 팁 정기 이메일
□ 만족도 조사 자동 발송
□ 불만 감지 시 에스컬레이션

목표: 이탈 방지 + 만족도 유지
자동화 효과: 고객 유지율 30% 향상
```

---

## 슬라이드 8
```
6단계: 재구매·추천 (Advocacy)

고객이 다시 사고, 추천하는 단계

자동화:
□ 재구매 리마인드 자동 발송
□ 리뷰 요청 자동화
□ 추천인 보상 자동 처리
□ VIP 고객 맞춤 오퍼

목표: 충성 고객 + 자연 추천
자동화 효과: 재구매율 25% 향상
```
