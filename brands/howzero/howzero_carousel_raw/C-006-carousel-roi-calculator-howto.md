## Slide 2
```
📐 ROI 공식

ROI = (절감액 - 투자액) / 투자액 × 100%

예시:
절감: 월 ₩240만 (연 ₩2,880만)
투자: ₩200만 (1회)
ROI = (2,880 - 200) / 200 × 100
    = 1,340%

"₩200만 쓰고 ₩2,880만 아끼는 겁니다"
```

## Slide 3
```
📊 절감액 계산법

절감액 = 자동화 전 비용 - 자동화 후 비용

자동화 전:
  반복 업무 시간 × 시급 × 근무일
  = 4시간 × ₩25,000 × 22일
  = 월 ₩2,200,000

자동화 후:
  검토 시간 × 시급 × 근무일
  = 30분 × ₩25,000 × 22일
  = 월 ₩275,000

절감: ₩1,925,000/월
```

## Slide 4
```
⏱️ 투자 회수 기간

투자 회수 = 투자액 / 월 절감액

예시:
₩200만 / ₩192.5만 = 1.04개월

→ "1달이면 본전. 2달째부터 순이익."
```

## Slide 5
```
❌ ROI 계산 실수 TOP 3

1. "시간 절감"만 계산하고 "기회비용" 빠뜨림
   → 아낀 시간에 할 수 있는 매출 활동도 포함

2. 연간이 아니라 월간만 계산
   → 자동화는 매달 절감. 연 단위로 봐야 진짜 가치

3. "세팅 시간"을 투자에 안 넣음
   → 세팅에 쓴 시간도 비용. (하지만 15분이니까 무시 가능 😄)
```
