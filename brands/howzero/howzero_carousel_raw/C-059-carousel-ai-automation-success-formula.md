# C-059: 인스타 캐러셀 — "AI 자동화 성공 공식 3가지"

> **유형**: 인스타 캐러셀 | **슬라이드**: 7장 | **톤**: 인사이트+직설

---

## 슬라이드 1 (표지)
```
AI 자동화 성공 공식 3가지

수백 개 기업을 봤습니다.
성공하는 곳은 이 3가지가 다릅니다.

🎯 1개라도 빠지면 실패합니다
```

---

## 슬라이드 2
```
공식 1: 프로세스 먼저, 도구 나중에

실패하는 기업:
"이 AI 도구 좋다더라" → 구매 → 어디에 쓰지?

성공하는 기업:
"이 업무가 비효율적이다" → 프로세스 정리 → 맞는 도구 선택

순서가 바뀌면 돈만 날립니다.

성공 공식:
문제 정의 → 프로세스 맵 → 도구 선택
```

---

## 슬라이드 3
```
공식 2: 작게 시작, 빠르게 증명

실패하는 기업:
"전사적 AI 전환 프로젝트" → 6개월 → 성과 없음

성공하는 기업:
"CS 자동 응답 1개" → 2주 세팅 → 바로 효과 체감

첫 자동화는 2주 안에 끝낼 수 있는 것으로.

성공 공식:
Low-hanging Fruit 1개 → 2주 내 세팅 → ROI 증명
```

---

## 슬라이드 4
```
공식 3: 측정하고 반복

실패하는 기업:
세팅 → 방치 → "자동화 별로인데?"

성공하는 기업:
세팅 → 측정 → 개선 → 확대

측정 지표:
• 절감 시간 (Before vs After)
• 에러율 변화
• 비용 절감액
• 고객 만족도

성공 공식:
세팅 → 1주 모니터링 → 최적화 → 다음 자동화
```

---

## 슬라이드 5
```
3가지 공식 정리

1️⃣ 프로세스 먼저, 도구 나중에
   문제 → 프로세스 → 도구

2️⃣ 작게 시작, 빠르게 증명
   1개 → 2주 → ROI 증명

3️⃣ 측정하고 반복
   세팅 → 측정 → 개선 → 확대

이 3가지를 지킨 기업은
예외 없이 성과를 냈습니다.
```

---

## 슬라이드 6
```
실제 사례

한 이커머스 기업:

1️⃣ 문제: CS 응답 평균 4시간
2️⃣ 첫 자동화: FAQ 자동 응답 챗봇 (2주 세팅)
3️⃣ 결과: 응답 시간 4시간 → 30초

다음 스텝:
리뷰 자동 요청 → 재고 알림 → 보고서 자동화

1개 성공이 전체를 바꿨습니다.
```
