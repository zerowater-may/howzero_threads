# 리드 마그넷 CTA 템플릿 라이브러리

> 채널별, 맥락별 CTA 카피. 복사해서 바로 사용 가능.

---

## 유튜브 숏폼 (끝 자막 + 음성)

### 뉴스레터 구독 유도
```
"매주 화요일, 비개발자 AI 자동화 꿀팁을 보내드립니다.
프로필 링크에서 뉴스레터 구독하세요."
```

### 무료 오딧 유도
```
"이거 직접 해보고 싶으시면, 무료 AI 오딧 30분 신청하세요.
프로필 링크에서 예약 가능합니다. 세일즈 없습니다."
```

### 미니코스 유도
```
"3일 만에 첫 자동화 만드는 무료 미니코스.
프로필 링크에서 이메일 입력하면 바로 시작됩니다."
```

## 유튜브 롱폼 (영상 중간 + 설명란)

### 템플릿 리드마그넷
```
"영상에서 보여드린 이 자동화 템플릿, 무료로 드립니다.
설명란 첫 번째 링크에서 이메일 입력하시면 즉시 발송됩니다."
```

### 영상 끝 CTA
```
"이 영상이 도움이 됐다면, 매주 이런 인사이트를 이메일로 보내드리는
하우제로 위클리를 구독해주세요. 설명란에 링크 있습니다."
```

### 고정 댓글
```
📩 이 영상에서 사용한 자동화 템플릿 (무료):
👉 [뉴스레터 구독 링크]

🔥 무료 AI 오딧 30분 (세일즈 없음):
👉 [오딧 신청 링크]
```
