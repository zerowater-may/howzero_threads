안녕하세요, 하우제로입니다.

"영업 이메일 100통 보냈는데 답이 3개입니다."

이런 말씀 많이 하시죠. 답장률 3%면 양호한 편이에요. 대부분 1% 이하입니다. 왜냐면 받는 사람 입장에서 스팸이니까요.

근데 같은 100통이라도, 각 이메일이 그 사람만을 위해 쓴 것처럼 느껴지면? 답장률이 15~25%까지 올라갑니다.

문제는 100통을 전부 개인화하면 하루가 끝나죠. AI로 합니다.

---

🍳 **이번 주 레시피: 영업 이메일 자동화**

**난이도**: ★★★☆☆ (중급)
**세팅 시간**: 20분
**효과**: 답장률 3% → 18%

---

📋 **재료**

- Claude Code (월 $20)
- Google Sheets (리드 리스트)
- 이메일 발송 도구 (Gmail, 스티비 등)

---

🔧 **만드는 법**

**Step 1: 리드 리서치 자동화 (10분)**

```
리드 리스트의 각 회사에 대해 자동 리서치해줘:

1. 회사 웹사이트 → 주요 서비스/제품
2. 최근 뉴스 (채용, 투자, 신제품 등)
3. LinkedIn → 담당자 최근 포스트
4. 업종별 자동화 니즈 예측

결과를 Google Sheets에 정리.
```

**Step 2: 개인화 이메일 자동 생성 (5분)**

```
리서치 결과를 기반으로 각 리드에게 맞춤 이메일을 작성해줘.

규칙:
- 첫 문장에 상대방 회사/상황에 대한 구체적 언급
- 우리가 해결할 수 있는 문제 1가지만 제시
- 성과 데이터 1개 포함 (예: "유사 기업 매출 30% 향상")
- 3줄 이내. 길면 안 읽습니다
- CTA는 질문형 (예: "15분 통화 가능하신가요?")
- 톤: 영업하는 느낌 없이, 진짜 도움을 주려는 느낌

절대 하지 말 것:
- "귀사의 무궁한 발전을 기원합니다" 같은 형식적 표현
- 첨부파일 (스팸 필터 발동)
- 한 이메일에 여러 CTA
```

**Step 3: 팔로업 시퀀스 자동화 (5분)**

```
이메일 발송 후 자동 팔로업:

- 3일 후 미답변: "바쁘신 걸 알지만" + 짧은 가치 제안 1줄
- 7일 후 미답변: 관련 사례/콘텐츠 링크 공유
- 14일 후 미답변: "타이밍이 안 맞으시면 알려주세요" + 마지막 연락

답변 오면 → 자동으로 팔로업 중단 + CRM 업데이트
3회 후 미답변 → 3개월 후 재접촉 리스트로 이동
```
