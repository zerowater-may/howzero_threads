안녕하세요, 하우제로입니다.

SaaS 월간 이탈률(churn rate)이 8%였습니다.

매달 100명 중 8명이 떠나는 구조. 1년이면 고객의 거의 전부가 바뀝니다. 이걸 신규 고객으로 메우려면 마케팅비가 끝없이 늘어납니다. 새 고객 1명 유치 비용이 기존 고객 유지 비용의 5배라는 건 다들 아시잖아요.

이탈률을 8%에서 3.5%로 줄였습니다. 자동화로.

---

🍳 **이번 주 레시피: 구독 이탈 방지 자동화**

**난이도**: ★★★☆☆ (중급)
**세팅 시간**: 5시간
**효과**: 이탈률 50% 이상 감소

---

🔧 **자동화 구조**

**1. 이탈 신호 감지**

```
행동 기반 이탈 예측:
→ 로그인 빈도 50% 이상 감소 → 경고
→ 핵심 기능 미사용 2주 이상 → 경고
→ 고객 지원 티켓 3회 이상 (불만 누적) → 경고
→ 결제 실패 1회 이상 → 즉시 알림
→ 경쟁사 비교 페이지 방문 → 경고
→ 리스크 점수 자동 산출 (0~100)
```

**2. 단계별 자동 개입**

```
리스크 점수별 대응:
→ 30~49점 (주의): 가치 리마인드 이메일 (활용 팁, 성공 사례)
→ 50~69점 (위험): 담당자 자동 배정 + 1:1 콜 제안
→ 70점 이상 (긴급): CS 리더에게 즉시 알림 + 특별 혜택 제안 권한
→ 구독 취소 페이지 진입 시: 실시간 팝업 (사유 선택 + 대안 제시)
```

**3. 취소 방어 플로우**

```
취소 프로세스 최적화:
→ 취소 사유 선택 (필수)
→ 사유별 맞춤 대안 자동 제시:
  - "너무 비싸요" → 할인 플랜 또는 일시 정지 제안
  - "잘 안 써요" → 맞춤 온보딩 세션 제안
  - "기능이 부족해요" → 로드맵 공유 + 피드백 반영 약속
  - "다른 서비스로" → 경쟁사 대비 차별점 요약
→ 최종 취소 시 "언제든 돌아오세요" + 30일 데이터 보관 안내
```

⚠️ **주의사항**

- 취소를 어렵게 만들지 마세요. 다크 패턴은 단기적으로 이탈을 막지만 장기적으로 브랜드를 망칩니다
- 할인은 최후의 수단으로. 할인으로 잡은 고객은 다음에도 할인을 기대합니다
- 이탈한 고객에게도 3개월 후 "잘 지내시죠?" 이메일 1통. 30%가 다시 돌아옵니다
