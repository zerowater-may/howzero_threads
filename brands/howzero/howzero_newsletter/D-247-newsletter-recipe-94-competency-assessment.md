역량 진단.
대부분의 중소기업이 안 하는 것.

이유: "시간이 없어서." "양식 만들기 귀찮아서." "결과 정리가 너무 오래 걸려서."

자동화하면 전부 해결됩니다.

---

구조.

1단계: 역량 모델 설정 (직무별 핵심 역량 5~7개)
2단계: 분기 1회 자동 설문 발송 (자기 평가 + 동료 평가 + 상사 평가)
3단계: 응답 자동 수집 → 역량 점수 자동 산출
4단계: 개인별 강점/약점 리포트 자동 생성
5단계: 성장 계획 템플릿 자동 제안 (약점 기반 학습 콘텐츠 매칭)

역량 진단의 목적은 "평가"가 아닙니다.
"성장 방향 제시"입니다.
