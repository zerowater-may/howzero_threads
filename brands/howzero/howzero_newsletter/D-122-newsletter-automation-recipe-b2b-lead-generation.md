B2B 영업에서 가장 시간이 많이 드는 작업이 뭘까요? 리드 발굴입니다.

LinkedIn에서 사람 찾고, 회사 정보 조사하고, 이메일 주소 확인하고, CRM에 입력하고. 이 과정을 수동으로 하면 하루 3시간은 기본입니다. 영업 담당자가 3명이면 하루 9시간.

오늘 레시피는 이 전체 과정을 자동화합니다.

---

🍳 **이번 주 레시피: B2B 리드 발굴 자동화**

**난이도**: ★★★☆☆ (중급)
**세팅 시간**: 2시간
**효과**: 리드 발굴 시간 90% 절감, 주당 200+ 리드 자동 수집

---

🔧 **전체 구조**

```
[ICP 정의] → [자동 스크래핑] → [데이터 정제] → [스코어링] → [CRM 자동 등록]
     ↓              ↓               ↓              ↓              ↓
  타겟 조건      LinkedIn/DB     AI 검증/보강    우선순위 산정    자동 배분
```

**Step 1: ICP(이상적 고객 프로필) 설정**

```
업종: SaaS, 이커머스, 제조업
규모: 직원 10~200명
직급: 대표, CTO, 운영 총괄
지역: 한국, 동남아
```

이걸 스프레드시트에 조건으로 정리합니다. 이 조건이 자동화의 필터가 됩니다.

**Step 2: 자동 수집 파이프라인**

```
Make(Integromat) 시나리오:
→ LinkedIn Sales Navigator API로 조건 매칭 프로필 수집
→ Apollo.io에서 이메일 주소 보강
→ 회사 정보(매출, 직원수, 최근 뉴스) 자동 수집
→ Google Sheets에 원본 데이터 적재
```

**Step 3: AI 기반 리드 스코어링**

```
GPT-4 API 호출:
→ 회사 웹사이트 자동 분석
→ "AI 자동화 도입 가능성" 0~100점 스코어링
→ 스코어 80점 이상 = Hot Lead
→ 50~79점 = Warm Lead
→ 49점 이하 = Cold Lead (보류)
```

**Step 4: CRM 자동 등록 + 알림**

```
Hot Lead → HubSpot CRM 자동 등록
→ 담당자 자동 배분 (라운드 로빈)
→ Slack #sales 채널에 알림
→ 24시간 내 팔로업 이메일 자동 발송
```

⚠️ **주의사항**

- LinkedIn 스크래핑은 이용약관을 확인하세요. 공식 API 사용을 권장합니다
