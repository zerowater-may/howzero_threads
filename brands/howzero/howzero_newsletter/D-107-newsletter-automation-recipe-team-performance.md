안녕하세요, 하우제로입니다.

"지난주 우리 팀 성과가 어땠어?"

이 질문에 답하려면 보통 이런 과정을 거치죠. 각 팀원한테 주간보고 받고, 엑셀에 취합하고, 그래프 만들고, PPT에 옮기고. 팀장의 월요일이 보고서 만드는 데 사라집니다.

팀이 5명이 되면서 저도 이 문제를 겪었습니다. 그래서 만들었어요.

---

🍳 **이번 주 레시피: 팀 성과 추적 자동화**

**난이도**: ★★★☆☆ (중급)
**세팅 시간**: 20분
**효과**: 관리 시간 월 10시간 절감

---

📋 **재료**

- Claude Code (월 $20)
- Google Sheets (대시보드)
- Slack 또는 노션 (데이터 수집)

---

🔧 **만드는 법**

**Step 1: KPI 자동 수집 (10분)**

```
각 팀원의 핵심 KPI를 자동으로 수집해줘:

콘텐츠팀:
- 영상 발행 수, 조회수, 구독자 증가

세일즈:
- 미팅 수, 제안서 발송, 계약 체결

커뮤니티:
- 신규 멤버, 활성 멤버, 질문 답변 수

각 데이터를 해당 도구에서 자동 수집하고
Google Sheets 대시보드에 일별 기록.
```

**Step 2: 주간 성과 리포트 자동 생성 (5분)**

```
매주 금요일 오후 5시에 주간 리포트를 만들어줘:

1. 팀 전체 KPI 요약 (목표 대비 달성률)
2. 팀원별 성과 (상위 3명 하이라이트)
3. 전주 대비 변화 (증감률)
4. 주의 필요 항목 (목표 대비 70% 미만)
5. 다음 주 중점 추진 사항 자동 제안

리포트를 Slack에 자동 발송.
```

**Step 3: 1:1 미팅 브리핑 자동 생성 (5분)**

```
팀원별 1:1 미팅 전에 브리핑 자료를 만들어줘:

- 최근 4주 성과 트렌드
- 목표 대비 달성률
- 강점 (잘 하고 있는 것)
- 개선 필요 (떨어지고 있는 것)
- 대화 포인트 제안 3개

미팅 1시간 전에 자동 전송.
```

하우제로 팀 실제 적용 결과:

| 지표 | 자동화 전 | 자동화 후 |
|------|-----------|-----------|
| 주간 보고 취합 시간 | 3시간 | 0시간 (자동) |
| 1:1 미팅 준비 | 30분/명 | 5분/명 |
| 성과 파악 속도 | 주 1회 | 실시간 |
| 이슈 발견 속도 | 2주 후 | 당일 |

**관리자가 보고서 만드는 대신, 팀원과 대화하는 데 시간을 씁니다.**

---

💡 **하우제로 팀에서 실제로 쓰는 방법**

매주 월요일 아침, 팀 전체 Slack 채널에 자동 리포트가 올라옵니다.

제가 따로 "지난주 어땠어?" 안 물어봐도 됩니다. 숫자가 다 말해줘요. 1:1 미팅에서는 숫자가 아니라 "왜 그런 결과가 나왔는지"에 집중합니다. 훨씬 생산적입니다.

---

⚠️ **주의사항**

- 성과 추적이 감시가 되면 안 됩니다. 목적은 지원이지 통제가 아닙니다
- KPI는 3~5개로 제한. 너무 많으면 의미가 흐려집니다
- 정량 데이터만으로 사람을 판단하지 마세요. 정성적 피드백도 함께
