---

**레시피 #72 구성:**

1. **타이밍 자동화**
   - 구매 후 3일, 사용 후 7일, 해지 직전 — 각 시점에 맞는 설문 자동 발송
   - "지금 바쁘시면 한 줄만" 옵션으로 응답 허들 최소화

2. **맥락 기반 질문 생성**
   - 고객의 사용 패턴 데이터 → AI가 맞춤형 질문 생성
   - 모든 고객에게 같은 질문? 의미 없습니다.
   - A 고객에겐 "기능 X를 많이 쓰시는데, 불편한 점은?"
   - B 고객에겐 "최근 로그인이 줄었는데, 이유가 있을까요?"

3. **응답 실시간 분석**
   - 텍스트 응답 → AI 감성 분석 + 키워드 추출
   - 부정 응답 즉시 CS팀 알림 (불만이 리뷰로 가기 전에 잡기)
   - 주간 트렌드 리포트 자동 생성

4. **피드백 루프 자동화**
   - 응답 기반 제품 개선 티켓 자동 생성 (Jira/Linear 연동)
   - 개선 완료 시 해당 고객에게 "반영했습니다" 자동 알림

---

결과:
- 설문 응답률: 8% → 34%
- 부정 피드백 대응 시간: 평균 72시간 → 평균 4시간
- 고객 이탈률: 12% → 7.8%

설문은 데이터 수집이 아닙니다.
고객과의 대화입니다.

대화를 자동화하면, 고객이 떠나기 전에 잡을 수 있습니다.
