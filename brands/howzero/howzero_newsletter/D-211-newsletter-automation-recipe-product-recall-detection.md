SNS에서 불만이 바이럴됩니다.

---

**레시피 #76 구성:**

1. **멀티채널 불만 감지**
   - 고객센터 문의, SNS 멘션, 리뷰 사이트 — 실시간 모니터링
   - 같은 제품에 대한 유사 불만이 3건 이상 → 자동 플래그
   - 감성 분석: "짜증" 수준 vs "위험" 수준 자동 분류

2. **패턴 분석 자동화**
   - 불만 데이터 클러스터링: 같은 증상인지 자동 판단
   - 생산 로트 번호 자동 매칭
   - "특정 기간 생산분에 집중된 문제인가" 즉시 분석

3. **리콜 의사결정 지원**
   - 영향 범위 자동 산출 (해당 로트 판매 수량, 지역 분포)
   - 리콜 비용 vs 미대응 리스크 비교 리포트 자동 생성
   - 규제 기관 신고 양식 자동 초안 작성

4. **리콜 실행 자동화**
   - 해당 고객 리스트 자동 추출
   - 리콜 안내 이메일/SMS 자동 발송
   - 반품 물류 프로세스 자동 트리거

- 리콜 대응 고객 만족도: 6.2/10 → 8.9/10
