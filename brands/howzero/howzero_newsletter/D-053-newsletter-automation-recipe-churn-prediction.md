🍳 **이번 주 레시피: 고객 이탈 예측**

**난이도**: ★★★☆☆ (중급)
**세팅 시간**: 20분
**효과**: 이탈률 50% 감소 가능

---

📋 **재료**

- Claude Code (월 $20)
- Google Sheets (무료)
- 고객 데이터 (접속 빈도, 문의 내역, 결제 이력)

---

🔧 **만드는 법**

**Step 1: 이탈 신호 정의 (5분)**

| 신호 | 위험 수준 |
|------|-----------|
| 2주 이상 미접속 | 🟡 주의 |
| 문의 빈도 급증 (불만) | 🟠 경고 |
| 결제 실패 후 미갱신 | 🔴 위험 |
| 이용량 50% 이상 감소 | 🟠 경고 |
| 경쟁사 언급 | 🔴 위험 |

여러분 비즈니스에 맞게 수정하세요.

**Step 2: 데이터 수집 자동화 (10분)**

```
매일 아침 고객 데이터를 확인해서
다음 이탈 신호가 있는 고객을 찾아줘:

1. 2주 이상 미접속
2. 최근 문의에 부정 키워드 포함 (불만, 해지, 환불, 다른 서비스)
3. 이용량이 전월 대비 50% 이상 감소
4. 결제 실패 후 3일 이상 미갱신

결과를 Google Sheets에 기록하고,
🔴 위험 고객은 즉시 이메일 알림.
```

**Step 3: 자동 대응 시나리오 (5분)**

```
위험 수준별 대응 액션을 만들어줘:

🟡 주의: "안녕하세요, 최근 어떠신가요?" 안부 이메일 초안
🟠 경고: 담당자에게 즉시 알림 + 전화 권유
🔴 위험: 긴급 미팅 요청 이메일 + 특별 혜택 제안 초안
```
