📌 **Week 3 미션: 리뷰 자동 수집 + 부정 알림**

이번 주는 "별점 3 이하 리뷰가 올라오면 10초 만에 알림."

**왜 이게 중요한가:**
부정 리뷰 24시간 내 대응 → 재구매율 33% 회복.
72시간 후 대응 → 7% 회복.
**5배 차이.** 10초 알림이 이 차이를 만듭니다.

**스텝 (15분)**:
1. 리뷰 소스 연결 — 쿠팡/스마트스토어 리뷰 페이지 URL (3분)
2. AI 감성 분석 — "이 리뷰가 긍정/부정/중립?" (3분)
3. 조건 분기 — 별점 3 이하만 알림 (3분)
4. 슬랙/카톡 알림 — 리뷰 내용 + 추천 답변 초안 (3분)
5. 일일 요약 — 매일 저녁 리뷰 요약 리포트 (3분)

---

**이걸 만들면?**

Before: 리뷰 확인 하루 30분~1시간. 부정 리뷰 3일 후 발견.
After: 확인 시간 0분. 부정 리뷰 10초 만에 알림 + 답변 초안.
