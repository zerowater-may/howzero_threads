━━━━━━━━━━━━━━━━━━━━━━━━━━━━

채용 공고 올리고 기다리는 시대는 끝났습니다.

하우제로 팀이 15명까지 성장하면서 깨달은 것.
좋은 사람은 공고를 보고 오지 않습니다.
좋은 사람은 **브랜드를 보고 옵니다.**

문제는 채용 브랜딩에 시간을 쏟을 여력이 없다는 것.
그래서 자동화했습니다.

---

**레시피 #71 구성:**

1. **채용 콘텐츠 자동 생성**
   - 팀원 인터뷰 → AI가 채용 블로그 초안 작성
   - 사내 문화 사진 → 자동 캡션 + SNS 포스팅 예약

2. **채널별 자동 배포**
   - 링크드인, 원티드, 잡플래닛에 동시 게시
   - 채널별 톤앤매너 자동 조정 (같은 내용, 다른 포맷)

3. **지원자 파이프라인 자동화**
   - 지원서 접수 → 슬랙 알림 → 자동 분류
   - 1차 스크리닝 질문 자동 발송 → 답변 요약 리포트

4. **후보자 경험 관리**
   - 접수 확인 메일 즉시 발송
   - 진행 상황 자동 업데이트 (지원자가 묻기 전에)

---

결과:
- 채용 콘텐츠 제작 시간: 주 8시간 → 주 1시간
- 지원자 응답 속도: 평균 48시간 → 평균 2시간
- 채용 전환율: 2.1% → 4.8%
