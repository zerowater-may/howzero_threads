**난이도**: ★★★☆☆ (중급)
**세팅 시간**: 3시간 (템플릿 포함)
**효과**: 제안서 작성 시간 90% 절감, 품질 일관성 확보

---

🔧 **5단계 자동화 파이프라인**

**Step 1: 고객 정보 자동 수집**

```
CRM에 딜 등록 시 자동 실행:
→ 고객사 웹사이트 크롤링 (사업 영역, 규모, 최근 뉴스)
→ 산업 보고서 자동 검색 (관련 트렌드 3개)
→ 기존 유사 고객 사례 자동 매칭
→ 구조화된 고객 프로필 JSON 생성
```

**Step 2: AI 제안서 초안 생성**

```
GPT-4 API + 커스텀 프롬프트:
→ 고객 프로필 입력
→ 과제 정의, 솔루션 제안, 기대 효과 자동 작성
→ 유사 사례의 실제 성과 데이터 자동 삽입
→ 3가지 옵션 (Basic/Standard/Premium) 자동 구성
```

**Step 3: 디자인 자동 적용**

```
Google Slides API / Canva API:
→ 브랜드 템플릿에 AI 생성 콘텐츠 자동 삽입
→ 고객사 로고 자동 배치
→ 차트/그래프 자동 생성 (기대 ROI, 타임라인)
→ PDF 자동 변환
```

**Step 4: 가격 자동 산출**

```
프라이싱 매트릭스 기반:
→ 프로젝트 범위 × 난이도 × 기간 → 자동 견적
→ 할인율 자동 적용 (고객 등급별)
→ 3가지 패키지 가격표 자동 생성
→ ROI 예측 자동 계산 ("투자 대비 X개월 내 회수")
```

**Step 5: 발송 + 추적**

```
→ 담당자 검토 (AI 초안 수정 15~20분)
→ 이메일 자동 발송 (추적 링크 포함)
→ 열람 여부 실시간 알림
→ 3일 미열람 시 자동 팔로업
→ 열람 후 5분 이상 체류 시 "관심 높음" 태깅
```
