안녕하세요, 하우제로입니다.

NPS(Net Promoter Score)를 수동으로 측정하고 계신가요?

분기마다 설문지 돌리고, 엑셀에 취합하고, 리포트 만드는 데 며칠 쓰는 구조. 저도 그랬습니다. 문제는 설문 응답률이 15%도 안 됐다는 겁니다. 표본이 부족하니 점수가 의미 없었어요.

자동화하니 응답률이 48%로 올랐습니다. 이유는 간단합니다. 타이밍이 정확해졌거든요.

---

🍳 **이번 주 레시피: NPS 자동 측정**

**난이도**: ★★☆☆☆ (초중급)
**세팅 시간**: 2시간
**효과**: 응답률 3배 증가, 이탈 조기 감지

---

🔧 **자동화 구조**

**1. 트리거 기반 설문 발송**

```
고객 행동 기반 자동 발송:
→ 서비스 이용 30일 후: 첫 NPS 설문
→ 주요 마일스톤 달성 시: 만족도 체크
→ CS 티켓 해결 24시간 후: 서비스 경험 평가
→ 구독 갱신 7일 전: 갱신 의사 + NPS
→ 분기 말: 정기 NPS (기존 방식 유지)
```

**2. 점수 자동 분류 + 대응**

```
NPS 점수별 자동 워크플로:
→ 추천자 (9~10점): 감사 메일 + 리뷰 요청 + 추천 프로그램 안내
→ 중립자 (7~8점): 개선 가능 영역 추가 질문 자동 발송
→ 비추천자 (0~6점): 즉시 CS팀 알림 + 48시간 내 전화 연락
→ 미응답: 3일 후 리마인드 1회 (강제 X)
```

**3. 실시간 대시보드**

```
자동 집계:
→ 전체 NPS 점수 (실시간 업데이트)
→ 세그먼트별 NPS (업종/규모/이용 기간)
→ 월별 추이 그래프
→ 비추천 사유 워드클라우드
→ 추천자 → 실제 추천 전환율
```

**4. 이탈 조기 경보**

```
경고 시스템:
→ NPS 2회 연속 하락 고객 → "위험" 태그
→ 비추천자 비율 20% 초과 시 → 경영진 알림
→ 특정 세그먼트 NPS 급락 → 원인 분석 리포트 자동 생성
```

📊 **하우제로 NPS 현황**

| 지표 | 수동 시절 | 자동화 후 |
|------|----------|----------|
| 설문 응답률 | 15% | 48% |
| 전체 NPS | 32 | 54 |
| 비추천자 대응 시간 | 7일 | 1.5일 |
| 추천자 → 실추천 전환 | 5% | 18% |
| 월 투입 시간 | 20시간 | 2시간 |

NPS가 32에서 54로 오른 건 고객이 갑자기 만족해서가 아닙니다. 불만족 신호를 빨리 잡아서 해결했기 때문입니다.

---

⚠️ **주의사항**

- 설문 피로도를 관리하세요. 한 고객에게 월 1회 이상 설문 보내지 마세요
- NPS 점수 자체에 집착하지 말고, 비추천 사유에 집중하세요. 점수는 후행 지표입니다
- 자동 대응이 있더라도 비추천자에게는 반드시 사람이 연락하세요
