**레시피 #81 구성:**

1. **시장 규모 자동 분석**
   - 타겟 국가 + 업종 입력
   - AI가 공개 데이터 소스 20개+ 자동 수집
   - TAM/SAM/SOM 자동 계산
   - 시장 성장률 트렌드 시각화

2. **경쟁사 자동 매핑**
   - 해당 국가의 동종 서비스 자동 검색
   - 가격, 기능, 리뷰, 시장 점유율 자동 정리
   - 포지셔닝 맵 자동 생성 (가격 × 기능 매트릭스)
   - 경쟁사 약점/기회 자동 분석

3. **현지 트렌드 모니터링**
   - 현지 뉴스, SNS, 포럼 — 키워드 기반 자동 크롤링
   - 주간 트렌드 리포트 (현지어 → 한국어 자동 번역)
   - "이 시장에서 지금 뜨는 것 / 지는 것" 자동 분류

4. **진입 전략 자동 제안**
   - 리서치 데이터 기반 진입 시나리오 3가지 자동 생성
   - 각 시나리오별 예상 비용, 기간, 리스크 자동 산출
   - 추천 파트너/유통 채널 자동 리스트업

---
