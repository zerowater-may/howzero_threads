안녕하세요, 하우제로입니다.

고객이 우리 서비스를 처음 알게 되는 순간부터 충성 고객이 되기까지. 이 여정에는 수십 개의 터치포인트가 있습니다.

문제는 이 여정 중 "어디서 고객이 이탈하는지"를 모른다는 겁니다. 대부분 매출이 안 오르면 "마케팅을 더 해야지"라고 생각합니다. 하지만 실제로는 퍼널의 한 지점에서 80%가 빠져나가고 있을 수 있어요.

그 지점을 자동으로 찾아줍니다.

---

🍳 **이번 주 레시피: 고객 여정 분석 자동화**

**난이도**: ★★★★☆ (고급)
**세팅 시간**: 4시간
**효과**: 이탈 지점 자동 감지, 전환율 최적화

---

🔧 **전체 구조**

```
[데이터 수집] → [여정 매핑] → [이탈 분석] → [인사이트] → [자동 대응]
모든 터치포인트   단계별 분류    병목 지점 탐지   AI 분석      자동 개입
```

**Step 1: 터치포인트 데이터 통합**

```
수집 대상:
→ 웹사이트: 페이지뷰, 스크롤 깊이, 클릭 (GA4)
→ 이메일: 오픈, 클릭, 답장 (Beehiiv/SendGrid)
→ 제품: 로그인, 기능 사용, 세션 시간 (Mixpanel)
→ CS: 문의, 불만, NPS (Intercom)
→ 결제: 구매, 환불, 업그레이드 (Stripe)
→ 전부 하나의 데이터 웨어하우스에 통합
```

**Step 2: 여정 단계 자동 매핑**

```
고객별 여정 자동 구성:
→ 인지: 첫 방문 채널 + 콘텐츠
→ 관심: AI Score 진단, 뉴스레터 구독
→ 고려: 코스 페이지 방문, 가격 페이지 체류
→ 구매: 결제 완료
→ 활성: 로그인 빈도, 기능 사용률
→ 충성: NPS 9-10점, 추천 활동
```

**Step 3: 이탈 병목 자동 탐지**

```
단계별 전환율 자동 추적:
인지 → 관심:     12% (정상)
관심 → 고려:      8% (⚠️ 병목!)
고려 → 구매:     22% (정상)
구매 → 활성:     65% (정상)
활성 → 충성:     30% (개선 필요)

AI 분석: "관심→고려 단계에서 8%만 전환.
원인: AI Score 결과 페이지에서 다음 액션(코스 안내)이 
명확하지 않음. CTA 추가 권장."
```

**Step 4: 자동 대응**

```
이탈 감지 시 자동 개입:
→ 가격 페이지에서 이탈 → 24시간 후 할인 쿠폰 이메일
→ 코스 7일간 미접속 → 리마인드 + 인기 콘텐츠 추천
→ NPS 6점 이하 → CS팀 자동 배정 + 1:1 연락
→ 결제 페이지 이탈 → 장바구니 리마인드 (3시간 후)
```
