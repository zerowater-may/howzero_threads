📌 **이번 주 챌린지: 이메일 자동 분류**

오늘부터 '주 1개 자동화 챌린지'를 시작합니다.

매주 1개, 15분이면 만들 수 있는 자동화를 알려드리겠습니다. 따라하시고, 답장으로 "완료!"만 보내주세요.

---

**Week 1 미션: 이메일 자동 분류**

목표: 이메일이 오면 AI가 자동으로 분류(환불/배송/문의/기타)

준비물:
- Gmail 계정
- Claude Code 계정 (무료)
- 15분

스텝:
1. Claude Code 가입 (3분)
2. Gmail 트리거 연결 — "새 이메일 오면 시작" (3분)
3. AI 프롬프트 — "이 이메일을 환불/배송/제품/기타로 분류해줘" (3분)
4. 결과를 슬랙/카톡으로 알림 (3분)
5. 테스트 이메일 보내기 (3분)

끝!

---

**이걸 만들면 뭐가 바뀌나요?**

Before: 이메일 50개를 하나하나 열어서 '이건 환불이네, 이건 배송이네' 수동 분류. 1시간.
After: AI가 3초 만에 분류하고 알림. 나는 중요한 것만 확인. 5분.
