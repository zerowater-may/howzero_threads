---

**레시피 #75 구성:**

1. **비동기 스탠드업 자동화**
   - 매일 오전 9시, 슬랙 봇이 3가지 질문
   - "어제 한 일 / 오늘 할 일 / 블로커"
   - AI가 전체 팀 스탠드업 요약 → 한눈에 파악

2. **포커스 타임 보호**
   - 캘린더에 "딥워크" 블록 자동 생성
   - 딥워크 시간엔 슬랙 알림 자동 뮤트
   - 긴급도 판단: 정말 급한 건만 통과

3. **성과 기반 대시보드**
   - 근무 시간이 아니라 **완료된 태스크** 추적
   - Jira/Linear 완료 데이터 → 주간 생산성 리포트 자동 생성
   - 개인별 생산성 트렌드 (본인만 열람 가능)

4. **팀 연결 자동화**
   - 매주 랜덤 1:1 커피챗 매칭
   - 월 1회 팀 전체 온라인 모임 자동 스케줄링
   - 분기 1회 오프라인 워크숍 (이것만 직접)

---

결과 (도입 6개월 후):
- 불필요한 회의: 주 12시간 → 주 4시간
- 태스크 완료율: 68% → 89%
- 팀 만족도 (익명 설문): 7.2/10 → 8.8/10
- 사무실 임대료 절감: 연 ₩6,000만 원
