안녕하세요, 하우제로입니다.

리뷰에 답변 안 하고 계신 분, 손 들어보세요.

솔직히 이해합니다. 바쁘니까요. 근데 데이터가 말하는 건 이겁니다: 리뷰에 답변하는 업체는 재구매율이 평균 18% 높습니다.

문제는 리뷰가 하루에 10개, 20개씩 쌓이면 답변할 시간이 없다는 거죠. 그래서 자동화합니다.

---

🍳 **이번 주 레시피: 리뷰 답변 자동화**

**난이도**: ★★☆☆☆ (초급)
**세팅 시간**: 15분
**효과**: 리뷰 답변 시간 90% 절감

---

📋 **재료**

- Claude Code (월 $20)
- Google Sheets (무료)
- 리뷰 플랫폼 (네이버, 쿠팡, 구글 등)

---

🔧 **만드는 법**

**Step 1: 리뷰 수집 자동화 (5분)**

```
다음 플랫폼에서 새 리뷰를 매일 수집해줘:
- 네이버 스마트스토어
- 쿠팡
- 구글 비즈니스

수집 데이터:
- 작성자, 별점, 리뷰 내용, 작성일
- Google Sheets에 자동 기록
```

**Step 2: 리뷰 분류 + 답변 초안 생성 (5분)**

```
리뷰를 읽고 다음 기준으로 분류해줘:

1. 긍정 리뷰 (4~5점): 감사 + 재방문 유도 답변
2. 중립 리뷰 (3점): 아쉬운 점 공감 + 개선 약속 답변
3. 부정 리뷰 (1~2점): 사과 + 해결 방안 + 연락처 안내
4. 긴급 (불량, 오배송 등): 즉시 알림 + 긴급 대응 답변

톤: 진심 어린, 개인화된, 복사-붙여넣기 느낌 없는.
리뷰에 언급된 구체적 내용을 반드시 답변에 반영.
```

**Step 3: 승인 후 자동 게시 (5분)**

```
생성된 답변 초안을 검토 시트에 정리해줘.

- 긍정 리뷰 답변: 자동 게시 (검토 없이)
- 중립 리뷰 답변: 검토 후 원클릭 게시
- 부정 리뷰 답변: 반드시 수동 검토 후 게시
- 긴급: 담당자에게 즉시 알림

매일 아침 "오늘 답변할 리뷰 [N]건" 리포트 발송.
```

✅ **완성!**

| 지표 | 자동화 전 | 자동화 후 |
|------|-----------|-----------|
| 일 리뷰 답변 시간 | 45분 | 5분 (검토만) |
| 답변율 | 40% | 98% |
| 평균 답변 속도 | 3일 | 12시간 |
| 재구매율 변화 | 기준 | +15% |

**답변율 40% → 98%. 이것만으로 매출이 바뀝니다.**

---

⚠️ **주의사항**

- 부정 리뷰는 반드시 사람이 최종 확인. AI가 톤을 잘못 잡으면 역효과
- "모든 리뷰에 똑같은 답변" 느낌이 나면 안 됩니다. 개인화가 핵심
- 거짓 정보로 답변하면 안 됩니다. 확실한 것만 답변에 포함
