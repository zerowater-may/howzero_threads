🔧 **만드는 법**

**Step 1: 영수증 자동 인식 (5분)**

```
영수증 사진을 업로드하면 자동으로 읽어서:

1. 날짜
2. 금액
3. 가맹점명
4. 항목 분류 (식비, 교통비, 업무용품 등)

이 4가지를 추출해서 Google Sheets에 자동 입력해줘.
사진이 흐려도 최대한 인식. 인식 불가 시 "수동 입력 필요" 표시.
```

**Step 2: 자동 분류 + 한도 체크 (5분)**

```
정산 데이터를 자동으로 분류하고 검증해줘:

- 항목별 월 한도 체크 (식비 ₩30만, 교통비 ₩15만 등)
- 한도 초과 시 자동 알림
- 중복 정산 감지 (같은 날짜 + 같은 금액)
- 개인 경비 vs 법인 경비 자동 분류
- 세금계산서 필요 여부 표시
```

**Step 3: 승인 워크플로우 자동화 (5분)**

```
정산 건이 등록되면:

1. 자동으로 승인권자에게 알림
2. ₩10만 미만: 자동 승인
3. ₩10만~₩50만: 팀장 승인
4. ₩50만 이상: 대표 승인

승인되면 → 회계팀 시트에 자동 이동
반려되면 → 사유와 함께 직원에게 알림
월말에 부서별 경비 리포트 자동 생성.
```
