🔧 **만드는 법**

**Step 1: 모니터링 항목 정하기 (3분)**

| 항목 | 체크 방법 |
|------|-----------|
| 가격 변동 | 제품 페이지 |
| 신제품 출시 | 블로그/뉴스 |
| SNS 반응 | 인스타/트위터 |
| 채용 공고 | 채용 페이지 |
| 마케팅 캠페인 | 광고 라이브러리 |

전부 할 필요 없습니다. 우리 비즈니스에 직결되는 2~3개만.

**Step 2: 수집 자동화 (7분)**

```
매일 아침 9시에 다음 경쟁사의 변동사항을 확인해줘:
1. [경쟁사A] — 가격 페이지, 블로그
2. [경쟁사B] — 신제품 페이지, SNS
3. [경쟁사C] — 채용 공고, 블로그

변동 있으면:
- 변경 내용 요약
- 우리에게 미치는 영향 분석
- 추천 대응 액션

결과를 Google Sheets에 날짜별로 기록.
변동 있을 때만 이메일 알림.
```

**Step 3: 주간 리포트 자동 생성 (5분)**

```
이번 주 경쟁사 변동사항을 요약해서
주간 인텔리전스 리포트를 만들어줘.
핵심 트렌드, 위협, 기회 포함.
```
