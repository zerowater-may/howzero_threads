🔧 **만드는 법**

**Step 1: 원본 데이터 정리 (5분)**

Google Sheets에 다음 컬럼을 준비하세요:

| 컬럼 | 내용 |
|------|------|
| A | 상품명 (원본) |
| B | 가격 |
| C | 주요 스펙 |
| D | 카테고리 |

나머지는 AI가 채워줍니다.

**Step 2: AI 상품 설명 생성 (5분)**

```
상품 데이터를 읽고 각 상품에 대해 다음을 생성해줘:

1. SEO 최적화 상품명 (50자 이내, 핵심 키워드 포함)
2. 상세 설명 (300자, 혜택 중심, 구매 유도)
3. 검색 키워드 5개
4. 카테고리 자동 분류

톤: 신뢰감 있고 간결하게.
결과를 Google Sheets E~H 컬럼에 기록.
```

**Step 3: 일괄 업로드 자동화 (5분)**

```
Google Sheets의 완성된 데이터를 읽고
쇼핑몰 플랫폼 업로드용 CSV 포맷으로 변환해줘.
플랫폼별 필수 필드 매핑:
- 스마트스토어
- 쿠팡
- 카페24
```
