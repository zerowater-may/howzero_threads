🍳 **이번 주 레시피: 직원 만족도 자동 조사**

**난이도**: ★★☆☆☆ (초중급)
**세팅 시간**: 1.5시간
**효과**: 이탈 신호 조기 감지, 팀 만족도 지속 추적

---

🔧 **3가지 자동 조사 시스템**

**1. 주간 펄스 체크 (Pulse Check)**

```
매주 금요일 오후 4시 자동 발송 (Slack Bot):
→ 3개 질문만 (응답 시간 30초):
  Q1: 이번 주 업무 만족도? (1~5 이모지)
  Q2: 도움이 필요한 부분? (선택형 3개 + 기타)
  Q3: 한마디 (선택, 익명)
→ 자동 집계 → Google Sheets 기록
→ 평균 3.0 이하 시 대표에게 알림
```

**2. 월간 심층 설문**

```
매월 마지막 수요일 자동 발송 (Typeform):
→ 7개 항목 (5분):
  업무량, 성장 기회, 팀 소통, 리더십, 보상, 워라밸, 추천 의향(eNPS)
→ 완전 익명 (6명 이하 팀에서는 특히 중요)
→ AI 분석: 전월 대비 변화 + 트렌드 감지
→ 리포트 자동 생성 → 대표 + HR에게 전달
```

**3. 이탈 위험 자동 감지**

```
행동 데이터 기반 (비침해적 범위 내):
→ Slack 활동 빈도 급감 (30% 이상 감소)
→ 주간 펄스 체크 3주 연속 3.0 이하
→ 월간 설문 미응답 2회 연속
→ 유급 휴가 사용률 급증
→ 복합 점수 산출 → "관심 필요" 알림
```

📊 **하우제로 6개월 운영 결과**

| 지표 | Before | After |
|------|--------|-------|
| 팀 평균 만족도 | 측정 안 함 | 4.2/5.0 |
| 이슈 발견 → 대응 시간 | 퇴사 시 | 평균 3일 |
| 자발적 이직률 | 66% (2/3) | 0% (6개월간) |
| 1:1 미팅 품질 | 막연한 대화 | 데이터 기반 대화 |

가장 큰 변화는 1:1 미팅입니다. "요즘 어때?" 대신 "이번 달 업무량 점수가 2.8이었는데, 어떤 부분이 부담이야?"로 대화가 바뀌었어요. 구체적인 대화가 구체적인 해결로 이어집니다.

⚠️ **주의사항**

- **익명성을 반드시 보장하세요.** 6명 이하 소규모 팀은 답변 패턴으로 개인이 특정될 수 있습니다. 결과는 전체 평균만 공유
