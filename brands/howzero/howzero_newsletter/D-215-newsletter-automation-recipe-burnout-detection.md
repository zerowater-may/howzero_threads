**레시피 #78 구성:**

(주의: 이건 감시 도구가 아닙니다. 케어 도구입니다.)

1. **행동 패턴 모니터링 (익명/집계 기반)**
   - 슬랙 활동량 변화 추적 (개인 내용 열람 없음, 빈도만)
   - 근무 시간 패턴: 야근 빈도, 주말 접속 증가 감지
   - 태스크 완료율 트렌드: 개인별 기준선 대비 변화

2. **위험 신호 자동 감지**
   - "번아웃 위험" 3단계: 관심 → 주의 → 경고
   - 복합 지표: 활동량 감소 + 근무 시간 증가 = 비효율적 과로
   - 팀 리더에게 익명화된 알림 (누가 아닌 "팀에 위험 신호")

3. **선제적 케어 자동화**
   - 주의 단계: 매니저에게 "1:1 미팅 추천" 알림
   - 경고 단계: HR에게 "워크로드 리밸런싱 필요" 알림
   - 전체 팀 웰빙 지수 주간 리포트

4. **구조적 원인 분석**
   - 특정 프로젝트 참여자들만 번아웃 증가? → 프로젝트 문제
   - 특정 시기에 전체 팀 번아웃 증가? → 시즌 이슈
   - 반복 패턴 → 구조적 개선 제안 자동 생성
