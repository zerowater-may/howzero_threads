병원, 미용실, 레스토랑, 컨설팅 — 예약 기반 비즈니스를 하시는 분들.

전화로 예약 받고, 수첩에 적고, 노쇼 나면 매출 손실. 이 사이클을 아직도 반복하고 계신가요?

예약 관리는 자동화 효과가 가장 즉각적으로 나타나는 영역입니다. 왜냐면 예약 → 확인 → 리마인더 → 노쇼 관리가 전부 규칙 기반이거든요. 규칙 기반 = 자동화 최적.

**난이도**: ★★☆☆☆ (초급)
**세팅 시간**: 15분
**효과**: 노쇼율 60% 감소, 관리 시간 80% 절감

📋 **재료**

- Claude Code (월 $20)
- Google Calendar (무료)
- Google Sheets (무료)
- 카카오 알림톡 또는 문자 API (건당 ₩15~30)

---

🔧 **만드는 법**

**Step 1: 온라인 예약 폼 연동 (5분)**

```
Google Forms로 예약 폼을 만들고
제출되면 자동으로:

1. Google Calendar에 일정 추가
2. Google Sheets에 고객 정보 기록
3. 고객에게 예약 확인 메시지 발송

메시지 템플릿:
"[고객명]님, [날짜] [시간] 예약이 확정되었습니다.
변경/취소는 이 링크에서: [링크]"
```

**Step 2: 자동 리마인더 시스템 (5분)**

```
예약일 기준으로 리마인더를 자동 발송해줘:

- 3일 전: "예약 확인 부탁드립니다" + 확인/변경/취소 버튼
- 1일 전: "내일 방문 예정입니다" 알림
- 당일 2시간 전: 최종 리마인더

고객이 '확인' 누르면 → 상태 업데이트
고객이 '취소' 누르면 → 대기자에게 자동 안내
응답 없으면 → 전화 필요 리스트에 추가
```

**Step 3: 노쇼 관리 자동화 (5분)**

```
노쇼 발생 시 자동으로:

1. 고객 DB에 노쇼 기록
2. 다음 예약 시 선결제 요청 플래그
3. 해당 시간 빈자리 알림 (대기자에게)
4. 주간 노쇼 리포트 생성
```
