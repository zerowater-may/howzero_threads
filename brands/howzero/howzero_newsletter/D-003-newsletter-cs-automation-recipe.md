📌 **이번 주 핵심 인사이트**

**"CS 자동화, 15분이면 됩니다. 레시피 공개."**

지난주에 "프로세스 맵 채워보세요"라고 드렸는데, 답장 46개를 받았습니다. 감사합니다.

46개 중 38개의 1순위가 같았습니다. **CS 문의 처리.**

그래서 오늘은 CS 자동화 레시피를 드립니다. 진짜 15분이면 됩니다.

**레시피 재료:**
- AI API (Claude or ChatGPT) — 문의 분류 + 답변 생성
- 자동화 도구 (Claude Code or Make.com) — 연결
- 이메일/카톡 — 문의 수신 채널
- 슬랙 or 카톡 — 담당자 알림

**레시피 순서:**

```
Step 1 (3분): 이메일/카톡 수신 트리거 설정
Step 2 (5분): AI에게 "분류해줘" 프롬프트 연결
  → "이 문의를 환불/배송/제품/기타 중 하나로 분류해줘"
Step 3 (5분): 분류별 답변 초안 자동 생성
  → "환불이면 환불 절차 안내를 친절하게 작성해줘"
Step 4 (2분): 담당자 슬랙/카톡 알림 + Notion 저장
```

**결과:**
건당 5분 → 건당 38초 (검토만)
하루 50건 기준: 4시간 10분 → 32분
월 절감: 약 240만 원

**핵심 팁:**
처음에는 '자동 발송' 하지 마세요. AI가 만든 초안을 사람이 확인하고 보내세요. 정확도 92%지만, 나머지 8%가 고객 신뢰를 깎을 수 있습니다.
