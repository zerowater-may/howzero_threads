안녕하세요, 하우제로입니다.

재고 관리, 감으로 하고 계시죠?

"이 정도면 되겠지" 하고 발주했는데 품절. 또는 "많이 시켜두자" 했는데 재고가 쌓여서 자금이 묶임. 둘 다 돈이 새는 겁니다.

품절 = 매출 손실. 과잉 재고 = 자금 묶임.
AI로 예측하면 둘 다 줄일 수 있습니다.

---

🍳 **이번 주 레시피: 재고 예측 자동화**

**난이도**: ★★★☆☆ (중급)
**세팅 시간**: 20분
**효과**: 재고 비용 30% 절감, 품절률 70% 감소

---

📋 **재료**

- Claude Code (월 $20)
- Google Sheets (판매/재고 데이터)
- 최소 3개월 판매 이력 데이터

---

🔧 **만드는 법**

**Step 1: 판매 데이터 분석 (10분)**

```
판매 데이터를 분석해서 각 상품별로:

1. 일 평균 판매량 (최근 3개월)
2. 주간 판매 패턴 (요일별 차이)
3. 월간 추세 (증가/감소/유지)
4. 계절성 패턴 (있으면)
5. 이상치 제거 (프로모션 기간 등)

결과를 Google Sheets에 상품별 정리.
```

**Step 2: 예측 모델 + 발주 시점 계산 (5분)**

```
판매 분석 데이터를 기반으로 각 상품의:

1. 향후 2주 예상 판매량
2. 안전 재고 수준 (품절 방지 최소치)
3. 발주 시점 (리드타임 고려)
4. 최적 발주 수량

안전 재고 = 일 평균 판매 × 리드타임(일) × 1.5
발주 시점 = 현재 재고 ≤ 안전 재고일 때
발주량 = 2주 예상 판매 - 현재 재고 + 안전 재고
```

**Step 3: 자동 알림 + 발주 리스트 (5분)**

```
매일 아침 재고 상태를 확인하고:

🔴 긴급: 3일 내 품절 예상 → 즉시 알림
🟡 주의: 7일 내 품절 예상 → 발주 리스트 추가
🟢 정상: 2주 이상 여유 → 모니터링만
⚫ 과잉: 4주 이상 재고 → 프로모션 제안

매주 월요일 "이번 주 발주 리스트" 자동 생성.
월간 재고 회전율 리포트 생성.
```

---

✅ **완성!**

컨설팅 고객사 적용 결과:

| 지표 | 자동화 전 | 자동화 후 |
|------|-----------|-----------|
| 월 품절 건수 | 12건 | 3건 |
| 과잉 재고 비율 | 35% | 15% |
| 재고 비용 | 기준 | -30% |
| 재고 회전율 | 4회/년 | 6.5회/년 |

**품절 9건 방지 × 건당 ₩50만 = 월 ₩450만 매출 보호.**
**과잉 재고 20% 감소 = 월 ₩300만 자금 확보.**
