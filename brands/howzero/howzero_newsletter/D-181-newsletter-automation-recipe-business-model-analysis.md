안녕하세요, 하우제로입니다.

새 사업을 시작하기 전에 가장 먼저 해야 할 일은 뭘까요?

시장 조사? 사업계획서? 아닙니다. "이 모델이 돈이 되는 구조인가"를 숫자로 검증하는 겁니다.

3년간 컨설팅하면서 가장 많이 본 실패 유형: "열심히 만들었는데, 구조적으로 수익이 안 나는 모델." 열정은 충분한데 수학이 부족한 경우. 이걸 자동으로 점검하는 시스템을 만들었습니다.

---

🍳 **이번 주 레시피: 비즈니스 모델 분석 자동화**

**난이도**: ★★★☆☆ (중급)
**세팅 시간**: 4시간
**효과**: 신규 사업 검증 시간 80% 절감, 실패 확률 감소

---

🔧 **자동화 구조**

**1. 모델 입력 & 자동 분석**

```
핵심 변수 입력 (10분):
→ 타겟 시장 규모 (TAM/SAM/SOM)
→ 가격 모델 (구독/건당/프리미엄)
→ 고객 획득 비용 (CAC) 추정
→ 고객 생애 가치 (LTV) 추정
→ 고정비/변동비 구조
→ 예상 전환율

자동 산출:
→ LTV/CAC 비율 (3 이상이면 건강)
→ 손익분기점 (BEP) 시점
→ 월별/연별 현금흐름 예측
→ 단위 경제학 (Unit Economics) 요약
```

**2. 벤치마크 자동 비교**

```
유사 비즈니스 모델 비교:
→ 동종 업계 평균 마진율 자동 제공
→ 유사 모델 성공/실패 사례 자동 매칭
→ 업계 평균 CAC, LTV, 이탈률 비교
→ "이 모델은 업계 평균 대비 ___" 자동 진단
```

**3. 시나리오 시뮬레이션**

```
자동 스트레스 테스트:
→ "전환율이 절반이면?" → 수익성 변화 자동 계산
→ "CAC가 2배면?" → BEP 시점 변화
→ "이탈률이 10%면?" → LTV 변화
→ "가격을 20% 낮추면?" → 볼륨 얼마나 늘어야 하나
→ 최악/기본/최선 3가지 시나리오 자동 생성
```

---

📊 **실제 사례: 하우제로 일본 진출 검증**

| 변수 | 낙관 | 기본 | 비관 |
|------|------|------|------|
| 월 신규 고객 | 15곳 | 8곳 | 3곳 |
| CAC | ₩30만 | ₩50만 | ₩80만 |
| 월 해지율 | 2% | 5% | 8% |
| LTV | ₩720만 | ₩360만 | ₩180만 |
| BEP | 4개월 | 8개월 | 24개월 |

기본 시나리오에서 8개월 만에 BEP. 비관 시나리오에서도 24개월. 투자 가능하다고 판단했고, 실제로 6개월 만에 BEP를 넘었습니다.

이 시뮬레이션 없이 "일본 시장 크니까 가보자"고 했으면 위험했을 겁니다.

---

⚠️ **주의사항**

- 모델 분석은 가정(assumption)에 기반합니다. 가정이 틀리면 결과도 틀립니다. 핵심 가정은 반드시 검증하세요
- 시뮬레이션이 좋다고 반드시 성공하는 건 아닙니다. 실행력이 50%입니다
- 분석 마비(Analysis Paralysis)에 빠지지 마세요. 시뮬레이션은 1주일 안에 끝내고 실행하세요
