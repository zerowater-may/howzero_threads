🍳 **이번 주 레시피: 커뮤니티 운영 자동화**

**난이도**: ★★★☆☆ (중급)
**세팅 시간**: 3시간
**효과**: 운영 시간 80% 절감, 멤버 활성도 유지

---

🔧 **5가지 자동화 모듈**

**1. 신규 멤버 온보딩**

```
가입 시 자동 실행:
→ 환영 DM (이름 개인화) 자동 발송
→ 자기소개 양식 링크 제공
→ 필수 채널 안내 + 가이드 문서 공유
→ 소그룹 자동 배정 (관심 분야 기반)
→ Day 3: "첫 질문을 올려보세요" 넛지
→ Day 7: 미활동 시 리마인드
→ Day 14: 활동 여부 확인 + 소그룹 소개
```

**2. 콘텐츠 자동 발행**

```
→ 매주 월요일: 주간 하이라이트 자동 게시 (지난주 인기 글 3개)
→ 매주 수요일: 자동화 레시피 미니 팁 (뉴스레터에서 발췌)
→ 매주 금요일: #wins 공유 리마인드 (성과 공유 독려)
→ 월간: 이달의 MVP 자동 선정 + 축하 게시
→ 이벤트 자동 알림 (세미나, AMA, 챌린지)
```

**3. 활성도 관리**

```
멤버별 활동 점수 자동 산출:
→ 글 작성: +5점, 답변: +3점, 반응: +1점
→ 주간 활동 0점 → 4주 연속 시 "비활성 멤버" 태그
→ 비활성 멤버 → 자동 리마인드 DM
→ 8주 연속 비활성 → 관리자에게 알림 (개인 접촉 필요)
→ 상위 10% 활동 멤버 → "기여자" 배지 자동 부여
```

**4. 질문·답변 자동 지원**

```
커뮤니티 AI 봇:
→ 질문 올라오면 기존 FAQ/레시피에서 관련 답변 자동 추천
→ "이 질문에 대한 답변이 레시피 #28에 있습니다" 자동 링크
→ AI가 답변 못 하는 경우 → 해당 분야 기여자에게 자동 태그
→ 48시간 미답변 → 운영진에게 알림
```

**5. 분석 대시보드**

```
주간 자동 리포트:
→ 신규 가입 / 탈퇴 / 순증
→ 활성 멤버 비율 (DAU/WAU)
→ 인기 토픽 + 감성 분석
→ 소그룹별 활성도 비교
→ 전환 지표 (멤버 → 코스/아카데미/오딧)
```
