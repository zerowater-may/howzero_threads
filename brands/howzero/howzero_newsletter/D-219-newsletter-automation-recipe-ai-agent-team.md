**레시피 #80 구성:**

하우제로에서 실제 운영 중인 AI 에이전트 5명(?)을 소개합니다.

1. **에이전트 "리서치"**
   - 역할: 시장 동향, 경쟁사, 법규 변경 모니터링
   - 매일 아침 8시, 리서치 브리핑 자동 생성
   - 중요도 판단 → 슬랙 채널에 자동 게시

2. **에이전트 "CS"**
   - 역할: 고객 문의 1차 대응
   - 질문 분류 → 답변 초안 생성 → 검토 대기열에 넣기
   - 단순 문의는 자동 답변 (인간 검토 없이)
   - 복잡한 문의만 팀원에게 에스컬레이션

3. **에이전트 "콘텐츠"**
   - 역할: 뉴스레터/SNS 초안 작성
   - 주간 주제 제안 → 초안 작성 → 편집 대기열
   - 최종 편집과 톤앤매너는 제가 직접

4. **에이전트 "데이터"**
   - 역할: 매출, 마케팅, 제품 데이터 분석
   - 이상치 감지 → 자동 알림
   - 주간/월간 리포트 자동 생성

5. **에이전트 "오퍼레이션"**
   - 역할: 팀 태스크 관리, 일정 조율
   - 미완료 태스크 리마인더
   - 미팅 노트 요약 + 액션 아이템 추출

---

AI 에이전트 팀 도입 결과:
- 팀원 업무 시간 절감: 평균 주 10시간
- CS 1차 응답 시간: 4시간 → 5분
- 리포트 작성 시간: 주 8시간 → 0시간 (자동)
- 인건비 절감 효과: 약 ₩3억/년

주의할 점.
AI 에이전트는 팀원을 대체하는 게 아닙니다.
팀원이 더 중요한 일에 집중하게 해주는 겁니다.

15명 + AI 에이전트 5개 = 사실상 20명의 팀.
