안녕하세요, 하우제로입니다.

아카데미 1기가 끝났습니다. 4주간의 결과를 숨김없이 공개합니다.

먼저 솔직하게. 20명 전원이 성공한 건 아닙니다. 그리고 저도 완벽한 프로그램을 만든 건 아닙니다. 하지만 배운 게 많습니다.

---

📊 **1기 전체 결과**

| 항목 | 수치 |
|------|------|
| 신청자 | 87명 |
| 선발 | 20명 |
| 수료 | 17명 (85%) |
| 자동화 시스템 구축 완료 | 14명 (70%) |
| 만족도 | 4.6/5.0 |
| 평균 시간 절감 | 주당 8.3시간 |
| 매출 증가 보고 | 5명 (평균 +23%) |

---

💡 **잘된 것 3가지**

**1. 실습 중심 커리큘럼이 먹혔습니다.**

매주 하나의 자동화를 직접 구축하는 방식. 이론은 30%, 실습이 70%. 수료 후 설문에서 "실제로 만들어보는 게 가장 도움됐다"가 1위였어요.

**2. 동기 부여 효과가 컸습니다.**

혼자 하면 3일이면 포기하는데, 20명이 함께 하니까 끝까지 갔습니다. 슬랙에서 매일 진행 상황을 공유하는 구조가 효과적이었어요.

**3. 실무 적용률이 높았습니다.**

자동화를 "배운" 게 아니라 "적용한" 비율이 70%. 이건 온라인 코스(평균 적용률 15~20%)와는 차원이 다릅니다.

---

🔥 **못된 것 3가지**

**1. 3명이 중도 이탈했습니다.**

이유: 업무가 바빠서(2명), 기술 수준 차이(1명). 사전 레벨 테스트를 더 꼼꼼하게 해야 했습니다.

**2. 1:1 멘토링 시간이 부족했습니다.**

20명에게 개인별 피드백을 주는 게 물리적으로 빡빡했어요. 2기에서는 정원을 줄이거나, 인증 강사와 분담해야 합니다.

**3. 과제 난이도 편차가 컸습니다.**

Week 1은 너무 쉽고, Week 4는 너무 어려웠다는 피드백. 중간 단계를 하나 더 넣어야 합니다.

---

🎯 **수료생 성과 하이라이트**

```
이커머스 대표 A: CS 자동화로 주당 12시간 절감
컨설팅 회사 B: 리포트 자동화로 월 ₩200만 인건비 절감
1인 기업 C: 이메일 시퀀스 자동화로 전환율 18% → 27%
스타트업 D: 온보딩 자동화로 고객 이탈률 40% 감소
프리랜서 E: 견적서 자동화로 응답 시간 24시간 → 2시간
```

이 분들의 변화를 보면서, 아카데미를 만들길 잘했다고 느꼈습니다.
