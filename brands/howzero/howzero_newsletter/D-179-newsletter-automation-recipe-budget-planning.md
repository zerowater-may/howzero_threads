안녕하세요, 하우제로입니다.

연말이면 돌아오는 악몽. 내년도 예산 수립.

각 팀에서 올라오는 엑셀 파일을 취합하고, 숫자가 안 맞아서 다시 보내고, 경영진 피드백 받아서 수정하고. 이 과정을 3주~4주 반복합니다.

하우제로는 이걸 3일로 줄였습니다. 60번째 레시피답게, 좀 야심찬 자동화입니다.

🍳 **이번 주 레시피: 예산 자동 수립**

**난이도**: ★★★★☆ (중상급)
**세팅 시간**: 8시간
**효과**: 예산 수립 기간 80% 절감, 예산 정확도 향상

---

🔧 **자동화 구조**

**1. 과거 데이터 기반 초안 생성**

```
자동 예산 초안:
→ 최근 12개월 실적 데이터 자동 추출 (매출, 비용, 투자)
→ 항목별 증가율 트렌드 자동 계산
→ 시즌별 패턴 반영 (분기별 편차 보정)
→ 인플레이션율 + 업계 성장률 자동 반영
→ 부서별 예산 초안 자동 생성
→ 전년 대비 증감 자동 표시
```

**2. 시나리오 시뮬레이션**

```
3가지 시나리오 자동 생성:
→ 보수적 (Conservative): 전년 대비 +5%
→ 기본 (Base): 트렌드 기반 예측
→ 공격적 (Aggressive): 성장 목표 반영
→ 각 시나리오별: 매출, 비용, 영업이익, 현금흐름 자동 산출
→ 손익분기점 자동 계산
→ "이 투자를 하면 ROI가 얼마인가" 자동 시뮬레이션
```

**3. 협업 자동화**

```
예산 수립 워크플로:
→ 부서별 예산 초안 자동 배포 (담당자에게)
→ 수정 요청 → 변경 이력 자동 추적
→ 부서 간 충돌 항목 자동 감지 (예: 마케팅↑ vs 비용절감↓)
→ 경영진 승인 요청 자동 발송
→ 승인 후 ERP 자동 반영
→ 월별 예산 vs 실적 자동 비교 리포트
```

📊 **하우제로 예산 수립 변화**

| 지표 | 수동 (작년) | 자동화 (올해) |
|------|-----------|-------------|
| 수립 기간 | 4주 | 3일 |
| 관련자 투입 시간 | 120시간 | 15시간 |
| 수정 횟수 | 7회 | 2회 |
| 예산 vs 실적 오차 | ±25% | ±12% |
| 시나리오 수 | 1개 | 3개 |

오차율이 25%에서 12%로 줄어든 게 가장 큰 변화입니다. 과거 데이터 기반 예측이 사람의 감보다 정확했어요.

⚠️ **주의사항**

- 자동 생성된 예산은 "출발점"이지 "최종안"이 아닙니다. 전략적 판단은 사람이 해야 합니다
- 새로운 사업이나 신규 투자는 과거 데이터가 없으므로 별도로 수립하세요
- 예산은 세우는 것보다 추적하는 게 더 중요합니다. 월별 자동 비교 리포트를 꼭 활용하세요
