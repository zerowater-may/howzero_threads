안녕하세요, 하우제로입니다.

드디어 됐습니다. 하우제로 인증 강사 1호.

왜 이게 중요한지 설명드리겠습니다.

지난 1년간 기업 교육은 전부 제가 직접 했습니다. 월 5~6건이 물리적 한계. 대기 리스트가 3개월치 밀려 있었어요. 이게 매출 ₩5,000만 벽의 원인이었습니다.

오늘부터 그 벽이 깨지기 시작합니다.

---

👤 **인증 강사 1호: 김정민님**

- 경력: AI 컨설팅 5년, 대기업 DX 프로젝트 리드
- 하우제로 코스 시즌1~3 전과정 수료
- 커뮤니티 활동 8개월, 멘토링 참여 30건+
- 3개월 강사 양성 프로그램 수료

정민님은 커뮤니티에서 먼저 다른 멤버들을 도우면서 눈에 띈 분입니다. 지식도 중요하지만, "남을 가르치는 능력"은 별개의 스킬이거든요. 그 두 가지를 다 가진 분이었습니다.

---

📋 **강사 양성 프로그램 — 어떻게 만들었나**

3개월 과정입니다. 대충 가르치고 보내지 않았습니다.

**1개월차: 콘텐츠 마스터리**
```
→ 하우제로 커리큘럼 전체 재학습
→ 각 모듈별 핵심 포인트 자체 정리
→ 주 2회 모의 강의 + 피드백
```

**2개월차: 실전 투입**
```
→ 하우제로 교육에 보조 강사로 참여 (5건)
→ 수강생 피드백 직접 수집 + 반영
→ 자체 케이스 스터디 3건 개발
```

**3개월차: 독립 강의**
```
→ 단독 강의 3건 진행
→ 수강생 만족도 평가 (기준: 4.5/5.0 이상)
→ 정민님 결과: 4.7/5.0 ✓
```

---

💡 **이 구조가 비즈니스에 미치는 영향**

```
Before: 월 교육 5~6건 (제가 직접)
After:  월 교육 10~12건 (제가 5건 + 정민님 5건)

교육 매출 예상 변화:
Before: 월 ₩2,340만
After:  월 ₩4,200만 (+80%)
```

여기서 끝이 아닙니다. 인증 강사 2호, 3호가 나오면 제가 교육에서 완전히 빠질 수 있어요. 그러면 저는 콘텐츠와 시스템 구축에만 집중할 수 있습니다.

이게 "나 없이도 돌아가는 비즈니스"의 첫 번째 퍼즐입니다.
