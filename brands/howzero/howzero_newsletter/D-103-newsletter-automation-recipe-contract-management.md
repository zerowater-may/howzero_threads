계약서 관리는 "중요한데 안 하는" 업무의 대표입니다. 안 하다가 사고 나면 크게 터지죠.

🔧 **만드는 법**

**Step 1: 계약서 중앙 저장소 구축 (5분)**

```
모든 계약서를 Google Drive 한 폴더에 모아줘.
각 계약서 PDF를 읽고 다음 정보를 추출해서
Google Sheets에 자동 정리:

1. 계약 상대방
2. 계약 유형 (서비스, 고용, 임대 등)
3. 시작일 / 종료일
4. 자동 갱신 여부 및 조건
5. 핵심 조건 (금액, SLA, 해지 조건)
6. 담당자
```

**Step 2: 만료/갱신 알림 자동화 (5분)**

```
계약 관리 시트를 매일 확인하고:

- 만료 90일 전: "검토 필요" 알림 + 갱신/해지 판단 요청
- 만료 30일 전: "조치 필요" 경고 + 담당자 리마인더
- 만료 7일 전: "긴급" 알림 + 대표에게 에스컬레이션
- 자동 갱신 계약: 갱신 30일 전 사전 알림

알림에 포함할 정보:
- 계약 요약 (핵심 조건 3줄)
- 갱신 시 예상 비용
- 작년 대비 변동 사항
```

**Step 3: 계약서 질의응답 (5분)**

```
계약서 내용에 대해 질문하면 답변해줘:

예시 질문:
- "A사 계약의 해지 통보 기간은?"
- "이번 달 만료되는 계약은?"
- "₩100만 이상 계약 목록"
- "자동 갱신되는 계약 전부"

계약서 원문에서 근거를 찾아서 답변.
근거를 못 찾으면 "해당 계약서에서 확인 불가" 표시.
```
