회의가 끝나면 뭘 하시나요?
대부분 이럽니다:

1. 녹음 파일 돌려듣기 (30분)
2. 핵심 내용 정리 (20분)
3. 액션 아이템 뽑기 (10분)
4. 참석자에게 공유 (5분)

1시간짜리 회의에 1시간짜리 후처리.
이건 시간 낭비입니다.

---

🍳 **이번 주 레시피: 회의록 자동화**

**난이도**: ★★☆☆☆ (초보 가능)
**세팅 시간**: 5분
**절감 효과**: 회의당 40~60분

---

📋 **재료**

- 녹음/녹화 도구 (Zoom, Google Meet, 또는 폰 녹음)
- Clova Note 또는 Whisper (음성→텍스트 변환)
- Claude Code (요약 + 구조화)

---

🔧 **만드는 법**

**Step 1: 회의 녹음 (0분 — 원래 하던 대로)**

Zoom이면 녹화 버튼, 대면이면 폰 녹음.
이미 하고 계실 겁니다.

**Step 2: 텍스트 변환 (자동 — 2분)**

녹음 파일을 Clova Note나 Whisper에 넣으면
자동으로 텍스트로 변환됩니다.
정확도 95% 이상.

**Step 3: AI 구조화 (3분)**

변환된 텍스트를 Claude Code에 넣고:

```
이 회의 녹취록을 다음 구조로 정리해줘:

1. 회의 요약 (3줄)
2. 핵심 논의 사항 (항목별)
3. 결정된 사항
4. 액션 아이템 (담당자 + 마감일)
5. 다음 회의 안건

톤: 비즈니스 문서, 간결하게.
```

---

✅ **완성!**

수동: 녹음 돌려듣기 30분 + 정리 20분 + 공유 5분 = 55분
AI: 변환 2분 + AI 정리 1분 + 검토 3분 = 6분

**절감: 회의당 49분**

주 5회 회의라면 → **주 4시간 절감**

---

⚡ **프로 팁**

1. 회의 시작 시 "오늘 안건: A, B, C"를 말하면 AI 정리 정확도 올라감
2. 액션 아이템은 "누가, 언제까지, 뭘"을 명확히 말하세요
3. 정기 회의는 템플릿을 만들어두면 매번 같은 구조로 정리됨

---

⚠️ **주의사항**

- 민감한 회의(인사, 법률)는 외부 AI 사용 전 보안 확인
- 녹음 전 참석자 동의 필수 (법적 요구사항)
- AI 회의록도 반드시 검토 후 공유
