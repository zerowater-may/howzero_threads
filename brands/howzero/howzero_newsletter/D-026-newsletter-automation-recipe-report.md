**난이도**: ★★☆☆☆

---

📋 **재료**

- Claude Code
- 데이터 파일 (엑셀, CSV, 구글 시트)
- 보고서 양식 (있으면 좋고, 없어도 됨)

---

🔧 **만드는 법**

**Step 1: 데이터 준비 (2분)**

보고서에 넣을 데이터를 준비합니다.
매출 데이터든, 마케팅 데이터든, 프로젝트 진행률이든.

**Step 2: 보고서 구조 지정 (3분)**

```
이 데이터로 주간 보고서를 만들어줘.

구조:
1. 핵심 요약 (3줄 이내)
2. 주요 지표 표 (이번 주 / 전주 / 변화율)
3. 주목할 점 (좋은 것 2개, 개선할 것 2개)
4. 다음 주 계획 (3가지)

톤: 간결하고 데이터 기반으로.
표와 볼드를 사용해줘.
```

**Step 3: 생성+검토 (5분)**

AI가 생성한 보고서를 검토합니다.
수정할 부분만 "이 부분은 이렇게 바꿔줘"라고 하면 됩니다.

---

💡 **프로 팁**

1. **양식 저장**: 한 번 만든 좋은 양식은 저장해두면 다음부터 더 빠름
2. **정기 자동화**: "매주 월요일 9시에 이 데이터로 보고서 만들어줘" → 자동
3. **다중 보고서**: 같은 데이터 → 상사용(요약), 팀용(상세), 고객용(핵심만)

---

⏱️ **절감 효과**

| 보고서 | 수동 | AI |
|--------|------|-----|
| 주간 보고 | 45분 | 2분 |
| 월간 보고 | 3시간 | 5분 |
| 프로젝트 보고 | 1시간 | 3분 |

주간 보고만 자동화해도 월 3시간 절감.
