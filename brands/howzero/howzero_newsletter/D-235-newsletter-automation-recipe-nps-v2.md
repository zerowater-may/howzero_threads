---

**레시피 #88 구성:**

1. **실시간 NPS 트래킹**
   - 분기별 설문이 아니라 지속적 수집
   - 주요 터치포인트(온보딩 완료, 3개월, 6개월, 1년)마다 자동 측정
   - NPS 트렌드 실시간 대시보드

2. **NPS 예측 모델**
   - 고객의 행동 데이터 → NPS 점수 예측
   - 설문에 답하지 않은 고객도 예측 NPS 산출
   - "이 고객은 다음 설문에서 Detractor가 될 확률 78%"
   - 사전 개입 대상 자동 리스트업

3. **세그먼트별 심화 분석**
   - Promoter(9-10): 왜 좋아하는지 → 강점 강화 인사이트
   - Passive(7-8): 뭘 바꾸면 Promoter가 되는지 → 개선 포인트
   - Detractor(0-6): 왜 불만인지 → 즉각 대응 트리거
   - 업종별, 요금제별, 사용 기간별 세그먼트 자동 분석

4. **NPS 기반 자동 액션**
   - Promoter → 리뷰 요청 + 레퍼럴 프로그램 자동 초대
   - Passive → 맞춤 혜택 자동 제안 (업그레이드 할인 등)
   - Detractor → CS 매니저 즉시 연결 + 이슈 해결 추적
   - 모든 액션의 결과를 다시 NPS에 반영 → 피드백 루프
