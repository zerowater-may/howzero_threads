SNS 콘텐츠 만드느라 하루 2시간 쓰고 계신다면,
이번 레시피가 인생을 바꿔줄 겁니다.

저도 예전엔 플랫폼마다 따로 만들었습니다.
지금은 콘텐츠 1개 만들면 7개가 자동으로 나옵니다.

---

🍳 **이번 주 레시피: SNS 콘텐츠 자동화**

**난이도**: ★★☆☆☆ (초보 가능)
**세팅 시간**: 10분
**절감 효과**: 하루 1.5~2시간

---

📋 **재료**

- Claude Code (월 $20)
- 블로그 글 또는 뉴스레터 원본 1개
- 각 플랫폼 계정

---

🔧 **만드는 법**

**Step 1: 원본 콘텐츠 준비 (이미 있음)**

블로그 글, 뉴스레터, 유튜브 대본 — 뭐든 됩니다.
핵심은 "원본 1개가 있어야 한다"는 겁니다.

**Step 2: 변환 프롬프트 입력 (3분)**

```
이 글을 다음 플랫폼에 맞게 변환해줘:

1. 링크드인 포스트 (300자, 전문적 톤, 줄바꿈 많이)
2. 트위터/X 스레드 (5개 트윗, 각 280자)
3. 인스타그램 캡션 (200자 + 해시태그 10개)
4. 유튜브 쇼츠 대본 (30초, 훅→문제→해결→CTA)
5. 틱톡 대본 (15초, 임팩트 중심)
6. 페이스북 포스트 (200자, 캐주얼 톤)
7. 블로그 요약 (100자, SEO 메타 디스크립션)

각 플랫폼의 알고리즘 특성을 반영해줘.
```

**Step 3: 검토 + 발행 (5분)**

AI가 만든 초안을 훑어보고,
어색한 표현만 수정하세요.

---

✅ **완성!**

수동: 7개 플랫폼 × 20분 = 140분
AI: 생성 30초 + 검토 5분 = 6분

**절감: 2시간 14분/일**

---

⚡ **프로 팁**

1. 요일별 메인 플랫폼을 정하세요 (월=링크드인, 화=트위터...)
2. 한 주에 쓸 원본 3개만 만들면 21개 콘텐츠가 나옵니다
3. 성과 좋은 포맷은 템플릿화해서 재사용

---

⚠️ **주의사항**

- 각 플랫폼 톤이 다릅니다. 링크드인 글을 틱톡에 그대로 쓰면 안 됩니다
- 해시태그는 트렌드에 맞게 주기적으로 업데이트
