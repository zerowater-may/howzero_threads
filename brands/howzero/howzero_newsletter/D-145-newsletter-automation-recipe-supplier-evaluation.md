안녕하세요, 하우제로입니다.

외주 업체, SaaS 공급사, 파트너사. 비즈니스를 하면 공급업체 선정이 반복됩니다.

보통 이 과정이 이렇습니다: 구글링, 지인 추천, 미팅 3~4번, 견적 비교, 레퍼런스 체크. 2주가 걸려요. 그리고 나서도 "이게 최선이었나?" 확신이 없습니다.

시스템으로 바꿉니다.

---

🍳 **이번 주 레시피: 공급업체 평가 자동화**

**난이도**: ★★★☆☆ (중급)
**세팅 시간**: 2~3시간
**효과**: 평가 시간 80% 절감, 객관적 의사결정

---

🔧 **4단계 자동 평가 파이프라인**

**Step 1: 후보 자동 수집**

```
평가 요청 등록 시:
→ 카테고리별 공급업체 DB에서 후보 자동 추출
→ G2, Capterra 등에서 평점·리뷰 자동 수집
→ 회사 정보 (설립일, 규모, 매출) 자동 조회
→ 후보 10~15개 자동 리스트업
```

**Step 2: 자동 스코어링**

```
평가 기준 (가중치 커스텀 가능):
→ 가격 경쟁력 (25%)
→ 기능 적합도 (25%)
→ 시장 평판 (20%)
→ 기술 지원 (15%)
→ 확장성/연동성 (15%)

AI 분석:
→ 각 후보를 기준별 0~100점 자동 평가
→ 가중 합산 총점 산출
→ Top 3 자동 선정
```

**Step 3: 비교 리포트 자동 생성**

```
GPT-4 API로 비교 분석 리포트:
→ Top 3 업체 상세 비교표
→ 각 업체의 장단점 요약
→ "하우제로 상황에서는 A업체 추천. 이유: ..."
→ 리스크 요인 별도 표기
→ PDF 자동 생성
```

**Step 4: 의사결정 + 기록**

```
→ 리포트 기반 최종 선정
→ 선정 이유 + 미선정 이유 자동 기록
→ 계약 후 분기별 성과 자동 추적
→ 성과 데이터 → 다음 평가 시 학습 데이터로 활용
```

📊 **하우제로 적용 사례**

최근 영상 편집 외주 업체를 선정할 때 이 시스템을 사용했습니다.

| 항목 | 수동 (이전) | 자동 (현재) |
|------|------------|------------|
| 후보 조사 | 5일 | 30분 |
| 비교 분석 | 3일 | 1시간 |
| 최종 결정 | 2일 | 당일 |
| 총 소요 시간 | 10일 | 2시간 |
| 선정 후 만족도 | 3.5/5 | 4.3/5 |

만족도가 올라간 이유? 감이 아니라 데이터로 결정하니까, "혹시 더 좋은 데가 있었을까" 하는 후회가 없어요.

---

⚠️ **주의사항**

- 자동 수집 데이터는 공개 정보 기반입니다. 최종 결정 전 직접 미팅은 필수
- 스코어링 기준은 프로젝트마다 달라야 합니다. 만능 기준은 없어요
- 기존 공급업체도 정기 평가하세요. "처음엔 좋았는데..."를 방지합니다
