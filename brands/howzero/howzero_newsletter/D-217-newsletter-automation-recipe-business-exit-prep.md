"사업 매각이요? 저는 매각 생각 없는데요."

그래도 읽어보세요.
매각 준비가 잘 된 사업 = 잘 운영되는 사업이니까.

매각할 때 바이어가 보는 것:
- 매출이 대표 개인에 의존하는가?
- 시스템으로 돌아가는가?
- 데이터가 정리되어 있는가?

이 3가지가 곧 좋은 사업의 조건입니다.

---

**레시피 #79 구성:**

1. **사업 가치 자동 산정**
   - 월 매출, 이익률, 성장률, 이탈률 데이터 자동 수집
   - SDE(Seller's Discretionary Earnings) 자동 계산
   - 업종별 멀티플 적용 → 예상 매각가 자동 산출
   - 매월 업데이트: "지금 팔면 얼마?"

2. **대표 의존도 분석**
   - 대표가 직접 관여하는 업무 자동 식별
   - 대표 부재 시 매출 영향도 시뮬레이션
   - "대표 의존도 점수": 100점(완전 의존) → 0점(완전 시스템화)
   - 점수 낮추기 위한 구체적 액션 자동 제안

3. **재무 데이터 자동 정리**
   - 손익계산서, 현금흐름표 자동 생성 (월별/분기별)
   - 매출원별 분류 자동화
   - 비용 항목 정리 + 이상치 자동 감지
   - 실사(Due Diligence) 대응 자료 자동 패키징

4. **SOP(표준운영절차) 자동 문서화**
   - 팀의 업무 프로세스 → AI가 자동으로 SOP 문서화
   - 미문서화 업무 자동 감지 → 작성 요청
   - SOP 커버리지 대시보드: "전체 업무의 몇 %가 문서화?"

---

하우제로 자체 진단 결과:
- 예상 매각가: ₩80~120억 (매출 ₩24억 × 멀티플 3.3~5)
- 대표 의존도: 35점 (1년 전 65점에서 개선)
- SOP 커버리지: 82%

매각할 생각은 아직 없습니다.
하지만 "언제든 매각할 수 있는 상태"가 목표입니다.
