안녕하세요, 하우제로입니다.

리스크는 터진 후에 아는 게 아닙니다. 터지기 전에 잡아야 합니다.

하우제로가 15개월차에 해외 매출을 확대할 때, 환율 변동으로 예상 수익의 12%가 증발한 적 있습니다. 미리 환율 리스크를 모니터링했으면 헤징을 걸었을 거예요. 그때 만든 게 이 시스템입니다.

---

🍳 **이번 주 레시피: 비즈니스 리스크 분석 자동화**

**난이도**: ★★★★☆ (고급)
**세팅 시간**: 4시간
**효과**: 리스크 사전 감지, 의사결정 속도 2배 향상

---

🔧 **4가지 리스크 모니터링 모듈**

**1. 매출 리스크**

```
자동 모니터링:
→ 고객 집중도: 상위 3개 고객 매출 비중 추적
  (50% 이상 → "위험" 알림)
→ 구독 이탈 추세: 3개월 이동 평균
→ 파이프라인 건강도: 예상 매출 vs 목표 갭
→ 계절성 패턴: 전년 동기 대비 이상치 감지
```

**2. 현금흐름 리스크**

```
→ 런웨이 자동 계산 (현재 현금 / 월 고정비)
→ 3개월 현금흐름 예측
→ 대형 지출 예정 자동 알림 (D-30)
→ 미수금 에이징 추적 (30/60/90일)
→ 런웨이 6개월 미만 → 경고
```

**3. 운영 리스크**

```
→ 핵심 인력 의존도 분석 (특정인 업무 비중 60% 이상 → 경고)
→ 주요 SaaS/API 가동률 모니터링
→ 자동화 시스템 에러율 추적
→ 데이터 백업 상태 확인 (자동)
→ 보안 취약점 스캔 (월 1회)
```

**4. 시장 리스크**

```
AI 자동 모니터링:
→ 경쟁사 가격 변동 (웹 스크래핑)
→ 산업 뉴스 자동 수집 + AI 요약
→ 규제 변경 탐지 (개인정보보호법, AI 규제 등)
→ 환율 변동 추적 (해외 매출이 있을 경우)
→ 주간 시장 리스크 브리핑 자동 생성
```
