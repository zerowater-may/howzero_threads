안녕하세요, 하우제로입니다.

유튜브 구독자가 합산 30,000명을 넘었습니다. 한국어 채널 25,000명, 영어 채널 5,200명.

17개월 전에 채널을 만들었을 때 첫 영상 조회수가 23회였습니다. 댓글 0, 좋아요 2 (하나는 제 것). 그때 "이거 의미 있나?" 싶었어요.

의미 있었습니다. 30,000명이 증명합니다.

---

| 항목 | 수치 | 전월 대비 |
|------|------|----------|
| 월 매출 | ₩6,500만 | +4.8% |

| 유튜브 (영어) | 5,200명 | +1,100 |

| 유튜브 → 매출 기여 | ₩1,950만 | +22% |

---

💡 **유튜브가 비즈니스에 미치는 영향**

```
유튜브 → 비즈니스 전환 경로:

영상 시청 → 뉴스레터 구독 (전환율 1.2%)
           → AI Score 진단 (전환율 0.8%)
           → 코스 구매 (전환율 0.3%)
           → 오딧 의뢰 (전환율 0.05%)

월 480,000 조회수 기준:
→ 뉴스레터 신규: ~5,760명/년
→ AI Score 진단: ~3,840건/년
→ 코스 구매: ~1,440건/년
→ 오딧 의뢰: ~240건/년
```

유튜브는 광고 수익이 아닙니다. 매출의 30%를 유튜브가 만들어줍니다. 콘텐츠가 24시간 일하는 영업사원이에요.

---

🔥 **30,000명까지의 성장 패턴**

```
 0~1,000명 (0~4개월):   가장 힘든 구간. 포기할 뻔
 1,000~5,000명 (5~8개월):  알고리즘이 밀어주기 시작
 5,000~15,000명 (9~13개월): 안정적 성장
 15,000~30,000명 (14~17개월): 가속
```

**전환점은 5,000명이었습니다.**

5,000명까지는 "좋은 콘텐츠를 꾸준히 올리면 된다"였고, 5,000명 이후는 "시스템으로 올리면 된다"로 바뀌었어요.

---

🎯 **콘텐츠 시스템**

```
주간 콘텐츠 생산 루틴 (총 6시간):

월요일: 주제 선정 + 대본 AI 초안 (1시간)
화요일: 대본 수정 + 촬영 (2시간)
수요일: 편집 (에디터 외주)
목요일: 썸네일 + SEO 최적화 (30분)
금요일: 업로드 + 멀티 채널 배포 자동화 (30분)

1개 영상 → 7개 콘텐츠로 변환:
유튜브 롱폼, 유튜브 쇼츠 3개,
LinkedIn 포스트, 뉴스레터 인용, 블로그 포스트
```

주 6시간으로 7개 플랫폼에 콘텐츠를 올립니다. 효율의 핵심은 "원 소스 멀티 유즈"입니다.

📈 **다음 목표: 실버 버튼 (100,000명)**

솔직히 30,000명에서 100,000명은 다른 차원의 게임입니다. 하지만 지금의 성장률이 유지되면 12~14개월 후에 가능합니다.
