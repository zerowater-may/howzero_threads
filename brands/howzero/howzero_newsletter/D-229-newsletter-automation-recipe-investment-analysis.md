사업이 커지면 투자 제안을 받습니다.
"투자 받으시겠어요?"

반대로, 사업의 여유 자금을 어디에 넣을지도 고민됩니다.
"이 사업에 투자할까?"

두 방향 모두, 데이터 없이 결정하면 위험합니다.

---

**레시피 #85 구성:**

1. **투자 제안 자동 분석 (받는 입장)**
   - VC/엔젤 투자 제안서 업로드 → AI 자동 분석
   - 밸류에이션 적정성 판단 (업종별 멀티플 비교)
   - 투자 조건 핵심 요약 (지분율, 이사회 구성, 청산 우선권 등)
   - "이 조건으로 투자 받으면 5년 후 시나리오" 시뮬레이션

2. **사업 투자 자동 실사 (주는 입장)**
   - 대상 기업 공개 정보 자동 수집 (재무, 뉴스, 리뷰, 직원 평판)
   - 핵심 지표 자동 계산 (매출 성장률, 이탈률, LTV/CAC 등)
   - 경쟁사 비교 분석 자동 생성
   - 리스크 요인 자동 식별 (3단계: Low/Medium/High)

3. **포트폴리오 대시보드**
   - 기투자 건 성과 실시간 추적
   - 예상 ROI vs 실제 ROI 비교
   - 포트폴리오 전체 리스크 분산도 분석

4. **투자 메모 자동 생성**
   - 분석 결과 → 투자 의사결정 메모 자동 작성
   - "투자 추천 / 보류 / 거절" 의견과 근거
   - 이사회/파트너 공유용 리포트 자동 포맷팅

---

하우제로의 실제 활용:

지난 1년간 투자 제안 8건 수신.
AI 분석 결과:
- 추천: 2건
- 보류: 3건
- 거절: 3건

실제 의사결정:
- 모두 거절. (지금은 투자보다 사업 성장에 집중.)

하지만 이 분석 도구 덕분에 "왜 거절하는지"를 명확히 설명할 수 있었습니다.
"감사합니다만, 현재 우리 성장 단계에서 이 밸류에이션은..."
데이터 기반 거절. 관계는 유지.

투자 결정은 감이 아니라 데이터로.
