문제는 취합이 끝날 때쯤이면 이미 한 달이 지나 있다는 겁니다. 뒤늦게 "이 매장 매출이 떨어졌네" 확인해봤자 이미 늦은 거예요.

실시간으로 비교하고, 이상 신호를 즉시 잡는 시스템을 만들었습니다.

🍳 **이번 주 레시피: 프랜차이즈 벤치마크 자동화**

**난이도**: ★★★☆☆ (중급)
**세팅 시간**: 5시간
**효과**: 매장 성과 분석 시간 90% 절감, 저성과 매장 조기 발견

---

🔧 **자동화 구조**

**1. 매장 데이터 자동 수집**

```
POS/ERP 연동 자동 수집:
→ 일매출, 주매출, 월매출
→ 객단가, 방문 고객 수
→ 인건비율, 원재료비율
→ 재방문율 (멤버십 데이터 기반)
→ 고객 리뷰 평점 (네이버/카카오맵 크롤링)
```

**2. 벤치마크 자동 분석**

```
매장 간 자동 비교:
→ 전체 평균 대비 각 매장 성과 (상/중/하 자동 분류)
→ 동일 상권 매장끼리 비교 (공정한 비교)
→ 월별 추세 분석 (성장/정체/하락)
→ TOP 5 / BOTTOM 5 매장 자동 추출
→ "성공 매장 패턴" 자동 분석 (무엇이 다른가)
```

**3. 알림 시스템**

```
이상 감지:
→ 매출 전월 대비 20% 이상 하락: 즉시 알림
→ 원재료비율 업계 평균 초과: 주간 알림
→ 고객 리뷰 평점 4.0 이하 하락: 즉시 알림
→ 3개월 연속 하위 10% 매장: 본사 특별 관리 대상 자동 지정
```
