🔧 **자동화 구조**

**1. 이탈 신호 데이터 수집**

```
자동 추적 지표 (개인정보 최소화 원칙):
→ 슬랙/메신저 활동량 변화 (메시지 수, 반응 수)
→ 1:1 미팅 참석률 + 미팅 분위기 점수 (관리자 입력)
→ 프로젝트 참여도 (태스크 완료율, 지연율)
→ 근무 패턴 변화 (갑작스러운 야근 증가/감소)
→ 설문 응답 변화 (펄스 서베이 점수 추이)
```

**2. 리스크 점수 자동 산출**

```
가중치 기반 종합 점수:
→ 활동량 30% 이상 감소: +25점
→ 1:1 미팅 2회 연속 스킵: +20점
→ 펄스 서베이 점수 하락: +20점
→ 프로젝트 지연율 50% 이상: +15점
→ 근무 패턴 급변: +10점
→ 입사 기념일 ± 1개월: +10점 (이직 피크 타이밍)

→ 70점 이상: 🔴 고위험
→ 40~69점: 🟡 주의
→ 39점 이하: 🟢 안정
```

**3. 자동 대응**

```
리스크 단계별 알림:
→ 🟡 주의: 매니저에게 주간 리포트에 포함
→ 🔴 고위험: 매니저 + HR에 즉시 알림
→ 매니저에게 "대화 가이드" 자동 제공
→ 월간: 팀별 이탈 리스크 요약 리포트
```
