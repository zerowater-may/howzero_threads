**레시피 #87 구성:**

1. **경쟁사 특허 자동 추적**
   - 경쟁사 리스트 등록 → 신규 특허 출원 시 즉시 알림
   - KIPRIS(한국), USPTO(미국), JPO(일본) 자동 크롤링
   - 키워드 기반 관련 특허 자동 필터링

2. **특허 분석 자동화**
   - 신규 특허 청구항 → AI 요약 (비전문가가 읽을 수 있게)
   - "우리 제품/기술과 겹치는가?" 자동 유사도 분석
   - 유사도 High → 즉시 알림 + 법무 검토 요청 자동 생성

3. **특허 맵 자동 생성**
   - 우리 업종의 특허 현황 시각화
   - 누가 어떤 영역에 특허를 가지고 있는지 맵핑
   - "특허 공백 지역" 자동 식별 (출원 기회)
   - 시간별 특허 트렌드: 어떤 기술이 뜨고 있는지

4. **자사 특허 관리**
   - 보유 특허 만료일 자동 추적
   - 갱신 비용 및 기한 자동 알림
   - 유지/포기 의사결정 지원 (활용도 기반)
