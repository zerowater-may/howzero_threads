안녕하세요, 하우제로입니다.

팀이 10명이 되니까 한 가지 문제가 생겼습니다.

누가 뭘 잘하는지 정확히 모릅니다. 5명일 때는 다 알았어요. 매일 같이 일하니까. 10명이 되면 "A가 파이썬 할 줄 아나?", "B가 영어 미팅 가능한가?" 같은 질문에 바로 답이 안 됩니다.

그래서 스킬맵을 만들었습니다. 수동으로 시작했다가 자동화했습니다.

---

🍳 **이번 주 레시피: 직원 스킬맵 자동화**

**난이도**: ★★☆☆☆ (초중급)
**세팅 시간**: 3시간
**효과**: 적재적소 배치, 교육 계획 수립 효율화

---

🔧 **자동화 구조**

**1. 스킬 데이터 수집**

```
분기별 자동 수집:
→ 자기 평가 설문 (5점 척도, 10분 소요)
→ 동료 평가 (선택, 익명)
→ 프로젝트 이력에서 사용 기술 자동 추출
→ 외부 교육/자격증 이력 자동 반영
→ 스킬 카테고리: 기술, 비즈니스, 커뮤니케이션, 리더십
```

**2. 스킬맵 자동 생성**

```
시각화 자동 산출:
→ 팀 전체 스킬 히트맵 (강점/약점 한눈에)
→ 개인별 레이더 차트 (8개 핵심 역량)
→ 팀 내 스킬 갭 분석 (필요 역량 vs 보유 역량)
→ 역량별 Top 3 팀원 자동 태그
→ 분기 대비 성장 추이
```

**3. 자동 매칭 & 추천**

```
실무 활용 자동화:
→ 새 프로젝트 → 필요 스킬 입력 → 최적 팀 자동 추천
→ 스킬 갭 발견 → 관련 교육 과정 자동 추천
→ 멘토-멘티 자동 매칭 (고스킬자 ↔ 저스킬자)
→ 채용 시 "팀에 부족한 스킬" 자동 반영
```

---

📊 **하우제로 팀 스킬맵 요약**

| 스킬 영역 | 팀 평균 | 최고 | 갭(필요-보유) |
|----------|--------|------|-------------|
| AI/자동화 | 4.2 | 5.0 | -0.3 (충분) |
| 세일즈 | 3.5 | 4.5 | +0.8 (부족) |
| 콘텐츠 제작 | 3.8 | 4.8 | -0.1 (충분) |
| 영어 커뮤니케이션 | 2.8 | 4.0 | +1.5 (심각) |
| 프로젝트 관리 | 3.2 | 4.0 | +0.5 (보통) |

이 맵을 보고 결정한 것: 다음 채용은 "영어 + 세일즈" 역량을 가진 사람. 데이터가 우선순위를 정해줬습니다.

---

⚠️ **주의사항**

- 스킬맵은 평가가 아니라 성장 도구입니다. 인사고과에 직접 연결하면 정직한 자기 평가가 사라집니다
- 동료 평가는 반드시 익명으로. 솔직한 피드백이 가능해집니다
- 스킬 점수가 낮은 것은 "문제"가 아니라 "기회"입니다
