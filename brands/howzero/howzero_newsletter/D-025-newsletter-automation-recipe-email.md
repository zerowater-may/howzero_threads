🍳 **이번 주 레시피: 이메일 자동 분류+답변**

**난이도**: ★☆☆☆☆ (초보 가능)
**세팅 시간**: 5분
**절감 효과**: 하루 1~2시간

---

📋 **재료**

- Claude Code (월 $20)
- 이메일 수신함
- 지난 1주일간 받은 이메일 목록

---

🔧 **만드는 법**

**Step 1: 이메일 카테고리 정하기 (1분)**

대부분 이렇게 나뉩니다:
- 🔴 긴급 — 24시간 내 답변 필요
- 🟡 일반 — 이번 주 내 답변
- 🔵 참조 — 읽기만
- ⚫ 스팸/광고 — 무시

여러분 상황에 맞게 수정하세요.

**Step 2: 분류 프롬프트 입력 (2분)**

Claude Code에 이렇게 입력하세요:

```
이 이메일들을 다음 기준으로 분류해줘:
- 긴급: 거래처 클레임, 결제 관련, 마감 임박
- 일반: 문의, 제안, 협업 요청
- 참조: 뉴스레터, 공지, CC
- 스팸: 광고, 홍보

각 이메일에:
1. 카테고리
2. 1줄 요약
3. 추천 액션

을 붙여줘.
```

**Step 3: 답변 초안 자동 생성 (2분)**

```
긴급 이메일 3개에 대해 답변 초안을 작성해줘.
톤: 정중하지만 간결하게.
길이: 각 200자 이내.
```

---

✅ **완성!**

수동: 이메일 50개 분류 + 답변 = 2시간
AI: 분류 5초 + 답변 초안 10초 + 검토 10분 = 11분

**절감: 1시간 49분/일**

---

⚠️ **주의사항**

1. AI 답변은 반드시 검토 후 발송
2. 민감한 내용(계약, 법적 이슈)은 직접 작성
3. 처음 1주일은 분류 정확도 확인하면서 기준 조정
