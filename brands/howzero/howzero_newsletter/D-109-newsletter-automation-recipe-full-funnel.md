30번째 레시피입니다. 마지막이니까 가장 큰 그림을 그려봅니다.

지금까지 레시피 29개는 각각 하나의 업무를 자동화했습니다. 오늘은 이걸 하나로 연결합니다. 인지 → 관심 → 고려 → 구매 → 유지 → 추천. 고객 여정 전체를 하나의 파이프라인으로 만드는 겁니다.

이게 완성되면, 비즈니스가 시스템으로 돌아갑니다.

---

🍳 **이번 주 레시피: 풀퍼널 자동화**

**난이도**: ★★★★☆ (고급)
**세팅 시간**: 기존 레시피 조합이므로 1~2시간 (개별 레시피가 완성된 상태에서)
**효과**: 매출 파이프라인 전체 자동화

---

🔧 **6단계 풀퍼널 구조**

```
[인지] → [관심] → [고려] → [구매] → [유지] → [추천]
  ↓        ↓        ↓        ↓        ↓        ↓
콘텐츠   리드캡처  팔로업    견적/결제  CS+리텐션  NPS+추천
자동화    자동화   자동화    자동화     자동화    자동화
```

**Stage 1: 인지 — 콘텐츠 자동화 (레시피 #25)**

```
유튜브 영상 1개 → 7개 플랫폼 자동 배포
→ 잠재 고객이 콘텐츠를 발견
```

**Stage 2: 관심 — 리드 캡처 자동화**

```
콘텐츠 CTA → AI Score 진단 페이지
→ 이메일 수집 → 자동 태그 부여 (업종, 관심사)
→ 뉴스레터 자동 구독
```

**Stage 3: 고려 — 팔로업 자동화 (레시피 #24)**

```
구독 후 이메일 시퀀스 자동 발송:
Day 1: 환영 + AI Score 결과 해석
Day 3: 가장 관련 높은 레시피 추천
Day 7: 케이스 스터디 공유
Day 14: 오딧/교육 안내
```

**Stage 4: 구매 — 견적/결제 자동화 (레시피 #18)**

```
오딧 신청 → 일정 자동 예약 (레시피 #20)
→ 진단 결과 + 견적서 자동 생성
→ 전자 계약 + 결제 링크 자동 발송
```

**Stage 5: 유지 — CS + 리텐션 자동화 (레시피 #8, #21, #23)**

```
서비스 이용 중:
→ CS 문의 자동 분류 + 답변 (레시피 #8)
→ 이탈 신호 감지 + 선제 대응
→ 고객 세분화 + 맞춤 커뮤니케이션 (레시피 #23)
```

**Stage 6: 추천 — NPS + 추천 자동화**

```
서비스 3개월 후:
→ NPS 설문 자동 발송
→ 높은 점수 고객 → 추천 요청 + 인센티브 안내
→ 리뷰 요청 자동 발송 (레시피 #21)
→ 추천 유입 → Stage 1로 복귀 (선순환)
```
