안녕하세요, 하우제로입니다.

모든 고객에게 같은 이메일을 보내고 계신가요?

그럼 오픈율은 아마 15% 이하일 겁니다. 고객마다 관심사가 다르고, 구매 단계가 다르고, 니즈가 다릅니다. 같은 메시지로는 아무도 움직이지 않아요.

해답은 고객 세분화입니다. 근데 수동으로 하면 미칩니다. AI로 자동화하면 됩니다.

---

🍳 **이번 주 레시피: 고객 세분화 자동화**

**난이도**: ★★★☆☆ (중급)
**세팅 시간**: 20분
**효과**: 이메일 오픈율 2배, 전환율 3배

---

📋 **재료**

- Claude Code (월 $20)
- Google Sheets (고객 데이터)
- 이메일 마케팅 도구 (스티비, Mailchimp 등)

---

🔧 **만드는 법**

**Step 1: 고객 데이터 통합 (10분)**

```
다음 데이터를 Google Sheets에 모아줘:

- 구매 이력 (금액, 빈도, 최근 구매일)
- 이메일 활동 (오픈, 클릭, 미오픈)
- 웹사이트 방문 (페이지, 체류 시간)
- 코스/서비스 이용 현황
```

**Step 2: RFM 기반 자동 세분화 (5분)**

```
고객 데이터를 분석해서 5개 그룹으로 나눠줘:

1. VIP (상위 10%): 최근 구매 + 높은 빈도 + 높은 금액
2. 충성 고객: 꾸준한 구매, 중간 금액
3. 성장 가능 고객: 최근 첫 구매 or 빈도 증가 중
4. 이탈 위험 고객: 구매 빈도 감소 or 3개월 미구매
5. 휴면 고객: 6개월 이상 미활동

각 고객에 그룹 태그를 자동 부여하고,
매주 월요일에 그룹 변동 리포트 생성.
```

**Step 3: 그룹별 자동 액션 (5분)**

```
그룹별로 다른 액션을 자동 실행해줘:

VIP → 신규 서비스 얼리액세스 초대 이메일
충성 고객 → 감사 + 추천 프로그램 안내
성장 가능 → 다음 단계 코스/서비스 추천
이탈 위험 → "무엇이 아쉬우셨나요?" 설문 + 특별 혜택
휴면 → 재활성화 시퀀스 (3회 이메일 후 정리)

각 그룹별 메시지 톤과 CTA를 다르게 설정.
```

✅ **완성!**

하우제로 실제 적용 결과:

| 지표 | 세분화 전 | 세분화 후 |
|------|-----------|-----------|

| 클릭률 | 3.2% | 8.7% |
| 코스 전환율 | 2% | 6.5% |

**같은 수의 이메일을 보내도, 맞춤형이면 3배 더 팔립니다.**

⚠️ **주의사항**

- 고객 데이터는 개인정보보호법 준수 필수
- 세분화는 분기별 재검증. 고객 행동은 계속 바뀝니다
- 그룹 수가 너무 많으면 관리 불가. 5개가 적정
