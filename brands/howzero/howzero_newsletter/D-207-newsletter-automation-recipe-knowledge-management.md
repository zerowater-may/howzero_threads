"아, 그건 민수 씨가 알 텐데... 민수 씨 퇴사했죠."

이 대화, 익숙하시죠?

팀이 10명 넘으면 반드시 터지는 문제.
지식이 사람 머릿속에만 있습니다.

하우제로 팀도 15명이 되면서 같은 문제를 겪었습니다.
그래서 만들었습니다. AI 기반 지식 관리 시스템.

---

**레시피 #74 구성:**

1. **자동 지식 수집**
   - 슬랙 대화에서 Q&A 자동 감지 → 지식 베이스에 저장
   - 회의 녹음 → AI 요약 → 핵심 의사결정 자동 아카이빙
   - 노션/구글독스 변경사항 → 자동 인덱싱

2. **AI 검색 인터페이스**
   - "우리 환불 정책이 뭐야?" → 슬랙 봇이 즉시 답변
   - 자연어 검색. 문서 제목 몰라도 됩니다.
   - 답변 출처(원본 문서 링크) 항상 표시

3. **지식 갭 감지**
   - 같은 질문이 3번 이상 반복 → "문서화 필요" 알림
   - 6개월 이상 업데이트 안 된 문서 → 검토 요청 자동 발송
   - 신규 입사자 질문 패턴 분석 → 온보딩 문서 개선 제안

4. **퇴사자 지식 보존**
   - 퇴사 예정자의 슬랙/문서 활동 분석
   - "이 사람만 아는 지식" 자동 식별
   - 인수인계 체크리스트 자동 생성

---

결과:
- 신규 입사자 온보딩 기간: 4주 → 2주
- "이거 누가 알아요?" 질문: 일 평균 8건 → 1건
- 지식 검색 시간: 평균 15분 → 평균 30초

사람은 떠날 수 있습니다.
지식은 남아야 합니다.
