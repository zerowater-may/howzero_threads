📌 **Week 2 미션: 주간 보고서 자동화**

이번 주는 "매주 2시간 쓰던 보고서를 0분으로 만들기."

**목표**: 월요일 아침 출근하면 보고서가 이미 이메일에 와있는 상태.

**준비물**: Google Sheets (데이터) + Claude Code + 이메일 계정

**스텝 (11분)**:
1. Google Sheets 연결 (3분) — 매출/CS/고객 데이터 시트
2. AI 요약 프롬프트 (3분) — "이 데이터를 분석해서 주간 보고서 만들어줘. 포함: 핵심 지표, 전주 대비 변화, 인사이트 3개."
3. 이메일 자동 발송 (3분) — 수신자 설정 + 발송
4. 스케줄 설정 (2분) — 매주 월요일 09:00

**끝! 11분.**
