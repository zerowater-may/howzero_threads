안녕하세요, 하우제로입니다.

연간 계획을 세우는 건 쉽습니다. 지키는 게 어렵습니다.

하우제로도 1년차에 연간 계획을 세웠습니다. 2분기가 되니까 1분기 목표가 뭐였는지도 기억이 안 나더라고요. "계획을 세우는 것"과 "계획을 추적하는 것"은 완전히 다른 일입니다.

2년차에는 시스템을 만들었습니다.

---

🍳 **이번 주 레시피: 연간 계획 자동화**

**난이도**: ★★★☆☆ (중급)
**세팅 시간**: 3시간
**효과**: 목표 달성률 2배 향상, 전략 정렬

---

🔧 **4단계 시스템**

**Step 1: OKR 구조화**

```
연간 목표 → 분기 OKR → 월간 KPI → 주간 태스크

예시:
연간: 연매출 ₩10억
→ Q1 OKR: 파트너 프로그램 런칭, 해외 매출 20%
  → 1월 KPI: 파트너 5명 확보, 일본 세미나 개최
    → W1: 파트너 모집 페이지 오픈
    → W2: 지원서 검토 + 면담

전체 구조를 Notion DB에 자동 연결
```

**Step 2: 자동 추적 대시보드**

```
실시간 자동 집계:
→ 매출 데이터 (Stripe/은행 API) → KPI 자동 업데이트
→ 유튜브/SNS 데이터 → 성장 지표 자동 반영
→ 프로젝트 진행률 (Linear/Notion) → 자동 반영
→ 전체 OKR 달성률 자동 계산
→ 대시보드 매일 자동 갱신
```

**Step 3: 정기 리뷰 자동화**

```
주간 리뷰 (매주 금요일 16시 자동 생성):
→ 이번 주 KPI 달성 현황
→ 미달성 항목 + 원인 분석 (AI)
→ 다음 주 우선순위 자동 제안

월간 리뷰 (매월 말 자동 생성):
→ 월간 OKR 진척도
→ 전월 대비 변화
→ 목표 수정이 필요한 항목 플래그

분기 리뷰 (분기 말 자동 생성):
→ 분기 OKR 최종 평가
→ 연간 목표 대비 진도
→ 다음 분기 OKR 초안 AI 제안
```

**Step 4: 알림 + 넛지**

```
→ 주간 목표 미달성 시 월요일 오전 알림
→ KPI가 목표 대비 80% 미만 → 리마인드
→ 프로젝트 마감 D-3 → 자동 리마인드
→ 팀 전체 OKR 점수 주간 공유 (Slack)
→ 월간 MVP (가장 높은 달성률) 자동 선정
```

📊 **하우제로 실제 효과**

| 항목 | 시스템 도입 전 | 도입 후 |
|------|-------------|--------|
| 연간 목표 달성률 | 측정 안 함 | 78% |
| 분기 OKR 리뷰 실행 | 1/4 분기만 | 4/4 분기 |
| 주간 리뷰 | 가끔 | 매주 (자동) |
| 목표 수정 빈도 | 연 1회 (연초) | 분기 1회 (적응적) |
| 팀 목표 인지도 | 낮음 | 높음 (주간 공유) |

가장 큰 변화: 목표를 "기억"하지 않아도 됩니다. 시스템이 매주 상기시켜주니까요.

---

⚠️ **주의사항**

- OKR은 3~5개가 적당합니다. 10개 넘으면 집중력이 분산됩니다
- KPI는 "측정 가능한 것"만 넣으세요. "브랜드 인지도 향상"은 KPI가 아닙니다
- 분기 리뷰에서 목표를 수정하는 건 실패가 아니라 적응입니다
