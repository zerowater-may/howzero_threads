전체 자동화 v3.

v1 (레시피 #30): 풀퍼널 자동화. 인지→구매→유지.
v2 (레시피 #70): AI 에이전트 추가. 사람이 트리거, AI가 실행.
v3 (레시피 #89): **AI가 AI를 관리하는 시스템.**

---

**v3가 v2와 다른 점:**

v2: 사람이 AI 에이전트에게 지시.
v3: AI 오케스트레이터가 AI 에이전트들을 관리.

사람은 목표만 설정합니다.
"이번 달 신규 고객 50명"
나머지는 시스템이 알아서.

---

**레시피 #89 구성:**

1. **AI 오케스트레이터**
   - 비즈니스 목표 입력 → 하위 태스크 자동 분해
   - 각 태스크를 적합한 에이전트에게 자동 배정
   - 진행 상황 모니터링 + 이상 시 재배정
   - 주간 목표 달성률 리포트 자동 생성

2. **에이전트 간 협업**
   - 리서치 에이전트 → 콘텐츠 에이전트: "이 트렌드 기반으로 글 써줘"
   - CS 에이전트 → 데이터 에이전트: "이 불만 패턴 분석해줘"
   - 데이터 에이전트 → 오퍼레이션 에이전트: "이 이슈 태스크로 만들어줘"
   - 에이전트끼리 직접 소통. 사람 개입 최소화.

3. **자가 학습 루프**
   - 각 에이전트의 성과 자동 측정
   - 성과가 낮은 에이전트 → 프롬프트/설정 자동 조정
   - 새로운 업무 패턴 감지 → 새 에이전트 생성 제안
   - 주간 시스템 최적화 리포트

4. **인간 감독 레이어**
   - 중요 의사결정은 여전히 인간 승인 필요
   - "이 에이전트가 이런 결정을 내리려 합니다. 승인하시겠습니까?"
   - 에스컬레이션 기준 자동 설정 (금액, 리스크, 신규성)

---

v3 도입 후 변화:
- 대표(저)의 일일 의사결정 횟수: 50건 → 8건
- 에이전트 자율 처리율: 60% → 85%
- 팀원 루틴 업무 비중: 40% → 15%
- 전략적 업무 비중: 30% → 60%

AI가 AI를 관리합니다.
