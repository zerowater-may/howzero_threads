아카데미 2기 모집을 시작합니다. 이번에는 30명입니다.

1기는 20명으로 시작해서 17명이 수료했습니다. 솔직하게 공개했던 실패 포인트 — 난이도 편차, 1:1 멘토링 부족, 중도 이탈 — 이걸 전부 반영해서 2기를 설계했습니다.

---

📋 **2기에서 달라진 점**

| 항목 | 1기 | 2기 | 변경 이유 |
|------|-----|-----|----------|
| 정원 | 20명 | 30명 | 인증 강사 합류로 멘토링 여력 확보 |
| 기간 | 4주 | 5주 | Week 2.5 추가 (중간 난이도) |
| 멘토 | 1명 (저) | 3명 (저 + 인증 강사 2명) | 1:1 피드백 강화 |
| 사전 레벨 테스트 | 없음 | 있음 | 수준 차이로 인한 이탈 방지 |
| 소그룹 | 없음 | 6명씩 5팀 | 동료 학습 효과 |
| 수료 기준 | 과제 제출 | 과제 + 실전 적용 증빙 | 진짜 "적용"까지 확인 |

---

💡 **5주 커리큘럼**

```
Week 1: 기초 — AI 자동화의 원리와 첫 번째 자동화 구축
        (Make 기본, API 연결, 트리거 설정)

Week 2: 중급 — 비즈니스 프로세스 분석 + 자동화 설계
        (프로세스 맵핑, 우선순위 매트릭스, 3개 자동화 구축)

Week 2.5: 브릿지 — 실전 트러블슈팅 + 에러 핸들링
          (1기에서 가장 요청 많았던 내용)

Week 3: 고급 — AI 연동 자동화 + 데이터 파이프라인
        (GPT-4 API, 데이터 분석, 대시보드 구축)

Week 4: 심화 — 풀퍼널 자동화 + 최적화
        (전체 비즈니스 파이프라인 통합)

Week 5: 실전 — 자기 비즈니스에 적용 + 발표
        (각자의 비즈니스에 실제 적용 후 성과 발표)
```
