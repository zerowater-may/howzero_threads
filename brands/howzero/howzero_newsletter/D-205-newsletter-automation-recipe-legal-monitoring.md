**레시피 #73 구성:**

1. **법규 변경 감지**
   - 관보, 국회 입법예고, 규제 샌드박스 — 매일 자동 크롤링
   - 업종별 키워드 필터 (이커머스, SaaS, 개인정보, 노동법 등)

2. **영향도 자동 분석**
   - 새 법규 텍스트 → AI가 "우리 사업에 해당되는가" 판단
   - 영향도 High/Medium/Low 자동 분류
   - High인 경우 즉시 슬랙 알림 + 담당자 태그

3. **대응 가이드 자동 생성**
   - 해당 법규 요약 (3줄)
   - 필요한 조치 사항 리스트
   - 시행일까지 남은 기간 카운트다운
   - 유사 판례/사례 자동 검색

4. **컴플라이언스 대시보드**
   - 현재 준수 상태 실시간 모니터링
   - 기한 임박 항목 자동 리마인더
   - 분기별 컴플라이언스 리포트 자동 생성

---

- 법규 변경 인지 속도: 평균 45일 → 평균 1일

- 법무 담당자 모니터링 시간: 주 10시간 → 주 1시간
