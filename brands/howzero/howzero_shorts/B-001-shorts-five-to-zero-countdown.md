# B-001: 숏폼 스크립트 3개 — "5→0 카운트다운" 시리즈

> **유형**: 쇼츠/틱톡/릴스 | **각 30~45초 | **시리즈**: #하우제로 #5to0

---

## 숏폼 1: "5→0 카운트다운" (30초)

```
[0~3초] 텍스트: "5 → 0" (큰 숫자, 배경 전환)
나레이션: "직원 5명이 하던 일을 0명이 합니다."

[3~8초] 화면: CS 분류 화면
"1. CS 문의 분류 — 하루 2시간 → 자동. 0."

[8~13초] 화면: 이메일 응답 화면
"2. 고객 응답 — 하루 1시간 → AI 초안. 0."

[13~18초] 화면: 리뷰 대시보드
"3. 리뷰 관리 — 하루 30분 → 자동 수집. 0."

[18~23초] 화면: 보고서 생성
"4. 보고서 — 주 2시간 → 자동 생성. 0."

[23~27초] 화면: 데이터 시트
"5. 데이터 정리 — 하루 1시간 → 자동. 0."

[27~30초] 텍스트: "코딩 0줄. 하우제로."
"코딩 0줄로 전부 자동화했습니다. 방법은 프로필에."
```

**해시태그**: #하우제로 #AI자동화 #5to0 #비개발자 #코딩없이 #직원대체

---

## 숏폼 2: "아직도 수동으로 CS 해요?" (40초)

```
[0~3초] 카메라 직접 + 텍스트: "아직도 수동으로 CS 해요?"
나레이션: "아직도 CS 문의를 사람이 하나하나 읽고 분류하고 답변 쓰시나요?"

[3~8초] 카메라
"저도 그랬습니다. 하루 4시간. 직원 2명이 매달려서."

[8~15초] 화면 전환 → 자동화 데모 (화면 녹화)
"지금은요? 이메일 들어오면 — AI가 3초 만에 분류합니다."
(화면: 이메일 → 자동 분류 → 카테고리 태그)

[15~22초] 화면 녹화 계속
"답변 초안도 5초면 나옵니다."
(화면: AI가 응답 초안 생성)

[22~30초] 화면 녹화
"담당자한테 카톡 알림까지 자동."
(화면: 알림 발송)

[30~35초] 카메라 복귀
"이 전체 과정에 코딩 0줄입니다. 세팅 시간 15분."

[35~40초] 텍스트: "무료 AI 오딧 → 프로필"
"우리 회사에도 되는지 궁금하면 → 프로필 링크에서 무료 진단."
```

**해시태그**: #CS자동화 #AI자동화 #하우제로 #비개발자 #인건비절감 #업무자동화

## 숏폼 3: "코딩 실력 제로인 제가 10억 벌었습니다" (35초)

```
[0~3초] 카메라 직접 + 텍스트: "코딩 실력: 0"
나레이션: "코딩 실력 제로."

[3~5초] 텍스트 추가: "개발 경험: 0"
"개발 경험 제로."

[5~7초] 텍스트 추가: "IT 인맥: 0"
"IT 인맥 제로."

[7~10초] 텍스트 전환 → "연매출: 10억"
"연매출 10억."

[10~12초] 잠깐 멈춤 (임팩트)

[12~25초] 카메라
"전부 제로인 상태에서 GPT-3로 SaaS를 만들었고,
 1년 만에 10억을 벌었습니다.
 비결요? 코딩이 아니라 프로세스입니다.
 AI 도구를 먼저 찾지 마세요.
 반복 업무를 먼저 정리하세요.
 그게 제가 3번 실패하고 배운 겁니다."

[25~30초] 텍스트: "제로에서 10억까지. 코딩도 제로."
"제로에서 시작해서, 제로로 만드는 사람. 하우제로."

[30~35초] CTA
"전 과정 공개 영상은 유튜브에. 프로필 링크."
```

**해시태그**: #하우제로 #코딩없이10억 #제로에서시작 #AI자동화 #비개발자 #SaaS

## 3개 공통 게시 가이드

```
유튜브 쇼츠: 그대로 게시
틱톡: 트렌딩 사운드 추가 + 자막 크기 키우기
인스타 릴스: 비율 확인 (9:16) + 커버 이미지 설정

게시 순서: 틱톡 → 유튜브 쇼츠 → 인스타 릴스 (24시간 간격)
```
