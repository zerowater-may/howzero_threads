## 숏폼 1: "키즈 코딩 교육 자동화 Before vs After" (35초)

```
[0~3초] 텍스트: "키즈 코딩 교육 + AI = ?"
나레이션: "키즈 코딩 교육에서 커리큘럼 업데이트 느림. 아직도 사람이 합니다."

[3~10초] 화면: Before 상황 시각화
"커리큘럼 업데이트 느림에 매일 몇 시간씩 쓰고 있습니다."

[10~20초] 화면: AI 자동화 데모
"AI를 붙이면? AI 적응형 학습. 자동입니다."

[20~30초] 화면: 결과 수치
"결과: 커리큘럼 업데이트 주기 10배 단축."

[30~35초] 텍스트: "코딩 0줄. 하우제로."
"코딩 0줄입니다. 방법은 프로필에."
```

**해시태그**: #하우제로 #AI자동화 #키즈 코딩 교육 #비개발자 #코딩없이

## 숏폼 2: "키즈 코딩 교육에서 AI가 대체하는 업무 3가지" (40초)

```
[0~3초] 카메라 직접 + 텍스트: "키즈 코딩 교육에서 AI가 대체하는 업무"
나레이션: "키즈 코딩 교육 업종에서 AI가 당장 대체할 수 있는 업무 3가지."

[3~13초] 화면: 업무 1
"첫째. AI 적응형 학습. 사람이 할 필요 없습니다."

[13~23초] 화면: 업무 2
"둘째. 자동 난이도 조절. 이것도 AI가 합니다."

[23~33초] 화면: 업무 3
"셋째. 다국어 커리큘럼 자동 현지화. 전부 자동."

[33~40초] 카메라 복귀
"이 3가지만 자동화해도 커리큘럼 업데이트 주기 10배 단축. 코딩? 0줄. 하우제로."
```

**해시태그**: #하우제로 #키즈 코딩 교육자동화 #AI #비개발자 #업무자동화

## 숏폼 3: "이 업종도 AI 됩니다 — 키즈 코딩 교육" (30초)

```
[0~3초] 텍스트: "이 업종도 AI 됩니다"
나레이션: "키즈 코딩 교육? AI랑 관계없다고요?"

[3~8초] 카메라
"저도 처음엔 그렇게 생각했습니다."

[8~18초] 화면: 자동화 데모
"그런데 실제로 적용해 보니 — 커리큘럼 업데이트 주기 10배 단축, 학습 효과 40% 향상."

[18~25초] 카메라
"비결은 간단합니다. AI 도구를 먼저 찾지 마세요. 프로세스 맵부터 그리세요."

[25~30초] 텍스트: "무료 AI 진단 → 프로필"
"우리 회사에도 되는지 궁금하면? 프로필 링크에서 무료 진단."
```
