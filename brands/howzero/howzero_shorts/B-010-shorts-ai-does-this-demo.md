# B-010: 숏폼 3개 — "AI가 이것도 해요?" 데모 시리즈

> **유형**: 쇼츠/틱톡/릴스 | **앵글**: 순수 데모 충격. 설명 최소. 보여주기 최대.

---

## 숏폼 1: "이메일 100개를 30초 만에 분류합니다" (30초)

```
[0~3초] 텍스트: "이메일 100개. 30초."
(이메일 수신함 화면 — 읽지 않은 메일 100개)

[3~5초] "사람이 하면 2시간."

[5~8초] "AI가 하면?"
(버튼 클릭)

[8~25초] 화면 녹화 — 이메일이 빠르게 분류되는 화면
(환불 → 빨강 / 배송 → 파랑 / 제품문의 → 초록 / 기타 → 회색)
스르르 분류되는 화면 + 카운터 올라감
"10개... 30개... 50개... 80개... 100개. 완료."

[25~28초] 텍스트: "소요 시간: 27초"

[28~30초] "코딩 0줄. 방법 → 프로필."
```

**포인트**: 말 거의 없음. 화면이 말하게. 시각적 충격.

---

## 숏폼 2: "AI가 보고서를 만들어서 이메일로 보냈습니다" (25초)

```
[0~3초] 텍스트: "월요일 아침 9시."
(이메일 수신함 화면)

[3~8초] "누가 보냈지?"
(이메일 열기 — 보낸 사람: AI 자동화 시스템)

[8~18초] 화면: 보고서 내용 스크롤
"이번 주 매출: ₩4,200만 (+8%)"
"가장 많이 팔린 상품: [X]"
"CS 문의: 187건"
"핵심 인사이트: 배송 문의 30% 증가"

[18~22초] 카메라 (여유로운 표정)
"제가 쓴 글자: 0. AI가 일요일 밤에 만들어놨습니다."

[22~25초] "세팅 11분. → 프로필."
```

---

## 숏폼 3: "부정 리뷰 올라오면 10초 만에 알림 옵니다" (30초)

```
[0~3초] 텍스트: "별점 1점 리뷰가 올라왔습니다 ⚠️"

[3~8초] 화면: 쿠팡 리뷰 페이지 — 별점 1점 리뷰 등록됨

[8~15초] 화면: 슬랙 알림 (10초 후 도착)
"⚠️ 부정 리뷰 감지
 상품: [X]
 별점: ★☆☆☆☆
 내용: '배송이 너무 느려요'
 감성: 부정 (95%)
 추천 대응: '고객님, 불편을 드려 죄송합니다...'"

[15~22초] 화면: 추천 답변 클릭 → 리뷰 답변 작성

[22~27초] 카메라
"리뷰 올라오고 10초 만에 알림.
 답변 초안까지 AI가 만들어줌.
 10분 내 대응하면 별점 하락 막을 수 있습니다."

[27~30초] "자동 세팅법 → 프로필."
```
