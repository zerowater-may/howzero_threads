# B-500: 숏폼 스크립트 3개 — "AI 자동화 500번째 — 13년의 모든 것과 다음 500개" 시리즈

> **유형**: 쇼츠/틱톡/릴스 | **각 30~60초** | **시리즈**: #하우제로 #AI자동화

---

## 숏폼 1: "30초 데모: 500번째 특집 AI" (30초)

```
[0~3초] 텍스트: "AI 자동화 500번째 — 13년의 모든 것과 다음 500개" (큰 글씨, 배경 전환)
나레이션: "500번째입니다."

[3~10초] 화면: 수동 작업 장면
"500번째 특집 — 13년의 여정을 정리하지 않은 채 달려온 것."

[10~18초] 화면: 자동화 데모
"AI가 하면? 회고 데이터 자동 정리 → 핵심 교훈 추출 AI → 다음 500개 로드맵 생성 3초."

[18~25초] 화면: 결과 대시보드
"월 40시간 → 거의 0. 실측 데이터입니다."

[25~30초] 텍스트: "코딩 0줄. 하우제로."
"방법은 프로필에. 하우제로."
```

**해시태그**: #하우제로 #AI자동화 #500번째특집 #코딩없이 #업무자동화

## 숏폼 2: "비포/애프터: 500번째 특집" (45초)

```
[0~3초] 화면 분할: Before(빨강) / After(초록)
나레이션: "500번째 특집, Before & After."

[3~12초] Before 화면
"Before: 13년의 여정을 정리하지 않은 채 달려온 것.
 시간: 하루 2~3시간. 실수: 빈번. 야근: 기본."

[12~22초] After 화면 (자동화 데모)
"After: AI가 자동으로 회고 데이터 자동 정리 → 핵심 교훈 추출 AI → 다음 500개 로드맵 생성.
 시간: 15분. 실수: 거의 0. 야근: 없음."

[22~35초] 결과 수치
"500번째입니다. 13년간 AI 자동화만 파온 사람의 모든 것을 공개합니다.
 이게 코딩 0줄로 가능합니다."

[35~40초] 카메라 직접
"프로세스부터 정리하세요. 도구는 그 다음입니다."

[40~45초] CTA
"무료 진단 → 프로필 링크. 하우제로."
```

**해시태그**: #비포애프터 #AI자동화 #하우제로 #500번째특집 #자동화전후

## 숏폼 3: "1분 자동화: 500번째 특집 핵심" (40초)

```
[0~3초] 텍스트: "1분 자동화" (카운트다운 시작)
나레이션: "500번째 특집 자동화, 핵심만 1분에."

[3~12초] Step 1
"Step 1: 회고 데이터 자동 정리"
(화면: 트리거 설정)

[12~21초] Step 2
"Step 2: 핵심 교훈 추출 AI"
(화면: AI 처리 과정)

[21~30초] Step 3
"Step 3: 다음 500개 로드맵 생성"
(화면: 결과 확인)

[30~35초] 결과
"코딩 0줄. 세팅 15분. 월 비용 3만 원 이하."

[35~40초] CTA
"전체 과정은 롱폼에서. 링크는 프로필.
코딩도 제로. 하우제로."
```

**해시태그**: #1분자동화 #500번째특집 #AI자동화 #하우제로 #노코드
