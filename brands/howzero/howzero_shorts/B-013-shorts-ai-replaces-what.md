# B-013: 숏폼 3개 — "AI가 대체하는 것 vs 못하는 것" 시리즈

> **시리즈**: #AI가대체하는것 | **앵글**: 교육+역발상 동시

---

## 숏폼 1: "AI가 대체하는 것 ✅ vs 못하는 것 ❌" (35초)

```
[0~3초] 텍스트: "AI가 대체하는 것 / 못하는 것"

[3~18초] 빠른 텍스트 전환
✅ 이메일 분류 → AI
❌ 고객 공감 → 사람
✅ 보고서 작성 → AI
❌ 전략 수립 → 사람
✅ 데이터 입력 → AI
❌ 관계 구축 → 사람
✅ 리뷰 분석 → AI
❌ 창의적 기획 → 사람
✅ 일정 관리 → AI
❌ 협상 → 사람

[18~28초] 카메라
"AI는 '지루한 일'을 가져갑니다.
 사람은 '중요한 일'에 집중합니다.
 해고가 아니라 해방입니다."

[28~35초] "어떤 걸 AI에게 맡길지 → 프로필."
```

---

## 숏폼 2: "월급 250만 원 vs 월 8만 원" (25초)

```
[0~3초] 텍스트: "직원 1명 vs AI 시스템"

[3~10초] 화면 분할
좌: "직원 1명"
   월급 ₩250만
   주 40시간
   야근 싫어함
   휴가 씀
   퇴사 가능

우: "AI 시스템"
   월 ₩8만
   주 168시간 (24/7)
   야근 없음 (항상 일함)
   휴가 없음
   퇴사 안 함

[10~20초] 카메라
"물론 AI가 사람을 완전히 대체하진 못합니다.
 근데 '복붙, 분류, 정리' 같은 반복 업무는?
 AI가 31배 저렴하게 합니다."

[20~25초] "방법 → 프로필."
```

---

## 숏폼 3: "사장님의 시급, 계산해봤습니다" (30초)

```
[0~3초] 텍스트: "사장님의 진짜 시급"

[3~10초] 카메라 + 계산
"사장님 연봉이 1억이라고요? 시급 ₩48,000.
 그 시급으로 CS 복붙을 하고 계십니다.
 시급 ₩48,000짜리 인력이 시급 ₩10,000짜리 일을."

[10~22초] 카메라
"AI한테 ₩10,000짜리 일을 맡기세요.
 사장님은 ₩48,000짜리 일을 하세요.
 영업. 전략. 관계. 성장.
 그게 사장님이 진짜 해야 할 일입니다."

[22~27초] 텍스트: "AI = ₩10,000짜리 일 담당"

[27~30초] "시작 → 프로필. 하우제로."
```
