## 숏폼 1: "30초 데모: 스마트 시티/도시 인프라 AI" (30초)

```
[0~3초] 텍스트: "스마트 시티/도시 인프라 AI 자동화" (큰 글씨, 배경 전환)
나레이션: "스마트 시티/도시 인프라, 자동화 가능합니다."

[3~10초] 화면: 수동 작업 장면
"도시 인프라 자동 관리 수동으로 하고 있죠?"

[10~18초] 화면: 자동화 데모
"AI가 하면? 도시 인프라 자동 관리 → 교통 데이터 실시간 자동 분석 → 에너지 소비 자동 최적화 3초."

[18~25초] 화면: 결과 대시보드
"월 40시간 → 거의 0. 실측 데이터입니다."

[25~30초] 텍스트: "코딩 0줄. 하우제로."
"방법은 프로필에. 하우제로."
```

**해시태그**: #하우제로 #AI자동화 #스마트시티도시인프라 #코딩없이 #업무자동화

## 숏폼 2: "비포/애프터: 스마트 시티/도시 인프라" (45초)

```
[0~3초] 화면 분할: Before(빨강) / After(초록)
나레이션: "스마트 시티/도시 인프라, Before & After."

[3~12초] Before 화면
"Before: 도시 인프라 자동 관리 수동 처리.
 시간: 하루 2~3시간. 실수: 빈번. 야근: 기본."

[12~22초] After 화면 (자동화 데모)
"After: AI가 자동으로 도시 인프라 자동 관리 → 교통 데이터 실시간 자동 분석 → 에너지 소비 자동 최적화.
 시간: 15분. 실수: 거의 0. 야근: 없음."

[22~35초] 결과 수치
"스마트 시티/도시 인프라 자동화, 과장 아닙니다.
 이게 코딩 0줄로 가능합니다."

[35~40초] 카메라 직접
"프로세스부터 정리하세요. 도구는 그 다음입니다."

[40~45초] CTA
"무료 진단 → 프로필 링크. 하우제로."
```

**해시태그**: #비포애프터 #AI자동화 #하우제로 #스마트시티도시인프라 #자동화전후

## 숏폼 3: "1분 자동화: 스마트 시티/도시 인프라 핵심" (40초)

```
[0~3초] 텍스트: "1분 안에 핵심만"
나레이션: "스마트 시티/도시 인프라 자동화, 핵심만 1분."

[3~13초] 핵심 1
"첫째: 도시 인프라 자동 관리.
 AI가 데이터를 자동으로 수집, 분류합니다."

[13~23초] 핵심 2
"둘째: 교통 데이터 실시간 자동 분석.
 수동 반나절 → AI 3분."

[23~33초] 핵심 3
"셋째: 에너지 소비 자동 최적화.
 실시간 자동 처리. 실수율 거의 0."

[33~38초] 결과
"코딩 0줄. 월 40시간 → 거의 0."

[38~40초] CTA
"howzero.com/ai-score 하우제로."
```
