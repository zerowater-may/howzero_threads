# B-516: 숏폼 스크립트 3개 — "AI로 사업 파트너 적합성 자동 평가" 시리즈

> **유형**: 쇼츠/틱톡/릴스 | **각 30~60초** | **시리즈**: #하우제로 #AI자동화

---

## 숏폼 1: "30초 데모: 파트너 평가 AI" (30초)

```
[0~3초] 텍스트: "AI로 사업 파트너 적합성 자동 평가" (큰 글씨, 배경 전환)
나레이션: "사업 파트너를 감으로 고르면 실패 확률 70%."

[3~10초] 화면: 수동 작업 장면
"파트너 평가 — 파트너 적합성을 감으로 판단."

[10~18초] 화면: 자동화 데모
"AI가 하면? 파트너 데이터 자동 수집 → 적합성 AI 분석 → 리스크 자동 평가 3초."

[18~25초] 화면: 결과 대시보드
"월 40시간 → 거의 0. 실측 데이터입니다."

[25~30초] 텍스트: "코딩 0줄. 하우제로."
"방법은 프로필에. 하우제로."
```

**해시태그**: #하우제로 #AI자동화 #파트너평가 #코딩없이 #업무자동화

---

## 숏폼 2: "비포/애프터: 파트너 평가" (45초)

```
[0~3초] 화면 분할: Before(빨강) / After(초록)
나레이션: "파트너 평가, Before & After."

[3~12초] Before 화면
"Before: 파트너 적합성을 감으로 판단.
 시간: 하루 2~3시간. 실수: 빈번. 야근: 기본."

[12~22초] After 화면 (자동화 데모)
"After: AI가 자동으로 파트너 데이터 자동 수집 → 적합성 AI 분석 → 리스크 자동 평가.
 시간: 15분. 실수: 거의 0. 야근: 없음."

[22~35초] 결과 수치
"사업 파트너를 감으로 고르면 실패 확률 70%. 데이터로 고르면 30%입니다.
 이게 코딩 0줄로 가능합니다."

[35~40초] 카메라 직접
"프로세스부터 정리하세요. 도구는 그 다음입니다."

[40~45초] CTA
"무료 진단 → 프로필 링크. 하우제로."
```

**해시태그**: #비포애프터 #AI자동화 #하우제로 #파트너평가 #자동화전후

---

## 숏폼 3: "1분 자동화: 파트너 평가 핵심" (40초)

```
[0~3초] 텍스트: "1분 자동화" (카운트다운 시작)
나레이션: "파트너 평가 자동화, 핵심만 1분에."

[3~12초] Step 1
"Step 1: 파트너 데이터 자동 수집"
(화면: 트리거 설정)

[12~21초] Step 2
"Step 2: 적합성 AI 분석"
(화면: AI 처리 과정)

[21~30초] Step 3
"Step 3: 리스크 자동 평가"
(화면: 결과 확인)

[30~35초] 결과
"코딩 0줄. 세팅 15분. 월 비용 3만 원 이하."

[35~40초] CTA
"전체 과정은 롱폼에서. 링크는 프로필.
코딩도 제로. 하우제로."
```

**해시태그**: #1분자동화 #파트너평가 #AI자동화 #하우제로 #노코드
