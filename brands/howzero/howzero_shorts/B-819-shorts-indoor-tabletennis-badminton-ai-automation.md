```

"실내 탁구/배드민턴장 — 코트 예약 최적화 및 자동 배정 수동으로 하고 있죠?"

[10~18초] 화면: 자동화 데모
"AI가 하면? 코트 예약 최적화 및 자동 배정 → 피크 타임 분석 기반 요금 동적 조정 → 회원 이용 패턴 분석 및 맞춤 프로모션 3초."

## 숏폼 2: "비포/애프터: 실내 탁구/배드민턴장" (45초)

```
[0~3초] 화면 분할: Before(빨강) / After(초록)
나레이션: "실내 탁구/배드민턴장, Before & After."

[3~12초] Before 화면
"Before: 코트 예약 최적화 및 자동 배정 수동 처리.
 시간: 하루 2~3시간. 실수: 빈번. 야근: 기본."

[12~22초] After 화면 (자동화 데모)
"After: AI가 자동으로 코트 예약 최적화 및 자동 배정 → 피크 타임 분석 기반 요금 동적 조정 → 회원 이용 패턴 분석 및 맞춤 프로모션.
 시간: 15분. 실수: 거의 0. 야근: 없음."

[22~35초] 결과 수치
"실내 탁구/배드민턴장 자동화, 과장 아닙니다.
 이게 코딩 0줄로 가능합니다."

[35~40초] 카메라 직접
"프로세스부터 정리하세요. 도구는 그 다음입니다."

[40~45초] CTA
"무료 진단 → 프로필 링크. 하우제로."
```

## 숏폼 3: "1분 자동화: 실내 탁구/배드민턴장 핵심" (40초)

```
[0~3초] 텍스트: "1분 안에 설명하는 실내 탁구/배드민턴장 AI 자동화"
나레이션: "1분 안에 보여드립니다."

[3~13초] Step 1
"첫째, 코트 예약 최적화 및 자동 배정.
 AI가 자동으로 수집하고 분류합니다."

[13~23초] Step 2
"둘째, 피크 타임 분석 기반 요금 동적 조정.
 수동으로 반나절 걸리던 게 3분."

[23~33초] Step 3
"셋째, 회원 이용 패턴 분석 및 맞춤 프로모션.
 사람이 할 필요 없습니다."

[33~38초] 결과
"결과: 월 40시간 → 거의 0."

[38~40초] CTA
"코딩 0줄. 하우제로. 프로필 링크."
