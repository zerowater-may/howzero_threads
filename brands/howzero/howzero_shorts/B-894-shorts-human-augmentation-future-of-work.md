## 숏폼 1: "30초 데모: 인간 증강과 미래 노동 AI" (30초)

```
[0~3초] 텍스트: "인간 증강과 미래 노동 AI 자동화" (큰 글씨, 배경 전환)
나레이션: "인간 증강과 미래 노동, 자동화 가능합니다."

[3~10초] 화면: 수동 작업 장면
"인간 증강 기술 트렌드 자동 리서치 수동으로 하고 있죠?"

[10~18초] 화면: 자동화 데모
"AI가 하면? 인간 증강 기술 트렌드 자동 리서치 → AI+인간 협업 생산성 자동 측정 → 미래 노동 시나리오 자동 시뮬레이션 3초."

[18~25초] 화면: 결과 대시보드
"월 40시간 → 거의 0. 실측 데이터입니다."

[25~30초] 텍스트: "코딩 0줄. 하우제로."
"방법은 프로필에. 하우제로."
```

**해시태그**: #하우제로 #AI자동화 #인간증강과미래노동 #코딩없이 #업무자동화

## 숏폼 2: "비포/애프터: 인간 증강과 미래 노동" (45초)

```
[0~3초] 화면 분할: Before(빨강) / After(초록)
나레이션: "인간 증강과 미래 노동, Before & After."

[3~12초] Before 화면
"Before: 인간 증강 기술 트렌드 자동 리서치 수동 처리.
 시간: 하루 2~3시간. 실수: 빈번. 야근: 기본."

[12~22초] After 화면 (자동화 데모)
"After: AI가 자동으로 인간 증강 기술 트렌드 자동 리서치 → AI+인간 협업 생산성 자동 측정 → 미래 노동 시나리오 자동 시뮬레이션.
 시간: 15분. 실수: 거의 0. 야근: 없음."

[22~35초] 결과 수치
"인간 증강과 미래 노동 자동화, 과장 아닙니다.
 이게 코딩 0줄로 가능합니다."

[35~40초] 카메라 직접
"프로세스부터 정리하세요. 도구는 그 다음입니다."

[40~45초] CTA
"무료 진단 → 프로필 링크. 하우제로."
```

## 숏폼 3: "1분 자동화: 인간 증강과 미래 노동 핵심" (40초)

```
[0~3초] 텍스트: "1분 안에 설명하는 인간 증강과 미래 노동 AI 자동화"
나레이션: "1분 안에 보여드립니다."

[3~13초] Step 1
"첫째, 인간 증강 기술 트렌드 자동 리서치.
 AI가 자동으로 수집하고 분류합니다."

[13~23초] Step 2
"둘째, AI+인간 협업 생산성 자동 측정.
 수동으로 반나절 걸리던 게 3분."

[23~33초] Step 3
"셋째, 미래 노동 시나리오 자동 시뮬레이션.
 사람이 할 필요 없습니다."

[33~38초] 결과
"결과: 월 40시간 → 거의 0."

[38~40초] CTA
"howzero.com/ai-score. 하우제로."
```
