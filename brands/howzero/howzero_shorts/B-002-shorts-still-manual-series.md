# B-002: 숏폼 3개 — "이거 아직도 수동으로 해요?" 시리즈

> **유형**: 쇼츠/틱톡/릴스 | **각 30~40초 | **시리즈**: #아직도수동

---

## 숏폼 1: "아직도 엑셀로 보고서 만들어요?" (35초)

```
[0~3초] 카메라 + 텍스트: "아직도 엑셀로 보고서요?"
"아직도 매주 엑셀 열어서 보고서 만드세요?"

[3~8초] 카메라
"데이터 복붙하고, 그래프 그리고, 메일 쓰고. 매주 2시간."

[8~20초] 화면 전환 → 자동화 데모
"지금 보여드릴게요."
(화면: 데이터 자동 수집 → AI 요약 → PDF 생성 → 이메일 발송)
"데이터 수집부터 발송까지. 전자동. 2시간 → 0."

[20~30초] 카메라
"세팅 시간요? 20분. 코딩? 0줄."

[30~35초] CTA
"방법 → 프로필 링크"
```

---

## 숏폼 2: "아직도 리뷰를 눈으로 확인해요?" (30초)

```
[0~3초] 텍스트: "리뷰 100개를 눈으로?" + 카메라
"고객 리뷰를 매일 하나하나 눈으로 읽으세요?"

[3~12초] 화면 녹화
"AI가 자동으로 수집하고"
(화면: 리뷰 크롤링)
"긍정/부정 분류하고"
(화면: 감성 분석 결과)
"부정 리뷰만 알림 보내줍니다"
(화면: 슬랙 알림)

[12~25초] 카메라
"부정 리뷰 놓쳐서 별점 떨어지는 거, 이제 끝."

[25~30초] CTA + 텍스트
"세팅 15분. 코딩 0줄. → 프로필"
```

---

## 숏폼 3: "아직도 고객 문의를 복붙으로 답해요?" (40초)

```
[0~3초] 텍스트: "복붙 CS의 시대는 끝났습니다"
"환불 문의 오면 복붙. 배송 문의 오면 복붙. 하루에 50번 복붙."

[3~8초] 카메라 (약간 지친 표정)
"이거 사람이 할 일이 아닙니다."

[8~25초] 화면 녹화
"문의가 들어옵니다."
(화면: 이메일 수신)
"AI가 읽고 분류합니다. 3초."
(화면: 자동 분류)
"맞춤 답변 초안을 만듭니다. 5초."
(화면: AI 답변 생성)
"담당자는 '보내기'만 누르면 됩니다."
(화면: 원클릭 발송)

[25~35초] 카메라
"복붙 50번 → 클릭 50번. 시간? 4시간 → 30분."

[35~40초] CTA
"우리 회사에도 되는지? → 프로필 링크에서 무료 진단"
```
