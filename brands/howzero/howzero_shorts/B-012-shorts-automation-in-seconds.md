# B-012: 숏폼 3개 — "N초 만에 끝" 타임어택 시리즈

> **시리즈**: #N초자동화 | **앵글**: 시간 카운터 + 데모 = 충격

---

## 숏폼 1: "경쟁사 가격 30초 만에 전부 확인" (30초)

```
[0~3초] 텍스트 + 타이머: "경쟁사 가격 10개. 30초."

[3~25초] 화면 녹화 (실시간 타이머)
경쟁 상품 10개 URL이 입력되어 있음
→ 버튼 클릭
→ 가격이 하나씩 빠르게 수집됨
→ 비교표 자동 생성
→ "우리보다 싼 상품: 3개 ⚠️"
→ 타이머: 23초.

[25~30초] 카메라
"수동으로 하면 2시간. AI로 23초. → 프로필."
```

---

## 숏폼 2: "고객 리뷰 50개 감성 분석 15초" (25초)

```
[0~3초] 텍스트: "리뷰 50개. 15초."

[3~18초] 화면 녹화
리뷰 50개 텍스트 → AI 분석 시작
→ 긍정 32개 ✅ / 부정 11개 ❌ / 중립 7개 ➖
→ "가장 많이 언급된 불만: 배송 속도"
→ 타이머: 12초.

[18~22초] 카메라
"1시간 → 12초. 부정 리뷰만 즉시 대응."

[22~25초] "세팅법 → 프로필."
```

---

## 숏폼 3: "5분 만에 상품 설명 AI가 써줌" (30초)

```
[0~3초] 텍스트: "상품 설명 AI가 써줌"

[3~8초] 카메라
"타오바오 상품 하나. 중국어. 뭐라 써있는지 모르겠죠?"

[8~22초] 화면 녹화
→ 상품 URL 입력
→ AI가 중국어 읽음
→ 한국어로 번역 + 쿠팡 스타일 상품명 생성
→ 상품 설명 + 키워드 + 가격 자동 계산(환율+관세+마진)

[22~27초] 카메라
"건당 30분 → 5분. 번역 외주비 월 50만 원 → 0원."

[27~30초] "셀러분들 → 프로필 링크."
```
