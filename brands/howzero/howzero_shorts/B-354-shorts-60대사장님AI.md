# B-354: 숏폼 스크립트 3개 — "60대 사장님도 할 수 있다" 시리즈

> **유형**: 쇼츠/틱톡/릴스 | **각 30~60초** | **시리즈**: #하우제로 #AI자동화

---

## 숏폼 1: "30초 데모: 60대 사장님도 할 수 있다" (30초)

```
[0~3초] 텍스트: "60대 사장님도 할 수 있다" (큰 글씨, 배경 전환)
나레이션: "60대 사장님도 할 수 있다, 아직 수동으로 하세요?"

[3~10초] 화면: 수동 작업 장면
"나이는 장벽이 아니다 — 사람이 하면 하루 3시간."

[10~18초] 화면: 자동화 데모
"AI가 하면? 실제 60대 도입 사례까지 3초."

[18~25초] 화면: 결과 대시보드
"기술이 아니라 의지의 문제다."

[25~30초] 텍스트: "코딩 0줄. 하우제로."
"방법은 프로필에. 하우제로."
```

**해시태그**: #하우제로 #AI자동화 #60대사장님도할수있다 #코딩없이 #업무자동화

---

## 숏폼 2: "비포/애프터: 60대 사장님도 할 수 있다" (45초)

```
[0~3초] 화면 분할: Before(빨강) / After(초록)
나레이션: "60대 사장님도 할 수 있다, Before & After."

[3~12초] Before 화면
"Before: 나이는 장벽이 아니다부터 지원 체계까지 전부 수동.
 시간: 하루 3~4시간. 실수: 빈번. 야근: 기본."

[12~22초] After 화면 (자동화 데모)
"After: AI가 자동으로 나이는 장벽이 아니다 → 실제 60대 도입 사례 → 단계별 가이드 → 지원 체계.
 시간: 15분. 실수: 거의 0. 야근: 없음."

[22~35초] 결과 수치
"기술이 아니라 의지의 문제다.
 이게 코딩 0줄로 가능합니다."

[35~40초] 카메라 직접
"프로세스부터 정리하세요. 도구는 그 다음입니다."

[40~45초] CTA
"무료 진단 → 프로필 링크. 하우제로."
```

**해시태그**: #비포애프터 #AI자동화 #하우제로 #60대사장님도할수있다 #자동화전후

---

## 숏폼 3: "1분 자동화: 60대 사장님도 할 수 있다 핵심" (40초)

```
[0~3초] 텍스트: "1분 자동화" (카운트다운 시작)
나레이션: "60대 사장님도 할 수 있다 자동화, 핵심만 1분에."

[3~12초] Step 1
"Step 1: 나이는 장벽이 아니다"
(화면: 트리거 설정)

[12~21초] Step 2
"Step 2: 실제 60대 도입 사례"
(화면: AI 처리 과정)

[21~30초] Step 3
"Step 3: 단계별 가이드"
(화면: 결과 확인)

[30~35초] 결과
"기술이 아니라 의지의 문제다."

[35~40초] CTA
"코딩 0줄. 세팅 15분. 하우제로."
```

**해시태그**: #1분자동화 #하우제로 #AI자동화 #60대사장님도할수있다 #비개발자

---

## 3개 공통 게시 가이드

```
유튜브 쇼츠: 그대로 게시
틱톡: 트렌딩 사운드 추가 + 자막 크기 키우기
인스타 릴스: 비율 확인 (9:16) + 커버 이미지 설정

게시 순서: 틱톡 → 유튜브 쇼츠 → 인스타 릴스 (24시간 간격)
```
