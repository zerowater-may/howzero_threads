> **유형**: 쇼츠/틱톡/릴스 | **시리즈**: #AI오해파괴

---

## 숏폼 1: "AI 자동화 = 직원 해고?" (30초)

```
[0~3초] 텍스트: "AI 자동화 = 직원 짜르기?" + 큰 X 표시
"AI 자동화가 직원 짜르는 거라고요?"

[3~10초] 카메라 (진지하게)
"아닙니다. 제 고객사 중 자동화 후에 직원을 자른 곳은 0곳입니다."

[10~22초] 카메라
"대신 뭐가 바뀌었냐면 —
 CS 복붙하던 직원이 → 고객 관계 관리를 합니다.
 데이터 입력하던 직원이 → 데이터 분석을 합니다.
 보고서 쓰던 직원이 → 전략 기획을 합니다.
 단순 노동에서 해방시켜주는 겁니다."

[22~30초] 텍스트: "AI = 해고가 아니라 해방"
"AI는 해고가 아니라 해방입니다. 하우제로."
```

## 숏폼 2: "코딩 모르면 AI 못 써요?" (25초)

```
[0~3초] 텍스트: "코딩 모르면 AI 못 쓴다?" + 큰 X
"코딩 모르면 AI 자동화 못 한다고요?"

[3~8초] 카메라 (웃으면서)
"제가 코딩 0줄로 10억 벌었는데요."

[8~18초] 빠른 화면 전환 — 자동화 데모 몽타주
"CS 자동 분류 — 코딩 0줄."
"보고서 자동 생성 — 코딩 0줄."
"리뷰 자동 수집 — 코딩 0줄."
"CRM 자동화 — 코딩 0줄."

[18~25초] 카메라
"2026년에 코딩 몰라서 못 한다는 건 핑계입니다. 방법은 프로필에."
```

## 숏폼 3: "AI가 다 해주는 거 아니에요?" (35초)

```
[0~3초] 텍스트: "AI가 알아서 다 해준다?" + 큰 X
"AI 붙이면 알아서 다 해준다고요?"

[3~10초] 카메라 (진지)
"안 됩니다. AI는 마법이 아닙니다.

AI가 잘하는 것: 분류, 요약, 반복 작업, 패턴 인식.
AI가 못하는 것: 판단, 공감, 창의적 결정, 관계 구축."

[10~25초] 카메라
"그래서 제가 항상 하는 말이 —
 '프로세스를 먼저 정리하세요.'
 AI한테 뭘 시킬지 모르면서 AI 도구부터 사는 건
 운전 못 하면서 페라리 사는 거랑 같습니다."

[25~30초] 카메라
"AI는 도구입니다. 도구는 쓰는 사람이 중요합니다."

[30~35초] CTA
"제대로 쓰는 법 → 프로필 링크"
```
