# B-802: 숏폼 스크립트 3개 — "AI로 고객 옴니채널 여정 자동 통합" 시리즈

> **유형**: 쇼츠/틱톡/릴스 | **각 30~60초** | **시리즈**: #하우제로 #AI자동화

---

## 숏폼 1: "30초 데모: 옴니채널 여정 통합 AI" (30초)

```
[0~3초] 텍스트: "옴니채널 여정 통합 AI 자동화" (큰 글씨, 배경 전환)
나레이션: "옴니채널 여정 통합, 자동화 가능합니다."

[3~10초] 화면: 수동 작업 장면
"옴니채널 여정 통합 — 오프라인-온라인 고객 접점 데이터 통합 수동으로 하고 있죠?"

[10~18초] 화면: 자동화 데모
"AI가 하면? 오프라인-온라인 고객 접점 데이터 통합 → 채널별 고객 여정 맵 자동 생성 → 이탈 구간 자동 감지 및 리텐션 시나리오 3초."

[18~25초] 화면: 결과 대시보드
"월 40시간 → 거의 0. 실측 데이터입니다."

[25~30초] 텍스트: "코딩 0줄. 하우제로."
"방법은 프로필에. 하우제로."
```

**해시태그**: #하우제로 #AI자동화 #옴니채널여정통합 #코딩없이 #업무자동화

## 숏폼 2: "비포/애프터: 옴니채널 여정 통합" (45초)

```
[0~3초] 화면 분할: Before(빨강) / After(초록)
나레이션: "옴니채널 여정 통합, Before & After."

[3~12초] Before 화면
"Before: 오프라인-온라인 고객 접점 데이터 통합 수동 처리.
 시간: 하루 2~3시간. 실수: 빈번. 야근: 기본."

[12~22초] After 화면 (자동화 데모)
"After: AI가 자동으로 오프라인-온라인 고객 접점 데이터 통합 → 채널별 고객 여정 맵 자동 생성 → 이탈 구간 자동 감지 및 리텐션 시나리오.
 시간: 15분. 실수: 거의 0. 야근: 없음."

[22~35초] 결과 수치
"옴니채널 여정 통합 자동화, 과장 아닙니다.
 이게 코딩 0줄로 가능합니다."

[35~40초] 카메라 직접
"프로세스부터 정리하세요. 도구는 그 다음입니다."

[40~45초] CTA
"무료 진단 → 프로필 링크. 하우제로."
```

**해시태그**: #비포애프터 #AI자동화 #하우제로 #옴니채널여정통합 #자동화전후

## 숏폼 3: "1분 자동화: 옴니채널 여정 통합 핵심" (40초)

```
[0~3초] 텍스트: "1분 안에 설명하는 옴니채널 여정 통합 AI 자동화"
나레이션: "1분 안에 보여드립니다."

[3~13초] Step 1
"첫째, 오프라인-온라인 고객 접점 데이터 통합.
 AI가 자동으로 수집하고 분류합니다."

[13~23초] Step 2
"둘째, 채널별 고객 여정 맵 자동 생성.
 수동으로 반나절 걸리던 게 3분."

[23~33초] Step 3
"셋째, 이탈 구간 자동 감지 및 리텐션 시나리오.
 사람이 할 필요 없습니다."

[33~38초] 결과
"결과: 월 40시간 → 거의 0."

[38~40초] CTA
"코딩 0줄. 하우제로. 프로필 링크."
```

**해시태그**: #1분설명 #AI자동화 #하우제로 #옴니채널여정통합 #자동화핵심
