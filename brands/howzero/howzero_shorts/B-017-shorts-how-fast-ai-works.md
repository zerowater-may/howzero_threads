# B-017: 숏폼 3개 — "이 속도 실화?" AI 속도 충격 시리즈

> **시리즈**: #AI속도실화 | **앵글**: 실시간 타이머로 속도 충격

---

## 숏폼 1: "계약서 초안 45초" (25초)

```
[0~3초] 텍스트 + 타이머: "계약서 초안. 타이머 시작."

[3~18초] 화면 녹화 (실시간)
"조건 입력: 보증금 5,000만, 월세 100만, 2년, 특약 2개."
→ AI 생성 중...
→ 2페이지 계약서 초안 완성.
타이머: 45초.

[18~22초] 카메라
"변호사님이 2시간 걸리던 초안. AI가 45초."

[22~25초] "검토만 하세요. → 프로필."
```

---

## 숏폼 2: "상품 설명 번역+작성 38초" (25초)

```
[0~3초] 텍스트: "중국어 상품 → 한국어 상품 설명. 타이머."

[3~18초] 화면 녹화
타오바오 상품 URL 입력
→ AI가 중국어 읽음
→ 한국어 상품명 생성
→ 상품 설명 작성
→ 키워드 추출
→ 가격 자동 계산 (환율+관세+마진)
타이머: 38초.

[18~22초] 카메라
"셀러분들. 건당 30분 → 38초."

[22~25초] "프로필."
```

---

## 숏폼 3: "뉴스레터 초안 32초" (25초)

```
[0~3초] 텍스트: "뉴스레터 1,500자 초안. 타이머."

[3~18초] 화면 녹화
"주제: 'AI 도입 전 프로세스 정리의 중요성'"
→ AI 생성 중...
→ 1,500자 뉴스레터 초안 완성
→ 제목 3개 + 본문 + CTA 포함
타이머: 32초.

[18~22초] 카메라
"매주 2시간 쓰던 뉴스레터. AI 초안 32초 + 검토 5분 = 끝."

[22~25초] "방법 → 프로필."
```
