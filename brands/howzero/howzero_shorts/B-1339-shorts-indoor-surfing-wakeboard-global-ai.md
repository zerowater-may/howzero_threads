## 숏폼 1: "실내 서핑·웨이크보드 체인 자동화 Before vs After" (35초)

```
[0~3초] 텍스트: "실내 서핑·웨이크보드 체인 + AI = ?"
나레이션: "실내 서핑·웨이크보드 체인에서 파도 머신 세팅·수질 관리가 수작업. 아직도 사람이 합니다."

[3~10초] 화면: Before 상황 시각화
"이용자 레벨별 프로그램 배정이 수동."

[10~20초] 화면: AI 자동화 데모
"AI를 붙이면? 파도 머신 AI 자동 세팅 + 수질 모니터링. 자동입니다."

[20~30초] 화면: 결과 수치
"결과: 장비 고장 예측 정확도 94%, 이용자 만족도 45% 상승."

[30~35초] 텍스트: "코딩 0줄. 하우제로."
"코딩 0줄입니다. 방법은 프로필에."
```

## 숏폼 2: "실내 서핑·웨이크보드 체인에서 AI가 대체하는 업무 3가지" (40초)

```
[0~3초] 카메라 직접 + 텍스트: "실내 서핑·웨이크보드 체인에서 AI가 대체하는 업무"
나레이션: "실내 서핑·웨이크보드 체인 업종에서 AI가 당장 대체할 수 있는 업무 3가지."

[3~13초] 화면: 업무 1
"첫째. 파도 머신 AI 자동 세팅 + 수질 모니터링. 사람이 할 필요 없습니다."

[13~23초] 화면: 업무 2
"둘째. 이용자 레벨 AI 자동 분석 + 프로그램 매칭. 이것도 AI가 합니다."

[23~33초] 화면: 업무 3
"셋째. 장비 유지보수 AI 예측 시스템. 전부 자동."

[33~40초] 카메라 복귀
"이 3가지만 자동화해도 장비 고장 예측 정확도 94%, 이용자 만족도 45% 상승. 코딩? 0줄. 하우제로."
```

**해시태그**: #하우제로 #실내서핑웨이크보드체인자동화 #AI #비개발자 #업무자동화
