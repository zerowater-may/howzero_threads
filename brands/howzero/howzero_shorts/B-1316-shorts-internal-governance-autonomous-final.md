## 숏폼 1: "사내 자동화 거버넌스 자동화 Before vs After" (35초)

```
[0~3초] 텍스트: "사내 자동화 거버넌스 + AI = ?"
나레이션: "사내 자동화 거버넌스에서 정책 준수를 수동 점검. 아직도 사람이 합니다."

[3~10초] 화면: Before 상황 시각화
"거버넌스 리포트를 분기마다 수동 작성."

[10~20초] 화면: AI 자동화 데모
"AI를 붙이면? 자동화 정책 준수 자동 모니터링. 자동입니다."

[20~30초] 화면: 결과 수치
"결과: 정책 준수율 70% → 98%."

[30~35초] 텍스트: "코딩 0줄. 하우제로."
"코딩 0줄입니다. 방법은 프로필에."

## 숏폼 2: "사내 자동화 거버넌스에서 AI가 대체하는 업무 3가지" (40초)

```
[0~3초] 카메라 직접 + 텍스트: "사내 자동화 거버넌스에서 AI가 대체하는 업무"
나레이션: "사내 자동화 거버넌스 업종에서 AI가 당장 대체할 수 있는 업무 3가지."

[3~13초] 화면: 업무 1
"첫째. 자동화 정책 준수 자동 모니터링. 사람이 할 필요 없습니다."

[13~23초] 화면: 업무 2
"둘째. 거버넌스 리포트 자동 생성. 이것도 AI가 합니다."

[23~33초] 화면: 업무 3
"셋째. 위반 사항 자동 알림 및 시정. 전부 자동."

[33~40초] 카메라 복귀
"이 3가지만 자동화해도 정책 준수율 70% → 98%. 코딩? 0줄. 하우제로."
```

## 숏폼 3: "이 업종도 AI 됩니다 — 사내 자동화 거버넌스" (30초)

```
