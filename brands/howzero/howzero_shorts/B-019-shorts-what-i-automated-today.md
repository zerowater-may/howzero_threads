# B-019: 숏폼 3개 — "오늘 자동화한 것" 일기 시리즈

> **시리즈**: #오늘자동화 | **앵글**: 리얼 일상 + 구체적 자동화 + 공감

---

## 숏폼 1: "오늘 자동화한 것: 견적서 자동 발송" (30초)

```
[0~3초] 텍스트: "오늘 자동화한 것 🔧"
(카메라: 편안한 톤)

[3~10초] "고객이 오딧 끝나면 견적서를 보내야 해요.
 매번 수동으로 만들고, PDF 변환하고, 이메일 쓰고, 첨부하고.
 건당 20분."

[10~22초] 화면 녹화
"이제는요. 오딧 완료 체크하면 →
 견적서 자동 생성 →
 PDF 변환 →
 이메일에 첨부해서 자동 발송.
 건당 20분 → 0분."

[22~27초] 카메라
"오늘 30분짜리 자동화 하나 더 만들었습니다."

[27~30초] "세팅 시간: 8분. → 프로필."
```

---

## 숏폼 2: "오늘 자동화한 것: 미팅 후 자동 팔로업" (25초)

```
[0~3초] 텍스트: "오늘 자동화한 것 🔧"

[3~10초] 카메라
"미팅 끝나면 팔로업 이메일 보내야 하잖아요.
 '감사합니다, 논의한 내용 정리해서 보내드립니다.'
 매번 까먹거나 미루게 돼요."

[10~20초] 화면 녹화
"이제는. 캘린더에서 미팅 종료 감지 →
 AI가 미팅 내용 요약 →
 팔로업 이메일 초안 자동 생성 →
 제가 '보내기'만."

[20~25초] "까먹는 것 → 0. 프로필."
```

---

## 숏폼 3: "오늘 자동화한 것: 매일 아침 뉴스 브리핑" (25초)

```
[0~3초] 텍스트: "오늘 자동화한 것 🔧"

[3~10초] 카메라
"AI 뉴스를 매일 확인해야 하는데
 사이트 5개 돌면서 읽을 시간이 없어요."

[10~20초] 화면 녹화
"이제는. 매일 아침 7시 →
 AI가 5개 사이트에서 뉴스 수집 →
 비개발자에게 중요한 것만 3개 선별 →
 요약해서 슬랙으로."

[20~25초] "5개 사이트 × 30분 = 2.5시간 → 3분 읽기. → 프로필."
```
