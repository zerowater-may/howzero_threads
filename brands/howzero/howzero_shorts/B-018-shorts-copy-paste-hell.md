## 숏폼 1: "Ctrl+C, Ctrl+V 하루에 몇 번?" (30초)

```
[0~3초] (타이핑 소리 + 텍스트)
"Ctrl+C... Ctrl+V..."

[3~8초] 카메라 (지친 표정)
"오늘 하루에 복붙 몇 번 하셨어요?
 50번? 100번? 200번?"

[8~15초] 텍스트 애니메이션
"CS 답변 복붙: 50번"
"데이터 입력 복붙: 30번"
"보고서 수치 복붙: 20번"
"= 하루 100번+"

[15~22초] 카메라 (진지)
"이걸 사람이 할 이유가 없습니다.
 2026년에 복붙은 AI의 일입니다."

[22~27초] 화면: AI가 자동으로 처리하는 데모 (3초)

[27~30초] "복붙 탈출 → 프로필."
```

## 숏폼 2: "엑셀 → 엑셀 → 엑셀 → 그만" (25초)

```
[0~3초] 텍스트: "엑셀 A에서 복사 → 엑셀 B에 붙여넣기 → 엑셀 C에 또 붙여넣기"
(빠른 화면 전환)

[3~10초] 카메라
"이거 매일 하시는 분 손🙋. 하루 1시간.
 월 22시간. 연 264시간.
 시급 ₩25,000이면 연 ₩660만 원."

[10~18초] 화면: 자동화 데모
"엑셀 A의 데이터가 자동으로 B와 C에 동기화."
"복붙 0번."

[18~22초] 카메라
"₩660만 어치의 복붙을 AI가 0원에 합니다."

[22~25초] "방법 → 프로필."
```

## 숏폼 3: "복붙 vs 자동화, 1년 차이" (30초)

```
[0~3초] 텍스트: "복붙하는 사람 vs 자동화한 사람"

[3~10초] 화면 분할
좌 (복붙): "1년 후"
  "스킬: 복붙 숙련도 S급"
  "매출: 제자리"
  "야근: 매일"
  "번아웃: 진행 중"

우 (자동화): "1년 후"
  "스킬: AI 시스템 설계"
  "매출: 3배"
  "야근: 거의 없음"
  "여유: 넘침"

[10~22초] 카메라
"같은 1년인데 결과가 다릅니다.
 차이는 15분.
 15분 만에 첫 자동화를 만드느냐 마느냐."

[22~27초] 텍스트: "15분이 1년을 바꿉니다."

[27~30초] "시작 → 프로필."
```
