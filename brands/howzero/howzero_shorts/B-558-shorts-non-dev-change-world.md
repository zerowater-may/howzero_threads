# B-558: 숏폼 스크립트 3개 — "AI 자동화 — 비개발자가 세상을 바꾸는 방법" 시리즈

> **유형**: 쇼츠/틱톡/릴스 | **각 30~60초** | **시리즈**: #하우제로 #AI자동화

---

## 숏폼 1: "30초 데모: 비개발자 세상 변화 AI" (30초)

```
[0~3초] 텍스트: "비개발자 세상 변화 AI 자동화" (큰 글씨, 배경 전환)
나레이션: "비개발자 세상 변화, 자동화 가능합니다."

[3~10초] 화면: 수동 작업 장면
"비개발자 자동화 운동 — 자동화 교육 콘텐츠 자동 생성 수동으로 하고 있죠?"

[10~18초] 화면: 자동화 데모
"AI가 하면? 자동화 교육 콘텐츠 자동 생성 → 성공 사례 자동 수집 → 글로벌 커뮤니티 자동 매칭 3초."

[18~25초] 화면: 결과 대시보드
"월 40시간 → 거의 0. 실측 데이터입니다."

[25~30초] 텍스트: "코딩 0줄. 하우제로."
"방법은 프로필에. 하우제로."
```

**해시태그**: #하우제로 #AI자동화 #비개발자세상변화 #코딩없이 #업무자동화

---

## 숏폼 2: "비포/애프터: 비개발자 세상 변화" (45초)

```
[0~3초] 화면 분할: Before(빨강) / After(초록)
나레이션: "비개발자 세상 변화, Before & After."

[3~12초] Before 화면
"Before: 자동화 교육 콘텐츠 자동 생성 수동 처리.
 시간: 하루 2~3시간. 실수: 빈번. 야근: 기본."

[12~22초] After 화면 (자동화 데모)
"After: AI가 자동으로 자동화 교육 콘텐츠 자동 생성 → 성공 사례 자동 수집 → 글로벌 커뮤니티 자동 매칭.
 시간: 15분. 실수: 거의 0. 야근: 없음."

[22~35초] 결과 수치
"비개발자 자동화 운동 자동화, 과장 아닙니다.
 이게 코딩 0줄로 가능합니다."

[35~40초] 카메라 직접
"프로세스부터 정리하세요. 도구는 그 다음입니다."

[40~45초] CTA
"무료 진단 → 프로필 링크. 하우제로."
```

**해시태그**: #비포애프터 #AI자동화 #하우제로 #비개발자세상변화 #자동화전후

---

## 숏폼 3: "1분 자동화: 비개발자 세상 변화 핵심" (40초)

```
[0~3초] 텍스트: "1분 자동화" (카운트다운 시작)
나레이션: "비개발자 세상 변화 자동화, 핵심만 1분에."

[3~12초] Step 1
"Step 1: 자동화 교육 콘텐츠 자동 생성"
(화면: 트리거 설정)

[12~21초] Step 2
"Step 2: 성공 사례 자동 수집"
(화면: AI 처리 과정)

[21~30초] Step 3
"Step 3: 글로벌 커뮤니티 자동 매칭"
(화면: 결과 확인)

[30~35초] 결과
"코딩 0줄. 세팅 15분. 월 비용 3만 원 이하."

[35~40초] CTA
"전체 과정은 롱폼에서. 링크는 프로필.
코딩도 제로. 하우제로."
```

**해시태그**: #1분자동화 #비개발자세상변화 #AI자동화 #하우제로 #노코드
