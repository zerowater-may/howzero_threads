## 숏폼 1: "30초 데모: 한방 화장품 AI" (30초)

```
[0~3초] 텍스트: "한방 화장품/K-뷰티 AI 자동화" (큰 글씨, 배경 전환)
나레이션: "한방 화장품을 해외에 팔려면 서류만 100장."

[3~10초] 화면: 수동 작업 장면
"한방 화장품 — 성분 관리와 마케팅을 수동으로 처리."

[10~18초] 화면: 자동화 데모
"AI가 하면? 성분 데이터 자동 관리 → AI 마케팅 콘텐츠 생성 → 해외 수출 서류 자동화 3초."

[18~25초] 화면: 결과 대시보드
"월 40시간 → 거의 0. 실측 데이터입니다."

[25~30초] 텍스트: "코딩 0줄. 하우제로."
"방법은 프로필에. 하우제로."
```

## 숏폼 2: "비포/애프터: 한방 화장품" (45초)

```
[0~3초] 화면 분할: Before(빨강) / After(초록)
나레이션: "한방 화장품, Before & After."

[3~12초] Before 화면
"Before: 성분 관리와 마케팅을 수동으로 처리.
 시간: 하루 2~3시간. 실수: 빈번. 야근: 기본."

[12~22초] After 화면 (자동화 데모)
"After: AI가 자동으로 성분 데이터 자동 관리 → AI 마케팅 콘텐츠 생성 → 해외 수출 서류 자동화.
 시간: 15분. 실수: 거의 0. 야근: 없음."

[22~35초] 결과 수치
"한방 화장품을 해외에 팔려면 서류만 100장. AI는 자동으로 만듭니다.
 이게 코딩 0줄로 가능합니다."

[35~40초] 카메라 직접
"프로세스부터 정리하세요. 도구는 그 다음입니다."

[40~45초] CTA
"무료 진단 → 프로필 링크. 하우제로."
```

**해시태그**: #비포애프터 #AI자동화 #하우제로 #한방화장품 #자동화전후

## 숏폼 3: "1분 자동화: 한방 화장품 핵심" (40초)

```
[0~3초] 텍스트: "1분 자동화" (카운트다운 시작)

코딩도 제로. 하우제로."
```
