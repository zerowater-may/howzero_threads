# B-005: 숏폼 3개 — "1분 자동화 챌린지" 시리즈

> **유형**: 쇼츠/틱톡/릴스 | **시리즈**: #1분자동화챌린지

---

## 숏폼 1: "1분 안에 이메일 자동 분류 만들기" (50초)

```
[0~3초] 텍스트: "1분 챌린지 🔥" + 타이머 시작
"1분 안에 이메일 자동 분류를 만들겠습니다. 시작."

[3~50초] 화면 녹화 (2배속) + 보이스오버
"트리거 — 새 이메일 수신. 3초."
"AI 노드 연결 — '환불/배송/제품/기타로 분류해줘'. 15초."
"분류 결과 → 슬랙 알림 연결. 10초."
"테스트 이메일 발송... 3, 2, 1..."
"분류 완료. '환불'. 정확합니다."
"타이머 — 47초! 1분 안에 완성."

[마지막 5초] 카메라
"코딩 0줄. 47초. 이게 AI 자동화입니다. 방법 → 프로필."
```

---

## 숏폼 2: "1분 안에 자동 보고서 만들기" (45초)

```
[0~3초] 텍스트: "1분 챌린지 #2 🔥" + 타이머
"이번엔 주간 보고서를 자동으로 만들겠습니다. 시작."

[3~40초] 화면 녹화 (2배속)
"Google Sheets 데이터 연결. 5초."
"AI에게 '이 데이터를 요약해서 보고서 만들어줘'. 10초."
"결과를 이메일로 자동 발송. 10초."
"스케줄 설정 — 매주 월요일 9시. 5초."
"테스트... 보고서가 이메일로 왔습니다."
"52초! 완성."

[40~45초] 카메라
"이제 매주 월요일 보고서가 자동으로 옵니다. 프로필에서 더."
```

---

## 숏폼 3: "1분 안에 슬랙 알림 봇 만들기" (40초)

```
[0~3초] 텍스트: "1분 챌린지 #3 🔥" + 타이머
"슬랙 알림 봇. 1분. 갑니다."

[3~35초] 화면 녹화 (2배속)
"특정 조건 트리거 — '리뷰 별점 3 이하'. 5초."
"AI가 리뷰 내용 요약. 10초."
"슬랙 채널에 자동 알림 발송. 10초."
"테스트... 슬랙에 알림 도착!"
"38초! 여유 있게 완성."

[35~40초] 카메라
"부정 리뷰 놓치는 거, 오늘부터 끝. 방법 → 프로필."
```
