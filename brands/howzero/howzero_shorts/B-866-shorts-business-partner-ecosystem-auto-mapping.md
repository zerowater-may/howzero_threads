## 숏폼 1: "30초 데모: AI로 비즈니스 파트너 생태계 자동 맵핑 AI" (30초)

```
[0~3초] 텍스트: "AI로 비즈니스 파트너 생태계 자동 맵핑 AI 자동화" (큰 글씨, 배경 전환)
나레이션: "AI로 비즈니스 파트너 생태계 자동 맵핑, 자동화 가능합니다."

[3~10초] 화면: 수동 작업 장면
"AI로 비즈니스 파트너 생태계 자동 맵핑 — 파트너사 데이터 자동 수집 및 분류 수동으로 하고 있죠?"

[10~18초] 화면: 자동화 데모
"AI가 하면? 파트너사 데이터 자동 수집 및 분류 → 협력 관계 네트워크 시각화 자동 생성 → 시너지 기회 자동 탐지 및 알림 3초."

[18~25초] 화면: 결과 대시보드
"월 40시간 → 거의 0. 실측 데이터입니다."

[25~30초] 텍스트: "코딩 0줄. 하우제로."
"방법은 프로필에. 하우제로."
```

**해시태그**: #하우제로 #AI자동화 #AI로비즈니스파트너생태계자동맵핑 #코딩없이 #업무자동화

## 숏폼 2: "비포/애프터: AI로 비즈니스 파트너 생태계 자동 맵핑" (45초)

```
[0~3초] 화면 분할: Before(빨강) / After(초록)
나레이션: "AI로 비즈니스 파트너 생태계 자동 맵핑, Before & After."

[3~12초] Before 화면
"Before: 파트너사 데이터 자동 수집 및 분류 수동 처리.
 시간: 하루 2~3시간. 실수: 빈번. 야근: 기본."

[12~22초] After 화면 (자동화 데모)
"After: AI가 자동으로 파트너사 데이터 자동 수집 및 분류 → 협력 관계 네트워크 시각화 자동 생성 → 시너지 기회 자동 탐지 및 알림.
 시간: 15분. 실수: 거의 0. 야근: 없음."

[22~35초] 결과 수치
"AI로 비즈니스 파트너 생태계 자동 맵핑 자동화, 과장 아닙니다.
 이게 코딩 0줄로 가능합니다."

[35~40초] 카메라 직접
"프로세스부터 정리하세요. 도구는 그 다음입니다."

[40~45초] CTA
"무료 진단 → 프로필 링크. 하우제로."
```

**해시태그**: #비포애프터 #AI자동화 #하우제로 #AI로비즈니스파트너생태계자동맵핑 #자동화전후

## 숏폼 3: "1분 자동화: AI로 비즈니스 파트너 생태계 자동 맵핑 핵심" (40초)

```
[0~3초] 텍스트: "1분 안에 설명하는 AI로 비즈니스 파트너 생태계 자동 맵핑 AI 자동화"
나레이션: "1분 안에 보여드립니다."

[3~13초] Step 1
"첫째, 파트너사 데이터 자동 수집 및 분류.
 AI가 자동으로 수집하고 분류합니다."

[13~23초] Step 2
"둘째, 협력 관계 네트워크 시각화 자동 생성.
 수동으로 반나절 걸리던 게 3분."

[23~33초] Step 3
"셋째, 시너지 기회 자동 탐지 및 알림.
 사람이 할 필요 없습니다."

[33~38초] 결과
"결과: 월 40시간 → 거의 0."

[38~40초] CTA
"자동화 방법 → 프로필 링크. 하우제로."
```
