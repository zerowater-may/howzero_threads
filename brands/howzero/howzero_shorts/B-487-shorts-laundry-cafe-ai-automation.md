## 숏폼 1: "30초 데모: 런드리카페 AI" (30초)

```
[0~3초] 텍스트: "런드리카페/빨래방카페 AI 자동화" (큰 글씨, 배경 전환)
나레이션: "빨래방카페에서 세탁기 고장 알림을 손님이 알려준다면, 자동화가 필요합니다."

[3~10초] 화면: 수동 작업 장면
"런드리카페 — 기기 상태와 대기열을 수동으로 관리."

[10~18초] 화면: 자동화 데모
"AI가 하면? 기기 상태 자동 모니터링 → 대기열 AI 최적화 → 결제·정산 자동화 3초."

[18~25초] 화면: 결과 대시보드
"월 40시간 → 거의 0. 실측 데이터입니다."

[25~30초] 텍스트: "코딩 0줄. 하우제로."
"방법은 프로필에. 하우제로."
```

## 숏폼 2: "비포/애프터: 런드리카페" (45초)

```
[0~3초] 화면 분할: Before(빨강) / After(초록)
나레이션: "런드리카페, Before & After."

[3~12초] Before 화면
"Before: 기기 상태와 대기열을 수동으로 관리.
 시간: 하루 2~3시간. 실수: 빈번. 야근: 기본."

[12~22초] After 화면 (자동화 데모)
"After: AI가 자동으로 기기 상태 자동 모니터링 → 대기열 AI 최적화 → 결제·정산 자동화.
 시간: 15분. 실수: 거의 0. 야근: 없음."

[22~35초] 결과 수치
"빨래방카페에서 세탁기 고장 알림을 손님이 알려준다면, 자동화가 필요합니다.
 이게 코딩 0줄로 가능합니다."

[35~40초] 카메라 직접
"프로세스부터 정리하세요. 도구는 그 다음입니다."

[40~45초] CTA
"무료 진단 → 프로필 링크. 하우제로."
```

**해시태그**: #비포애프터 #AI자동화 #하우제로 #런드리카페 #자동화전후
