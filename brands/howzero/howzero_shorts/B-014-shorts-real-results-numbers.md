## 숏폼 1: "자동화 세팅 12분. 절감 연 2,880만 원." (20초)

```
[0~2초] 텍스트만 (검은 배경, 흰 글씨):
"세팅 시간: 12분"

[2~5초] "절감 시간: 하루 3.5시간"

[5~8초] "월 절감: ₩240만"

[8~11초] "연 절감: ₩2,880만"

[11~14초] "투자: ₩0 (무료 도구)"

[14~17초] "ROI: ∞"

[17~20초] "코딩: 0줄. 하우제로. → 프로필."
```

**포인트**: 말 0. 숫자만. 임팩트 최대.

---

## 숏폼 2: "100곳 진단 결과, 1위 비효율은?" (25초)

```
[0~3초] 텍스트: "100곳 AI 오딧 결과"

[3~6초] "가장 큰 비효율 1위:"
(드럼롤 효과)

[6~10초] 큰 텍스트: "CS 문의 처리 (68%)"

[10~15초] "2위: 데이터 수동 입력 (54%)"
"3위: 보고서 작성 (48%)"
"4위: 이메일 관리 (42%)"
"5위: 재고 관리 (35%)"

[15~20초] 카메라
"68%의 기업이 CS에서 시간을 가장 많이 낭비하고 있었습니다."

[20~25초] "우리 회사 1위는? → 프로필에서 진단."
```

## 숏폼 3: "자동화 전후 하루 비교" (25초)

```
[0~2초] 텍스트: "사장님의 하루"

[2~5초] "Before: 반복 업무 5시간 + 성장 일 2시간"
(파이차트: 빨강 71% / 초록 29%)

[5~8초] "After: 반복 업무 30분 + 성장 일 7시간"
(파이차트: 빨강 6% / 초록 94%)

[8~15초] 텍스트 애니메이션:
"같은 8시간."
"성장에 쓰는 시간: 2시간 → 7시간."
"3.5배."

[15~20초] 카메라
"1년이면 어떤 사장님이 더 성장할까요?"

[20~25초] "시작 → 프로필. 하우제로."
```
