## 숏폼 1: "병원 원장님, 노쇼 50% 줄이는 법" (30초)

```
[0~3초] 텍스트: "병원 원장님들 보세요"
"예약 잡아놓고 안 오는 환자. 노쇼."

[3~10초] 카메라
"노쇼 1건 = 빈 슬롯 30분 = 매출 ₩50,000 손실.
 하루 3건이면 월 ₩300만 날아갑니다."

[10~22초] 화면 녹화
"예약 확정 → 자동 확인 문자."
"D-1 → 자동 리마인더."
"당일 2시간 전 → 자동 리마인더."
"3번 리마인더 → 노쇼 50% 감소."

[22~27초] 카메라
"세팅 10분. 비용 월 ₩0. 노쇼 절반."

[27~30초] "원장님들 → 프로필."
```

## 숏폼 2: "학원 원장님, 학부모 문의 자동 응답" (30초)

```
[0~3초] 텍스트: "학원 원장님들 보세요"
"학부모 카톡 문의, 하루에 몇 건이에요?"

[3~10초] 카메라
"수강료 얼마예요? 몇 시에 해요? 주차 되나요?
 매일 같은 질문. 매일 같은 답변. 하루 1시간."

[10~22초] 화면 녹화
"AI 챗봇이 자동 응답."
"Q: 수강료 얼마예요? → '초등반 월 ₩30만입니다. 상담 예약 →'"
"Q: 시간표 알려주세요 → '화목 3시, 수금 4시입니다.'"
"80% 자동 응답. 나머지 20%만 원장님이."

[22~27초] 카메라
"하루 1시간 → 12분."

[27~30초] "학원 원장님들 → 프로필."
```

## 숏폼 3: "쇼핑몰 사장님, 반품 처리 자동화" (30초)

```
[0~3초] 텍스트: "쇼핑몰 사장님들 보세요"
"반품 요청. 가장 귀찮은 업무."

[3~10초] 카메라
"반품 사유 확인하고, 주문 조회하고, 택배 접수하고, 환불 처리하고.
 건당 15분."

[10~22초] 화면 녹화
"반품 요청 접수 → AI가 사유 분류."
"주문번호 자동 조회 → 반품 가능 여부 확인."
"택배 접수 링크 자동 발송."
"환불 처리 요청 자동 생성."
"건당 15분 → 2분."

[22~27초] 카메라
"하루 10건이면 2시간 10분 절감."

[27~30초] "사장님들 → 프로필."
```
