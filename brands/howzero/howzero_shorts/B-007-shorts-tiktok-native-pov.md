# B-007: 숏폼 3개 — 틱톡 네이티브 POV 시리즈

> **유형**: 틱톡 네이티브 (POV 포맷) | **시리즈**: #비개발자AI자동화

---

## 숏폼 1: "POV: 대표가 AI 도입하라는데" (35초)

```
[0~3초] 텍스트: "POV: 대표님이 AI 도입하라고 하는데 개발자가 없다"
(당황한 표정으로 카메라 봄)

[3~8초] 자막: "나: 개발자도 없는데 AI를요...?"
(고개 갸우뚱)

[8~12초] 자막: "근데 이 사람이 코딩 0줄로 10억 벌었다고?"
(눈 커지며 핸드폰 보는 연기)

[12~25초] 화면 전환 → 데모 몽타주 (빠르게)
"CS 자동 분류 — 됨."
"보고서 자동 생성 — 됨."
"리뷰 자동 수집 — 됨."
전부 코딩 없이.

[25~30초] 다시 카메라 (자신감 넘치는 표정)
자막: "나: 대표님, 제가 해보겠습니다."

[30~35초] 텍스트: "방법 → 프로필"
```

---

## 숏폼 2: "POV: 매일 엑셀 지옥인 직장인" (30초)

```
[0~3초] 텍스트: "POV: 매일 엑셀에 데이터 복붙하는 직장인"
(지친 표정, 키보드 타이핑)

[3~8초] 자막: "Ctrl+C... Ctrl+V... Ctrl+C... Ctrl+V..."
(반복 동작)

[8~12초] 자막: "매일 1시간. 매달 22시간. 매년 264시간."
(화면에 숫자 커짐)

[12~22초] 화면 전환 → 자동화 데모
자막: "근데 이걸 30초 만에 하는 방법이 있습니다."
(데이터 자동 수집 → 자동 입력 → 완료)

[22~27초] 카메라 (편안한 표정)
자막: "264시간 돌려받음."

[27~30초] "방법 → 프로필. 코딩 0줄."
```

---

## 숏폼 3: "POV: 사장님이 '보고서 어디있어?' 할 때" (35초)

```
[0~3초] 텍스트: "POV: 월요일 아침 사장님 카톡"
자막: "사장님: 주간 보고서 어디있어?"

[3~8초] 카메라 (당황)
자막: "나: 아직 안 만들었는데... (금요일에 퇴근했잖아)"

[8~15초] 텍스트 + 카메라
자막: "근데 만약..."
"AI가 일요일 밤에 자동으로 만들어서"
"월요일 아침에 이미 메일함에 있으면?"

[15~25초] 화면 녹화
"일요일 23:00 자동 실행"
"데이터 수집 → AI 요약 → 보고서 생성 → 이메일 발송"
"월요일 09:00 — 보고서 도착."

[25~30초] 카메라 (여유로운 표정)
자막: "나: 사장님, 이미 보내드렸습니다. ☕"

[30~35초] "세팅 시간: 11분. 코딩: 0줄. → 프로필"
```
