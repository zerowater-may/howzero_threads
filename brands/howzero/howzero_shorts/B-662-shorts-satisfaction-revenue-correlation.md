## 숏폼 1: "30초 데모: 고객 만족도-매출 상관관계 분석 AI" (30초)

```
[0~3초] 텍스트: "고객 만족도-매출 상관관계 분석 AI 자동화" (큰 글씨, 배경 전환)
나레이션: "고객 만족도-매출 상관관계 분석, 자동화 가능합니다."

[3~10초] 화면: 수동 작업 장면
"고객 만족도-매출 상관관계 분석 — 만족도 데이터 자동 수집 수동으로 하고 있죠?"

[10~18초] 화면: 자동화 데모
"AI가 하면? 만족도 데이터 자동 수집 → 매출 데이터 연동 자동화 → 상관관계 AI 분석 리포트 3초."

[18~25초] 화면: 결과 대시보드
"월 40시간 → 거의 0. 실측 데이터입니다."

[25~30초] 텍스트: "코딩 0줄. 하우제로."
"방법은 프로필에. 하우제로."

## 숏폼 2: "비포/애프터: 고객 만족도-매출 상관관계 분석" (45초)

```
[0~3초] 화면 분할: Before(빨강) / After(초록)
나레이션: "고객 만족도-매출 상관관계 분석, Before & After."

[3~12초] Before 화면
"Before: 만족도 데이터 자동 수집 수동 처리.
 시간: 하루 2~3시간. 실수: 빈번. 야근: 기본."

[12~22초] After 화면 (자동화 데모)
"After: AI가 자동으로 만족도 데이터 자동 수집 → 매출 데이터 연동 자동화 → 상관관계 AI 분석 리포트.
 시간: 15분. 실수: 거의 0. 야근: 없음."

[22~35초] 결과 수치
"고객 만족도-매출 상관관계 분석 자동화, 과장 아닙니다.
 이게 코딩 0줄로 가능합니다."

[35~40초] 카메라 직접
"프로세스부터 정리하세요. 도구는 그 다음입니다."

[40~45초] CTA
"무료 진단 → 프로필 링크. 하우제로."
```

## 숏폼 3: "1분 자동화: 고객 만족도-매출 상관관계 분석 핵심" (40초)

```

"코딩 0줄. 세팅 15분. 월 비용 3만 원 이하."

"전체 과정은 유튜브에. 하우제로."
