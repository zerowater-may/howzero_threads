# B-015: 숏폼 3개 — "자동화 일상" 비하인드 시리즈

> **시리즈**: #자동화일상 | **앵글**: "이 사람이 실제로 이렇게 사는구나" 리얼리티

## 숏폼 1: "아침에 일어나면 이미 다 되어 있음" (30초)

```
[0~3초] 텍스트: "☀️ 아침 8시. 출근."
(카메라: 컴퓨터 앞에 앉는 연기)

[3~8초] "이메일 열어볼게요."
(화면: 이메일 수신함)
"밤새 CS 문의 23건 → AI가 전부 분류 완료."

[8~13초] "슬랙 확인."
(화면: 슬랙)
"부정 리뷰 1건 → 답변 초안 준비 완료."

[13~18초] "보고서?"
(화면: 이메일 — 주간 보고서 도착)
"어젯밤 11시에 AI가 만들어서 보내놨네."

[18~25초] 카메라 (커피 들고 여유)
"제가 한 것: 0. AI가 밤새 일했습니다."

[25~30초] "이 시스템 → 프로필에서."
```

---

## 숏폼 2: "퇴근 후에도 AI는 일합니다" (25초)

```
[0~3초] 텍스트: "🌙 저녁 9시. 퇴근한 지 3시간."

[3~10초] (폰 알림 화면)
"알림 1: 새 리드가 웹사이트에서 문의 → CRM 자동 저장."
"알림 2: 경쟁사 A 가격 12% 하락 → 내일 확인."
"알림 3: 뉴스레터 내일 아침 8시 자동 발송 예약 완료."

[10~20초] 카메라 (소파에 누워서)
"저는 넷플릭스 보고 있었습니다.
 AI는 일하고 있었습니다.
 24시간 365일. 야근 없이."

[20~25초] "방법 → 프로필."
```

## 숏폼 3: "월요일 아침이 달라졌습니다" (30초)

```
[0~3초] 텍스트: "Before: 월요일 아침"
(지친 표정)

[3~8초] "보고서 만들어야 하고..."
"CS 밀린 거 처리해야 하고..."
"리뷰 확인해야 하고..."
"월요병 최고조."

[8~12초] 텍스트 전환: "After: 월요일 아침"
(여유로운 표정)

[12~22초] "보고서? 이미 메일함에 와 있음. ✅"
"CS? AI가 밤새 분류해놨음. ✅"
"리뷰? 부정 리뷰 0건. 알림 없음 = 문제 없음. ✅"

[22~27초] 카메라 (커피 마시며)
"월요일 아침에 할 일: 커피 한 잔 마시면서 검토만."

[27~30초] "이게 자동화입니다. → 프로필."
```
