# B-466: 숏폼 스크립트 3개 — "AI로 비즈니스 프로세스 리엔지니어링(BPR) 자동화" 시리즈

---

## 숏폼 1: "30초 데모: BPR자동화 AI" (30초)

```
[0~3초] 텍스트: "AI로 비즈니스 프로세스 리엔지니어링(BPR) 자동화" (큰 글씨, 배경 전환)
나레이션: "BPR자동화, 아직 수동으로 하세요?"

[3~10초] 화면: 수동 작업 장면
"프로세스 매핑 — 사람이 하면 하루 2~3시간."

[10~18초] 화면: 자동화 데모
"AI가 하면? 프로세스 마이닝 AI → 병목 자동 식별 → AI 기반 프로세스 재설계 3초."

[18~25초] 화면: 결과 대시보드
"월 40시간 → 거의 0. 실측 데이터입니다."

[25~30초] 텍스트: "코딩 0줄. 하우제로."
"방법은 프로필에. 하우제로."
```

## 숏폼 2: "비포/애프터: BPR자동화" (45초)

```
[0~3초] 화면 분할: Before(빨강) / After(초록)
나레이션: "BPR자동화, Before & After."

[3~12초] Before 화면
"Before: 프로세스 매핑 전부 수동.
 시간: 하루 2~3시간. 실수: 빈번. 야근: 기본."

[12~22초] After 화면 (자동화 데모)
"After: AI가 자동으로 프로세스 마이닝 AI → 병목 자동 식별 → AI 기반 프로세스 재설계.
 시간: 15분. 실수: 거의 0. 야근: 없음."

[22~35초] 결과 수치
"BPR을 아시나요? 90년대 유행했던 건데, AI를 만나면 완전히 다른 차원이 됩니다.
 이게 코딩 0줄로 가능합니다."

[35~40초] 카메라 직접
"프로세스부터 정리하세요. 도구는 그 다음입니다."

[40~45초] CTA
"무료 진단 → 프로필 링크. 하우제로."
```

## 숏폼 3: "1분 자동화: BPR자동화 핵심" (40초)

```

코딩도 제로. 하우제로."
```
