## 숏폼 1: "CS 비용 제로" (30초)

```
[0~2초] 텍스트: "CS 비용: ₩300만/월"
나레이션: "매달 CS에 300만 원 쓰고 계시죠?"

[2~5초] 텍스트 변환 애니메이션: "₩300만 → ₩0"
"제로로 만들었습니다."

[5~15초] 화면 녹화 — CS 자동화 데모 (빠르게)
"이메일 들어오면, AI가 분류하고, 답변 만들고, 알림 보내고."

[15~22초] 카메라
"세팅 15분. 코딩 0줄. 월 300만 원 → 0원."

[22~27초] 텍스트: "야근 제로. 코딩 제로. CS 비용 제로."
"야근도 제로. 코딩도 제로. CS 비용도 제로."

[27~30초] 텍스트: "하우제로."
"How to Zero. 하우제로."
```

## 숏폼 2: "코딩 제로로 만든 것들" (35초)

```
[0~3초] 카메라 + 텍스트: "코딩 0줄로 만든 것들"
"제 코딩 실력은 0입니다. 근데 이것들을 다 만들었습니다."

[3~8초] 화면: SaaS 대시보드
"1. SaaS 서비스 — 연매출 10억"

[8~13초] 화면: CS 자동화
"2. CS 자동화 — 직원 5명분 대체"

[13~18초] 화면: 보고서 자동 생성
"3. 자동 보고서 — 주 2시간 절감"

[18~23초] 화면: CRM 대시보드
"4. CRM 시스템 — 리드 관리 완전 자동"

[23~28초] 화면: 뉴스레터 자동 발송
"5. 뉴스레터 — 작성부터 발송까지 자동"

[28~35초] 카메라
"전부 코딩 0줄. 비결은요? 코딩이 아니라 프로세스.
 프로필 링크에서 방법 알려드립니다."
```

## 숏폼 3: "사장님들 이거 보세요" (틱톡 네이티브, 40초)

```
[0~3초] POV 텍스트: "POV: 사장님이 AI 자동화 하라고 하는데 개발자가 없다"
(살짝 당황한 표정)

[3~8초] 카메라 (당황)
"개발자 없이 AI 자동화를요? 그게 돼요?"

[8~12초] 표정 전환 (자신감)
"됩니다. 저도 개발자 0명으로 10억 벌었으니까."

[12~25초] 빠른 화면 전환 (데모 몽타주)
"CS 자동 분류 — 됩니다."
"보고서 자동 생성 — 됩니다."
"리뷰 자동 수집 — 됩니다."
"고객 알림 자동 발송 — 됩니다."

[25~33초] 카메라
"전부 코딩 0줄.
 비개발자가 15분이면 세팅합니다.
 개발자 없다고 핑계 대지 마세요."

[33~40초] 텍스트: "제로에서 10억까지. 코딩도 제로."
"방법은 프로필에. 하우제로."
```
