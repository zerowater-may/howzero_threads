## 숏폼 1: "하우제로 철학 자동화 Before vs After" (35초)

```
[0~3초] 텍스트: "하우제로 철학 + AI = ?"
나레이션: "하우제로 철학에서 코딩 장벽. 아직도 사람이 합니다."

[3~10초] 화면: Before 상황 시각화
"코딩 장벽에 매일 몇 시간씩 쓰고 있습니다."

[10~20초] 화면: AI 자동화 데모
"AI를 붙이면? 제로 프레임워크. 자동입니다."

[20~30초] 화면: 결과 수치
"결과: 모든 제약 요소의 체계적 제거."

[30~35초] 텍스트: "코딩 0줄. 하우제로."
"코딩 0줄입니다. 방법은 프로필에."
```

**해시태그**: #하우제로 #AI자동화 #하우제로 철학 #비개발자 #코딩없이

## 숏폼 2: "하우제로 철학에서 AI가 대체하는 업무 3가지" (40초)

```
[0~3초] 카메라 직접 + 텍스트: "하우제로 철학에서 AI가 대체하는 업무"
나레이션: "하우제로 철학 업종에서 AI가 당장 대체할 수 있는 업무 3가지."

[3~13초] 화면: 업무 1
"첫째. 제로 프레임워크. 사람이 할 필요 없습니다."

[13~23초] 화면: 업무 2
"둘째. 제약 해체 AI. 이것도 AI가 합니다."

[23~33초] 화면: 업무 3
"셋째. 영원의 자동화 아키텍처. 전부 자동."

[33~40초] 카메라 복귀
"이 3가지만 자동화해도 모든 제약 요소의 체계적 제거. 코딩? 0줄. 하우제로."
```

**해시태그**: #하우제로 #하우제로 철학자동화 #AI #비개발자 #업무자동화

## 숏폼 3: "이 업종도 AI 됩니다 — 하우제로 철학" (30초)

```
[0~3초] 텍스트: "이 업종도 AI 됩니다"
나레이션: "하우제로 철학? AI랑 관계없다고요?"

[3~8초] 카메라
"저도 처음엔 그렇게 생각했습니다."

[8~18초] 화면: 자동화 데모
"그런데 실제로 적용해 보니 — 모든 제약 요소의 체계적 제거, 무한 확장 가능 시스템."

[18~25초] 카메라
"비결은 간단합니다. AI 도구를 먼저 찾지 마세요. 프로세스 맵부터 그리세요."

[25~30초] 텍스트: "무료 AI 진단 → 프로필"
"우리 회사에도 되는지 궁금하면? 프로필 링크에서 무료 진단."
```
