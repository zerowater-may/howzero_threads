---

## 숏폼 1: "회의 끝나면 회의록이 이미 와있음" (25초)

```
[0~3초] 텍스트: "줌 미팅 끝나고 이메일 확인했는데"

[3~8초] 화면: 이메일 수신 — "회의 요약 리포트"
"회의록이 이미 와있습니다."

[8~15초] 화면: 회의록 내용 스크롤
"논의 사항 3가지.
 액션 아이템: [A]는 [누가] [수요일]까지.
 다음 미팅: [날짜]."

[15~20초] 카메라 (놀란 표정 → 만족)
"제가 쓴 글자: 0. AI가 미팅 끝나자마자 자동으로."

[20~25초] "세팅 5분. → 프로필."
```

---

## 숏폼 2: "고객이 예약하면 알아서 다 됨" (30초)

```
[0~3초] 텍스트: "고객이 상담 예약을 했습니다"

[3~10초] 화면 녹화 — 고객 시점
고객이 예약 페이지에서 시간 선택 → 예약 완료

[10~18초] 화면 — 뒤에서 일어나는 일 (자동)
"자동 확인 문자 발송 ✓"
"캘린더에 일정 추가 ✓"
"CRM에 고객 정보 저장 ✓"
"D-1 리마인더 예약 ✓"
"당일 줌 링크 자동 발송 ✓"

[18~25초] 카메라
"제가 한 것: 0. 전부 자동."

[25~30초] "전화 왔다갔다 15분 → 0분. 프로필."
```

---

## 숏폼 3: "30일 안 온 고객한테 자동으로 연락감" (30초)

```
[0~3초] 텍스트: "이 고객, 30일째 안 왔습니다"
(CRM 화면 — 비활성 고객 표시)

[3~8초] "보통이면? 모르고 넘어갑니다."

[8~18초] 화면 녹화 — 자동 이메일 발송
"근데 AI가 자동으로 감지합니다."
"30일 비활성 → 자동 이메일 발송:"
"'오랜만이에요! 놓치신 소식 3가지가 있어요 →'"

[18~23초] 화면: 고객 재방문 알림
"3일 후 — 이 고객이 다시 왔습니다."

[23~27초] 카메라
"이탈 고객 50% 복구. 자동으로."

[27~30초] "세팅법 → 프로필."
```
