## 숏폼 1: "홀로그래픽 리테일 자동화 Before vs After" (35초)

```
[0~3초] 텍스트: "홀로그래픽 리테일 + AI = ?"
나레이션: "홀로그래픽 리테일에서 매장 디스플레이 콘텐츠 교체가 수작업. 아직도 사람이 합니다."

[3~10초] 화면: Before 상황 시각화
"고객 동선 분석이 수동 관찰."

[10~20초] 화면: AI 자동화 데모
"AI를 붙이면? 홀로그래픽 콘텐츠 AI 자동 생성·교체. 자동입니다."

[20~30초] 화면: 결과 수치
"결과: 매장 체류 시간 48% 증가, 구매 전환율 35% 상승."

[30~35초] 텍스트: "코딩 0줄. 하우제로."
"코딩 0줄입니다. 방법은 프로필에."
```

**해시태그**: #하우제로 #AI자동화 #홀로그래픽리테일 #비개발자 #코딩없이

## 숏폼 2: "홀로그래픽 리테일에서 AI가 대체하는 업무 3가지" (40초)

```
[0~3초] 카메라 직접 + 텍스트: "홀로그래픽 리테일에서 AI가 대체하는 업무"
나레이션: "홀로그래픽 리테일 업종에서 AI가 당장 대체할 수 있는 업무 3가지."

[3~13초] 화면: 업무 1
"첫째. 홀로그래픽 콘텐츠 AI 자동 생성·교체. 사람이 할 필요 없습니다."

[13~23초] 화면: 업무 2
"둘째. 고객 동선 AI 분석 + 디스플레이 최적 배치. 이것도 AI가 합니다."

[23~33초] 화면: 업무 3
"셋째. 실시간 구매 전환 AI 추적. 전부 자동."

[33~40초] 카메라 복귀
"이 3가지만 자동화해도 매장 체류 시간 48% 증가, 구매 전환율 35% 상승. 코딩? 0줄. 하우제로."
```

## 숏폼 3: "이 업종도 AI 됩니다 — 홀로그래픽 리테일" (30초)

```
[0~3초] 텍스트: "이 업종도 AI 됩니다"
나레이션: "홀로그래픽 리테일? AI랑 관계없다고요?"
```
