# B-009: 숏폼 3개 — 이커머스 셀러 공감 시리즈

> **유형**: 쇼츠/틱톡/릴스 | **타겟**: 이커머스 셀러 전용

---

## 숏폼 1: "셀러의 하루 요약" (35초, 틱톡 네이티브)

```
[0~3초] 텍스트: "이커머스 셀러의 진짜 하루"
(지친 표정)

[3~7초] "아침 9시. 카톡 50개. 전부 CS."
(핸드폰 보는 연기)

[7~11초] "10시. 환불 3건 처리. 배송 문의 12건."
(타이핑 연기)

[11~15초] "12시. 리뷰 확인. 별점 1짜리 올라옴."
(충격 표정)

[15~19초] "2시. 신상품 등록하려는데 번역이 안 끝남."
(머리 감싸기)

[19~23초] "5시. 경쟁사 가격 확인하는데 벌써 5시."
(시계 보기)

[23~27초] "7시. 이제 보고서 써야 하는데..."
(절망)

[27~32초] 표정 전환 (자신감)
"근데 이걸 전부 AI가 해주면요?"

[32~35초] "코딩 0줄. 세팅 15분. → 프로필"
```

---

## 숏폼 2: "쿠팡 환불 자동 처리 30초 데모" (30초)

```
[0~3초] 텍스트: "쿠팡 환불 요청, 아직 수동으로?"
"환불 요청 왔습니다."

[3~10초] 화면 녹화
"AI가 주문번호 자동 추출."
"환불 조건 자동 체크. 7일 이내? → 가능."
"환불 안내 답변 자동 생성."

[10~20초] 화면 녹화 계속
"담당자 카톡 알림: '환불 승인하시겠습니까?'"
"담당자: 승인 클릭. 끝."

[20~25초] 카메라
"건당 15분 → 30초. 하루 5건이면 1시간 10분 절감."

[25~30초] "세팅 10분. 코딩 0줄. → 프로필"
```

---

## 숏폼 3: "스마트스토어 셀러 수익 계산기" (35초)

```
[0~3초] 텍스트: "셀러님, 이거 계산해보셨어요?"
"매일 CS에 몇 시간 쓰세요?"

[3~8초] 카메라 + 계산기 화면
"하루 3시간이라고요? 계산해볼게요."

[8~18초] 텍스트 애니메이션 (숫자 올라감)
"3시간 × 22일 = 66시간/월"
"시급 ₩25,000 기준"
"= ₩1,650,000/월"
"= ₩19,800,000/년"

[18~25초] 카메라 (충격 표정)
"CS에 연간 2,000만 원 쓰고 계신 겁니다.
 이 시간에 소싱했으면 매출이 올랐을 텐데."

[25~30초] 텍스트: "이걸 0으로 만들면?"
"AI 자동화 비용: ₩200만 (1회). ROI: 890%."

[30~35초] "방법 → 프로필. 하우제로."
```
