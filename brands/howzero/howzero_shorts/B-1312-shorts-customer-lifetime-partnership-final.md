## 숏폼 1: "고객 생애 파트너십 자동화 Before vs After" (35초)

```
[0~3초] 텍스트: "고객 생애 파트너십 + AI = ?"
나레이션: "고객 생애 파트너십에서 고객 가치를 매출로만 판단. 아직도 사람이 합니다."

[3~10초] 화면: Before 상황 시각화
"장기 고객 관리가 담당자 기억에 의존."

[10~20초] 화면: AI 자동화 데모
"AI를 붙이면? 고객 생애 가치 AI 자동 산출. 자동입니다."

[20~30초] 화면: 결과 수치
"결과: 고객 생애 가치 평균 3.2배 증가."

[30~35초] 텍스트: "코딩 0줄. 하우제로."
"코딩 0줄입니다. 방법은 프로필에."
```

## 숏폼 2: "고객 생애 파트너십에서 AI가 대체하는 업무 3가지" (40초)

```
[0~3초] 카메라 직접 + 텍스트: "고객 생애 파트너십에서 AI가 대체하는 업무"
나레이션: "고객 생애 파트너십 업종에서 AI가 당장 대체할 수 있는 업무 3가지."

[3~13초] 화면: 업무 1
"첫째. 고객 생애 가치 AI 자동 산출. 사람이 할 필요 없습니다."

[13~23초] 화면: 업무 2
"둘째. 파트너십 단계별 자동 관리. 이것도 AI가 합니다."

[23~33초] 화면: 업무 3
"셋째. 장기 고객 자동 리워드 시스템. 전부 자동."

[33~40초] 카메라 복귀
"이 3가지만 자동화해도 고객 생애 가치 평균 3.2배 증가. 코딩? 0줄. 하우제로."
```

## 숏폼 3: "이 업종도 AI 됩니다 — 고객 생애 파트너십" (30초)

```
[0~3초] 텍스트: "이 업종도 AI 됩니다"
나레이션: "고객 생애 파트너십? AI랑 관계없다고요?"

"저도 처음엔 그렇게 생각했습니다."

"우리 회사에도 되는지 궁금하면? 프로필 링크에서 무료 진단."
```
