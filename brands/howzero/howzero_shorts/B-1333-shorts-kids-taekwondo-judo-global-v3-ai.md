## 숏폼 1: "키즈 태권도·유도 글로벌 도장 자동화 Before vs After" (35초)

```
[0~3초] 텍스트: "키즈 태권도·유도 글로벌 도장 + AI = ?"
나레이션: "키즈 태권도·유도 글로벌 도장에서 수련생 승급 관리·출석이 수작업. 아직도 사람이 합니다."

[3~10초] 화면: Before 상황 시각화
"글로벌 도장 간 커리큘럼 표준화 어려움."

[10~20초] 화면: AI 자동화 데모
"AI를 붙이면? 수련생 승급 AI 자동 관리 + 역량 분석. 자동입니다."

[20~30초] 화면: 결과 수치
"결과: 승급 심사 효율 70% 향상, 글로벌 커리큘럼 통일률 95%."

[30~35초] 텍스트: "코딩 0줄. 하우제로."
"코딩 0줄입니다. 방법은 프로필에."
```

## 숏폼 2: "키즈 태권도·유도 글로벌 도장에서 AI가 대체하는 업무 3가지" (40초)

```
[0~3초] 카메라 직접 + 텍스트: "키즈 태권도·유도 글로벌 도장에서 AI가 대체하는 업무"
나레이션: "키즈 태권도·유도 글로벌 도장 업종에서 AI가 당장 대체할 수 있는 업무 3가지."

[3~13초] 화면: 업무 1
"첫째. 수련생 승급 AI 자동 관리 + 역량 분석. 사람이 할 필요 없습니다."

[13~23초] 화면: 업무 2
"둘째. 글로벌 커리큘럼 AI 표준화 시스템. 이것도 AI가 합니다."

[23~33초] 화면: 업무 3
"셋째. 학부모 리포트 자동 생성 + 다국어 지원. 전부 자동."

[33~40초] 카메라 복귀
"이 3가지만 자동화해도 승급 심사 효율 70% 향상, 글로벌 커리큘럼 통일률 95%. 코딩? 0줄. 하우제로."
```

## 숏폼 3: "이 업종도 AI 됩니다 — 키즈 태권도·유도 글로벌 도장" (30초)

