## 대본

### 훅 (0:00–0:40)
네일샵 운영하시는 분, 노쇼(no-show) 때문에 빈 좌석 생기는 거 한 달에 몇 건이세요? 평균 15~20%. 매출로 환산하면 월 200만 원 이상 날아갑니다. AI가 예약 확인부터 노쇼 예측, 대기 고객 자동 배정, 시술 트렌드 분석까지 합니다. 노쇼 80% 감소, 시술 단가 15% 상승. 데이터입니다.

### 문제 정의 (0:40–3:00)
네일/뷰티 업계의 병목. 첫째, 예약 관리 — 전화·DM·네이버 예약이 섞여서 더블 부킹, 노쇼가 빈번합니다. 둘째, 시술 추천 — 고객별 선호 스타일, 피부/손톱 상태를 기억하기 어렵습니다. 셋째, 트렌드 대응 — 시즌별 인기 디자인이 빠르게 바뀌는데, 감으로 대응합니다. 넷째, 재료 관리 — 젤, 파츠, 컬러 재고를 수동으로 관리합니다. 인기 컬러 품절, 비인기 컬러 과잉 재고.

### 자동화 설계 (3:00–8:00)
**예약 통합+노쇼 방지**: 모든 채널 예약 자동 통합 → 24시간 전·2시간 전 자동 리마인더 → 노쇼 이력 고객 플래그 + 사전 예약금 안내 → 취소 발생 시 대기 고객 자동 알림.

**AI 시술 추천**: 고객별 시술 이력 + 선호 스타일 자동 기록 → 재방문 시 AI가 "지난번 프렌치 네일 하셨는데, 이번엔 시즌 트렌드인 아우라 네일 어떠세요?" 추천 → 업셀링 성공률 증가.

**트렌드 자동 분석**: 인스타그램·핀터레스트 네일 트렌드 AI 크롤링 → 월간 인기 디자인 TOP 10 자동 리포트 → 시술 메뉴 업데이트 제안.

**재료 자동 관리**: 시술 건별 사용 재료 자동 차감 → 재고 부족 시 자동 발주 → 인기/비인기 컬러 데이터 기반 발주량 최적화.
