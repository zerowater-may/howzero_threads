## 대본

### 훅 (0:00–0:40)
직원이 퇴사 의향을 밝히면 이미 늦습니다. 마음을 되돌리기 거의 불가능합니다. 이직을 결심하기까지 평균 6개월. 그 6개월 동안 신호가 있었습니다. 불만, 번아웃, 팀 갈등. 그런데 1년에 한 번 하는 직원 만족도 조사로는 못 잡습니다. AI가 매주 자동으로 펄스 체크하고, 감성 분석하고, 이직 위험 신호를 실시간으로 감지합니다. 이직률 40% 감소.

### 문제 정의 (0:40–3:00)
직원 피드백 관리의 현실. 첫째, 빈도가 너무 낮음 — 연 1~2회 만족도 조사. 6개월 전 불만이 지금도 유효한지 모릅니다. 둘째, 형식적 응답 — 실명 조사면 솔직 안 하고, 익명이면 후속 조치가 어렵습니다. 셋째, 분석 불가 — 서술형 응답 수백 개를 사람이 읽을 수 없습니다. 요약도 안 됩니다. 넷째, 액션 연결 부재 — 피드백 결과가 경영진 보고서로 끝. 실제 변화가 없으니 다음에는 응답도 안 합니다.

### 자동화 설계 (3:00–8:00)
**자동 펄스 서베이**: 매주 금요일 2~3문항 자동 발송(슬랙/이메일) → 30초면 응답 가능 → 주제 자동 로테이션(업무 만족·팀 분위기·성장 기회·워크라이프 밸런스) → 응답률 자동 추적 + 미응답자 부드러운 리마인더.

**AI 감성 분석**: 서술형 응답 AI 자동 분석 → 감성 스코어(긍정/중립/부정) 자동 산출 → 키워드 자동 추출("야근", "소통 부족", "인정") → 부서별·직급별 감성 트렌드 자동 비교.

**이직 위험 신호 감지**: 개인별 감성 스코어 3주 연속 하락 → 응답 내 이직 관련 키워드 감지("다른 회사", "성장 한계") → 업무량 급증 + 만족도 급락 패턴 → 위험 직원 HR에 자동 알림(익명성 보장하면서 팀 단위 알림).

**액션 자동 연결**: 피드백에서 반복 등장하는 이슈 자동 클러스터링 → 우선순위 자동 산정(빈도×심각도) → 경영진에게 액션 아이템 자동 제안 → 개선 조치 후 관련 피드백 변화 자동 추적("3개월 전 야근 불만이 많았는데, 유연근무 도입 후 해당 키워드 80% 감소").
