## 대본

### 훅 (0:00–0:40)
피부과·성형외과에서 가장 많은 전화가 뭔지 아십니까. "레이저 시술 가격이 얼마예요?" "보톡스 예약하려는데요." "시술 후 주의사항이 뭐예요?" 이 3가지가 전화의 70%입니다. 간호사가 전화 받느라 진료 보조를 못 합니다. AI로 예약, 상담, 시술 후 관리까지 자동화해서 전화량 80% 줄이고, 재시술률 2배 올린 피부과 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:30)
피부과/성형외과의 3대 병목. 첫째, 상담 문의 폭주 — 시술 종류별 가격, 부작용, 소요 시간 등 반복 질문이 일 50건+. 간호사 2명이 전화·카톡에 매달림. 둘째, 노쇼·예약 관리 — 시술 예약 노쇼율 평균 15%. 시술실·장비 가동률 직접 타격. 시간대별 예약 최적화도 수동. 셋째, 시술 후 관리 — 시술 후 주의사항 안내, 경과 확인, 재시술 시기 안내를 수동으로 하면 누락. 재시술률이 낮아지는 주요 원인.

### 자동화 설계 (3:30–9:30)
**AI 상담 챗봇**: 카카오 챗봇으로 시술 안내(종류별 설명, 가격, 소요 시간, 전후 사진) → 피부 고민 입력 시 AI가 맞춤 시술 추천 → 예약까지 원스톱 → 복잡한 상담만 실장에게 연결.

**스마트 예약 관리**: 온라인 예약 → 시술실·장비·원장 스케줄 자동 확인 → 최적 시간대 배치 → 예약 확정 알림 → 시술 전날 + 당일 2시간 전 리마인드 → 노쇼 시 대기 환자 자동 연락.

**시술 후 자동 케어**: 시술 완료 → 당일 주의사항 카카오 알림(세안 시기, 자외선 차단, 금지 행위) → 3일 후 경과 확인 메시지("시술 부위 이상은 없으신가요?") → 이상 응답 시 즉시 진료 예약 제안 → 재시술 적정 시기(보톡스 3개월, 레이저 4주 등) 자동 안내.

**고객 이력 관리**: 전 시술 이력, 피부 상태, 알레르기, 선호 시술 구글 시트 관리 → 원장 진료 전 환자 이력 자동 브리핑 → 시술 간격·횟수 자동 추적.
