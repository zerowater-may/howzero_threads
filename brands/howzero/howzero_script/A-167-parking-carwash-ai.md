## 대본

### 훅 (0:00–0:40)
주차장·세차장 체인 운영하시는 분들. 현장 매니저 없으면 현황 파악이 안 되고, 있어도 엑셀에 수기로 적어서 보고합니다. 점포 5개 넘으면 누가 뭘 하는지 파악이 불가능합니다. AI로 점포별 실시간 현황부터 수요 예측 기반 인력 배치까지 자동화한 사례. 코딩 제로입니다.

### 문제 정의 (0:40–3:30)
주차장·세차장 체인의 3대 병목. 첫째, 요금·매출 관리 — 점포별 요금 체계 다름, 할인 적용 기준 불일치, 매출 집계 수작업. 둘째, 예약 관리 — 세차 예약 전화로만 받음. 대기 시간 안내 불가. 노쇼. 셋째, 인력 배치 — 비 오는 날 세차 수요 급감인데 인력 동일 배치. 주말 주차장 만차인데 인력 부족.

### 자동화 설계 (3:30–9:30)
**매출·요금 통합 관리**: 점포별 POS/결제 데이터 자동 수집 → 일별·점포별 매출 대시보드 자동 생성 → 요금 체계 중앙 관리(본사에서 일괄 변경) → 할인·프로모션 적용 자동 기록.

**세차 예약 자동화**: 온라인 예약 페이지 → 실시간 대기 현황 표시 → 예약 확정 시 도착 시간·소요 시간 자동 안내 → 노쇼 방지: D-1, H-1 자동 리마인드 → 세차 완료 시 픽업 알림.

**AI 수요 예측+인력 배치**: 과거 데이터 + 날씨 예보 + 요일 패턴 → 점포별 익일 수요 예측(주차: 만차 예상 시간, 세차: 예상 건수) → 수요 기반 인력 배치 자동 산출 → 매니저에게 전일 저녁 자동 알림.

**점포 비교 분석**: 점포별 KPI(매출, 가동률, 고객 만족도, 인건비율) 자동 비교 → 하위 점포 자동 경고 → 상위 점포 성공 패턴 자동 분석.

### 데모/사례 (9:30–13:30)
세차장 체인 W. 7개 점포. 자동화 전: 매출 집계 월 1회 수작업, 예약 전화만, 인력 배치 고정형, 점포 비교 불가. 자동화 후: 매출 실시간 대시보드. 세차 예약 온라인 전환 70%. 날씨 기반 인력 최적화로 인건비 18%↓. 점포별 KPI 비교로 하위 2개 점포 개선 → 전체 매출 11%↑.

도구: POS 연동 + 구글 시트 + Make + GPT-4 + 날씨 API + 카카오 알림톡. 월 운영비 약 10만 원.
