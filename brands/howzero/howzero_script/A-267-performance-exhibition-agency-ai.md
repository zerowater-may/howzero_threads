## 대본

### 훅 (0:00–0:40)
공연/전시 기획하시는 분, 좌석 점유율 80% 넘기기 힘드시죠? 마케팅은 열심히 하는데, 누구에게 어떤 메시지를 보내야 할지 감으로 합니다. 잔여석 소진은 공연 전날 할인 폭탄. AI가 관객 데이터 기반으로 타겟 마케팅을 자동 실행하고, 다이나믹 프라이싱으로 좌석 점유율을 최적화합니다. 좌석 점유율 72%에서 92%로. 빈 좌석은 곧 버리는 돈입니다.

### 문제 정의 (0:40–3:00)
공연/전시 기획의 병목. 첫째, 관객 타겟팅 — 어떤 관객이 어떤 공연을 좋아하는지 데이터가 없습니다. 전체 DB에 같은 메시지 발송. 둘째, 가격 최적화 — 고정 가격이라 초반에 안 팔리면 결국 할인. 수익이 깎입니다. 셋째, 현장 운영 — 입장 대기, 동선, 안내를 사람이 수동으로 관리합니다. 넷째, 사후 관리 — 관객 피드백 수집, 다음 공연 마케팅 연결이 안 됩니다.

### 자동화 설계 (3:00–8:00)
**관객 세그먼트+맞춤 마케팅**: 과거 관람 이력 + 장르 선호 + 가격 민감도 AI 분석 → 관객 세그먼트 자동 분류(클래식 마니아/가족 관람/데이트족/문화 소비자) → 세그먼트별 맞춤 메시지 자동 발송 → "지난 번 재즈 공연 좋아하셨죠? 다음 달 같은 아티스트 내한 공연 있어요" 자동.

**다이나믹 프라이싱**: 판매 추이 실시간 분석 → 빠르게 팔리는 좌석 구간 가격 자동 상향 → 느리게 팔리는 구간 조기 할인 → 공연 D-7 잔여석 기반 최종 가격 자동 조정 → 총 매출 최적화.

**현장 운영 자동화**: 전자 티켓 QR 자동 발권 → 입장 시간대별 관객 수 예측 → 안내 인력 배치 자동 제안 → 동선 혼잡도 실시간 모니터링 → 긴급 공지(지연·변경) 관객 일괄 자동 발송.

**사후 관리 자동화**: 공연 다음 날 만족도 조사 자동 발송 → 별점+텍스트 피드백 AI 분석 → 인사이트 요약 리포트 자동 생성 → 관람 이력 기반 다음 공연 추천 자동 발송.

### 데모/사례 (8:00–12:30)
공연 기획사 AO. 연간 공연 24건, 평균 좌석 800석. 자동화 전: 평균 좌석 점유율 72%, 할인 판매 비율 35%, 관객 DB 활용 없음, 사후 피드백 수집률 5%. 자동화 후: 평균 좌석 점유율 92%, 할인 판매 비율 15%(다이나믹 프라이싱), 맞춤 추천 발송 후 구매 전환율 18%, 사후 피드백 수집률 42%.

도구: 구글 시트 + Make + GPT-4 + 카카오 알림톡 + 티켓 시스템 API. 월 운영비 약 10만 원.
