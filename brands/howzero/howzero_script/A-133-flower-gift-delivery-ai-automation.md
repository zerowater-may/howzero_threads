## 대본

### 훅 (0:00–0:40)
꽃배달 사업에서 가장 중요한 날이 언제인지 아십니까. 밸런타인데이, 어버이날, 크리스마스. 이 3일에 연매출의 40%가 몰립니다. 그런데 주문이 폭주하면 배송 지연, 주문 누락, 메시지 카드 오류가 터집니다. AI로 피크 시즌 주문 관리부터, 평소 기념일 리마인드까지 자동화한 꽃배달 업체 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
꽃배달/선물 배달의 3대 병목. 첫째, 주문 집중 — 특정일에 평소 10배 주문 폭주. 전화·카톡 동시에 쏟아지면 주문 누락, 배송 오류. 둘째, 배송 관리 — 시간 지정 배달, 서프라이즈 배달, 부재 시 처리 등 변수가 많음. 배달 기사와의 커뮤니케이션 복잡. 셋째, 재주문 유도 — 1회성 구매 고객이 70%+. 기념일 리마인드가 없으면 다음 해에 다른 업체로.

### 자동화 설계 (3:00–8:00)
**주문 자동 접수**: 카카오 챗봇 + 웹 주문 폼 → 상품 선택, 배송지, 배송일시, 메시지 카드 문구 입력 → 자동 주문 확인 → 결제 링크 자동 발송 → 주문 현황 구글 시트 자동 업데이트.

**배송 자동 관리**: 주문 확정 → 배송 지역별 자동 분류 → 배달 기사 자동 배정 → 배달 루트 최적화 제안 → 배달 시작·완료 시 고객 자동 알림 → 부재 시 대응 시나리오(문앞 배치 + 사진 전송).

**기념일 리마인드**: 주문 시 기념일 정보 수집(생일, 결혼기념일, 어버이날 등) → 기념일 14일 전 자동 리마인드("○○님 생일이 2주 앞으로 다가왔습니다. 작년에 보내신 꽃다발과 비슷한 상품을 준비했습니다") → 원클릭 재주문 → 정기 구독 전환 유도.

**피크 시즌 자동화**: D-7부터 주문 현황 일일 리포트 → 배달 용량 초과 시 자동 마감 → 추가 배달 기사 필요 시 알림 → 피크 당일 실시간 배송 현황 대시보드.

### 데모/사례 (8:00–13:30)
꽃배달 업체 FL. 일 평균 주문 15건, 피크 시즌 일 150건+. 자동화 전: 피크 시즌 주문 누락 일 5건+, 배송 지연율 20%, 기념일 재주문율 8%, 전화 응대 마비. 자동화 후: 챗봇으로 주문 접수 80% 처리 — 누락 0건. 배송 자동 배정으로 지연율 20%→5%. 기념일 리마인드로 재주문율 8%→32%. 피크 시즌 매출 45%↑.
