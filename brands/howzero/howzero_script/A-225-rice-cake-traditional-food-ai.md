## 대본

### 훅 (0:00–0:40)
떡집 사장님, 명절 시즌에 주문 몇 건 받으세요? 전화로 받고, 노트에 적고, 수량 계산하고, 배송 일정 잡고. 명절 때 잠을 못 자시죠. AI가 주문 접수부터 생산량 계산, 배송 스케줄까지 전부 자동으로 처리하는 시스템. 명절에도 사장님은 떡 만드는 데만 집중합니다.

### 문제 정의 (0:40–3:00)
떡집·전통식품의 병목. 첫째, 주문 관리 — 전화·카톡 주문이 대부분. 수량, 종류, 배송일 기록이 수동. 명절 시즌 주문 폭주 시 실수 빈발. 둘째, 생산 계획 — 주문량에 맞춘 재료 준비, 생산 스케줄 수립이 감에 의존. 셋째, 배송 관리 — 당일 제조·당일 배송 특성상 배송 타이밍이 중요한데 수동 조율.

### 자동화 설계 (3:00–7:30)
**주문 자동 접수**: 카카오 채널 주문 폼(제품, 수량, 배송일, 주소) → 자동으로 주문 DB 등록 → 주문 확인 + 입금 안내 자동 발송 → 입금 확인 자동.

**생산 계획 자동화**: 전체 주문 집계 → 제품별 필요 재료 자동 계산 → 재료 발주 자동 생성 → 일별 생산 스케줄(제조 순서, 수량) 자동 배정.

**배송 자동 관리**: 제조 완료 → 배송 건별 운송장 자동 생성 → 고객에게 배송 시작 알림 + 추적 번호 자동 발송 → 배송 완료 확인.

**시즌 예측 자동화**: 전년도 명절 주문 데이터 → AI가 올해 예상 주문량 예측 → 사전 재료 확보 추천 → 조기 주문 할인 프로모션 자동 발송.

### 데모/사례 (7:30–11:30)
전통 떡 전문점 M. 월 주문 200건, 설·추석 시즌 월 800건. 자동화 전: 명절 주문 전화 일 60통, 주문 실수(수량 오기, 배송일 착오) 월 5~8건, 재료 부족으로 긴급 발주 월 3회, 사장님 명절 시즌 하루 16시간 근무. 자동화 후: 주문 90% 온라인 자동, 주문 실수 0~1건, 재료 사전 확보 → 긴급 발주 0회, 사장님 명절 시즌 하루 10시간(제조 집중).

명절 매출 20% 증가(주문 처리 용량 증가). 고객 불만 85% 감소.
