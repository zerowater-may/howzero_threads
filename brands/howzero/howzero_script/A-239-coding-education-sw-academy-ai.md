**진도 추적 자동화**: 수강생별 학습 진행률, 과제 제출률, 점수 추이 자동 추적 → 주간 진도 대시보드 → 평균 이하 수강생 자동 플래그 → 강사에게 "1:1 케어 추천" 알림.

**탈락 위험 조기 감지**: 출석 패턴 + 과제 제출 지연 + 점수 하락 + 질문 빈도 감소 → AI가 탈락 위험도 점수 산출 → 위험 수강생에게 자동 격려 메시지 + 강사 알림.

**수준별 학습 경로 자동화**: 과제 점수 + 학습 속도 기반 → AI가 수강생별 보충 학습 자료 자동 추천 → 빠른 수강생에게 심화 과제 자동 제공 → 느린 수강생에게 복습 자료 자동 제공.
