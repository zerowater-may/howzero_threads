## 대본

### 훅 (0:00–0:40)
ESG, 탄소 중립. 대기업만의 이야기라고요? 2026년부터 중소기업도 공급망 탄소 배출 보고 의무가 확대됩니다. 준비 안 하면 거래처에서 잘립니다. 그런데 탄소 배출 측정만 해도 컨설팅 비용 수천만 원. AI가 에너지 사용량 자동 추적, 탄소 배출량 자동 산출, 감축 포인트 자동 식별, 보고서 자동 생성까지 합니다. 비용의 1/10로 ESG 대응. 실전 방법을 알려드립니다.

### 왜 지금인가 (0:40–3:30)
**규제 현실**: EU CBAM(탄소국경조정제도) 시행 중. 한국 탄소 배출권 거래제 확대. 대기업 공급망 실사 의무화 → 중소 협력업체에 탄소 배출 데이터 요구 증가. 2026년부터 매출 500억 이상 기업 ESG 보고 의무화(예정).

**비즈니스 기회**: ESG 잘하는 기업 → 금리 우대(최대 0.3%p) + 정부 보조금 + 대기업 공급망 우선 선정. 탄소 절감 = 에너지 비용 절감 = 수익 증가. 친환경 브랜드 이미지 → 소비자 프리미엄.

핵심: 탄소 중립은 비용이 아니라 투자입니다. 그런데 측정과 보고가 너무 어렵습니다. AI가 이걸 해결합니다.

### AI 자동화로 탄소 관리하는 4단계 (3:30–10:00)
**1단계: 자동 측정**: 전기·가스·수도 사용량 자동 수집(관리비 고지서 OCR 또는 스마트 미터 연동) → 차량 연료비 자동 집계 → 출장 교통 데이터 자동 수집 → 배출 계수 적용해 탄소 배출량 자동 산출(Scope 1·2·3 구분).

**2단계: 자동 분석+감축 포인트 식별**: 월별·부서별·활동별 배출량 자동 비교 → 에너지 사용 패턴 AI 분석 → "매주 토요일 사무실 에어컨이 12시간 가동되고 있습니다"(1인 출근에 전체 가동) → 감축 가능 포인트 자동 식별+우선순위.

**3단계: 감축 실행 자동화**: 조명·냉난방 스케줄 자동 최적화 → 비사용 시간대 자동 절전 → 물류 루트 최적화(배송 거리 단축 = 연료 절감) → 종이 사용량 모니터링+디지털 전환 제안.

**4단계: 보고서 자동 생성**: GRI·TCFD·CDP 등 글로벌 프레임워크 기준 보고서 자동 생성 → 전년 대비 감축률 자동 계산 → 목표 대비 달성률 대시보드 → 거래처/투자자 요청 시 원클릭 제출.

### 실전 사례 (10:00–14:00)

1년 차 감축 성과: 에너지 최적화로 연 2,880만 원 절감 + 탄소 배출 18% 감축. 거래처 ESG 평가 통과 → 신규 계약 2건(연 5억 원). 정부 친환경 보조금 1,200만 원 확보.
