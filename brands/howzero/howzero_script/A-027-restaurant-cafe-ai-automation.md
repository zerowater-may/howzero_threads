### [1:30~3:30] 문제 정의

"외식업 사장님의 마감 후 루틴:

1. **일일 정산** — POS 데이터 확인, 매출 기록 (30분)
2. **재료 재고 확인** — 뭐가 부족한지 체크 (20분)
3. **발주** — 부족한 재료 업체에 주문 (15분)
4. **SNS 업데이트** — 오늘의 메뉴/후기 인스타 포스팅 (30분)
5. **리뷰 확인+답변** — 배달앱/네이버 리뷰 (20분)

합계: 약 2시간.
주 6일 영업이면 주 12시간. 월 48시간."

### [3:30~7:00] 자동화 설계

"**시스템 1: 자동 정산+리포트**
- POS 매출 데이터 → 일간 매출 요약 자동
- 전일 대비, 전주 대비 변화 자동 계산
- 월간 매출 추이 자동 그래프

**시스템 2: 재고+발주 자동화**
- 매일 사용량 기반 재고 자동 계산
- 발주 필요 품목 자동 알림
- 발주 메시지 자동 생성 (업체별)

**시스템 3: SNS+리뷰 자동화**
- 오늘의 메뉴/추천 메뉴 인스타 캡션 자동
- 배달앱 리뷰 자동 수집+감성 분석
- 리뷰 답변 초안 자동 생성"
