## 대본

### 훅 (0:00–0:40)
서핑, 다이빙, 패러글라이딩, 농장 체험. 레저·액티비티 사업의 가장 큰 변수가 뭡니까. 날씨입니다. 비 오면 취소, 바람 세면 취소. 예약 20건 중 5건이 당일 취소. 거기에 예약 관리, 강사 배정, 안전 서류, 리뷰 관리까지. AI로 날씨 연동 자동 안내부터 예약·인력 관리까지 자동화한 사례. 코딩 제로입니다.

### 문제 정의 (0:40–3:30)
레저·액티비티의 3대 병목. 첫째, 예약·취소 관리 — 날씨 의존도 높은데 취소 정책·환불 처리 수작업. 당일 취소로 빈 슬롯 재판매 못 함. 둘째, 강사·장비 배정 — 강사 스케줄, 자격, 선호와 고객 인원·레벨 매칭 수작업. 셋째, 안전·서류 — 서약서, 보험 안내, 안전 교육 체크리스트 수작업. 누락 시 법적 리스크.

### 자동화 설계 (3:30–9:30)
**날씨 연동 예약 관리**: 날씨 API 연동 → 예약일 D-2 날씨 예보 자동 확인 → 진행 불가 조건(풍속, 강수량 기준) 시 자동 안내("예약하신 서핑 체험 — 내일 풍속 10m/s 예상으로 진행 어려울 수 있습니다. 일정 변경 또는 취소를 선택해주세요") → 취소 시 빈 슬롯 대기 고객에게 자동 안내 → 날씨 좋은 날 자동 프로모션("내일 맑음! 당일 예약 20% 할인").

**강사·장비 자동 배정**: 강사별 자격(서핑 초급/중급, 다이빙 오픈워터/어드밴스드), 스케줄, 선호 DB → 예약 접수 시 고객 레벨에 맞는 강사 자동 배정 → 장비 사이즈·수량 자동 체크 → 배정 결과 강사·고객 양쪽에 자동 안내.

**안전 서류 자동화**: 예약 확정 시 전자 서약서 자동 발송 → 고객 모바일 서명 → 보험 안내 자동 → 안전 교육 체크리스트 강사 앱에 자동 생성 → 체크 완료 시 체험 시작 가능 → 전 과정 자동 기록(법적 증빙).

**리뷰·재방문 관리**: 체험 후 만족도 자동 설문 + 사진·영상 자동 전송 → 리뷰 작성 유도(다음 예약 할인) → 시즌별 재방문 프로모션 자동 발송.

### 데모/사례 (9:30–13:30)
제주 서핑숍 W. 강사 5명, 일 최대 예약 30명. 자동화 전: 날씨 취소 대응 수작업(당일 빈 슬롯 재판매 0%), 강사 배정 카톡으로, 서약서 현장 종이, 리뷰 수집 안 함. 자동화 후: 날씨 사전 안내로 당일 취소 40%↓. 빈 슬롯 자동 재판매로 가동률 15%↑. 강사 배정 자동화로 운영 시간 50%↓. 전자 서약서로 현장 대기 시간 10분↓. 리뷰 수집률 5%→35%.

도구: 날씨 API + 구글 시트 + Make + GPT-4 + 카카오 알림톡. 월 운영비 약 8만 원.
