## 대본

### 훅 (0:00–0:40)
비즈니스 모델 캔버스, 한 번쯤 그려보셨죠. 9칸 채우다가 3칸째에서 멈추거나, 채워도 "이게 맞나" 확신이 안 서서 서랍에 넣어둔 경험. AI에게 우리 회사 정보를 넣으면 9칸 전부 채워주고, 경쟁사 대비 약한 칸까지 짚어주는 시스템. 10분이면 끝납니다.

### 문제 정의 (0:40–3:00)
BMC의 현실적 문제. 첫째, 작성 난이도 — 가치 제안, 고객 세그먼트까지는 쓰겠는데 핵심 파트너, 비용 구조는 막연. 둘째, 주관성 — 대표 혼자 쓰면 자기 확신 편향. "우리 강점은 이거지"가 고객 시각과 다름. 셋째, 갱신 안 됨 — 한 번 쓰고 1년째 방치. 시장이 바뀌어도 BMC는 그대로.

### 자동화 설계 (3:00–8:30)
**BMC 자동 생성**: 회사 기본 정보(업종, 주요 제품, 매출 구조, 직원 수) 입력 → AI가 9개 블록 초안 자동 생성 → 각 블록별 근거와 참고 사례 첨부 → 구글 시트 BMC 템플릿에 자동 매핑.

**경쟁사 벤치마크**: 경쟁사 2~3곳 이름 입력 → AI가 공개 정보 기반으로 경쟁사 BMC 추정 생성 → 우리 BMC와 9개 블록별 비교 분석 → 강점·약점 자동 하이라이트("가치 제안은 유사하나 채널 다변화에서 경쟁사 우위").

**분기별 자동 리뷰**: 분기 1회 자동 트리거 → 최근 3개월 매출 데이터·고객 피드백 반영 → BMC 변경 제안 자동 생성("고객 세그먼트에 B2B 추가 권장 — B2B 문의 전 분기 대비 40%↑") → 대표에게 리뷰 요청 알림.

**투자자/파트너용 자동 변환**: BMC 데이터를 기반으로 1장짜리 사업 요약서 자동 생성 → 투자 피칭용 슬라이드 초안 자동 생성.

### 데모/사례 (8:30–13:00)
D2C 브랜드 M. 연 매출 8억, 직원 12명. 대표가 BMC를 2년 전에 그려놓고 방치. AI로 재분석한 결과: 고객 세그먼트가 2년 전과 완전히 달라짐(20대 여성→30대 남녀). 채널 블록에서 인스타그램 편중 발견 → 네이버 스마트스토어·쿠팡 채널 추가 권장. 경쟁사 대비 핵심 파트너 블록 취약 → 물류 파트너십 강화 제안. 실행 후 6개월 매출 22%↑.
