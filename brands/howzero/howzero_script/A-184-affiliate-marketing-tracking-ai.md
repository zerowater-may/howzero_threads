## 대본

### 훅 (0:00–0:40)
제휴 마케팅, 인플루언서 마케팅 하시는 분들. 파트너가 10명 넘어가면 누가 매출을 만드는지 파악이 안 됩니다. 클릭은 많은데 전환이 없는 파트너, 소수지만 고객 단가가 높은 파트너. 이 구분 없이 동일 수수료를 주고 있다면 돈이 새고 있는 겁니다. AI로 파트너별 실시간 성과 추적, ROI 분석, 수수료 최적화까지. 코딩 제로입니다.

### 문제 정의 (0:40–3:30)
제휴 마케팅의 3대 병목. 첫째, 성과 추적 분산 — 파트너마다 다른 채널(블로그, 인스타, 유튜브, 카카오). UTM 관리 엉망. 어디서 매출이 왔는지 모름. 둘째, ROI 불투명 — 파트너 수수료 총액은 아는데, 파트너별 수수료 대비 매출이 얼마인지 모름. 셋째, 파트너 관리 — 성과 리포트 수작업 발송, 수수료 정산 수작업, 비활성 파트너 방치.

### 자동화 설계 (3:30–9:30)
**UTM 자동 관리**: 파트너별 고유 UTM 링크 자동 생성 → 링크 관리 대시보드 → 클릭·전환·매출 실시간 추적 → 채널별(블로그/인스타/유튜브) 성과 분리 추적.

**파트너별 ROI 자동 분석**: 파트너별 매출 기여 자동 집계 → 수수료 대비 ROI 자동 산출 → 파트너 등급 자동 분류(A등급: ROI 300%↑, B등급: 100~300%, C등급: 100%↓) → 등급별 관리 전략 자동 제안.

**수수료 최적화**: A등급 파트너 → 수수료 인상+독점 프로모션 제공 자동 제안 → C등급 파트너 → 활성화 방안 or 계약 종료 자동 제안 → 전체 수수료 구조 시뮬레이션("A등급 수수료 5%↑, C등급 제외 시 전체 ROI 35%↑").

**자동 리포트·정산**: 파트너별 월간 성과 리포트 자동 생성·자동 발송 → 수수료 자동 정산(매출 확정 후 자동 계산) → 정산 내역 파트너에게 자동 전송 → 지급 스케줄 자동 관리.

### 데모/사례 (9:30–13:30)
건강식품 D2C V. 제휴 파트너 25명. 자동화 전: UTM 관리 수작업(누락 40%), 파트너별 ROI 모름, 수수료 동일 적용(8%), 리포트·정산 월 3일 소요. 자동화 후: UTM 자동 추적으로 전 파트너 성과 실시간 파악. ROI 분석 결과 → 상위 5명이 매출의 70% 기여 발견. 수수료 차등 적용(A등급 10%, C등급 계약 종료) → 전체 제휴 매출 동일, 수수료 비용 25%↓. 리포트·정산 3일→자동(검수 30분만).

도구: 구글 시트 + Make + GPT-4 + UTM 빌더 + GA4 연동. 월 운영비 약 7만 원.
