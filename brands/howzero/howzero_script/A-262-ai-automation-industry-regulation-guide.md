## 대본

### 훅 (0:00–0:40)
AI 자동화 열심히 도입했는데, 갑자기 규제 위반 통보가 옵니다. 과징금. 이런 일이 실제로 일어나고 있습니다. 의료 데이터 무단 처리, 금융 고객 정보 동의 없는 활용, 교육 분야 학생 데이터 해외 전송. 산업마다 규제가 다르고, AI 자동화에 적용되는 규제는 더 복잡합니다. 오늘 산업별로 정확히 무엇을 주의해야 하는지 완전 정리합니다.

### 공통 규제 (0:40–4:00)
모든 산업에 적용되는 기본 규제.
**개인정보보호법**: AI가 개인정보를 처리하면 수집·이용 동의 필수. 목적 외 이용 금지. 보유 기간 제한. 자동화된 의사결정에 대한 설명 요구권(2025년 시행). 위반 시 매출의 3% 과징금.

**AI 기본법(2026년 시행 예정)**: 고위험 AI 분류 기준 → 안전성 평가 의무 → 투명성 보고 → 인간 개입 보장. 자동화된 의사결정의 설명 가능성 요구.

**전자상거래법**: AI 챗봇이 고객 응대 시 "AI 응답임" 고지 의무. 자동 결제·구독 관련 소비자 보호 규정.

핵심: "모르고 위반"해도 과징금은 동일합니다. 몰랐다는 건 변명이 안 됩니다.

### 의료/헬스케어 (4:00–7:00)
**적용 규제**: 의료법 + 개인정보보호법 민감정보 조항 + 의료기기법.
**주의 포인트**: 건강 데이터는 민감정보 → 별도 동의 필수 + 암호화 저장 의무. AI 진단 보조 도구는 의료기기 인허가 대상 가능성. 환자 데이터 해외 서버 저장 제한.
**자동화 시 체크리스트**: 건강 데이터 별도 동의 확보 자동화 → 데이터 암호화 자동 적용 → 국내 서버 저장 확인 → AI 판단에 의료진 최종 확인 프로세스 반드시 포함.

### 금융/핀테크 (7:00–10:00)
**적용 규제**: 전자금융거래법 + 신용정보법 + 금융소비자보호법.
**주의 포인트**: 신용정보 처리 시 금융위 가이드라인 준수. 자동 심사/평가 시 설명 의무(왜 거절됐는지). AI 이상거래 탐지 시 오탐에 대한 소비자 이의제기 절차 필수. 데이터 제3자 제공 엄격 제한.
**자동화 시 체크리스트**: 자동 심사 거절 시 사유 자동 생성+통보 → 이의제기 채널 자동 안내 → 신용정보 보관 기간 자동 관리 → 이상거래 탐지 로그 5년 보관 자동화.

### 교육 (10:00–12:00)
**적용 규제**: 교육기본법 + 학생 개인정보 특별 보호 + 아동 관련 법규.
**주의 포인트**: 14세 미만 아동 데이터는 법정대리인 동의 필수. 학습 데이터의 프로파일링 제한. AI 평가 결과의 차별 금지(성별·지역·소득 편향 점검). 학습 데이터 해외 전송 제한.
**자동화 시 체크리스트**: 법정대리인 동의 수집 자동화 → AI 평가 편향 정기 점검 자동 → 학습 데이터 보관·삭제 자동화 → 졸업/퇴학 시 데이터 삭제 자동 트리거.

### 이커머스/유통 (12:00–14:00)
**적용 규제**: 전자상거래법 + 소비자보호법 + 개인정보보호법.
**주의 포인트**: AI 추천의 "광고성" 고지. 자동 가격 조정(다이나믹 프라이싱)의 소비자 기만 리스크. 자동 리뷰 필터링의 공정성. 구독 자동 결제 해지 용이성 보장.
**자동화 시 체크리스트**: AI 추천에 "맞춤 추천" 표시 자동 → 가격 변동 이력 투명 공개 → 구독 해지 원클릭 보장 → 리뷰 삭제 시 사유 자동 통보.

### 제조/물류 (14:00–15:30)
**적용 규제**: 산업안전보건법 + 제품안전법 + 환경 규제.
**주의 포인트**: AI 품질 검사의 최종 판단은 사람이 해야 합니다(안전 관련 제품). AI 예측 정비의 기록 보관 의무. 환경 데이터 자동 보고 시 정확성 검증 의무.
**자동화 시 체크리스트**: 품질 판정 AI + 사람 최종 확인 프로세스 → 정비 기록 자동 보관(법정 기간) → 환경 데이터 자동 보고 전 검증 단계 포함.

### 실전 대응 프레임워크 (15:30–18:00)
**1단계**: 우리 산업에 적용되는 규제 목록 작성 (AI가 자동 스캔 가능).
**2단계**: 현재 자동화 시나리오와 규제 요건 자동 대조 → 미준수 항목 식별.
**3단계**: 미준수 항목 우선순위 지정(과징금 리스크 순) → 자동 개선.
**4단계**: 법규 변경 모니터링 자동화 → 영향 받는 자동화 시나리오 자동 플래그.

비용: 규제 대응 자동화는 추가 비용이 거의 없습니다. 기존 자동화에 체크포인트를 추가하는 수준. 과징금 한 건이면 자동화 100년치 비용입니다.
