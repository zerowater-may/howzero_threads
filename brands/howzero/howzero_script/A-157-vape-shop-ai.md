### 훅 (0:00–0:40)

### 문제 정의 (0:40–3:00)
바이프샵의 3대 병목. 첫째, 재고 관리 — 액상 맛 종류만 100가지+, 니코틴 농도별, 디바이스 종류별. SKU가 많아서 재고 파악이 어렵고, 인기 없는 맛에 재고가 묶임. 둘째, 고객 취향 — 단맛, 시원한 맛, 담배맛 등 취향이 극명하게 다른데 체계적으로 기록 안 함. 셋째, 규제 대응 — 연령 인증, 광고 규제, 성분 표시 등 규정 준수 관리가 복잡.

### 자동화 설계 (3:00–8:00)
**고객 취향 DB**: 첫 구매 시 취향 설문(선호 맛 계열, 니코틴 농도, 디바이스 유형, 흡연 빈도) → 구매 이력과 연동 → 고객별 취향 프로필 자동 구축 → 재방문 시 즉시 조회.

**AI 맞춤 추천**: 신제품 입고 → AI가 취향 프로필 매칭 → 해당 고객에게 맞춤 안내("○○님이 좋아하시는 과일 계열 신상 액상이 나왔습니다") → 카카오 알림 → 온라인 주문 가능.

**재구매 자동화**: 액상 소비 주기 자동 추정(구매 빈도 기반) → 소진 예상일 3일 전 리마인드 → 정기 배송 전환 유도 → 디바이스 코일/팟 교체 주기 안내.

**재고 관리 자동화**: SKU별 판매 속도 자동 추적 → 인기 상품 자동 발주 → 체류 재고(90일 미판매) 자동 식별 → 할인 프로모션 제안 → 시즌별 수요 예측(여름: 시원한 맛↑).

**연령 인증 프로세스**: 온라인 주문 시 본인 인증 자동 연동 → 인증 완료만 주문 진행 → 인증 이력 자동 기록 → 규제 변경 시 자동 알림.

### 데모/사례 (8:00–13:30)
바이프샵 VP. 오프라인 1곳 + 온라인몰, SKU 200개, 월 고객 300명. 자동화 전: 취향 기록 없음(점원 기억 의존), 재구매 안내 없음, 재고 파악 주 1회(수동), 체류 재고 비율 25%. 자동화 후: 취향 DB로 맞춤 추천 → 전환율 2배. 재구매 리마인드로 재구매 주기 35일→28일. 재고 자동 관리로 체류 재고 25%→8%. 연령 인증 자동으로 규정 준수 100%.
