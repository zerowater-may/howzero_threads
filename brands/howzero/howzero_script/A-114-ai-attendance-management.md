## 대본

### 훅 (0:00–0:40)
직원 10명만 돼도 근태 관리가 골칩니다. 누가 지각했는지, 초과근무는 몇 시간인지, 연차는 며칠 남았는지… 엑셀로 관리하다가 급여 정산 때 맞지 않으면 직원과 마찰. AI로 출퇴근 기록부터 급여 계산까지 전자동으로 만든 시스템. 급여 정산 오차 제로, HR 담당자 업무 시간 80% 절감입니다.

### 문제 정의 (0:40–3:00)
근태 관리의 3대 병목. 첫째, 기록 — 출퇴근 시간을 수기나 엑셀로 관리. 대리 출석, 기록 누락 발생. 둘째, 계산 — 기본근무, 연장근무, 야간근무, 휴일근무 수당이 각각 다름. 수동 계산 시 오류 빈번. 셋째, 연차 관리 — 발생·사용·잔여 연차 추적. 노동법 기준 정확해야 함.

### 자동화 설계 (3:00–8:00)
**출퇴근 자동 기록**: 모바일 앱(GPS 기반) 또는 사무실 QR 스캔 → 출퇴근 시간 자동 기록 → 구글 시트 DB 자동 저장. 지각/조퇴/결근 자동 감지 → 관리자 알림.

**근무 시간 자동 계산**: 일별 근무 시간 자동 산출 → 주 52시간 초과 경고 → 연장·야간·휴일 수당 자동 계산(노동법 기준 적용).

**연차 자동 관리**: 입사일 기준 연차 자동 발생 → 사용 신청(슬랙/카카오) → 자동 승인 워크플로우 → 잔여 연차 실시간 확인 → 소멸 전 알림.

**급여 정산 자동화**: 월말 자동으로 — 기본급 + 수당 + 공제(4대 보험, 소득세) 자동 계산 → 급여 명세서 자동 생성 → 직원별 카카오 알림톡 발송.

### 데모/사례 (8:00–12:30)
중소기업 AE사. 직원 30명. 자동화 전: 출퇴근 엑셀 관리, HR 담당자 근태 정리 월 2일, 급여 정산 오차 월 3~5건(직원 불만), 연차 관리 수동. 자동화 후: 출퇴근 앱 자동 기록. 근태 정리 자동. 급여 정산 오차 제로. 연차 잔여 실시간 확인. HR 담당자 근태+급여 업무 월 4일 → 반나절.

도구: 출퇴근 웹앱(간단 구축) + 구글 시트 + Make + GPT-4 + 카카오 알림톡. 월 운영비 약 6만 원.
