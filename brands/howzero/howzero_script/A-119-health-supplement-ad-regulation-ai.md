## 대본

### 훅 (0:00–0:40)
건강기능식품 광고 잘못 하면 과징금 억 단위입니다. "이 제품 먹으면 살 빠져요" — 위반. "암 예방에 좋아요" — 위반. "의사가 추천했어요" — 위반. 어디까지가 되고 어디서부터 안 되는지 경계가 모호합니다. AI로 광고 카피, 상세 페이지, SNS 포스트를 자동 체크해서 위반 요소를 사전에 잡아내는 시스템. 과징금 리스크를 제로로 만듭니다.

### 문제 정의 (0:40–3:00)
건기식 광고 규제의 어려움. 첫째, 규정 복잡 — 식약처, 공정위, 건강기능식품법, 표시광고법 등 규정이 산재. 둘째, 경계 모호 — "피로 회복에 도움" vs "피로 치료" — 한 끗 차이로 위반. 셋째, 채널 다수 — 상세 페이지, 블로그, 인스타, 유튜브, 라이브커머스… 모든 채널의 카피를 검수해야 함. 넷째, 비용 — 법률 자문을 매 카피마다 받을 수 없음.

### 자동화 설계 (3:00–9:00)
**규제 DB 구축**: 건강기능식품법, 식약처 가이드라인, 과거 적발 사례, 허용 표현/금지 표현 DB를 AI가 학습.

**실시간 체크**: 광고 카피 입력 → AI가 문장별로 분석 → 위반 위험 표현 하이라이트 + 위반 근거 조항 표시 + 대체 표현 자동 제안.

예시: "다이어트에 효과적" → 위반 위험(질병 치료 암시) → 대체: "체지방 감소에 도움을 줄 수 있음(개인차 있음)"

**멀티 채널 일괄 검수**: 상세 페이지, SNS 카피, 라이브 스크립트 등 전 채널 카피를 한 번에 업로드 → 일괄 검수 → 채널별 수정본 자동 생성.

**변경 모니터링**: 규정 변경 시 자동 알림 + 기존 카피 중 영향받는 항목 자동 검출 → 수정 필요 카피 리스트 자동 생성.
