## 대본

### 훅 (0:00–0:40)
스포츠 용품은 시즌이 생명입니다. 여름에 수영용품 못 팔면 1년 재고. 겨울 스키 장비 발주 늦으면 품절. 감으로 발주하는 시대는 끝났습니다. AI로 시즌 수요를 예측하고, 마케팅까지 자동화한 브랜드. 재고 적체 40% 줄이고, 시즌 매출 55% 올렸습니다.

### 문제 정의 (0:40–3:00)
스포츠 용품 브랜드의 3대 병목. 첫째, 수요 예측 — 시즌, 날씨, 트렌드에 따라 급변. 감으로 발주하면 과잉 재고 또는 품절. 둘째, 시즌 마케팅 — 타이밍이 핵심. 너무 빨라도, 늦어도 효과 없음. 셋째, 사이즈/컬러 재고 관리 — SKU가 많아 관리 복잡(같은 신발 10사이즈 × 5컬러 = 50SKU).

### 자동화 설계 (3:00–8:00)
**수요 예측**: 과거 3년 판매 데이터 + 날씨 API + 포털 검색 트렌드 + SNS 언급량 → AI가 카테고리별·SKU별 주간 수요 예측 → 발주량 자동 추천.

**시즌 마케팅 자동화**: 수요 상승 신호 감지(검색량↑, 기온 변화 등) → 해당 카테고리 광고 자동 활성화 → 시즌 종료 신호 시 자동 종료 + 재고 정리 프로모션 트리거.

**사이즈/컬러 재고 최적화**: SKU별 판매 속도 실시간 추적 → 인기 사이즈/컬러 자동 리오더 알림 → 비인기 SKU 할인 자동 트리거.

**콘텐츠 자동화**: 신상품 등록 시 → 상세 페이지, SNS 포스트, 광고 카피 자동 생성 → 시즌별 톤 자동 조절(여름: 활동적, 겨울: 아웃도어).

### 데모/사례 (8:00–12:30)
스포츠 용품 온라인 브랜드 AF. SKU 400개, 시즌 4개. 자동화 전: 발주 감에 의존, 시즌 종료 후 재고 적체율 35%, 마케팅 타이밍 수동. 자동화 후: AI 수요 예측 정확도 82%(감에 의존 시 추정 55%). 재고 적체율 35%→21%. 시즌 마케팅 자동 트리거로 광고 효율 40%↑. 시즌 매출 55%↑.
