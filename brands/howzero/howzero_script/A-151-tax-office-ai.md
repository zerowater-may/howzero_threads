## 대본

### 훅 (0:00–0:40)
세무사 사무실에서 가장 시간을 많이 먹는 업무가 뭔지 아십니까. 세무 판단이 아닙니다. 자료 수집입니다. 고객에게 "매출 자료 보내주세요", "카드 내역 보내주세요", "통장 내역 보내주세요" — 이 독촉만으로 사무원 하루의 절반이 갑니다. AI로 자료 수집 자동화부터 절세 포인트 자동 감지까지, 세무사 사무실을 바꾼 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:30)
세무사 사무실의 3대 병목. 첫째, 자료 수집 — 고객사마다 자료 포맷이 다르고, 제출 시기가 늦고, 누락이 빈번. 사무원이 고객 50곳에 매월 전화·카톡으로 독촉. 둘째, 자료 분류·정리 — 수집된 영수증, 카드 내역, 통장 내역을 계정 과목별로 분류. 고객 1곳당 2~3시간. 셋째, 신고 시즌 과부하 — 부가세, 종합소득세, 법인세 신고 시즌에 업무량 3~5배 폭증. 야근이 일상.

### 자동화 설계 (3:30–9:30)
**자료 수집 자동화**: 매월 1일 고객에게 자동 안내("이번 달 세무 자료 제출 기한: 5일까지") → 자료 제출 포맷 가이드 포함 → 카카오/이메일로 자료 접수 → 미제출 고객 3일 전 자동 리마인드 → 제출 완료 시 자동 확인.

**AI 자료 분류**: 수집된 자료(카드 내역, 통장 내역, 영수증 사진) → AI OCR로 자동 인식 → GPT-4가 계정 과목 자동 분류(접대비, 차량 유지비, 복리후생비 등) → 분류 결과를 구글 시트에 자동 정리 → 사무원이 검수(10분).

**절세 포인트 자동 감지**: 고객별 데이터를 AI가 분석 → 놓치기 쉬운 공제 항목 자동 감지("창업 세액 감면 대상인데 적용 안 됨", "가사 사용분 안분 미적용") → 세무사에게 알림 → 절세 효과 추정 금액 포함.

**세무 달력 자동화**: 고객별 신고 일정 자동 관리 → D-30, D-7, D-3 자동 알림 → 필요 자료 체크리스트 자동 생성 → 신고 완료 후 고객에게 결과 안내 자동 발송.
