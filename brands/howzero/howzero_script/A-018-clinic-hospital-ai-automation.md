## [0:07~2:00] 병원의 3대 비효율

"병원/의원을 오딧하면 항상 같은 3가지가 나옵니다.

1. **예약 노쇼**: 월 평균 15~30건. 매출 손실 ₩75~150만.
2. **전화 문의**: '몇 시에 해요?' '주차 되나요?' 하루 20~30건 × 3분 = 1~1.5시간.
3. **리뷰 관리**: 네이버/카카오맵 리뷰 확인 + 답변 = 하루 20~30분. 안 하면 별점 하락.

총 반복 업무: 하루 2~3시간. 원장님이 진료에 집중해야 하는 시간을 빼앗깁니다."

## [2:00~10:00] 3가지 자동화 데모

"**자동화 1: 예약 노쇼 방지 (2:00~5:00)**
(화면: Cal.com + 자동 알림 설정)
온라인 예약 시스템 → 예약 확정 시 자동 문자 → D-1 리마인더 → 당일 2시간 전 리마인더.
3번 리마인더 = 노쇼 50% 감소.
추가: 노쇼 시 자동 대기자 문자 → 빈 슬롯 채우기.
세팅 15분. 비용 ₩0.

**자동화 2: 문의 자동 응답 (5:00~7:30)**
(화면: AI 챗봇 설정)
카카오톡 채널 + AI 챗봇.
'진료 시간이요?' → '평일 9~6시, 토 9~1시입니다.'
'주차 되나요?' → '건물 지하 주차장 2시간 무료입니다.'
'어떤 진료 보시나요?' → '피부과, 미용, 레이저... 예약 →'
문의의 80% 자동 응답. 나머지 20%만 직원이.
세팅 20분.

**자동화 3: 리뷰 자동 관리 (7:30~10:00)**
(화면: 리뷰 자동화 데모)
진료 완료 D+3 → 자동 리뷰 요청 문자 → '오늘 진료 만족하셨나요? 리뷰 남겨주시면 감사하겠습니다 → [링크]'
리뷰 올라오면 → AI 감성 분석 → 부정이면 즉시 알림 + 답변 초안.
긍정이면 → 감사 답글 자동 생성.
리뷰 수 3배 증가 + 부정 리뷰 즉시 대응."
