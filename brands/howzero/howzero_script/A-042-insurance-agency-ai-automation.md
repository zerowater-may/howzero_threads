### [1:30~3:30] 문제 정의

"보험설계사의 핵심 업무:

1. **갱신 관리** — 고객별 만기일 추적, 사전 연락
2. **제안서 작성** — 고객 상황에 맞는 상품 제안
3. **고객 생일/기념일** — 관계 유지 연락
4. **신규 리드 관리** — 문의 → 상담 → 계약 프로세스
5. **클레임 지원** — 보험금 청구 안내 + 서류 준비

고객 200명 기준: 매일 2~3시간이 관리.
상담할 시간이 부족. 영업할 시간이 부족."

---

### [3:30~7:00] 자동화 설계

"**시스템 1: 갱신 자동 알림**
- 고객 DB에서 만기 30일/14일/7일 전 자동 감지
- 개인화된 갱신 안내 메시지 자동 생성
- 갱신 미응답 시 팔로업 자동

**시스템 2: 제안서 자동 생성**
- 고객 정보(나이, 직업, 가족 구성, 기존 보험) 입력
- AI가 최적 상품 조합 추천
- 비교표 포함 제안서 자동 생성

**시스템 3: 관계 관리 자동화**
- 생일/결혼기념일/가입 기념일 자동 축하 메시지
- 분기별 보장 점검 안내 자동
- 계절별 건강 정보 + 보험 팁 자동 발송

**시스템 4: 리드 파이프라인**
- 신규 문의 → 자동 분류(HOT/WARM/COLD)
- 상담 예약 자동 제안
- 미계약 리드 → 정기 팔로업 자동"

---

### [7:00~13:00] 라이브 데모

"[갱신 알림]
'이 고객 200명 데이터에서 30일 이내 갱신 대상을 찾아줘.
각 고객에게 보낼 개인화된 갱신 안내 메시지를 만들어줘.
기존 보험 내용과 갱신 시 달라지는 점을 포함.'

(실행 — 12명 감지 + 12개 개인화 메시지)

수동이면 12명 × 15분 = 3시간. AI는 30초.

[제안서 자동 생성]
'고객 정보:
김민수, 34세, 자영업, 배우자+자녀1명
기존: 종합보험(삼성), 자동차보험(현대해상)
미보장: 암보험, 연금

이 고객에게 맞는 보험 제안서를 만들어줘.
현재 보장 분석 + 미보장 영역 + 추천 상품 + 예상 보험료 포함.'

(실행 — 전문적인 제안서)

제안서 1건 수동: 40분. AI: 2분.

[관계 관리]
'이번 주 생일인 고객 5명에게 개인화된 축하 메시지를 만들어줘.
각 고객의 보험 가입 내용을 반영해서 자연스럽게.'

[리드 관리]
'지난주 신규 문의 8건을 분석해서
HOT/WARM/COLD로 분류하고, 각각 팔로업 메시지 초안을 만들어줘.'

전부 합쳐서 5분."
