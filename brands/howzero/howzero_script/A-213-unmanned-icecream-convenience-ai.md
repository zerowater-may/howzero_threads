## 대본

### 훅 (0:00–0:40)
무인 매장 3개 운영하시면서 하루에 매장 몇 번 나가세요? CCTV 체크하고, 재고 확인하고, 청소 상태 보고, 기계 점검하고. 무인인데 사장님은 바빠요. 진짜 무인 매장으로 만든 사장님이 있습니다. AI가 매장 상태를 모니터링하고, 문제 있을 때만 알려줍니다. 매장 방문 주 7회 → 주 2회.

### 문제 정의 (0:40–3:00)
무인 매장의 병목. 첫째, 재고 관리 — 매장에 직접 가서 눈으로 확인. 품절 발생 시 매출 손실. 둘째, 매장 상태 관리 — 청소, 기기 작동, 온도 관리를 원격으로 확인 어려움. 셋째, 도난·이상 상황 — CCTV를 24시간 볼 수 없음. 결제 이상, 도난 감지 지연.

### 자동화 설계 (3:00–8:00)
**재고 자동 관리**: POS 판매 데이터 기반 실시간 재고 추적 → 품절 임박 상품 자동 알림 → 발주 리스트 자동 생성 → 매장별 배송 스케줄 자동 조율.

**매장 상태 모니터링**: IoT 센서(온도, 습도) + CCTV → 냉동고 온도 이상 시 즉시 알림 → 청소 필요 시 청소 업체 자동 호출 → 기기 이상 감지 시 A/S 자동 접수.

**매출·고객 분석 자동화**: 시간대별 매출 자동 분석 → 인기 상품/비인기 상품 자동 분류 → 상품 배치 추천 → 월간 매장별 비교 리포트.

**이상 감지 자동화**: 결제 없는 상품 반출 감지 → 관리자 즉시 알림 → CCTV 해당 시간 자동 캡처 → 비정상 결제 패턴 감지(연속 취소 등).

### 데모/사례 (8:00–12:30)
무인 아이스크림 매장 3개 운영 사장님 K. 자동화 전: 매장별 매일 방문(하루 3시간), 품절로 인한 매출 손실 월 40~60만 원, 냉동고 고장 발견 지연으로 재고 폐기 경험. 자동화 후: 매장 방문 주 2회(재고 입고 시), 품절 손실 월 5만 원 이하, 냉동고 이상 5분 내 감지.

3매장 합산 매출 월 12% 증가(품절 감소 + 최적 상품 배치). 매장 관리 시간 월 90시간 → 20시간.

도구: POS API + IoT 센서 + 구글 시트 + Make + GPT-4 + 카카오 알림톡. 월 운영비 약 9만 원.
