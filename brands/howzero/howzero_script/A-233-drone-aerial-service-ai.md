## 대본

### 훅 (0:00–0:40)
드론 촬영 업체 운영하시는 분, 견적 내고, 비행 허가 신청하고, 날씨 체크하고, 촬영하고, 편집하고 — 촬영보다 행정이 더 많죠? AI가 견적 산출, 비행 허가 서류 준비, 날씨 모니터링, 편집 초안까지 자동으로 해줍니다. 파일럿은 촬영에만 집중.

### 문제 정의 (0:40–3:00)
드론 서비스의 병목. 첫째, 견적·상담 — 촬영 장소, 시간, 용도에 따라 견적이 매번 다릅니다. 건당 상담 30분 이상. 둘째, 행정 — 비행 승인, 보험, 안전 계획서 등 서류 작업이 많습니다. 셋째, 후반 작업 — 촬영 원본에서 편집까지 시간이 오래 걸립니다.

### 자동화 설계 (3:00–8:00)
**견적 자동 산출**: 고객이 촬영 위치(주소) + 용도(부동산, 행사, 건설 현장 등) + 시간 입력 → AI가 거리, 비행 난이도, 소요 시간 자동 계산 → 자동 견적서 생성 + 포트폴리오 유사 작업 자동 첨부.

**비행 허가 자동 준비**: 촬영 위치 확정 → 비행 금지 구역 자동 체크 → 필요 시 비행 승인 신청서 자동 생성(장소, 일시, 기체 정보 자동 입력) → 제출 체크리스트.

**날씨 자동 모니터링**: 촬영일 D-7부터 → 해당 지역 날씨(풍속, 강수, 시정) 일일 자동 체크 → 촬영 불가 조건 시 고객+파일럿 자동 알림 → 대체 일정 자동 제안.

**편집 워크플로우 자동화**: 촬영 완료 → 원본 자동 분류(위치, 시간, 앵글) → AI가 하이라이트 컷 자동 선별 → 편집 초안 자동 생성 → 편집자 최종 작업 시간 60% 절감.

### 데모/사례 (8:00–12:30)
드론 촬영 업체 D. 파일럿 2명, 월 촬영 15건. 자동화 전: 견적 상담 건당 40분, 비행 허가 서류 건당 1시간, 날씨 수동 체크, 편집 건당 6시간. 자동화 후: 견적 자동(고객 확인만), 비행 허가 서류 자동 생성(확인만), 날씨 자동 모니터링, 편집 건당 2시간.

월 처리 건수 15건 → 25건(행정 시간 절감 → 촬영 가능 일수 증가). 견적 응답 시간 평균 4시간 → 15분.

도구: 구글 시트 + Make + GPT-4 + 기상청 API + 카카오 알림톡. 월 운영비 약 6만 원.
