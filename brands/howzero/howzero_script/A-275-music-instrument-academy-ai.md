## 대본

### 훅 (0:00–0:40)
음악 학원 운영하시는 분, 체험 레슨 후 등록 전환율 얼마세요? 평균 25~30%. 10명이 체험하면 7명이 안 옵니다. 이유는 단순합니다. 체험 후 팔로업을 안 하니까. 바빠서 전화 못 하고, 문자 보내기도 까먹고. AI가 체험 후 자동 팔로업, 학생별 연습 관리, 악기 구매 추천, 레슨 스케줄까지 자동으로 관리합니다. 등록 전환율 2.2배.

### 문제 정의 (0:40–3:00)
음악 학원/악기 판매의 병목. 첫째, 상담→등록 전환 — 전화/방문 상담 후 팔로업이 안 됩니다. "생각해볼게요" 하면 그대로 끝. 둘째, 레슨 스케줄 — 강사별·학생별·악기별 시간표 관리가 복잡합니다. 보강, 대체 수업까지 고려하면 수동으로 한계. 셋째, 학생 이탈 — 초보자가 3개월 내 이탈하는 비율 50%. 연습이 안 되니 실력이 안 늘고, 재미가 없어 그만둡니다. 넷째, 악기 판매 연결 — 레슨 학생에게 악기·액세서리 판매 기회를 놓칩니다.

### 자동화 설계 (3:00–8:00)
**상담→등록 자동 전환**: 상담/체험 레슨 후 당일 감사 메시지 자동 발송 → 3일 후 "체험 어떠셨나요? 궁금한 점 있으시면 편하게 물어보세요" → 7일 후 등록 미진행 시 한정 혜택 안내 → 14일 후 마지막 리마인더 → 각 단계 반응에 따라 다음 시나리오 자동 분기.

**레슨 스케줄 자동 관리**: 강사 가용 시간 + 학생 선호 시간 + 교실/악기 가용 현황 → AI가 최적 스케줄 자동 생성 → 결석 시 보강 자동 제안(강사·학생 양쪽 일정 매칭) → 스케줄 변경 시 관련자 자동 알림.

**학생 연습+이탈 방지**: 주간 연습 목표 자동 설정("이번 주 목표: 코드 전환 연습 하루 15분") → 연습 리마인더 자동 발송 → 연습 기록 제출 시 AI 피드백 → 2주 연속 미접속/미연습 시 강사에게 자동 알림 → 격려 메시지 자동 발송.

**악기·액세서리 자동 추천**: 레슨 진행 단계 기반 추천("3개월 차에는 메트로놈이 필요해요") → 사용 중인 악기 업그레이드 시점 안내("초보용 기타로 6개월 하셨으니 중급 기타 추천드려요") → 학원 내 판매 또는 제휴 쇼핑몰 연결.
