## 대본

### 훅 (0:00–0:40)
중고 명품 플랫폼 운영하시는 분, 감정사 한 명이 하루에 몇 건 처리하세요? 사진 확인, 시리얼 넘버 대조, 상태 등급 매기기, 시세 확인해서 가격 책정. 건당 30분이면 하루 15건이 한계입니다. AI가 사진만으로 1차 감정하고, 시세 자동 분석하고, 가격을 자동 책정하면? 감정 시간 80% 단축, 건당 6분. 같은 인력으로 5배 처리합니다.

### 문제 정의 (0:40–3:00)
리셀 플랫폼의 병목. 첫째, 감정 — 전문 감정사가 필요하고, 처리 속도가 곧 입고 속도를 제한합니다. 둘째, 가격 책정 — 시세가 매일 변동하고, 상태·희소성·시즌에 따라 달라집니다. 경험에 의존합니다. 셋째, 위조품 리스크 — 위조품이 통과하면 플랫폼 신뢰가 무너집니다. 사람도 틀립니다. 넷째, 판매 속도 — 가격이 높으면 안 팔리고, 낮으면 위탁자가 불만. 최적가를 잡기 어렵습니다.

### 자동화 설계 (3:00–8:30)
**AI 1차 감정**: 판매자가 사진 5~8장 업로드 → AI가 브랜드·모델·컬러 자동 식별 → 시리얼 넘버 OCR 자동 읽기 → 위조 의심 포인트 자동 플래그(스티칭 패턴·로고 위치·소재 질감) → 1차 통과/반려/정밀 감정 필요 자동 분류.

**상태 자동 등급화**: 사진 AI 분석 → 스크래치·변색·사용감 자동 감지 → S/A/B/C 등급 자동 부여 → 감정사는 AI 판정 결과 검토+최종 확인만.

**시세 자동 분석+가격 책정**: 국내외 리셀 플랫폼(크림·번개장터·베스티에르 등) 실시간 시세 크롤링 → 동일 모델·상태별 거래가 분포 AI 분석 → 최적 판매가 자동 산출(빠른 판매 vs 높은 마진 옵션) → 위탁자에게 자동 제안.

**판매 최적화**: 리스팅 후 조회·관심·문의 데이터 실시간 추적 → 반응이 낮으면 가격 자동 조정 제안 → 시세 변동 시 가격 자동 업데이트 알림 → 시즌별 인기 브랜드/모델 트렌드 리포트 자동.
