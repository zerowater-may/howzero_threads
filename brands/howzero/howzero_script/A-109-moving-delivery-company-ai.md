### 훅 (0:00–0:40)
이사 업체 사장님, 견적 문의 하나 처리하는 데 얼마나 걸려요? 고객이 전화로 "원룸이요, 짐 좀 있어요" 하면 뭘 얼마나 갖고 있는지 한참 물어봐야 합니다. 그리고 견적 내면 3곳 비교하고 연락 안 옵니다. AI로 견적을 자동으로 내고, 배차도 자동으로 잡고, 완료 후 리뷰까지 자동 수집. 견적 전환율 25%에서 48%로 올린 사례입니다.

### 문제 정의 (0:40–3:00)
이사/용달 업체의 3대 병목. 첫째, 견적 — 고객마다 짐의 양이 달라 표준화 어려움. 전화로 30분씩 상담. 둘째, 배차 — 차량 크기, 기사 스케줄, 이동 거리 고려한 최적 배차. 수동. 셋째, 리뷰 — 이사 후 리뷰 요청을 깜빡. 부정 경험 고객은 알아서 리뷰를 씀.

### 자동화 설계 (3:00–8:00)
**견적 자동화**: 카카오 챗봇 또는 웹 폼으로 — 이사 유형(원룸/투룸/가정), 짐 사진 업로드, 출발지/도착지, 희망일 입력 → AI가 사진 분석으로 짐 양 추정 + 거리/시간 계산 + 자동 견적 산출 → 고객에게 즉시 견적서 발송.

**배차 자동화**: 확정된 이사 건 → AI가 차량 크기 매칭 + 기사 스케줄 확인 + 경로 최적화 → 자동 배차 → 기사에게 자동 안내.

**고객 커뮤니케이션**: 이사 전날 자동 안내(시간, 준비사항) → 당일 기사 출발 알림 → 이사 완료 확인 → 2시간 후 만족도 조사 + 리뷰 요청.

**가격 최적화**: 시즌/요일/시간대별 수요 데이터 분석 → 동적 가격 제안(비수기 할인, 성수기 프리미엄).
