### 훅 (0:00–0:40)

### 문제 정의 (0:40–3:00)
건강식품 쇼핑몰의 3대 병목. 첫째, 상품 추천 — 고객 건강 상태, 나이, 성별에 따라 맞는 제품이 다른데 일괄 추천. 둘째, 복용 관리 — 건강식품은 꾸준히 먹어야 효과. 고객이 까먹으면 재구매 안 함. 셋째, CS — "이 제품 어디에 좋아요?", "같이 먹어도 되나요?" 같은 질문이 반복.

### 자동화 설계 (3:00–8:30)
**맞춤 추천**: 첫 구매 시 간단 설문(나이, 성별, 건강 고민, 복용 약물) → AI가 맞춤 제품 추천 + 조합 제안(예: 비타민D + 마그네슘). 재방문 시 구매 이력 기반 자동 추천.

**복용 리마인드**: 구매 제품 기반 복용 스케줄 자동 생성 → 매일 아침 카카오톡 리마인드 → 복용 기록 간편 입력(O/X) → 복용률 대시보드.

**재구매 자동화**: 제품별 소진 예상일 자동 계산(30일분 제품 → 25일차에 재구매 안내) → 할인 쿠폰 자동 발급 → 원클릭 재주문 링크.

**CS 자동화**: 제품별 FAQ DB 구축 → AI 챗봇이 성분, 효능, 복용법, 주의사항, 제품 간 상호작용 자동 응답.
