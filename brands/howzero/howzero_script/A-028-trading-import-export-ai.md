### [1:30~4:00] 문제 정의

"무역회사 일상:

1. **이메일 처리** — 영어/중국어 바이어 메일 50~100통/일
2. **서류 작성** — 인보이스, P/L, 원산지 증명 등
3. **환율 모니터링** — 매시간 체크, 결제 타이밍 판단
4. **선적 추적** — 컨테이너별 현재 위치, ETA 확인
5. **견적 대응** — 바이어 문의에 견적 + 조건 회신

이 5가지에 하루 6~8시간 쓰고 있습니다.
새 바이어 발굴할 시간이 없죠."

---

### [4:00~8:00] 자동화 설계

"**시스템 1: 이메일 자동 분류+번역+답변 초안**
- 영어/중국어 메일 → 자동 한국어 번역
- 긴급/일반/참조 자동 분류
- 유형별 답변 초안 (영어/중국어) 자동 생성

**시스템 2: 서류 자동 생성**
- 주문 데이터 입력 → Commercial Invoice 자동
- 동일 데이터 → Packing List 자동
- 기존 서류에서 정보 추출 → 새 서류에 자동 매핑

**시스템 3: 환율+선적 모니터링**
- 주요 통화 환율 자동 추적
- 목표 환율 도달 시 알림
- 선적 번호 입력 → ETA 자동 추적 + 지연 알림

**시스템 4: 견적 자동 대응**
- 상품 DB 기반 자동 견적서 생성
- MOQ, 리드타임, 결제조건 자동 포함
- 바이어 국가별 언어 자동 전환"

### [8:00~14:00] 라이브 데모

"[이메일 처리]
'이 영어 이메일 10개를 처리해줘:
1. 한국어로 번역
2. 긴급/일반/참조 분류
3. 긴급 건에 대한 영문 답변 초안'

(실행 — 10개 메일 분류+번역+답변 초안 → 15초)

수동으로 하면 1시간 30분. AI는 15초.

[서류 자동 생성]
'다음 주문 정보로 Commercial Invoice를 만들어줘:
Buyer: ABC Trading Co., Ltd.
Items: Widget A x 1000pcs @$2.50, Widget B x 500pcs @$3.80
Terms: FOB Busan, T/T 30days
Shipping: Container 20ft'

(실행 — 표준 형식 인보이스)

'같은 정보로 Packing List도 만들어줘.
Net Weight, Gross Weight, Carton 수 포함.'

2분 만에 인보이스 + 패킹리스트 완성.
수동이면 30분.

[견적 자동 대응]
'이 바이어의 문의에 견적서를 만들어줘:
문의: We need 2000pcs of Widget A. What is your best price?
우리 조건: MOQ 500, 단가 $2.50 (2000pcs 이상 $2.30), FOB Busan, T/T 30days
영어로 작성.'

(실행 — 전문적인 견적 이메일)

견적 대응 시간: 20분 → 2분."
