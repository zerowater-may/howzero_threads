**리밸런싱 신호 자동 감지**: 목표 자산 배분(예: 주식 60, 채권 30, 현금 10) 대비 현재 비중 편차 자동 계산 → 편차 5% 초과 시 리밸런싱 알림 → AI가 매수/매도 종목 + 수량 자동 제안.

**시장 맥락 자동 분석**: 주요 경제 지표(금리, 환율, VIX) + 뉴스 요약 → AI가 "현재 시장 상황에서 리밸런싱 적절성" 의견 제시 → 단순 규칙이 아닌 맥락 기반 판단 보조.
