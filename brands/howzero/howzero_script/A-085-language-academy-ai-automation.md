### 문제 정의 (0:40–3:00)
어학원 행정 병목 세 가지. 첫째, 수강 관리 — 등록, 반 배정, 결석 관리, 보강 스케줄링. 둘째, 레벨 테스트 — 출제, 채점, 결과 분석, 반 배치 추천. 셋째, 학부모 커뮤니케이션 — 학습 리포트, 상담 일정, 공지. 매달 이 루틴에 원장+직원 합산 주 40시간 이상 소모.

### 자동화 설계 (3:00–8:30)
**수강 관리**: 온라인 등록 폼 → 자동 반 배정(레벨+시간대 매칭) → 수강료 자동 청구(카카오페이) → 결석 시 자동 보강 스케줄 제안.

**레벨 테스트**: AI 문제 은행에서 자동 출제(난이도 조절) → 온라인 응시 → 자동 채점 → GPT가 결과 분석 + 반 배치 추천 + 학습 커리큘럼 제안.

**학부모 리포트**: 월말 자동 생성 — 출석률, 시험 점수 추이, 강사 코멘트 종합 → 개인화된 리포트 PDF → 카카오 알림톡으로 자동 발송.

### 데모/사례 (8:30–13:30)
수도권 영어학원 E. 수강생 180명, 강사 5명, 행정직원 2명. 자동화 전: 레벨 테스트 채점 월 8시간, 학부모 리포트 월 15시간, 수강 관리 주 10시간. 자동화 후: 레벨 테스트 전 과정 자동(채점+분석+배치). 학부모 리포트 자동 생성+발송(월 15시간→30분). 결석 보강 자동 스케줄링.

도구: 구글 폼 + 구글 시트 + Make + GPT-4 + 카카오 알림톡. 월 운영비 약 7만 원.
