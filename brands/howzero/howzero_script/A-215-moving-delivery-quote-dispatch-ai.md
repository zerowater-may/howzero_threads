### 자동화 설계 (3:00–8:00)

**배차 자동 최적화**: 확정 예약 → 차량 크기 자동 배정(물량 기반) → 인력 자동 배정 → 이동 경로 최적화 → 당일 스케줄 자동 생성.

**고객 커뮤니케이션 자동화**: 견적 확정 → 준비사항 체크리스트 자동 발송 → D-3 리마인더 → 당일 기사 정보·도착 예정 시간 자동 안내 → 완료 후 만족도 조사 + 리뷰 요청.

**정산 자동화**: 작업 완료 → 기사별 작업 내역 자동 집계 → 일당·추가 수당 자동 계산 → 월말 정산 리포트.
