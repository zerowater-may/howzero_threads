## 대본

### 훅 (0:00–0:40)
"좋은 사람이 안 뽑혀요." 소규모 회사 대표님들한테 가장 많이 듣는 말입니다. 근데 채용 공고 보면 이유를 알겠어요. 2019년에 쓴 것 같은 문구, 복붙한 복리후생, 회사가 뭘 하는지도 안 보이는 채용 페이지. AI로 채용 브랜딩 전체를 바꾼 회사가 있습니다. 지원자 수 3배, 합격자 이탈률 60% 감소. 코딩 0줄.

### 문제 정의 (0:40–3:00)
채용 브랜딩의 병목. 첫째, 채용 공고 품질 — 매번 비슷한 문구 복붙. 회사 문화, 비전이 안 보입니다. 둘째, 채용 콘텐츠 부재 — 회사 소개 영상, 팀 인터뷰, 직무 소개 만들 여력이 없습니다. 셋째, 프로세스 비효율 — 이력서 스크리닝, 면접 일정 조율, 결과 통보 전부 수동.

### 자동화 설계 (3:00–8:30)
**채용 공고 자동 생성**: 직무 기술서(JD) 핵심 키워드 입력 → AI가 회사 톤앤매너에 맞는 채용 공고 자동 작성 → 채용 플랫폼별(사람인, 잡코리아, 원티드) 포맷 자동 변환.

**채용 콘텐츠 자동화**: 팀원 인터뷰 질문지 자동 생성 → 답변 기반 채용 블로그 포스트 자동 작성 → SNS용 짧은 콘텐츠 자동 변환.

**이력서 사전 스크리닝**: 지원자 이력서 수신 → AI가 JD 기준 적합도 점수화 → 상위 30% 자동 분류 → 면접 일정 자동 제안 발송.

**온보딩 자동화**: 합격 통보 → 입사 서류 안내 자동 발송 → 첫 주 가이드(사내 도구, 팀 소개, 미션) 자동 생성 → Day 1/7/30 체크인 알림.
