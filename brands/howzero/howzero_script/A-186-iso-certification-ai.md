## 대본

### 훅 (0:00–0:40)
ISO 인증, 받고 싶은데 문서 작업이 두려워서 미루고 있는 분들 많습니다. ISO 9001이든, 27001이든 핵심은 프로세스 문서화와 기록 관리입니다. 이걸 수작업으로 하면 3~6개월에 인력 2~3명이 매달립니다. AI로 문서 자동 생성부터 내부 감사, 사후 관리까지 자동화해서 ISO 인증 준비 기간을 절반으로 줄인 사례. 코딩 제로입니다.

### 문제 정의 (0:40–3:30)
ISO 인증 준비의 3대 병목. 첫째, 문서화 — 품질 매뉴얼, 절차서, 작업 지침서, 기록 양식 수십~수백 종. 처음부터 작성하면 3개월+. 둘째, 내부 감사 — 프로세스별 적합성 검증, 부적합 사항 기록, 시정 조치 추적. 감사원 양성·실행에 시간. 셋째, 사후 관리 — 인증 받고 방치. 1년 뒤 사후 심사에서 부적합 → 인증 취소 위험.

### 자동화 설계 (3:30–10:00)
**문서 자동 생성**: 회사 현재 프로세스 정보 입력(업종, 조직 구조, 주요 프로세스) → AI가 ISO 요구사항 기반 문서 초안 자동 생성(품질 방침, 절차서, 작업 지침서, 기록 양식) → 담당자 검수·수정 → 문서 번호·버전 자동 관리.

**갭 분석 자동화**: ISO 요구사항 체크리스트 vs 현재 프로세스 자동 비교 → 미충족 항목 자동 식별 → 항목별 개선 방안 AI 제안 → 개선 우선순위 자동 산출(중요도×난이도) → 개선 일정 자동 생성.

**내부 감사 자동화**: 감사 일정 자동 생성(프로세스별 주기 설정) → 감사 체크리스트 AI 자동 생성 → 감사 결과 기록 양식 자동 제공 → 부적합 사항 등록 시 시정 조치 기한 자동 설정 → 시정 완료 확인 자동 추적 → 감사 리포트 자동 생성.

**사후 관리 자동화**: 사후 심사 일정 자동 관리(D-90, D-30, D-7 알림) → 심사 준비 체크리스트 자동 생성 → 지난 1년 문서 변경·부적합 이력 자동 요약 → 경영 검토 보고서 초안 자동 생성.

### 데모/사례 (10:00–14:30)
