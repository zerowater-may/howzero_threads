### 문제 정의 (0:40–3:00)

### 자동화 설계 (3:00–7:30)
**예약·결제 자동화**: 온라인 예약 페이지(날짜, 시간, 클래스 종류) → 자동 결제 → 예약 확인 + 준비물 안내 자동 발송 → D-1 리마인더 → 노쇼 시 자동 환불 정책 적용.

**재료 발주 자동화**: 확정 수강생 수 → 클래스별 재료 리스트 자동 산출 → 꽃시장 발주서 자동 생성 → 여유분 자동 포함(10%) → 입고 확인 체크리스트.

**콘텐츠 자동화**: 수업 중/완성작 사진 촬영 → AI가 캡션 + 해시태그 자동 생성 → 인스타 피드·스토리·릴스 스크립트 자동 작성 → 예약 발행.

**수강생 관리 자동화**: 수업 후 → 꽃 관리 팁 자동 발송 → 다음 클래스 추천(수강 이력 기반) → 정기 수강생 할인 쿠폰 자동 발송 → 리뷰 요청.
