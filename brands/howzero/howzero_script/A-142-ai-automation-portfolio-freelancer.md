## 대본

### 훅 (0:00–0:40)
AI 자동화를 배웠습니다. Make도 쓸 줄 알고, GPT API도 연동할 줄 압니다. 그런데 클라이언트가 없습니다. 왜? 포트폴리오가 없으니까. "뭘 해봤는데요?"라는 질문에 "이것저것..."이라고 답하면 계약은 없습니다. 오늘 AI 자동화 프리랜서가 반드시 만들어야 할 포트폴리오 구성법과, 포트폴리오 없이 시작하는 방법까지 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
AI 자동화 프리랜서의 3대 허들. 첫째, 사례 부재 — 처음 시작하면 보여줄 게 없음. "경험이 없으니 일을 못 받고, 일을 못 받으니 경험이 없는" 순환. 둘째, 차별화 — "AI 자동화 해드립니다"만으로는 100명 중 1명. 어떤 자동화를, 어떤 업종에, 어떤 결과를 냈는지가 차별화. 셋째, 가격 설정 — 사례가 없으니 가격 근거도 없음. 저가 경쟁에 빠짐.

### 자동화 설계 (3:00–9:00)
**사례 없이 포트폴리오 만드는 법**: 첫째, 자기 자신의 자동화를 사례로 만듭니다 — 내 이메일 관리 자동화, 내 콘텐츠 일정 자동화, 내 고객 관리 자동화. 이것이 첫 번째 포트폴리오. 둘째, 지인 1~2명에게 무료 프로젝트 제안. "월 10만 원 수준의 자동화를 무료로 해드리겠습니다. 대신 사례로 쓰겠습니다." 셋째, 가상 프로젝트 — 특정 업종(예: 부동산 중개소)의 자동화를 설계하고 데모 영상 제작.

**포트폴리오 구성 공식**: 프로젝트당 1페이지. 구성: 고객 상황(Before) → 해결한 문제 → 구축한 시스템(도구 목록) → 결과(숫자) → 고객 후기. AI가 각 프로젝트의 정보를 입력하면 포트폴리오 페이지를 자동 생성합니다.

**포트폴리오 자동 업데이트**: 프로젝트 완료 시 결과 데이터 입력 → AI가 사례 페이지 자동 생성 → 노션/웹사이트에 자동 게시 → SNS 공유용 요약 자동 생성.

**클라이언트 확보 자동화**: 포트폴리오 링크를 활용한 아웃바운드 — 업종별 타겟 리스트 → 맞춤 제안서(해당 업종 사례 포함) AI 자동 생성 → 이메일 발송 → 팔로업 자동.

### 데모/사례 (9:00–13:30)
AI 자동화 프리랜서 PF. 시작 시 사례 0개. 1개월 차: 자기 자동화 3개 + 지인 무료 프로젝트 2개 = 포트폴리오 5개. 2개월 차: 포트폴리오 기반 아웃바운드 30건 → 미팅 5건 → 계약 2건(건당 150만 원). 6개월 차: 포트폴리오 12개, 월 계약 4~5건, 월 수익 600만 원+.

핵심: 무료 프로젝트의 ROI를 명확히 기록 → "이 자동화로 월 30시간 절감, 연 환산 1,800만 원 가치" → 다음 클라이언트에게 가격 근거로 활용.
