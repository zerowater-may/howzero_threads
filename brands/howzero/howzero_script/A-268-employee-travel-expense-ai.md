## 대본

### 문제 정의 (0:40–3:00)
경비 정산의 병목. 첫째, 영수증 관리 — 종이 영수증 모으기, 분실, 퇴색. 둘째, 수동 입력 — 날짜, 금액, 사용처, 항목을 사람이 타이핑합니다. 오타, 잘못된 항목 분류. 셋째, 정책 확인 — 식비 한도, 숙박비 기준, 교통비 규정을 매번 확인합니다. 초과 여부를 경리가 일일이 대조합니다. 넷째, 부정 청구 감지 — 중복 청구, 개인 경비 혼입, 영수증 위변조를 사람 눈으로 잡기 어렵습니다.

### 자동화 설계 (3:00–8:00)
**영수증 AI 자동 인식**: 영수증 사진 촬영 → OCR + AI가 자동 추출(날짜·금액·사용처·항목) → 항목 자동 분류(식비·교통·숙박·기타) → 정산서에 자동 입력 → 원본 이미지 자동 보관.

**정책 자동 검증**: 회사 경비 정책(식비 1인 1만5천원, 숙박 15만원 이내 등) 자동 적용 → 초과 시 즉시 알림("숙박비가 기준 15만원을 초과했습니다. 사유를 입력해주세요") → 정책 변경 시 자동 업데이트.

**자동 결재 워크플로**: 정산서 완성 → 금액 기준 결재 라인 자동 설정(10만원 이하 팀장, 이상 이사) → 결재자에게 자동 알림 → 승인/반려 원클릭 → 승인 시 급여 연동 자동 처리.

**부정 청구 AI 감지**: 중복 영수증 자동 감지(같은 금액·날짜·사용처) → 주말/휴일 경비 자동 플래그 → 개인별 경비 패턴 학습 → 이상치 자동 감지(평소 월 30만원인데 이번 달 150만원) → 경리 담당자에게 검토 요청.

### 데모/사례 (8:00–12:30)
