### 문제 정의 (0:40–3:00)

### 자동화 설계 (3:00–7:30)
**장비 모니터링 자동화**: 세탁기·건조기 IoT 연동 → 가동 횟수, 에러 코드 실시간 추적 → 이상 감지 시 관리자 즉시 알림 → A/S 업체 자동 호출.

**소모품 자동 관리**: 세제 디스펜서 잔량 센서 → 기준치 이하 시 보충 알림 → 주기적 발주 자동 생성.

**청소 관리 자동화**: 청소 업체 자동 스케줄링 → 청소 완료 사진 수신 → AI가 청결 상태 자동 체크 → 미흡 시 재청소 요청.

**매출·가동 분석**: 매장별·시간대별 가동률 자동 분석 → 비수기 할인 쿠폰 자동 발행 → 월간 매장별 비교 리포트.
