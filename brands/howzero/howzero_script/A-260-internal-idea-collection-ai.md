## 대본

### 훅 (0:00–0:40)
사내 아이디어 제안 제도 운영하고 계세요? 대부분 이런 패턴입니다. 처음엔 열심히 제안하다가, 3개월 후 아무도 안 합니다. 왜냐면 제안해도 아무 일도 안 일어나니까요. 검토가 느리고, 피드백이 없고, 실행은 더 없습니다. AI가 아이디어를 실시간 수집하고, 자동 분류·평가하고, 실행 가능성과 기대 효과를 자동 산출합니다. 제안에서 실행까지 평균 3개월 → 2주. 실행률 3%에서 24%로.

### 문제 정의 (0:40–3:00)
사내 아이디어 관리의 현실. 첫째, 수집 채널 부재 — 아이디어를 제안할 마땅한 채널이 없거나, 있어도 불편합니다. 둘째, 평가가 느림 — 위원회 심사, 부서장 검토 등 절차가 복잡합니다. 제안 후 3개월 지나 "검토 중" 답변. 셋째, 중복·유사 아이디어 — 같은 아이디어가 반복 제안되지만 알 수 없습니다. 넷째, 실행 연결 부재 — 좋은 아이디어가 선정되어도 담당자 배정, 자원 확보, 진행 추적이 안 됩니다.

### 자동화 설계 (3:00–8:00)
**다채널 아이디어 수집**: 슬랙 봇 + 웹폼 + 카카오톡 → 간편하게 아이디어 제출(제목+설명+기대 효과 3줄이면 충분) → AI가 자동 구조화(카테고리·키워드·관련 부서 태깅).

**중복 감지+유사 아이디어 연결**: 신규 아이디어 제출 시 과거 아이디어 DB와 AI 유사도 검색 → 유사 아이디어 존재 시 자동 알림("비슷한 아이디어가 2건 있어요. 합치시겠어요?") → 투표/공감 기능으로 인기도 자동 집계.

**AI 자동 평가**: 아이디어별 실행 가능성(기술 난이도·필요 자원·기간) + 기대 효과(매출 영향·비용 절감·고객 만족) AI 자동 스코어링 → 난이도-가치 매트릭스 자동 배치 → 낮은 과일(Low-hanging fruit) 자동 하이라이트.

**실행 파이프라인**: 평가 통과 아이디어 → 담당 부서 자동 배정 → 실행 계획 템플릿 자동 제공 → 진행률 주간 자동 추적 → 완료 시 제안자에게 결과 자동 피드백 + 보상 트리거.
