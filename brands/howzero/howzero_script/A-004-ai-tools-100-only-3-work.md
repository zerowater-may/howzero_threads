"저는 코딩을 모릅니다. 그래서 '비개발자도 쓸 수 있다'고 광고하는 AI 도구는 전부 써봤습니다. 노코드 자동화, AI 챗봇 빌더, 앱 빌더, 프롬프트 도구, RPA... 정말 100개 넘게.

결론부터 말하면 — 대부분 '비개발자도 가능'이라고 해놓고, 실제로 해보면 개발자가 필요했습니다.

'노코드'라고 쓰면서 API 키 설정하라고 하고, JSON 구조를 이해해야 하고, 에러 나면 로그를 읽어야 하고.

진짜 비개발자가, 혼자서, 15분 안에 실전 자동화를 만들 수 있는 도구. 그게 3개였습니다."
