## 대본

### 훅 (0:00–0:40)
제품 출시일입니다. 그런데 상세페이지가 안 올라갔습니다. 아니, 법무팀 검토가 아직이었습니다. 아니, 물류 창고에 재고가 아직 안 들어갔습니다. 제품 출시에 관여하는 부서가 5개, 체크 항목이 50개 이상인데, 엑셀 하나로 관리하면 반드시 누락됩니다. AI로 제품 출시 체크리스트를 자동 생성하고, 부서별 할 일을 자동 배분하고, 실시간 추적하는 시스템을 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
제품 출시의 3대 병목. 첫째, 체크 항목 파편화 — 기획, 디자인, 마케팅, 물류, 법무, CS 등 부서마다 해야 할 일이 있는데 통합 관리가 안 됨. 둘째, 의존성 관리 — "상세페이지는 법무 검토 후에만 게시 가능" 같은 선후 관계. 수동으로 추적하면 병목 발생. 셋째, 커뮤니케이션 — "어디까지 됐어요?" 확인 전화·메시지가 하루 20통+.

### 자동화 설계 (3:00–8:30)
**AI 체크리스트 자동 생성**: 제품 유형(신제품/리뉴얼/한정판)과 판매 채널(자사몰/마켓플레이스/오프라인) 입력 → AI가 해당 유형에 맞는 체크리스트 자동 생성(50~80개 항목) → 부서별 자동 분류 → 의존성(선후 관계) 자동 설정.

**부서별 자동 배분**: 체크리스트 확정 → 각 부서 담당자에게 슬랙/카카오로 할 일 자동 전달 → 마감일 포함 → 완료 시 체크 → 다음 의존 항목 담당자에게 자동 알림("법무 검토 완료, 상세페이지 게시 가능합니다").

**실시간 진행 대시보드**: 전체 체크리스트 완료율 실시간 표시 → 부서별 진행률 시각화 → 마감 D-3 이내 미완료 항목 자동 경고 → 병목(오래 걸리는 항목) 자동 식별 → PM에게 일일 진행 리포트 자동 발송.

**출시 후 자동화**: 출시 당일 체크리스트(가격 확인, 결제 테스트, CS 매뉴얼 준비 등) → 출시 후 24시간 모니터링(주문 현황, CS 문의 현황, 재고 소진 속도) → 이상 시 즉시 알림.
