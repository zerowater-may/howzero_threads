## 대본

### 훅 (0:00–0:40)
"이 공급업체 괜찮은 데인데..." 감으로 공급업체를 평가하고 계시죠. 납기 지연이 몇 번이었는지, 불량률이 얼마인지, 경쟁사 대비 가격이 적정한지 — 데이터로 답할 수 있는 사장님이 거의 없습니다. AI로 공급업체를 데이터 기반으로 자동 평가하고, 최적의 업체를 선택하는 시스템을 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
공급업체 평가의 3대 병목. 첫째, 데이터 부재 — 납품 기록, 불량 이력, 가격 변동 이력이 체계적으로 관리되지 않음. 둘째, 주관적 평가 — "오래 거래해서", "사장님 아는 사이라" 등 관계 기반 평가. 객관적 비교 불가. 셋째, 평가 주기 — 공급업체 평가를 연 1회 하거나, 아예 안 하거나. 문제가 터져야 업체를 바꿈.

### 자동화 설계 (3:00–8:00)
**납품 데이터 자동 수집**: 발주→납품 전 과정을 구글 시트에 기록 → 납기 준수율 자동 계산 → 불량 발생 시 기록 → 불량률 자동 산출 → 가격 변동 이력 추적.

**AI 종합 평가**: 분기 1회 자동 평가 — 납기 준수율(30%), 품질(불량률 25%), 가격 경쟁력(20%), 대응 속도(15%), 유연성(10%) → 100점 만점 종합 점수 자동 산출 → 등급(A/B/C/D) 자동 부여.

**벤치마크 분석**: 동일 품목 공급업체 간 자동 비교 → 가격 대비 품질 매트릭스 시각화 → 대체 업체 필요 시 AI가 후보 리스트 제안.

**자동 리포트 & 알림**: 분기 공급업체 평가 리포트 자동 생성 → D등급 업체 즉시 경고 → 개선 요청 자동 발송 → 2분기 연속 D등급 시 대체 검토 알림.
