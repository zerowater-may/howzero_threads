## 대본

### 훅 (0:00–0:40)
고객 충성도 프로그램, 대부분 이렇게 운영하죠. 포인트 적립해주고, 가끔 쿠폰 뿌리고, 생일에 문자 보내고. 그런데 재구매율은 안 올라갑니다. 왜냐면 모든 고객에게 같은 혜택을 주니까요. AI가 고객별 구매 패턴을 분석해서, 이탈 직전 고객에게는 파격 혜택을, VIP에게는 프리미엄 경험을, 신규 고객에게는 두 번째 구매를 유도하는 맞춤 전략을 자동 실행합니다. 재구매율 2.4배. 데이터입니다.

### 문제 정의 (0:40–3:00)
충성도 프로그램의 현실. 첫째, 일률적 혜택 — 모든 고객에게 같은 쿠폰, 같은 적립률. 돈만 나가고 효과는 미미합니다. 둘째, 이탈 감지 불가 — 구매 주기가 길어지는 고객을 알아채지 못하고, 이미 떠난 후에야 깨닫습니다. 셋째, 수동 운영 — 등급 산정, 혜택 발송, 포인트 정산을 매번 수작업합니다. 넷째, 효과 측정 불가 — 프로그램이 실제로 매출에 기여하는지 감으로만 판단합니다.

### 자동화 설계 (3:00–8:30)
**고객 세그먼트 자동 분류**: 구매 이력 데이터 → AI가 RFM(최근성·빈도·금액) 자동 분석 → 5단계 자동 등급 산정 → 등급 변동 시 실시간 반영.

**이탈 예측+자동 개입**: 구매 주기 패턴 AI 학습 → 예상 구매일 대비 지연 감지 → 이탈 확률 70% 이상 고객 자동 플래그 → 맞춤 혜택(할인·무료배송·한정 제품 우선 접근) 자동 발송.

**VIP 자동 관리**: 상위 5% 고객 자동 식별 → 프리미엄 혜택(전용 할인·신제품 사전 공개·1:1 담당자 배정) 자동 트리거 → 분기별 감사 메시지 자동 발송.

**신규 고객 2차 구매 유도**: 첫 구매 후 7일째 AI가 관련 제품 추천 → 14일째 후기 작성 유도 + 포인트 → 30일째 미구매 시 맞춤 쿠폰 → 전환 여부에 따라 다음 시나리오 자동 분기.

**효과 대시보드**: 세그먼트별 재구매율, LTV(고객 생애 가치), 이탈률, 프로그램 ROI 자동 집계.
