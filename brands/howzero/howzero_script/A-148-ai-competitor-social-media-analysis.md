## 대본

### 훅 (0:00–0:40)
경쟁사 인스타그램을 매일 들여다보고 계시죠. "이 브랜드는 왜 잘 되지?" 느낌은 있는데, 데이터가 없습니다. 어떤 콘텐츠가 반응이 좋은지, 게시 빈도가 어떤지, 해시태그 전략이 뭔지 — 감이 아니라 데이터로 분석하면 전략이 보입니다. AI로 경쟁 브랜드 SNS를 자동 분석하는 시스템을 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
경쟁사 분석의 3대 병목. 첫째, 수동 모니터링 — 경쟁사 3~5개의 인스타/유튜브/블로그를 수동 체크. 시간만 쓰고 체계적 분석 안 됨. 둘째, 정량 분석 부재 — "이 포스트 반응 좋네" 수준. 좋아요 수, 댓글 수, 인게이지먼트 비율, 게시 시간대 등 정량 비교가 없음. 셋째, 인사이트 도출 — 데이터가 있어도 "그래서 우리는 뭘 해야 하는지"로 연결 못함.

### 자동화 설계 (3:00–8:00)
**경쟁사 콘텐츠 자동 수집**: 경쟁사 SNS 계정 등록 → 매일 새 게시물 자동 수집(텍스트, 이미지 유형, 해시태그, 게시 시간) → 구글 시트에 자동 저장 → 좋아요/댓글/공유 수 자동 기록.

**AI 자동 분석**: 주간 데이터 집계 → GPT-4가 자동 분석 — 인게이지먼트 비율 상위 10% 콘텐츠 특징, 게시 빈도 패턴, 해시태그 전략, 콘텐츠 유형별 성과(이미지 vs 영상 vs 캐러셀), 톤앤매너 분석.

**트렌드 감지**: 경쟁사의 새로운 시도 자동 탐지(신제품 출시, 캠페인, 협업) → 즉시 슬랙 알림 → 대응 전략 제안.

**벤치마크 리포트**: 월 1회 자동 생성 — 우리 vs 경쟁사 비교표(게시 빈도, 인게이지먼트, 팔로워 증가율) → 경쟁사 대비 우리의 강점/약점 AI 분석 → "이번 달 실행 추천 3가지" 자동 제안.

### 데모/사례 (8:00–13:30)
뷰티 브랜드 SC. 경쟁사 4개 모니터링. 자동화 전: 마케터가 주 3시간 경쟁사 SNS 수동 체크, 체계적 분석 없음, 콘텐츠 전략이 감 의존. 자동화 후: 자동 수집+분석으로 주 3시간→30분(리포트 검토만). 경쟁사 인기 콘텐츠 유형 발견(비포/애프터 캐러셀) → 도입 후 인게이지먼트 3.2배↑. 최적 게시 시간대 발견(화·목 저녁 8시) → 적용 후 도달율 40%↑. 경쟁사 신제품 출시 당일 감지 → 대응 콘텐츠 48시간 내 발행.
