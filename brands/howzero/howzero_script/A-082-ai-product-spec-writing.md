# A-082: 유튜브 롱폼 — "AI로 제품 기획서 자동 작성"
> **유형**: 유튜브 롱폼 | **길이**: 16분
## 메타데이터
- **제목**: AI로 제품 기획서 자동 작성 — 3시간 → 20분으로
- **썸네일 텍스트**: 기획서 자동 작성 | AI 20분 완성
- **태그**: 제품 기획서, AI 문서 자동화, PRD 자동 생성, 노코드, 하우제로
## 대본

### 훅 (0:00–0:40)
제품 기획서 하나 쓰는 데 반나절 날리고 있지 않습니까? 시장 조사, 경쟁 분석, 기능 정의, 일정 수립… 매번 빈 문서부터 시작하면 3시간은 기본입니다. 저는 이걸 20분으로 줄였습니다. AI가 초안을 만들고, 사람은 판단만 합니다. 오늘 그 시스템 전부 보여드립니다.

### 문제 정의 (0:40–3:00)
기획서 작성의 병목은 세 가지입니다. 첫째, 정보 수집. 시장 데이터, 경쟁사 분석, 내부 요구사항 — 여기저기 흩어져 있음. 둘째, 구조화. 수집한 정보를 PRD 포맷에 맞춰 정리하는 건 순전히 노동. 셋째, 반복 수정. 팀 피드백 → 수정 → 재피드백 → 재수정의 무한 루프. 기획자의 진짜 가치는 '판단'인데, 시간의 70%를 '정리'에 쓰고 있습니다.

### 자동화 설계 (3:00–8:00)
**입력**: 슬랙/노션에 제품 아이디어 한 줄 + 타겟 고객 + 핵심 문제 입력. **AI 처리**: GPT-4가 자동으로 — 시장 규모 추정(퍼플렉시티 API 연동) + 경쟁사 3개 자동 분석 + 핵심 기능 리스트 도출 + 사용자 스토리 생성 + 성공 지표(KPI) 제안 + 간트 차트용 일정 초안. **출력**: 노션 템플릿에 자동 채워넣기. 기획자는 검토 + 수정만.

추가 자동화: 팀 피드백을 슬랙 스레드로 수집 → AI가 피드백 분류 + 기획서 자동 반영 + 변경 이력 관리.

### 데모/사례 (8:00–12:30)
라이브 시연: "반려동물 건강 관리 앱" 기획서를 실시간 생성. 아이디어 한 줄 입력 → 3분 후 12페이지 PRD 초안 완성. 시장 규모, 경쟁사 분석, 기능 명세, 와이어프레임 설명, KPI까지 포함. 실제 스타트업 B사 적용 사례: 월 평균 기획서 8건 작성. 기획서 1건당 소요 시간 3.5시간 → 40분. 월 기준 22시간 절감.

### 효과 정리 (12:30–14:30)
- 기획서 작성 시간 80% 절감
- 정보 수집 자동화로 누락 항목 90% 감소
- 피드백 반영 주기 3일 → 당일
- 기획자가 '정리'가 아닌 '판단'에 집중
