## 대본

### 훅 (0:00–0:40)
월말에 KPI 리포트 만드느라 이틀 날리고 있지 않습니까? 각 부서에서 엑셀 받아서 취합하고, 차트 만들고, PPT에 옮기고… 이 작업을 매달 반복합니다. AI가 데이터를 자동으로 수집하고, 대시보드를 실시간으로 업데이트하고, 이상치를 자동 감지해서 알려주는 시스템. 15분이면 구축합니다.

### 문제 정의 (0:40–3:00)
KPI 관리의 현실. 첫째, 데이터 분산 — 매출은 ERP, 마케팅은 GA, CS는 제노데스크, HR은 엑셀. 둘째, 수동 취합 — 부서별 데이터를 일일이 복사/붙여넣기. 셋째, 사후 분석 — 월말에야 수치를 보니 문제 발견이 늦음. 넷째, 해석 부재 — 숫자는 있는데 "그래서 뭘 해야 하는지" 모름.

### 자동화 설계 (3:00–8:00)
**데이터 자동 수집**: 각 시스템에서 구글 시트로 자동 동기화 — ERP 매출, GA 트래픽, CRM 리드, CS 티켓 등. API 또는 Make 커넥터 활용.

**실시간 대시보드**: 구글 시트 데이터 → Looker Studio 자동 시각화. 매출, CAC, LTV, 이탈률, NPS 등 핵심 KPI 실시간 표시.

**AI 인사이트**: 매주 월요일 GPT-4가 자동으로 — 전주 대비 이상치 감지 + 원인 가설 제시 + 개선 액션 제안 → 슬랙으로 자동 브리핑.

**월간 리포트 자동 생성**: 월말 자동으로 PPT/PDF 리포트 생성. 핵심 수치 + 차트 + AI 해석 + 다음 달 제안.
