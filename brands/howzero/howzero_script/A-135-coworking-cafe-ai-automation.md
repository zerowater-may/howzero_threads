## 대본

### 훅 (0:00–0:40)
코워킹 카페가 수익을 내려면 좌석 가동률이 70% 이상이어야 합니다. 그런데 대부분의 코워킹 카페가 평일 오전에는 텅 비고, 오후에는 만석입니다. 좌석 가동률 편차가 40%입니다. AI로 좌석 예약, 멤버십 관리, 커뮤니티까지 자동화해서 가동률을 85%로 올린 코워킹 카페 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
코워킹 카페의 3대 병목. 첫째, 좌석 관리 — 예약 없이 워크인이 대부분. 만석인지 아닌지 고객이 와봐야 앎. 빈 좌석이 있어도 짐만 놓고 자리를 비우는 경우. 둘째, 멤버십 관리 — 시간권, 일권, 월권, 패키지 등 요금제 복잡. 수동 관리하면 결제 누락, 만료 미통보 빈번. 셋째, 차별화 — 주변 카페와의 차이가 "콘센트와 와이파이" 뿐. 커뮤니티 기능 없으면 충성 고객 확보 어려움.

### 자동화 설계 (3:00–8:00)
**스마트 좌석 관리**: 카카오 챗봇으로 실시간 잔여 좌석 확인 + 좌석 예약 → QR 체크인/체크아웃 → 이용 시간 자동 계산 → 2시간 이상 자리 비움 시 알림 → 비수기 시간대(오전) 할인 자동 적용.

**멤버십 자동화**: 회원 가입 시 이용 패턴 분석 → 최적 요금제 자동 추천 → 결제 자동 → 이용권 만료 7일 전 자동 안내 → 업그레이드 제안(시간권→월권 전환 시 혜택).

**커뮤니티 자동화**: 회원 프로필(직업, 관심사) 기반 네트워킹 매칭 → 월 1회 자동 네트워킹 이벤트 공지 → 커뮤니티 게시판 관리(질문 자동 분류, 답변 알림) → 회원 생일 자동 축하 + 무료 음료 쿠폰.

**운영 데이터 분석**: 시간대별·요일별 가동률 자동 분석 → 비수기 프로모션 자동 기획(오전 커피+좌석 패키지) → 인기 좌석 히트맵 → 직원 배치 최적화 제안.

### 데모/사례 (8:00–13:30)
코워킹 카페 CW. 좌석 40석, 월 방문자 800명. 자동화 전: 좌석 가동률 평균 55%(오전 30%, 오후 80%), 멤버십 결제 누락 월 8건, 커뮤니티 활동 없음, 재방문율 40%. 자동화 후: 좌석 가동률 55%→85%(오전 할인 효과+예약 시스템). 결제 누락 0건. 네트워킹 이벤트로 커뮤니티 활성화 — 재방문율 40%→72%. 멤버십 전환율(시간권→월권) 25%↑.
