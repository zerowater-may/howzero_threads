### [1:30~4:00] 고객 생애주기

"고객 관리의 본질은 '생애주기'입니다.

**1단계: 인지** → 잠재 고객이 나를 발견
**2단계: 관심** → 뉴스레터 구독, SNS 팔로우
**3단계: 문의** → 구체적 질문/상담 요청
**4단계: 구매** → 첫 거래
**5단계: 유지** → 재구매/리테이너
**6단계: 추천** → 주변에 소개

각 단계에서 해야 할 액션이 있는데
대부분 4단계(구매)까지만 하고 5~6단계를 놓칩니다.

5~6단계에서 놓치는 매출이 전체의 60%입니다."

### [4:00~8:00] AI CRM 설계

"**시스템 1: 리드 자동 분류**
- 문의 들어오면 자동 기록
- HOT/WARM/COLD 자동 분류
- 각 단계별 팔로업 알림

**시스템 2: 팔로업 자동화**
- HOT → 24시간 내 개인화 이메일 자동
- WARM → 3일 후 사례 자료 발송
- COLD → 월간 뉴스레터 자동 편입
- 무응답 3회 → 자동 아카이브

**시스템 3: 구매 후 관리**
- 구매 완료 → 감사 메일 + 사용 가이드
- 7일 후 → '잘 쓰고 계세요?' 체크인
- 30일 후 → 리뷰 요청
- 90일 후 → 재구매/업그레이드 제안

**시스템 4: 추천 프로그램**
- 만족도 높은 고객 자동 식별
- 추천 요청 이메일 자동 발송
- 추천 발생 시 감사 메일 + 혜택 안내"
