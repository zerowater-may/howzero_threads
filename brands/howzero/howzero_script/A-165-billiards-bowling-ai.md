### 문제 정의 (0:40–3:30)
당구장·볼링장의 3대 병목. 첫째, 예약·대기 관리 — 피크 시간 전화 폭주. 대기 명단 수기 관리. 노쇼 대응 불가. 둘째, 테이블/레인 배정 — 몇 번 테이블이 비는지 실시간 파악 어려움. 배정 비효율로 회전율 하락. 셋째, 비수기 매출 — 평일 오후 가동률 30% 이하. 프로모션 해도 타겟팅 없이 뿌리기식.

### 자동화 설계 (3:30–9:30)
**예약·대기 자동화**: 카카오/네이버 예약 연동 → 실시간 테이블/레인 현황 자동 업데이트 → 예약 확정 시 자동 안내(위치, 주차, 예상 대기) → 노쇼 방지: 예약 30분 전 자동 확인 문자 → 미응답 시 자동 취소+대기자 자동 승격.

**테이블/레인 자동 배정**: IoT 센서 없이도 가능 — POS 결제 시점 기준 사용 시작, 결제 종료 기준 사용 완료 자동 기록 → 실시간 사용 현황 대시보드 → 예상 종료 시간 AI 예측(과거 데이터 기반 평균 이용 시간) → 프론트 직원에게 "3번 테이블 10분 후 종료 예상" 자동 알림.

**비수기 프로모션 자동화**: 가동률 50% 이하 시간대 자동 식별 → 해당 시간대 타겟 프로모션 자동 생성("화요일 14~17시 2시간 이용 시 1시간 무료") → 지난 3개월 내 방문 고객에게 자동 발송 → 프로모션 효과 자동 측정.

**매출·운영 분석**: 일별·시간대별·테이블별 매출 자동 집계 → 고객 재방문 주기 분석 → 테이블별 수익성 분석(코너 테이블 vs 중앙 테이블) → 월간 운영 리포트 자동 생성.
