## 대본

### 훅 (0:00–0:40)
복지시설에서 일하시는 분들. 이용자 케어에 집중하고 싶은데, 행정 업무에 하루가 다 갑니다. 일지 작성, 보고서 제출, 프로그램 일정 관리, 보호자 연락. 사회복지사가 복지 업무보다 서류 업무를 더 많이 합니다. AI가 행정 업무의 60%를 자동화하면? 직원들은 이용자 케어에 집중하고, 보호자 소통은 실시간이 되고, 프로그램 품질은 올라갑니다. 코딩 제로.

### 문제 정의 (0:40–3:00)
복지시설 운영의 병목. 첫째, 행정 과부하 — 일일 일지, 월간 보고서, 정부 제출 서류. 직원 시간의 40%가 여기에 갑니다. 둘째, 이용자별 맞춤 케어 — 이용자 50명의 개별 상태, 투약, 프로그램 참여를 추적하기 어렵습니다. 셋째, 보호자 소통 — "오늘 우리 아이 어떻게 지냈나요?" 매번 전화로 확인합니다. 넷째, 프로그램 성과 측정 — 어떤 프로그램이 이용자에게 효과적인지 데이터가 없습니다.

### 자동화 설계 (3:00–8:30)
**일지+보고서 자동 생성**: 직원이 간단 키워드/체크리스트 입력 → AI가 정형화된 일일 일지 자동 작성 → 월간 종합 보고서 자동 생성 → 정부 제출 양식에 맞춰 자동 변환 → 작성 시간 70% 감소.

**이용자 상태 자동 추적**: 이용자별 일일 컨디션·투약·식사·프로그램 참여 체크리스트 → 자동 기록+추이 분석 → "이용자 A, 최근 2주 식사량 30% 감소, 컨디션 하락 추세" 자동 알림 → 조기 대응.

**보호자 자동 소통**: 일일 활동 요약 자동 생성 → 카카오톡으로 보호자에게 오후 6시 자동 발송 → "오늘 미술 프로그램 참여, 점심 잘 드셨어요, 기분 좋은 하루였습니다" → 보호자 만족도 상승.

**프로그램 성과 자동 분석**: 프로그램별 참여율·이용자 반응·행동 변화 자동 측정 → 분기 성과 리포트 자동 생성 → "음악 치료 프로그램 참여 이용자, 사회성 점수 평균 25% 향상" 데이터 근거.
