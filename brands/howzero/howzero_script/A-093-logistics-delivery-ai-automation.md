### 문제 정의 (0:40–3:00)
운송 업체의 3대 병목. 첫째, 배차 — 기사·차량·물량·지역을 동시에 고려해야 하는 최적화 문제. 사람이 하면 비효율. 둘째, 경로 — 네비게이션은 1건 최적이지, 다건 배송 경로 최적화는 못함. 셋째, 정산 — 기사별 운행 거리, 물량, 수당 계산이 복잡. 수기 오류 잦음.

### 자동화 설계 (3:00–8:30)
**배차 자동화**: 당일 배송 건 자동 수집 → AI가 기사별 최적 배분(지역, 물량, 근무 시간 고려) → 배차표 자동 생성 → 기사 앱으로 자동 전송.

**경로 최적화**: 기사별 배송지 리스트 → AI가 최단 거리+시간 경로 자동 산출(교통 상황 반영) → 네비게이션 연동.

**실시간 추적**: GPS 기반 배송 현황 실시간 대시보드 → 지연 시 자동 알림(관리자+고객).

**정산 자동화**: GPS 운행 기록 기반 거리 자동 계산 → 기사별 수당·유류비 자동 산출 → 월말 정산 리포트 자동 생성.
