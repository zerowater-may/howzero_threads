## 대본

### 훅 (0:00–0:50)
에이스 직원이 퇴사합니다. 갑자기. 그 직원 머릿속에만 있던 고객 관리 노하우, 엑셀 매크로, 거래처 연락 방법 — 전부 사라집니다. 한국 중소기업 평균 직원 근속 연수 2.7년입니다. 3년마다 모든 노하우가 리셋됩니다. 그런데 시스템으로 만들어 놓으면? 직원이 바뀌어도 업무는 그대로 돌아갑니다. 오늘은 "사람 의존"에서 "시스템 의존"으로 전환하는 AI 자동화 이야기를 하겠습니다.

### 문제 정의 (0:50–4:00)
직원 퇴사의 숨겨진 비용. 채용 비용(공고, 면접, 교육) 평균 연봉의 30~50%. 하지만 진짜 비용은 "지식 유실"입니다. 첫째, 암묵지 소실 — 거래처 담당자 성향, 특수 주문 처리 방법, 고객 클레임 대응 노하우. 문서화되지 않은 것들. 둘째, 생산성 공백 — 새 직원이 이전 수준에 도달하기까지 평균 6개월. 그 기간 동안 생산성 40%↓. 셋째, 반복되는 실수 — 이전 직원이 겪고 해결한 문제를 새 직원이 똑같이 겪음.

### 자동화 설계 (4:00–10:00)
**업무 자동 문서화**: 모든 반복 업무를 AI가 자동 기록 — 이메일 패턴, CS 응대 기록, 주문 처리 흐름 → 자동으로 업무 매뉴얼 생성 → 실시간 업데이트.

**지식 베이스 자동 구축**: 직원의 카카오톡 업무 대화, 이메일, 메모를 AI가 분류 → 업무별 FAQ 자동 생성 → 검색 가능한 지식 베이스 구축 → 새 직원이 질문하면 AI가 기존 지식에서 답변.

**프로세스 자동화**: 사람이 하던 반복 업무를 시스템으로 전환 — 견적 발송, 주문 확인, 재고 알림, 고객 팔로업. 사람이 바뀌어도 프로세스는 동일하게 작동.

**온보딩 자동화**: 새 직원 입사 → 해당 직무 SOP 자동 안내 → 1주차/2주차/4주차 학습 체크리스트 자동 발송 → 질문 시 AI 챗봇이 지식 베이스에서 답변 → 멘토(이전 직원이 아닌 시스템)가 가이드.

### 데모/사례 (10:00–14:30)
