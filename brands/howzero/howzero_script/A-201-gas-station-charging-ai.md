## 대본

### 훅 (0:00–0:40)
주유소 운영하시는 분들, 유가 변동 체크하고, 재고 파악하고, 직원 시프트 짜고, 세차 예약 관리하고 — 이거 하루에 몇 시간 쓰세요? 제가 컨설팅한 주유소 하나는 이 전부를 AI로 돌리고 있습니다. 사장님은 매장에 하루 2시간만 나옵니다. 코딩 0줄로 만든 시스템입니다.

### 문제 정의 (0:40–3:00)
주유소·충전소의 핵심 병목 세 가지. 첫째, 유가 변동 대응 — 국제 유가, 세금, 경쟁사 가격을 매일 체크해서 가격을 조정해야 합니다. 둘째, 재고 관리 — 유종별 탱크 잔량 확인, 발주 타이밍 판단, 배송 스케줄 조율. 셋째, 고객 관리 — 세차 예약, 멤버십, 쿠폰 발송 전부 수동. EV 충전소는 충전기 상태 모니터링까지 추가됩니다.

### 자동화 설계 (3:00–8:00)
**유가 모니터링 자동화**: 오피넷 데이터 + 경쟁사 가격 자동 수집 → AI가 최적 가격 추천 → 사장님 승인 시 POS 반영. 매일 아침 8시에 알림톡으로 리포트 발송.

**재고·발주 자동화**: 탱크 잔량 데이터 연동 → 판매 속도 기반 소진 예측 → 잔량 기준치 이하 시 자동 발주 요청서 생성 + 배송사 발송.

**고객 관리 자동화**: 주유 이력 기반 멤버십 등급 자동 분류 → 등급별 쿠폰 자동 발송 → 세차 예약 알림 자동화. 리뷰 남기면 자동 감사 메시지.

**EV 충전기 모니터링**: 충전기 상태 API 연동 → 고장·점유 상태 실시간 알림 → 정비 요청 자동 접수.

### 데모/사례 (8:00–12:30)
경기도 소재 주유소 K. 유종 3종, 셀프+풀서비스, 세차장 병행. 자동화 전: 사장님 매장 상주 12시간, 가격 조정 주 5회 수동, 발주 실수로 월 1~2회 품절. 자동화 후: 매장 출근 2시간, 가격 조정 자동 추천(승인만 클릭), 품절 0건(6개월 연속). 세차 예약률 35%↑, 단골 재방문율 22%↑.

도구: 구글 시트 + Make + GPT-4 + 카카오 알림톡 + POS API 연동. 월 운영비 약 8만 원.
