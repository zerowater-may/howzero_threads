### 문제 정의 (0:40–3:00)

### 자동화 설계 (3:00–8:30)
**데일리 스탠드업 자동화**: 매일 오전 9시 슬랙 봇이 "어제 한 일 / 오늘 할 일 / 블로커" 자동 수집 → AI가 팀 전체 요약 생성 → 미팅 없이 현황 파악.

**업무 진행 자동 트래킹**: 노션/지라 태스크 상태 변경 → 자동으로 주간 진행률 대시보드 갱신 → 3일 이상 정체된 태스크 자동 알림.

**미팅 자동 최적화**: 줌 미팅 녹화 → AI 자동 요약 + 액션 아이템 추출 → 태스크 자동 생성 → 담당자 자동 할당.

**비동기 커뮤니케이션 허브**: 팀원이 메시지 남기면 → AI가 맥락 요약 → 관련 문서/이전 논의 자동 링크 → 응답 필요 시 담당자 멘션 자동.

**주간 리포트 자동 생성**: 금요일 → AI가 팀별 완료 태스크, 진행 중 태스크, 블로커, KPI 진행률 자동 집계 → 리더에게 자동 발송.
