### [1:30~3:30] 문제 정의

"학원 행정 업무를 정리하면:

1. **수강 문의 응대** — 카톡/전화 반복 답변 (하루 30분~1시간)
2. **출석 관리** — 매 수업 체크 + 엑셀 입력 (하루 30분)
3. **학부모 알림** — 진도, 출석, 공지 개별 전송 (하루 1시간)
4. **수납 관리** — 미납 확인 + 알림 (주 1시간)
5. **보강/일정 조율** — 개별 연락 (주 1시간)

합하면 하루 2~3시간.
원장님이 직접 하실 일은 사실 수업과 커리큘럼뿐입니다."

---

### [3:30~7:00] 자동화 설계

"**시스템 1: 수강 문의 자동 응대**
- 자주 묻는 질문 자동 답변 (수업 시간, 비용, 커리큘럼)
- 상담 예약 자동 잡기
- 복잡한 문의만 원장님에게 알림

**시스템 2: 출석+학부모 알림 자동화**
- 수업 시작 시 출석 체크 → 자동 기록
- 결석 시 학부모에게 자동 알림
- 주간 학습 리포트 자동 생성+전송

**시스템 3: 수납+일정 자동 관리**
- 수납일 3일 전 자동 알림
- 미납 시 리마인드 메시지 자동
- 보강 일정 자동 매칭+알림"

---

### [7:00~12:00] 라이브 데모

"[수강 문의 자동 답변]
(화면: 카톡 문의 시뮬레이션)

문의: '수학 중등반 수업 시간이 어떻게 되나요?'

AI 자동 답변:
'안녕하세요! 중등 수학반은 화/목 16:00~17:30입니다.
현재 3자리 남아있습니다.
상담 예약을 원하시면 아래에서 시간을 선택해주세요.'

문의부터 상담 예약까지 자동.
원장님이 개입할 필요 없습니다.

[학부모 알림 자동화]
'오늘 수업에 참석한 15명 학생의 학부모에게
출석 확인 + 오늘 수업 내용 요약 알림을 보내줘.'

(실행 — 15명분 개별 메시지 생성)

이걸 수동으로 하면 1시간.
AI로 하면 2분.

[수납 관리]
'이번 달 수납 현황을 확인하고,
미납자 3명에게 리마인드 메시지를 작성해줘.
톤: 정중하지만 명확하게.'

수납 알림까지 자동."
