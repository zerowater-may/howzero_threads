### 훅 (0:00–0:40)
펫 미용실 운영하시는 분, 고객이 올 때마다 다시 물어보지 않으세요? "어떤 스타일로 할까요?", "피부 예민한 데 있나요?", "지난번 컷 만족하셨나요?" 단골이어도 기억이 안 됩니다. AI가 반려동물별 미용 기록, 피부 상태, 선호 스타일을 전부 기억하고, 예약부터 사후 관리까지 자동 처리합니다. 재방문율 2.3배. 반려동물 이름을 기억하는 매장이 이깁니다.

### 문제 정의 (0:40–3:00)
펫 미용업의 병목. 첫째, 예약 관리 — 전화·카카오톡·네이버 예약이 혼재. 대형견은 시간이 더 필요한데 일반 예약과 같은 시간 배정. 둘째, 반려동물 정보 관리 — 견종·크기·피부 상태·공격성·선호 스타일. 미용사가 전부 기억할 수 없습니다. 셋째, 노쇼 — 반려동물 동반이라 노쇼가 더 많습니다. 평균 20%. 넷째, 사후 관리 부재 — 미용 후 사진 공유, 다음 미용 주기 안내를 안 합니다. 재방문 유도가 안 됩니다.

### 자동화 설계 (3:00–8:00)
**스마트 예약**: 반려동물 크기·견종·시술 내용 기반 소요 시간 자동 산정 → 미용사 스케줄에 최적 배치 → 노쇼 이력 고객 사전 예약금 안내 → 24시간 전+2시간 전 자동 리마인더("오늘 코코 미용 예약 있으시죠?").

**반려동물 프로필 자동 관리**: 첫 방문 시 반려동물 정보 수집(이름·견종·나이·무게·피부 상태·주의사항) → 매 방문 미용 기록 자동 저장(스타일·사용 제품·특이사항) → 다음 방문 시 미용사에게 자동 브리핑("코코, 포메라니안 3살, 피부 건조 주의, 곰돌이컷 선호, 지난번 뒷다리 피부 발진 있었음").

**미용 후 자동 관리**: 미용 완료 후 비포/애프터 사진 보호자에게 자동 발송 → 미용 주기(견종별 권장 4~6주) 기반 다음 예약 자동 제안 → "코코 다음 미용 예정일은 4월 25일이에요. 예약할까요?" → 계절별 케어 팁 자동 발송.

**매출 분석 자동화**: 미용사별·시술별·시간대별 매출 자동 분석 → 인기 시술·비인기 시간대 파악 → 비인기 시간대 할인 프로모션 자동 실행 → 회원 등급별 혜택 자동 적용.
