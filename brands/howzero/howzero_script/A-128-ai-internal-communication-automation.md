## 대본

### 훅 (0:00–0:40)
"공지 봤어요?" "아, 못 봤어요." 이 대화가 회사에서 하루에 몇 번 반복됩니까. 공지는 카톡으로, 보고는 이메일로, 피드백은 구두로. 채널이 분산되면 정보가 사라집니다. 직원 10명 이상이면 사내 소통 비용이 매출의 5~10%를 잡아먹습니다. AI로 공지, 보고, 피드백을 하나의 시스템으로 묶은 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
사내 소통의 3대 병목. 첫째, 공지 전달률 — 카톡 공지방에 올려도 읽지 않는 직원 30%+. 중요 공지 누락으로 업무 차질. 둘째, 보고 지연 — 일일/주간 보고를 독촉해야 받는 문화. 보고서 양식도 제각각. 셋째, 피드백 부재 — 상사의 피드백은 바빠서 생략, 직원의 건의는 "나중에"로 묻힘. 양방향 소통이 단방향이 됨.

### 자동화 설계 (3:00–8:00)
**스마트 공지 시스템**: 공지 작성 → AI가 핵심 요약 자동 생성 → 슬랙+카카오+이메일 동시 발송 → 읽음 확인 자동 추적 → 미확인자에게 24시간 후 재발송 → 중요 공지는 퀴즈형 확인("이번 달 재고 마감일은?") → 전달률 95%+.

**자동 보고 시스템**: 매일 오후 5시 슬랙봇이 보고 양식 발송(오늘 한 일, 이슈, 내일 계획) → 직원이 3줄 입력 → AI가 팀별·프로젝트별 자동 집계 → 주간 보고서 자동 생성 → 대표에게 월요일 오전 자동 발송.

**피드백 자동화**: 프로젝트 완료 시 자동 피드백 요청 → 동료 평가(간단 체크리스트) → AI가 분기별 성과 리포트에 통합 → 건의사항 익명 접수 → AI가 카테고리 분류 후 담당 부서에 자동 전달.

**회의 자동화**: 회의 녹음 → Whisper로 자동 요약 → 액션 아이템 자동 추출 → 담당자에게 할일 자동 배정 → 마감일 리마인드.
