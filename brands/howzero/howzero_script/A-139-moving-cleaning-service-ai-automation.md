## 대본

### 훅 (0:00–0:40)
이삿짐센터 사장님, 견적 전화 한 통에 얼마나 걸리십니까. "몇 평이세요? 짐 많으세요? 엘리베이터 있어요? 날짜는요?" 최소 10분입니다. 하루 견적 문의 20건이면 전화만 3시간 넘습니다. 그런데 실제 계약 전환율은 30% 미만. AI로 견적 자동 산출, 배차 최적화, 후기 관리까지 자동화한 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
이삿짐/청소대행의 3대 병목. 첫째, 견적 산출 — 평수, 짐 양, 거리, 층수, 엘리베이터 유무, 특수 짐 등 변수가 많음. 경험으로만 산출 가능. 견적 편차도 크고, 고객 신뢰 하락. 둘째, 배차 관리 — 기사·차량·시간을 맞추는 퍼즐. 수동 배차는 비효율(공차 운행, 동선 낭비). 셋째, 후기 관리 — 서비스 후 후기 요청 안 하면 안 남기고, 불만이 있으면 네이버에 바로 악평. 선제 대응 없음.

### 자동화 설계 (3:00–8:00)
**AI 자동 견적**: 카카오 챗봇 또는 웹폼으로 정보 입력(평수, 짐 사진, 출발/도착 주소, 층수, 날짜) → AI가 사진 분석으로 짐 양 추정 + 거리·층수 기반 자동 견적 → 3등급 견적 제시(기본/표준/프리미엄) → 고객 선택 시 계약 자동 진행.

**배차 자동화**: 확정된 이사/청소 건을 구글 시트에 집계 → 기사 위치·가용 시간 기반 자동 배차 → 동선 최적화(같은 지역 건 연속 배치) → 기사에게 자동 일정 알림 → 고객에게 기사 정보·도착 예정 시간 자동 안내.

**후기 관리 자동화**: 서비스 완료 2시간 후 만족도 조사 자동 발송 → 4~5점: 네이버/카카오 후기 작성 유도(링크 포함) → 1~3점: 즉시 사장님에게 알림 + 불만 해소 대응 → 월별 후기 분석 리포트 자동.

**재주문 유도**: 이사 고객 → 3개월 후 입주 청소 프로모션 → 청소 고객 → 정기 청소 구독 안내 → 계절별 에어컨 청소, 대청소 자동 안내.

### 데모/사례 (8:00–13:30)
