## 대본

### 훅 (0:00–0:40)
"우리 회사 복리후생이 뭐가 있어요?" 직원이 물을 때마다 인사팀이 같은 설명을 반복하고 있습니다. 건강검진, 도서비, 자기개발비, 경조사 지원, 생일 선물 — 종류는 많은데 관리가 안 됩니다. 신청서 제출, 영수증 정리, 예산 추적이 모두 수동. AI로 복리후생 신청부터 정산까지 자동화한 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
복리후생 관리의 3대 병목. 첫째, 신청 프로세스 — 복리후생 종류마다 신청 방법이 다르고, 어떤 건 이메일, 어떤 건 구두, 어떤 건 양식. 직원이 복잡해서 안 씁니다. 둘째, 정산 — 영수증 수집, 한도 확인, 승인, 입금까지 수동이면 인사팀 월말 3일 매달림. 셋째, 활용도 파악 — 어떤 복리후생이 인기 있는지, 예산이 얼마나 남았는지 실시간 파악 불가. 좋은 제도인데 활용률 30% 미만인 경우 빈번.

### 자동화 설계 (3:00–8:00)
**원스톱 신청 시스템**: 슬랙봇/카카오 챗봇으로 전 복리후생 항목 안내 → 항목 선택 → 필요 정보 입력(금액, 영수증 사진) → 자동 한도 확인 → 팀장 승인 요청 자동 → 승인 시 정산 자동 진행.

**자동 정산**: 영수증 사진 → AI OCR로 금액·날짜·사업자 자동 인식 → 항목별 예산 차감 → 월말 일괄 정산 자동 → 급여 반영 데이터 자동 생성.

**대시보드**: 직원별 복리후생 사용 현황, 항목별 예산 잔여, 활용률 실시간 → 활용률 하위 항목 분석 → 반기 1회 복리후생 만족도 설문 자동 → 결과 기반 제도 개선 제안.

**자동 안내**: 신규 입사자에게 복리후생 전체 안내 자동 발송 → 미사용 복리후생 분기 1회 리마인드("올해 도서비 아직 20만 원 남으셨습니다") → 생일·기념일 자동 축하 + 선물 자동 발송.
