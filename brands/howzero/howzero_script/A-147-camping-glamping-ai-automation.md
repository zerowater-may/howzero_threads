## 대본

### 훅 (0:00–0:40)
캠핑장 사장님, 성수기에 전화기 녹으시죠. "자리 있어요?" "바비큐 가능해요?" "체크인 몇 시예요?" 성수기 주말에는 전화만 일 80통 넘습니다. 그런데 비수기에는 예약률 20% 미만. 이 극단적 편차를 AI로 관리한 캠핑장 사례를 보여드리겠습니다. 성수기 전화 폭주 해결부터, 비수기 가동률 3배 올리기까지.

### 문제 정의 (0:40–3:00)
캠핑장/글램핑장의 3대 병목. 첫째, 예약 관리 — 네이버 예약, 전화, 카톡, 플랫폼 예약이 혼재. 더블 부킹 발생, 잔여 사이트 파악 실시간 불가. 둘째, 시즌 편차 — 성수기(7~8월, 연휴) 예약률 100%, 비수기(11~2월) 20% 미만. 비수기 매출이 바닥. 셋째, 현장 운영 — 체크인/아웃 안내, 시설 이용 규칙, 주변 관광 안내 등 반복 안내 업무.

### 자동화 설계 (3:00–8:00)
**통합 예약 시스템**: 모든 예약 채널을 구글 시트 하나로 통합 → 실시간 잔여 사이트 표시 → 카카오 챗봇으로 사이트 유형(텐트/카라반/글램핑), 날짜, 인원 선택 → 즉시 예약 확정 → 결제 링크 자동 발송.

**비수기 가동률 자동화**: 비수기 시작 2개월 전 기존 고객에게 할인 프로모션 자동 발송 → 주중 특가 패키지(숙박+바비큐 세트) → 기업 워크숍/단체 타겟 이메일 캠페인 → 예약률 실시간 모니터링 → 목표 미달 시 추가 프로모션 자동 트리거.

**현장 운영 자동화**: 예약 확정 시 체크인 안내(시간, 위치, 준비물) 자동 발송 → 체크인 당일 오전 날씨+교통 정보 자동 안내 → 사이트 배정 QR 코드 → 시설 이용 규칙 카카오 메시지 → 체크아웃 안내 + 후기 요청.

**리뷰 관리**: 체크아웃 2시간 후 만족도 조사 → 4~5점: 네이버/카카오 후기 유도 → 1~3점: 즉시 사장님 알림 + 개선 대응 → 월별 리뷰 분석(키워드 추출 — 시설, 청결, 경치 등).

### 데모/사례 (8:00–13:30)
글램핑장 CP. 사이트 15개, 연간 운영. 자동화 전: 전화 예약 80%, 비수기 가동률 18%, 더블 부킹 월 2~3건, 리뷰 관리 없음(별점 3.9). 자동화 후: 온라인 예약 전환 75% — 전화 감소 70%. 비수기 프로모션 자동화로 가동률 18%→52%. 더블 부킹 0건. 리뷰 관리로 별점 3.9→4.6.
