## 대본

### 훅 (0:00–0:40)
광고 카피 하나 쓰는 데 얼마나 고민하세요? 30분? 1시간? 그리고 그 카피가 성과가 좋을지 나쁠지는 돌려봐야 압니다. AI로 카피 50개를 5분 만에 만들고, 자동 A/B 테스트로 최적 카피를 찾고, 성과 나쁜 건 자동으로 끄는 시스템. 광고 ROAS 2배 올린 방법입니다.

### 문제 정의 (0:40–3:00)
SNS 광고의 3대 병목. 첫째, 카피 제작 — 매번 새 카피를 쓰는 게 고통. 아이디어 고갈. 둘째, A/B 테스트 — 해야 하는 건 아는데 수동으로 하면 너무 번거로움. 대부분 안 함. 셋째, 최적화 — 성과가 떨어지는 광고를 며칠째 방치. 돈이 새고 있는데 모름.

### 자동화 설계 (3:00–9:00)
**카피 자동 생성**: 제품 정보 + 타겟 오디언스 + 톤 입력 → AI가 — 헤드라인 10개 + 본문 10개 + CTA 5개 조합 → 50개+ 카피 변형 자동 생성. 과거 성과 데이터 반영(클릭률 높았던 패턴 우선).

**자동 A/B 테스트 세팅**: 생성된 카피 중 상위 10개 선정 → Meta/Google Ads에 자동 업로드 → 소액 예산 균등 배분.

**성과 모니터링 자동화**: 24시간 단위 성과 수집(CTR, CPC, ROAS) → 하위 30% 자동 OFF → 상위 30%에 예산 자동 재배분.

**학습 루프**: 매주 성과 데이터 AI 분석 → "이런 표현이 타겟에게 효과적" 패턴 축적 → 다음 카피 생성 시 반영.

**주간 리포트**: 카피별 성과, 트렌드 변화, 추천 액션 → 슬랙/이메일 자동 발송.
