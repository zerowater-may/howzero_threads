## 대본

### 훅 (0:00–0:40)
직원이 퇴사하면 채용·교육 비용으로 연봉의 50~200%가 날아갑니다. 문제는 퇴사 의사를 밝힌 시점엔 이미 늦다는 겁니다. 3개월 전에 징후를 잡았으면 막을 수 있었던 퇴사가 대부분입니다. AI로 퇴사 징후를 사전에 감지하고, 자동으로 리텐션 조치를 실행하는 시스템. 코딩 제로로 만듭니다.

### 문제 정의 (0:40–3:30)
HR 팀의 이직 관리 현실. 첫째, 사후 대응 — 퇴사 의사 표명 후 면담. 이미 결심한 사람을 붙잡는 건 성공률 20% 이하. 둘째, 데이터 사각 — 출퇴근 기록, 성과 데이터, 설문 결과가 각각 다른 시스템에 흩어져 있어 종합 분석 불가. 셋째, 관리자 감(感) 의존 — "요즘 표정이 안 좋더라"로 판단. 팀원 10명 넘으면 파악 불가.

### 자동화 설계 (3:30–10:00)
**데이터 통합**: 출퇴근 기록(야근 빈도, 지각 증가) + 성과 데이터(KPI 달성률 변화) + 내부 커뮤니케이션(슬랙 활동량 변화) + 휴가 사용(연차 몰아쓰기) + 설문(만족도 점수 추이) → 구글 시트로 자동 수집.

**AI 이직 위험도 스코어링**: 수집 데이터를 GPT-4가 분석 → 직원별 이직 위험도 점수(0~100) 산출 → 위험 요인 Top 3 자동 분석("야근 50%↑, 만족도 20%↓, 슬랙 활동 30%↓") → 주간 리포트 자동 생성.

**자동 리텐션 트리거**: 위험도 70점 이상 → HR 매니저에게 자동 알림 + 해당 직원 프로필·위험 요인 요약 제공 → 1:1 면담 일정 자동 제안 → 면담 후 조치 사항 자동 추적(급여 조정, 부서 이동, 교육 지원 등).

**퇴사 후 분석**: 퇴사자 데이터 자동 분석 → 퇴사 사유 패턴 분류 → 분기별 리텐션 리포트 자동 생성 → 조직 전체 이직률 트렌드 시각화.

### 데모/사례 (10:00–14:30)
