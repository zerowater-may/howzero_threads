## 대본

### 훅 (0:00–0:40)
마케팅 담당자가 가장 싫어하는 업무가 뭔지 아십니까. 성과 리포트 만들기입니다. 네이버 광고, 구글 애즈, 메타 광고, 카카오 모먼트 — 4개 플랫폼에서 데이터 뽑아서 엑셀에 정리하고, 차트 그리고, ROAS 계산하고. 매주 반나절을 이것에 씁니다. AI로 10분에 끝내는 방법을 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
마케팅 성과 분석의 3대 병목. 첫째, 데이터 산재 — 광고 플랫폼마다 대시보드가 다르고, 지표 정의도 다름. 수동으로 통합하면 오류 발생률 15%. 둘째, 분석 시간 — 주간 리포트 작성에 평균 3~4시간. 월간 리포트는 하루 꼬박. 이 시간에 다음 캠페인을 기획해야 합니다. 셋째, 인사이트 부재 — 숫자는 정리하지만 "왜 이 캠페인이 잘 됐는지", "다음에 뭘 바꿔야 하는지"는 경험에 의존.

### 자동화 설계 (3:00–8:00)
**데이터 자동 수집**: Make로 각 광고 플랫폼 API 연동 → 매일 오전 9시 전일 데이터 자동 수집 → 구글 시트에 통합 정리. 노출수, 클릭수, 전환수, 비용, ROAS, CPC, CTR 자동 계산.

**AI 성과 분석**: 주간 데이터가 쌓이면 GPT-4가 자동 분석 — 캠페인별 성과 비교, 전주 대비 변화율, 이상 징후 탐지("CPC가 전주 대비 40% 상승 — 소재 피로도 의심"), 예산 재배분 제안.

**자동 리포트 생성**: 분석 결과를 구글 독스 템플릿에 자동 삽입 → 차트 자동 생성 → 슬랙으로 팀에 공유 + 이메일로 대표에게 발송. 매주 월요일 오전 10시, 리포트가 알아서 도착합니다.

**이상 알림**: 일일 모니터링 — ROAS가 기준치 이하로 떨어지면 즉시 슬랙 알림. 예산 소진 속도가 비정상이면 경고. 24시간 감시를 사람이 아닌 시스템이 합니다.

### 데모/사례 (8:00–13:30)
온라인 쇼핑몰 CM. 네이버+메타+구글 월 광고비 1,500만 원. 자동화 전: 주간 리포트 4시간, 월간 리포트 8시간, 이상 탐지 수동(매일 플랫폼별 확인). 자동화 후: 리포트 생성 시간 4시간→10분(자동). 이상 탐지 평균 대응 시간 48시간→2시간. ROAS 개선 18%(빠른 예산 재배분 덕분). 마케터가 분석 대신 기획에 집중 → 신규 캠페인 수 월 2개→5개.

도구: 구글 시트 + Make + GPT-4 + 구글 독스 + 슬랙. 월 운영비 약 8만 원.
