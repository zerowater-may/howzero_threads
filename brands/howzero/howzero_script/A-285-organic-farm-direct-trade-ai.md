### 자동화 설계 (3:00–8:00)
**통합 주문 자동 관리**: 카카오톡·네이버 스토어·자사몰 주문 → 하나의 시트에 자동 통합 → 배송지·품목·수량 자동 정리 → 송장 자동 생성 → 택배사 자동 접수.

**정기 구독 자동화**: 구독 고객 DB 자동 관리 → 매주/격주 배송 일정 자동 생성 → 수확 가능 품목에 맞춰 꾸러미 자동 구성 → "이번 주 꾸러미: 방울토마토, 상추, 오이" 자동 안내 → 결제 자동 처리.

**수확-배송 자동 연동**: 수확 예정일 등록 → 수확 완료 체크 → 당일 오후 택배 자동 접수 → 고객에게 "오늘 아침 수확, 내일 도착" 자동 알림 → 신선도 스토리 자동 마케팅.

**시즌 마케팅 자동화**: 작물별 수확 시기 캘린더 → 수확 2주 전 사전 예약 자동 오픈 → 기존 구매 고객 자동 알림 → "제철 블루베리 사전 예약 시작" → 완판율 증가.
