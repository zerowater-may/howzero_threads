## 대본

### 훅 (0:00–0:40)
발렌타인데이 마케팅을 2월 10일에 기획하고 있으면 이미 늦은 겁니다. 경쟁사는 1월에 끝냈습니다. 문제는 연간 시즌 이벤트가 30개 넘는데, 매번 "이번엔 뭐 하지?"부터 시작한다는 겁니다. AI로 연간 마케팅 캘린더 자동 생성부터 시즌별 캠페인 기획, 콘텐츠 제작, 성과 분석까지 한 번에. 코딩 제로입니다.

### 문제 정의 (0:40–3:30)
시즌 마케팅의 3대 병목. 첫째, 기획 지연 — 시즌 이벤트 때마다 뒤늦게 허둥지둥. 사전 준비 부재. 둘째, 콘텐츠 제작 — 시즌마다 배너, SNS 포스팅, 이메일 등 제작물 쏟아짐. 디자이너·마케터 과부하. 셋째, 성과 미측정 — "작년 크리스마스 캠페인 매출이 얼마였더라?" 기록 안 남아서 매년 처음부터.

### 자동화 설계 (3:30–10:00)
**연간 마케팅 캘린더 자동 생성**: 업종+타겟 고객 입력 → AI가 연간 시즌 이벤트 캘린더 자동 생성(설날, 발렌타인, 화이트데이, 어린이날, 여름 바캉스, 추석, 블랙프라이데이, 크리스마스 등 30+개) → 이벤트별 준비 시작일·실행일·종료일 자동 산출 → 구글 캘린더 자동 등록.

**캠페인 기획 자동화**: 각 시즌 이벤트 D-30 자동 트리거 → AI가 캠페인 초안 생성(테마, 할인율 제안, 타겟 고객, 채널별 콘텐츠 방향) → 작년 동일 시즌 성과 데이터 자동 반영 → 마케터가 검수·확정.

**콘텐츠 자동 생성**: 확정된 캠페인 기반 → AI가 SNS 포스팅 문구(인스타·페이스북·카카오) 자동 생성 → 이메일 뉴스레터 초안 자동 생성 → 배너 카피 자동 생성 → 예약 게시 자동 등록.

**성과 자동 분석**: 캠페인 종료 시 자동 성과 리포트 생성(매출 기여, 트래픽, 전환율, ROI) → 작년 대비 성장률 자동 계산 → 인사이트 자동 추출("올해 발렌타인 — 카카오 채널 전환율이 인스타 대비 3배. 내년 카카오 비중↑ 권장") → 다음 해 캘린더에 자동 반영.
