### 훅 (0:00–0:40)
사진관 운영하시는 분들, 촬영 말고 가장 많은 시간 쓰는 업무가 뭡니까. 사진 보정입니다. 촬영 1시간, 보정 4시간. 여기에 예약 관리, 고객 응대, 파일 납품까지 더하면 하루가 어떻게 가는지 모릅니다. AI로 예약부터 자동 보정, 납품까지 시스템화한 사진관 사례를 보여드리겠습니다. 코딩 제로입니다.

### 문제 정의 (0:40–3:30)
사진관/스튜디오의 3대 병목. 첫째, 예약 관리 — 카카오톡, 전화, 인스타 DM 세 군데서 예약이 들어옴. 겹침 사고 월 2~3건. 둘째, 사진 보정 — 한 건 촬영에 보정 시간 3~5시간. 증명사진 배경 제거, 피부 보정, 색감 통일 등 반복 작업. 셋째, 납품·후속 관리 — 파일 정리, 고객에게 다운로드 링크 전송, 리뷰 요청까지 수작업. 고객 100명이면 100번 반복.

### 자동화 설계 (3:30–9:30)
**예약 통합 자동화**: 카카오·인스타·네이버 예약을 구글 캘린더 한 곳으로 통합 → 촬영 유형별 소요 시간 자동 배정(증명 30분, 프로필 1시간, 가족 2시간) → 예약 확정 시 고객에게 준비물 안내 자동 발송 → D-1 리마인드.

**AI 자동 보정**: 촬영 원본을 폴더에 드롭 → AI가 촬영 유형 인식(증명/프로필/웨딩) → 유형별 프리셋 자동 적용(배경 제거, 피부 보정, 색감 보정) → 보정 결과물 미리보기 폴더 자동 생성 → 사진사가 최종 검수(5분).

**납품 자동화**: 보정 완료 시 고객에게 비밀번호 걸린 다운로드 링크 자동 전송 → 다운로드 확인 후 3일 뒤 리뷰 요청 자동 발송 → 리뷰 작성 시 다음 촬영 10% 할인 쿠폰 자동 발급.

**매출 분석**: 촬영 유형별·월별 매출 자동 집계 → 재방문 고객 비율 추적 → 비수기 자동 프로모션 트리거(지난달 매출 20%↓이면 기존 고객에게 할인 안내).
