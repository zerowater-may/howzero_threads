## 대본

### 훅 (0:00–0:40)
유치원 원장님, 선생님들이 아이 돌보는 시간보다 알림장 쓰는 시간이 더 길지 않습니까? 알림장, 출결 관리, 학부모 연락, 식단표, 행사 안내… 행정 업무가 교육의 질을 갉아먹고 있습니다. AI로 행정 업무의 60%를 자동화하고, 선생님들이 진짜 아이들에게 집중할 수 있게 만든 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
유치원/어린이집 행정 병목 네 가지. 첫째, 알림장 — 원아 20명이면 매일 20개의 개별 알림장. 선생님 1인당 일 40분~1시간 소요. 둘째, 출결 — 등/하원 시간 기록, 결석 사유 확인, 보육료 정산 연동. 셋째, 학부모 소통 — 행사, 준비물, 긴급 연락 등 채널 분산(전화, 문자, 카카오). 넷째, 서류 — 보육일지, 관찰 기록, 안전 점검표 등 관공서 제출 서류.

### 자동화 설계 (3:00–8:30)
**알림장 자동 생성**: 선생님이 오늘 활동 키워드만 입력(예: 미술놀이, 바깥놀이, 급식 잘 먹음) → AI가 원아별 맞춤 알림장 자동 작성(원아 특성, 전날 내용과 겹치지 않게) → 선생님 확인 후 카카오톡 자동 발송.

**출결 자동화**: 등원 시 QR/NFC 태그 → 자동 출결 기록 → 미등원 시 학부모에게 자동 확인 연락 → 월말 보육료 자동 정산.

**학부모 통합 소통**: 행사 안내, 준비물, 식단표 → 카카오 알림톡으로 일괄 자동 발송 → 회신 자동 수집.

**서류 자동화**: 일일 보육일지, 월간 관찰 기록 → 알림장+활동 데이터 기반 AI 자동 초안 생성 → 선생님 확인 후 저장.

### 데모/사례 (8:30–13:30)
경기 소재 어린이집 R. 원아 60명, 교사 6명. 자동화 전: 알림장 작성 교사 1인당 일 50분, 보육일지 주 2시간, 학부모 개별 연락 일 30분. 자동화 후: 알림장 작성 일 50분 → 10분(키워드만 입력). 보육일지 주 2시간 → 20분. 학부모 연락 자동화로 일 5분. 교사 만족도 대폭 상승 → 이직률 40%→15%.

도구: 구글 시트 + Make + GPT-4 + 카카오 알림톡. 월 운영비 약 5만 원.
