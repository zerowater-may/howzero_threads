## 대본

### 훅 (0:00–0:40)
럭셔리 리조트 운영하시는 분, 객단가를 올리고 싶으시죠? 럭셔리의 본질은 "나를 알아주는 경험"입니다. 그런데 100명의 게스트 취향을 직원이 전부 기억할 수 없습니다. AI가 게스트별 선호도를 전부 기억하고, 도착 전부터 맞춤 경험을 설계하고, 체류 중 업셀링을 자연스럽게 합니다. 객단가 55% 상승. 럭셔리는 기억력입니다.

### 문제 정의 (0:40–3:00)
럭셔리 숙소의 특수한 병목. 첫째, 게스트 프로필 관리 — 음식 선호, 알레르기, 베개 종류, 룸 온도, 와인 취향. 재방문 게스트가 "또 물어봐야 해요?" 하면 럭셔리가 아닙니다. 둘째, 업셀링 기회 — 스파, 다이닝, 투어, 와인 등 부대시설 매출이 객실 매출을 넘길 수 있지만, 안내가 어색하거나 타이밍을 놓칩니다. 셋째, 컨시어지 과부하 — 성수기에 요청이 폭주하면 응대 품질이 떨어집니다. 넷째, 후기 관리 — 럭셔리 고객은 후기에 엄격합니다. 사소한 불만이 치명적.

### 자동화 설계 (3:00–8:30)
**게스트 프로필 자동 학습**: 첫 예약 시 사전 설문(음식 선호·알레르기·목적·동행자·특별 요청) → 체류 중 주문·이용 기록 자동 축적 → 재방문 시 AI가 프로필 브리핑 자동 제공("김OO 고객: 락 베개, 룸 온도 22도, 씨푸드 알레르기, 화이트 와인 선호").

**맞춤 경험 자동 설계**: 체크인 2일 전 → 게스트 프로필 기반 웰컴 세팅 자동 지시(선호 과일·음료·룸 세팅) → 체류 일정 기반 맞춤 추천("둘째 날 저녁, 선셋 디너 코스 예약해드릴까요? 지난번 좋아하셨던 시그니처 스테이크 포함입니다") → 기념일 자동 감지+축하 이벤트.

**스마트 업셀링**: 체크인 당일 → 스파 할인 안내(오후 3시, 룸 세팅 기다리는 시간) → 체류 2일 차 → 프라이빗 투어 추천(날씨 좋은 날 자동 감지) → 마지막 날 → 선물 패키지 추천 → 모든 업셀링이 자연스럽게 콘시어지 메시지로 전달.

**AI 컨시어지 보조**: 게스트 요청 접수(인앱/카카오톡) → 일반 요청 AI 즉시 처리(룸서비스·택시·관광지 정보) → 특별 요청만 담당 직원 연결 → 요청 처리 상태 게스트에게 실시간 자동 안내.

**후기 모니터링+선제 대응**: 체류 중 간단 만족도 체크(체크인 4시간 후 "불편한 점 없으세요?") → 불만 감지 시 즉시 관리자 알림 → 체크아웃 전 해결 → 체크아웃 후 리뷰 요청(만족 고객만 선별).
