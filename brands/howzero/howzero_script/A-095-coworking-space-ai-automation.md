### 문제 정의 (0:40–3:00)
공유오피스 운영의 4대 병목. 첫째, 입주 문의·계약 — 문의 응대, 투어 스케줄링, 계약서 관리가 수동. 둘째, 시설 예약 — 회의실, 세미나실 예약 충돌, 노쇼 관리. 셋째, 시설 관리 — 냉난방, 프린터, 정수기 등 민원 접수+처리. 넷째, 정산 — 입주사별 사용량(인쇄, 회의실, 추가 서비스) 집계+청구.

### 자동화 설계 (3:00–8:00)
**입주 문의 자동화**: 웹사이트 챗봇으로 24시간 문의 응대 → 관심 고객 정보 자동 수집 → 투어 일정 자동 제안 → 계약 진행 시 전자 계약서 자동 생성.

**예약 시스템**: 입주자 전용 앱/웹에서 회의실 실시간 예약 → 30분 전 자동 리마인드 → 10분 노쇼 시 자동 취소+대기자 알림.

**시설 민원 AI**: 카카오톡으로 민원 접수 → AI가 유형 분류(시설, 청소, 택배 등) → 담당자에게 자동 배정 → 처리 완료 시 자동 알림.

**정산 자동화**: 출입 기록 + 회의실 사용 + 인쇄량 자동 집계 → 입주사별 월 청구서 자동 생성 → 자동 발송.
