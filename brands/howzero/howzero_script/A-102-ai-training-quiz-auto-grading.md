## 대본

### 훅 (0:00–0:40)
사내 교육, 형식적으로 하고 있지 않습니까? PPT 보여주고, 참석 확인하고, 끝. 교육 효과를 측정하는 회사는 10%도 안 됩니다. AI로 교육 자료 기반 퀴즈를 자동 생성하고, 자동 채점하고, 부족한 부분을 자동으로 추가 학습시키는 시스템. 교육 효과가 3배 올라갑니다.

### 문제 정의 (0:40–3:00)
사내 교육의 3대 문제. 첫째, 교육 자료 제작 — 매번 새로운 교육 콘텐츠를 만드는 건 HR 담당자에게 큰 부담. 둘째, 효과 측정 부재 — 교육 후 뭘 배웠는지 확인 안 함. 셋째, 개인별 차이 — 같은 교육을 들어도 이해도가 다른데 일괄 진행.

### 자동화 설계 (3:00–8:00)
**퀴즈 자동 생성**: 교육 자료(PDF, PPT, 영상 스크립트) 업로드 → AI가 자동으로 — 객관식, 주관식, OX 퀴즈 생성 + 난이도 조절(초급/중급/고급) + 해설 자동 작성.

**자동 채점**: 구글 폼으로 응시 → 객관식 즉시 채점 + 주관식은 AI가 키워드+맥락 기반 채점 + 점수 자동 집계.

**개인별 맞춤 학습**: 오답 분석 → 취약 영역 자동 파악 → 해당 영역 보충 자료 자동 추천/발송.

**리포트 자동화**: 부서별, 개인별 교육 이수율·평균 점수·취약 영역 대시보드 자동 생성 → HR에게 월간 리포트 자동 발송.

### 데모/사례 (8:00–12:30)
제조업 T사. 직원 120명. 분기 1회 안전 교육 + 월 1회 직무 교육. 자동화 전: 퀴즈 출제 건당 3시간, 채점 건당 4시간, 교육 효과 측정 없음. 자동화 후: 퀴즈 출제 10분(AI 자동), 채점 즉시(자동), 개인별 취약 영역 파악+보충 학습. 안전사고 발생률 전년 대비 45%↓(교육 이해도 향상).

도구: 구글 폼 + 구글 시트 + Make + GPT-4. 월 운영비 약 5만 원.
