## 메타데이터
- **제목**: AI로 특허 침해 자동 모니터링 — 침해 발견, 빨라야 이긴다
- **썸네일 텍스트**: 특허 침해 AI | 자동 감지
- **태그**: 특허, 지식재산, 침해 모니터링, IP 관리, AI 자동화, 하우제로
## 대본

### 훅 (0:00–0:40)
특허 있으신 분들, 누가 침해하고 있는지 어떻게 확인하세요? 대부분 모릅니다. 경쟁사가 우리 특허를 슬쩍 베껴도, 발견하는 건 보통 1~2년 후. 그때는 이미 시장을 뺏긴 후입니다. AI가 매주 전 세계 특허 출원, 제품 출시, 논문을 스캔해서 우리 특허 침해 가능성을 자동 감지합니다.

### 문제 정의 (0:40–3:00)
특허 관리의 병목. 첫째, 침해 발견 — 전 세계 특허 출원 연 350만 건. 수동 모니터링 불가능. 둘째, 유사도 판단 — 청구항 대 청구항 비교는 전문 변리사가 필요. 비용 높고 시간 오래 걸립니다. 셋째, 방어 전략 — 우리 특허가 침해당하는 것뿐 아니라, 우리가 타사 특허를 침해하고 있을 수도 있습니다.

### 자동화 설계 (3:00–8:30)
**특허 출원 자동 모니터링**: KIPRIS(한국), USPTO(미국), EPO(유럽) 등 주요 특허청 DB 자동 검색 → 우리 특허 핵심 키워드 + 기술 분류 기반 신규 출원 자동 수집 → 주 1회 리포트.

**AI 유사도 분석**: 수집된 신규 출원 → AI가 우리 특허 청구항과 유사도 자동 분석 → 유사도 70% 이상 자동 플래그 → 변리사 검토 우선순위 리스트 자동 생성.

**경쟁사 제품 모니터링**: 경쟁사 신제품 출시, 보도자료, 전시회 발표 자동 수집 → 우리 특허 기술 적용 가능성 AI 분석 → 이상 감지 시 알림.

**IP 포트폴리오 관리 자동화**: 보유 특허 만기일·유지료 납부 자동 알림 → 특허별 활용도(라이선스, 제품 적용) 자동 추적 → 연간 IP 전략 리포트 자동 생성.

### 데모/사례 (8:30–13:00)
의료기기 스타트업 T. 특허 12건 보유. 자동화 전: 변리사에게 연 2회 선행 기술 조사 의뢰(건당 200만 원), 침해 발견은 우연히(전시회에서 경쟁사 제품 발견), 특허 유지료 납부 누락 1건(특허 소멸 위기). 자동화 후: 주간 자동 모니터링, 유사도 분석 자동, 유지료 납부 알림 자동.

6개월간 유사 출원 23건 자동 감지 → 그 중 유사도 80% 이상 3건 → 변리사 검토 후 1건 정보 제공 의견서 제출 → 경쟁사 출원 거절. 변리사 비용 연 70% 절감(AI가 사전 필터링).

도구: Make + GPT-4 + 구글 시트 + 슬랙 + KIPRIS API. 월 운영비 약 8만 원.
