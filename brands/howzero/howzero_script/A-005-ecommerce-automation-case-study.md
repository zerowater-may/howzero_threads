## [5:00~10:00] 자동화 구축 과정 (5단계)

**[화면: 실제 자동화 화면 녹화]**

"이 회사에 적용한 자동화 5가지를 순서대로 보여드리겠습니다.

**1. CS 문의 자동 분류 + 답변 초안 (세팅 15분)**
(화면: 데모)
카톡/이메일 문의 → AI가 환불/배송/제품/기타 분류 → 답변 초안 생성 → 담당자 알림
결과: 건당 5분 → 30초

**2. 리뷰 자동 수집 + 감성 분석 (세팅 20분)**
(화면: 데모)
쿠팡/스마트스토어 리뷰 자동 크롤링 → 긍정/부정/중립 분류 → 부정만 즉시 알림
결과: 주 5시간 → 0시간 (알림만 확인)

**3. 환불 요청 자동 접수 (세팅 10분)**
환불 키워드 감지 → 주문 정보 자동 조회 → 환불 조건 체크 → 승인/거절 제안
결과: 건당 15분 → 2분

**4. 경쟁사 가격 자동 모니터링 (세팅 15분)**

결과: 주 2시간 → 0시간 (알림만)

**5. 일일 매출/CS 자동 보고서 (세팅 10분)**
매일 저녁 6시 자동 집계 → AI 요약 → 카톡/이메일 자동 발송
