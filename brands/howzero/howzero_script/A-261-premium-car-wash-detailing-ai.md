## 대본

### 훅 (0:00–0:40)
프리미엄 세차장 운영하시는 분, 일반 세차만 하는 고객을 코팅·디테일링까지 업셀링하고 싶으시죠? 그런데 바쁘면 안내를 못 하고, 고객 차량 상태를 매번 기억 못 합니다. AI가 차량별 세차 기록, 코팅 유지 상태, 다음 관리 시점을 전부 추적하고, 최적 타이밍에 맞춤 서비스를 자동 추천합니다. 객단가 40% 상승. 프리미엄은 기억력입니다.

### 문제 정의 (0:40–3:00)
프리미엄 세차의 병목. 첫째, 예약 관리 — 차종별 소요 시간이 다르고(경차 30분, SUV 1시간, 디테일링 3시간), 베이 배정이 복잡합니다. 둘째, 차량 기록 — 고객 차량의 코팅 종류, 코팅 시점, 특수 관리 사항을 기억할 수 없습니다. 셋째, 업셀링 기회 누락 — 코팅 효과 감소 시점, 시즌별 관리 필요 시점에 안내를 못 합니다. 넷째, 기상 변수 — 비 예보 시 예약 취소가 급증하지만 대응이 느립니다.

### 자동화 설계 (3:00–8:00)
**스마트 예약**: 차종+서비스 종류 선택 → 소요 시간 자동 산정 → 베이 배정 자동 최적화 → 기상 예보 연동(비 예보 시 실내 서비스 자동 전환 또는 일정 변경 제안) → 24시간 전 리마인더.

**차량 프로필 자동 관리**: 첫 방문 시 차량 정보 수집(차종·색상·코팅 이력·주차 환경) → 매 방문 서비스 기록 자동 저장(서비스 종류·사용 제품·작업 사진) → 코팅 유효기간 자동 추적.

**맞춤 업셀링**: 코팅 후 6개월 시점 → "코팅 유지 관리 시기입니다. 코팅 부스터 서비스 20% 할인" 자동 발송 → 겨울 시즌 → "하부 세차+염화칼슘 제거 패키지" 자동 추천 → 세차 3회차 방문 시 → "프리미엄 디테일링 첫 체험 30% 할인" 업셀링.

**작업 품질 관리**: 작업 전/후 사진 자동 촬영+보관 → 고객에게 비포/애프터 자동 발송 → 서비스 만족도 자동 수집 → 불만족 시 즉시 CS 연결.
