## 대본

### 훅 (0:00–0:40)
북카페 운영하시는 분들, 신간 입고를 어떻게 결정하십니까. "베스트셀러 위주로", "내 취향대로". 문제는 고객 취향과 안 맞는다는 겁니다. 입고한 책의 40%가 한 달간 한 번도 안 읽힙니다. AI로 고객 데이터 기반 입고 결정부터 맞춤 도서 추천, 독서 이벤트 자동 기획까지. 코딩 제로입니다.

### 문제 정의 (0:40–3:30)
도서관·북카페의 3대 병목. 첫째, 도서 선정 — 신간 수천 종 중 뭘 입고할지 감으로 결정. 비인기 도서 재고 쌓임. 둘째, 도서 추천 — "이런 책 있나요?" 직원 개인 역량 의존. 고객별 맞춤 추천 불가. 셋째, 이벤트·프로그램 — 독서 모임, 저자 강연 등 기획·홍보·참가자 관리 전부 수작업.

### 자동화 설계 (3:30–9:30)
**AI 기반 도서 입고**: 대출/열람 데이터 분석 → 장르별·주제별 수요 트렌드 파악 → 신간 리스트에서 우리 고객 취향 매칭 점수 자동 산출 → 입고 추천 리스트 자동 생성(예상 열람 횟수 포함) → 담당자 검수 후 발주.

**맞춤 도서 추천**: 회원별 대출·열람 이력 분석 → AI가 개인별 관심 프로필 자동 생성 → 신간 입고 시 관심 매칭 회원에게 자동 알림("고객님이 좋아하실 만한 새 책이 들어왔습니다") → 추천 수락률 자동 추적.

**이벤트 자동 기획·운영**: 인기 도서·장르 데이터 기반 독서 모임 주제 자동 제안 → 참가 신청 폼 자동 생성 → 참가자에게 리마인드·안내 자동 발송 → 이벤트 후 만족도 설문 자동 → 다음 이벤트 기획에 피드백 반영.

**운영 대시보드**: 일별 방문자 수, 인기 도서 Top 10, 좌석 이용률, 음료 매출 자동 집계 → 월간 운영 리포트 자동 생성.
