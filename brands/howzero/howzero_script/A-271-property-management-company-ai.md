### 문제 정의 (0:40–3:00)

### 자동화 설계 (3:00–8:00)
**임대료 자동 수납 관리**: 계좌 입금 자동 매칭(입금자명+금액+호수) → 미입금 호수 자동 감지 → D+1 자동 리마인더 → D+7 공식 독촉 → D+30 경고 → 단계별 자동 에스컬레이션 → 수납 현황 대시보드 실시간.

**공실 자동 마케팅**: 퇴거 예정일 2개월 전 자동 감지 → 매물 정보 자동 생성(사진+면적+가격+위치 설명) → 네이버 부동산·직방·피터팬 자동 등록 → 문의 AI 자동 응대(24시간) → 현장 안내 일정 자동 조율.

**시설 관리 자동 접수**: 임차인 카카오톡/웹폼으로 하자 접수 → AI가 유형 자동 분류(배관·전기·도배·기타) → 해당 업체 자동 배정 → 방문 일정 자동 조율 → 수리 완료 후 임차인 만족도 자동 확인.

**계약 라이프사이클 자동화**: 만기 6개월 전 자동 감지 → 시세 기반 갱신 조건 자동 산출 → 갱신 안내 자동 발송 → 갱신/해지 의사 수집 → 해지 시 퇴거 프로세스+공실 마케팅 자동 시작.
