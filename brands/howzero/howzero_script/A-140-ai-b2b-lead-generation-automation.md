## 대본

### 훅 (0:00–0:40)
B2B 세일즈에서 가장 힘든 단계가 뭔지 아십니까. 미팅이 아닙니다. 미팅 잡기입니다. 잠재 고객 찾기 → 연락처 확보 → 콜드 이메일 작성 → 팔로업. 이 과정에 세일즈 담당의 시간 60%가 들어갑니다. AI로 이 과정을 자동화하면 세일즈 담당은 미팅과 클로징에만 집중할 수 있습니다. 월 미팅 수 3배 늘린 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:30)
B2B 리드 발굴의 3대 병목. 첫째, 타겟 리스트 구축 — "우리 고객이 될 만한 회사"를 찾아서 담당자 이름, 직책, 이메일을 수집하는 데 주 10시간+. 둘째, 개인화 메시지 — 같은 템플릿을 100명에게 보내면 응답률 1%. 회사별 맞춤 메시지를 써야 하는데, 건당 15분. 셋째, 팔로업 — 첫 이메일에 응답하는 비율 5% 미만. 3~5회 팔로업이 필요한데 수동으로는 관리 불가.

### 자동화 설계 (3:30–9:30)
**AI 타겟 리스트 자동 구축**: 이상적 고객 프로필(ICP) 정의 — 업종, 규모, 지역, 매출 규모 → AI가 링크드인에서 조건에 맞는 회사·담당자 자동 탐색 → 회사 웹사이트, 뉴스, SNS 자동 스캔으로 연락처 + 컨텍스트 수집 → 구글 시트에 타겟 리스트 자동 구축.

**AI 개인화 콜드 이메일**: 타겟 회사의 최근 뉴스, 채용 공고, SNS 게시물을 AI가 분석 → 개인화 포인트 자동 추출("최근 물류 센터를 확장하셨더군요") → GPT-4가 맞춤 이메일 초안 생성 → 검토 후 발송.

**자동 팔로업 시퀀스**: 첫 이메일 → 3일 후 미응답 시 2차 이메일(다른 각도) → 7일 후 3차(사례 공유) → 14일 후 4차(마지막 기회) → 응답 감지 시 시퀀스 자동 중단 + 담당자 알림.

**링크드인 연동**: 이메일 시퀀스와 병행 — 링크드인 연결 요청 + 맞춤 메시지 → 연결 승인 시 가치 제공 메시지(인사이트 공유) → 미팅 제안.

**파이프라인 대시보드**: 리드 단계별 현황(발굴→접촉→응답→미팅→계약) 자동 추적 → 전환율 자동 계산 → 병목 단계 자동 식별.

### 데모/사례 (9:30–14:30)
