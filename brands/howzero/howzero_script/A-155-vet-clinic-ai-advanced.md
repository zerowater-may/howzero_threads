## 대본

### 훅 (0:00–0:40)
동물병원 원장님, 진료 기록을 언제 쓰십니까. 진료 중에요, 아니면 진료 후에요? 대부분 진료 후에 기억에 의존해서 씁니다. 하루 30마리 보면, 저녁에 기록하는 것만 1시간 넘습니다. 빠진 내용도 생기고. AI로 진료 중 음성 기록, 접종 스케줄 자동 관리, 처방 이력 추적까지 — 수의사가 진료에만 집중하는 시스템을 보여드리겠습니다.

### 문제 정의 (0:40–3:30)
동물병원의 3대 병목. 첫째, 진료 기록 — 진료 중 메모할 시간이 없고, 끝나고 쓰면 누락. 차트 정리에 일 1~2시간. 둘째, 접종 관리 — 강아지 종합 접종 5회, 광견병 매년, 심장사상충 매월 — 환자 200마리의 접종 스케줄을 수동 관리하면 누락 필연. 보호자가 까먹고, 병원도 안내 못함. 셋째, 처방·약 관리 — 환자별 처방 이력, 약물 알레르기, 복용 중인 약 확인이 차트를 뒤져야 가능.

### 자동화 설계 (3:30–9:30)
**AI 진료 기록**: 진료 중 음성 녹음(간단한 마이크) → Whisper API로 자동 텍스트 변환 → GPT-4가 진료 기록 포맷으로 정리(주증상, 검사 소견, 진단, 처방, 다음 방문 계획) → 수의사 확인 후 저장. 진료 기록 시간 1시간→10분.

**접종 스케줄 자동 관리**: 환자 등록 시 종, 나이, 접종 이력 입력 → AI가 개별 접종 스케줄 자동 생성 → 접종일 7일 전 보호자에게 카카오 알림("멍멍이 종합 접종 2차 시기입니다") → 예약 링크 포함 → 미방문 시 재안내.

**처방 이력 통합**: 환자별 전 처방 이력 구글 시트 관리 → 진료 시 자동 조회 → 약물 상호작용 AI 체크 → 동일 처방 반복 시 자동 알림 → 보호자에게 투약 안내(시간, 용량, 주의사항) 자동 발송.

**보호자 케어**: 수술/시술 후 회복 가이드 자동 발송 → 3일 후 경과 확인 메시지 → 건강검진 시기 자동 안내(7세+ 시니어 반려동물) → 보호자 만족도 조사.

### 데모/사례 (9:30–14:30)
동물병원 VT. 수의사 2명, 테크니션 3명, 등록 환자 350마리, 일 진료 35마리. 자동화 전: 진료 기록 일 1.5시간, 접종 안내 수동(누락 월 20건+), 처방 이력 차트 뒤적임(건당 3분), 보호자 팔로업 거의 없음. 자동화 후: 진료 기록 AI 자동화로 1.5시간→15분. 접종 안내 자동화로 접종 이행률 60%→88%. 처방 이력 즉시 조회(건당 10초). 보호자 케어로 재방문율 45%→68%.
