## 대본

### 훅 (0:00–0:40)
아이돌봄 서비스 운영하시는 분, 매칭 한 건에 시간 얼마나 드세요? 부모님 요구사항 확인하고, 가능한 시터 찾고, 스케줄 맞추고, 소개하고. 건당 1시간은 걸리죠. AI가 부모 요구사항과 시터 프로필을 자동 매칭합니다. 매칭 시간 5분. 매칭 만족도 4.7점. 실제 사례입니다.

### 문제 정의 (0:40–3:00)
돌봄 매칭의 병목. 첫째, 매칭 복잡성 — 위치, 시간, 아이 연령, 특수 요구(영어, 예체능, 알레르기 대응), 시터 경험·자격을 모두 고려해야 합니다. 둘째, 신뢰 구축 — 부모님은 낯선 사람에게 아이를 맡기는 게 불안합니다. 충분한 정보 제공이 필요. 셋째, 스케줄 관리 — 정기/비정기, 긴급 요청, 시터 가용 시간 조율이 복잡.

### 자동화 설계 (3:00–8:00)
**AI 자동 매칭**: 부모님 요구사항(위치, 시간, 연령, 특수 요구) 입력 → AI가 시터 DB에서 적합도 점수 산출 → 상위 3명 자동 추천 → 프로필(경력, 자격증, 리뷰 요약) 자동 제공.

**신뢰 시스템 자동화**: 시터 등록 시 → 자격 서류 자동 검증 안내 → 이용 후 양방향 리뷰 자동 수집 → AI가 시터별 신뢰 점수 자동 갱신 → 이상 리뷰 자동 플래그.

**스케줄 자동 관리**: 정기 돌봄 → 자동 반복 예약 → 시터 불가 시 대체 시터 자동 매칭 → 긴급 요청 → 실시간 가용 시터 검색 → 자동 매칭.

**커뮤니케이션 자동화**: 매칭 확정 → 상호 연락처 교환 → 돌봄 당일 체크인/체크아웃 자동 기록 → 돌봄 리포트(활동, 식사, 특이사항) 자동 발송.

### 데모/사례 (8:00–12:30)
돌봄 매칭 서비스 D. 등록 시터 200명, 월 매칭 300건. 자동화 전: 매칭 담당자 3명, 건당 평균 45분, 매칭 실패(부모 거절) 30%, 긴급 요청 대응 평균 3시간. 자동화 후: 매칭 담당자 1명(예외 처리), 건당 5분(자동 매칭 + 부모 확인), 매칭 실패 12%(적합도 높은 추천), 긴급 요청 대응 30분.

월 매칭 300건 → 500건(처리 용량 증가). 부모 만족도 4.2 → 4.7. 시터 재등록율 70% → 88%.

도구: 구글 시트 + Make + GPT-4 + 카카오 알림톡 + 자체 DB. 월 운영비 약 12만 원.
