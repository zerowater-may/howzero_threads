## 대본

### 훅 (0:00–0:40)
골프연습장 사장님, 피크 타임에 타석 없어서 돌려보내는 고객이 하루 몇 명입니까. 반대로 오전 시간대에는 타석의 60%가 비어있죠. 타석 가동률이 매출의 전부인 비즈니스에서, 수동 관리는 돈을 버리는 겁니다. AI로 타석 예약, 회원 관리, 레슨 스케줄까지 자동화한 골프연습장 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
골프연습장/스포츠 시설의 3대 병목. 첫째, 타석(시설) 가동률 — 피크 타임 만석인데 비수기 시간대는 텅 빔. 가동률 편차가 매출을 깎음. 둘째, 회원 관리 — 시간권, 월정액, 가족 회원, 기업 회원 등 요금제 복잡. 만료·갱신 관리 수동. 셋째, 레슨 관리 — 프로 강사 스케줄, 레슨 예약, 진도 기록이 각각 분리.

### 자동화 설계 (3:00–8:00)
**스마트 타석 예약**: 카카오 챗봇으로 실시간 타석 현황 확인 + 예약 → 인기 시간대 자동 안내 + 비수기 할인 자동 적용("오전 10시 이전 30% 할인") → QR 체크인 → 이용 시간 자동 계산 → 종료 10분 전 알림.

**회원 관리 자동화**: 가입 시 이용 패턴 분석 → 최적 요금제 추천 → 자동 결제 → 만료 7일 전 자동 안내 → 3개월 미방문 시 복귀 프로모션 → 회원 등급(이용 빈도 기반) 자동 산정.

**레슨 관리 자동화**: 프로 강사 스케줄 + 타석 배정 + 수강생 예약 통합 관리 → 레슨 전날 리마인드 → 레슨 후 간단 피드백 자동 발송(강사가 키워드 입력 → AI가 문장으로 변환) → 수강생 진도 누적 기록.

**운영 분석**: 시간대별 가동률 분석 → 비수기 프로모션 자동 기획 → 회원 이탈 예측(이용 빈도 감소 패턴) → 월별 매출·회원 현황 리포트 자동.
