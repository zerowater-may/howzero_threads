### 문제 정의 (0:40–3:00)
스튜디오 운영의 3대 병목. 첫째, 예약 관리 — 전화·카톡으로 예약 받으면 더블 부킹, 노쇼, 변경 누락이 반복. 클래스당 정원이 6~12명인데 노쇼 1명이면 매출 직격탄. 둘째, 회원 이탈 — 3개월 이용권 끝나면 연장 안 하는 비율 40%+. 개별 케어를 해야 하는데 시간이 없음. 셋째, 수업 스케줄 — 강사 변경, 대체 수업, 특별 클래스 공지를 수동으로 하면 혼선.

### 자동화 설계 (3:00–8:30)
**AI 예약 시스템**: 카카오 챗봇으로 24시간 예약 접수 → 잔여석 실시간 확인 → 예약 확정 자동 발송 → 수업 2시간 전 리마인드 → 노쇼 시 자동 대기자 알림.

**출결 자동 관리**: QR 체크인 → 구글 시트에 자동 기록 → 월말 출석률 자동 집계 → 출석률 낮은 회원 자동 케어 메시지("○○님, 이번 달 아직 3회 남으셨어요. 이번 주 화요일 7시 수업 예약해드릴까요?").

**회원 이탈 방지**: 이용권 만료 14일 전 자동 안내 → 장기회원 혜택 제시 → 만료 후 7일간 재등록 프로모션 → 1개월 미방문 회원 자동 안부 메시지.

**수업 공지 자동화**: 강사 변경·휴무·특별 클래스 → 해당 수업 예약자에게만 자동 알림 → 대체 수업 추천 → 일괄 카카오 발송.
