## 대본

### 훅 (0:00–0:40)
요양시설 운영하시는 분, 직원들이 어르신 케어보다 서류 작성에 시간을 더 쓰고 있지 않으세요? 투약 기록, 식사량 기록, 건강 상태 일지, 보호자 보고서. 하루에 2~3시간입니다. AI가 돌봄 기록을 80% 자동 작성하고, 건강 이상 징후를 조기 감지하고, 보호자에게 일일 리포트를 자동 발송합니다. 직원이 어르신 곁에 있는 시간이 늘어납니다. 이게 핵심입니다.

### 문제 정의 (0:40–3:00)
요양시설의 고질적 병목. 첫째, 돌봄 기록 — 어르신별 투약, 식사, 활동, 건강 상태를 매일 기록해야 합니다. 법적 의무입니다. 하지만 수기 또는 단순 입력으로 시간이 엄청나게 듭니다. 둘째, 건강 이상 감지 — 혈압·체온·식사량 변화를 사람이 패턴으로 감지하기 어렵습니다. 이상이 뚜렷해진 후에야 대응합니다. 셋째, 보호자 소통 — 보호자는 매일 궁금합니다. "오늘 어머니 식사는 하셨나요?" 전화 문의가 하루 수십 통. 넷째, 인력 관리 — 교대 근무 스케줄, 인수인계가 복잡합니다.

### 자동화 설계 (3:00–8:30)
**돌봄 기록 자동화**: 직원이 태블릿에 음성으로 "김OO 어르신, 점심 80% 섭취, 혈압 130/85" → AI가 자동 구조화 기록 → 투약 체크리스트 자동 생성 → 미기록 항목 알림.

**건강 이상 조기 감지**: 어르신별 바이탈 데이터 일일 자동 수집 → AI가 개인별 정상 범위 학습 → 3일 연속 식사량 감소, 혈압 상승 추세 등 패턴 감지 → 간호사에게 즉시 알림 + 의사 상담 자동 연결.

**보호자 자동 리포트**: 일일 돌봄 기록 AI 요약 → 보호자 앱/카카오톡으로 자동 발송(식사·활동·건강 상태·오늘의 사진) → 보호자 질문에 AI 챗봇 1차 응답 → 상세 문의만 직원 연결.

**근무 스케줄+인수인계**: 직원 근무표 자동 생성(자격·선호·법정 근무시간 반영) → 교대 시 AI가 인수인계 요약 자동 생성(특이사항·주의 어르신·예정 일정).
