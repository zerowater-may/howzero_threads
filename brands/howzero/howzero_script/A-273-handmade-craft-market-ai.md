### 자동화 설계 (3:00–8:00)
**상품 등록 자동 지원**: 작가가 사진 업로드 → AI가 자동 보정(밝기·배경) + 상품 설명 초안 생성 + 카테고리·태그 자동 부여 + SEO 최적화 제목 제안 → 작가는 확인+수정만.

**고객 문의 자동 라우팅**: 일반 문의(배송·교환·반품) → AI 자동 응답 → 커스텀 문의("사이즈 변경 가능?") → 해당 작가에게 자동 전달 + 고객에게 "작가에게 확인 중" 자동 안내 → 작가 응답 → 고객에게 자동 전달.

**자동 정산**: 주문 완료 → 판매가 - 수수료 - 배송비 - 할인 자동 계산 → 작가별 월간 정산서 자동 생성 → 정산일에 자동 송금 트리거 → 세금계산서 자동 발행.

**AI 큐레이션+트렌드 매칭**: 고객 검색·구매 이력 분석 → 맞춤 상품 추천("손뜨개 좋아하시는 분, 이 작가 신상 확인해보세요") → 시즌 트렌드 자동 분석 → 트렌드에 맞는 작가에게 기획전 참여 자동 제안.
