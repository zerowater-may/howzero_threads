## 대본

### 훅 (0:00–0:40)
꽃집 사장님, 기념일 시즌에 주문 폭주하면 어떻게 되세요? 카카오톡으로 주문 받고, 수기로 적고, 꽃 재고 파악하고, 배송 일정 잡고… 꽃 만지는 시간보다 폰 보는 시간이 더 길어집니다. AI로 주문 접수부터 배송 완료 알림까지 자동화한 플라워샵, 기념일 시즌 매출이 2배가 됐습니다.

### 문제 정의 (0:40–3:00)
플라워샵 운영의 3대 병목. 첫째, 주문 관리 — 카톡, 전화, 인스타 DM으로 주문이 들어오는데 채널이 분산. 주문 누락, 내용 착오 빈번. 둘째, 재고 — 꽃은 유통기한이 짧아 재고 관리가 핵심. 감으로 발주하면 폐기 또는 품절. 셋째, 배송 — 시간대별 배송 스케줄링이 복잡.

### 자동화 설계 (3:00–8:00)
**주문 통합**: 카카오톡 챗봇 + 인스타그램 DM 자동 응답 → 주문 폼으로 유도(꽃 종류, 예산, 배송일, 메시지 카드 문구) → 구글 시트 주문 DB 자동 등록 → 결제 링크 자동 발송.

**재고 관리**: 입고 시 재고 등록 → 주문 확정 시 자동 차감 → 재고 부족 시 자동 경고 → 판매 데이터 기반 AI 발주 추천(요일별, 시즌별 패턴 분석).

**배송 스케줄링**: 주문 확정 → 배송 시간대별 자동 배분 → 배송 기사에게 자동 배차 → 배송 출발/완료 시 고객에게 자동 알림.

**기념일 리마인드**: 고객 DB에서 기념일 기록 → 7일 전 자동 알림("작년에 보내신 꽃다발, 올해도 보내실까요?") → 재주문 유도.
