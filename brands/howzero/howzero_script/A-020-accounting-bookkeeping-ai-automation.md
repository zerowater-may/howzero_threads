### [1:30~4:00] 문제 정의 — 소규모 사업자의 경리 현실

"소규모 사업자의 경리 업무를 정리하면 이렇습니다.

1. **통장 내역 정리** — 매일 or 매주. 어디서 뭐가 들어오고 나갔는지.
2. **항목 분류** — 이게 매출인지, 경비인지, 개인 지출인지.
3. **영수증 관리** — 종이든 사진이든 모아서 대조.
4. **매출/매입 집계** — 부가세 신고용.
5. **세무사 전달** — 정리된 자료를 보내는 것.

이 중에서 사장님이 직접 해야 하는 건 뭘까요?
솔직히 말하면 — 거의 없습니다.

1~4번은 전부 '규칙 기반 반복 작업'입니다.
AI가 가장 잘하는 영역이에요."

### [4:00~8:00] 자동화 설계 — 3개 시스템

"오늘 만들 시스템은 3개입니다.

**시스템 1: 통장 내역 자동 분류**
- 통장 CSV를 업로드하면
- AI가 각 거래를 '매출/매입/경비/개인' 자동 분류
- 분류 결과를 구글 시트에 정리

**시스템 2: 영수증 자동 인식**
- 영수증 사진을 찍으면
- AI가 날짜, 금액, 항목 자동 추출
- 통장 내역과 자동 매칭

**시스템 3: 월간 보고서 자동 생성**
- 매월 1일, 전월 데이터 자동 집계
- 매출/매입/경비 요약 보고서 생성
- 세무사에게 자동 이메일 전송

이 3개가 돌아가면 경리 업무의 90%가 자동입니다."
