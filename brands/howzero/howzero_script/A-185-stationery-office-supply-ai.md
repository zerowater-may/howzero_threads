### 문제 정의 (0:40–3:30)

### 자동화 설계 (3:30–9:30)
**상품 등록 자동화**: 공급사 카탈로그(PDF/엑셀) 업로드 → AI가 자동 파싱(상품명, 스펙, 가격, 이미지) → 카테고리 자동 분류 → 상품 설명 AI 자동 생성(SEO 최적화) → 쇼핑몰 일괄 등록.

**동적 가격 관리**: 경쟁사 주요 상품 가격 자동 모니터링(주 1회) → 가격 차이 자동 분석 → 가격 조정 제안("A4 용지 — 경쟁사 대비 5% 높음, 3% 인하 권장") → B2B 거래처별 단가 테이블 자동 관리 → 견적서 자동 생성(거래처 선택+상품 선택→즉시 PDF).

**재고·발주 최적화**: ABC 분석 자동 실행(A등급: 매출 80% 기여 상위 20% 상품) → A등급 상품 안전 재고 자동 관리 → 재주문점 도달 시 자동 발주 알림 → C등급 상품 중 6개월 미판매 자동 경고("단종 또는 할인 처리 권장").

**B2B 영업 자동화**: 거래처별 구매 이력 분석 → 재구매 주기 예측("거래처 X: A4 용지 평균 45일 주기 — 3일 후 재구매 예상") → 자동 재주문 제안 발송 → 신규 거래처 리드 자동 관리.
