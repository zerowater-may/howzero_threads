## 대본

### 훅 (0:00–0:40)
ESG 보고서, 대기업만의 일이라고요? 아닙니다. 대기업 공급망에 있으면 납품사인 여러분도 ESG 데이터를 제출해야 합니다. 2026년부터 의무화 범위가 확대됩니다. 준비 안 하면 납품 자격을 잃을 수 있습니다. 근데 ESG 전담 인력을 뽑을 여력은 없죠. AI가 데이터 수집부터 보고서 생성까지 자동으로 해줍니다.

### 문제 정의 (0:40–3:00)
ESG 보고의 병목. 첫째, 데이터 수집 — 탄소 배출량, 에너지 사용량, 폐기물, 용수 사용, 직원 다양성 등 수십 개 지표를 수동 수집. 둘째, 프레임워크 대응 — GRI, TCFD, SASB 등 프레임워크마다 요구 항목이 다릅니다. 셋째, 보고서 작성 — 데이터를 모아도 보고서 형태로 만드는 데 전문 지식 필요.

### 자동화 설계 (3:00–8:30)
**데이터 자동 수집**: 전기 사용량(한전 API) + 가스 사용량 + 용수 사용량 + 폐기물 배출량 → 월별 자동 수집 → 탄소 배출량 자동 환산(Scope 1, 2).

**공급망 ESG 데이터 수집**: 주요 협력사에게 분기별 ESG 설문 자동 발송 → 응답 수집 → 미응답 리마인더 → 공급망 ESG 점수 자동 산출.

**보고서 자동 생성**: 수집된 데이터 → 선택한 프레임워크(GRI/TCFD/SASB)에 맞는 보고서 자동 생성 → 전년 대비 변화 자동 분석 → 개선 영역 자동 식별.

**목표 대비 추적 자동화**: ESG 목표(탄소 배출 10% 감축 등) 설정 → 월별 진행률 자동 추적 → 목표 미달 시 원인 분석 + 개선 추천 → 분기별 경영진 리포트.
