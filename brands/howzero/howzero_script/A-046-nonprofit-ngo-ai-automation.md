## [1:30~4:00] 문제 정의 — NGO 행정의 3대 병목 (2분 30초)

**[화면: 업무 비율 파이차트]**

**대본:**
"비영리단체 행정 병목 3가지.

**1번: 후원자 관리 + 감사 커뮤니케이션**
후원이 들어오면 감사 메일 보내고, 후원 이력 업데이트하고, 연말에 기부금 영수증 발급하고, 정기 후원 현황 보고서 만들고. 후원자 500명이면 이 작업만 주 15시간입니다.

**2번: 보고서 작성**
정부 보조금 받으면 중간보고서, 결과보고서 제출해야 합니다. 기업 CSR 후원이면 분기별 보고서. 이사회 보고서도 따로. 같은 데이터를 포맷만 바꿔서 5번 씁니다.

**3번: 홍보 콘텐츠 제작**
SNS 운영, 뉴스레터 발행, 캠페인 페이지 제작. 전담 인력이 없어서 실무자가 본업 외에 추가로 합니다. 결과적으로 홍보 품질이 떨어지고, 신규 후원자 유입이 안 됩니다."

## [4:00~7:30] 자동화 설계 (3분 30초)

**[화면: 시스템 설계도]**

**대본:**
"NGO용 AI 자동화 시스템 3단계입니다.

**1단계: 후원자 자동 관리 시스템**
후원 결제가 들어오면 → 자동으로 감사 메일 발송 → 구글 시트에 후원 이력 업데이트 → 후원 유형별 태그 자동 부여.

감사 메일도 획일적인 게 아닙니다. AI가 후원자의 이름, 후원 금액, 후원 이력(첫 후원인지, 정기 후원인지)에 맞게 개인화된 메시지를 생성합니다.

'김영희 님, 3년째 매월 보내주시는 후원에 진심으로 감사드립니다. 님의 후원 덕분에 올해 청소년 장학금 수혜자가 15명 추가되었습니다.'

이런 메시지가 자동으로 나갑니다.

**2단계: 보고서 자동 생성**
프로그램 데이터(수혜자 수, 집행 예산, 활동 사진)를 구글 시트에 기록합니다. 보고서 제출 시점에 AI가 이 데이터를 가져와서 보고서 초안을 자동 작성합니다.

정부 양식이든, 기업 양식이든, 이사회 양식이든 — 템플릿만 설정해두면 같은 데이터로 다른 포맷의 보고서가 나옵니다.

**3단계: 홍보 콘텐츠 자동화**
프로그램 활동 데이터를 기반으로 AI가 SNS 포스팅 초안, 뉴스레터 초안을 자동 생성합니다. 사진과 수혜 사례를 넣으면 감동적인 스토리까지 만들어줘요.

주의할 점: 수혜자 개인정보 보호는 반드시 지켜야 합니다. AI 프롬프트에 '실명 사용 금지, 식별 가능한 정보 제외'를 명시하세요."

(테스트: 실제 보고서 초안 생성 과정)

기존에 3일 걸리던 보고서가 30분이면 끝납니다."
