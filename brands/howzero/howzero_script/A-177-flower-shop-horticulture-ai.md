## 대본

### 훅 (0:00–0:40)
꽃집 운영하시는 분들, 가장 큰 리스크가 뭡니까. 재고입니다. 꽃은 유통기한이 3~7일입니다. 발주 많이 하면 폐기, 적게 하면 기회 손실. 어버이날, 졸업시즌에는 물량 확보 전쟁. AI로 수요 예측 기반 발주부터 재고 관리, 배송 최적화까지 자동화한 사례. 코딩 제로입니다.

### 문제 정의 (0:40–3:30)
꽃집·원예 도매의 3대 병목. 첫째, 발주 — 경험과 감으로 발주. 날씨, 기념일, 트렌드 반영 부족. 장미 100송이 발주했는데 50송이 폐기. 둘째, 재고 관리 — 신선도 기반 재고 회전 관리 안 됨. "어제 들어온 장미"와 "3일 된 장미" 구분 없이 판매. 셋째, 배송 — 배달 주문 증가인데 배송 동선 비효율. 기사 1명이 하루 20건, 동선 꼬이면 시간 낭비.

### 자동화 설계 (3:30–9:30)
**AI 수요 예측·발주**: 지난 1년 판매 데이터 + 기념일 캘린더(어버이날, 졸업, 발렌타인, 화이트데이, 성년의날 등) + 날씨 예보 + SNS 트렌드(인기 꽃 키워드) → 품목별 주간 수요 예측 → 발주량 자동 산출 → 도매시장 시세 반영 → 발주서 자동 생성.

**신선도 기반 재고 관리**: 입고일 자동 기록 → 품목별 신선도 D-Day 자동 추적 → D-2 경고("장미 30송이 내일 폐기 위험 — 할인 판매 권장") → 당일 할인 자동 안내(SNS/카카오) → 폐기율 자동 추적.

**배송 동선 최적화**: 당일 배달 주문 수집 → AI가 최적 배송 동선 자동 산출(거리+시간 최소화) → 배송 기사에게 경로 자동 안내 → 배달 완료 시 고객에게 자동 확인 문자+꽃 관리법 안내.

**트렌드 분석**: 인스타그램·핀터레스트 꽃 트렌드 자동 수집 → 인기 꽃·색상·스타일 주간 리포트 → 신메뉴(꽃다발·화분) 기획 제안.

### 데모/사례 (9:30–13:30)
동네 꽃집 F. 일 평균 주문 25건. 자동화 전: 폐기율 18%, 배달 동선 비효율(기사 하루 8시간), 기념일 시즌 물량 부족 빈발, 트렌드 반영 안 됨. 자동화 후: 수요 예측으로 폐기율 18%→5%. 기념일 시즌 발주 정확도 90%↑(물량 부족 제로). 배송 동선 최적화로 기사 시간 8→6시간(배달 건수 동일). 트렌드 반영 신상품으로 객단가 15%↑.
