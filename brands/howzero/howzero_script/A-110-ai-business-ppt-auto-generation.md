# A-110: 유튜브 롱폼 — "AI로 사업 성과 발표 자료(PPT) 자동 생성"

## 메타데이터
- **제목**: AI로 사업 성과 발표 자료(PPT) 자동 생성 — 데이터만 넣으면 끝
- **썸네일 텍스트**: PPT 자동 생성 | AI가 만든다
- **태그**: PPT, 발표 자료, 성과 리포트, 자동 생성, AI, 하우제로
## 대본

### 훅 (0:00–0:40)
월말, 분기말마다 발표 자료 만드느라 밤새고 있지 않습니까? 데이터 뽑고, 차트 만들고, PPT 디자인하고, 멘트 준비하고… 내용은 같은데 매달 반복합니다. AI한테 데이터만 주면 PPT가 자동으로 나오는 시스템. 발표 준비 시간을 이틀에서 30분으로 줄인 방법입니다.

### 문제 정의 (0:40–3:00)
발표 자료 작성의 병목. 첫째, 데이터 수집 — ERP, GA, CRM에서 각각 데이터를 뽑아야 함. 둘째, 시각화 — 차트, 그래프를 수동으로 만들고 포맷 맞춤. 셋째, 스토리라인 — 숫자를 나열하면 아무도 안 들음. 인사이트를 뽑아서 스토리로 만들어야 함. 넷째, 디자인 — CI에 맞게 슬라이드 디자인. 이 네 단계를 합치면 최소 8시간~2일.

### 자동화 설계 (3:00–8:00)
**데이터 자동 수집**: 월말 자동으로 — 매출(ERP), 트래픽(GA), 리드(CRM), 고객 만족도(설문) 데이터를 구글 시트로 통합.

**AI 분석+스토리라인**: GPT-4가 데이터를 분석 → 핵심 인사이트 3가지 도출 + 전월 대비 변화 + 목표 대비 달성률 + 개선 제안 → 발표 스토리라인 자동 구성.

**PPT 자동 생성**: 스토리라인 + 데이터 → 구글 슬라이드/PPT 템플릿에 자동 삽입. 차트 자동 생성(매출 추이, 채널별 성과, 고객 세그먼트 등). CI 컬러, 폰트, 레이아웃 자동 적용.

**발표 멘트 자동 생성**: 슬라이드별 발표 스크립트 자동 생성 → 발표자 메모에 삽입.

### 데모/사례 (8:00–12:30)
라이브 시연: 실제 월간 성과 데이터 투입 → 3분 후 15슬라이드 PPT 완성. 표지, 목차, KPI 대시보드, 매출 분석, 채널별 성과, 고객 분석, 문제점+개선 제안, 다음 달 계획, Q&A 슬라이드 포함.

실제 사례: 중견기업 AB사. 월간 경영 회의 자료. 자동화 전: 기획팀 2명이 2일 작업. 자동화 후: 데이터 확인 + AI 생성 + 검토 = 총 30분. 연간 절감 시간 약 400시간.

도구: 구글 시트 + Make + GPT-4 + 구글 슬라이드 API. 월 운영비 약 8만 원.
