# A-019: 유튜브 롱폼 — "콘텐츠 1개 만들면 10개 채널에 자동 배포되는 시스템"

> **유형**: 유튜브 롱폼 | **길이**: 15분 | **시리즈**: 자동화 레시피 EP.06

---

## [0:00~0:08] 훅

"유튜브, 인스타, 틱톡, 링크드인, 블로그, 뉴스레터 — 다 해야 하는 거 알겠는데 시간이 없잖아요. 오늘 콘텐츠 1개만 만들면 나머지 9개가 자동으로 만들어지고 배포되는 시스템을 보여드리겠습니다."

## [0:08~2:00] 문제 — 멀티채널의 고통

"콘텐츠 마케팅, 중요한 거 아시죠. 근데 채널이 너무 많아요.

유튜브 롱폼 + 쇼츠 + 틱톡 + 릴스 + 인스타 피드 + 링크드인 + 블로그 + 뉴스레터 + X + 쓰레드.

각각 따로 만들면? 하루가 콘텐츠 제작으로 끝납니다. 비즈니스 할 시간이 없어요.

해결: 1개만 제대로 만들고, 나머지 9개는 AI가 자동으로 변환+배포."

## [2:00~5:00] 원 소스 멀티유즈 구조

**[화면: 구조도]**

"이게 제가 실제로 쓰는 구조입니다.

유튜브 롱폼 1개 촬영 (30분)
↓ 자동 파이프라인
├── 쇼츠 5~8개 (AI 자동 클리핑)
├── 틱톡 5~8개 (쇼츠 리포맷)
├── 릴스 3~5개 (쇼츠 리포맷)
├── 링크드인 포스트 2개 (대본 → 텍스트 변환)
├── 블로그 1개 (대본 → SEO 글)
├── 뉴스레터 소재 1개 (핵심 인사이트 추출)
├── X 스레드 1개 (핵심 7단계 추출)
├── 쓰레드 2개 (대화체 변환)
└── 인스타 캐러셀 1개 (핵심 포인트 카드뉴스)

1개 → 20개+. 추가 제작 시간: 검토 30분."

## [5:00~12:00] 라이브 빌드 (7분)

**[화면: 무편집]**

"실제로 만들어 보겠습니다.

**5:00~6:30 — 트랜스크립트 자동 추출**
유튜브에 영상 올리면 → 자동으로 대본 텍스트 추출.
(화면: Whisper API or Clova Note 연동)

**6:30~8:00 — 숏폼 자동 클리핑**
OpusClip에 영상 URL 넣으면 → AI가 바이럴 가능한 구간 5~8개 자동 선택 → 자막까지 자동.
(화면: OpusClip 데모)
3분이면 숏폼 8개 완성.

**8:00~10:00 — 텍스트 콘텐츠 자동 변환**
Claude API로 대본을 한 번에 8가지로 변환:
'이 대본을 블로그(2,000자)/링크드인(1,000자)/뉴스레터(500자)/X스레드(7트윗)/쓰레드(2개)로 변환해줘. 각각 톤 다르게.'
(화면: AI가 8개 결과 동시 출력 — 30초)

**10:00~11:30 — 캐러셀 자동 생성**
'이 대본에서 핵심 포인트 7개를 추출해서 캐러셀 텍스트로.'
(화면: 7장 슬라이드 텍스트 자동 생성)
Canva 템플릿에 붙여넣기만.

**11:30~12:00 — 예약 게시**
Buffer에 전부 예약.
쇼츠 → 화목토
틱톡 → 매일
릴스 → 화목토
링크드인 → 화목
블로그 → 수
뉴스레터 → 화

전부 예약 완료."
