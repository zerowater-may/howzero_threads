# A-118: 유튜브 롱폼 — "AI로 투자자/파트너 미팅 브리핑 자동 생성"
> **유형**: 유튜브 롱폼 | **길이**: 16분
## 메타데이터
- **제목**: AI로 투자자·파트너 미팅 브리핑 자동 생성 — 5분 준비로 완벽한 미팅
- **썸네일 텍스트**: 미팅 브리핑 AI 자동화 | 5분 준비
- **태그**: 투자자 미팅, 파트너, 브리핑, 미팅 준비, IR, AI, 하우제로
## 대본

### 훅 (0:00–0:40)
내일 투자자 미팅인데, 준비를 아직 못 했습니다. 그 투자자가 어떤 회사에 투자했는지, 최근 관심사가 뭔지, 우리 회사 데이터 중 뭘 강조해야 하는지… 이걸 밤새 리서치하고 있으면 정작 미팅에서 컨디션이 최악입니다. AI가 미팅 상대방 분석 + 우리 회사 핵심 데이터 + 예상 질문까지 자동으로 브리핑을 만들어줍니다.

### 문제 정의 (0:40–3:00)
미팅 준비의 병목. 첫째, 상대방 리서치 — 투자자/파트너의 포트폴리오, 최근 활동, 관심 분야를 일일이 검색. 둘째, 우리 데이터 정리 — 상대방이 관심 가질 만한 우리 회사 수치를 골라서 정리. 셋째, 예상 질문 — 어떤 질문이 올지 예측하고 답변 준비. 미팅 1건 준비에 평균 3시간.

### 자동화 설계 (3:00–8:00)
**미팅 등록**: 캘린더에 미팅 등록 시 참석자 이름/회사명 입력 → 자동 트리거.

**상대방 자동 리서치**: 이름+회사명 기반 → 링크드인 프로필, 최근 인터뷰/기사, 투자 포트폴리오(VC의 경우), 회사 최근 뉴스 자동 수집 → AI가 핵심 요약(5줄).

**우리 데이터 맞춤 정리**: 상대방 관심사 기반 → 우리 회사 데이터에서 관련 수치 자동 추출(예: VC가 SaaS 전문이면 MRR, Churn, LTV 강조).

**예상 질문+답변**: AI가 상대방 프로필 + 우리 데이터 기반으로 예상 질문 5개 + 모범 답변 자동 생성.

**브리핑 문서**: 위 내용을 1장짜리 브리핑 문서로 자동 정리 → 이메일/슬랙으로 미팅 1시간 전 자동 발송.

### 데모/사례 (8:00–12:30)
스타트업 AH사. 대표가 월 평균 미팅 12건(투자자 5건, 파트너 4건, 고객 3건). 자동화 전: 미팅 준비 건당 3시간. 월 36시간(거의 1주일). 자동화 후: 캘린더 등록만 하면 자동 브리핑 생성. 검토 시간 5분. 미팅 준비 시간 97% 절감. 미팅 질 향상(상대방 맥락 파악이 정확) → 투자 유치 성공률↑.

도구: 구글 캘린더 + Make + GPT-4 + 퍼플렉시티 API + 슬랙. 월 운영비 약 8만 원.
