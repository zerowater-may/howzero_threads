## 대본

### 훅 (0:00–0:40)
사내 컴플라이언스, 어떻게 관리하세요? 분기마다 체크리스트 돌리고 끝? 문제는 법규 위반이 분기 점검 사이에 발생한다는 겁니다. 계약서 한 줄 빠뜨린 게 3년 뒤 소송으로 돌아옵니다. AI가 매일 자동으로 점검합니다. 계약서, 이메일, 결재 문서, 내부 규정까지. 위반 징후 발견 즉시 알림. 연간 법무 비용 절감은 덤입니다. 코딩 제로.

### 문제 정의 (0:40–3:00)
컴플라이언스 관리의 병목. 첫째, 사후 점검 — 이미 위반이 발생한 후에 발견합니다. 과태료·소송 비용 발생. 둘째, 수동 검토 — 계약서 하나 검토에 법무팀 2시간. 월 수십 건이면 감당 불가. 셋째, 법규 변경 추적 — 관련 법률이 바뀌어도 사내 규정 업데이트가 늦습니다. 넷째, 직원 인식 부족 — 컴플라이언스 교육을 해도 실제 업무에서 적용이 안 됩니다.

### 자동화 설계 (3:00–9:00)
**계약서 자동 검토**: 신규 계약서 업로드 → AI가 필수 조항 누락·불리한 조건·법규 위반 소지 자동 스캔 → "제7조 손해배상 조항 — 상한 미지정, 리스크 높음" 자동 플래그 → 법무팀 검토 우선순위 자동 배정.

**내부 문서 실시간 모니터링**: 결재 문서·이메일·메신저 → AI가 컴플라이언스 위반 키워드+패턴 자동 감지 → "개인정보 무단 공유 징후" "공정거래법 위반 소지 표현" → 즉시 담당자 알림.

**법규 변경 자동 추적**: 관련 법령·시행령 개정 자동 모니터링 → 사내 규정과 자동 대조 → "개인정보보호법 제30조 개정 → 사내 규정 제12조 업데이트 필요" → 법무팀 자동 통보.

**컴플라이언스 교육 자동화**: 직원 직무별 필수 교육 자동 배정 → 이수 현황 자동 추적 → 미이수자 자동 리마인더 → 분기별 교육 이수율 리포트 자동 생성.
