### 자동화 설계 (3:00–8:00)
**예약 자동화**: 온라인 예약 폼(견종, 체중, 예방접종, 특이사항) → AI가 적합한 방·그룹 자동 배정 → 보호자에게 예약 확인 + 준비물 안내 자동 발송 → 체크인 전일 리마인더.

**보호자 리포트 자동화**: 직원이 하루 2회 사진 촬영 + 간단 메모 → AI가 "오늘의 리포트"(사진 + 활동 요약 + 건강 상태) 자동 생성 → 보호자 알림톡 자동 발송.

**건강·행동 로그 자동화**: 직원이 체크리스트(식사/배변/활동/특이사항) 터치 입력 → 자동 기록 → AI가 이상 패턴 감지(식사량 급감, 활동량 저하) → 즉시 보호자 알림 + 수의사 연계 추천.

**마케팅 자동화**: 이용 후 리뷰 요청 자동 발송 → 재방문 쿠폰 자동 발송(마지막 이용 30일 후) → 시즌별(명절, 여름 휴가) 프로모션 자동.
