## 대본

### 훅 (0:00–0:40)
구독 비즈니스의 진짜 수익은 신규 가입이 아닙니다. 이탈 방지입니다. 정수기, 공기청정기, 매트리스, SaaS — 렌탈/구독 모델은 결국 "얼마나 오래 유지시키느냐"가 전부입니다. 그런데 대부분의 렌탈 업체가 결제 실패 알림도 수동으로 보내고 있습니다. 오늘 AI 자동화로 구독 이탈률 34% 낮춘 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:30)
렌탈/구독 비즈니스의 3대 병목. 첫째, 결제 실패 관리 — 카드 만료, 잔액 부족 등으로 결제가 실패하면 수동으로 전화·문자. 평균 결제 실패율 8~12%, 이 중 30%는 그대로 이탈. 둘째, 해지 예측 불가 — 고객이 해지 버튼을 누르기 전까지 이탈 징후를 모릅니다. CS 불만 접수, 사용 빈도 감소, 결제 연체 반복 — 다 데이터에 있는데 활용을 못 합니다. 셋째, 리마인드 타이밍 — 필터 교체, 점검 예정일, 계약 만료 안내 등이 엑셀로 관리되면서 누락. 고객 불만의 40%가 "연락이 없었다"입니다.

### 자동화 설계 (3:30–9:00)
**결제 실패 자동 복구 시스템**: 결제 실패 감지 → 1시간 내 카카오 알림톡 발송("결제 수단을 확인해주세요" + 원클릭 결제 링크) → 24시간 미결제 시 2차 안내 → 72시간 미결제 시 담당자에게 슬랙 알림. 이 4단계만으로 결제 실패 이탈의 60%를 자동 복구합니다.

**이탈 예측 AI**: 고객 데이터를 구글 시트에 모읍니다 — CS 접수 횟수, 결제 연체 횟수, 서비스 이용 빈도, 가입 기간. GPT-4에게 이 데이터를 주고 이탈 위험 점수를 산출. 점수 70 이상이면 자동으로 프로모션 발송 — 1개월 무료 연장, 업그레이드 할인 등. 점수 90 이상이면 담당자 직접 전화 알림.

**자동 리마인드 시스템**: 필터 교체일 7일 전 알림 → 교체 예약 링크 → 점검 완료 후 만족도 조사 자동 발송 → 계약 만료 30일 전 갱신 안내 + 장기 고객 혜택 제시.

**해지 방어 시나리오**: 해지 요청 접수 → AI가 해지 사유 분석 → 사유별 맞춤 대응(가격 → 할인 제안, 불만 → 즉시 개선 약속, 이사 → 이전 설치 무료) → 해지 방어 성공률 자동 집계.
