## 대본

### 훅 (0:00–0:40)
세탁 프랜차이즈 본사에서 10개 매장 관리하시는 분. 각 매장 매출, 재고, 고객 불만, 장비 상태 — 이거 어떻게 파악하세요? 매장마다 전화해서 물어보시죠? 그거 AI가 알아서 수집하고, 이상 있으면 알려주고, 월말 리포트까지 만들어줍니다. 10개 매장, 관리 인원 1명. 실제 사례입니다.

### 문제 정의 (0:40–3:00)
세탁 프랜차이즈 본사의 병목. 첫째, 매장별 현황 파악 — 매출, 주문 수, 고객 불만 건수를 매장마다 수동 취합. 둘째, 장비 관리 — 세탁기, 건조기 고장이 매출 직결인데, 사후 대응만 합니다. 셋째, 가맹점 지원 — 운영 매뉴얼 업데이트, 프로모션 배포, 교육 자료 전달이 느립니다.

### 자동화 설계 (3:00–8:00)
**매장 현황 자동 집계**: POS 데이터 자동 수집 → 매장별 일매출, 주문 수, 객단가 대시보드 자동 갱신 → 전일 대비 20% 이상 이상치 발생 시 본사 알림.

**장비 예측 정비**: 장비 가동 시간 로그 자동 수집 → AI가 고장 예측(평균 수명 기반) → 정비 필요 장비 자동 알림 → 정비 업체 예약 자동 연동.

**가맹점 커뮤니케이션 자동화**: 본사 공지·매뉴얼 업데이트 시 → 전 매장 자동 발송 → 읽음 확인 자동 추적 → 미확인 매장 리마인더.

**고객 불만 중앙 관리**: 전 매장 리뷰·CS 자동 수집 → 감성 분석 → 부정 리뷰 즉시 본사 알림 → 대응 템플릿 자동 제안.
