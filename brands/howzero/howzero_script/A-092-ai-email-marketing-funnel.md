# A-092: 유튜브 롱폼 — "AI 이메일 마케팅 퍼널 자동화 완전 가이드"
> **유형**: 유튜브 롱폼 | **길이**: 19분
## 메타데이터
- **제목**: AI 이메일 마케팅 퍼널 자동화 완전 가이드 — 리드 수집부터 전환까지
- **썸네일 텍스트**: 이메일 퍼널 AI 자동화 | 완전 가이드
- **태그**: 이메일 마케팅, 퍼널, 자동화, 리드 너처링, 전환율, AI, 하우제로
## 대본

### 훅 (0:00–0:40)
이메일 마케팅, 죽었다고요? 전혀요. ROI 기준 모든 마케팅 채널 중 1위입니다. 1달러 투자에 36달러 회수. 문제는 대부분 수동으로 하니까 효과를 못 보는 겁니다. AI로 퍼널을 자동화하면 — 리드 수집 → 세그먼트 → 개인화 콘텐츠 → 발송 타이밍 최적화 → 전환까지 전부 자동입니다. 오늘 완전 가이드를 드립니다.

### 문제 정의 (0:40–3:30)
이메일 마케팅의 4대 병목. 첫째, 리드 수집 — 이메일을 모아도 품질이 낮음. 둘째, 세그먼트 — 전체 발송하면 오픈률 15% 이하. 셋째, 콘텐츠 — 매번 같은 내용이면 구독 해지. 넷째, 타이밍 — 언제 보내야 열어볼까? 감으로 보냄.

### 자동화 설계 (3:30–9:30)
**리드 수집 자동화**: 블로그/유튜브 CTA → 랜딩 페이지 → 리드 마그넷(무료 진단, 템플릿, 체크리스트) → 이메일 수집 → CRM 자동 등록.

**자동 세그먼트**: 유입 경로, 다운로드 콘텐츠, 웹사이트 행동 기반으로 AI가 자동 분류(관심 단계, 업종, 규모).

**개인화 콘텐츠 생성**: 세그먼트별 맞춤 이메일 시리즈를 AI가 자동 생성. 웰컴 → 가치 제공 → 케이스 스터디 → 오퍼 → 팔로업. 제목, 본문, CTA 모두 세그먼트에 맞게 개인화.

**발송 타이밍 최적화**: 수신자별 과거 오픈 시간 데이터 분석 → AI가 개인별 최적 발송 시간 자동 결정.

**A/B 테스트 자동화**: 제목 2개, 본문 2개 자동 생성 → 소규모 테스트 → 성과 좋은 버전 자동 선택 → 전체 발송.

### 데모/사례 (9:30–15:00)
B2B SaaS L사. 리드 리스트 2,400명. 자동화 전: 월 2회 전체 발송, 오픈률 14%, 클릭률 1.2%, 전환률 0.3%. 자동화 후: 주 1회 세그먼트별 자동 발송. 오픈률 14%→38%. 클릭률 1.2%→6.8%. 전환률 0.3%→2.1%. 월 이메일 기반 매출 200만 원→1,400만 원.

라이브 시연: 실제 5단계 너처링 시퀀스 구축 과정 시연. 리드 유입 → AI가 세그먼트 → 5일간 자동 이메일 시리즈 → 최종 CTA.

도구: ConvertKit(또는 Mailchimp) + Make + GPT-4 + 구글 시트. 월 운영비 약 8만 원.

### 효과 정리 (15:00–17:00)
- 오픈률 14% → 38%
- 클릭률 1.2% → 6.8%
- 전환률 0.3% → 2.1%
