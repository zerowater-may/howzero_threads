### 문제 정의 (0:40–3:00)
멀티 플랫폼 셀러의 5대 병목. 첫째, 상품 등록 — 플랫폼마다 양식이 달라 같은 상품을 2~3번 등록. 둘째, 가격/재고 동기화 — 수동 관리 시 가격 불일치, 품절 미반영. 셋째, CS — 같은 질문이 플랫폼별로 반복. 넷째, 리뷰 관리 — 분산된 리뷰 일일이 확인. 다섯째, 광고 최적화 — 감으로 광고비 쏟고 있음.

### 자동화 설계 (3:00–9:00)
**상품 등록 원클릭**: 마스터 DB(구글 시트)에 상품 정보 1회 입력 → AI가 플랫폼별 최적화(제목, 키워드, 카테고리, 상세 설명) → 쿠팡/네이버 동시 등록.

**가격/재고 실시간 동기화**: 마스터 DB 기준으로 전 플랫폼 가격·재고 실시간 연동. 경쟁사 가격 모니터링 → 자동 가격 조정 제안.

**CS 자동 응답**: 자주 묻는 질문(배송, 교환, 반품) 80% AI 자동 응답 → 복잡한 건만 사람이 처리.

**리뷰 통합 관리**: 전 플랫폼 리뷰 자동 수집 → 감성 분석 → 부정 리뷰 즉시 알림 → 답글 초안 생성.

**광고 최적화**: ROAS 데이터 자동 수집 → 성과 낮은 키워드 자동 OFF → 성과 좋은 키워드 예산 자동 증액.
