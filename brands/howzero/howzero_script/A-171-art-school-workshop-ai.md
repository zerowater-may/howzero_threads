### 자동화 설계 (3:30–9:30)
**수강 관리 자동화**: 온라인 등록 폼 → 결제 자동 확인 → 수강 일정 자동 배정 → 출석 체크(QR/태블릿) → 결석 시 자동 알림+보강 일정 자동 제안 → 진도 기록 자동화(수업별 체크리스트).

**작품 포트폴리오 자동화**: 수업 후 작품 사진 촬영(태블릿/폰) → 자동 보정(밝기, 배경 정리) → 수강생별 디지털 포트폴리오 자동 생성 → 학부모/본인에게 월간 포트폴리오 자동 발송("이번 달 OO님의 작품 4점").

**SNS 자동 포스팅**: 작품 사진에서 AI가 자동 캡션 생성(작품 설명+해시태그) → 인스타그램 예약 포스팅 자동 등록 → 주 3회 자동 게시 → 포스팅 성과(좋아요, 도달) 자동 추적.

**재등록·리텐션**: 수강 기간 종료 2주 전 자동 안내 → 재등록 할인 자동 적용 → 3개월 이상 미등록 수강생에게 복귀 프로모션 자동 발송.
