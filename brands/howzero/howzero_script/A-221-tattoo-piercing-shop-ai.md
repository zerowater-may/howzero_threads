## 대본

### 훅 (0:00–0:40)
타투이스트분들, 작업은 좋아하는데 예약 관리, 디자인 상담, SNS 포트폴리오 올리기 — 이런 거 싫으시죠? DM으로 상담하고, 견적 내고, 일정 잡고, 사후 관리 안내하고. 이 전부를 AI가 대신합니다. 아티스트는 작업에만 집중. 예약은 2배 늘었습니다.

### 문제 정의 (0:40–3:00)
타투·피어싱샵의 병목. 첫째, 상담·견적 — DM/카톡으로 디자인 상담, 사이즈·위치·가격 문의가 하루 수십 건. 대부분 비슷한 질문. 둘째, 예약·일정 — 시술 시간이 제각각(소형 1시간, 대형 8시간)이라 스케줄 관리가 복잡. 셋째, 포트폴리오 관리 — 작품 촬영 후 인스타에 올리고, 해시태그 달고, 스토리 만들 시간이 없습니다.

### 자동화 설계 (3:00–7:30)
**상담·견적 자동화**: 카톡/DM 문의 → AI 챗봇이 사이즈, 위치, 스타일 확인 → 레퍼런스 이미지 기반 자동 견적 → 복잡한 상담만 아티스트에게 전달.

**예약 자동 관리**: 견적 확정 → 시술 예상 시간 자동 산정 → 가능 시간 자동 제안 → 예약금 결제 링크 발송 → D-1 리마인더 + 사전 주의사항 안내.

**포트폴리오 자동화**: 시술 완료 사진 업로드 → AI가 캡션(스타일, 시술 시간, 부위) 자동 생성 → 인스타 피드 + 스토리 자동 예약 발행 → 해시태그 자동 최적화.

**사후 관리 자동화**: 시술 후 → 케어 가이드 자동 발송(D+1, D+3, D+7) → 힐링 상태 체크 메시지 → 터치업 시기 알림 → 1년 후 재방문 쿠폰.

### 데모/사례 (7:30–11:30)
홍대 타투샵, 아티스트 2명. 자동화 전: DM 상담 일 20건(응대 3시간), 예약 노쇼율 15%, 인스타 포스팅 주 2회, 사후 관리 안 함. 자동화 후: AI 챗봇 응대(아티스트 응대 일 30분), 예약 노쇼율 3%(예약금 + 리마인더), 인스타 포스팅 주 5회 자동, 사후 관리 100%.

월 시술 건수 40건 → 65건(상담 시간 절감 → 시술 시간 확보). 리뷰 수 3배 증가(사후 관리 시 리뷰 요청).

도구: 카카오 채널 + 구글 시트 + Make + GPT-4 + 인스타그램 API. 월 운영비 약 5만 원.
