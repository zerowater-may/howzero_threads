## 대본

### 훅 (0:00–0:40)
에어비앤비 3채 운영하시는 분, 하루에 메시지 몇 개 보내세요? 체크인 안내, 와이파이 비번, 주변 맛집, 체크아웃 안내, 청소 스케줄. 같은 메시지를 매번 수동으로 보냅니다. AI가 예약 확정부터 체크아웃 후 리뷰 요청까지 전 과정을 자동으로 운영합니다. 운영 시간 80% 절감. 10채를 혼자 운영할 수 있습니다.

### 문제 정의 (0:40–3:00)
숙소 운영의 병목. 첫째, 게스트 커뮤니케이션 — 예약 전 문의, 체크인 안내, 체류 중 질문, 체크아웃 안내. 다국어 대응까지 하면 끝이 없습니다. 둘째, 멀티 플랫폼 관리 — 에어비앤비, 부킹닷컴, 야놀자, 직접 예약. 각 플랫폼 캘린더를 동기화하지 않으면 더블 부킹. 셋째, 청소/관리 스케줄 — 체크아웃과 체크인 사이 청소 배정, 소모품 보충, 시설 점검. 넷째, 가격 최적화 — 시즌, 요일, 주변 행사에 따라 가격을 조정해야 하지만 수동으로 못 합니다.

### 자동화 설계 (3:00–8:00)
**게스트 커뮤니케이션 자동화**: 예약 확정 → 감사 메시지 + 체크인 가이드 자동 발송 → 체크인 3일 전 상세 안내(위치·비번·주차) → 체크인 당일 웰컴 메시지 → 체류 중 "불편한 점 있으세요?" 자동 체크 → 체크아웃 안내 → 리뷰 요청. 한/영/중/일 다국어 자동.

**멀티 플랫폼 동기화**: 모든 OTA 예약 자동 수집 → 캘린더 실시간 동기화 → 더블 부킹 방지 → 각 플랫폼 메시지 통합 관리(하나의 대시보드에서).

**청소/관리 자동 배정**: 체크아웃 감지 → 청소팀에 자동 알림(숙소 위치·다음 체크인 시간·특이사항) → 청소 완료 확인 → 소모품 부족 시 자동 발주 → 시설 이상 발견 시 수리 자동 접수.

**다이나믹 프라이싱**: 주변 숙소 가격 실시간 모니터링 + 예약률 + 시즌/행사 캘린더 → AI가 최적 가격 자동 산출 → OTA에 가격 자동 업데이트 → 예약률이 낮으면 가격 자동 조정.

### 데모/사례 (8:00–12:30)
에어비앤비 호스트 AQ. 숙소 5채, 1인 운영. 자동화 전: 일 메시지 응대 2시간, 월 2회 더블 부킹, 가격 조정 월 1회(수동), 청소 배정 전화/카톡, 리뷰 요청 안 함. 운영 시간 일 5시간. 자동화 후: 메시지 응대 자동(검토만 일 15분), 더블 부킹 0건, 가격 매일 자동 최적화, 청소 자동 배정, 리뷰 요청 자동(수집률 65%). 운영 시간 일 1시간.

객실 단가 평균 15% 상승(다이나믹 프라이싱). 점유율 75% → 88%. 에어비앤비 평점 4.5 → 4.85(빠른 응답+리뷰 관리). 5채에서 10채로 확장해도 1인 운영 가능.

도구: 구글 시트 + Make + GPT-4 + 에어비앤비 API + 카카오 알림톡. 월 운영비 약 8만 원.
