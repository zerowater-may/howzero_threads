## 대본

### 훅 (0:00–0:40)
"우리 팀에 파이썬 할 줄 아는 사람 있나요?" 이 질문에 HR이 즉답 못 하면, 직원 스킬 관리가 안 되고 있는 겁니다. 외부 채용에 수천만 원 쓰기 전에 내부에 이미 그 역량이 있는지부터 확인해야 합니다. AI로 직원 스킬맵을 자동 구축하고, 역량 갭 분석부터 성장 경로 추천까지 자동화한 시스템. 코딩 제로입니다.

### 문제 정의 (0:40–3:30)
스킬 관리의 3대 병목. 첫째, 파악 불가 — 직원이 30명만 넘어도 누가 뭘 할 수 있는지 HR이 파악 못 함. 엑셀에 한 번 정리하고 업데이트 안 됨. 둘째, 역량 갭 — 회사가 필요한 역량과 현재 보유 역량의 차이를 모름. 항상 외부 채용으로 해결. 셋째, 성장 경로 부재 — 직원이 "내가 뭘 배워야 승진하나?" 모름. 동기 저하 → 이직.

### 자동화 설계 (3:30–9:30)
**스킬맵 자동 구축**: 직원별 자기 진단 설문(분기 1회, 구글 폼) → 업무 이력·프로젝트 참여 데이터 자동 수집 → AI가 직원별 스킬 프로필 자동 생성(기술 스킬+소프트 스킬, 숙련도 1~5) → 부서별·직급별 스킬 히트맵 자동 시각화.

**역량 갭 분석**: 회사 전략 목표에 필요한 스킬셋 정의 → 현재 조직 보유 스킬과 자동 비교 → 갭 자동 식별("데이터 분석 역량: 필요 5명, 보유 2명, 갭 3명") → 내부 육성 vs 외부 채용 자동 제안.

**개인별 성장 경로 추천**: 직원의 현재 스킬 + 목표 포지션(본인 설정 또는 관리자 설정) → AI가 필요 스킬 갭 분석 → 맞춤 학습 콘텐츠 자동 추천(사내 교육, 외부 강좌, 자격증) → 분기별 성장 리포트 자동 생성.

**프로젝트 매칭**: 새 프로젝트 필요 스킬 입력 → 스킬맵에서 적합 인력 자동 추천 → 팀 구성 시뮬레이션(스킬 커버리지 점수) → PM에게 추천 결과 자동 제공.
