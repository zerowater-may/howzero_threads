### 문제 정의 (0:40–3:30)

### 자동화 설계 (3:30–9:00)
**AI 자동 견적 시스템**: 고객이 카카오 챗봇이나 웹폼으로 사양 입력(용지, 사이즈, 수량, 후가공, 납기) → AI가 단가표 DB 참조하여 즉시 견적 산출 → 견적서 PDF 자동 생성 → 고객에게 자동 발송. 복잡한 사양은 "유사 견적 3건" 제시 후 담당자 확인.

**생산 일정 자동화**: 주문 확정 → 제작 공정별 소요 시간 자동 계산 → 기계 가동 현황과 대조하여 최적 일정 배치 → 납기일 충돌 시 자동 경고 → 일정표 실시간 업데이트 → 진행 상황 고객 자동 안내("인쇄 완료, 후가공 진행 중").

**납품·정산 자동화**: 출고 시 납품 확인 알림 → 고객 수령 확인 → 세금계산서 자동 발행 → 미수금 30일 경과 시 자동 안내 → 월별 매출·미수금 리포트 자동 생성.
