## 대본

### 훅 (0:00–0:40)
키즈카페 사장님, 주말 오후에 입장 대기 줄이 어떻습니까. 안에는 정원 초과, 밖에는 대기 30분. 그런데 평일 오전에는 텅 비어있죠. 정원 관리, 안전 점검, 생일파티 예약 — 이 세 가지가 키즈카페 운영의 핵심인데, 대부분 수동입니다. AI로 예약부터 안전 점검까지 자동화한 키즈카페 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
키즈카페의 3대 병목. 첫째, 정원 관리 — 실시간 입장 인원 파악이 안 되면 과밀+안전 리스크. 대기 시간 안내도 감으로. 둘째, 안전 점검 — 놀이기구 점검, 소독, 안전 사고 대응 매뉴얼이 있지만 체크가 불규칙. 사고 시 법적 리스크. 셋째, 생일파티 — 전체 매출의 30~40%를 차지하는데, 예약·준비·진행 관리가 복잡. 누락 시 클레임 직결.

### 자동화 설계 (3:00–8:00)
**스마트 입장 관리**: 카카오 챗봇으로 실시간 입장 가능 여부 확인 + 시간대 예약 → QR 입장 → 실시간 인원 카운트 → 정원 80% 시 "혼잡" 안내, 100% 시 자동 마감 → 대기자에게 예상 시간 자동 안내 + 빈자리 발생 시 알림.

**안전 점검 자동화**: 일일 오픈 전 안전 점검 체크리스트 자동 발송(놀이기구별 점검 항목) → 직원이 사진+체크 → 미점검 항목 빨간색 경고 → 월 점검 리포트 자동 생성 → 소독 일정 자동 스케줄 + 완료 기록.

**생일파티 자동화**: 온라인 예약 폼(날짜, 인원, 패키지, 케이크, 데코레이션) → 자동 견적 + 결제 → D-3 준비물 체크리스트 직원에게 자동 전달 → D-1 학부모에게 확인 안내 → 당일 진행표 자동 → 완료 후 사진 발송 + 후기 요청.

**비수기 가동률**: 평일 오전 할인 자동 안내 → 유치원/어린이집 단체 패키지 프로모션 → 재방문 쿠폰 자동 발송(첫 방문 7일 후).

### 데모/사례 (8:00–13:30)
키즈카페 KD. 정원 60명, 생일파티 월 12건. 자동화 전: 정원 파악 수동(아르바이트 카운터), 안전 점검 불규칙(주 3회), 생일파티 준비 누락 월 2건, 평일 가동률 25%. 자동화 후: 실시간 정원 관리로 과밀 0건 + 대기 시간 정확 안내. 안전 점검 100% 이행(일일 자동 체크). 생일파티 누락 0건. 비수기 프로모션으로 평일 가동률 25%→48%.

도구: 구글 시트 + Make + GPT-4 + 카카오 챗봇 + QR. 월 운영비 약 7만 원.
