## 대본

### 훅 (0:00–0:40)
중고거래 많이 하시는 분들, 사진 찍고 설명 쓰고 가격 정하고 채팅 응대하고… 물건 하나 파는 데 시간이 얼마나 드세요? 리셀러나 정리 대행업 하시는 분들은 하루에 수십 건씩 올려야 합니다. AI로 사진만 찍으면 상품 설명 자동 작성, 시세 자동 조회, 채팅 자동 응대까지 되는 시스템. 일 50건 등록하는 리셀러의 비밀입니다.

### 문제 정의 (0:40–3:00)
중고거래 셀러의 3대 병목. 첫째, 상품 등록 — 사진 촬영, 상품명, 상세 설명, 카테고리, 상태(S/A/B/C) 작성. 건당 15~20분. 둘째, 가격 책정 — 시세를 모르면 너무 비싸게(안 팔림) 또는 너무 싸게(손해). 둘째, 채팅 — "직거래 돼요?", "네고 가능?", "상태 어때요?" 같은 반복 질문에 시간 소모.

### 자동화 설계 (3:00–8:00)
**상품 등록 자동화**: 사진 촬영 → GPT-4 Vision이 상품 분석(브랜드, 모델, 상태, 카테고리 자동 판별) → 상품명, 상세 설명, 해시태그 자동 생성 → 플랫폼 등록.

**시세 자동 조회**: 상품명+브랜드+상태 기반 → 당근마켓, 번개장터, 중고나라 최근 거래가 자동 수집 → 적정 가격 범위 제안(하한~상한).

**채팅 자동 응대**: FAQ 패턴 인식 → "직거래 가능합니다(OO역 기준)", "네고 가능합니다(최대 O% 할인)", "사진 추가 보내드립니다" 등 자동 응답. 복잡한 질문만 사람에게 전달.

**끌올/재등록 자동화**: 3일 미판매 시 자동 끌올 → 7일 미판매 시 가격 조정 제안 → 14일 미판매 시 다른 플랫폼 교차 등록.

### 데모/사례 (8:00–12:30)
중고 리셀러 AD씨. 월 판매 150건+. 자동화 전: 등록 건당 20분(일 10건이 한계), 시세 조회 감에 의존, 채팅 일 3시간. 자동화 후: 등록 건당 3분(사진만 촬영), 일 50건 등록 가능. 시세 기반 가격 책정으로 평균 판매 기간 5일→2.5일. 채팅 자동 응대 70%.

도구: GPT-4 Vision + 구글 시트 + Make + 자동 응답 스크립트. 월 운영비 약 5만 원.
