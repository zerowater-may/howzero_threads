## 대본

### 훅 (0:00–0:40)
파트너가 5개면 관리할 만합니다. 20개 넘으면? 계약 만료일 놓치고, 성과 정산 밀리고, 커뮤니케이션 누락됩니다. 제휴 파트너 관리는 사업이 커질수록 복잡해지는데, 대부분의 중소기업이 엑셀 하나로 버티고 있습니다. AI로 계약 관리, 성과 추적, 정산까지 자동화한 방법을 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
파트너 관리의 3대 병목. 첫째, 계약 관리 — 파트너마다 계약 조건, 수수료율, 만료일이 다른데 엑셀 한 줄로 관리. 갱신 시기를 놓쳐서 불리한 조건으로 자동 연장되는 경우 빈번. 둘째, 성과 추적 — 어떤 파트너가 실제로 매출에 기여하는지 수동 집계. 분기 1회 정리하면 이미 3개월 늦음. 셋째, 정산 — 파트너별 수수료 계산, 세금계산서 발행, 입금 확인까지 수동이면 월말 3~4일 소요.

### 자동화 설계 (3:00–8:00)
**계약 관리 자동화**: 파트너 DB(구글 시트) — 파트너명, 계약 조건, 수수료율, 시작일, 만료일, 담당자 → 만료 60일 전 자동 알림 → 30일 전 갱신 협상 리마인드 → 계약 갱신 시 이력 자동 기록.

**성과 추적 자동화**: 파트너별 추천 코드/UTM 자동 추적 → 일별 유입, 전환, 매출 자동 집계 → 주간 파트너 성과 대시보드 자동 업데이트 → 성과 하위 10% 파트너 자동 알림(재협상 대상).

**정산 자동화**: 월말 자동 정산 — 성과 데이터 기반 수수료 자동 계산 → 정산서 자동 생성 → 파트너에게 이메일 발송 → 입금 후 확인 자동 기록 → 미정산 건 자동 추적.

**파트너 커뮤니케이션**: 월간 뉴스레터 자동 발송(신상품, 프로모션, 성과 현황) → 분기 리뷰 미팅 자동 스케줄링 → 우수 파트너 자동 인센티브 안내.

### 데모/사례 (8:00–13:30)
이커머스 브랜드 PM. 제휴 파트너 28개. 자동화 전: 계약 만료 놓친 건 연 4회, 성과 집계 분기 1회(수동 3일), 월 정산 4일 소요, 파트너 커뮤니케이션 불규칙. 자동화 후: 계약 만료 놓침 0건. 성과 추적 실시간(주간 리포트 자동). 정산 4일→4시간(자동+검수). 월간 뉴스레터로 파트너 만족도 향상 — 파트너 이탈률 15%→5%.

도구: 구글 시트 + Make + GPT-4 + 이메일. 월 운영비 약 6만 원.
