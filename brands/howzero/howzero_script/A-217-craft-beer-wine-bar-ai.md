## 대본

### 훅 (0:00–0:40)

### 문제 정의 (0:40–3:00)
수제맥주·와인바의 병목. 첫째, 재고·탭 관리 — 케그 잔량 확인, 소진 예측, 발주 타이밍 판단을 감으로 합니다. 둘째, 고객 추천 — 100종 이상의 맥주/와인 중 고객 취향에 맞는 추천을 직원 경험에 의존. 셋째, 이벤트·마케팅 — 신규 입고, 한정판, 페어링 이벤트 기획과 홍보에 시간이 없습니다.

### 자동화 설계 (3:00–8:00)
**탭/재고 자동 관리**: POS 판매 데이터 기반 탭별 소진 속도 추적 → AI가 탭 교체 시점 예측 → 자동 발주 리스트 생성 → 입고 시 메뉴판 자동 업데이트.

**고객 추천 AI**: 고객 과거 주문 이력 + 선호 키워드(IPA, 스타우트, 프루티 등) → AI가 맞춤 추천 → 직원용 태블릿에 추천 리스트 표시 → 신메뉴 출시 시 선호 고객에게 자동 알림.

**이벤트·콘텐츠 자동화**: 신규 입고 맥주 → AI가 맛 노트, 양조장 스토리 기반 SNS 콘텐츠 자동 생성 → 페어링 메뉴 추천 자동 생성 → 이벤트 알림 자동 발송.

**매출 분석 자동화**: 맥주·와인별 마진율, 회전율 자동 분석 → 저회전 상품 교체 추천 → 시간대별 인기 메뉴 분석 → 해피아워 최적 메뉴 자동 제안.
