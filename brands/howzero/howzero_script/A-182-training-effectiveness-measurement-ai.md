## 대본

### 훅 (0:00–0:40)
직원 교육에 연간 얼마 쓰시나요. 중소기업 평균 직원 1인당 50~80만 원입니다. 50명이면 연 2,500만~4,000만 원. 그런데 이 교육이 실제로 업무 성과에 기여했는지 측정하는 회사는 10% 미만입니다. "교육 만족도 4.2점"이 끝입니다. AI로 교육 전후 성과 변화를 자동 측정하고, ROI까지 산출하는 시스템. 코딩 제로입니다.

### 문제 정의 (0:40–3:30)
교육 효과 측정의 3대 병목. 첫째, 만족도만 측정 — 커크패트릭 4단계(반응→학습→행동→결과) 중 1단계(반응=만족도)만 측정. 실제 행동 변화, 업무 성과 영향은 모름. 둘째, 데이터 연결 불가 — 교육 데이터(수료율, 점수)와 업무 성과 데이터(KPI)가 별도 시스템. 상관관계 분석 불가. 셋째, 리포트 부재 — 경영진에게 "교육 투자 대비 효과"를 숫자로 보여줄 수 없음 → 교육 예산 삭감.

### 자동화 설계 (3:30–9:30)
**교육 데이터 통합**: 교육 플랫폼(수료율, 시험 점수, 학습 시간) + 만족도 설문 + 사전/사후 역량 평가 → 구글 시트 자동 통합.

**성과 연결 자동 분석**: 교육 수료자의 교육 전후 KPI 변화 자동 추적(교육 전 3개월 vs 교육 후 3개월) → 비수료자 대비 성과 차이 자동 비교(컨트롤 그룹) → AI가 교육-성과 상관관계 분석("세일즈 교육 수료자: 수주율 12%↑ vs 비수료자: 변동 없음").

**교육 ROI 자동 산출**: 교육 비용(강사비+플랫폼비+인건비) 자동 집계 → 성과 개선분 금액 환산(수주율 12%↑ = 추가 매출 X원) → ROI 자동 산출("세일즈 교육 ROI: 340%") → 교육 프로그램별 ROI 랭킹.

**개인별 학습 경로 최적화**: 직원별 역량 갭 + 성과 데이터 + 학습 이력 → AI가 맞춤 교육 추천("김OO: 프레젠테이션 스킬 교육 권장 — 제안 발표 전환율 하위 20%") → 교육 후 성과 변화 자동 추적.

### 데모/사례 (9:30–13:30)
중견 제조기업 M. 직원 150명, 연간 교육 예산 8,000만 원. 자동화 전: 만족도만 측정(4.1/5), 성과 연결 분석 없음, 교육 ROI 모름, 교육 프로그램 10개 동일 예산 배분. 자동화 후: 교육 프로그램별 ROI 자동 산출 → 상위 3개 프로그램 ROI 200%↑, 하위 3개 프로그램 ROI 마이너스 발견. 예산 재배분(고ROI에 집중) → 교육 예산 동일, 성과 효과 2.5배↑. 경영진에게 ROI 데이터 제출 → 교육 예산 오히려 20%↑ 승인.

도구: 구글 시트 + Make + GPT-4 + 구글 폼. 월 운영비 약 6만 원.
