## 대본

### 훅 (0:00–0:40)
수입차 정비하시는 사장님들. BMW 5시리즈 부품 하나 발주하는데 며칠 걸리세요? 3일? 5일? 고객은 기다리다 딜러센터로 갑니다. 그리고 예약 관리요. 전화 받고, 일정 맞추고, 정비 이력 찾고. 수입차 정비의 핵심은 '신뢰'입니다. AI가 정비 이력 자동 관리, 부품 자동 발주, 정비 완료 후 자동 리포트까지 해줍니다. 딜러센터보다 빠르고 저렴한데, 신뢰도는 동급. 코딩 제로.

### 문제 정의 (0:40–3:00)
수입차 정비의 병목. 첫째, 부품 수급 — 수입차 부품은 재고가 없으면 해외 발주. 시간+비용 증가. 둘째, 정비 이력 관리 — 고객별·차량별 정비 기록이 노트에 있거나 없습니다. 셋째, 예약+일정 관리 — 전화 예약, 수기 기록. 더블 부킹 발생. 넷째, 고객 신뢰 — 딜러센터 대비 "사설 정비소가 정확히 해줄까?" 불안감.

### 자동화 설계 (3:00–8:30)
**부품 자동 발주 시스템**: 정비 접수 시 차종·연식 입력 → AI가 필요 부품 자동 식별 → 국내 재고 자동 조회(제일 빠른 공급처) → 없으면 해외 발주 자동 진행 → 도착 예정일 고객 자동 안내.

**차량별 정비 이력 자동 관리**: 차량 번호 입력 → 전체 정비 이력 즉시 조회 → "마지막 오일 교환 8개월 전, 브레이크 패드 교환 시기 도래" 자동 알림 → 예방 정비 안내 자동 발송.

**예약+일정 자동화**: 온라인 예약 폼(네이버/카카오) → 정비 유형별 소요 시간 자동 산출 → 가용 슬롯 자동 배정 → 확인 문자 자동 발송 → 전일 리마인더.

**정비 리포트 자동 생성**: 정비 완료 후 사진+체크리스트 입력 → AI가 전문 리포트 자동 생성 → "교체 부품: OEM 브레이크 패드, 잔여 수명 예측: 18개월" → 고객에게 자동 발송 → 딜러센터급 신뢰.

서울 수입차 전문 정비소 G. 리프트 3대, 정비사 4명. 자동화 전: 부품 발주 수동(평균 4일), 정비 이력 종이 기록, 예약 전화 전담 1명, 월 정비 대수 120대, 재방문율 35%. 자동화 후: 부품 발주 자동(평균 1.5일), 정비 이력 DB화, 온라인 예약 70%, 월 정비 대수 165대, 재방문율 68%.

월 매출: 4,500만 원 → 6,200만 원(38% 증가). 부품 수급 시간: 4일 → 1.5일. 재방문율: 35% → 68%.
