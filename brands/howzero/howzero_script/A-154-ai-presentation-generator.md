- **썸네일 텍스트**: PPT 10분 완성 | AI 자동 생성

## 대본

### 훅 (0:00–0:40)
월요일 아침 대표님이 말합니다. "오후 미팅에 쓸 PPT 15장 만들어줘." 데이터 정리, 슬라이드 구조 잡기, 텍스트 작성, 차트 삽입, 디자인 맞추기 — 반나절입니다. 그런데 이 과정의 80%를 AI가 할 수 있습니다. 데이터와 핵심 메시지만 입력하면 10분에 프레젠테이션이 나오는 시스템을 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
프레젠테이션 제작의 3대 병목. 첫째, 구조 설계 — "뭐부터 말하지?"에서 멈춤. 전체 흐름을 잡는 데 1시간+. 둘째, 텍스트 작성 — 슬라이드마다 핵심 메시지, 불렛 포인트, 설명문 작성. 15장이면 2~3시간. 셋째, 시각화 — 차트 만들기, 디자인 정리, 색상·폰트 통일. 내용은 좋은데 보기 싫으면 설득 실패.

### 자동화 설계 (3:00–9:00)
**1단계: 입력**: 프레젠테이션 목적(보고/제안/교육), 청중(경영진/고객/팀원), 핵심 메시지 3~5개, 관련 데이터(구글 시트 or 텍스트)를 입력합니다.

**2단계: AI 구조 자동 생성**: GPT-4가 목적과 청중에 맞는 슬라이드 구조 자동 설계 — 표지, 목차, 문제 정의, 데이터 분석, 솔루션, 실행 계획, 요약, Q&A. 각 슬라이드의 제목, 핵심 텍스트, 발표자 노트까지 자동 생성.

**3단계: 데이터 시각화**: 입력된 데이터를 AI가 분석 → 최적 차트 유형 자동 선택(막대, 라인, 파이, 워터폴) → 차트 자동 생성 → 핵심 인사이트 텍스트 자동 추가("전년 대비 23% 성장, 특히 Q3에 집중").

**4단계: 디자인 자동화**: Canva AI 또는 구글 슬라이드 템플릿에 자동 삽입 → 브랜드 색상·폰트 자동 적용 → 이미지 자동 배치 → 최종 결과물 PDF/PPTX 출력.

**전체 프로세스**: 입력(5분) → AI 생성(3분) → 검수·수정(5~10분) = 총 15~20분. 기존 4시간 대비 90%↓.

### 데모/사례 (9:00–13:30)
실제 시연. 주간 매출 리포트 프레젠테이션 15장을 실시간으로 만듭니다. 입력: "대상: 경영진, 목적: 주간 매출 보고, 핵심 데이터: 주간 매출 1.2억(전주 대비 +8%), 채널별: 자사몰 45%, 마켓플레이스 35%, 오프라인 20%, 이슈: 마켓플레이스 광고비 20% 증가."

AI 출력: 12장 슬라이드 — 표지, 주간 요약, 매출 추이 차트, 채널별 비중 파이차트, 전주 대비 변화 분석, 채널별 상세(3장), 광고비 이슈 분석, 대응 제안, 다음 주 계획, 마무리. 총 소요 시간: 12분.

팀 활용법: 마케팅 팀 — 캠페인 결과 보고. 영업 팀 — 고객 제안서. 경영진 — 투자자 미팅 자료. 모두 같은 시스템으로.
