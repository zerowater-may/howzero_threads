## 대본

### 훅 (0:00–0:40)
코인노래방 운영하시는 분들, 무인이라고 해놓고 실제론 무인이 아니죠. CCTV 수시 확인, 기기 고장 대응, 정산, 청소 스케줄. 점포 3개면 매일 돌아다닙니다. 진짜 무인 운영 시스템이 뭔지 보여드리겠습니다. AI로 룸 현황 실시간 파악, 이상 감지 자동 알림, 매출 최적화까지. 코딩 제로입니다.

### 문제 정의 (0:40–3:00)
노래방의 3대 병목. 첫째, 룸 관리 — 어떤 룸이 사용 중인지, 청소가 필요한지 실시간 파악 어려움. 다점포면 더 심각. 둘째, 매출 관리 — 시간대별 요금 차등이 비효율적. 비수기 프로모션 없음. 둘째, 유지보수 — 기기 고장, 에어컨 이상, 조명 문제 대응 지연. 고객 불만 → 리뷰 하락.

### 자동화 설계 (3:00–8:30)
**룸 현황 실시간 관리**: 결제 시스템 연동으로 룸별 사용 시작/종료 자동 기록 → 실시간 대시보드(사용 중/빈 룸/청소 필요) → 청소 스케줄 자동 생성(종료 후 자동 배정) → 다점포 통합 현황.

**매출 최적화**: 시간대별·요일별 가동률 자동 분석 → 비수기 시간대 자동 식별 → AI가 동적 프로모션 제안("월~목 14~17시 30% 할인 시 예상 가동률 40%→65%") → 프로모션 효과 자동 측정.

**이상 감지·유지보수**: 결제 기기·노래방 기기·에어컨 상태 데이터 수집(결제 에러 로그, 고객 신고 버튼) → 이상 패턴 AI 감지 → 즉시 관리자 알림 + 수리 업체 자동 연락 → 수리 이력 자동 기록.

**고객 리뷰 관리**: 이용 후 만족도 자동 설문(QR) → 부정 리뷰 즉시 알림 → AI가 응대 초안 자동 생성 → 월간 고객 만족도 트렌드 리포트.

### 데모/사례 (8:30–12:30)
코인노래방 체인 N. 5개 점포. 자동화 전: 매일 점포 순회 방문, 가동률 파악 불가, 기기 고장 고객 신고 후 대응, 프로모션 없음. 자동화 후: 전 점포 실시간 대시보드로 순회 80%↓. 비수기 프로모션으로 평일 오후 가동률 35%→58%. 기기 고장 대응 시간 평균 4시간→40분. 월 매출 15%↑.
