## 대본

### 훅 (0:00–0:40)
세탁소 사장님, 하루에 "제 옷 언제 돼요?" 전화 몇 통 받으세요? 접수는 수기, 추적은 기억, 완료 알림은 전화… 세탁 기술은 최고인데 관리는 20년 전 방식입니다. AI로 접수부터 완료 알림까지 전자동으로 바꾼 세탁소 이야기를 해드리겠습니다.

### 문제 정의 (0:40–3:00)
세탁소 운영 병목 세 가지. 첫째, 접수 — 고객 정보, 세탁물 종류, 특이사항을 수기로 작성. 빠뜨리면 클레임. 둘째, 추적 — 세탁물이 지금 어느 단계(접수→세탁→건조→다림질→포장)인지 파악 불가. 셋째, 알림 — 완료되면 전화해야 하는데, 바빠서 누락. 고객이 먼저 전화.

### 자동화 설계 (3:00–8:00)
**디지털 접수**: 태블릿/키오스크로 접수 → 고객 정보 + 세탁물 사진 촬영 + 특이사항 체크 → QR코드 라벨 자동 출력 → DB 자동 기록.

**세탁물 추적**: 각 공정 단계에서 QR 스캔 → 실시간 상태 업데이트 → 고객이 카카오톡으로 "내 세탁물 어디?" 질문하면 AI 챗봇이 자동 응답.

**완료 알림**: 포장 완료 시 자동 카카오 알림톡 발송 → 24시간 미수령 시 리마인드 → 3일 미수령 시 추가 알림.

**정기 세탁 관리**: 단골 고객 주기 파악(2주마다 와이셔츠 5벌 등) → 자동 리마인드 → 예약 접수.

### 데모/사례 (8:00–12:30)
서울 소재 프리미엄 세탁소 K. 일 평균 접수 45건. 자동화 전: "언제 돼요?" 전화 일 20통, 접수 오류 월 8건, 미수령 세탁물 월 15건. 자동화 후: 문의 전화 일 3통(AI 챗봇 처리), 접수 오류 제로(디지털화), 미수령 건 월 2건(자동 알림). 정기 세탁 예약으로 단골 매출 30%↑.

도구: 태블릿 + 구글 시트 + Make + GPT-4 + 카카오 알림톡 + QR 라벨 프린터. 초기 세팅비 약 50만 원, 월 운영비 약 5만 원.
