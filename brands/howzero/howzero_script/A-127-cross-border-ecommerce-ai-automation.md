## 대본

### 훅 (0:00–0:40)
해외 직구·역직구 셀러들이 가장 시간을 많이 쓰는 게 뭔지 아십니까. 판매가 아닙니다. 번역, 통관 서류, 해외 CS입니다. 일본어 상품 설명 번역, 미국 FDA 서류 준비, 영어 CS 대응 — 이 세 가지에 하루 4시간 이상 쓰는 셀러가 대부분입니다. AI로 이 시간을 90% 줄인 역직구 셀러 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:30)
직구/역직구의 3대 병목. 첫째, 다국어 상품 관리 — 상품명, 설명, 키워드를 현지 언어로 번역해야 합니다. 단순 번역이 아니라 현지 검색어에 맞춘 SEO 번역. SKU 100개면 번역량만 엄청남. 둘째, 통관·규정 — 국가별 수입 규정, HS코드, 성분 표시, 인증서 등. 하나 틀리면 반송. 셋째, 해외 CS — 시차 때문에 실시간 대응 어려움. 영어·일어·중국어 CS를 직접 하면 인건비 폭등.

### 자동화 설계 (3:30–9:30)
**AI 다국어 상품 등록**: 한국어 상품 정보 입력 → GPT-4가 영어/일어/중국어로 SEO 최적화 번역 → 현지 마켓플레이스 포맷에 맞춰 자동 변환 → 구글 시트에 다국어 상품 DB 자동 구축. 번역 품질 검수 포인트 — AI 번역 후 현지 표현 체크리스트 자동 적용.

**통관 서류 자동화**: 상품 카테고리 입력 → AI가 국가별 필요 서류 목록 자동 생성 → HS코드 자동 매칭 → 통관 서류 템플릿 자동 채움 → 체크리스트로 누락 방지.

**24시간 AI CS**: 해외 고객 문의(영어/일어) → AI 챗봇이 자동 응대(배송 조회, 반품 안내, 상품 문의) → 복잡한 건만 담당자에게 에스컬레이션 + 번역 포함 → 담당자가 한국어로 답변하면 AI가 자동 번역 후 발송.

**환율·가격 자동화**: 환율 변동 실시간 모니터링 → 마진율 기준치 이하 시 자동 가격 조정 제안 → 승인 후 마켓플레이스 가격 자동 업데이트.
