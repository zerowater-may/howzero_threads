## 대본

### [0:00~1:30] 훅

"프리랜서 여러분.
작업 시간보다 행정 시간이 더 긴 적 있으시죠?

견적서 작성, 계약서 정리, 작업 일지 기록, 청구서 발행, 세금 계산...
이게 실제 일하는 시간의 30~40%입니다.

오늘 이 행정 업무 전부를 AI로 자동화합니다.
프리랜서는 실력으로 돈을 버는 사람입니다.
행정으로 시간 쓰지 마세요."

### [1:30~3:30] 문제 정의

"프리랜서의 비즈니스 프로세스:

1. **리드 관리** — 문의 온 클라이언트 정보 정리
2. **견적서 작성** — 매번 새로 만들기
3. **계약서** — 표준 계약서 수정
4. **작업 관리** — 진행 상황 추적
5. **청구서 발행** — 작업 완료 후 정산
6. **세금/경비 관리** — 매출/경비 기록

이 6단계 중 프리랜서의 핵심 경쟁력은?
4번 '작업' 뿐입니다.
나머지 5개는 AI가 더 잘합니다."

---

### [3:30~7:00] 자동화 설계

"4개 시스템:

**시스템 1: 리드+견적 자동화**
- 문의 이메일 → AI가 자동 분류
- 프로젝트 유형별 견적서 자동 생성
- 견적서 PDF 자동 생성+발송

**시스템 2: 계약+작업 관리**
- 표준 계약서에 클라이언트 정보 자동 입력
- 작업 시작 시 마일스톤 자동 생성
- 진행률 자동 리포트

**시스템 3: 청구+정산**
- 작업 완료 시 청구서 자동 생성
- 입금 확인 자동 추적
- 미입금 리마인드 자동

**시스템 4: 세금/경비 자동 정리**
- 매출/경비 자동 분류
- 월간 수입/지출 리포트
- 종합소득세 예상액 자동 계산"

### [7:00~13:00] 라이브 데모

"[견적서 자동 생성]
(화면: Claude Code)

'웹디자인 프로젝트 견적서를 만들어줘.
클라이언트: ABC 주식회사
범위: 반응형 웹사이트 5페이지 + 모바일 최적화
기간: 3주
내 시급: ₩80,000
예상 작업시간: 40시간'

(실행 — 깔끔한 견적서)

항목별 금액, 결제 조건, 유효기간까지 포함.
이걸 매번 처음부터 만들면 30분.
AI로 2분.

[계약서 자동 생성]
'위 견적서 기반으로 프리랜서 용역 계약서를 만들어줘.
착수금 30%, 중간 30%, 완료 40% 조건으로.'

법적으로 기본 요소가 다 들어간 계약서가 나옵니다.
(물론 중요한 계약은 법률 검토를 받으세요.)

[청구서 자동화]
'프로젝트 완료. 최종 청구서를 만들어줘.
착수금 ₩960,000 수령 완료, 중간금 ₩960,000 수령 완료.
잔금 ₩1,280,000 청구.'

[월간 리포트]
'이번 달 프리랜서 수입/지출 리포트를 만들어줘.
매출: 프로젝트 A ₩3,200,000 + 프로젝트 B ₩1,500,000
경비: 장비 ₩150,000 + 소프트웨어 ₩50,000 + 교통비 ₩30,000'

깔끔한 월간 리포트. 세무사에게 바로 보낼 수 있습니다."
