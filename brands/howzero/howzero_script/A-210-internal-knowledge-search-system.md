## 대본

### 훅 (0:00–0:40)
"이거 어디 있었지?" 회사에서 하루에 몇 번 말하세요? 노션, 드라이브, 슬랙, 이메일에 흩어진 정보. 찾는 데만 하루 1시간. 직원이 퇴사하면 지식도 같이 사라집니다. AI 사내 지식 검색 시스템 하나 만들었더니, 신입이 첫 주부터 5년차처럼 일합니다. 코딩 0줄.

### 문제 정의 (0:40–3:00)
사내 지식 관리의 병목. 첫째, 정보 파편화 — 노션, 구글 드라이브, 슬랙, 이메일, 개인 메모에 분산. 둘째, 검색 불능 — 키워드 검색으로는 원하는 답을 못 찾습니다. "지난번 A 고객 이슈 어떻게 해결했지?" 이런 질문에 답을 못 합니다. 셋째, 지식 소실 — 직원 퇴사 시 머릿속 노하우가 함께 사라집니다.

### 자동화 설계 (3:00–8:30)
**지식 자동 수집**: 노션 페이지, 구글 드라이브 문서, 슬랙 주요 채널 대화 → 자동으로 지식 베이스에 인덱싱 → 주 1회 갱신.

**AI 검색 인터페이스**: 슬랙에서 "/ask 질문" 입력 → AI가 지식 베이스에서 관련 문서 검색 + 답변 생성 → 출처 문서 링크 함께 제공.

**FAQ 자동 생성**: 가장 많이 검색되는 질문 TOP 20 → AI가 자동으로 FAQ 문서 생성 → 주 1회 갱신.

**온보딩 지식 자동화**: 신입 입사 시 → 직무별 필수 문서 자동 추천 → "첫 주 가이드" 자동 생성 → 질문 시 AI가 사내 맥락 기반 답변.

**지식 갭 감지**: 검색했는데 답을 못 찾은 질문 자동 수집 → "문서화가 필요한 주제" 리스트 자동 생성 → 담당자에게 작성 요청.

### 데모/사례 (8:30–13:00)
마케팅 에이전시 E. 직원 20명, 노션 페이지 2,000+, 구글 드라이브 문서 5,000+. 자동화 전: 정보 찾는 시간 인당 일 45분, 신입 온보딩 기간 4주, 퇴사자 인수인계 평균 1주(그래도 누락). 자동화 후: 정보 찾는 시간 인당 일 5분, 신입 온보딩 기간 1.5주, 퇴사자 지식 90% 이상 시스템에 보존.

AI 검색 정확도 87%(3개월 후 학습으로 93%). 월간 질의 수 1,200건. "선배한테 물어보기" 70% 감소.

도구: 노션 API + 구글 드라이브 API + 슬랙 + Make + GPT-4 + Pinecone(벡터DB). 월 운영비 약 15만 원.
