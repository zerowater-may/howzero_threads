## 대본

### 훅 (0:00–0:40)
해외 진출 고민하시는 대표님, 어떤 나라로 가야 할지 어떻게 결정하세요? 컨설팅 업체에 5,000만 원 주고 보고서 받으세요? 아니면 감으로? AI가 200개국 시장 데이터를 분석해서, 우리 제품에 가장 적합한 시장 TOP 5를 자동으로 뽑아줍니다. 리서치 비용 90% 절감. 실제 해외 진출 성공 사례입니다.

### 문제 정의 (0:40–3:00)
글로벌 리서치의 병목. 첫째, 데이터 접근 — 각국 시장 규모, 경쟁 환경, 규제, 소비자 특성 데이터가 흩어져 있습니다. 언어 장벽도 있습니다. 둘째, 분석 역량 — 데이터를 모아도 비교 분석할 전문 인력이 없습니다. 셋째, 비용 — 글로벌 컨설팅 보고서 1건에 수천만 원. 중소기업에겐 부담.

### 자동화 설계 (3:00–8:30)
**시장 데이터 자동 수집**: World Bank, UN Comtrade, Google Trends, Amazon 판매 데이터 등 → 국가별 시장 규모, 성장률, 경쟁 강도, 관세 정보 자동 수집 → 주 1회 갱신.

**적합도 자동 분석**: 우리 제품 특성(카테고리, 가격대, 타겟층) 입력 → AI가 국가별 적합도 점수 산출 → 시장 규모, 경쟁 강도, 진입 장벽, 물류 비용, 문화 적합성 종합 평가 → TOP 5 국가 자동 추천.

**경쟁사 분석 자동화**: 타겟 국가 선정 후 → 현지 경쟁사 자동 검색 → 가격대, 포지셔닝, 리뷰 분석 → 우리 제품의 경쟁 우위/열위 자동 분석.

**진출 로드맵 자동 생성**: 타겟 국가 확정 → AI가 진출 단계별 체크리스트(인허가, 물류, 마케팅, 현지화) 자동 생성 → 예상 비용·일정 자동 산출 → 리스크 요인 정리.
