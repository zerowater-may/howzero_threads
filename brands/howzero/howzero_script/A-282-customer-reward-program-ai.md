## 대본

### 훅 (0:00–0:40)
리워드 프로그램 운영하시는 분들. 포인트 적립해주고, 쿠폰 뿌리고. 근데 재구매율은 안 오르죠? 이유가 뭔지 아세요? 고객 전체한테 똑같은 혜택을 주고 있기 때문입니다. VIP한테도 첫 구매 고객한테도 똑같은 5% 할인. AI는 고객별로 다른 리워드를 줍니다. 이탈 직전 고객에겐 파격 쿠폰, VIP에겐 전용 혜택, 휴면 고객에겐 복귀 인센티브. 이탈률 40% 감소. 코딩 제로.

### 문제 정의 (0:40–3:00)
리워드 프로그램의 병목. 첫째, 일률적 혜택 — 모든 고객에게 같은 포인트율, 같은 쿠폰. ROI가 낮습니다. 둘째, 이탈 감지 불가 — 고객이 떠나기 전에 알 수 없습니다. 떠난 후에야 "왜 안 오시지?" 셋째, 수동 운영 — 쿠폰 발행, 문자 발송, 포인트 정산 전부 수작업. 넷째, 효과 측정 불가 — 어떤 리워드가 실제 재구매로 이어졌는지 모릅니다. 감으로 운영합니다.

### 자동화 설계 (3:00–8:00)
**고객 세그먼트 자동 분류**: RFM(최근 구매·빈도·금액) 분석 자동화 → VIP/일반/이탈 위험/휴면 자동 분류 → 매주 자동 갱신 → "이 고객은 구매 빈도가 3주 전 대비 60% 감소, 이탈 위험" 자동 감지.

**세그먼트별 리워드 자동 설계**: VIP → 전용 얼리 액세스+추가 포인트 / 이탈 위험 → 파격 할인 쿠폰 자동 발송 / 휴면 고객 → 복귀 인센티브 / 신규 → 두 번째 구매 유도 쿠폰. 전부 자동 발송.

**효과 자동 측정**: 리워드별 사용률·재구매 전환율·LTV 변화 자동 추적 → 월간 리포트 자동 생성 → "이탈 위험 고객 파격 쿠폰 → 복귀율 32%, 일반 쿠폰 → 복귀율 8%" 비교 분석.

**포인트 정산 자동화**: 적립·사용·소멸 자동 계산 → 소멸 예정 알림 자동 발송 → "3,200 포인트가 이번 달 말 소멸됩니다" → 사용 유도 → 재방문.
