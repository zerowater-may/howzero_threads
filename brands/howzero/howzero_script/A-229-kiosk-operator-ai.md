## 대본

### 훅 (0:00–0:40)
키오스크 운영 업체 사장님, 키오스크 100대 관리하는 데 인원 몇 명이세요? 결제 오류, 프린터 먹통, 메뉴 업데이트, 소프트웨어 패치 — 이거 건마다 현장 출동이죠? AI가 키오스크 100대를 원격으로 모니터링하고, 80%의 이슈를 원격으로 해결합니다. 현장 출동 80% 감소.

### 문제 정의 (0:40–3:00)
키오스크 운영의 병목. 첫째, 장애 대응 — 결제 모듈 오류, 프린터 에러, 네트워크 끊김을 매장에서 신고해야 알게 됩니다. 둘째, 콘텐츠 관리 — 메뉴 변경, 가격 수정, 프로모션 적용을 키오스크별로 수동 업데이트. 셋째, 성능 분석 — 어떤 메뉴가 잘 팔리는지, 어떤 시간대에 사용률이 높은지 데이터를 못 씁니다.

### 자동화 설계 (3:00–8:00)
**원격 모니터링 자동화**: 키오스크별 상태(네트워크, 결제 모듈, 프린터, 디스플레이) 실시간 수집 → 이상 감지 시 즉시 알림 → AI가 원격 재부팅/리셋 자동 시도 → 미해결 시 현장 출동 자동 배정.

**콘텐츠 자동 배포**: 중앙 관리 시스템에서 메뉴·가격·프로모션 변경 → 전체/선택 키오스크에 자동 배포 → 적용 확인 자동 → 실패 건 재시도.

**고객 행동 분석 자동화**: 주문 데이터 + 터치 히트맵 → 인기 메뉴, 이탈 포인트, 평균 주문 시간 AI 분석 → UX 개선 추천 → 매장별 맞춤 메뉴 배치 제안.

**정기 관리 자동화**: 소프트웨어 패치 → 비수기 시간(새벽) 자동 배포 → 업데이트 결과 자동 리포트 → 하드웨어 정기 점검 스케줄 자동 생성.

### 데모/사례 (8:00–12:30)
