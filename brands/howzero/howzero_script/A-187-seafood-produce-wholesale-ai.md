## 대본

### 훅 (0:00–0:40)
수산물, 농산물 도매업. 가장 큰 리스크 두 가지. 시세 변동과 폐기입니다. 어제 킬로당 5,000원이던 오징어가 오늘 8,000원. 100킬로 발주했는데 70킬로만 팔림 → 30킬로 폐기. 이 두 리스크를 데이터로 관리하는 도매상은 5% 미만입니다. AI로 시세 예측, 수요 기반 발주, 유통 경로 최적화까지. 코딩 제로입니다.

### 문제 정의 (0:40–3:30)
수산·농산 도매의 3대 병목. 첫째, 시세 관리 — 경매 시세가 매일 변동. 적정 매입가 판단이 경험 의존. 고매입→마진 하락, 저매입 시도→물량 확보 실패. 둘째, 수요 예측·발주 — 거래처(식당, 마트, 급식업체)별 주문 패턴 파악 안 됨. 감으로 매입. 폐기율 10~20%. 셋째, 유통 관리 — 입고→분류→배송 과정에서 신선도 관리 어려움. 배송 동선 비효율.

### 자동화 설계 (3:30–10:00)
**시세 모니터링·예측**: 주요 도매시장 경매 시세 자동 수집(일별) → 품목별 시세 트렌드 시각화 → AI가 내일/이번 주 시세 예측(과거 패턴+계절+어획량/작황 데이터) → 매입 의사결정 지원("오징어: 이번 주 시세 하락 예상 — 매입 1~2일 지연 권장").

**수요 예측·발주 최적화**: 거래처별 과거 주문 이력 분석 → 요일·시즌·메뉴 트렌드 반영 → 품목별 일별 수요 예측 → 적정 매입량 자동 산출 → 시세 예측 반영한 최적 매입 타이밍 제안.

**유통·배송 자동화**: 입고 시 품목·수량·신선도 등급 자동 기록 → 거래처 주문 자동 수집 → 주문별 피킹 리스트 자동 생성 → 배송 동선 자동 최적화 → 배송 완료·수령 확인 자동 기록.

**거래처 관리**: 거래처별 주문 이력·결제 이력·매출 기여도 자동 분석 → 거래처 등급 관리(A/B/C) → 장기 미주문 거래처 자동 알림("거래처 Y: 3주 미주문 — 연락 권장") → 월간 거래처 리포트.

### 데모/사례 (10:00–14:30)
수산물 도매 S. 취급 품목 30종, 거래처 50곳. 자동화 전: 시세 확인 매일 새벽 수작업, 매입량 감으로(폐기율 15%), 배송 동선 비효율(기사 2명 하루 10시간), 거래처 관리 수작업. 자동화 후: 시세 예측 정확도 75%→AI 적중률 85%. 수요 예측+발주 최적화로 폐기율 15%→4%(연 2,400만 원 절감). 배송 동선 최적화로 기사 시간 10→7시간(배송 건수 동일). 거래처 리텐션 자동 관리로 이탈률 20%↓.
