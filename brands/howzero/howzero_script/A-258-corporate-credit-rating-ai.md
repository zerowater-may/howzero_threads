- **제목**: AI로 기업 신용 평가 자동 분석 — 재무제표부터 리스크 스코어까지

## 대본

### 훅 (0:00–0:40)
거래처 신용 확인, 어떻게 하세요? 신용평가 보고서 하나 떼는 데 2~3만 원, 분석은 담당자가 엑셀로. 거래처가 100개 넘으면 분기마다 확인하는 것도 불가능합니다. 그래서 대금 미회수 사고가 터집니다. AI가 거래처 재무제표 자동 수집, 리스크 스코어 자동 산출, 위험 징후 실시간 알림을 합니다. 분석 시간 90% 단축. 대금 미회수 0건이 목표입니다.

### 문제 정의 (0:40–3:00)
기업 신용 관리의 현실. 첫째, 정보 수집이 수동 — 전자공시(DART), 신용평가사 보고서, 뉴스를 사람이 하나씩 확인합니다. 둘째, 분석이 단편적 — 재무제표 숫자만 봅니다. 업종 트렌드, 경영진 변동, 법적 분쟁 등 비재무 리스크를 놓칩니다. 셋째, 모니터링이 안 됨 — 거래 시작 때 한 번 확인하고 끝. 거래 중에 상대방 재무 상태가 악화되는 걸 모릅니다. 넷째, 의사결정이 느림 — 신규 거래처 여신 승인에 1~2주 걸립니다.

### 자동화 설계 (3:00–8:30)
**재무 데이터 자동 수집**: DART 전자공시 자동 크롤링 → 거래처별 재무제표(매출·영업이익·부채비율·현금흐름) 자동 추출 → 비재무 데이터(뉴스·법적 분쟁·경영진 변동) 자동 모니터링.

**AI 리스크 스코어**: 재무 지표 5개 + 비재무 지표 3개 종합 분석 → 100점 만점 리스크 스코어 자동 산출 → 업종 평균 대비 비교 → 전년 동기 대비 변화 추이 → A(안전)~E(위험) 등급 자동 부여.

**실시간 위험 알림**: 거래처 리스크 스코어 급변 시 즉시 알림 → 부도·회생·대표 변경 뉴스 자동 감지 → 연체 패턴 AI 분석(결제 주기 지연 추세) → 위험 등급 D 이상 시 여신 한도 자동 재검토 트리거.

**여신 심사 자동화**: 신규 거래처 사업자등록번호 입력 → AI가 5분 이내 리스크 리포트 자동 생성 → 여신 한도 자동 추천 → 심사역은 리포트 검토+최종 승인만.
