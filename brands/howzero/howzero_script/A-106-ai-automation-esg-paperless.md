### 문제 정의 (0:40–3:00)
종이 의존 사무실의 문제. 첫째, 비용 — 용지, 토너, 프린터 유지보수, 보관 공간. 직원 50명 기준 연 600만 원+. 둘째, 비효율 — 종이 문서 검색 시간, 분실 위험, 버전 관리 불가. 셋째, ESG 리스크 — 투자자·고객이 ESG 경영을 요구. 종이 사용량은 가장 눈에 보이는 지표.

### 자동화 설계 (3:00–8:00)
**문서 디지털화**: 기존 종이 문서 스캔 → AI OCR로 텍스트 추출 → 자동 분류+태깅 → 클라우드 저장 → 검색 가능한 DB 구축.

**결재 전자화**: 종이 품의서/결재서 → 전자 결재 워크플로우(슬랙+구글 폼) → 결재 라인 자동 알림 → 서명 전자화 → 이력 자동 보관.

**계약서 자동화**: 계약서 템플릿 DB → 계약 조건 입력 → AI가 자동 생성 → 전자 서명(모두싸인/도큐사인) → 자동 보관+만료 알림.

**인쇄 모니터링**: 프린터 사용량 자동 추적 → 부서별 인쇄량 대시보드 → 월간 트렌드 → 목표 대비 알림 → ESG 리포트 자동 생성.
