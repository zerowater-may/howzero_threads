## 대본

### 훅 (0:00–0:40)
네일 아티스트분들, 시술하면서 전화 받느라 힘들지 않으세요? 시술 중에 전화가 오면 받을 수도 없고, 나중에 콜백하면 이미 다른 데 예약한 고객도 있습니다. 예약 전화 때문에 놓치는 매출이 월 얼마인지 아세요? AI로 24시간 예약을 자동으로 받고, 고객 취향까지 자동 관리하는 시스템. 매출 30% 올린 네일샵 사례입니다.

### 문제 정의 (0:40–3:00)
뷰티 살롱의 3대 병목. 첫째, 예약 — 전화/카카오톡 예약을 시술 중에 처리 불가. 노쇼도 문제. 둘째, 고객 관리 — 고객별 시술 이력, 알러지, 선호 디자인 기억에 의존. 셋째, 재방문 유도 — 시술 후 팔로업 없음. 재방문 주기가 늘어남.

### 자동화 설계 (3:00–8:00)
**24시간 예약**: 네이버 예약/카카오 챗봇 연동 → 실시간 빈 시간대 확인+예약 → 예약 확정 알림 → 24시간 전 리마인드 → 노쇼 시 대기 고객 자동 안내.

**고객 프로필 자동화**: 시술 후 사진 촬영 → 고객 DB에 자동 저장(날짜, 시술 종류, 컬러, 디자인) → 다음 방문 시 "지난번 프렌치 네일 하셨는데, 이번엔 어떻게 해드릴까요?" 자동 제안.

**재방문 유도**: 평균 재방문 주기 분석(네일 3~4주, 왁싱 4~6주) → 주기 도래 3일 전 자동 메시지 → 이벤트/신규 디자인 안내 → 예약 링크 포함.

**리뷰 관리**: 시술 완료 2시간 후 자동 리뷰 요청(네이버/인스타) → 리뷰 작성 시 포인트 자동 적립 → 부정 리뷰 즉시 알림.

### 데모/사례 (8:00–12:30)
서울 강남 네일샵 U. 네일 아티스트 2명. 자동화 전: 부재중 전화 일 15통(놓치는 예약 추정 일 5건), 노쇼 월 12건, 재방문 주기 평균 5주. 자동화 후: 예약 100% 온라인(부재중 전화 제로), 노쇼 월 2건(리마인드 효과), 재방문 주기 5주→3.5주(자동 리마인드), 리뷰 수 월 8건→25건.

도구: 네이버 예약 + 구글 시트 + Make + GPT-4 + 카카오 알림톡. 월 운영비 약 4만 원.
