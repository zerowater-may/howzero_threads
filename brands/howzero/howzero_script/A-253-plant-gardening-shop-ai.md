## 대본

### 훅 (0:00–0:40)
반려식물 쇼핑몰 운영하시는 분, 구매 후 "식물이 죽었어요" 클레임 몇 건이세요? 식물은 배송 후가 진짜 승부입니다. 관리를 못 하면 죽고, 죽으면 재구매가 아니라 이탈입니다. AI가 구매 식물별 맞춤 관리 가이드를 자동 발송하고, 물 주기 리마인더를 보내고, 계절별 관리 팁을 보내고, 관련 용품을 자동 추천합니다. 클레임 70% 감소, 재구매율 2.1배.

### 문제 정의 (0:40–3:00)
식물 쇼핑몰의 특수한 병목. 첫째, 사후 관리 — 식물은 사고 나서가 시작입니다. 관리법을 모르면 죽고, 책임은 판매자에게 돌아옵니다. 둘째, 계절 변수 — 같은 식물이라도 여름과 겨울의 관리법이 다릅니다. 이걸 일일이 안내할 인력이 없습니다. 셋째, 재구매 유도 — 식물은 소모품이 아닙니다. 추가 구매를 유도하려면 화분·흙·비료 등 관련 용품 추천이 필요합니다. 넷째, 상담 — "잎이 누래졌어요", "벌레가 생겼어요" 같은 문의가 매일 수십 건.

### 자동화 설계 (3:00–8:00)
**구매 후 맞춤 관리 가이드**: 구매 식물 종류 자동 감지 → 해당 식물 전용 관리 가이드(물주기·일조량·온도·습도) 자동 발송 → 배송 당일 "분갈이 가이드" → 1주 후 "적응기 관리법" → 계절 변동 시 자동 업데이트.

**물주기 리마인더**: 식물 종류별 최적 물주기 주기 설정 → 구매일 기반 리마인더 자동 발송 → 계절·날씨 반영 주기 자동 조정 → "오늘 몬스테라 물줄 날이에요. 화분 밑으로 물이 빠질 때까지 충분히."

**AI 식물 상담**: 고객이 사진+증상 전송 → AI가 이미지 분석(과습·과건·병충해·영양 결핍) → 원인 진단 + 조치 방법 자동 안내 → 심각한 경우 전문가 상담 연결.

**관련 용품 자동 추천**: 구매 식물에 맞는 화분·흙·비료 자동 추천(구매 3일 후) → 계절별 필요 용품 추천(겨울: 가습기·식물등, 여름: 살충제·차광막) → 구매 이력 기반 중복 추천 방지.
