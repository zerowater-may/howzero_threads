## 대본

### 훅 (0:00–0:40)
연말 인사평가, 회사 전체가 2주 멈추지 않으세요? 평가 양식 배포하고, 자기평가 독촉하고, 상사평가 취합하고, 등급 조정 회의하고. 이 과정에서 실제로 직원이 성장하는 건 하나도 없습니다. AI가 365일 역량 데이터를 자동 수집하고, 실시간으로 강점·약점을 진단하고, 개인별 성장 로드맵을 자동 제안합니다. 평가가 아니라 성장 시스템으로 바꾸는 거죠.

### 문제 정의 (0:40–3:00)
직원 역량 관리의 현실. 첫째, 연 1~2회 형식적 평가 — 1년치 성과를 기억에 의존해 평가합니다. 최근 3개월 편향(Recency Bias)이 심합니다. 둘째, 주관적 평가 — 상사의 호불호가 반영됩니다. 객관적 데이터가 없습니다. 셋째, 성장 연결 부재 — 평가 결과가 "올해 B등급"으로 끝납니다. 구체적으로 뭘 어떻게 개선해야 하는지 안내가 없습니다. 넷째, 운영 부담 — 인사팀이 평가 시즌에 다른 일을 못 합니다.

### 자동화 설계 (3:00–8:30)
**역량 데이터 자동 수집**: 프로젝트 완료율, 고객 피드백, 동료 피드백, 교육 이수, 자격증 취득 등 다양한 데이터 소스에서 자동 수집 → 역량 프레임워크(직무별 핵심 역량 5~7개)에 자동 매핑.

**실시간 역량 대시보드**: 직원별 역량 레이더 차트 자동 생성 → 팀 평균 대비 강점·약점 자동 식별 → 분기별 변화 추이 자동 추적 → 관리자 대시보드에 팀 전체 역량 현황 요약.

**AI 성장 로드맵**: 약점 영역 자동 식별 → 해당 역량 개발을 위한 교육·프로젝트·멘토 자동 추천 → "김OO님, 데이터 분석 역량이 팀 평균 대비 낮습니다. 추천: SQL 기초 과정(2주) + 다음 분기 데이터 분석 프로젝트 참여" → 진행률 자동 추적.

**평가 자동화**: 수집된 데이터 기반 평가 초안 자동 생성 → 관리자는 검토+조정만 → 등급 분포 자동 시뮬레이션 → 평가 결과 피드백 메시지 자동 생성.
