### 문제 정의 (0:40–3:00)

### 자동화 설계 (3:00–8:00)
**로스팅 스케줄 자동화**: 주문량 예측(과거 데이터 + 구독 스케줄) → 로스팅 배치 자동 생성 → 숙성 완료일 계산 → 발송일 자동 세팅.

**구독 관리 자동화**: 구독 결제일 자동 처리 → 발송 전 "이번 달 원두 소개" 알림톡 자동 발송 → 구독 3개월째 고객에게 업셀(블렌드 추가) 자동 제안 → 해지 요청 시 리텐션 쿠폰 자동 발송.

**콘텐츠 자동 생성**: 새 원두 등록 시 → AI가 산지 정보 + 맛 노트 기반 상세 페이지 자동 작성 → 블로그 포스트(브루잉 가이드) 자동 생성 → SNS 콘텐츠 자동 변환.

**고객 취향 분석**: 구매 이력 + 리뷰 분석 → AI가 고객별 선호 프로파일 생성 → 맞춤 원두 추천 자동 발송.
