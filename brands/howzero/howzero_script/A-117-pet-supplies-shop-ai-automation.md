### 문제 정의 (0:40–3:00)
펫 쇼핑몰의 3대 병목. 첫째, 개인화 부재 — 강아지/고양이, 소형/대형, 퍼피/시니어 다 다른데 같은 상품을 추천. 둘째, 소모품 재구매 — 사료, 간식, 패드는 정기적으로 필요한데 재구매 안내가 없음. 셋째, 정보 제공 — "소형견에 좋은 사료가 뭐예요?" 같은 질문에 전문적 응대가 어려움.

### 자동화 설계 (3:00–8:30)
**펫 프로필 기반 추천**: 회원가입 시 반려동물 프로필 등록(종, 품종, 나이, 체중, 알러지, 건강 이슈) → AI가 프로필 기반 맞춤 상품 추천 → 메인 페이지, 이메일, 카카오톡에 개인화 추천 노출.

**재구매 자동화**: 소모품 구매 시 소진 예상일 자동 계산(사료 5kg, 소형견 1마리 = 약 30일) → 소진 5일 전 자동 알림 → 원클릭 재주문 → 정기배송 전환 유도.

**AI 펫 컨설턴트**: 카카오 챗봇에 AI 배치 → "8개월 된 포메라니안, 알러지 있는데 사료 추천해주세요" → AI가 프로필+상품 DB 기반 추천 + 수의사 감수 정보 제공.

**커뮤니티 자동화**: 반려동물 생일 자동 축하 메시지 + 나이대별 건강 팁 자동 발송 + 포토 리뷰 이벤트 자동 운영.
