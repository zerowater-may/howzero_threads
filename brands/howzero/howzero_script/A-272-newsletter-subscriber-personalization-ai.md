## 대본

### 훅 (0:00–0:40)
뉴스레터 5,000명에게 같은 내용 보내시죠? 오픈율 20%. 나쁘지 않다고요? 같은 5,000명에게 5개 버전으로 세분화해서 보내면 오픈율 40%. 같은 노력으로 2배 효과. AI가 구독자를 자동으로 세분화하고, 세그먼트별 관심사에 맞는 콘텐츠를 자동으로 큐레이션하고, 제목까지 개인화합니다. 뉴스레터는 개인화가 전부입니다.

### 문제 정의 (0:40–3:00)
뉴스레터 운영의 병목. 첫째, 전체 발송 — 모든 구독자에게 같은 내용. 마케팅 담당자에게 보내는 내용과 개발자에게 보내는 내용이 같습니다. 둘째, 세분화가 수동 — 구독자를 분류하려면 설문을 돌려야 합니다. 응답률 10%. 셋째, 콘텐츠 제작 부담 — 세그먼트별로 다른 콘텐츠를 만들면 작업량이 3~5배. 넷째, 효과 분석 부재 — 어떤 구독자가 어떤 콘텐츠에 반응하는지 분석 안 합니다.

### 자동화 설계 (3:00–8:00)
**행동 기반 자동 세분화**: 오픈 이력·클릭 이력·유입 경로 AI 분석 → 관심사 자동 태깅("이 구독자는 마케팅+자동화 관심") → 5~7개 세그먼트 자동 생성 → 세그먼트 자동 업데이트(행동 변화 반영).

**콘텐츠 자동 큐레이션**: 메인 콘텐츠 1개 작성 → AI가 세그먼트별 각도 자동 변형(마케팅 관점 버전, 운영 관점 버전, 기술 관점 버전) → 도입부와 CTA만 세그먼트별 차별화 → 작업량 최소화.

**제목 개인화**: 세그먼트별 과거 오픈율 높았던 제목 패턴 AI 학습 → 세그먼트별 제목 자동 생성 → A/B 테스트 자동 실행 → 승자 제목 확대.

**발송 시점 최적화**: 구독자별 과거 오픈 시간대 분석 → 개인별 최적 발송 시간 자동 산출 → 같은 뉴스레터를 구독자마다 다른 시간에 발송.

**구독자 건강도 관리**: 3개월 미오픈 구독자 자동 플래그 → 재참여 캠페인 자동 발송("요즘 뉴스레터 안 보시는 것 같아서요. 관심사가 바뀌셨나요?") → 6개월 미반응 시 자동 정리 → 리스트 건강도 유지.
