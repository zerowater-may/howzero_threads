## 대본

### 훅 (0:00–0:40)
네일아트 샵 원장님, 단골 고객이 들어왔는데 "지난번에 한 디자인으로 해주세요"라고 합니다. 기억나십니까? 네일 색상, 파츠 종류, 길이, 모양 — 한 달에 고객 100명이면 전부 다릅니다. 시술 이력을 체계적으로 관리하면 고객 만족도가 확 올라갑니다. AI로 시술 이력, 재방문 유도, 인스타그램 운영까지 자동화한 심화 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
네일/속눈썹 샵의 3대 병목. 첫째, 시술 이력 — 고객마다 선호 디자인, 알레르기, 손톱 상태, 지난 시술 내용이 다른데 기록이 없음. 사진을 찍어도 누구 건지 분류 안 됨. 둘째, 재방문 주기 — 네일 평균 3~4주, 속눈썹 2~3주 주기인데 고객이 까먹으면 안 옴. 경쟁 샵으로 이동. 셋째, SNS 운영 — 시술 사진을 인스타에 올려야 신규 고객이 오는데, 사진 촬영·캡션 작성·해시태그·게시 시간까지 신경 쓸 여유가 없음.

### 자동화 설계 (3:00–8:00)
**시술 이력 자동 관리**: 시술 완료 → 사진 촬영 → 구글 시트에 고객명+시술 내용(색상, 디자인, 파츠, 길이)+사진 자동 저장 → 다음 방문 시 "지난 시술" 즉시 조회 → 고객 선호도 패턴 자동 분석(항상 파스텔 톤, 매트 마감 선호 등).

**재방문 자동 유도**: 시술 후 재방문 적정 시기 자동 계산(네일 3주, 속눈썹 2.5주) → 적정일 3일 전 카카오 알림("○○님, 네일 교체 시기가 다가왔습니다. 이번엔 봄 신상 디자인은 어떠세요?") → 예약 링크 포함 → 미방문 시 1주 후 재안내 + 할인 쿠폰.

**인스타그램 자동화**: 시술 사진 촬영 → AI가 캡션 자동 생성(시술 내용+해시태그+CTA) → 게시 예약(인스타 최적 시간대) → 주간 게시 스케줄 자동 관리. 예: "#봄네일 #벚꽃네일 신상 디자인으로 달라진 손끝 💅 예약 문의는 프로필 링크!"

**고객 세분화 마케팅**: 시술 이력 기반 고객 그룹 분류(네일만/속눈썹만/복합/VIP) → 그룹별 맞춤 프로모션 자동 발송 → 신규 메뉴 출시 시 관련 고객에게만 안내.
