## 대본

### 훅 (0:00–0:40)
ESG 보고서, 의무 공시 대상이 매년 확대되고 있습니다. 2026년부터 자산 2조 원 이상 기업 의무화. 중소기업도 공급망 ESG 요구를 받고 있습니다. 문제는 ESG 데이터 수집에 인력 2~3명이 3개월 매달린다는 겁니다. AI로 데이터 수집부터 지표 산출, 보고서 초안 생성까지 자동화한 시스템. 코딩 제로입니다.

### 문제 정의 (0:40–3:30)
ESG 보고서의 3대 병목. 첫째, 데이터 수집 — 환경(전기·가스·수도 사용량, 폐기물), 사회(직원 다양성, 산업재해, 교육 시간), 지배구조(이사회 구성, 윤리 경영) 데이터가 부서별로 흩어져 있음. 수작업 수집에 2~3개월. 둘째, 지표 산출 — GRI, SASB, TCFD 등 프레임워크별 요구 지표가 다름. 수기 계산 오류. 셋째, 보고서 작성 — 100페이지+ 보고서를 매년 처음부터. 외부 컨설팅비 3,000만~1억 원.

### 자동화 설계 (3:30–10:00)
**데이터 자동 수집**: 전기·가스·수도 요금 고지서 OCR 자동 인식 → 폐기물 처리 업체 리포트 자동 수집 → HR 시스템에서 인사 데이터 자동 추출 → 구매 시스템에서 공급망 데이터 추출 → 전 데이터 구글 시트 통합.

**ESG 지표 자동 산출**: 수집 데이터 기반 → GRI 스탠다드 지표 자동 계산(탄소 배출량 Scope 1·2, 용수 사용량, 폐기물 재활용률, 직원 이직률, 교육 시간 등) → 전년 대비 변화율 자동 산출 → 업종 평균 대비 벤치마크.

**보고서 초안 자동 생성**: 산출된 지표 + 회사 ESG 전략·정책 정보 → AI가 GRI 형식 보고서 초안 자동 생성(CEO 메시지 초안, 환경·사회·지배구조 섹션별 본문, 데이터 표·그래프) → 담당자가 검수·수정.

**분기별 모니터링**: 분기 1회 자동 중간 점검 → 연간 목표 대비 진행률 자동 산출 → 목표 미달 항목 경고 → 개선 조치 자동 제안.
