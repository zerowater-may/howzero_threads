## 대본

### 훅 (0:00–0:40)
와이너리나 농장 체험 프로그램 운영하시는 분, 주말마다 예약 전화 폭주하죠? 인원 확인하고, 프로그램 설명하고, 날씨 따라 일정 바꾸고, 체험 후 와인·농산물 판매까지. 이 전부를 AI가 관리하는 와이너리가 있습니다. 체험 예약 3배, 부대 매출 2배.

### 문제 정의 (0:40–3:00)
체험 관광의 병목. 첫째, 예약 관리 — 전화·카톡 예약, 인원 변경, 취소 처리가 수동. 주말 시즌에 전화를 못 받아서 놓치는 예약이 30%. 둘째, 날씨·시즌 대응 — 비 오면 프로그램 변경, 수확 시기에 따라 체험 내용 변경을 실시간으로 안내해야 합니다. 셋째, 부대 수익 — 체험 후 와인·농산물 구매 전환이 자연스럽지 않습니다.

### 자동화 설계 (3:00–7:30)
**예약 자동화**: 온라인 예약 시스템(날짜, 인원, 프로그램) → 실시간 잔여석 확인 → 자동 결제 → 오시는 길 + 준비물 + 프로그램 안내 자동 발송 → D-1 날씨 기반 안내(우천 시 대체 프로그램 자동 안내).

**프로그램 관리 자동화**: 시즌별 가용 프로그램 자동 전환(포도 수확 → 와인 시음 → 와인 만들기) → 홈페이지·예약 시스템·SNS 자동 업데이트.

**부대 매출 자동화**: 체험 후 → 당일 체험한 와인/농산물 구매 링크 자동 발송 → 체험 참여자 한정 할인 쿠폰 → 정기 구독(와인 클럽) 안내 자동.

**콘텐츠·마케팅 자동화**: 체험 사진 → AI가 후기형 콘텐츠 자동 생성 → SNS 자동 포스팅 → 방문자 리뷰 요청 자동 → 시즌별 프로모션 자동 발송.

### 데모/사례 (7:30–11:30)
경북 와이너리 W. 주말 체험 프로그램 3종, 연간 방문 2,000명. 자동화 전: 예약 전화 놓침 30%, 우천 시 개별 연락 3시간, 체험 후 와인 구매 전환율 15%, SNS 포스팅 월 2회. 자동화 후: 예약 100% 온라인 자동, 우천 안내 자동(0분), 와인 구매 전환율 42%, SNS 포스팅 주 4회 자동.

연간 방문 2,000명 → 5,500명. 부대 매출(와인 판매) 2.3배 증가. 와인 클럽 구독 120명 확보.
