## 대본

### 훅 (0:00–0:40)
3D프린팅 서비스 운영하시는 분, 견적 하나 내는 데 시간 얼마나 걸리세요? 파일 확인하고, 소재 선택하고, 출력 시간 계산하고, 후가공 비용 추가하고. 건당 30분은 걸리죠. AI가 STL 파일 올리면 30초 만에 자동 견적을 내고, 프린터 배정까지 자동으로 합니다. 처리량 2배.

### 문제 정의 (0:40–3:00)
3D프린팅 서비스의 병목. 첫째, 견적 산출 — 파일 크기, 소재, 후가공, 수량에 따라 매번 수동 계산. 복잡한 형상은 출력 가능 여부 판단도 필요. 둘째, 프린터 관리 — 여러 대의 프린터 가동 스케줄, 소재 잔량, 출력 상태 모니터링이 수동. 셋째, 주문 관리 — 다수 고객의 주문 진행 상태 추적, 완료 알림, 배송 처리가 복잡.

### 자동화 설계 (3:00–8:00)
**AI 자동 견적**: 고객이 STL 파일 업로드 → AI가 형상 분석(부피, 서포트 필요량, 출력 시간) → 소재별 가격 자동 산출 → 후가공 옵션 추가 → 30초 이내 견적 제공 → 출력 불가 형상 자동 감지 + 수정 제안.

**프린터 스케줄 자동화**: 주문 확정 → 가용 프린터 자동 배정(소재, 크기 기반) → 출력 시작/완료 예상 시간 고객 자동 안내 → 출력 중 에러 감지 시 즉시 알림 + 재출력 자동 스케줄.

**재료 관리 자동화**: 프린터별 필라멘트/레진 잔량 추적 → 잔량 기준치 이하 시 자동 발주 → 소재별 사용량 통계 자동.

**고객 커뮤니케이션 자동화**: 주문 확정 → 출력 시작 → 출력 완료 → 후가공 → 배송 각 단계 자동 알림 → 수령 후 리뷰 요청.
