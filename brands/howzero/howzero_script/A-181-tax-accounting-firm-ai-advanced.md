### 문제 정의 (0:40–3:30)
세무·회계법인의 심화 병목. 첫째, 감사 업무 — 증빙 대조, 이상 거래 검출, 감사 조서 작성에 시간 폭발. 시니어 회계사의 70%가 반복 검증 작업. 둘째, 세무 자문 — 고객 질문에 법령·판례 검색 후 답변. 건당 30분~2시간. 주니어가 못 하고 시니어만 가능. 셋째, 고객 관리 — 고객사 100곳+의 계약 갱신, 수임료 관리, 업셀 기회 파악 불가.

### 자동화 설계 (3:30–10:30)
**감사 자동화**: 고객사 회계 데이터 자동 수집 → AI가 전표 자동 대조(장부 vs 증빙) → 이상 거래 자동 감지("동일 거래처 월 3회 이상 100만 원 이하 분할 결제 — 세금계산서 회피 의심") → 감사 조서 초안 자동 생성 → 회계사 검수.

**세무 자문 AI 어시스턴트**: 법인 내부 지식 DB 구축(세법, 판례, 예규, 과거 자문 사례) → 고객 질문 입력 → AI가 관련 법령·판례 자동 검색 + 자문 초안 생성 → 시니어 회계사 검수·보완 → 주니어도 AI 지원으로 기본 자문 가능.

**고객 포트폴리오 관리**: 고객사별 계약 정보, 수임료, 서비스 내역, 매출 규모 자동 관리 → 계약 갱신 D-60 자동 알림 → 고객 성장에 따른 업셀 기회 자동 감지("고객사 A 매출 30%↑ — 법인 전환 자문 제안 권장") → 고객 만족도 자동 추적.

**법인 경영 대시보드**: 담당자별 수임 건수·매출·가동률 자동 추적 → 서비스별 수익성 분석 → 신규 고객 획득 파이프라인 관리 → 월간 경영 리포트 자동 생성.
