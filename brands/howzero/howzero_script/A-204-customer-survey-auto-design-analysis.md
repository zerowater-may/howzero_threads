## 대본

### 훅 (0:00–0:40)
고객 설문조사 해보신 적 있으세요? 질문 만드는 데 3일, 배포하고 기다리는 데 2주, 결과 분석하는 데 또 1주. 그래서 안 하게 됩니다. 근데 설문 설계부터 분석까지 전부 AI가 하면 어떨까요? 질문 설계 10분, 분석 자동, 인사이트 리포트까지 알아서 나옵니다. 실제 사례 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
설문조사의 병목. 첫째, 질문 설계 — 어떤 질문을 해야 유의미한 데이터가 나오는지 모릅니다. 대충 만들면 대충 나옵니다. 둘째, 응답률 — 길면 안 하고, 짧으면 인사이트가 없습니다. 최적 길이와 타이밍을 모릅니다. 셋째, 분석 — 엑셀에 수백 개 응답 쌓여 있는데, 교차 분석, 감성 분석은 엄두를 못 냅니다.

### 자동화 설계 (3:00–8:00)
**질문 자동 설계**: 설문 목적(만족도, NPS, 신제품 반응 등) 입력 → AI가 목적에 맞는 질문 세트 자동 생성 → 질문 수, 유형(객관식/주관식/척도) 최적 배분.

**배포 자동화**: 설문 완성 → 타겟 고객 세그먼트 자동 추출 → 이메일/알림톡으로 자동 발송 → 미응답자 리마인더 자동 발송(D+3, D+7).

**실시간 분석 자동화**: 응답 수신 즉시 → 객관식 통계 자동 집계 → 주관식 감성 분석(긍정/부정/중립) 자동 분류 → 핵심 키워드 추출.

**인사이트 리포트 자동 생성**: 분석 완료 → AI가 "주요 발견 3가지 + 개선 액션 추천" 형태의 1페이지 리포트 자동 생성 → 의사결정자에게 자동 발송.
