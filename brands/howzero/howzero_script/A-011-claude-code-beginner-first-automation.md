# A-011: 유튜브 롱폼 — "비개발자의 첫 AI 자동화, Claude Code로 따라하세요"

> **유형**: 유튜브 롱폼 | **길이**: 16분 | **앵글**: 완전 초보가 따라할 수 있는 튜토리얼
> **키워드**: Claude Code 사용법, AI 자동화 시작, 비개발자 자동화

## [0:00~0:08] 훅

"AI 자동화 해보고 싶은데 뭐부터 해야 할지 모르겠다고요? 오늘 이 영상 하나면 됩니다. 완전 초보도 따라할 수 있게, 첫 자동화를 처음부터 끝까지 만들어 보겠습니다. 15분 후에 여러분의 첫 자동화가 돌아갑니다."

## [0:08~2:00] 왜 Claude Code인가

"자동화 도구가 수십 개 있는데 왜 Claude Code냐.

3가지 이유:
1. AI 연동이 가장 강력합니다. ChatGPT, Claude 다 연결 가능.
2. 비개발자도 시작할 수 있습니다. 코딩 0줄.
3. 제가 10억 SaaS를 만들 때 쓴 도구입니다. 실전 검증 완료.

오늘 만들 것: '이메일이 오면 → AI가 읽고 분류해서 → 슬랙으로 알림 보내주는' 자동화. 가장 기본이면서 가장 효과 큰 것."

## [2:00~4:00] Step 1: 환경 준비 (2분)

**[화면: 셋업 과정]**

"먼저 필요한 것 3가지.

1. Gmail 계정 (있으시죠?)
2. 슬랙 or 카카오톡 (알림 받을 곳)
3. Claude Code 계정 (지금 만들어도 됩니다)

Claude Code 가입 방법:
(화면: 가입 과정 1분)
무료 플랜으로 시작. 카드 등록 안 해도 됩니다.

로그인하면 이런 화면이 나옵니다.
(화면: 메인 화면)
무서워 보이죠? 걱정 마세요. 오늘 쓸 건 3가지뿐입니다."

## [4:00~12:00] Step 2: 첫 자동화 만들기 (8분, 무편집)

**[화면: 전체 화면 녹화 — 무편집으로 실시간]**

"자, 시작합니다. 타이머 켭니다.

**4:00~5:30 — 트리거: '이메일 오면 시작'**
새 프로젝트를 만듭니다.
(화면: 프로젝트 생성)
'트리거'를 추가합니다. 쉽게 말하면 '이 일이 일어나면 자동화 시작해'라는 조건이에요.
Gmail 선택 → 계정 연결 → '새 이메일 수신 시' 선택.
(화면: 트리거 설정)
끝. 이제 이메일이 오면 자동으로 감지합니다.

**5:30~8:00 — AI 분류: '이메일을 읽고 분류해'**
AI 노드를 추가합니다.
(화면: AI 노드 추가)
프롬프트를 씁니다. 이게 핵심이에요.

'이 이메일의 내용을 읽고, 다음 4가지 카테고리 중 하나로 분류해줘:
1. 환불 요청
2. 배송 문의
3. 제품 문의
4. 기타
카테고리 이름만 답변해줘.'

(화면: 프롬프트 입력)
이게 전부입니다. AI한테 '뭘 해달라'고 말하는 것 = 프롬프트.

**8:00~10:00 — 알림: '결과를 슬랙으로 보내'**
슬랙 노드를 추가합니다.
(화면: 슬랙 연결)
메시지 내용:
'새 이메일 도착!
분류: {{AI 분류 결과}}
발신자: {{보낸 사람}}
제목: {{이메일 제목}}'
(화면: 메시지 설정)

**10:00~12:00 — 테스트!**
전체 워크플로우를 저장하고 테스트합니다.
테스트 이메일을 보내볼게요.

(화면: 테스트 이메일 발송)
'환불 해주세요. 주문번호 12345. 사이즈가 안 맞아요.'

3... 2... 1...

(화면: 슬랙 알림 도착)
'새 이메일 도착!
분류: 환불 요청
발신자: test@test.com
제목: 환불 요청합니다'

됐습니다! 타이머 — 11분 23초.

12분도 안 돼서 첫 자동화가 완성됐습니다."

## [12:00~14:00] Step 3: 업그레이드 팁

"기본이 되면 이것도 추가하세요.

1. **AI 답변 초안 자동 생성**: 분류 결과에 따라 답변까지 만들어줌.

2. **Notion에 자동 저장**: 모든 이메일을 Notion 데이터베이스에 자동 기록. 나중에 분석용.

3. **조건 분기**: 환불이면 → A 담당자, 배송이면 → B 담당자에게 각각 알림.

이건 다음 영상에서 다룰게요. 오늘은 기본만."
