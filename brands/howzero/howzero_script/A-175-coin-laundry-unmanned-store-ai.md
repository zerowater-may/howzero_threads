### 문제 정의 (0:40–3:30)
코인세탁·무인매장의 3대 병목. 첫째, 장비 관리 — 세탁기·건조기 고장을 고객 신고 후에야 파악. 고장 1대당 일 매출 2~3만 원 손실. 둘째, 현장 관리 — 청소 상태, 잔돈기 보충, 세제 보충 상태 원격 파악 불가. 셋째, 고객 대응 — 세탁물 손상, 기기 작동 안 됨 등 민원을 전화로만 처리. 심야 시간 대응 불가.

### 자동화 설계 (3:30–9:30)
**장비 모니터링 자동화**: 세탁기·건조기 가동 데이터 수집(결제 기록 기반 — IoT 없이 가능) → 비정상 패턴 AI 감지("3번 세탁기 일 가동 횟수 평소 대비 60%↓ — 고장 의심") → 즉시 관리자 알림 → 수리 업체 자동 연락 → 수리 이력 자동 기록.

**현장 관리 자동화**: 일별 체크리스트 자동 생성(청소, 잔돈 보충, 세제 보충, 쓰레기통) → 관리 직원에게 자동 배정 → 완료 보고(사진 첨부) → 미완료 시 자동 에스컬레이션.

**고객 대응 자동화**: 카카오 챗봇으로 24시간 기본 문의 자동 응대(사용법, 요금, 영업시간) → 민원 접수 자동화(기기 번호+증상 입력) → 긴급 민원(세탁물 손상) 즉시 관리자 알림 → 환불 기준 내 자동 처리.

**매출 최적화**: 시간대별·요일별 가동률 자동 분석 → 비수기 프로모션 자동 생성("평일 오전 10~12시 20% 할인") → 주변 아파트·오피스텔 대상 타겟 마케팅 → 프로모션 효과 자동 측정.
