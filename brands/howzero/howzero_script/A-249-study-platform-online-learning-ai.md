## 대본

### 훅 (0:00–0:40)
온라인 스터디 플랫폼 운영하시는 분, 완주율 몇 퍼센트세요? 업계 평균 15%입니다. 85%가 중도 이탈합니다. 이유는 간단합니다. 혼자 하면 포기하고, 맞지 않는 스터디에 배정되고, 진도 관리가 안 되니까요. AI가 학습자 매칭, 진도 추적, 이탈 예측, 맞춤 리마인더를 자동 처리하면 완주율 3배. 15%에서 48%로. 진짜 숫자입니다.

### 문제 정의 (0:40–3:00)
스터디/학습 플랫폼의 병목. 첫째, 스터디 매칭 — 수준·목표·시간대가 맞는 사람끼리 묶어야 하는데 수동입니다. 미스매칭이 이탈의 1번 원인. 둘째, 진도 관리 — 수백 개 스터디의 진행 상태를 운영자가 트래킹하기 불가능합니다. 셋째, 이탈 방지 — 이탈 징후(접속 감소, 과제 미제출)를 감지할 수 없습니다. 떠나고 나서 알게 됩니다. 넷째, 콘텐츠 큐레이션 — 학습자 수준에 맞는 콘텐츠를 추천하지 못합니다.

### 자동화 설계 (3:00–8:30)
**AI 스터디 매칭**: 가입 시 목표·수준·가용 시간·선호 학습 방식 수집 → AI가 최적 그룹 자동 매칭(수준 유사성 + 시간대 일치 + 목표 정렬) → 매칭 후 첫 주 만족도 자동 수집 → 불만족 시 재매칭 자동 제안.

**진도 자동 추적+리포트**: 학습자별 접속 빈도, 과제 완료율, 퀴즈 점수 자동 수집 → 주간 진도 리포트 자동 발송(개인+그룹) → 우수 학습자 자동 인정(배지·리더보드) → 운영자 대시보드에 전체 스터디 현황 요약.

**이탈 예측+자동 개입**: 접속 감소, 과제 미제출, 커뮤니케이션 감소 패턴 AI 감지 → 이탈 확률 70% 이상 시 자동 개입(격려 메시지 → 1:1 상담 연결 → 스터디 변경 제안) → 개입 후 반응 추적.

**맞춤 콘텐츠 추천**: 학습 이력 + 퀴즈 결과 분석 → 부족한 영역 자동 파악 → 보충 콘텐츠 자동 추천 → 난이도 자동 조절.
