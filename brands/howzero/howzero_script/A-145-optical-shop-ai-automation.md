## 대본

### 훅 (0:00–0:40)
안경점에서 가장 아까운 고객이 누구인지 아십니까. 2년 전에 안경 맞추고 다시 안 오는 고객입니다. 시력 변화, 렌즈 교체 시기, 자외선 차단 코팅 마모 — 재방문 사유는 충분한데 안내를 안 합니다. 안경점의 평균 재방문율이 25%입니다. AI로 고객 관리, 렌즈 맞춤 추천, 재구매 유도를 자동화해서 재방문율을 2배 올린 사례를 보여드리겠습니다.

### 문제 정의 (0:40–3:00)
안경점의 3대 병목. 첫째, 고객 데이터 활용 — 시력, 렌즈 종류, 프레임 선호, 구매 이력이 다 있는데 활용을 못 합니다. 종이 처방전에 쓰고 서랍에 넣어둠. 둘째, 맞춤 추천 — 고객마다 생활 패턴(사무직/야외/학생), 시력 변화 속도, 예산이 다른데 일괄 안내. 셋째, 재방문 유도 — 안경은 1~2년 주기 교체 상품인데, 교체 시기 안내 시스템이 없으면 경쟁 안경점으로 이동.

### 자동화 설계 (3:00–8:00)
**고객 DB 자동 구축**: 안경 구매 시 고객 정보 구글 시트 입력 — 시력, 렌즈 종류, 프레임, 생활 패턴, 구매일 → 전 고객 이력 검색 가능 → 재방문 시 즉시 이력 조회.

**AI 맞춤 렌즈 추천**: 고객 정보(시력, 나이, 직업, 기존 렌즈) 입력 → AI가 최적 렌즈 조합 추천(블루라이트 차단, 변색 렌즈, 누진다초점 등) → 가격대별 옵션 3가지 제시 → 추천 근거 설명.

**재방문 자동화**: 구매 후 1년 시점 안내("시력 변화 확인 시기입니다. 무료 시력 검사 예약하세요") → 렌즈 코팅 수명 기반 교체 안내(18개월) → 시즌별 프로모션(신학기 학생 할인, 여름 선글라스) → 생일 쿠폰 자동 발송.

**시력 변화 추적**: 방문 시마다 시력 기록 → 변화 추이 자동 분석 → 급격한 변화 시 안과 방문 권유 → 고객에게 시력 리포트 제공(신뢰 구축).

### 데모/사례 (8:00–13:30)
안경점 OT. 월 고객 150명, 단골 재방문율 25%. 자동화 전: 고객 이력 종이 관리, 재방문 안내 없음, 렌즈 추천 경험 의존, 시즌 프로모션 수동(불규칙). 자동화 후: 고객 DB 디지털화 — 재방문 시 즉시 이력 확인(고객 감동). 재방문 안내 자동화로 재방문율 25%→52%. AI 렌즈 추천으로 고가 렌즈 전환율 15%↑. 시즌 프로모션 자동 발송으로 프로모션 매출 35%↑.
