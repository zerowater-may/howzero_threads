## 대본

### 훅 (0:00–0:40)
회사 규정집 마지막으로 업데이트한 게 언제세요? 근로기준법은 바뀌었는데 취업규칙은 2년 전 그대로? 개인정보보호법 개정됐는데 사내 정책은 옛날 버전? 이거 노동청 감사 들어오면 바로 걸립니다. AI가 법 바뀌면 관련 사내 규정을 자동으로 찾아서 수정 제안합니다. 코딩 0줄.

### 문제 정의 (0:40–3:00)
사내 규정 관리의 병목. 첫째, 법 변경 추적 — 근로기준법, 개인정보보호법, 산업안전보건법 등 관련 법규가 수시로 바뀝니다. 둘째, 영향 매핑 — 법이 바뀌면 우리 사내 규정 중 어떤 조항이 영향받는지 찾아야 합니다. 규정집이 200페이지면 일일이 대조해야 합니다. 셋째, 배포·교육 — 규정이 바뀌었다는 걸 직원들에게 알리고, 교육해야 합니다.

### 자동화 설계 (3:00–8:00)
**법 변경 → 규정 영향 자동 매핑**: 법규 변경 감지(A-208 시스템 연동) → AI가 사내 규정집에서 관련 조항 자동 검색 → 영향받는 조항 리스트 + 현행 내용 vs 법 변경 내용 비교표 자동 생성.

**수정안 자동 제안**: 영향받는 조항 → AI가 법 변경에 맞춘 수정안 자동 작성 → 인사/법무 담당자 검토 → 승인 시 규정집 자동 업데이트.

**버전 관리 자동화**: 규정 변경 시 → 변경 이력(날짜, 변경 조항, 사유) 자동 기록 → 이전 버전 자동 보관 → 변경 히스토리 검색 가능.

**직원 공지·교육 자동화**: 규정 업데이트 → 전 직원 알림 자동 발송 → 변경 사항 요약 + "읽었습니다" 확인 수집 → 주요 변경은 퀴즈형 교육 자동 생성.
