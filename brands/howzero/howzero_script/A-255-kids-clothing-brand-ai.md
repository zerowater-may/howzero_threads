### 문제 정의 (0:40–3:00)
키즈 의류의 특수한 병목. 첫째, 사이즈 이슈 — 브랜드마다 사이즈 기준이 다르고, 아이 성장 속도가 제각각입니다. 둘째, 시즌성 극심 — 입학 시즌, 여름, 겨울 수요 편차가 큽니다. 재고 예측 실패 시 시즌 지나면 가치 0. 셋째, 타겟 고객 소통 — 구매자(부모)와 사용자(아이)가 다릅니다. 마케팅 메시지가 복잡합니다. 넷째, 트렌드 변화 — 캐릭터·컬러 트렌드가 빠르게 바뀝니다. 맘카페, 인스타 등에서 실시간 파악이 필요합니다.

### 자동화 설계 (3:00–8:00)
**AI 사이즈 추천**: 아이 나이·키·몸무게 입력 → 해당 브랜드 사이즈 차트 + 과거 구매자 피팅 데이터 AI 분석 → 최적 사이즈 추천 + "6개월 후에는 이 사이즈" 성장 예측 → 교환/반품 시 사이즈 데이터 자동 업데이트.

**시즌 수요 예측**: 과거 3년 시즌별 판매 데이터 + 날씨 예보 + 입학/졸업 시즌 캘린더 → AI가 품목별 수요 예측 → 생산/발주량 자동 산출 → 재고 과잉/부족 시뮬레이션.

**맘카페/SNS 트렌드 자동 분석**: 주요 맘카페·인스타·핀터레스트 자동 모니터링 → 인기 캐릭터·컬러·스타일 트렌드 주간 리포트 → 다음 시즌 기획에 데이터 반영.

**생애주기 마케팅**: 첫 구매 시 아이 생년월일 수집 → 성장 단계별 사이즈 업 시점에 맞춤 추천("00이가 곧 110사이즈가 필요할 시기예요") → 시즌 시작 2주 전 선제적 추천 → 형제/자매 등록 시 추가 할인.
