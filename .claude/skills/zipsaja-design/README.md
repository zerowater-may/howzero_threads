# 집사자 (zipsaja) Design System

> 20–30대 부린이·신혼부부를 위한 내집마련 부동산 큐레이션 인스타그램 계정.
> 노란 사자 + 흰 날개 마스코트가 **반말로 부동산 정보를 친구처럼 정리해주는 카드뉴스 콘텐츠**.

집사자의 결과물은 단 하나의 표면 — **인스타그램 캐러셀 (1080×1350px, 4:5, 10장 이하)** — 으로 출력되는 카드뉴스다. 이 디자인 시스템은 "옆집 오빠가 노트에 정리해준" 느낌을 픽셀 단위로 재현하기 위한 토큰·컴포넌트·문체 가이드의 모음이다.

## 컨텍스트

- **타겟**: 20–30대, 부동산 첫 경험자(부린이) + 신혼부부.
- **표면**: Instagram Carousel (1080×1350 / 4:5).
- **포맷**: 10장 구성 — 커버 → 도입 → 조건 → 매물 5개 → 정리 → CTA.
- **목적**: 매물 큐레이션을 친구의 노트 필기처럼 빠르게 훑게 한다.

## CONTENT FUNDAMENTALS — 카피 라이팅

### 톤앤매너
- **반말**. 무조건. "옆집 오빠/누나가 노트에 휘갈겨 준 느낌".
- **친근**. 따뜻하고 약간의 농담 섞임. 친구 톤.
- **존댓말 절대 금지**. "~합니다 / ~입니다" 사용 금지.
- **도발·위협 금지**. "끝났다 / 망한다 / 90%가 모른다" 등 자극형 훅 금지 (이는 sibling 브랜드 `braveyong` 톤이며 집사자와는 정반대).
- **이모지 금지**. (캐러셀 본문, 캡션 모두.)
- **그라디언트 금지**. (시각·언어 모두 — 메시지를 흐리게 하는 모든 것.)

### 인칭
- 화자 = **집사자(사자 마스코트)** = "내가 / 우리가". 가끔 "집사자가" 3인칭 자칭.
- 청자 = **너 / 너희** ("당신", "여러분" 사용 금지).

### 케이싱 / 부호
- 한국어 문장은 **마침표 생략 가능** — 노트 메모 톤이라 더 자연스럽다.
- 느낌표는 슬라이드당 최대 1개. 남발 금지.
- 숫자는 **아라비아 숫자** + 단위 한글 ("3억 5천", "전용 59㎡", "역세권 5분").
- 강조 단어는 **오렌지 Pill** 또는 **오렌지 형광펜** 으로 시각 처리 (텍스트로 *별표* 같은 것 쓰지 말 것).

### 좋은 예시
- "이 가격에 이 컨디션? 진짜 드물어"
- "내가 정리해봤어 — 너 조건이면 이 5개"
- "체크포인트 — 등기부 꼭 확인"
- "발품 팔기 전에 이거 먼저 봐"
- "예산 5억, 신혼부부, 출퇴근 1시간 이내"

### 나쁜 예시 (이 톤에서 금지)
- ❌ "지금 안 사면 후회합니다" (위협 + 존댓말)
- ❌ "90%가 모르는 부동산 진실" (도발형 훅)
- ❌ "안녕하세요 여러분 ✨" (존댓말 + 이모지)
- ❌ "내 집 마련 가이드를 알아보아요!" (어미 흐림 + 권유형)

### 슬라이드별 카피 길이
- **커버 hook**: 한 줄. 5~10자 핵심 단어. (예: "5억 이하 신혼집")
- **본문 헤드라인**: 1~2줄. 30자 이내.
- **본문 설명**: 3줄 이내. 슬라이드 = 1메시지 원칙.
- **마스코트 말풍선** (Gaegu): 1줄. 추임새·첨언·반응. (예: "이 매물 진짜 괜찮아", "근데 단점은…")

## VISUAL FOUNDATIONS

### 색
- **단 한 가지 액센트** = 오렌지 `#EA2E00`. 유일한 "컬러"다. (CSS 변수는 호환성 위해 `--zs-yellow`로 유지, 값만 오렌지)
- 보조 액센트 (pressed / 진한 강조) = `#C42600`.
- 배경은 거의 항상 **베이지 `#F0E7D6`** (메인 캔버스). 보조 박스 = **연한 베이지 `#F5EDE0`** 또는 **순백 `#FFFFFF`** (사진 컨테이너용).
- 텍스트·테두리는 **잉크 블랙 `#1A1A1A`** (순흑 `#000` 사용 금지 — 너무 차갑다).
- 보조 의미색(통과/경고/금지)은 작은 아이콘 표시에만, 면적 색으로 사용 금지.
- **그라디언트 금지**. 단색 면 + 단색 보더만.

### 타이포 — 2 + 1 시스템
| 역할 | 폰트 | 무게 | 용도 |
|------|------|------|------|
| **Headline** | `Gmarket Sans` | Bold 700 | 제목·하이라이트·숫자·브랜드명. 둥근 한글. |
| **Body** | `Gmarket Sans` | Medium 500 / Bold 700 | 본문·설명·라벨. 폴백 `Noto Sans KR`. |
| **Hand / Whisper** | `Gaegu` | 700 | 마스코트 말풍선·사이드 코멘트·손글씨 코너. |

> Gmarket Sans 는 로컬 `fonts/` 에 TTF 동봉 (Light/Medium/Bold). Gaegu·Noto Sans KR 는 Google Fonts CDN.
> CDN + @font-face = `colors_and_type.css` 상단에 모두 정의됨. **substitution flag: 없음.**

### 배경
- **베이지** 캔버스가 기본. 흰색은 사진 박스나 카드 강조 시에만 보조로 사용.
- 패턴/텍스처는 **오렌지 Pill의 점박이 hatching** 한 군데에서만 사용.
- 텍스처·그레인 효과 금지.

### 보더 / 카드 / 라디우스
- **검은 굵은 테두리 3px solid `#1A1A1A`** — 모든 박스의 시그니처.
- 카드 라디우스 = **16px** (작은 태그는 8px, 필은 999px).
- 모서리 컷·각진 박스 금지 — 부드럽고 둥글게.

### 그림자
- 외부 그림자는 **오프셋 블록 셰도우** `4px 4px 0 0 #1A1A1A` (스탬프형) 또는 미사용.
- 흐릿한 blurred drop shadow 금지 — 노트북 메모 톤과 안 맞다.
- 인너 셰도우 금지.

### 핵심 시각 패턴 (signature)
1. **오렌지 Pill 하이라이트** — `border-radius: 999px`, 점박이 hatching. 핵심 단어 강조.
2. **검은 굵은 테두리 박스** — `border-radius: 16px`. 모든 정보 컨테이너의 기본형.
3. **Check Point 박스** — 진한 오렌지(`#C42600`) 면 + 흰 텍스트 + 검은 테두리 + Gmarket Sans 헤드라인. 슬라이드당 1개.
4. **마스코트 말풍선** — Gaegu 폰트. 꼬리(SVG) 가 마스코트 입과 연결.
5. **연한 베이지 정보 박스 `#F5EDE0`** — 부가 정보 / 표 / 매물 스펙 컨테이너.
6. **오렌지 형광펜** — 본문 텍스트 강조. 오렌지만.

### 레이아웃
- 슬라이드 패딩: 좌우 80px / 상단 100px / 하단 140px (footer 영역 확보).
- 그리드 12-column 사용 안 함 — 콘텐츠 중심 자유 배치.
- 모든 텍스트 **좌측 정렬** 기본. 커버·CTA·통계만 중앙 정렬.
- footer 고정: 좌측 `@zipsaja` 핸들 + 우측 페이지 번호 `1/10`.

### 애니메이션 / 인터랙션
- 출력물이 **정적 PNG** 이라 애니메이션은 디자인 토큰에 없다.
- 웹 미리보기에서만: hover 시 카드 `transform: translate(-2px, -2px)` + shadow 깊어짐. ease 200ms.
- press 시 카드 shadow 0 으로 흡수 (스탬프가 눌린 느낌).

### 투명도 / 블러
- 사용 안 함. 투명도는 워터마크(`opacity: 0.08~0.12`) 한 곳에서만.
- backdrop-filter blur 금지.

### 이미지 톤
- 마스코트가 메인 비주얼이라 사진 거의 없음. 매물 사진이 들어갈 때는 `border-radius: 16px` + 검은 3px 테두리로 통일.
- 컬러 보정 없음 (warm/cool/grain 등). 원본 그대로.

## ICONOGRAPHY

- **메인 비주얼 = 마스코트** (`assets/mascots/`). 표정 변형으로 톤을 전달한다.
- **이모지 사용 금지** (브랜드 룰).
- **유니코드 기호 제한적 사용 가능**:
  - `→` (다음 슬라이드 안내, footer)
  - `✓` (체크포인트), `✗` (금지)
  - `●` (불릿)
- 라인 아이콘이 필요하면 [Lucide](https://lucide.dev/) (CDN) — 검은 1.5~2px stroke, 18~24px. 마스코트의 라인 아트 톤과 매치된다.
- **AI가 그린 SVG 일러스트 금지** — 마스코트 외 일러스트는 사용하지 않는다.
- **대체 / 누락 플래그**:
  - 마스코트 10종 전부 첨부 완료 (hero / default / blank / smile / happy / shining / surprise / worried / angry / side).
  - Lucide 는 substitution 이 아니라 보조 라이브러리 — 브랜드 사이트나 캐러셀에 거의 등장하지 않음.

## 사용 가이드

```html
<link rel="stylesheet" href="colors_and_type.css">
```

```html
<h1 class="t-h1">5억 이하 <span class="zs-pill">신혼집</span> 5개</h1>
<div class="zs-box-cream zs-stamp">
  <p class="t-body">서울 동북권 · 전용 59㎡ · 출퇴근 1시간 이내</p>
</div>
<p class="t-bubble">내가 발품 다 팔아봤어</p>
```

캐러셀 슬라이드 전체를 볼 거라면 `ui_kits/carousel/index.html` 을 열어라.
