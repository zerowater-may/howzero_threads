## 타겟 오디언스

- AI 도구 일상적으로 쓰는 개발자/직장인
- 테크 밈에 공감하는 20~40대
- "여보 이거 봐봐" 하면서 GPT/Claude 자랑하는 사람들
- AI 커뮤니티 (트위터, 인스타, 디스코드)

## 톤앤매너

- **공감형 유머**: "아 맞아 나도 이러는데ㅋㅋ" 반응이 목표
- **부부 대화체**: 존댓말/반말 혼용. "여보~", "이봐", "아하하하... 그...러게요?"
- **자조적 과장**: AI 광인의 행동을 극단적으로 과장해서 웃김 만듦
- **테크 용어 자연 삽입**: 토큰, API rate limit, Opus, Sonnet, 컨텍스트 등을 일상 대화에 섞음
- **4컷 만화 리듬**: 기승전결. 마지막 컷에 펀치라인/반전

### 좋은 문장 예시

- "후후 이건 완전 제 전문 분야라 볼 수 있죠"
- "시즌 138번째 ai 제안이군"
- "안 써봐주면 빼지는 녀석아"
- "개발 완료 직전에 토큰이 다 떨어짐"
- "와 여보 이거 미.. ...2년?"

### 나쁜 문장 (이 스타일에서 금지)
- "AI 트렌드를 분석해보겠습니다" (강의 톤 → 만화 톤 깨짐)
- "지금 당장 시작하세요!" (CTA 강요 → 이 계정은 CTA 없이 순수 콘텐츠)
- 전문가 어투 / 인사이트 정리 / 리스트형 정보 전달

## 규칙 — 스타일 DNA

### 반드시 할 것

- **순백 배경 + 검정 선화만**: 색은 아내의 빨간 리본(`#E53935`)만. 나머지 전부 흑백.
- **떡 캐릭터**: 모든 인물은 둥근 떡/찐빵 형태. 몸통과 머리 구분 없음. 팔다리는 단순 선.
- **안경 = 남편, 리본 = 아내**: 이 두 가지만으로 캐릭터 식별.
- **에피소드 번호 박스**: 각 슬라이드 좌상단에 `N. 제목문장` 형태의 검은 테두리 박스.
- **대사는 캐릭터 옆에 직접 배치**: 말풍선 없이 캐릭터 근처에 텍스트 배치 (간혹 말풍선 사용).
- **의성어/효과음 손글씨**: "뿌에에엥-", "피상", "졸졸졸..." 등 Gaegu 폰트로 산발적 배치.
- **컷 구분 없음**: 4컷 만화이지만 칸 나누기 선 없이 여백으로 흐름 구분.
- **실제 AI 도구명 언급**: GPT, Claude, Gemini, Opus, Sonnet, NotebookLM, 토큰 등 실명 사용.

### 절대 하지 말 것

| 금지 | 이유 |
|------|------|
| 컬러풀한 배경/그라디언트 | 흑백 손그림이 본질. 색 넣으면 웹툰 느낌 사라짐 |
| 깔끔한 벡터 일러스트 | AI 생성 느낌 남. 투박한 손그림이어야 함 |
| 정보 전달형 레이아웃 (리스트, 체크박스) | 이건 만화지 카드뉴스가 아님 |
| CTA (팔로우, 저장, 신청) | 순수 콘텐츠 계정. CTA 없음 |
| 이모지 | 의성어/효과음으로 감정 표현 |
| 캐릭터에 디테일 추가 (옷, 머리카락 등) | 떡 형태의 미니멀리즘이 정체성 |
| 빨간 리본 외 다른 색 사용 | 유일한 컬러 포인트를 지켜야 정체성 유지 |

2. **에피소드 1** — `1. 제목문장` 박스 + 2~3컷 만화

## 에피소드 제목 스타일

에피소드 제목은 **단정적 서술체**로 작성. "~한다.", "~있다." 끝맺음.

- "이들은 잘 때나 깨어있을 때나 24시간 단 한 순간도 쉬지 않고 AI를 한다."
- "조용하다가도 AI 이야기를 꺼내면 멈추질 않는다. 고장난 라디오 같다."
- "자꾸 뭘 만들어서 가져온다. 자꾸 써보라고 해서 상당히 귀찮다."
- "토큰이 뭔진 모르겠지만 아무튼 토큰이 다 떨어져가면 불안해 한다."
- "심심하면 아 인간 시대의 끝이 도래했다고 징징 댄다."
