# 용감한용팀장 - 인게이지먼트 플레이북

인스타그램 캐러셀 업로드 후 팔로워 인게이지먼트를 끌어올리는 설문·스토리·댓글 유도 전략.

## 핵심 원칙

1. **캐러셀만으로 끝내지 않는다.** 캡션→스토리→DM→단톡방으로 이어지는 깔때기를 만든다.
2. **자가진단 유도.** "내가 어디 단계지?"를 스스로 묻게 만들면 저장·공유가 따라온다.
3. **모든 설문은 단톡방 유입으로 귀결.** 답변 받으면 DM으로 맞춤 반응 후 단톡방 초대.
4. **도발은 유지.** "90%는 안 할 것"을 섞어야 상위 1% 심리가 작동한다.

---

## 1. 스토리 투표 스티커 (A vs B)

**목적**: 자가 포지셔닝 + 청중 세그먼트 파악.

### 템플릿

```
질문: {타겟의 현재 상태 묻기}
보기: 🅰 {일반적/잘못된 선택}   🅱 {올바른/상위 1% 선택}

→ A 선택자: "{경고 메시지}" DM
→ B 선택자: "{칭찬 + 다음 스텝}" DM + 단톡방 초대
```

### 실전 예시 (구매대행)

- Q: "당신의 스마트스토어, 지금 상품 몇 개?" A: 1,000개 미만 · B: 1,000개 이상
- Q: "상품명, 직접 쓰시나요?" A: 소싱 프로그램 자동 · B: 하나하나 직접 설계
- Q: "'1,000개 제한' 소식 첫 반응?" A: 망했다 · B: 오히려 기회다
- Q: "내 상품 1페이지에 뜨나요?" A: 네 · B: 확인 안 해봤음

## 2. 스토리 퀴즈 스티커 (4지선다)

**목적**: 지식 체크 + 정답 공개로 권위 강화.

### 템플릿

```
질문: {캐러셀 핵심 팩트를 문제화}
보기: ① {오답} ② {근접 오답} ③ {정답} ④ {함정}
정답: ③
해설: {왜 그런지 1~2줄}
```

### 실전 예시

- Q: "최근 3개월 매출 얼마면 5,000개 한도?" 정답: ③ 500만 원
- Q: "'판매 상품 비중 3%'가 의미하는 것?" 정답: ② 상품 종 개수 3%
- Q: "효자상품 1개가 100번 팔리면 몇 종?" 정답: ① 1종
- Q: "'업소용 육수통' 검색 실제 노출 카테고리?" 정답: ② 곰솥 들통

## 3. 질문 박스 스티커 (자유 답변)

**목적**: 다음 콘텐츠 주제 확보 + 1:1 관계 형성.

### 템플릿

```
"{정체성 질문 또는 공감 질문 1줄}
 {답변 가이드 1줄}"
```

### 실전 예시

- "당신은 '가짜 셀러'인가요 '진짜 셀러'인가요? 그렇게 생각한 이유 1줄로 써주세요."
- "{이슈} 듣고 가장 먼저 든 생각은?"
- "내 스토어 효자상품 1개만 말해주세요. 같이 분석해드립니다."
- "{주제} 가장 막막한 순간은? 익명으로 답해주세요. 다음 캐러셀 주제 됩니다."
- "{세부 행동} 어떻게 하세요? 같이 얘기해봅시다."

## 4. 슬라이더 스티커 (감정·공감)

**목적**: 가벼운 참여 유도. 수치화된 감정 데이터 수집.

### 템플릿

```
"{감정 또는 자가 평가 문구}"
(왼쪽 이모지/숫자 ———— 오른쪽 이모지/숫자)
```

### 실전 예시

- "{이슈} 얼마나 충격이었나요?" (😌 편안함 ———— 😱 절망)
- "내 {행동/자산}, 몇 점 주시겠어요?" (0점 ———— 100점)
- "{주제} 계속 할 확신, 몇 %?" (0% ———— 100%)

## 5. 캐러셀 댓글 유도 (캡션 끝 부착)

**목적**: 알고리즘용 댓글 수 확보 + 저장 유도.

### 패턴 5종

#### 패턴 ① 간단 선언
```
"이거 보고 '오늘 밤 점검한다' 댓글 남겨주세요.
 3일 뒤 결과 공유하겠습니다."
```

#### 패턴 ② 티어 체크
```
"댓글로 지금 당신 상태 알려주세요:
  🔴 A - {위험 상태}
  🟡 B - {중간 상태}
  🟢 C - {상위 1% 상태}
 A, B 나오신 분들 DM으로 무료 진단 도와드립니다."
```

#### 패턴 ③ 키워드 수집
```
"당신 {영역}에서 {X} 1개만 댓글로 적어주세요.
 {가치 제공} 해드립니다. (선착순 20명)"
```

#### 패턴 ④ 공감 유도
```
"'이거 내 얘기다' 싶으면 🙋‍♂️ 이모지 하나만.
 당신만 그런 게 아닙니다. 같이 바꿔봅시다."
```

#### 패턴 ⑤ 저장 유도
```
"이 캐러셀 저장 안 하고 넘기면
 3일 뒤 후회합니다. 진짭니다.
 🔖 저장 버튼 지금 누르세요."
```
